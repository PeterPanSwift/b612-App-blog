安裝 Flutter App 的第三方套件( package) — 以 google fonts 為例

開發 Flutter App 時，我們時常會安裝別人寫好的 package 快速實現某個功能。以下我們以 package google_fonts 為例，一步步示範如何在 Flutter App 加入第三方 package。

  • 收藏 package 的 pub.dev。
  • 搜尋 package。
  • 安裝 package。
  • 使用 package。

收藏 package 的 pub.dev

pub.dev 上收集了適合 Flutter & Dart 程式的 package，我們可以在上面挖寶尋找喜歡的 package。

Dart packages
Pub is the package manager for the Dart programming language, containing reusable libraries & packages for Flutter and…pub.dev

搜尋 package

搜尋 google fonts，結果如下圖所示，我們的目標是出現在第一個的 google_fonts，它的 LIKES、POINTS & DOWNLOADS 都極高，表示它是一級棒的 package。

google_fonts | Flutter Package
A Flutter package to use fonts from fonts.google.com. Supports HTTP fetching, caching, and asset bundling.pub.dev

點進 google_fonts 的頁面後，可以看到它的詳細介紹和說明。

安裝 package

  • 方法 1: 利用 flutter pub add。

打開 IDE 的 terminal。

輸入以下指令安裝 package，在 flutter pub add 後接 package 的名稱。

flutter pub add google_fonts

ps: 若是開發 dart 的 command line 程式，可用以下指令安裝 package。

dart pub add google_fonts

剛剛的指令會在 pubspec.yaml 加入 package 的名字和安裝 package，在 dependencies 下長出 google_fonts: ^6.1.0。

dependencies:
  flutter:
    sdk: flutter


  # The following adds the Cupertino Icons font to your application.
  # Use with the CupertinoIcons class for iOS style icons.
  cupertino_icons: ^1.0.2
  google_fonts: ^6.1.0

dependencies 區塊設定安裝的 package 清單，版本號碼開頭的 ^ 可限制 package 的版本，因此以上例子的 google_fonts 只會抓取 >= 6.1.0 和 < 7.0.0 的版本。關於 Package dependencies 的詳細說明可參考以下連結。

Package dependencies
Add other packages to your app. Specify package locations, version constraints, and more.dart.dev
  • 方法 2: 設定 pubspec.yaml 和取得 package。

在 pubspec.yaml 的 dependencies 加入 google_fonts。yaml 要注意縮排要一致，以下例子的 flutter、cupertino_icons、google_fonts 前都是兩個空白。

dependencies:
  flutter:
    sdk: flutter

  # The following adds the Cupertino Icons font to your application.
  # Use with the CupertinoIcons class for iOS style icons.
  cupertino_icons: ^1.0.2
  google_fonts: ^6.1.0

點選右上的下載按鈕安裝，或是在 VS Code / Android Studio 的 terminal 輸入 flutter pub get。

  • VS Code 的下載按鈕。
  • Android Studio 的下載按鈕 Pub get。
  • 方法 3: 點選 add xxx to dependencies。

若在程式裡 import package 出現找不到 dart 檔的錯誤，可點選 Quick Fix 的 add xxx to dependencies，它將自動幫我們安裝套件。

從 PROBLEMS 的 URI doesn’t exist 的右鍵選單也可以點選 add xxx to dependencies。

  • 方法 4: 使用 VS Code 的 extension Pubspec Assist。
Pubspec Assist - Visual Studio Marketplace
Extension for Visual Studio Code - Easily add and update dependencies to your Dart and Flutter project.marketplace.visualstudio.com

安裝 VS Code 的 extension Pubspec Assist。

從 Command Palette 點選 Pubspec Assist: Add/update dependencies。

在框框裡輸入 package 的名稱後按 enter 安裝，若有多個 package，還可以一次輸入，package 名字間用逗號分隔。以下我們輸入 google_fonts。

此時 pubspec.yaml 也會自動加入 google_fonts。

dependencies:
  cupertino_icons: ^1.0.2
  flutter:
    sdk: flutter
  google_fonts: ^6.1.0

使用 package

為了在程式裡使用 package，我們必須 import 它的相關檔案才能使用，import package:google_fonts/google_fonts.dart 後即可開心地使用豐富的 Google 字型，以下我們設定 Text 的字型是手寫字型 GoogleFonts.shadowsIntoLight。

ps: 從以下連結可瀏覽 Google Fonts 有哪些有趣的字體。

Browse Fonts - Google Fonts
Making the web more beautiful, fast, and open through great typographyfonts.google.com
  • 寫法 1。
import 'package:flutter/material.dart';
import 'package:google_fonts/google_fonts.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  // This widget is the root of your application.
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: Center(
          child: Text(
            'peter pan loves SwiftUI',
            style: GoogleFonts.shadowsIntoLight(
              fontSize: 30,
            ),
          ),
        ),
      ),
    );
  }
}
  • 寫法 2。
import 'package:flutter/material.dart';
import 'package:google_fonts/google_fonts.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  // This widget is the root of your application.
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: Center(
          child: Text(
            'peter pan loves SwiftUI',
            style: GoogleFonts.shadowsIntoLight(
              textStyle: const TextStyle(
                fontSize: 30,
              ),
            ),
          ),
        ),
      ),
    );
  }
}

App 畫面成功出現了漂亮的手寫字型。
