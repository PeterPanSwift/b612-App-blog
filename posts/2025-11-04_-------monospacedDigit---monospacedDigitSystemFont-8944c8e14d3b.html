讓數字等寬的 monospacedDigit & monospacedDigitSystemFont

當我們在 iOS App 顯示數字時，有時會發現數字變化時造成文字抖動，比方以下顯示時間的例子。

import SwiftUI

struct ContentView: View {
    var body: some View {
        TimelineView(.periodic(from: .now, by: 1)) { context in
            Text(context.date.formatted(date: .omitted, time: .standard))
                .font(.largeTitle)
        }
    }
}

問題來自數字有不同的寬度，因此數字改變時會有些許的位移。

在 SwiftUI 透過讓數字等寬的 modfier monospacedDigit 可解決此問題。

struct ContentView: View {
    var body: some View {
        TimelineView(.periodic(from: .now, by: 1)) { context in
            Text(context.date.formatted(date: .omitted, time: .standard))
                .font(.largeTitle.monospacedDigit())
        }
    }
}

若是 UIKit App 的 UIFont，則可透過 monospacedDigitSystemFont(ofSize:weight:) 生成等寬的數字。

UIFont.monospacedDigitSystemFont(ofSize: 30, weight: .bold)
