如何對 The Swift Programming Language 全文搜尋

從 The Swift Programming Language 官網我們可以閱讀 Swift 程式語法的詳細內容，不過有一個小小可惜的地方，它並沒有提供全文搜尋的功能。

從官網左下的 Filter 框框可以搜尋章節關鍵字，但是卻無法對 swift book 整本書進行搜尋，比方查詢 escaping 只會出現 No results found。

彼得潘稍微研究一下後，目前想對 The Swift Programming Language 全文搜尋有以下兩種方法。

  • 方法 1: 利用 Google 的站內搜尋。
  • 方法 2: 從 swift book 的 GitHub 搜尋。

方法 1: 利用 Google 的站內搜尋

指定搜尋的網站是 swift book 的官網，在 Google 搜尋時加上 site:docs.swift.org/swift-book/documentation/the-swift-programming-language。

以搜尋 escaping 為例，從 Google 的搜尋結果可看出 escaping 出現在 swift book 的 Closures 章節。

方法 2: 從 swift book 的 GitHub 搜尋

GitHub - swiftlang/swift-book: The Swift Programming Language book
The Swift Programming Language book. Contribute to swiftlang/swift-book development by creating an account on GitHub.github.com

點選右上方的 search 框。

在下圖的 search 框輸入想查詢的關鍵字。

以下我們輸入 escaping 後按 enter。

找到 escaping 出現的檔案清單。

從上圖可看出 escaping 出現在 Closures.md，點選查看它的 Preview 即可瀏覽 escaping 的相關資訊。
