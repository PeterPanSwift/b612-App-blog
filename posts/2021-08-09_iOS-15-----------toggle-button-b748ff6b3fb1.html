iOS 15 自動更新選取狀態的 toggle button

開發 iOS App 時，我們時常會看到 toggle button 的設計，點選 button 時可切換不同狀態，比方以下 Music App 的例子。

  • 點選右下的三條線，切換歌曲 / 清單模式。
  • 點選右上的 button 切換是否重覆播放。

從前若要實現這樣的 button，我們必須手動修改 button 的 property isSelected。在 iOS 15 不用這麼麻煩了，只要將 changesSelectionAsPrimaryAction 設為 true，點選 button 時將自動更新 isSelected，當 isSelected 原本為 true 時將變成 false，當 isSelected 原本為 false 時將變成 true。

接下來讓我們利用 changesSelectionAsPrimaryAction 實現以下兩種常見的 button UI 設計。

  • isSelected 改變時更新圖文和背景的顏色。
  • isSelected 改變時更新圖文，比方打勾勾的 checkbox。

isSelected 改變時更新圖文和背景的顏色

changesSelectionAsPrimaryAction 設為 true，點選 button 時將自動更新 isSelected。當 isSelected 原本為 true 時將變成 false，當 isSelected 原本為 false 時將變成 true。為了讓 button 在 isSelected true 時顯示背景顏色，我們可以設定 button 的 primaryAction，它將自動幫我們加入更新顏色的效果。

override func viewDidLoad() {
    super.viewDidLoad()
        
    let repeatAction = UIAction(title: "重覆播放") { _ in
    }
    let button = UIButton(primaryAction: repeatAction)
    button.frame = CGRect(x: 100, y: 100, width: 100, height: 100)
    button.changesSelectionAsPrimaryAction = true
    view.addSubview(button)
       
}

button 背景變色的效果跟 primaryAction 有關，只要設定 primaryAction，手動修改 isSelected 也可以實現變色效果。

override func viewDidLoad() {
     super.viewDidLoad()
     // Do any additional setup after loading the view.
     let repeatAction = UIAction(title: "重覆播放") { action in
         let button = action.sender as? UIButton
         button?.isSelected.toggle()
     }
     let button = UIButton(primaryAction: repeatAction)
     button.frame = CGRect(x: 100, y: 100, width: 100, height: 100)
     view.addSubview(button)
}

isSelected 改變時更新圖文

button 的 configurationUpdateHandler 將在狀態改變時觸發，我們利用 它更新 button 的圖文。

checkbox

在 isSelected true 時顯示勾勾，isSelected false 時顯示框框。

override func viewDidLoad() {
    super.viewDidLoad()
    // Do any additional setup after loading the view.
    let button = UIButton(configuration: .plain(), primaryAction: UIAction(handler: { _ in
    }))
    button.frame = CGRect(x: 100, y: 100, width: 100, height: 100)
    button.changesSelectionAsPrimaryAction = true
    button.configurationUpdateHandler = { button in
        var config = button.configuration
        config?.image = UIImage(systemName: "\(button.isSelected ? "checkmark." : "" )rectangle")
        button.configuration = config
    }
    view.addSubview(button)     
}

若是 button 點選時沒有要做額外的事，我們也可以省略 primaryAction。

let button = UIButton(configuration: .plain())

從 Interface Builder 加入的 button 也可以變成 toggle button，方法如下:

  • 勾選 Menu 的 Selection as Primary Action
  • 利用 property observer 設定 configurationUpdateHandler
@IBOutlet weak var button: UIButton! {
    didSet {
        button.configurationUpdateHandler = { button in
            var config = button.configuration
            config?.image = UIImage(systemName: "\(button.isSelected ? "checkmark." : "" )rectangle")
            button.configuration = config
        }
    }
}

音樂播放鍵

音樂正在播放時 isSelected 為 true，顯示暫停圖示。音樂停止播放時 isSelected 為 false，顯示三角形圖示。

override func viewDidLoad() {
    super.viewDidLoad()
    // Do any additional setup after loading the view.
        
    let button = UIButton(configuration: .plain(), primaryAction: UIAction(handler: { _ in
    }))
    button.frame = CGRect(x: 100, y: 100, width: 100, height: 100)
    button.changesSelectionAsPrimaryAction = true
    button.configurationUpdateHandler = { button in
        var config = button.configuration
        config?.image = UIImage(systemName: "\(button.isSelected ? "pause" : "play" ).fill")
        button.configuration = config
    }
    view.addSubview(button)
}

利用 configurationUpdateHandler 更新 button 樣式的方法，我們也可以讓 Interface Builder 加入的 button 在 isSelected 改變時更新背景和文字的顏色。

@IBOutlet weak var button: UIButton! {
    didSet {
        button.configurationUpdateHandler = { button in
            var config = button.configuration
            config?.background.backgroundColor = button.isSelected ? .systemBlue : .clear
            config?.attributedTitle?.foregroundColor = button.isSelected ? .white : .systemBlue
            button.configuration = config
        }
    }
}
