使用 ChatGPT 和 API 提升程式可讀性

對正在學程式寫作業的新手或不擅長英文的工程師來說，ChatGPT 是得力的程式小幫手。它可以幫忙改善變數，function 的命名，大大提升程式的可讀性。

ps: ChatGPT有時會給錯的程式，不過它的英文不錯，因此可以請它幫忙優化程式裡 type / property / function / variable 的命名。

以下我們介紹兩種使用 ChatGPT 提升程式可讀性的方法。

  • 方法 1: 從 ChatGPT 網頁操作。
  • 方法 2: 串接 ChatGPT API。

方法 1: 從 ChatGPT 網頁操作

  • 請 ChatGTP 列出名稱需調整的項目。

描述問題如下，然後再貼上程式碼。

提升程式的可讀性，列出名稱需要調整的項目，不需要說明。描述的範例格式如下
nextBtnOutlet to nextButton

除了手動貼上程式，GPT 4 有以下兩種更方便的方法。

  • Browse with Bing

附上 GitHub 上程式檔的連結。

  • Advanced Data Analysis

上傳程式檔。

描述問題如下，然後再貼上程式碼。

To improve the readability of the program, please optimize the names of the following items:
