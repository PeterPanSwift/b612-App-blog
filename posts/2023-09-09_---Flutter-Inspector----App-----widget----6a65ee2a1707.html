透過 Flutter Inspector 檢視 App 畫面的 widget 元件

開發 Flutter App 時，我們可以透過 Flutter Inspector 檢視 App 畫面的 widget 元件，它將幫助我們方便地查看 UI 相關的細節和問題。

以下我們以 VS Code 操作，用 flutter 專案預設建立的 counter App 為例，一步步介紹 Flutter Inspector 的常用操作和功能。

  • 啟動 App。
  • 打開 DevTools，顯示 Flutter Inspector。
  • 查看 widget 的相關資訊。
  • 查看 widget 對應的程式。
  • 更新 Widget Tree (Refresh Tree)。
  • 同步 App & widget tree 的 Select Widget Mode。
  • 修改 widget layout。

啟動 App

我們必須先啟動 App，才能使用 Flutter Inspector 檢視畫面裡的 widget。

顯示 Flutter Inspector

  • 方法 1: 在 VS Code 裡顯示 Flutter Inspector。

點選下圖紅色框框的按鈕。

或打開 Command Palette(cmd + shift + p)，輸入 inspector 後點選 View: Show Flutter DevTools: Inspector。

或從左側清單點選 Flutter DevTools: Inspector。

Flutter Inspector 將在 VS Code 裡顯示，出現在下圖的紅色框框。

  • 方法 2: 在瀏覽器顯示 Flutter Inspector。

打開 Command Palette(cmd + shift + p)，輸入 brow 後點選 Dart: Open DevTools in Browser。

切換到 Flutter Inspector 分頁。

查看 widget 的相關資訊

從下圖的 Widget Tree 可檢視 App 畫面上有哪些 widget。

點選 Widget Tree 裡的 widget 可查詢 widget 的相關資訊。以下圖為例，點選 Text 元件後，從 Widget Properties 頁面可查詢它的大小和相關資訊，大小為 307 * 20。

查看 widget 對應的程式

點選 Widget Tree 的 widget 時，VS Code 會自動顯示 widget 對應的程式。比方點選 Text 後，VS Code 自動顯示 main.dart 裡的 Text(
 'You have pushed the button this many times:',)。

更新 Widget Tree (Refresh Tree)

當我們在 App 做一些操作後，畫面可能有一些變化，比方點擊下圖右下的 + 後，畫面中間的數字變成 3。

此時 Flutter Inspector 的 Widget Tree 顯示的還是舊的資訊，因此若想顯示畫面最新的資訊，記得要點選下圖的紅色框框更新 Widget Tree。

同步 App & widget tree 的 Select Widget Mode

有時我們會想知道畫面上的元件對應到 widget tree 的哪個 widget，這時可以打開下圖的 Select Widget Mode，它將實現神奇的 App & widget tree 同步功能。

打開 Select Widget Mode 後，點選 widget tree 的 widget 時，App 畫面將自動標示對應的元件。點選 App 畫面的元件時，widget tree 也會標示對應的 widget。

修改 widget layout

我們也可以從 Flutter Inspector 修改 widget layout，查看 App 在不同 layout 的模樣。

以下我們試試調整 Column 的 main axis alignment，切換到 Flex explorer 頁面。

點選 Main Axis 下方的 center。

點選 start。

Column 裡的文字從原本的置中變成置頂。
