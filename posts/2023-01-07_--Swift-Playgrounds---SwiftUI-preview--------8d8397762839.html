從 Swift Playgrounds 的 SwiftUI preview 查看列印訊息

從 Xcode 開發 SwiftUI App 時，我們可以方便地從 Canvas 操作 App 的 preview 畫面。不過很可惜，在 preview 無法看到列印的訊息，想看列印的訊息只能執行 App。

最近發現 Swift Playgrounds 雖然不像 Xcode 那麼強大，但它卻能在 preview 時顯示列印的訊息。

比方我們在 Swift Playgrounds 輸入以下程式，點擊骰子會更新骰子的圖片和印出骰子的點數。

import SwiftUI

struct ContentView: View {
    @State private var number = 1

    var body: some View {
        Image(systemName: "die.face.\(number).fill")
            .resizable()
            .scaledToFit()
            .padding()
            .onTapGesture {
                number = .random(in: 1...6)
                print("骰子點數是\(number)")
            }
    }
}

此時最新列印的訊息會出現在下方。

不過也別高興得太早，它只會出現一下子，幾秒後就會消失。

如果想看之前印出的訊息，可點選剛剛訊息左邊的按鈕，它是顯示 / 隱藏 Console 的開關。

從 Console 視窗可看到曾經印出的訊息。

如果覺得訊息太多，我們也可以清除之前列印的訊息，從 Console 視窗按出右鍵選單，然後點選 Clear Console。
