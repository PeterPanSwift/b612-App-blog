NavigationView 造成的 SwiftUI animation 問題

更新: 問題在 iOS 16 已修正，舊版才會有問題

利用 SwiftUI 的 animation modifier，我們可以輕易地實作動畫效果，比方以下星星向右移動的例子。

struct ContentView: View {
    
    @State private var moveX: Double = 0
    
    var body: some View {
        
        Image(systemName: "star")
            .resizable()
            .frame(width: 200, height: 200)
            .offset(x: moveX)
            .animation(.linear(duration: 5), value: moveX)
            .onAppear {
                moveX = 200
            }
    }
}

不過當畫面多了 NavigationView 後，卻會產生奇怪的動畫效果。

struct ContentView: View {
    
    @State private var moveX: Double = 0
    
    var body: some View {
        
        NavigationView {
            Image(systemName: "star")
                .resizable()
                .frame(width: 200, height: 200)
                .offset(x: moveX)
                .animation(.linear(duration: 5), value: moveX)
                .onAppear {
                    moveX = 200
                }
                .navigationTitle("Star")
        }
    }
}

如下圖所示，星星變成從左上往右下移動，但我們想要的應該是水平向右移動。

面對如此奇怪的問題，我們有以下解法:

畫面出現後，設定 0.01 秒之後才改變內容觸發動畫

struct ContentView: View {
    
    @State private var moveX: Double = 0
    
    var body: some View {
        
        NavigationView {
            Image(systemName: "star")
                .resizable()
                .frame(width: 200, height: 200)
                .offset(x: moveX)
                .animation(.linear(duration: 5), value: moveX)
                .onAppear {
                    DispatchQueue.main.asyncAfter(deadline: .now() + 0.01) {
                        moveX = 200
                    }
                }
                .navigationTitle("Star")
        }
    }
}
