顯示 Xcode 的 Console

開發 iOS App 時，print 印出的訊息會顯示在 Xcode 右下角的 Console。比方我們半夜睡不著，一直點選畫面上的 Console 去哪了時，應該會看到 hello 不斷印出。

struct ContentView: View {
    var body: some View {
        Text("Console 去哪了")
            .padding()
            .onTapGesture(perform: {
                print("hello")
            })
    }
}

不過有朋友遇到怪事，Xcode 下方的視窗完全看不到 print 印出的訊息。

問題出在右下方的 Console 視窗不小心被關掉了。開啟 Console 有以下方法。

  • 點選下圖圈起的 button。
  • 從 Quick Actions 點選 Activate Console(快速鍵 cmd + shift + c)。
