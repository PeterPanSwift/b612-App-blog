利用多行字串在 Playground練習 JSON 解析

我們時常連到後台抓取 JSON 資料 ，不過如果想先驗證 JSON 解析的程式是否寫對，排除網路問題或後台工程師的 bug，我們可以先在 Playground 利用三個雙引號的多行字串( multiline string literal)包含 JSON 資料測試，比方以下例子:

import Foundation
struct Song: Codable {
   let name: String
   let movie: String
}

let data = """
[
{
"movie": "我的少女時代",
"name": "小幸運"
},
{
"movie": "擺渡人",
"name": "十年"
}
]
""".data(using: .utf8)!

let decoder = JSONDecoder()
do {
   let songs = try decoder.decode([Song].self, from: data)
   print(songs.count)
} catch {
   print(error)
}

說明:

利用三個雙引號的多行字串包含 JSON 資料，然後呼叫 function data(using:) 得到 Data 型別的 JSON 資料，此時得到的資料就像我們利用 URLSessionTask 從網路下載的資料。

不過有一點值得注意的地方，Playground 裡如果使用三個雙引號的多行字串，有時會造成手動執行失效，所以建議測試時改成 Automatically Run(自動執行)。
