#74 利用 CIFilter 實現美麗的圖片濾鏡
iOS 上照片類的 App 十分受歡迎，尤其是 App 裡各式各樣幫我們做出漂亮照片的濾鏡功能。其實基本的濾鏡功能不難做到，因為 iOS SDK 早已為我們提供實現濾鏡功能的 CIFilter。medium.com

然而基本的濾鏡效果已經無法滿足我們了，接下來我們來試試結合多個 CIFilter，實現更 Cool 的濾鏡效果。

在專案裡加入可愛的娃娃圖片

加入顯示圖片的 image view

加入 CoreImage.CIFilterBuiltins

import CoreImage.CIFilterBuiltins

定義 function，將圖片變成以線條繪製

利用 CIFilter.edgeWork 製作濾鏡效果。由於 CIFilter 接收的輸入圖片和產生的輸出圖片型別都是 CIImage，因此我們將參數 input 和 function 回傳的型別都宣告為 CIImage。

func edgeWorkFilter(_ input: CIImage, radius: Float) -> CIImage? {
    let filter = CIFilter.edgeWork()
    filter.inputImage = input
    filter.radius = radius
    return filter.outputImage
}

值得注意的，產生的圖片線條將是白色，所以我們要再搭配顏色反轉的濾鏡將線條變成黑色。

定義 function，將顏色反轉

我們希望產生黑色的線條，因此我們利用 CIFilter.colorInvert 製作濾鏡效果，將剛剛用 edgeWork 濾鏡產生的白色線條變成黑色。

func colorInvertFilter(_ input: CIImage) -> CIImage? {
    let filter = CIFilter.colorInvert()
    filter.inputImage = input
    return filter.outputImage
}

套用 edgeWork & colorInvert 的濾鏡效果

呼叫 function edgeWorkFilter 得到白色線條圖，然後再傳入 function colorInvertFilter 得到黑色線條圖，最後再將 CIImage 轉換成 UIImage 後，顯示到 UIImageView 上。

override func viewDidLoad() {
    super.viewDidLoad()
    // Do any additional setup after loading the view.
        
    let image = UIImage(named: "little prince")
        
    if let ciImage = CIImage(image: image!), 
       let edgeWorkOutput = edgeWorkFilter(ciImage, radius: 3.0),
       let colorInvertOutput = colorInvertFilter(edgeWorkOutput) {
       let filterImage = UIImage(ciImage: colorInvertOutput)
       imageView.image = filterImage
    }
}
