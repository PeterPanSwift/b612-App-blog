使用 AI 搭配 Shortcuts 一鍵翻譯選取的文字 — Codex、Gemini、Foundation Models

macOS 內建翻譯功能，不過它只會照字面的意思翻譯，因此翻譯的結果通常比不上我們常用的 AI，像是 ChatGPT、Claude、Gemini、Grok 等。

比方以下網頁的小王子名言。

內建翻譯的結果差強人意，It is only with the heart that one can see rightly 竟然翻成只有用心才能正確看待。

既然 AI 那麼會翻譯，彼得潘想到不妨用 macOS 的 Shortcuts 搭配 AI 實現一鍵翻譯選取的文字。

以下我們示範 3 種例子。

  • 使用 Codex CLI(付費)。
  • 使用 Gemini CLI(免費)。
  • 使用 Apple 的 Foundation Models(免費)。

使用 Codex CLI

訂閱 ChatGPT 的朋友可免費使用 Codex，使用前請先安裝 Codex CLI。

Codex CLI
Pair with Codex in your terminaldevelopers.openai.com

打開 Shortcuts App，點選 + 新增 Shortcut。

取名為 Translate to Chinese。

切換到 i 分頁，勾選 Use as Quick Action。

Receive 的內容勾選 Text 就好。

加入 Run Shell Script。

點選 Open Preferences。

勾選 Allow Running Scripts。

Shell 選擇 zsh，Pass Input 選擇 as arguments。

輸入 script，請 codex 翻譯選取的文字，若沒有選取的文字則翻譯剪貼簿的文字。(ps: 感謝 AI 幫忙寫出以下 script)

export LANG=zh_TW.UTF-8
export LC_ALL=zh_TW.UTF-8

text="$1"
# 若沒有選取文字，退而求其次用剪貼簿
if [[ -z "$text" ]]; then
  text="$(pbpaste)"
fi

# 跑 Codex、過濾雜訊，最後回到剪貼簿
result=$(codex exec "Translate to zh-tw: $text" --skip-git-repo-check \
  | awk 'BEGIN{last=""} /tokens used:/{print last; exit} NF{last=$0}')

# 複製到剪貼簿
echo "$result" | pbcopy

# 回傳給 Shortcuts 顯示
echo "$result"

點選 Shortcut Input，將 Type 改成 Text。

加入 Show Notification，在翻譯完成時顯示通知。

Show notification 的 Title 輸入翻譯完成，Attachments 選擇 Shell Script Result。

Shortcut 的完整設定如下。

設定快速鍵，切換到 i 分頁點選 Add Keyboard Shortcut。

輸入快速鍵，比方 cmd + ctrl + t。

之後在網頁或任何 App，只要選取文字，即可按快速鍵或從右鍵選單點選 Service > Translate to Chinese 翻成中文。

以剛剛的小王子名言為例，在網頁選取文字後按下 cmd + ctrl + t。

稍等幾秒後，右上方出現翻譯完成的通知。

按 cmd + v 還可貼上翻譯的結果，翻得比內建的翻譯好上許多。

使用 Gemini CLI

Gemini CLI 每天有一定的免費使用額度，使用前請先安裝 Gemini CLI。

GitHub - google-gemini/gemini-cli: An open-source AI agent that brings the power of Gemini directly…
An open-source AI agent that brings the power of Gemini directly into your terminal. - google-gemini/gemini-cligithub.com

剛剛 script 裡的翻譯程式改成這行即可變成用 Gemini CLI 翻譯。

result=$(gemini -p "Translate to zh-tw: $text" | tail -n 1)

完整的 script 如下。

export LANG=zh_TW.UTF-8
export LC_ALL=zh_TW.UTF-8

text="$1"
# 若沒有選取文字，退而求其次用剪貼簿
if [[ -z "$text" ]]; then
  text="$(pbpaste)"
fi

# 使用 gemini 翻譯
result=$(gemini -p "Translate to zh-tw: $text" | tail -n 1)

# 複製到剪貼簿
echo "$result" | pbcopy

# 回傳給 Shortcuts 顯示
echo "$result"

翻譯結果也挺不錯的，跟 Codex 不相上下。

使用 Foundation Models

Apple 的 Foundation Models 可免費使用。請修改剛剛 Run Shell Script 的設定，在 script 輸入以下 Swift 程式。

import Foundation
import AppKit
import FoundationModels


// 從命令列參數取得文字
var text: String? = CommandLine.arguments.dropFirst().first

// 如果沒有輸入，則改用剪貼簿內容
if text == nil || text!.isEmpty {
    text = NSPasteboard.general.string(forType: .string)
}

guard let text else {
    exit(1)
}

// 使用 Apple 的 Foundation Models 翻譯
let session = LanguageModelSession()
let semaphore = DispatchSemaphore(value: 0)
var result = ""

Task {
    do {
        result = try await session.respond(
            to: "Translate to zh-tw: \(text)"
        ).content
    } catch {
        result = "\(error)"
    }
    semaphore.signal()
}

_ = semaphore.wait(timeout: .distantFuture)

// 複製結果到剪貼簿
let pasteboard = NSPasteboard.general
pasteboard.clearContents()
pasteboard.setString(result, forType: .string)

// 印出結果（給 Shortcuts 用）
print(result)

Shell 改成 Swift。

Foundation Models 由於是本機 AI，效果會比雲端 AI 差一點，優點則是沒有連網時也能翻譯。
