當我們在程式裡生成 UNNotificationAttachment 物件，也就是通知包含的附檔時，需特別注意附檔名。如果附檔名不是上表提到的格式，將無法順利生成 UNNotificationAttachment。

比方以下例子，我們指定的附檔名為奇怪的 hello，它當然不支援，所以程式將跑到 catch，印出 error。

let url = Bundle.main.url(forResource: "demo", withExtension: "hello")
do {
   let attachment = try UNNotificationAttachment(identifier: "", url: url!, options: nil)
   print(attachment)
} catch {
   print(error)
}

error 訊息如下，它明白告訴我們，它不認得附檔名呀。(Unrecognized attachment file type)

Error Domain=UNErrorDomain Code=101 “Unrecognized attachment file type” UserInfo={NSLocalizedDescription=Unrecognized attachment file type}
