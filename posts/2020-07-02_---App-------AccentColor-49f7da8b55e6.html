設定 App 主要顏色的 AccentColor

Xcode 12 的 Assets 新增了 AccentColor，現在我們可以方便地透過 AccentColor 設定 App 的主要顏色。接下來就讓我們分別看看 SwiftUI & UIKit 的例子。

ps: 只有 iOS 14 以上版本會套用 Assets 裡設定的 AccentColor，舊版不會套用。

設定 Assets 的 AccentColor

從 Attributes inspector 的 Color 調整顏色

點選 Show Color Panel 選顏色。

SwiftUI

struct ContentView: View {
    @State private var value: Double = 1
     
    var body: some View {
        VStack {
            Image("heart")
                .resizable()
                .scaledToFit()
            Button("好想你") {
            }
            Slider(value: $value)
        }
        
    }
}

SwiftUI 的預覽畫面立即心靈感應 AccentColor，button & slider 都變成了桃紅色。

UIKit

UIKit 元件的顏色一樣可從 AccentColor 設定，不過從 storyboard 看不出來，要等 App 執行後才會變成 AccentColor 設定的顏色。

ps: 值得注意的，若有在 storyboard 設定 global tint，則會依據 global tint 設定的顏色，而不是 AccentColor。
