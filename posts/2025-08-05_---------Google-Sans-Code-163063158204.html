適合寫程式的字體 Google Sans Code

最近 Google 推出適合寫程式的新字體 Google Sans Code，使用 Android Studio 的朋友可直接在 Settings > Editor > Font 將字體設為 Google Sans Code。

使用其它 IDE 的朋友可從網路下載字體檔，然後再設成 IDE 採用的字體。

Google Sans Code - Google Fonts
Edit descriptionfonts.google.com

點選 Get font。

點選 Download all。

Google_Sans_Code 資料夾下有許多字體檔，以下我們安裝 GoogleSansCode-VariableFont_wght.ttf & GoogleSansCode-Italic-VariableFont_wght.ttf。

以下我們分別示範 Xcode & VS Code 採用 Google Sans Code 的方法。

  • Xcode

從 Quick Actions 點選 Edit Themes。

點選 Theme 的名字，然後新增 theme，取名為 Google Sans Code。

按 cmd + a 全選 Source Editor 裡的選項，然後點擊下圖的框框調整字體。

選擇 Google Sans Code。

程式的字體變成 Google Sans Code。

  • VS Code

從 Command Palette 點選 Preferences: Open Settings (UI)。

搜尋 font。

在 Editor: Font Family 的開頭加入 'Google Sans Code'。

程式的字體變成 Google Sans Code。
