table view cell 的 accessory view

如下圖所示，table view cell 的右邊有一塊呈現 accessory view 的區塊，接下來讓我們認識 accessory view 內建的四種樣式以及如何自訂 accessory view。

accessory view 內建的四種樣式

  • Disclosure Indicator
  • Detail Disclosure
  • Checkmark
  • Detail

自訂 accessory view

以圖片原本的大小當 accessoryView 的大小

  • 方法 1: 從 Interface Builder 設定

在 cell 裡加入要當 accessoryView 的 image view。它擺放的位置不重要，因為待會它將成為顯示在 cell 右邊的 accessoryView。

image view 的大小為圖片的大小，比方圖片在 1x 的大小為 40 * 64 pixels，image view 的大小請設為 40 * 64 points。

從 cell 按右鍵叫出右鍵選單，然後從 accessoryView 連到 image view peter。

此時 cell 的 accessoryView 將顯示 peter 的圖片。

結果

  • 方法 2: 從程式設定

設定 cell 的 accessoryView。(假設變數 cell 代表顯示的 UITableViewCell 元件)

let accessoryView = UIImageView(image: UIImage(systemName: "heart"))
cell.accessoryView = accessoryView

自訂 accessoryView 的大小

若要調整 accessoryView 的大小，則要用 view 當 accessoryView，然後在 view 裡加入要顯示的元件。

  • 方法 1: 從 Interface Builder 設定

cell 的 accessoryView 連到裝著愛心 image view 的 yellow view。

結果

  • 方法 2: 從程式設定

假設變數 cell 代表顯示的 UITableViewCell 元件。

let accessoryView = UIView(frame: CGRect(x: 0, y: 0, width: 117, height: 115))
accessoryView.backgroundColor = .systemYellow
let heartImageView = UIImageView(frame: CGRect(x: 8, y: 9, width: 101, height: 99))
heartImageView.image = UIImage(systemName: "heart")
heartImageView.contentMode = .scaleAspectFit
accessoryView.addSubview(heartImageView)
cell.accessoryView = accessoryView
