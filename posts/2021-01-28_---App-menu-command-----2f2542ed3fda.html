自訂 App menu command 快速鍵

快速鍵可以大幅提高我們做事的效率，因此很多 App 都為常用功能定義了快速鍵。然後App 不太可能為每個功能定義快速鍵，不過沒關係，只要出現在 App menu 的 command，我們都可以幫它加上快速鍵。

接下來就讓我們以 Xcode 的 Behaviors > Select all and reindent 為例，幫它定義快速鍵吧。

啟動 System Preferences App，點選 Keyboard

切換到 Shortcuts 下的 App Shortcuts，然後點選下方的 +

輸入快速鍵的相關資訊

Application 選擇 Xcode，Menu Title 輸入(Select all and reindent)，Keyboard Shortcut 輸入自訂的快速鍵 cmd + ctrl + a。值得注意的，Menu Title 輸入的名字一定要和 Xcode 裡的 menu command 一模一樣，否則會無法對應。

大功告成 ! 現在 Select all and reindent 也有專屬的快速鍵了，cmd + ctrl + a。
