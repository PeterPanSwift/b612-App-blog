模仿 LINE App 在 App 裡使用芫荽字體

最近 LINE App 推出新功能，使用者可選擇將 App 的字體改成包含台客語用字的可愛字體「 芫荽」。

芫荽是開源的免費字體，因此我們開發 App 時也可將 App 的字體改成芫荽字體。

下載芫荽字體的 ttf 檔

連到芫荽(iansui)的 GitHub 網站，點選右側 Releases 下方最新的版本(Latest)。

GitHub - ButTaiwan/iansui: 芫荽，基於 Klee One 改造的學習用台灣繁體字型
芫荽，基於 Klee One 改造的學習用台灣繁體字型. Contribute to ButTaiwan/iansui development by creating an account on GitHub.github.com

點選 Iansui.zip 下載。

解壓縮後資料夾內的 Iansui-Regular.ttf 即為字體檔。

在 App 裡使用芫荽字體

以下我們以 Swift iOS App 為例，在 iOS App 使用芫荽字體。

為 iOS App 加入客製字型 (custom font)
覺得 iOS App 內建的中文字型不夠可愛嗎 ? 沒關係，我們可另外加入字型檔，在 iOS App 裡顯示我們喜愛的字型。可加到專案裡的字體檔主要有 ttf ，otf & ttc 三種格式，其中 ttc 比較容易有問題，因此建議選擇…medium.com

在 Project navigator 加入 Iansui-Regular.ttf。

在 App 的 Info 頁面設定 Fonts provided by application 的字體為 Iansui-Regular.ttf。

以下我們示範幾種芫荽字體和系統字體的比較。

  • 中文字。
struct ContentView: View {
    var body: some View {
        VStack {
            Text("彼得潘")
                .font(.system(size: 100))
            Text("彼得潘")
                .font(.custom("Iansui", size: 100))

        }
    }
}
  • 注音符號。
struct ContentView: View {
    var body: some View {
        VStack {
            Text("ㄅㄆㄇㄈ")
                .font(.system(size: 100))
            Text("ㄅㄆㄇㄈ")
                .font(.custom("Iansui", size: 100))
        }
    }
}
  • 數字。
struct ContentView: View {
    var body: some View {
        VStack {
            Text("12345")
                .font(.system(size: 100))
            Text("12345")
                .font(.custom("Iansui", size: 100))
        }
    }
}
  • 英文字。
struct ContentView: View {
    var body: some View {
        VStack {
            Text("Peter")
                .font(.system(size: 100))
            Text("Peter")
                .font(.custom("Iansui", size: 100))
        }
    }
}
