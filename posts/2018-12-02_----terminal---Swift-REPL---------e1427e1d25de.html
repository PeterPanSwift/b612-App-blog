如何在 terminal 的 Swift REPL 執行非同步程式

當我們在 terminal 手動輸入 Swift 程式時，原本它執行後會馬上結束，沒有耐心等待網路抓資料這類非同步程式的執行結果。比方以下例子，URLSessionTask 需要幾秒之後才會抓到資料，但此時程式已經結束執行，因此我們看不到它印出的內容。

想在 terminal 看到非同步程式的執行結果，必須透過 CFRunLoop 的幫忙。

import Foundation
let runLoop = CFRunLoopGetCurrent()
if let url = URL(string: "https://itunes.apple.com/search?term=swift&media=music") {
   let task = URLSession.shared.dataTask(with: url) { (data, response , error) in
      if let data = data, let dic = try?     JSONSerialization.jsonObject(with: data) as? [String: Any] {
         print(dic)
         CFRunLoopStop(runLoop)
      }
   }
   task.resume()
   CFRunLoopRun()
}

說明

利用 CFRunLoopRun() 讓程式持續執行，然後在 URLSessionTask 抓到資料後呼叫 CFRunLoopStop 停止程式的執行。
