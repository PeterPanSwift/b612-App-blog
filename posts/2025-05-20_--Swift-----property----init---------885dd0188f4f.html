讓 Swift 的常數 property 能在 init 設定且有預設值

當我們利用 struct 或 class 定義型別時，property 可以宣告成變數或常數。不過當 property 是常數且直接用 = 設定內容時，property 的內容將完全無法修改。

以 struct 定義的 Baby 為例，當 let 宣告的 age 直接設為 1 時，它只會生成包含 name 參數的 init，age 將永遠為 1。

struct Baby {
    let name: String
    let age = 1
}

此時若另外定義 init 設定 age 的內容也是不行的，因為 age 已先被設為 1，顯示錯誤 Immutable value ‘self.age’ may only be initialized once。

若希望身為常數的 property 能夠在 init 設定且有預設值，預設值必須改成從 init 的參數設定。

因此我們可修改剛剛的例子，定義 Baby 的 init，從 init 將 age 的預設值設為 1。

struct Baby {
    let name: String
    let age: Int
    
    init(name: String, age: Int = 1) {
        self.name = name
        self.age = age
    }
}

此時生成 Baby 將有兩個選擇，可傳入 age 也可不傳 age，不傳 age 時將採用預設值 1。
