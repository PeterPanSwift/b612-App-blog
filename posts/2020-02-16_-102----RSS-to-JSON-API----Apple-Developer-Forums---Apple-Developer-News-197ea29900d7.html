以上兩個網站都有提供 RSS，有興趣的朋友不妨利用 RSS to JSON API 開發一個每天收看 Apple Developer News 或 Apple Developer Forums 的 App。

RSS 介紹

RSS 是一種透過 XML 制定的資料格式，它專門收集和發佈網站的資料內容，當我們透過 RSS 閱讀器訂閱後，將可快速取得網站的最新資訊。

Apple Developer Forums 的 RSS

Apple Developer Forums 將各種主題的問題以 tag 分類，每個 tag 有對應的 RSS，方便有興趣的朋友訂閱。

Apple Developer Forums Tags
Use these tags to categorize questions or search for content. Learn how to search for tags The App Store Connect API…developer.apple.com

彼得潘對 SwiftUI 特別有興趣，因此找到以下 SwiftUI 的 tag 頁面。

點選上圖紅色框框的部分，然後從右鍵選單選擇 Copy Link 複製 RSS 的網址。

SwiftUI 的 RSS 如下:

https://developer.apple.com/forums/tags/rssFeed/swiftui

Apple Developer News 的 RSS

Apple 提供眾多的 RSS，從音樂，電影，電視，App，Apple News 到 Apple Developer News，方便我們訂閱搶先收看最新的資訊。

RSS Information - Apple
Get Apple news, iTunes hit lists, and more delivered directly to your web browser or newsreader. Click the links below…www.apple.com

以下是 Apple Developer News 的 RSS 網址:

http://developer.apple.com/news/rss/news.rss

解析 RSS 回傳的 XML

RSS 回傳的資料格式是 XML，因此我們可以利用 XMLParser 解析後顯示在 App 裡。

將 RSS 變成 JSON

開發 iOS App 時，解析 JSON 會比 XML 輕鬆許多，因此接下來我們將透過網站 rss to json 提供的 API 幫忙，將 RSS 變成熟悉的 JSON。

Rss to json online converter - rss2json.com
Free online API to convert RSS to JSONrss2json.com

如下圖所示，我們可以在網站貼上 RSS 的網址，點選 Convert to JSON 將內容變成 JSON。

從產生的 JSON，我們看到第一筆資料的 title 是 Swift Playgrounds Now Available on Mac，連結是 https://developer.apple.com/news/?id=02122020a 。

利用 API 將 RSS 變 JSON

我們希望 App 能呈現 RSS 的最新內容，因此剛剛手動從網站將 RSS 轉換成 JSON 的方法並不可行。還好，rss to json 貼心地提供了 API，我們可以利用以下 API 取得轉換後的 JSON 資料。

https://api.rss2json.com/v1/api.json?rss_url=rss網址

因此，在參數 rss_url= 後貼上 http://developer.apple.com/news/rss/news.rss，即可抓取 Apple Developer News 的 JSON 資料。

https://api.rss2json.com/v1/api.json?rss_url=http://developer.apple.com/news/rss/news.rss

成功取得 JSON 後，接下來我們就可用熟悉的 JSONDecoder 解析資料，將它美美地呈現在 App 上了。

One More Thing，取得 Medium 文章清單的 JSON

Medium 網站上有許多優質的 iOS 開發文章，可惜官方並未提供抓取文章清單的 API。不過別難過，它提供了 RSS，比方彼得潘 blog 的 RSS 是

https://medium.com/feed/@apppeterpan

因此我們可以利用剛剛介紹的 API 將 RSS 變 JSON，網址如下

https://api.rss2json.com/v1/api.json?rss_url=https://medium.com/feed/@apppeterpan
