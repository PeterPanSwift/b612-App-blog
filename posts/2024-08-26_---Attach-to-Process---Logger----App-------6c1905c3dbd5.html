使用 Attach to Process & Logger 查看 App 列印的訊息

開發 iOS App 時，我們常常會從 Xcode 啟動 App 到模擬器，然後從 Console 查看 print 列印的訊息。不過最近有同學問彼得潘，從模擬器點擊 App Icon 啟動的 App，也能看到它列印的訊息嗎 ?

可以的 ! 只要將 print 改成用 Logger 列印，再搭配 Xcode 的 Attach to Process 或 Console App 即可看到列印的訊息。

我們用以下的程式做實驗，點擊圖片時用 logger.log 列印訊息。

import SwiftUI
import OSLog

struct ContentView: View {
    @State private var count = 0
    private let logger = Logger()
    
    var body: some View {
        VStack {
            Image(.pic)
                .resizable()
                .scaledToFit()
                .onTapGesture {
                    count += 1
                    logger.log("tap \(count)")
                }
            Text("\(count)")
                .font(.largeTitle)
        }
    }
}

從 Xcode Console 觀看 App 列印的訊息

若要從 Xcode 看到 App 列印的訊息，我們必須讓 Xcode 和執行中的 App 連結。請從 Xcode menu 點選 Debug > Attach to Process，然後點選 App 的名字，例如下圖的 Demo。

此時點擊圖片將在 Xcode 的 Console 看到列印的訊息。

從 Console App 觀看 App 列印的訊息

我們也可以從 Console App 觀看 App 列印的訊息。

啟動 Console App 後，切換到模擬器的分頁，點選 Start 開始記錄模擬器輸出的訊息。

在右上的過濾框輸入文字查詢有興趣的內容。

  • 顯示 demo App 相關的訊息。
  • 顯示 tap 相關的訊息。
