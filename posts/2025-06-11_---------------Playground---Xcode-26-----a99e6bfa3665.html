方便即時查看程式執行結果的 #Playground — Xcode 26 新功能

Xcode 的 Playground 可以直接執行程式顯示結果，很適合用來練習或測試程式的功能。

不過另外建立 Playground 檔還是有點麻煩，在 Xcode 26 我們有更簡單的方法，在 Project 的檔案加入 #Playground 即可立即預覽程式執行的結果。

以下我們實驗幾個例子。

  • 公式計算。
  • 定義類別和建立物件。
  • 抓取網路 API 的資料。
  • 設計 UI 畫面。

公式計算

使用 #Playground { } 包住想執行的程式，前面必須先 import Playgrounds 才能使用 #Playground。

import Playgrounds

#Playground {
    func celsiusToFahrenheit(_ celsius: Double) -> Double {
        (celsius * 9.0 / 5.0) + 32.0
    }
    let fahrenheit = celsiusToFahrenheit(100.0)
    print("100.0 degrees Celsius is equal to \(fahrenheit) degrees Fahrenheit.")
}

#Playground {
    func calculateBmi(_ height: Double, _ weight: Double) -> Double {
        weight / (height * height)
    }
    let bmi = calculateBmi(1.78, 60.0)
}

Xcode 右邊的 Canvas 頁面將顯示每行程式執行的結果，Canvas 將在我們輸入程式時立即更新結果。由於有兩個 #Playground，Canvas 將顯示兩個分頁，分頁名稱的數字對應 #Playground 所在的行數。

若有 print，下方的 Canvas Console 也會顯示列印的結果。

有多個 #Playground 時，我們也可在 #Playground( ) 裡傳入字串設定分頁的標題。

#Playground("溫度換算") {
    func celsiusToFahrenheit(_ celsius: Double) -> Double {
        (celsius * 9.0 / 5.0) + 32.0
    }
    let fahrenheit = celsiusToFahrenheit(100.0)
    print("100.0 degrees Celsius is equal to \(fahrenheit) degrees Fahrenheit.")
}

#Playground("BMI") {
    func calculateBmi(_ height: Double, _ weight: Double) -> Double {
        weight / (height * height)
    }
    let bmi = calculateBmi(1.78, 60.0)
}

當同時有寫 #Preview & #Playground 時，我們從分頁的 icon 區分，眼睛 icon 的分頁代表 Preview，量杯 icon 的分頁代表 Playground。

定義類別和建立物件

定義 Baby 類別。

class Baby {
    var name: String
    var age: Int
   
    init(name: String, age: Int) {
        self.name = name
        self.age = age
    }
}

建立和使用物件。

import Playgrounds

#Playground {
    var cuteBaby = Baby(name: "Peter", age: 1)
    cuteBaby.age = 2
}

抓取網路 API 的資料

import Playgrounds
import Foundation

#Playground {
    let url = URL(string: "https://wttr.in/Taipei?format=3")!
    Task {
        do {
            let (data, _) = try await URLSession.shared.data(from: url)
            if let string = String(data: data, encoding: .utf8) {
                print(string)
            }
        } catch {
            print(error)
        }
    }
}

設計 UI 畫面

import Playgrounds
import UIKit

#Playground {
    let redView = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
    redView.backgroundColor = .red
    redView.layer.cornerRadius = 50
    redView
}

重新執行 #Playground 的程式

在 #Playground 所在的檔案按空白鍵或 return 鍵可觸發重新執行 #Playground 的程式。

#Playground 的限制

  • 聽不到聲音。
  • 不能設定 liveView，不能操作畫面。
  • 看不到 controller 的畫面。
  • 看不到 SwiftUI 元件的畫面。
