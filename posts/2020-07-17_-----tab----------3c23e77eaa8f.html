未被點選 tab 的文字和圖案顏色

未被點選 tab 的文字和圖案顏色預設是灰色，不喜歡嗎? 我們可以將它設成任何喜歡的顏色。

比方改成浪漫的淡藍色。

設定的方法有以下兩種:

從 Interface Builder 設定

選取 Tab Bar。

在 User Defined Runtime Attributes 加入 unselectedItemTintColor，Type 選擇 Color，然後在 Value 輸入顏色。

從程式設定

在 AppDelegate.swift 的 application(_:didFinishLaunchingWithOptions:) 設定 UITabBar 的 unselectedItemTintColor。

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    // Override point for customization after application launch.
    UITabBar.appearance().unselectedItemTintColor = UIColor(red: 118/255, green: 214/255, blue: 1, alpha: 1)
    return true
}
