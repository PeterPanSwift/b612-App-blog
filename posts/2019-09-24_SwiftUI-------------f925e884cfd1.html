SwiftUI 調整元件位置的幾種方法

SwiftUI 預設將置中顯示元件，若想調整它的位置，一般有以下幾種做法。

  • stack 搭配參數 alignment & spacing 排列元件。
  • frame 控制對齊的 alignment。
  • Spacer & padding。
  • 利用 offset 位移。
  • 利用 ZStack，overlay & background 堆疊 SwiftUI 畫面。
  • 利用 position 定位。

利用 stack 搭配參數 alignment & spacing 排列元件

struct ContentView: View {
    var body: some View {
        VStack(alignment: .leading, spacing: 50) {
            Image(.peter)
                .background(.orange)
            Text("我要一步一步學App，等待朋友崇拜看著我的臉，小小的天有大大的夢想，重重的課做著輕輕的App。我要一步一步學App，在最餓點吃著蘋果寫著Code，小小的天流過的淚和汗，總有一天我有屬於我的App。")
        }
    }
}
