用 GitHub 存放方便 App 讀取的圖片，影片，音樂，JSON

GitHub 專門用來存放我們工程師最喜歡的程式碼，不過它基本上可以存放任何東西，所以我們也可以將圖片，影片，音樂，JSON 等資料存在 GitHub。

我們開發 App 時，可以將資料存在 GitHub，比方製作電影時刻表 App，將每週上映電影的圖片存在 GitHub，然後再寫程式從 iOS App 讀取。

ps: GitHub 有流量的限制，因此此做法較適合用來練習開發App 或是應用在使用者不多的 App。

從程式讀取 GitHub 的圖片十分簡單，因為存在 GitHub 的檔案會有固定的網址格式。比方彼得潘的 GitHub 帳號是 AppPeterPan，當我將圖片 peter1.jpg 存在 repository CutePic 下時，圖片的網址是

https://raw.githubusercontent.com/AppPeterPan/CutePic/main/peter1.jpg

接下來我們將示範用 GitHub 存放圖片，然後再從 iOS App 讀取顯示圖片。

登入 GitHub 網站

GitHub: Where the world builds software
GitHub is where over 83 million developers shape the future of software, together. Contribute to the open source…github.com

點選左上的 New 新增 repository

設定 repository 資訊

輸入 repository 名稱，將權限設為 Public，方便之後 App 讀取，然後點選 Create repository。

第一次上傳檔案

點選 choose your files 上傳檔案，在此我們上傳兩張 Peter 可愛的圖片。

點選 Commit changes。

現在 repository 裡出現我們剛剛上傳的 peter1.jpeg & peter2.jpeg。

之後只要透過以下網址即可讀取 peter1.jpeg。

https://raw.githubusercontent.com/AppPeterPan/CutePic/main/peter1.jpeg

之後再上傳檔案

點選右上的 Add file，然後點選 Upload files。

從 iOS App 讀取顯示圖片

  • SwiftUI 版本
import SwiftUI
struct ContentView: View {
    var body: some View {
        let url = URL(string: "https://raw.githubusercontent.com/AppPeterPan/CutePic/main/peter1.jpeg")
        AsyncImage(url: url) { image in
            image
                .resizable()
                .scaledToFill()
        } placeholder: {
            Color.gray
        }
        .frame(width: 300, height: 300)
        .clipShape(Circle())
    }
}

由於圖片網址的開頭是固定，程式裡只要知道圖片的名稱，之後再跟開頭的網址組合即可生出完整的網址。

import SwiftUI
struct ContentView: View {
    let pics = ["peter1", "peter2"]
    let baseURL = URL(string: "https://raw.githubusercontent.com/AppPeterPan/CutePic/main")!
    
    var body: some View {
        let pic = pics.randomElement()!
        let url = baseURL.appending(path: pic).appendingPathExtension(for: .jpeg)
        AsyncImage(url: url) { image in
            image
                .resizable()
                .scaledToFill()
        } placeholder: {
            Color.gray
        }
        .frame(width: 300, height: 300)
        .clipShape(Circle())
    }
}

方法 1: 使用 async & await

import UIKit
class ViewController: UIViewController {
　　 @IBOutlet weak var imageView: UIImageView!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        let url = URL(string: "https://raw.githubusercontent.com/AppPeterPan/CutePic/main/peter1.jpeg")!
        Task {
            do {
                let (data, _) = try await URLSession.shared.data(from: url)
                imageView.image = UIImage(data: data)
            } catch {
                print(error)
            }
        }
    }
}

方法 2: 沒有使用 async & await

import UIKit
class ViewController: UIViewController {
　　 @IBOutlet weak var imageView: UIImageView!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        let url = URL(string: "https://raw.githubusercontent.com/AppPeterPan/CutePic/main/peter1.jpeg")!
        URLSession.shared.dataTask(with: url) { data, response, error in
            if let data {
                DispatchQueue.main.async {
                    self.imageView.image = UIImage(data: data)
                }
            }
        }.resume()
    }
}
