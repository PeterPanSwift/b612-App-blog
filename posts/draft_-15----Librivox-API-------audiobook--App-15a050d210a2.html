API 例子。

  • 搜尋 peter pan 相關的有聲書。

完全比對，只會找到書名 peter pan 的書。

https://librivox.org/api/feed/audiobooks/?title=peter pan&format=json

部分比對，找到書名包含 peter pan 的書。

https://librivox.app/api/search?query=Peter Pan
  • 取得書本 id 816 的相關資訊。(ps: id 816 是書本 peter pan)
https://librivox.org/api/feed/audiobooks/?id=816&format=json

可下載整本書音檔的 zip，解壓縮後的內容如下。

  • 取得書本 id 816 的播放檔清單。(ps: id 816 是書本 peter pan)
https://librivox.org/api/feed/audiotracks/?project_id=816&format=json
