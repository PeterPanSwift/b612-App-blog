Swift 指定小數位數的方法

使用 Swift 開發 iOS App 時，我們有幾種方法可指定小數的位數。

  • 利用 .number.rounded。
  • 利用 .number.precision。
  • 利用 %.數字f。

以下我們以 number 為例說明。

let number: Double = 2 / 3

利用 .number.rounded

increment 0.01 表示四捨五入到小數第二位，結果是 0.67。

let numberString = number.formatted(.number.rounded(increment: 0.01))

rule 參數可控制進位的方法，比方 .down 的結果將小於原始數字，因此結果是 0.66。

let numberString = number.formatted(.number.rounded(rule: .down, increment: 0.01))

SwiftUI 的 Text 可在 String Interpolation 或 Text 裡傳入 format 參數設定，結果是 0.67。

Text("\(number, format: .number.rounded(increment: 0.01))")
Text(number, format: .number.rounded(increment: 0.01))

利用 .number.precision

fractionLength(2) 指定小數位數為 2 位，結果是 0.67。

let numberString = number.formatted(.number.precision(.fractionLength(2)))

若希望小數最多顯示到第二位，但當原本的小數不到兩位以原本的位數顯示時，可改成傳入數字範圍的寫法 fractionLength(0…2)。

let number: Double = 0
let numberString = number.formatted(.number.precision(.fractionLength(0...2)))

利用 %.數字f

String 的 format 參數 "%.2f" 表示四捨五入到小數第二位，結果是 0.67。

let numberString = String(format: "%.2f", number)

SwiftUI 的 Text 可在 String Interpolation specifier 參數設定，結果是 0.67。

Text("\(number, specifier: "%.2f")")
