取得某天的開始時間和結束時間

開發 iOS App 時，我們時常需要取得某一天的開始， 24:0:0。有很多方法可以做到，但最簡單的莫過於使用 Calendar 的 function startOfDay(for:)。至於一天的結束時間 23:59:59，很可惜 Calendar 沒有提供 endOfDay(for:)，不過沒關係，多寫幾行程式，一樣可以算得出來。

取得開始時間
import Foundation
let today = Date()
let calendar = Calendar.current
let todayStartTime = calendar.startOfDay(for: today)

利用 Calendar 的 startOfDay(for:) 取得某一天的 24:0:0。

startOfDay(for:) - NSCalendar | Apple Developer Documentation
Returns the first moment of a given date as a date instance.developer.apple.com

取得結束時間

寫法 1

import Foundation
let today = Date()
let calendar = Calendar.current
let todayStartTime = calendar.startOfDay(for: today)
var todayEndTime = calendar.date(bySettingHour: 23, minute: 59, second: 59, of: today)

利用 Calendar 的 function date(bySettingHour:minute:second:of:options:) 設定某一天的 23:59:59。

date(bySettingHour:minute:second:of:options:) - NSCalendar | Apple Developer Documentation
Creates a new date calculated with the given time.developer.apple.com

寫法 2

import Foundation
let today = Date()
let calendar = Calendar.current
let todayStartTime = calendar.startOfDay(for: today)
let todayEndTime = calendar.date(byAdding: .day, value: 1, to: todayStartTime)?.addingTimeInterval(-1)
