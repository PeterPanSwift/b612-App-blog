請 Windsurf 串接 API 顯示 / 上傳資料

最近彼得潘發現 Windsurf 的 Write with Cascade 可以抓取 API 回傳的資料，因此現在請 AI 串接 API 顯示 / 上傳資料變得更簡單了 ! 只要告訴 AI API 的網址或 curl 指令，AI 即可自動抓取研究回傳的資料，寫出串接 API 的 App。

以下我們分別示範串接 get API 顯示資料和串接 post API 上傳資料的例子。

  • 串接 API 顯示資料。
  • 串接 API 上傳資料。

串接 API 顯示資料

為了證明 AI 沒作弊，以下彼得潘用自訂的桌球 API 為例，請 AI 串接 API 顯示桌球相關資訊，開發 SwiftUI & Flutter 兩種版本的 App 。

prompt:

串接 API 顯示桌球的相關資訊 
https://raw.githubusercontent.com/PeterPanSwift/JSON_API/refs/heads/main/table_tennis.json
  • SwiftUI App

AI 完成的程式可參考以下連結。

GitHub - PeterPanSwift/TableTennis
Contribute to PeterPanSwift/TableTennis development by creating an account on GitHub.github.com

AI 成功完成串接 API 顯示桌球相關資訊的 App。

  • Flutter App

滑鼠移到 beta 上可看到讀取 URL 的功能目前是免費的。

點選 Accept 安裝 http 套件。

點選 Accept All Changes 接受 AI 完成的程式。

AI 完成的程式可參考以下連結。

GitHub - PeterPanSwift/table_tennis

串接 API 上傳資料

以下我們以 Reqres 的登入 API 為例請 AI 串接 API 上傳帳號密碼。

Reqres - A hosted REST-API ready to respond to your AJAX requests
A hosted REST-API ready to respond to your AJAX requestsreqres.in

prompt:

串接 API 實現登入頁面

curl -X 'POST' \
  'https://reqres.in/api/login' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -d '{
  "email": "eve.holt@reqres.in",
  "password": "cityslicka"
}'
