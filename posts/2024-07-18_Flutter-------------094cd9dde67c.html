CircleAvatar

radius 是圓形的半徑。

CircleAvatar(
  radius: 100,
  backgroundImage: NetworkImage('https://peterpanswift.github.io/peter.png'),
)

ClipOval 搭配 Image

圖片的寬高設為圓形的直徑。

ClipOval(
  child: Image.network(
    'https://peterpanswift.github.io/peter.png',
    width: 200,
    height: 200,
    fit: BoxFit.cover,
  ),
)

ClipRRect 搭配 Image，borderRadius 設定 BorderRadius.circular

BorderRadius.circular 設定圓形的半徑。

ClipRRect(
  borderRadius: BorderRadius.circular(100),
  child: Image.network(
    'https://peterpanswift.github.io/peter.png',
    width: 200,
    height: 200,
    fit: BoxFit.cover,
  ),
)

Container 設定 DecorationImage，BoxDecoration 的 shape 設為 BoxShape.circle

Container 的寬高設為圓形的直徑。

Container(
  width: 200,
  height: 200,
  decoration: const BoxDecoration(
    shape: BoxShape.circle,
    image: DecorationImage(
      image: NetworkImage('https://peterpanswift.github.io/peter.png'),
      fit: BoxFit.cover,
    ),
  ),
)

Container 設定 DecorationImage，ShapeDecoration 的 shape 設為 CircleBorder

Container(
  width: 200,
  height: 200,
  decoration: const ShapeDecoration(
    shape: CircleBorder(),
    image: DecorationImage(
      image: NetworkImage('https://peterpanswift.github.io/peter.png'),
      fit: BoxFit.cover,
    ),
  ),
)

Container 搭配 Image，BoxDecoration 的 shape 設為 BoxShape.circle

Container(
  decoration: const BoxDecoration(
    shape: BoxShape.circle,
  ),
  clipBehavior: Clip.antiAlias,
  child: Image.network(
    'https://peterpanswift.github.io/peter.png',
    width: 200,
    height: 200,
    fit: BoxFit.cover,
  ),
)

Container 搭配 Image，BoxDecoration 的 borderRadius 設為 BorderRadius.circular

Container(
  decoration: BoxDecoration(
    borderRadius: BorderRadius.circular(100),
  ),
  clipBehavior: Clip.antiAlias,
  child: Image.network(
    'https://peterpanswift.github.io/peter.png',
    width: 200,
    height: 200,
    fit: BoxFit.cover,
  ),
)

Container(

  child: Image.network(
