快速加入佔滿螢幕的背景圖片

1 從 Media Library 將圖片拖曳到 controller 的 view 上，生成 image view。

2 讓 image view 佔滿整個畫面。

從 Size Inspector 點選 Fill Container Horizontally 和 Fill Content Vertically。

3 將圖片的 Content Mode 設為 Aspect Fill。

4 點選 Reset to Suggested Constraints。(快速鍵 cmd + option + shift + = )

產生讓圖片佔滿螢幕的四個條件。
