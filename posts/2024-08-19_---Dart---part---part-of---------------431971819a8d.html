使用 Dart 的 part 和 part of 將大檔拆解成方便管理的小檔

開發程式時，我們常常會在不知不覺間加入愈來愈多的功能，造成一個檔案包含大量的程式，因此影響程式的可讀性和維護性。

此問題可透過 Dart 特別的 part 和 part of 語法解決，它能幫我們將大檔拆解成多個方便管理的小檔。

比方原本 a.dart 包含 3 個 function，functionA、functionB 和 functionC。

void functionA() {
  print('functionA');
  functionB();
  functionC();
}

void functionB() {
  print('functionB');
}

void functionC() {
  print('functionC');
}

我們可以將 a.dart 拆解成三個檔案，a.dart 定義 functionA，b.dart 定義 functionB，c.dart 定義 functionC。

在 b.dart & c.dart 前加上 part of 說明它們屬於哪個檔案，比方 part of 'a.dart';表示屬於 a.dart，是 a.dart 的一份子，因此以下的 b.dart 和 c.dart 都是 a.dart 的一部分。

  • b.dart。
part of 'a.dart';

void functionB() {
  print('functionB');
}
  • c.dart。
part of 'a.dart';

void functionC() {
  print('functionC');
}

在 a.dart 加入 part 說明它包含哪些檔案，比方 part 'b.dart'; 和 part 'b.dart'; 說明 a.dart 包含了 b.dart 和 c.dart 的程式，因此 functionA 可以呼叫 functionB 和 functionC。

  • a.dart。
part 'b.dart';
part 'c.dart';

void functionA() {
  print('functionA');
  functionB();
  functionC();
}

剛剛的例子我們從 a.dart 存取 b.dart 和 c.dart 的內容，不過要從 b.dart 存取 a.dart 和 c.dart，或從 c.dart 存取 a.dart 和 b.dart 也是 ok 的，因為它們可以當成同一個檔案，只是透過 part 和 part of 拆解。

之後其它的檔案若想使用 a.dart、b.dart、c.dart 的內容，必須 import a.dart，不能 import b.dart 或 c.dart，因為使用 part of 語法的 b.dart 跟 c.dart 只是 a.dart 的一部分。

最後我們再看一個實際的計算機例子說明 part 和 part of 的應用。做為主程式的 calculator.dart 使用 part 加入 operations.dart 和 display.dart 的內容，計算的邏輯被拆到 operations.dart，畫面顯示的邏輯被拆到 display.dart。

  • calculator.dart
part 'operations.dart';
part 'display.dart';

class Calculator {
  int _currentValue = 0;

  void add(int value) {
    _currentValue = addOperation(_currentValue, value);
    updateDisplay(_currentValue);
  }

  void subtract(int value) {
    _currentValue = subtractOperation(_currentValue, value);
    updateDisplay(_currentValue);
  }

  void multiply(int value) {
    _currentValue = multiplyOperation(_currentValue, value);
    updateDisplay(_currentValue);
  }

  void divide(int value) {
    _currentValue = divideOperation(_currentValue, value);
    updateDisplay(_currentValue);
  }

  void clear() {
    _currentValue = 0;
    updateDisplay(_currentValue);
  }
}
  • operations.dart。
part of 'calculator.dart';

int addOperation(int a, int b) {
  return a + b;
}

int subtractOperation(int a, int b) {
  return a - b;
}

int multiplyOperation(int a, int b) {
  return a * b;
}

int divideOperation(int a, int b) {
  if (b == 0) {
    print('錯誤：除數不能為零');
    return a;
  }
  return a ~/ b;  // 使用整數除法
}
  • display.dart。
part of 'calculator.dart';

void updateDisplay(int value) {
  print('當前值: $value');
}
