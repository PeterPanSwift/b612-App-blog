舊版做法

Random name API 可取得各種有趣的名字，像是男孩名，女孩名，國家名，城市名，電影名，遊戲名等。

Dry Names | Random name list generator and API - names.drycodes.com
Our Goal: Provide a simple, easy to use, random name generator API where you don't repeat yourself a.k.a DRY.names.drycodes.com

以 https://names.drycodes.com/10?nameOptions=boy_names&separator=space 為例，它將回傳 10 個隨機可愛男孩的名字，以下為輸出的範例。

["Edouard Jebediah","Mehtab Jediah","Ashton Gibson","Bryden Humberto","Breck Randolph","Calvin Roddy","Terry Simeon","Addison Duane","Ron Kolby","Jamison Ellis"]

請從 API 的參數 nameOptions 找尋有興趣的 4 個選項，製作一個顯示名字清單的表格頁面，使用者可從上方的 segmented control 選擇某一個 nameOption 顯示對應的名字清單。
