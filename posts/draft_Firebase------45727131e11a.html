Firebase 重點整理

1 一開始的設定 

(1) 建立專案，輸入 bundle ID。

(2) 下載 GoogleService-Info.plist，加到 Xcode 專案裡。

(3) 安裝 pod

pod 'Firebase/Core'

(4) 在 AppDelegate 設定 Firebase。

import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {

var window: UIWindow?

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplicationLaunchOptionsKey: Any]?) -> Bool {
// Override point for customization after application launch.
   FirebaseApp.configure()
   return true
}

2 登入功能

3 使用 Database 

(1) Database 的套件

pod 'Firebase/Database'

(2) 要登入後才能上傳資料

否則會有 permission_denied 的錯誤。

(3) 登入要用到 Auth 的套件

pod 'Firebase/Auth'

(4) email register。

Auth.auth().createUser(withEmail: "test@gmail.com", password: "123456") { (user, error) in
   print(user?.uid)
}

createUser 後即為登入狀態。

(2) setValue : 取代

let songDic = ["name": "你好不好"]
Database.database().reference().child("songs").setValue(songDic)

(3) updateChildValues: 更新

如果 key 原本就存在，它的 value 會被更新。

如果 key 原本不存在，會新增 key 和 value。

let songDic = ["name": "你好不好"]
Database.database().reference().child("songs").updateChildValues(songDic) { (error, reference) in
}
