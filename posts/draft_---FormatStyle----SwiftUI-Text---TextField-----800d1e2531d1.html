利用 FormatStyle 設定 SwiftUI Text & TextField 的格式

從 iOS 15 開始，我們可以很方便地利用 FormatStyle 設定 SwiftUI Text & TextField 的格式。FormatStyle 是 protocol，如下圖所示，有許多遵從 FormatStyle 的 style 可選擇，接下來讓我們試試幾種常見的應用。

指定到小數點後第幾位，number.precision

到小數點後第二位，小數點後第三位會四捨五入到第二位

struct ContentView: View {
    @State private var percent = 20.536
    
    var body: some View {
        VStack {
            Text(percent, format: .number.precision(.fractionLength(2)))
            TextField("percent", value: $percent, format: .number.precision(.fractionLength(2)))
                .textFieldStyle(.roundedBorder)
        }
        .font(.largeTitle)
        .padding()
        
    }
}

顯示百分比，percent

struct ContentView: View {
    @State private var percent = 10
    
    var body: some View {
        VStack {
            Text(percent, format: .percent)
            TextField("percent", value: $percent, format: .percent)
                .textFieldStyle(.roundedBorder)
        }
        .font(.largeTitle)
        .padding()
        
    }
}

顯示世界上最受歡迎的錢，currency(code:)

.currency(code: "TWD") 將顯示新台幣，.currency(code: Locale.current.currencyCode ?? "TWD") 將依據使用者手機設定的 Region 決定顯示的金錢符號，以下例子的手機將 region 設為日本，因此顯示 ¥1,000。

struct ContentView: View {
    @State private var percent = 1000
    
    var body: some View {
        VStack {
            Text(percent, format: .currency(code: "TWD"))
            TextField("percent", value: $percent, format: .currency(code: Locale.current.currencyCode ?? "TWD"))
                .textFieldStyle(.roundedBorder)
        }
        .font(.largeTitle)
        .padding()
        
    }
}
