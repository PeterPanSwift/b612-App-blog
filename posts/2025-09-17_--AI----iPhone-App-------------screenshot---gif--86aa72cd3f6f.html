請 AI 製作 iPhone App 截圖外框合成網站(包含 screenshot、gif、video)

Apple 的 Design Resources 網站提供了漂亮的 iPhone 外框， 因此我們可以將 iOS App 的 screenshot / gif / video 加上美麗的外框。不過除了辛苦地用美術軟體合成，有更簡單的方法嗎 ?

當然有呀，請 AI 幫我們寫個 iPhone App 截圖外框合成網站吧。

以下彼得潘使用 VS Code 搭配 Codex gpt-5-codex high 製作。

首先我們從 Apple Design Resources 下載 iPhone 17 & 16 的外框圖。

Apple Design Resources
Design apps accurately and quickly using official Apple design templates, icon production templates, color guides, and…developer.apple.com

建立網站的資料夾 iPhone Bezels，在裡面新增資料夾 Bezels，存放 iPhone 17 & 16 各個機型的 PNG。

使用 VS Code 打開資料夾 iPhone Bezels，打開 Codex，輸入以下 prompt。

  • 設計一個網頁，可上傳 iPhone App 的 screenshot，跟 directory Bezels 下的 iPhone Bezel 合成有外框的圖片。

為了更完善功能，彼得潘後續輸入以下 prompt。

  • 合成圖的四角多出白色三角形，請修正問題。
  • 除了選擇檔案，也可以貼上圖片。
  • 可合成 gif，幫 gif 加上外框。
  • 可合成 mov & mp4 格式的影片，幫影片加上外框。
  • 加上重設 button。
  • 使用 GitHub CLI 部署到 GitHub Pages，變成可以連線的網頁。

Codex 不負所託，成功完成 iPhone App 截圖外框合成工具。

工具的網址。

iPhone App 截圖外框合成工具
上傳或貼上 App 截圖或影片（支援 PNG、JPG、GIF、MP4、MOV）並選擇想要的 iPhone 外框，立即產出帶框成品。peterpanswift.github.io

專案的 GitHub 網址。

GitHub - PeterPanSwift/iphone-bezels
Contribute to PeterPanSwift/iphone-bezels development by creating an account on GitHub.github.com

網站的畫面如下。

可選擇 iPhone 的機型、顏色和方向。

接下來我們分別示範 screenshot & gif 的加框。

  • screenshot 加框。

貼上 screenshot，選擇機型、顏色、方向後，點選合成圖片。

成功生成添加宇宙橙色外框的螢幕截圖。

  • GIF 加框。

成功生成添加宇宙橙色外框的 gif。
