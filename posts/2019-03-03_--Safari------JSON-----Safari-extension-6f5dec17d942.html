讓 Safari 漂亮顯示 JSON 資料的 Safari extension

JSON 格式的資料有許多的 {} 和 []，時常讓我們看到眼花撩亂。

因此我們會另外透過一些方法將 JSON 資料變成淺顯易懂的樣子呈現，比方透過以下的 JSON 排版網站。

利用 JSON Editor Online 排版美美的 JSON 資料
開發 iOS App 時，我們時常串接 API 抓取 JSON 格式的資料。不過 JSON 卻讓我們又愛又恨，它豐富了我們的 App 內容，但要看懂 JSON 卻不是件易事。medium.com

不過如果想要更快的方法，想直接連到 JSON 網頁後將它美美地顯示，則可考慮直接安裝特別用來顯示 JSON 的 Safari extension，例如以下介紹的 SimplyJSON for Safari。

https://apps.apple.com/tw/app/simplyjson-for-safari/id1495193038?l=en&mt=12

安裝 SimplyJSON for Safari 後，當我們再次連到 JSON 資料的頁面時，原本黑白的 JSON 文字全變成一目瞭然的彩色文字，而且還可點選 + 或 — 展開/收起每一層資料呢。
