利用 SwiftUI repeatForever 製作搖晃動畫效果

利用 SwiftUI 的 repeatForever 我們可以輕易做出生動的搖晃動畫效果。

以下我們以 repeatForever 搭配 rotationEffect 示範我們小時候常見的搖頭唸書動作。

struct ContentView: View {
    
    @State private var isShaking = false
    
    var body: some View {
        Image(.readBook)
            .resizable()
            .scaledToFit()
            .frame(width: 300)
            .shadow(radius: 20)
            .rotationEffect(.degrees(isShaking ? 6 : -6))
            .animation(
                .easeInOut(duration: 0.5)
                .repeatForever(),
                value: isShaking
            )
            .onAppear {
                isShaking = true
            }
    }
}
