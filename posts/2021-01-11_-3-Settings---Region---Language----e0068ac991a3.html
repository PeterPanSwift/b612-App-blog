#3 Settings 的 Region & Language 選擇

範例

選擇 Region & Language，包含 search & 右邊的 index 功能。

提示

  • 取得全部的 regions
let regions = Locale.isoRegionCodes.map {
   Locale.current.localizedString(forRegionCode: $0)
}
  • 取得全部的 languages
let languages = Locale.isoLanguageCodes.map {
    Locale.current.localizedString(forLanguageCode: $0)
}
  • 模仿 Settings App 的 search bar 效果
let searchController = UISearchController(searchResultsController: nil)
searchController.searchResultsUpdater = self  
searchController.searchBar.showsCancelButton = true     
searchController.hidesNavigationBarDuringPresentation = false
searchController.obscuresBackgroundDuringPresentation = false
navigationItem.searchController = searchController
definesPresentationContext = true

showsCancelButton 設為 true : 顯示 cancel button。

hidesNavigationBarDuringPresentation 設為 false : 在 search 框輸入文字時，不要隱藏 navigation bar 的標題。

obscuresBackgroundDuringPresentation 設為 false: search 顯示結果時不要變暗。
