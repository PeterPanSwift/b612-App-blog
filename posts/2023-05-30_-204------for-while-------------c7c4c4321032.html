數字的次方(power of a number)

寫一個 function 計算數字的次方，參數是底數和指數，底數型別是浮點數，指數型別是整數。

Write a function to calculate the power of a number. The function should have parameters for the base (floating-point number) and exponent (integer)

範例:

2 的三次方為 8。

計算階層(factorial of a number)

寫一個 function 計算 number 的階層，參數 number 的型別是整數。

Write a function to calculate the factorial of a number. The function should have a parameter number of integer type

階層的定義如下:

n的階層（表示為n!）是指從1到n的所有正整數的乘積。

範例:

5 的階層為：120。

等差數列(arithmetic sequence)

寫一個 function 計算等差數列的總和，接受三個參數，分別是等差數列的首項、公差和項數。

Write a function to calculate the sum of an arithmetic sequence. The function should take three parameters: the first term, the common difference, and the number of terms in the arithmetic sequence.

等差數列的定義如下:

  • 首項是等差數列的第一個數字。
  • 公差是等差數列中相鄰兩項之間的差值。
  • 項數是指等差數列的項目個數。

範例:

首項 3，公差 2，項數 5，等差數列的總和為 35。

等比數列(geometric sequence)

寫一個 function 印出等比數列，接受三個參數，分別是等比數列的首項、公比和項數。

Write a function to print a geometric sequence. The function should take three parameters: the first term, the common ratio, and the number of terms in the geometric sequence.

等比數列的定義如下:

  • 首項是等比數列的第一個數字。
  • 公比是等比數列中相鄰兩項之間的差值。
  • 項數是指等比數列的項目個數。

範例:

首項 2，公比 3，項數 5， 等比數列為：2, 6, 18, 54, 162。

判斷數字有幾位數

寫一個 function 印出參數有幾位數，比方 1025 會印出 4。

Write a function that prints out the number of digits in the parameter. For example, it would print out 4 for 1025.

計算圓周率 π

寫一個 function 印出圓周率，使用萊布尼茲公式(Leibniz formula)。

Write a function to print the value of pi using the Leibniz formula.

π = 4 * (1 - 1/3 + 1/5 - 1/7 + 1/9 - 1/11 + ...)

參數 number 表示要計算萊布尼茲級數的多少項，比方 number 為 3 時公式如下。

π = 4 * (1–1/3 + 1/5)

計算歐拉數 e (Euler’s number)

寫一個 function 印出歐拉數 e。

Write a function to calculate the Euler’s number.

e = 1 + 1/1! + 1/2! + 1/3! + 1/4! + ...

參數 number 表示要計算多少項，比方 number 為 3 時公式如下。

e = 1 + 1/1! + 1/2!

整數的全部因數(all the factors of an integer)

寫一個 function 印出 number 的全部因數，參數 number 的型別是整數。

Write a function to print all the factors of a number. The function should have a parameter number of integer type.

範例:

12 的所有因數為：1, 2, 3, 4, 6, 12。

費氏數列(Fibonacci sequence)

寫一個 function 印出費氏數列中的前 n 個數字，參數 n 的型別是整數。

Write a function to print the first n numbers of the Fibonacci sequence. The function should have a parameter n of integer type.

費氏數列的定義如下：

  • 第 0 項和第 1 項都是 1。
  • 從第 2 項開始，每一項都是前兩項的和。

範例: 費氏數列的前7個數字為：1, 1, 2, 3, 5, 8, 13。

質數檢查(prime number)

寫一個 function 檢查參數是否為質數。質數是只有 1 和其本身為因數的數字。

Write a function to check if a parameter is a prime number. A prime number is a number that has only 1 and itself as factors.

印出前 n 個質數(prime number)

寫一個 function 印出前 n 個質數，參數 n 的型別是整數。

Write a function to print the first n prime numbers, where the parameter n is of type integer.

最大公因數和最小公倍數 — 輾轉相除法(GCD & LCM — Euclidean algorithm)

寫一個 function 印出數字 number1 & number2 的最大公因數和最小公倍數，參數 number1 & number2 的型別是整數，使用輾轉相除法。

Write a function to print the greatest common divisor (GCD) and the least common multiple (LCM) of two numbers, number1 and number2, both of integer type, using the Euclidean algorithm.

最大公因數（Greatest Common Divisor, GCD）是指兩個數字中最大的能夠同時整除兩個數字的正整數。

最小公倍數（Least Common Multiple, LCM）是指兩個數字中最小的能夠同時被兩個數字整除的正整數。

公式:

最大公因數可用輾轉相除法，最小公倍數的公式如下。

LCM(a, b) = (a × b) / GCD(a, b)

數字 12 和 18 的最大公因數為 6，最小公倍數為 36。

完全數檢查(perfect number)

寫一個 function 檢查參數是否為完全數。完全數是所有因數(不包括自己)的總和等於自己。例如 6 (因數為1、2、3)。

Write a function to check if a parameter is a perfect number.

回文數檢查(palindrome number)

寫一個 function 檢查參數是否為回文數。回文數是指從左向右讀和從右向左讀都是相同的數字，例如 12321。

Write a function to check if a parameter is a palindrome number.

阿姆斯壯數檢查(自戀數，Armstrong number)

寫一個 function 檢查參數是否為阿姆斯壯數。一個 n 位數的數字，如果每個位數上的數字的 n 次方之和等於該數字本身，那麼這個數就被稱為阿姆斯壯數。

Write a function to check if a parameter is an Armstrong number.

範例:

153 是一個 3 位數的阿姆斯壯數，因為 1³ + 5³ + 3³ = 153。

快樂數(happy number)

寫一個 function 檢查參數是否為快樂數。一個數字所有位數的平方和最終將會等於 1 稱之為快樂數。

Write a function to check if a parameter is a happy number.

範例:

19 是快樂數，因為 1² + 9² = 82，然後 8² + 2² = 68，再來 6² + 8² = 100，最後 1² + 0² + 0² = 1。

質因數分解(prime factorization)

寫一個 function 印出參數的質因數分解。

Write a function to print the prime factor decomposition of the parameter.

範例:

1025 的質因數分解是 5*5*41。

畢達哥拉斯數(Pythagorean number)

寫一個 function 印出畢達哥拉斯數，參數為整數型別的 number，找出所有的 a, b, c，使得它們形成一個畢達哥拉斯三元組並且 a, b, c 都小於或等於 number。

Write a function that prints out Pythagorean numbers. The parameter should be an integer named ‘number’. Find all ‘a’, ‘b’, ‘c’ such that they form a Pythagorean triplet and ‘a’, ‘b’, ‘c’ are all less than or equal to ‘numbe’.

畢達哥拉斯數（也稱為畢達哥拉斯三元組）是一組三個正整數 a, b, c，滿足 a² + b² = c²。例如，(3, 4, 5) 就是一個畢達哥拉斯三元組，因為 3² + 4² = 9 + 16 = 25 = 5²。

範例:

參數 10，印出 (3,4, 5) 和 (6, 8, 10)。

判斷兩序列是否為排列關係(permutation)

寫一個 function 判斷兩序列是否為排列關係，有兩個參數，分別代表序列 1 和序列 2，序列裡有五個整數。

舉例來說，假設我們有一個數字序列 1, 2, 3, 4, 5，這個序列的一個排列可能是 5, 4, 3, 2, 1 或 2, 3, 5, 1, 4 。只要這個新的序列包含了原來序列中的所有元素，且每個元素出現的次數一樣，我們就可以說新的序列是原序列的一個排列。

然而序列 1, 2, 2, 4, 5 並不是序列 1, 2, 3, 4, 5 的排列，因為數字 3 沒有出現，而且數字 2 出現了兩次。
