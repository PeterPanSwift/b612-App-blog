讓開不了口的 App 開口告白

很重要的一堂課，教你用 App 告白！

對一個剛學會變數，function，類別和物件，但是還不會 if else 和迴圈的初學者來說，可以做出什麼樣的 App 呢 ? 我想利用 AVSpeechSynthesizer 物件開口告白是個不錯的例子。

學生們將學會以下四個重點:

  1. 了解開發 App 時，如何運用變數，function，類別和物件等基本程式概念。
  2. 開發 App 時，想要什麼功能，往往透過生出某種東西實現，就好像想要打掃房間時，必須先準備 Dyson 吸塵器，才能開始打掃。

比方想做導航功能，就生出地圖，想要播音樂，就生成 player。而地圖 ，player 這些東西，Apple 早已貼地幫我們做好了，不需要我們重頭寫，大幅減少我們開發 App 的時間。

3. 運用 outlet 連線產生的變數存取畫面上的元件。

比方讀取使用者在 text field 輸入的告白文字。

4. 運用 action 連線產生事件發生時執行的 function。

比方點選嘴巴按鈕時觸發 function，執行開口講話告白的程式。

import UIKit
import AVFoundation
class ViewController: UIViewController {
   @IBOutlet weak var loveTextField: UITextField!
     
   @IBAction func buttonPressed(_ sender: Any) {
      let speechUtterance =  AVSpeechUtterance(string: loveTextField.text!)
      speechUtterance.voice =   AVSpeechSynthesisVoice(language: "zh-TW")
      let synthesizer = AVSpeechSynthesizer()
      synthesizer.speak(speechUtterance)
   }
}
