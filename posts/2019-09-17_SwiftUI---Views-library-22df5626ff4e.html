SwiftUI 的 Views library

SwiftUI 的 Views library 有許多可以加到畫面上的元件，可分成以下四大類別。

Paints

漸層元件。

Control Views

button，slider，list(表格)等常見的 UI 元件。

Other Views

顏色，形狀，圖片等元件。

Layout Views

排版畫面的元件，比方水平排版的 HStack，垂直排版的 VStack。
