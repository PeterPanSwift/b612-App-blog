如何點選 storyboard 裡的 view controller 物件

世界上每個東西都有它的顏色，在 storyboard 裡看到黃色的東西，有很大的可能是 view controller 物件。

想點選 storyboard 裡的 view controller 物件，有三個方法。

  • 點選 controller 畫面上方 bar 裡的黃色 icon 或雙擊 bar。
  • 從 Document Outline 點選黃色 icon。
  • 在 Document Outline 下方的搜尋輸入框輸入 controller 類別名字的前幾個字，快速找出我們想點選的 controller。
方法 1: 點選 controller 畫面上方 bar 裡的黃色 icon 或雙擊 bar。

黃色 icon 代表 controller 物件。

方法 2: 從 Document Outline 點選黃色 icon。

跟方法 1 一樣，同樣是黃色圓形包著白色正方形。

方法 3: 在 Document Outline 下方的搜尋輸入框輸入 controller 類別名字的前幾個字，快速找出我們想點選的 controller。
