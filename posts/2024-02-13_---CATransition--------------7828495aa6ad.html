使用 CATransition 實現圖片水平移動輪播動畫

實現 iOS App 的圖片水平移動輪播動畫有很多方法，比方透過 collection view 或 scroll view 實現。

不過如果目標只是圖片自動滑動，不需讓使用者手動滑動，其實只用一個 image view 加上搭配 push 的 CATransition 也可實現。

接下來我們以紐西蘭好吃的 Whittakers 巧克力為例說明。

在 asset 加入三個口味的 Whittakers 巧克力圖片

在 storyboard 加入 view 和 image view

我們特別用一個 view 裝著 image view，並將 view 的 Clip to Bounds 勾選，如此圖片水平移動時才能不超出 view 的範圍。

使用 CATransition 實現水平移動動畫

let transition = CATransition()
transition.duration = 0.5
transition.timingFunction = CAMediaTimingFunction(name: .easeInEaseOut)
transition.type = .push
transition.subtype = .fromRight
imageView.image = UIImage(named: "pic2")
imageView.layer.add(transition, forKey: nil)

以上程式將實現水平移動動畫，type 控制動畫效果，subtype 控制動畫方向。比方原本圖片是 pic1，type push 和 subtype fromRight 將讓動畫啟動時 pic2 從右邊移動進來，而 pic1 則被推著往左移動消失不見。

掌握動畫的技巧後，我們可再搭配 UIPageControlTimerProgress 或 Timer 實現無限輪播的效果，以下分別為兩種做法的完整程式。

UIPageControlTimerProgress

UIPageControlTimerProgress 是 iOS 17 才有的新功能，相關說明可參考以下連結。

使用 UIPageControlTimerProgress 無限輪播內容 — iOS 17 新功能
開發 iOS App 時，我們通常會用 Timer 實現內容輪播功能，不過最近彼得潘發現 iOS 17 有更簡單的方法。medium.com
import UIKit

class ViewController: UIViewController {
    
    @IBOutlet weak var pageControl: UIPageControl!
    @IBOutlet weak var imageView: UIImageView!
    let pics = ["pic1", "pic2", "pic3"]
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        imageView.image = UIImage(named: pics[0])
        let timerProgress = UIPageControlTimerProgress(preferredDuration: 3)
        pageControl.progress = timerProgress
        timerProgress.resetsToInitialPageAfterEnd = true
        timerProgress.resumeTimer()
        
    }
    @IBAction func pageControlValueChanged(_ sender: UIPageControl) {
        let transition = CATransition()
        transition.duration = 0.5
        transition.timingFunction = CAMediaTimingFunction(name: .easeInEaseOut)
        transition.type = .push
        transition.subtype = .fromRight
        imageView.image = UIImage(named: pics[sender.currentPage])
        imageView.layer.add(transition, forKey: nil)
    }
}


Timer

import UIKit

class ViewController: UIViewController {

    @IBOutlet weak var imageView: UIImageView!
    
    let pics = ["pic1", "pic2", "pic3"]
    var index = 0
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        imageView.image = UIImage(named: pics[index])
        
        Timer.scheduledTimer(withTimeInterval: 3, repeats: true) { _ in
            self.nextImage()
        }
        
    }

    func nextImage() {
        index =  (index + 1) % pics.count
        let transition = CATransition()
        transition.duration = 0.5
        transition.timingFunction = CAMediaTimingFunction(name: .easeInEaseOut)
        transition.type = .push
        transition.subtype = .fromRight
        imageView.image = UIImage(named: pics[index])
        imageView.layer.add(transition, forKey: nil)
    }
}
