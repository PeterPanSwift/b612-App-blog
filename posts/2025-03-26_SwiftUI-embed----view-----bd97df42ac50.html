SwiftUI embed 多個 view 的方法

開發 SwiftUI App 時，我們時常會用 VStack、HStack、List 等容器元件包住多個元件，比方以下例子。

VStack {
    Text("小王子")
    Text("玫瑰")
    Text("狐狸")
}

Xcode 的右鍵選單提供方便的 Embed 功能讓我們快速加入容器元件，然而它卻不夠聰明，它目前只能包住一個元件。

下圖我們選取 3 個 Text，點選 Embed 後卻只包含第一個 Text。

若想 embed 多個 view，目前有以下幾種方法。

  • 將多行程式快速 embed 的手動輸入法。
  • 點選 embed 再手動調整。
  • 請 AI 幫忙。

將多行程式快速 embed 的手動輸入法

將多行程式快速 embed 的方法 — Xcode 14 新功能
從 Xcode 14 開始，我們可以快速地將多行程式 embed，比方將兩個 Text embed 在 VStack。medium.com

點選 Embed 再手動調整

點選 Embed 先包住第一個元件，在框框裡輸入 VStack。

VStack {
    Text("小王子")
}
Text("玫瑰")
Text("狐狸")

選取其它元件。

按下 Move Line Up 將選取的程式上移。(快速鍵 cmd + option + [ )。

VStack {
    Text("小王子")
    Text("玫瑰")
    Text("狐狸")
}

請 AI 幫忙

選取程式後請 AI embed in VStack。

  • 使用 Xcode 。

選取程式後叫出 Coding tools。

輸入 embed in VStack。

生成包住三個 Text 的 VStack { }。

  • 使用 VS Code。

選取程式後，點選燈泡後點選 Modify using Copilot。

手打或語音輸入 embed in VStack。
