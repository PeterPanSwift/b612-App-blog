將 ( ) 裡的程式變成多行排列的 format to multiple lines — Xcode 15 新功能

Xcode 15 推出了 format to multiple lines，可以將 ( ) 裡的程式變成整齊的多行排列。

以下我們以計算 BMI 的 function calculateBMI 說明。

func calculateBMI(weight: Double, height: Double) -> Double {
     let bmi = weight / (height * height)
     return bmi
}

我們必須將滑鼠游標停在 ( ) 裡才能觸發 format to multiple lines，以下我們將游標停在 calculateBMI( 後。

接著我們可用三種方法觸發 Format to Multiple Lines。

  • 方法 1: 從 quick actions 觸發。
  • 方法 2: 點選快速鍵 ctrl + m。
  • 方法 3: 從 menu 點選。

方法 1: 從 quick actions 觸發

點選 cmd + shift + a 叫出 quick actions，輸入 mul 後點選 Format to Multiple Lines。

方法 2: 點選快速鍵 ctrl + m

方法 3: 從 menu 點選

從 menu 點選 Refactor > Structure > Format to Multiple Lines。

變成多行程式的成果如下，calculateBMI( ) 裡的參數會整齊地多行排列。

func calculateBMI(
    weight: Double,
    height: Double
) -> Double {
    let bmi = weight / (height * height)
    return bmi
}

呼叫 calculateBMI 時，( ) 裡的參數也可以用 format to multiple lines 多行排列，例如以下例子。

let bmi = calculateBMI(
    weight: 70,
    height: 1.75
)
print(bmi)
