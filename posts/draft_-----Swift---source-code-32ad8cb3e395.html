3 研究 Swift 程式。

例子 1: Bool 的 function toggle()。

Bool 的 function toggle 可以把 true 變 false，false 變 true，我們想了解此 function 是如何實現的。

(1) 在 Find navigator 裡 search func toggle。

(2) 點選 Bool.swift 下的 func toggle()。

(3) func toggle() 的程式碼。

extension Bool {
/// Toggles the Boolean variable's value.
///
/// Use this method to toggle a Boolean value from `true` to `false` or from
/// `false` to `true`.
///
///    var bools = [true, false]
///
///    bools[0].toggle()
///    // bools == [false, false]
   @inlinable
   public mutating func toggle() {
      self = !self
   }
}

原來 func toggle() 的程式是如此簡單，一行 self = !self 就可實現 !
