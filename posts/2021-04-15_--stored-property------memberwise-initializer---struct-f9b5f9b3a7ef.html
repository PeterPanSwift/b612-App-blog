從 stored property 自動生成 memberwise initializer 的 struct

比起 class，struct 少了繼承，但卻多了從 stored property 自動生成 memberwise initializer 的特異功能，方便我們在建立 struct 資料時傳入每個屬性的初始值。

自動產生 memberwise initializer 的 struct

接下來讓我們以可愛寶寶的 Baby 型別為例說明。

struct Baby {
    var name: String
    var age: Int
}
var cuteBaby = Baby(name: "peter", age: 18)

Baby 有兩個屬性，name 和 age。由於 struct 自動生成以每個屬性名字為參數的 memberwise initializer，因此我們可在產生寶寶時傳入每個屬性的內容。

若是我們自己定義 memberwise initializer，它的模樣將類似以下的 init。

struct Baby {
    var name: String
    var age: Int
    
    init(name: String, age: Int) {
        self.name = name
        self.age = age
    }
}

從這裡我們可以看出 memberwise initializer 自動生成是個超貼心的發明。如果沒有它，當 Baby 有十個屬性，而你希望產生寶寶時一一設定每個屬性，則須自己定義有十個參數的 init，打字打到手指長繭。

class 不會自動產生 memberwise initializer

由於有 memberwise initializer，在 struct Baby 的定義裡，我們不需另外指定初始值。相反的，若以 class 定義 Baby，這樣的寫法將馬上冒出紅色錯誤，因為物件建立時屬性一定要完成初始，但 class 卻不像 struct 有自動生成的 memberwise initializer 幫忙初始屬性。

自動產生 default initializer & memberwise initializer

如果 struct 的屬性在宣告時已完成初始，已設定內容或宣告為 optional，那麼 struct 將更貼心，買一送一，同時送給我們無參數的 default initializer 和 memberwise initializer。

struct Baby {
    var name = "peter"
    var age = 1
}
var cuteBaby = Baby()
cuteBaby = Baby(name: "wendy", age: 2)

當 struct 的 property 是 optional 或有初始值時

當 struct 的 property 是 optional 或有設定初始值時，它是變數或常數，自動產生的 initializer 將有些不一樣的地方。

struct Baby {
    var name: String?
    var age = 1
    var weight: Double
}

以上程式的 age 初始值為 1，name 初始值為 nil，因此產生 Baby 時 age & name 都是可傳可不傳的參數，但是 weight 一定要傳，因為它沒初始值。

接著我們將 name 和 age 改成 let。

struct Baby {
    let name: String?
    let age = 1
    var weight: Double
}

age 是常數，因此它永遠為 1，產生 Baby 時不用再傳 age。name 是 optional 常數，產生寶寶時一定要帶參數 name，因為常數的 optional 沒有預設值。如果 name 預設為 nil，那將是很可怕的事 ! 此時寶寶這輩子都別想有名字了，因為 name 是常數，無法改名，他會恨你一輩子的。

在 struct 裡自訂 init

struct 可以自動生成 memberwise initializer，不過我們也可以定義自己想要的 init。值得注意的，一旦自訂 init，struct 將不再好心生成 memberwise initializer ＆ default initializer，此時我們只能以自訂的 init 產生資料。

struct Baby {
    var name = "peter"
    var age = 1
    
    init(name: String) {
        self.name = name
        age = 1
    }
}
var cuteBaby = Baby(name: "peter")
