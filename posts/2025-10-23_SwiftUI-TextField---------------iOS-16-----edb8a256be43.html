SwiftUI TextField 可輸入多行文字自動長高 ~ iOS 16 新功能

從 iOS 16 開始，SwiftUI TextField 可輸入多行文字，而且還會隨著換行自動長高呢。

透過參數 axis 傳入 .vertical，即可讓 TextField 容納多行文字。

struct ContentView: View {
    @State private var note = ""
    
    var body: some View {
        TextField("note", text: $note, axis: .vertical)
            .textFieldStyle(.roundedBorder)
            .padding()
    }
}

當我們輸入文字時，TextField 將隨著文字換行自動長高。

剛剛的 TextField 可以長到多高呢 ? 就像彼得潘長到 17x 就不再長高一樣，TextField 的身高也有極限，當它長到佔滿可用空間時將停止長高。不過別擔心，此時 TextField 將多出上下捲動的功能，因此我們還是可持續輸入文字換行。

通常我們會希望限制 TextField 的高度，這樣畫面才塞得下其它東西。透過 lineLimit 可限制它的高度，lineLimit(3) 表示 TextField 最多長到三行的高度，之後變成上下捲動容納更多行。

struct ContentView: View {
    @State private var note = ""
    
    var body: some View {
        TextField("note", text: $note, axis: .vertical)
            .textFieldStyle(.roundedBorder)
            .lineLimit(3)
            .padding()
    }
}

當我們在 lineLimit 傳入數字範圍時，將讓 TextField 一開始就是多行的高度。

struct ContentView: View {
    @State private var note = ""
    
    var body: some View {
        TextField("note", text: $note, axis: .vertical)
            .textFieldStyle(.roundedBorder)
            .lineLimit(3...6)
            .padding()
    }
}

lineLimit(3…6) 讓 TextField 一開始就有三行的高度。

若是傳入的範圍沒有包含上限，TextFeild 長高時將可長到佔滿可用空間。

struct ContentView: View {
    @State private var note = ""
    
    var body: some View {
        TextField("note", text: $note, axis: .vertical)
            .textFieldStyle(.roundedBorder)
            .lineLimit(3...)
            .padding()
    }
}

參考範例連結

TextField("Log your small wins", text: $note, axis: .vertical)
    .multilineTextAlignment(.leading)
    .lineLimit(5...Int.max)
