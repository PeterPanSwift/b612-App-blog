重新執行 SwiftUI preview(Refresh Canvas)

開發 SwiftUI App 時，有時我們會希望 SwiftUI preview 能重新執行，回到初始的狀態。

比方彼得潘為了情人節約會開發了超重要的約會穿搭 App，點選 button 將隨機挑一件 benetton 漂亮的針織衣。

畫面的程式如下。

struct ContentView: View {
    @State private var datingOutfit = "datingOutfit"
 
    var body: some View {
        VStack {
            Image(datingOutfit)
                .resizable()
                .scaledToFit()
            
            Button("情人節約會穿什麼") {
                let clothes = [
                   "北歐風緹花羊毛針織上衣",
                   "多彩橫條圓領針織上衣",
                   "橘藍綠色塊拼接針織上衣",
                   "立體織紋羊毛針織上衣"
                ]
                datingOutfit = clothes.randomElement()!
            }
            .font(.largeTitle)
        }
    }
}

畫面一開始顯示約會挑衣服的照片。

點選情人節約會穿什麼後，顯示 benetton 漂亮的衣服。

此時若想讓 preview 重新執行，回到初始狀態顯示挑衣服的照片，我們有兩個方法。

  • 方法1: 點選三角形的 live 按鈕。
  • 方法2: 按下 cmd + option + p。

對應的 Xcode menu 是 Refresh Canvas。
