利用 present(_:animated:completion:) 顯示 controller 頁面

利用 UIViewController 的 function present(_:animated:completion:)，我們可以切換顯示下一頁的 controller 頁面。雖然在 storyboard 拉拉 segue 感覺比較輕鬆，不過 iOS SDK 有許多內建的 controller，比方拍照的 UIImagePickerController，播放影片的 AVPlayerViewController。這些 controller 往往需要額外設定屬性才能發揮作用，因此我們習慣從程式生成這些 controller，設定它們的屬性，然後再呼叫 present(_:animated:completion:) 顯示畫面。

接下來我們將以播放影片的 AVPlayerViewController 為例，用它來播放男主角也叫彼得的蜘蛛人電影。

1 加入 AVKit。
import AVKit

因為 AVPlayerViewController 定義在 AVKit framework 裡。

2 產生點擊按鈕觸發的 @IBAction function playMovie。
@IBAction func playMovie(_ sender: Any) {
}
3 在 function playMovie 裡生成並顯示 AVPlayerViewController。
@IBAction func playMovie(_ sender: Any) {
   let playerViewController = AVPlayerViewController()
   if let url = URL(string: "https://video-ssl.itunes.apple.com/apple-assets-us-std-000001/Video117/v4/2c/f6/e6/2cf6e630-8f12-e721-c80d-b689aa8f9804/mzvf_1371130011356140075.640x354.h264lc.U.p.m4v") {
      playerViewController.player = AVPlayer(url: url)
      present(playerViewController, animated: true, completion: nil)
   }
}

說明

playerViewController.player = AVPlayer(url: url)

播放影片的其實是 AVPlayer，而 AVPlayerViewController 則負責顯示 AVPlayer 播放的影片內容。因此我們利用 url 生成 AVPlayer 後，再將它設為 AVPlayerViewController 的 player。

present(playerViewController, animated: true, completion: nil)

呼叫 present 顯示 AVPlayerViewController 。

present(_:animated:completion:) — UIViewController | Apple Developer Documentation
The object on which you call this method may not always be the one that handles the presentation. Each presentation…developer.apple.com

function present(_:animated:completion:) 的宣告如下

func present(_ viewControllerToPresent: UIViewController, 
 animated flag: Bool, 
 completion: (() -> Void)? = nil)

參數說明

  • viewControllerToPresent: 即將顯示的 controller 。
  • flag: 頁面切換時是否有動畫。
  • completion: controller 畫面顯示後執行的程式。

大部份我們呼叫 present 時 completion 都是傳入 nil，因為我們只想顯示 controller 畫面，並不須在畫面顯示後做額外動作。

結果

Cool，因為 AVPlayerViewController，複雜的影片播放功能一下就完成了。可惜它目前有個小小缺點，我們得動動小指按下播放鍵後才會開始播放。

讓我們再微調一下程式，這次我們呼叫 present 時在參數 completion 傳入 closure，執行 playerViewController.player?.play()，如此即可在 AVPlayerViewController 畫面顯示後自動開始播放 !

present(playerViewController, animated: true) {
   playerViewController.player?.play()
}

學會用 present(_:animated:completion:) 顯示 AVPlayerViewController 後，顯示其它的 controller 也是一樣的方法，只是我們必須研究每種 controller 需要設定的屬性。有興趣練習的朋友不妨試試以下連結提到的 11 種 iOS SDK 的內建 controller，透過他們快速實現 App 的分享，寄信，拍照等特別功能。
