設定橫向的 SwiftUI preview

SwiftUI preview 預設會顯示直向的 App 畫面，不過有些畫面較適合以橫向顯示，例如常見的遊戲畫面。

以下我們介紹各種將 preview 變成橫向畫面的方法。

  • 設定 Device Settings 的 Orientation。(Xcode 14 以上)
  • 呼叫 modifier previewInterfaceOrientation。(iOS 15 以上)
  • 呼叫 modifier previewLayout。

設定 Device Settings 的 Orientation (Xcode 14 以上)

點選 preview 下方的 Device Settings 按鈕，然後將 Orientation 的開關打開，勾選 Landscape (left) 或 Landscape (right)。

呼叫 modifier previewInterfaceOrientation (iOS 15 以上)

呼叫 modifier previewInterfaceOrientation，傳入 .landscapeLeft 或 .landscapeRight。

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
            .previewInterfaceOrientation(.landscapeLeft)
    }
}

呼叫 modifier previewLayout

舊版的 Xcode & iOS 不支援前兩個方法，只能呼叫 modifier previewLayout 設定固定尺寸，比方設定 iPhone 14 Pro 橫向的寬高 852 * 393。

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
            .previewLayout(.fixed(width: 852, height: 393))
    }
}

值得注意的，此方法有個小小的缺點，當 preview 在 Live mode 時會變回直向，因此測試 App 功能時必須透過模擬器才能以橫向操作。
