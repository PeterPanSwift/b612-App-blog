從 Symbol Library 加入 SF Symbol

以下分別介紹 SwiftUI & UIKit 的做法。

SwiftUI

  • 將輸入的游標停在準備輸入的位置
  • 打開 Library 視窗。

從 Quick Actions 點選 Show Library。(快速鍵 cmd + shift + l)

  • 切換到 Symbols Library 頁面，找到喜歡的圖片後將它拖曳到程式裡。

在此我們拖曳 die.face.1.fill。

Xcode 自動幫我們輸入 Image(systemName: "die.face.1.fill")。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "die.face.1.fill")
        }
    }
}

ps: 若是雙擊 Library 視窗的 symbol，而不是用拖曳的方式，Xcode 只會幫我們輸入圖片名字。因此若雙擊 symbol die.face.1.fill，Xcode 將輸入 die.face.1.fill。

UIKit

  • 編輯 storyboard，打開 Library 視窗。

從 Quick Actions 點選 Show Library。(快速鍵 cmd + shift + l)

或點選 Document Outline 左下的 +。

  • 切換到 Symbols Library 頁面，找到喜歡的圖片後拖曳到畫面上
