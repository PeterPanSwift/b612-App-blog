值得注意的，參數 decoration 只會讓 Container 變圓角，若想讓 Container 的 child 也變圓角，必須再設定 clipBehavior 為 Clip.antiAlias。

  @override

    return Scaffold(
      body: Center(
        child: Padding(
          padding: const EdgeInsets.all(8.0),
          child: Container(
            clipBehavior: Clip.antiAlias,

              borderRadius: BorderRadius.circular(10),

Button 搭配 style，shape 設定 RoundedRectangleBorder

什麼都不用做，Button 預設即有圓角矩形背景。

ElevatedButton(
  onPressed: () {},
  child: const Text('button'),
)

設定 button 的 style，利用 RoundedRectangleBorder 調整圓角弧度。

ElevatedButton(
  onPressed: () {},
  style: ElevatedButton.styleFrom(
    shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(10),
    ),
  ),
  child: const Text('button'),
)

TextField 搭配 OutlineInputBorder，borderRadius 設定 BorderRadius.circular

InputDecoration 的 border 設定 OutlineInputBorder 即有圓角矩形效果。

TextField(
  decoration: InputDecoration(
    hintText: 'name',
    border: OutlineInputBorder(),
  ),
)

利用 OutlineInputBorder 的 borderRadius 調整圓角弧度。

TextField(
  decoration: InputDecoration(
    hintText: 'name',
    border: OutlineInputBorder(
      borderRadius: BorderRadius.all(
        Radius.circular(20),
      ),
    ),
  ),
)
