利用 Swift tuple 依序比大小

Swift 的 tuple 很適合用來依序比大小，以下我們用經典的撲克牌說明。

首先我們用 enum 定義撲克牌的花色(Suit) & 數字(Rank)。

enum Suit: String {
    case spades = "♠️"
    case hearts = "♥️"
    case diamonds = "♦️"
    case clubs = "♣️"

    var priority: Int {
        switch self {
        case .spades: 4
        case .hearts: 3
        case .diamonds: 2
        case .clubs: 1
        }
    }
}

enum Rank: Int {
    case two = 2
    case three, four, five, six, seven, eight, nine, ten, jack, queen, king, ace

    var displayName: String {
        switch self {
        case .jack: "J"
        case .queen: "Q"
        case .king: "K"
        case .ace: "A"
        default: "\(rawValue)"
        }
    }
}

接著用 struct Card 定義撲克牌。

struct Card {
    let rank: Rank
    let suit: Suit
}

為了比較 Card 的大小，我們可讓 Card 遵從 Comparable，然後定義 < function，利用 if else 比較 rank 的 rawValue & suit 的 priority。

struct Card: Comparable {
    let rank: Rank
    let suit: Suit
    
    static func < (lhs: Card, rhs: Card) -> Bool {
        if lhs.rank.rawValue == rhs.rank.rawValue {
            lhs.suit.priority < rhs.suit.priority
        } else {
            lhs.rank.rawValue < rhs.rank.rawValue
        }
    }
}

利用 Swift tuple 將可進一步簡化 if else 比大小的寫法，因為 tuple 可以依序比較，它會先比較第一個成員，若相等再比較第二個成員，其它以此類推。

以下程式運用 tuple 先比較 rank.rawValue，若相等再比較 suit.priority。

struct Card: Comparable {
    let rank: Rank
    let suit: Suit
    
    static func < (lhs: Card, rhs: Card) -> Bool {
        (lhs.rank.rawValue, lhs.suit.priority) < (rhs.rank.rawValue, rhs.suit.priority)
    }
}

若想進一步簡化 Card 的 < function，我們也可先將 Suit & Rank 變成 Comparable。

enum Suit: String, Comparable {
    case spades = "♠️"
    case hearts = "♥️"
    case diamonds = "♦️"
    case clubs = "♣️"

    var priority: Int {
        switch self {
        case .spades: 4
        case .hearts: 3
        case .diamonds: 2
        case .clubs: 1
        }
    }
    
    static func < (lhs: Suit, rhs: Suit) -> Bool {
        lhs.priority < rhs.priority
    }
}

enum Rank: Int, Comparable {
    case two = 2
    case three, four, five, six, seven, eight, nine, ten, jack, queen, king, ace

    var displayName: String {
        switch self {
        case .jack: "J"
        case .queen: "Q"
        case .king: "K"
        case .ace: "A"
        default: "\(rawValue)"
        }
    }
    
    static func < (lhs: Rank, rhs: Rank) -> Bool {
        lhs.rawValue < rhs.rawValue
    }
}

如此 Card 的 < function 將可直接比較 rank & suit。

struct Card: Comparable {
    let rank: Rank
    let suit: Suit
    
    static func < (lhs: Card, rhs: Card) -> Bool {
        (lhs.rank, lhs.suit) < (rhs.rank, rhs.suit)
    }
}

除了 Comparable 的 < function，排序也很適合運用 Swift tuple 比大小。

範例: 寶寶 array 排序時先比體重，再比身高。

struct Baby {
    let name: String
    let weight: Double
    let height: Double
}

let babies = [
    Baby(name: "小明", weight: 3.5, height: 50.0),
    Baby(name: "小華", weight: 3.2, height: 48.5),
    Baby(name: "小美", weight: 3.5, height: 51.2)
]

let sortedBabies = babies.sorted {
    ($0.weight, $0.height) < ($1.weight, $1.height)
}
