Could not instantiate class named WKWebView 問題

當 WKWebView 是從 Interface Builder 生成，App 執行時可能會發生閃退，出現以下的錯誤訊息。

Thread 1: "Could not instantiate class named WKWebView because no class named WKWebView was found; the class needs to be defined in source code or linked in from a library (ensure the class is part of the correct target)"

閃退的原因是因為不認得 WKWebView。解決的方法是從 Project navigator 點選專案後，切換到 TARGETS 下 App 的 General 頁面，然後點選 Frameworks, Libraries, and Embedded Content 下的 + 新增 WebKit framework。
