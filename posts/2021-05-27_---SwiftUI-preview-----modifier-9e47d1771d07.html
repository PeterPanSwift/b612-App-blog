SwiftUI 的 preview 樣式可以透過 modifier 做調整，方便我們預覽不同情況的 App 畫面，接下來我們將逐一認識這些調整 preview 樣式的 modifier。

調整 SwiftUI preview 樣式的 modifier

SwiftUI 的 preview 樣式可以透過 modifier 做調整，方便我們預覽不同情況的 App 畫面，接下來我們將逐一認識這些調整 preview 樣式的 modifier。

範例 ContentView 的畫面設計如下。

struct ContentView: View {
    var body: some View {
        VStack {
            Image("book")
                .resizable()
                .scaledToFit()
                .clipShape(Circle())
            Text("SwiftUI Apprentice")
                .font(.largeTitle)
        }
    }
}

設定裝置的 previewDevice

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        Group {
            ContentView()
                .previewDevice("iPhone 12")
            ContentView()
                .previewDevice("iPad Air (4th generation)")
        }
        
    }
}

輸入 preview modifier 的方法有以下三種:

  • 手動輸入。
  • 點選 ContentView，然後從 Attributes inspector 設定內容。
  • 點選 preview 畫面上方的 Inspect Preview 按鈕，然後從跳出的視窗設定。

設定 preview 名字的 previewDisplayName

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        Group {
            ContentView()
                .previewDisplayName("iPhone 12")
                .previewDevice("iPhone 12")
            ContentView()
                .previewDisplayName("iPad Air")
                .previewDevice("iPad Air (4th generation)")
        }
        
    }
}

設定畫面尺寸的 previewLayout

  • sizeThatFits

剛好呈現內容的尺寸。

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
            .previewLayout(.sizeThatFits)
    }
}
  • fixed

指定寬高。

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
            .previewLayout(.fixed(width: 300.0, height: 200))
        
    }
}

設定畫面顏色的 preferredColorScheme

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        Group {
            ContentView()
                .preferredColorScheme(.dark)
            ContentView()
                .preferredColorScheme(.light)
        }
    }
}

設定文字大小的 environment sizeCategory

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        Group {
            ContentView()
                .environment(\.sizeCategory, .extraSmall)
            ContentView()
                .environment(\.sizeCategory, .accessibilityExtraExtraExtraLarge)
        }        
        
    }
}

設定語言的 environment locale

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        Group {
            ContentView()
                .environment(\.locale, Locale(identifier: "en"))
            ContentView()
                .environment(\.locale, Locale(identifier: "zh-Hant"))
        }
     }
}
