在參數的輸入框按下 enter 鍵，加快程式開發速度

當我們撰寫 Swift 程式時，時常需要在呼叫 function 或生成東西的 ( ) 裡傳入參數。當我們輸入參數時，以下情況可直接在參數輸入框按 enter，加快程式的開發速度。

  • 輸入型別加 ( ) 生成的東西

想在輸入框輸入參數型別加 ( ) 時，可先按 enter，讓 Xcode 輸入參數型別名。

  • 輸入 closure

輸入框是 function 型別時，按 enter 後 Xcode 會幫我們輸入 closure。

例子 1 : 輸入型別加 ( ) 生成的東西

按下 enter 鍵讓 Xcode 幫我們輸入型別的名字。比方以下生成 UIButton 時，輸入 CGRect() 的例子。

在 CGRect 顯示框框時按下 enter 鍵，讓 Xcode 幫我們輸入 CGRect，不用再自己打。

接著輸入 x，選到想要的目標後按 enter。

例子2 : 輸入 closure

按下 enter 鍵讓 Xcode 幫我們輸入 closure 的部分程式，比方以下生成 timer 時，在參數 block 傳入 closure 的例子。

在 block 的 (Timer) -> Void 顯示框框時按下 enter 鍵，讓 Xcode 幫我們輸入 closure 的部分程式。

完成 closure 的程式碼。

Timer.scheduledTimer(withTimeInterval: 1, repeats: true) { _ in
    print("do something")
}
