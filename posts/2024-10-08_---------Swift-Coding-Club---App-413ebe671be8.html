邀請朋友一起參加 Swift Coding Club 寫 App

開發 App 是件開心的事，不過比起自己一個人孤單單在房間寫 App，大家聚在一起開發，一邊吃 pizza 一邊寫 App 會更開心。

最近 Apple 推出 Swift Coding Club，幫助我們更容易創建 Swift 的社團、門派、秘密組織，邀請朋友一起開開心心寫 App。

Swift Coding Clubs are a great way to learn app development, build community, and get ready for the Swift Student…developer.apple.com

Starter Kit

邀請朋友參加 Swift Coding Club 的第一步其實是最難的。如何讓他們知道 club 的存在和引起他們的興趣呢？

別擔心，Apple 為我們準備了 Starter Kit，包含許多適合用來宣傳和邀請的素材。

點選 See the Starter Kit

點選 Download 下載 Starter Kit。

點選 SF Mono font 旁的 here 連到字體網頁，點選 Download SF Mono。

安裝 SF Mono Fonts。

Starter Kit 解壓縮後會變成資料夾 Swift Coding Club assets，裡面包含許多可以客製的檔案。

  • Club-icon-editable。

Club 的封面投影片，在 Add location here 輸入 club 的名字或地點。

  • Invitation-editable。

邀請卡，輸入 club 的名字或地點、網址和 email。

  • 社群分享圖。

Social-card 1。

Social-card 2。

  • Tshirt Template。

利用 T-Shirt-Logo 製作帥氣的社服。

Apple 建議用黑色 Tshirt 搭配白色的 Swift logo。

Meeting plans

辛苦地建立 club，邀請許多朋友加入 club，可以稍微躺著休息一下了嗎？

最困難，最花時間的事情才要開始耶。要如何準備有趣的活動和教學內容，才不會第一堂課教室擠到坐不下，第五堂課只有小貓兩三隻呢？

這部分貼心的 Apple 也幫我們準備好了，教材 Meeting plans 包含 SwiftUI foundations、Data modeling、Spatial computing 三大主題。

Meeting plans - Swift Coding Club - Learn to code - Apple Developer
Swift Coding Club meeting plans provide suggestions for your club activities to help you learn the basics of app…developer.apple.com

每個主題包含多個單元教材，而且 club 成員還可在每次的 club 聚會分組完成有趣的小組活動(group activity)。

ps: Meeting plans 的教材來自 Develop in Swift Tutorials。
