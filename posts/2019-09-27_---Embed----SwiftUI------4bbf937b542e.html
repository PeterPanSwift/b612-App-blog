利用 Embed 加入 SwiftUI 容器元件

SwiftUI 有一些特別的容器元件，專門以 { } 容納其它的元件，比方 HStack，VStack，List 等，Xcode 貼心地在選單裡提供 Embed 功能，方便我們快速加入這些元件。

比方我們想將 Text 元件包在 VStack 裡。

struct ContentView: View {
   var body: some View {
      Text("我要一步一步學App，等待朋友崇拜看著我的臉，小小的天有大大的夢想，重重的課做著輕輕的App。我要一步一步學App，在最餓點吃著蘋果寫著Code，小小的天流過的淚和汗，總有一天我有屬於我的App。")
   }
}

從 Text 上叫出右鍵選單，然後點選 Embed。

點選 Embed 後的程式。

在框框輸入元件的名字，在此我們輸入 VStack。

VStack {
   Text("我要一步一步學App，等待朋友崇拜看著我的臉，小小的天有大大的夢想，重重的課做著輕輕的App。我要一步一步學App，在最餓點吃著蘋果寫著Code，小小的天流過的淚和汗，總有一天我有屬於我的App。")
}

ps:

  • 從 macOS 15 開始，按下 control + return 也可叫出右鍵選單。
