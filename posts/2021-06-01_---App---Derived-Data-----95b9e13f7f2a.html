刪除 App 的 Derived Data 資料夾

開發 iOS App 時，我們時常因為以下原因需要刪除 App 的 Derived Data 資料夾:

  • 遇到一些奇怪的 Xcode 問題，比方自動完成出現問題，或是 compile 出現奇怪的錯誤。
  • Mac 空間不夠，想要多一點儲存空間。

以下我們介紹幾種刪除 Derived Data 的方法。

  • 方法 1: 從 Mac 的 System Information 做刪除，刪除所有 App 的 Derived Data。
  • 方法 2: 從 Xcode 的 Preferences 做刪除，刪除所有 App 的 Derived Data。
  • 方法 3: 從 Xcode 的 Show Build Folder in Finder 做刪除，刪除某個 App 的 Derived Data。

方法 3: 從 Xcode 的 Show Build Folder in Finder 做刪除，刪除某個 App 的 Derived Data

從 Quick Actions 點選 Show Build Folder in Finder。

此時 Finder 將顯示 Build 資料夾。

從 Build 資料夾往回找到前一層的 app 資料夾，然後將它刪除。app 資料夾名稱將為專案名加上一串亂碼，以下圖為例，它是 Demo 加上一串亂碼。

ps: 從 Quick Actions 點選 Clean Build Folder 可刪除 Build 資料夾的內容。
