透過 Android Studio 的 Create Gist 快速分享程式

Android Studio 內建 Create Gist 功能，方便我們快速將程式分享給朋友，一鍵將程式片段變成 GitHub 的 gist。

Create Gist 前 Android Studio 必須已登入 GitHub 帳號，若是尚未登入，請先從 Find Action(cmd + shift + a) 打開 Settings 的 GitHub。

點選 Add account。

點選 Log In via GitHub。

從打開的網頁點選 Authorize in GitHub。

Android Studio 的 Settings GitHub 頁面成功顯示剛剛登入的帳號。

現在我們可以開始 Create Gist 了。首先選取想分享的程式片段。

從右鍵選單點選 Create Gist。

從 Find Action 也可點選 Create Gist。

設定 Gist 的相關資訊。

成功建立 Gist，GitHub Gist 網頁出現了我們分享的程式。

除了將選取的程式變成 Gist，我們也可以將整個檔案的程式變成 Gist，只要不要選取程式，直接在開啟的檔案上 Create Gist 即可。
