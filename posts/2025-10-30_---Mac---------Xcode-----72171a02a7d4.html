我的 Mac 可以馴養新版的 Xcode 嗎 ?

同學們剛開始學習開發 iOS App 時，常常會遇到以下問題。

  • 我的 Mac 可以馴養(安裝) 新版的 Xcode 嗎 ?
  • 不行的話要買哪一款 Mac 呢 ?
  • 沒錢買新 Mac 的話可以裝哪一版 Xcode 呢 ?

要回答以上問題有點麻煩，因為要查詢兩個網頁，查詢 Xcode 支援的 macOS 版本以及 Mac 支援的 macOS 版本。

Xcode
Learn about using Xcode, Apple’s integrated development environment, to develop software.developer.apple.com
macOS — Wikipedia

為了快速知道 Mac 是否能馴養 Xcode，彼得潘決定請 AI 幫忙寫個網頁，Can Your Mac Tame Xcode?

  • 請 AI 撰寫 python 程式，解析 Xcode 網頁的 Xcode Releases 表格，生成 Xcode 版本和 macOS 版本的 JSON。

xcode_releases.json。

  • 請 AI 撰寫 python 程式，解析 macOS 網頁的 Hardware compatibility 表格，生成 macOS 版本和 Mac 機型的 JSON。

hardware_compatibility_os_supported.json。

  • 加入 AI 繪製的馴養成功和馴養失敗圖片。
  • 請 AI 依據 JSON 製作網頁，判斷 Mac 是否能馴養 Xcode ?

AI 成功生成網頁，專案的 GitHub 連結如下。

以下我們分別示範幾個馴養成功和馴養失敗的例子。

當 Mac 馴養了 Xcode

  • MacBook Pro M5、Xcode 26.1。
  • MacBook Air M4、macOS 15.7、Xcode 26.1。

當 Xcode 不被 Mac 馴養

  • MacBook Air M4、macOS 15.5、Xcode 26.1。

馴養失敗，升級到 macOS 15.6 以上即可馴養。

  • MacBook Pro 2017、Xcode 26.1。

馴養失敗，MacBook Pro 2017 能安裝的最新 Xcode 是 Xcode 15.2，若要安裝 Xcode 26.1 則需購買新的 Mac。

  • MacBook Pro M5、macOS 26.1、Xcode 16.3。

馴養失敗，MacBook Pro M5 搭配 macOS 26.1 不能安裝 Xcode 16.3，請安裝最新版的 Xcode 26.1。
