設定 Android Studio 的背景圖片

寫程式時保持愉悅的心情是很重要的，為了提升自己寫程式的士氣，以下彼得潘將 Android Studio 的背景變成最愛的桌球。

從 Find Action(cmd + shift + a)點選 Set Background Image。

Image 選擇一手 MacBook 一手桌球拍的 Peter。

透明度(Opacity) 設定 15，點選 Set for Editor and Tools。

Yeah ! 寫 code 的每一秒都能看到帥氣的桌球拍。
