Swift 集合的 contains & allSatisfy — 判斷 集合裡的東西是否符合條件

開發 iOS App 時，我們時常需要判斷 array 或 dictionary 裡的東西是否符合條件，比方來自台灣，英國，拉斯維加斯和日本的四位大師參加百年一度的 Swift 程式大師大賽，滿分 100 分，array 裡儲存了它們的分數。

var grades = [70, 90, 80, 99]

我們可能會有以下兩個需求

1 想知道此次考試是否有高手。

array 裡是否有人分數大於 90。

2 想知道是否每個人都順利過關。

array 裡是否每個人的分數都大於等於 60。

以上兩個問題皆可用迴圈解決，但 Swift 早為我們準備了 contains(where:) & allSatisfy(_:)，幫助我們更容易解決這些問題。

contains(where:)
func contains(where predicate: (Element) throws -> Bool) rethrows -> Bool
contains(where:) - Array | Apple Developer Documentation
Returns a Boolean value indicating whether the sequence contains an element that satisfies the given predicate.developer.apple.com
var grades = [70, 90, 80, 99]
var hasMaster = grades.contains { (grade) -> Bool in
   grade > 90
}
grades = [70, 90, 80, 79]
hasMaster = grades.contains { (grade) -> Bool in
   grade > 90
}

contains 會依序對 array 的成員執行參數 closure 的程式，只要有一個成員運算後回傳的結果為 true，contains 就會回傳 true。因此剛剛的例子只要有一個人的分數大於 90，hasMaster 將等於 true。

除了 contains(where:)，還有個跟它很像的 function 叫 contains(_:)，可以用來判斷集合裡是否包含某個成員，不過成員的型別必須遵從 Equatable protocol 才能使用。

allSatisfy(_:)
func allSatisfy(_ predicate: (Element) throws -> Bool) rethrows -> Bool
allSatisfy(_:) - Array | Apple Developer Documentation
This documentation contains preliminary information about an API or technology in development. This information is…developer.apple.com
var grades = [70, 90, 80, 99]
var allPass = grades.allSatisfy { (grade) -> Bool in
   grade >= 60
}
grades = [70, 90, 50, 79]
allPass = grades.allSatisfy { (grade) -> Bool in
   grade >= 60
}

allSatisfy(_:) 是 Swift 4.2 的新功能。它會依序對 array 的成員執行參數 closure 的程式，必須每個成員運算後回傳的結果皆為 true，它才會回傳 true。因此剛剛的例子只要有一個人的分數小於 60，allPass 將等於 false。
