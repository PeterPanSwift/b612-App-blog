讓 swift for 迴圈數字由大到小的方法

利用 reversed 倒轉順序

for i in (1...10).reversed() {
    print(i)
}

利用 stride(from:to:by:)，不包含結尾的數字

從 10 印到 2。

for i in stride(from: 10, to: 1, by: -1) {
    print(i)
}

利用 stride(from:through:by:)，包含結尾的數字

從 10 印到 1。

for i in stride(from: 10, through: 1, by: -1) {
    print(i)
}
