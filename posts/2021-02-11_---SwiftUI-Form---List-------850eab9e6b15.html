設定 SwiftUI Form 或 List 的背景顏色

SwiftUI 的 Form & List 都有個預設的背景顏色。

struct ContentView: View {
    
    @State private var name = ""
    @State private var address = ""
       
    var body: some View {
        Form {
            TextField("name", text: $name)
            TextField("address", text: $address)
        }
        
    }
    
}

Form 的預設背景色是灰色。

struct ContentView: View {
    
    @State private var name = ""
    @State private var address = ""
    var body: some View {
        List {
            TextField("name", text: $name)
            TextField("address", text: $address)
        }
        
    }
    
}

List 的預設背景色是白色。(dark mode 是黑色）

想修改很簡單，只要設定 UITableView.appearance().backgroundColor 即可。

struct ContentView: View {
    
    @State private var name = ""
    @State private var address = ""
    init() {
        UITableView.appearance().backgroundColor = .yellow
    }
       
    var body: some View {
        Form {
            TextField("name", text: $name)
            TextField("address", text: $address)
        }
        
    }
    
   
}
