請 AI 生成 JSON API — 生成 JSON、建立 gist，取得 API 網址

以下我們使用 VS Code 的 GitHub Copilot，示範兩種做法。

  • 請 AI 生成 JSON、使用 GitHub MCP server 建立 gist，取得 API 網址。
