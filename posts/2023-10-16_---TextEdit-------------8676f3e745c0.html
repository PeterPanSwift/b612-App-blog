利用 TextEdit 回到過去瀏覽之前的程式

開發 iOS App 時我們會用 git 記錄程式在不同時間的版本，不過 git 只能查詢曾用 git commit 記錄的版本，因此如果程式不小心改壞了，想回到半小時前的版本，或是想瀏覽過去某個時間點沒有用 git 記錄的程式，git 是無能為力的。

彼得潘研究一下後，發現還是有方法瀏覽過去的程式，而且透過 Mac 內建的 TextEdit 即可實現。

在 Mac 上許多 App 會自動儲存文件在不同時間的版本，Xcode 也會，因此即使沒有用 git 記錄，我們在 Xcode 裡編輯的檔案早已包含過去不同時間的版本，包含 App 自動記錄和我們手動儲存的版本，之後用 TextEdit 打開檔案即可查詢過去的版本。

Cool，像時光機一樣酷炫的畫面出現在右邊，我們可以輕易瀏覽程式之前的版本，點選 Restore 還能讓程式回復到之前某個時間的版本。
