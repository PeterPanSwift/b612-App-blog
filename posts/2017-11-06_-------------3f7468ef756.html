設定某段時間之後執行程式

開發 iOS App 時，有時我們會想要程式在某段時間之後執行。比方在畫面上出現警告文字，然後幾秒鐘之後稍失。

有很多方法可以實現這樣的功能，以下我們介紹三種方法。

  • Task.sleep。
  • 透過 DispatchQueue 物件，呼叫它的 function asyncAfter 。
  • 呼叫 function perform(_:with:afterDelay:)。

ps: Timer 也可以做到，不過它通常用在重覆觸發的情境，比方每 10 秒執行一次。

Task.sleep

利用 Task { } 執行非同步的程式，在 { } 裡用 Task.sleep 暫停一段時間。.seconds(315360000) 表示暫停 315360000 秒，因此要等十年之後才會執行 print 印出訊息。( 315360000 = 365 * 10 * 24 * 60 * 60 )

Task {
    try await Task.sleep(for: .seconds(315360000))
    print("十年之後, 我們是朋友, 還可以問候")
}

ps: 此段程式其實難度很高，你要確保電腦一直開著，十年都不會遇到停電、地震等意外。

若想確保 Task { } 的程式在 main thread 執行，可在 { } 加上 @MainActor in。

Task { @MainActor in
    try await Task.sleep(for: .seconds(315360000))
    print("十年之後, 我們是朋友, 還可以問候")
}

透過 DispatchQueue 物件，呼叫它的 function asyncAfter

我們在參數 deadline 指定時間，它的型別是 DispatchTime。我們利用 .now() 取得現在時間，加上秒數 315360000，即可指定十年之後執行第二個參數 closure { } 裡的程式。

DispatchQueue.main.asyncAfter(deadline: .now() + 315360000) {
    print("十年之後, 我們是朋友, 還可以問候")
}

呼叫 function perform(_:with:afterDelay:)

若是繼承 NSObject 的物件，可呼叫 function perform(_:with:afterDelay:)。以下例子將在 315360000 秒後執行 function hi。

@objc func hi() {
    print("十年之後, 我們是朋友, 還可以問候")
}

func test() {      
    perform(#selector(hi), with: nil, afterDelay: 315360000)
}
