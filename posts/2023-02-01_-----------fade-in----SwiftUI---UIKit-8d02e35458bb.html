讓元件以淡入動畫出現(fade in) — SwiftUI & UIKit

開發 iOS App 時，我們時常會看到元件以淡入動畫出現的設計。以下我們分別介紹 SwiftUI & UIKit 實現的方法。

SwiftUI

  • withAnimation

呼叫 withAnimation，將文字從空字串變成有內容，即可實現淡入動畫效果。

struct ContentView: View {
    let array = ["原子", "分子", "電子", "中子", "質子"]
    @State private var habit = ""
    
    var body: some View {
        VStack {
            Text("學習 Swift 必須要有什麼習慣")
            Text("\(habit)")
                .frame(height: 100)
            Button("答案") {
                habit = ""
                withAnimation(.easeInOut(duration: 3)) {
                    habit = "\(array.randomElement()!)習慣"
                }
            }
            .buttonStyle(.borderedProminent)
            
            Spacer()
        }
    }
}

調整 opacity 也可實現淡入效果，例如以下例子。

struct ContentView: View {
    let array = ["原子", "分子", "電子", "中子", "質子"]
    @State private var opacity: Double = 0

    var body: some View {
        VStack {
            Text("學習 Swift 必須要有什麼習慣")
            Text("原子習慣")
                .frame(height: 100)
                .opacity(opacity)
            Button("答案") {
                opacity = 0
                withAnimation(.easeInOut(duration: 3)) {
                    opacity = 1
                }
            }
            .buttonStyle(.borderedProminent)
            
            Spacer()
        }
    }
}
  • transition

利用 if 控制元件是否顯示。呼叫 withAnimation 將 show 設為 true 時，將讓元件以淡入的方式顯示。

struct ContentView: View {
    @State private var show = false
    
    var body: some View {
        VStack {
            Text("學習 Swift 必須要有什麼習慣")
            if show {
                Text("原子習慣")
                    .frame(height: 100)
            } else {
                Color.clear
                    .frame(height: 100)
            }
            
            Button("答案") {
                withAnimation(.easeInOut(duration: 3)) {
                    show = true 
                }
            }
            .buttonStyle(.borderedProminent)
            
            Spacer()
        }
    }
}

剛剛的程式其實省略了 transition(.opacity)，加上它也是一樣的效果。

Text("原子習慣")
    .frame(height: 100)
    .transition(.opacity)

UIKit

  • UIViewPropertyAnimator

設定 alpha 從 0 到 1 實現淡入效果。

@IBAction func showText(_ sender: Any) {
    let array = ["原子", "分子", "電子", "中子", "質子"]
    label.alpha = 0
    label.text = "\(array.randomElement()!)習慣"
    UIViewPropertyAnimator.runningPropertyAnimator(withDuration: 3, delay: 0) {
        self.label.alpha = 1
    }
}
  • UIView.transition

利用 transitionCrossDissolve 實現淡入效果。值得注意的，我們利用 asyncAfter 延遲 0.01 秒再開始動畫。這是為了讓 label 先顯示空白的文字。若是沒有加上延遲，label 將從之前顯示的文字變成新文字，而不是文字從無到有的效果。

@IBAction func showText(_ sender: Any) {
    let array = ["原子", "分子", "電子", "中子", "質子"]
    label.text = ""
    DispatchQueue.main.asyncAfter(deadline: .now()+0.01) {
        UIView.transition(with: self.label, duration: 3, options: [.transitionCrossDissolve]) {
            self.label.text = array.randomElement()!
        }
    }
}
