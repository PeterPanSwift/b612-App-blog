使用 chevron SF Symbol 實現 App 常見的 V 形圖案

iOS App 時常會看到 V 形圖案的設計，比方左上的返回鍵和切換上一個下一個的按鈕。

  • 左上返回鍵的 V。
  • 切換上個月 / 下個月的 V。

感謝 Apple，這些跟 V 有關的圖案，現在都不用麻煩了，因為 SF Symbol 已幫我們準備好各種常用的 V。

在 Symbols library 搜尋 chevron 雖然找不到雪佛龍車子，但卻可以找到各式各樣的 V 形圖。

  • SwiftUI 範例。
struct ContentView: View {
    var body: some View {
        HStack {
            Button {
                
            } label: {
                Image(systemName: "chevron.backward")
            }
            
            Image("chess")
                .resizable()
                .scaledToFit()
            
            Button {
                
            } label: {
                Image(systemName: "chevron.forward")
            }
        }
        .font(.largeTitle)
        .buttonStyle(.borderedProminent)
        .padding()
        
    }
}
  • UIKit 範例。
