在 mac 上安裝 Android Studio

對彼得潘來說，Xcode 是世界上最棒的開發軟體，一天 24 小時應該要 1/3 給女朋友，1/3 給 Xcode。那剩下的 1/3 呢 ? 也許可以再切成十等分，分一點玩玩 Android Studio。

接下來彼得潘將展開在 mac 上開發 Android App 的第一步，安裝開發軟體 Android Studio !

連到 Android Studio 網站

Download Android Studio & App Tools - Android Developers
Create dynamic layouts with Jetpack Compose. Then preview your layouts on any screen size and inspect Compose…developer.android.com

點選 Download Android Studio

勾選 I have read and agree with the above terms and conditions，選擇 Mac with Intel chip 或 Mac with Apple chip

選擇 Mac with Apple chip 或 Mac with Intel chip 後開始下載。

雙擊剛剛下載的 android studio dmg 檔

雙擊後會出現以下畫面，請將左邊的 Android Studio 拖曳到右邊的 Applications。

雙擊 Applications 下的 Android Studio

點選 Open 同意啟動 Android Studio。

選擇 Do not import settings

選擇是否要加入之前安裝的 Android Studio 設定，由於是第一次安裝，因此我們選擇 Do not import settings。

跟隨 Android Studio Setup Wizard 的腳步設定 Android Studio

想做好人的話，選擇 Send usage statistics to Google。

選擇 Next。

Install Type 選擇 Standard。

選擇 Next。

同意 android-sdk-license。

同意 android-sdk-arm-dbt-license，點選 Finish。

Android Studio Setup Wizard 開始下載安裝相關的軟體。

經過一點點漫長的等待後，終於安裝完畢，點選 Finish。

Welcome to Android Studio

大功告成，出現 Welcome to Android Studio，可以開始開發 Android App 了。

更新 Android Studio

點選 Check for Updates。

點選 Update。

點選 Update and Restart。

Beta & Canary 版的 Android Studio

如果想測試最新的功能，也可從 Preview release 分頁下載安裝 Beta & Canary 版的 Android Studio。

ps: Canary 的意思是金絲雀，暗示著它是較不穩定的早期測試版。以下是 AI 關於金絲雀緣由的說明。

在早期的礦場中，礦工會帶著金絲雀進入礦坑，這不是因為他們愛聽鳥叫，而是因為金絲雀對有毒氣體（如一氧化碳或甲烷）特別敏感。金絲雀如果暴露在這些氣體中會比人類更早出現不適，甚至死亡，這樣礦工就能及時撤離，避免中毒。
