iOS 11 SDK 不再支援 FB, Twitter 的登入和分享

在 iOS 11 之前，Settings App 裡支援 FB, Twitter 等社群平台帳號的登入功能。使用者只要在 Settings App 登入社群平台的帳號，之後即可方便地在 各個 App 登入 FB, Twitter 和分享，不用重新輸入帳號密碼。

令人難過的，在 iOS 11，Apple 把社群帳號登入的功能從 Settings App 移除了 !

換句話說，原本我們利用 ACAccountStore 登入，SLComposeViewController 分享到社群平台的寫法也就失效了。

@available(iOS, introduced: 6.0, deprecated: 11.0)
public let SLServiceTypeTwitter: String
@available(iOS, introduced: 6.0, deprecated: 11.0)
public let SLServiceTypeFacebook: String
@available(iOS, introduced: 6.0, deprecated: 11.0)
public let SLServiceTypeSinaWeibo: String
@available(iOS, introduced: 6.0, deprecated: 11.0)
public let SLServiceTypeTencentWeibo: String
@available(iOS, introduced: 6.0, deprecated: 11.0)
public let SLServiceTypeLinkedIn: String
@available(iOS, introduced: 5.0, deprecated: 11.0, message: "Use Twitter SDK instead")
public let ACAccountTypeIdentifierTwitter: String
@available(iOS, introduced: 6.0, deprecated: 11.0, message: "Use Facebook SDK instead")
public let ACAccountTypeIdentifierFacebook: String
@available(iOS, introduced: 6.0, deprecated: 11.0, message: "Use Sina Weibo SDK instead")
public let ACAccountTypeIdentifierSinaWeibo: String
@available(iOS, introduced: 7.0, deprecated: 11.0, message: "Use Tencent Weibo SDK instead")
public let ACAccountTypeIdentifierTencentWeibo: String

如果只是單純想分享圖片文字到 FB, Twitter，還是可透過 UIActivityViewController 辦到，只要使用者有安裝登入 FB, Twitter App。

但是若想在 App 裡結合 FB, Twitter 的進階功能，請進一步搭配官方提供的 SDK。
