利用 cmd + f 在 storyboard 快速找東西

利用 cmd + f (find)我們可以針對某個檔案快速搜尋其中的內容，最常見的莫過於搜尋 swift 程式檔的內容。

不過其實它也可以幫我們搜尋 storyboard 檔的內容呢，比方以下例子:

1 搜尋 controller。

輸入 addroom，找到 class 是 AddRoomViewController 的物件。

2 搜尋 cell 。

輸入 ServiceCell 時，找到 class ServiceCell 的物件。

有時候運氣好我們會找到很多符合條件的對象，就像有時會同時有很多人喜歡我們一樣。如下圖所示，右邊 X 旁的 9 代表找到 9 個對象，點選 < > 可切換下一個 / 上一個。

3 搜尋 segue 。

輸入 Select 找到 id 為 Select Characteristic 的 segue。

4 搜尋 outlet & action。

輸入 segmentedcon 找到 outlet segmentedControl。
