從 Xcode 輸出和複製 preview screenshot — Xcode 15.3 新功能

最近 Xcode 15.3 推出了彼得潘夢寐以求的新功能 ! 現在我們可以從 Xcode 輸出和複製 preview screenshot，彼得潘在製作教材時終於能在修改程式時一鍵生成螢幕截圖。

以下我們用 Apple 的 SwiftUI 範例 Backyard Birds 示範。

輸出 preview 的畫面截圖

  • 方法 1: 從 Quick Actions 選單點選 Export Preview Screenshot。
  • 方法 2: 點選 Xcode menu 的 Editor > Canvas > Export Preview Screenshot。

成功輸出 preview 的畫面截圖。

複製 preview 的畫面截圖

從 Quick Actions 選單點選 Copy Preview Screenshot。

成功複製 preview 的畫面截圖，之後可以在任何地方按 cmd + v 貼上。

設定 preview screenshot 的快速鍵

我們也可以另外設定輸出 preview screenshot 的快速鍵。

打開 Xcode 的 Settings，切換到 Shortcuts 分頁後找到 Export Preview Screenshot 或 Copy Preview Screenshot。彼得潘較常用到 Copy Preview Screenshot，因此將它的快速鍵設為 cmd + p，同時為了避免衝突，也將原本採用 cmd + p 的 print 快速鍵取消。
