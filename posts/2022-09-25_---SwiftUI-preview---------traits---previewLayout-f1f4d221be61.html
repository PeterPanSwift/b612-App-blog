指定 SwiftUI preview 尺寸的方法 — traits & previewLayout

SwiftUI preview 預設會顯示整個 iPhone 尺寸的大小。因此以下的 MovieRow 雖然只是設計表格裡某部電影的畫面，preview 依然會傻傻地顯示整個 iPhone 大小。

import SwiftUI

struct MovieRow: View {
    var body: some View {
        HStack {
            Image(.cover)
                .resizable()
                .scaledToFill()
                .frame(width: 100, height: 100)
                .clipped()
            VStack(alignment: .leading) {
                Text("I Origins")
                    .font(.largeTitle)
                Text("Movie")
                    .foregroundStyle(.gray)
            }

            Spacer()

            Button {
                
            } label: {
                Image(systemName: "ellipsis")
            }
        }
    }
}

#Preview {
    MovieRow()
}

想要 preview 顯示指定的尺寸嗎 ? 其實是可以的，接下來我們將示範透過 traits 或 previewLayout 指定 preview 的大小，分成以下兩種 case:

  • 剛好容納內容的大小。
  • 指定寬高。

ps: 為了讓 preview 變成指定的大小，請點選 preview 下方的向上箭頭，切換到 Selectable 模式，因為預設的 live 模式只會顯示裝置機型的大小。

剛好容納內容的大小

在 #Preview 的參數 traits 傳入 .sizeThatFitsLayout，preview 的大小將不多也不少，剛好容納內容。

#Preview(traits: .sizeThatFitsLayout) {
    MovieRow()
}

指定寬高

在 #Preview 的參數 traits 傳入 .fixedLayout，從參數 width & height 傳入指定的寬高。

#Preview(traits: .fixedLayout(width: 393, height: 150)) {
    MovieRow()
}
