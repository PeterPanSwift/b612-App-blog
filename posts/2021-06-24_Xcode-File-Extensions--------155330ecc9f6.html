Xcode File Extensions 的顯示和隱藏

Xcode Project navigator 的檔案清單預設會隱藏附檔名。

如果想念附檔名，可以照以下步驟將它打開。

從 Quick Actions 點選 Settings。

找到 General 頁面的 File Extensions。

Hide All 將隱藏附檔名，切換成 Show All 將顯示附檔名。
