利用 Xcode 的 fold 折疊 { } 程式

開發 iOS App 時，我們主要都在 { } 裡撰寫程式。不過有時 { } 的程式十分龐大，造成程式碼不易閱讀。此時我們可以利用 Xcode 的 fold 功能將 { } 裡的程式暫時折疊，需要時再將它打開。

將 { } 裡的程式折疊有三種方法。

  • code folding ribbon。
  • 點選選單裡的 fold。
  • 按快速鍵。

code folding ribbon

Xcode 的 Code folding ribbon
我們常常不知不覺寫出長長的一大串程式，不過沒關係，Xcode 的 Code folding 功能可以幫我們將整段程式收起(折疊)，等想念它的時候再將它展開。medium.com

點選選單裡的 fold

滑鼠移到想折疊的 { 上，然後按住 cmd 鍵後從右鍵選單裡點選 Fold。

按快速鍵

游標停在想折疊的 { 上，然後按快速鍵 cmd + option + 向左的方向鍵。
