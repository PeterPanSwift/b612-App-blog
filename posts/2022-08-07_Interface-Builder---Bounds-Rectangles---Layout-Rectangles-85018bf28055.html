Interface Builder 的 Bounds Rectangles & Layout Rectangles

當我們在 Interface Builder 設計 App 畫面時，有時會不容易看出元件的佔據的空間，此時可另外打開 Bounds Rectangles 或 Layout Rectangles，讓元件顯示藍色的長方形邊框，方法如下:

Bounds Rectangles

勾選 Editor > Canvas > Bounds Rectangles。

如下圖所示，view & label 都出現了藍色的長方形邊框。別擔心，邊框只會在 Interface Builder 顯示， App 執行時是看不到的。

Layout Rectangles

勾選 Editor > Canvas > Layout Rectangles。

此時不只會顯示元件的長方形邊框，還會顯示其它影響畫面排版的邊框，比方 safe area，margin & 文字底部。

不想看到藍色邊框 ?

若是看到畫面上出現許多藍色邊框，通常是因為打開了 Bounds Rectangles 或 Layout Rectangles。

不想看到它們嗎? 很簡單，取消 Bounds Rectangles 或 Layout Rectangles 的勾選即可。
