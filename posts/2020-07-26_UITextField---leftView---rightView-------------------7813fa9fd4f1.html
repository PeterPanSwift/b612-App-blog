UITextField 的 leftView & rightView ~ 以搜尋放大鏡 & 密碼顯示開關為例

利用 UITextField 的 leftView & rightView，我們可以在 text field 的左邊和右邊加上想呈現的內容，比方下圖左邊的放大鏡 icon & 右邊的眼睛 button。

leftView 範例 - 搜尋輸入框的放大鏡

class SearchTextField: UITextField {
    
    required init?(coder: NSCoder) {
        super.init(coder: coder)
        leftViewMode = .always
        leftView = UIImageView(image: UIImage(systemName: "magnifyingglass"))
    }
}

假設我們在 storyboard 加入 SearchTextField，它產生時將觸發 init?(coder: NSCoder)，因此我們在 init?(coder: NSCoder) 裡設定它的內容。

我們將 UITextField 的 leftView property 設為顯示放大鏡 icon 的 image view，並將 leftViewMode 設為 always，讓 icon 永遠顯示。leftViewMode 的型別是 UITextField.ViewMode，有 never，whileEditing，unlessEditing，always 4 種選項，可參考以下連結的說明。

放大鏡完美地出現在 text field 的左邊，不過它有點太靠近邊邊了，能不能不要那麼近呢 ?

當然可以，我們可以自訂繼承 UITextField 的類別，然後覆寫 function leftViewRect(forBounds:)，透過它回傳的 CGRect 控制 leftView 的大小和位置。

以下例子裡，我們將 x 設為 10，height 設為 text field 的預設高度 34，因此 image view 不再跟 text field 的左邊界黏在一起，保持著 10 points 的安全距離。另外我們也將 imageView 的 contentMode 設為 scaleAspectFit，讓圖片維持比例縮放。

class SearchTextField: UITextField {
    
    required init?(coder: NSCoder) {
        super.init(coder: coder)
        leftViewMode = .always
        let imageView = UIImageView(image: UIImage(systemName: "magnifyingglass"))
        imageView.contentMode = .scaleAspectFit
        leftView = imageView
    }
    
    override func leftViewRect(forBounds bounds: CGRect) -> CGRect {
        let width = UIImage(systemName: "magnifyingglass")?.size.width ?? 0
        return CGRect(x: 10, y: 0, width: width, height: 34)
    }
}

rightView 範例 — 切換密碼是否顯示的眼睛 button

class PasswordTextField: UITextField {
    
    @objc func eyeButtonTap(button: UIButton) {
        button.isSelected.toggle()
        isSecureTextEntry = button.isSelected
    }
required init?(coder: NSCoder) {
        super.init(coder: coder)
        isSecureTextEntry = true
        rightViewMode = .always
        let button = UIButton(frame: CGRect(x: 0, y: 0, width: 34, height: 34))
        button.isSelected = true 
        button.setImage(UIImage(systemName: "eye.slash.fill"), for: .selected)
        button.setImage(UIImage(systemName: "eye.fill"), for: .normal)
        button.imageView?.contentMode = .scaleAspectFit
        rightView = button
        button.addTarget(self, action: #selector(eyeButtonTap(button:)), for: .touchUpInside)
    }
    
    override func rightViewRect(forBounds bounds: CGRect) -> CGRect {
        CGRect(x: bounds.width - 10 - 34, y: 0, width: 34, height: 34)
    }
}

眼睛完美地出現在 text field 的邊，不過它有點太靠近邊邊了，能不能不要那麼近呢 ?

當然可以，我們可以定義 UITextField 的 function rightViewRect(forBounds:)，透過它回傳的 CGRect 控制 rightView 的大小和位置。

override func rightViewRect(forBounds bounds: CGRect) -> CGRect {
        CGRect(x: bounds.width - 10 - 34, y: 0, width: 34, height: 34)
}

參數 bounds 的 width 為 text field 的寬度，眼睛 button 的寬度是 34，將 x 設為 bounds.width — 10 — 34 將讓眼睛跟 text field 的右邊界保持著 10 points 的安全距離。
