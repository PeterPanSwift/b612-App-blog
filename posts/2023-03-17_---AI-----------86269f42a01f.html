使用 AI 幫忙縮排程式和文字

最近發現使用 AI 幫忙縮排程式和文字滿方便的，以下彼得潘示範 3 個例子。

  • 縮排程式，並將程式裡的空行拿掉。
  • 縮排程式，符合 SwiftUI 一行一個 modifier 的排版。
  • 縮排以下文字，一行一個單字。

縮排程式，並將程式裡的換行拿掉

原本的程式如下，Xcode 的 Re-Indent 可以幫忙做基本的縮排，但是不能移除空行。

struct ContentView: View {
    
    var body: some View {
VStack {
    Image(systemName: "globe")
            
                .imageScale(.large)
                    .foregroundStyle(.tint)
            
            Text("Hello, world!")
        }
             .padding()
    }
    
        func 將進酒() {
        
        print("君不見，黃河之水天上來，奔流到海不復回。君不見，高堂明鏡悲白髮，朝如青絲暮成雪。人生得意須盡歡，莫使金樽空對月。天生我材必有用，千金散盡還復來。")
        
    }
    
    
}

只要打縮排程式，AI 就會聰明地縮排跟移除空行。

AI 縮排後的成果。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "globe")
                .imageScale(.large)
                .foregroundStyle(.tint)
            Text("Hello, world!")
        }
        .padding()
    }
    
    func 將進酒() {
        print("君不見，黃河之水天上來，奔流到海不復回。君不見，高堂明鏡悲白髮，朝如青絲暮成雪。人生得意須盡歡，莫使金樽空對月。天生我材必有用，千金散盡還復來。")
    }
}

縮排程式，符合 SwiftUI 一行一個 modifier 的排版

原本的程式如下，Xcode 的 Re-Indent 可以幫忙做基本的縮排，但是不能將 modifier 換行。

struct ContentView: View {
    var body: some View {
        Image(systemName: "star")
            .resizable()
            .scaledToFill().frame(width: 200, height: 200).clipShape(Circle())
            .background(LinearGradient(gradient: Gradient(colors: [.red, .blue]), startPoint: .leading, endPoint: .trailing).frame(width: 250, height: 250).rotationEffect(.degrees(10))
            ).shadow(color: .black, radius: 10)
        
    }
}

只要打縮排程式，AI 就會聰明地縮排跟符合 SwiftUI 一行一個 modifier 的排版。

AI 縮排後的成果。

struct ContentView: View {
    var body: some View {
        Image(systemName: "star")
            .resizable()
            .scaledToFill()
            .frame(width: 200, height: 200)
            .clipShape(Circle())
            .background(
                LinearGradient(
                    gradient: Gradient(colors: [.red, .blue]),
                    startPoint: .leading,
                    endPoint: .trailing
                )
                .frame(width: 250, height: 250)
                .rotationEffect(.degrees(10))
            )
            .shadow(color: .black, radius: 10)
    }
}
