顯示程式執行過程和變數內容的 PythonTutor

最近彼得潘發現有個滿適合初學者學程式和老師教學的工具 PythonTutor，它能以生動的圖表顯示程式執行的過程和變數的內容，而且支援 Python、JavaScript、C、C++、Java 五種常見的程式語言。

以下我們以 Python 為例，一步步認識程式學習的好幫手 PythonTutor。

連到 pythontutor.com

Learn Python, JavaScript, C, C++, and Java
This tool helps you learn Python, JavaScript, C, C++, and Java programming by visualizing code execution. You can use…pythontutor.com

點選 Python，進入 Python 程式的編輯和執行畫面

在框框裡輸入 python 程式

math_score = 90
english_score = 100
average_score = (math_score + english_score) / 2

點選 Visualize Execution 觀看程式執行的過程

執行程式和觀看程式執行的過程

綠色箭頭指到剛執行的程式，紅色箭頭指到下一行要執行的程式。從下圖可看到下一行要執行的程式是 math_score = 90。

點選 Next 會執行下一行要執行的程式，因此程式執行了 math_score = 90，從右邊看到 math_score 的內容變成 90，綠色箭頭指到 math_score = 90，紅色箭頭指到下一行要執行的 english_score = 100。

再次點選 Next，程式執行了 english_score = 100，從右邊看到 english_score 的內容變成 100，綠色箭頭指到 english_score = 100，紅色箭頭指到下一行要執行的 average_score = (math_score + english_score) / 2。

再次點選 Next，程式執行了 average_score = (math_score + english_score) / 2，從右邊看到 average_score 的內容變成 95.0，綠色箭頭指到 average_score = (math_score + english_score) / 2。此時 Next 已不能再點擊，因為它已執行完全部的程式。(ps: 點選 Prev 可回頭執行之前的程式)

修改程式

若想修改程式，可點選 Edit this code，進入程式的編輯畫面。
