Flutter 元件不能操作的方法

開發 Flutter App 時，如何讓元件變成不能操作呢 ?

很簡單，只要將元件操作觸發的 function 設為 null 即可，而且 null 還會讓元件變成淺色不能點選的樣子。以下我們介紹三種常見的應用。

  • 讓元件不能操作。
  • 讓元件依據條件決定是否可操作。
  • 讓畫面完全不能操作。

讓元件不能操作

以下程式將 ElevatedButton 的 onPressed 設為 null，Slider & Switch 的 onChanged 設為 null。不過 TextField 是其中的特例，不能打字的方法是將它的 enabled 設為 false。

class _MyHomePageState extends State<MyHomePage> {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('不能操作的元件')),
      body: Padding(
        padding: const EdgeInsets.all(8.0),
        child: Column(
          spacing: 10,
          children: [
            ElevatedButton(onPressed: null, child: Text('不能按的 button')),
            Slider(value: 0.5, onChanged: null),
            Switch(value: true, onChanged: null),
            TextField(
              enabled: false,
              decoration: InputDecoration(hintText: '不能打字的 TextField'),
            ),
          ],
        ),
      ),
    );
  }
}

讓元件依據條件決定是否可操作

使用 ? : 的語法依據條件決定元件操作觸發的程式是否為 null。

class _MyHomePageState extends State<MyHomePage> {
  bool _componentsEnabled = true;
  double _number = 0.5;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('讓元件依據條件決定是否可操作')),
      body: Padding(
        padding: const EdgeInsets.all(8.0),
        child: Column(
          spacing: 10,
          children: [
            Row(
              children: [
                Text(_componentsEnabled ? "啟用元件" : "停用元件"),
                Switch(
                  value: _componentsEnabled,
                  onChanged: (value) {
                    setState(() {
                      _componentsEnabled = value;
                    });
                  },
                ),
              ],
            ),
            ElevatedButton(
              onPressed: _componentsEnabled ? () {} : null,
              child: Text(_componentsEnabled ? '可以按的 button' : '不能按的 button'),
            ),
            Slider(
              value: _number,
              onChanged:
                  _componentsEnabled
                      ? (value) {
                        setState(() {
                          _number = value;
                        });
                      }
                      : null,
            ),

            TextField(
              enabled: _componentsEnabled,
              decoration: InputDecoration(
                hintText:
                    _componentsEnabled ? '可以打字的 TextField' : '不能打字的 TextField',
              ),
            ),
          ],
        ),
      ),
    );
  }
}

讓畫面完全不能操作

使用 AbsorbPointer 包住的 widget 將完全不能操作。

class _MyHomePageState extends State<MyHomePage> {
  bool _componentsEnabled = true;
  double _number = 0.5;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('不能操作的畫面')),
      body: AbsorbPointer(
        child: Padding(
          padding: const EdgeInsets.all(8.0),
          child: Column(
            spacing: 10,
            children: [
              Switch(
                value: _componentsEnabled,
                onChanged: (value) {
                  setState(() {
                    _componentsEnabled = value;
                  });
                },
              ),
              ElevatedButton(onPressed: () {}, child: Text('button')),
              Slider(
                value: _number,
                onChanged: (value) {
                  setState(() {
                    _number = value;
                  });
                },
              ),
              TextField(decoration: InputDecoration(hintText: 'TextField')),
            ],
          ),
        ),
      ),
    );
  }
}

為了讓使用者看出畫面不能操作，可以考慮再加一層半透明黑色遮罩。

class _MyHomePageState extends State<MyHomePage> {
  bool _componentsEnabled = true;
  double _number = 0.5;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('不能操作的畫面')),
      body: Stack(
        children: [
          // 原有內容
          AbsorbPointer(
            child: Padding(
              padding: const EdgeInsets.all(8.0),
              child: Column(
                children: [
                  Switch(
                    value: _componentsEnabled,
                    onChanged: (value) {
                      setState(() {
                        _componentsEnabled = value;
                      });
                    },
                  ),
                  ElevatedButton(onPressed: () {}, child: Text('button')),
                  Slider(
                    value: _number,
                    onChanged: (value) {
                      setState(() {
                        _number = value;
                      });
                    },
                  ),
                  TextField(decoration: InputDecoration(hintText: 'TextField')),
                ],
              ),
            ),
          ),
          Container(color: Colors.black.withAlpha(100)),
        ],
      ),
    );
  }
}
