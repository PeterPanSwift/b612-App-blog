利用 NSAttributedString 顯示豐富圖文內容的 rtf & rtfd 檔

rtf 是一種特別的文件格式，它可以顯示豐富的排版。如下圖所示，文件裡可設定不同的字型，顏色，而且還具有加入表格，點選連結顯示網頁等酷炫功能。

如果想顯示圖片，還有更厲害的 rtfd 格式，而且還能設定圖片點選後顯示網頁。

透過 NSAttributedString，我們將可在 iOS App 裡實現顯示 rtf & rtfd 檔的功能。

顯示 rtf 格式

class ViewController: UIViewController {
    @IBOutlet weak var textView: UITextView!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let url = Bundle.main.url(forResource: "peterpan", withExtension: "rtf")!
        if let content = try? NSAttributedString(url: url, options: [.documentType : NSAttributedString.DocumentType.rtf], documentAttributes: nil) {
            textView.attributedText = content
        }
    }
}

生成 NSAttributedString 時在參數 options 傳入 [.documentType : NSAttributedString.DocumentType.rtf]，讓 NSAttributedString 知道它要顯示 rtf 格式的內容。

結果:

rtf 的內容美美地呈現，而且點選 link 後還能顯示網頁。在此我們使用 UITextView，因此具有點選連結顯示網頁的功能。若是使用 UILabel 一樣能顯示 rtf 格式，但將失去點選連結的功能。

顯示 rtfd 格式

let url = Bundle.main.url(forResource: "peterpan", withExtension: "rtfd")!
if let content = try? NSAttributedString(url: url, options: [.documentType : NSAttributedString.DocumentType.rtfd], documentAttributes: nil) {
    textView.attributedText = content
}

參數 options 傳入的格式改成 NSAttributedString.DocumentType.rtfd。

結果:

帥氣的 peter 圖片成功出現在 text view，而且還可以點選圖片顯示網頁。
