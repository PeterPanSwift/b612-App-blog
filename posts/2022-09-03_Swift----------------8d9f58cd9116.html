Swift 程式輸入又快又準的幾個小技巧

最近家教時觀察程式初學者打程式，發現他們時常會一字一字輸入程式，因此時常會遇到以下四大問題:

  • 打很慢
  • 打錯字
  • 字少打
  • 字多打

要怎樣才能讓我們打程式又快又準呢？請記得以下輸入程式時重要的小技巧。

  • 熟記鍵盤上英文字母的位置。

程式主要以英文撰寫，因此熟記鍵盤上英文字母的位置還滿重要的。工程師通常都是眼睛看著螢幕打程式的。

  • 輸入關鍵字就好，然後再從選單選，不用輸入每個字。
  • 從 Xcode 的提示選單選擇後按 enter。
打程式時記得多從選單選，不在選單表示 Xcode 找不到它呀
初學者剛學程式時，常常會一字一字照著範例打程式。不過程式往往有著超長的英文跟標點符號，一字一字打超容易打錯的 !medium.com
  • 按 tab 跳到下一個參數的輸入框。

按 tab 鍵可跳到下一個參數的輸入框，不需動到滑鼠。

  • 使用 Xcode 的 predictive code completion。
Xcode 聰明的 AI 自動完成 predictive code completion — Xcode 16 新功能
從 Xcode 16 開始，Xcode 推出神奇的 AI 自動完成 predictive code completion，它能聰明地預測提示我們可能輸入的程式，以下我們一步步介紹和示範它的各種用法。medium.com
  • 輸入參數的各種方法。
Xcode 輸入參數的各種方法

  • 用型別加 ( ) 生成東西時，可輸入型別和參數名的關鍵字，然後再從選單選。

print(message)

  • 在參數 repeating 輸入 "，此時 Xcode 會自動幫忙補上另一個 "。
  • 輸入 I love U。
  • 按 tab 跳到下一個參數 count 的輸入框。
  • 輸入 100。
  • 按向右鍵或 ctrl + e，讓游標跑到 ) 的右邊。
  • 按 enter 鍵讓游標跑到下一行。
  • 輸入 pri，從選單選到 print 後按 enter。
  • 輸入 mes，從選單選到 message 後按 enter。

產生顏色 — UIKit 版本

import UIKit
class ViewController: UIViewController {
    @IBOutlet weak var redSlider: UISlider!
    @IBOutlet weak var alphaSlider: UISlider!
    @IBOutlet weak var blueSlider: UISlider!
    @IBOutlet weak var greenSlider: UISlider!
    @IBOutlet weak var imageView: UIImageView!
    
    override func viewDidLoad() {
        super.viewDidLoad()
    }
　　 @IBAction func changeColor(_ sender: Any) {
        imageView.backgroundColor = UIColor(red: CGFloat(redSlider.value), green: CGFloat(greenSlider.value), blue: CGFloat(blueSlider.value), alpha: CGFloat(alphaSlider.value))
    }
}

關鍵步驟

  • 輸入 uicred，從選單選到下圖的選項後按 enter。

因為關鍵字有 UIColor & red，因此我們輸入 uicred。

  • 在參數 red 的輸入框按 enter，讓 Xcode 幫忙輸入 CGFloat，然後輸入 ( 。
  • 輸入 red，從選單選擇 redSlider 後按 enter。

輸入 red 時 Xcode 會貼心地幫我們補上 )。

  • 輸入 .va，從選單選擇 value 後按 enter。
  • 按 tab 跳到下一個參數 green，以同樣的方法輸入參數 green，blue，alpha。
  • 如果準備在下一行繼續輸入程式，可按 ctrl + e 讓游標跑到 ) 的右邊。

或是按兩次向右鍵，因為有兩個 )。
