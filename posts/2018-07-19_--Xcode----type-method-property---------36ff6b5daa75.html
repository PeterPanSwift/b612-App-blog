在 Xcode 查詢 type、method、property 相關說明的方法

  • 方法 1: 按住 option 鍵再點擊，觸發 Show Quick Help Inspector。
  • 方法 2: Show Quick Help Inspector 的速鍵 cmd + ctrl + shift + ?。
  • 方法 3: 大力點擊放開，觸發 Show Quick Help Inspector。
  • 方法 4: 從 Quick Actions 點選 Show Quick Help for Selected Item。
  • 方法 5: 從右鍵選單點選 Show Quick Help。

接下來讓我們用以下的程式說明，假設我們想查詢 dateFormat 的說明。

import Foundation

let now = Date()
let dateFormatter = DateFormatter()
dateFormatter.dateFormat = "yyyy年MM月dd日"
let dateString = dateFormatter.string(from: now)

方法 1: 按住 option 鍵再點擊

  1. 游標移至目標 dateFormat，按住 option 鍵，出現問號圖示。

2. 點擊觸控板，顯示它的相關說明。

如果覺得字太小的，也可以點選右下方的 Open in Developer Documentation，打開 Documentation 視窗顯示更清楚的說明。

在 Documentation 視窗，我們還可以按 cmd + 或 cmd -將文字放大縮小。

從 Documentation 視窗上方的 bar，我們也可從 dateFormat 左邊的 DateFormatter 看出 dateFormat 是型別 DateFormatter 的 property。

方法 2: Show Quick Help Inspector 的快速鍵 cmd + ctrl + shift + ?

點選 dateFormat，然後按 Show Quick Help Inspector 的快速鍵 cmd + ctrl + shift + ?。

方法 3: 大力點擊放開，觸發 Show Quick Help Inspector

游標移至目標 dateFormat，大力點擊放開後觸發相關說明。

方法 4: 從 Quick Actions 點選 Show Quick Help for Selected Item

文字游標停在目標 dateFormat 上，按 cmd + shift + a 叫出 Quick Actions，輸入 he，點選 Show Quick Help for Selected Item。(快速鍵 ctrl + cmd + shift + ?)

當文字游標停在目標 dateFormat 時，按快速鍵 ctrl + cmd + shift + ? 也可觸發 Show Quick Help for Selected Item。

點選 Show Quick Help Inspector 則會打開右邊的 Inspector，在 Quick Help Inspector 顯示說明。(快速鍵 cmd + option + 3)

在 Quick Actions 輸入 sea，點選 Search Documentation for Selected Text 則可在 Document 視窗搜尋游標所在的目標 。(快速鍵 ctrl + option + cmd + /)

此時 Documentation 視窗將聰明地自動搜尋游標所在的目標，比方剛剛例子的 dateFormat，點選 dateFormat 即可顯示相關說明。

方法 5: 從右鍵選單點選 Show Quick Help

游標移至目標，從右鍵選單點選 Show Quick Help。(ps: 在 project 時右鍵選單才會有 Show Quick Help，playground 沒有。)
