設定中文版的 VS Code 介面

VS Code 支援多國語言，只要安裝對應的語言套件，即可讓 VS Code 的介面文字變成對應的語言。

以下彼得潘示範如何將 VS Code 變成中文版介面。

按下 cmd + shift + p 叫出 command palette，點選 Configure Display Language。

點選中文(繁體)。

點選 Restart 重新啟動 VS Code。

VS Code 成功變身中文介面 !

之後若想切換語言，再從 Configure Display Language 切換語言即可。
