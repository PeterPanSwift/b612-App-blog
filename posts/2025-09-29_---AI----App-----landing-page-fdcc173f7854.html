使用 AI 製作 App 精美的 landing page

最近彼得潘看到 Natalia 大大上架了超漂亮的咖啡 App Breve，而且搭配的 landing page 網站也超有質感的。

Breve
Breve is a beautifully designed iPhone and iPad app that brings the café experience to your kitchen with easy to follow…breve.coffee

彼得潘想到現在 AI 那麼厲害，應該也可以請 AI 製作 App 的 landing page，然後將它部署到 GitHub Pages。

以下彼得潘以上課的範例 AppB612 推理小說作家示範。

打開 VS Code，從 Command Palette 點選 Workspace: Add Folder to Workspace。

點選 New Folder。

輸入資料夾的名字。

點選 Add。

新增存放 App 截圖的資料夾 screenshot。

利用截圖外框合成工具幫 App 截圖加上美美的外框。

iPhone App 截圖外框合成工具
上傳或貼上 App 截圖或影片（支援 PNG、JPG、GIF、MP4、MOV）並選擇想要的 iPhone 外框，立即產出帶框成品。peterpanswift.github.io

將圖片加到資料夾 screenshot。

使用 Codex 製作 App 的 landing page，輸入以下 prompt。

製作 App "B612 推理小說作家"的 landing page

ps:

  • 若想製作 App 真正的 landing page，記得跟 AI 補充更多 App 的細節，AI 才不會胡說八道。
  • 若想在網頁加入精美的 Download on the App Store badge，可另外從 Apple 的官網下載再請 AI 加入。
Marketing Resources and Identity Guidelines — App Store — Apple Developer
Get information on using the App Store badge and Apple product images, as well as best practices for promoting apps on…developer.apple.com

AI 成功生成 landing page 的網頁 index.html。

在 Terminal 輸入以下指令啟動網頁，然後從瀏覽器連到 http://localhost:8000 檢視網頁。

python3 -m http.serve

覺得滿意後再請 AI 部署到 GitHub Pages。

使用 GitHub CLI 部署到 GitHub Pages，變成可以連線的網頁

ps: 如果沒有安裝 GitHub CLI，請先參考以下連結安裝。

安裝 GitHub CLI 和登入 GitHub 帳號
參考 GitHub CLI 網頁的教學安裝 GitHub CLI。medium.com

AI 成功部署，landing page 的網址如下。

B612 推理小說作家｜讓靈感化為懸疑巨作
B612 推理小說作家：專為懸疑與推理創作者打造的靈感引擎，從角色設定到情節轉折，一站式完成。peterpanswift.github.io

AI 成功打造看似專業的 landing page，不過因為我們沒有跟 AI 說明 App 的用途，AI 有點在天馬行空編故事了。若想製作真正的 landing page，記得跟 AI 補充更多 App 的細節。
