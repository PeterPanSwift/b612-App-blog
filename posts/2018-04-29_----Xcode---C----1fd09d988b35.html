如何在 Xcode 寫 C 語言

Xcode 主要是拿來寫 Swift & Objective-C 的，不過如果你想拿它來寫 C 或 C++，其實也可以呀。

1 建立新專案。

File > New > Project。

2 選擇 macOS 下的 Command Line Tool 樣版。

3 Language 欄位選 C。

4 看到 main.c 裡熟悉的 main function 了。

5 點選左上角的三角形按鈕執行。

6 執行後可在右下角的 Console 看到列印的結果。
