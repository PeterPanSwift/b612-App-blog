當我們在 App 畫面顯示文字時，通常是由左而右由上而下排列。不過有時我們希望文字能像有生命般靈活排列，比方最近七夕彼得潘想跟 Wendy 告白，想要喜歡你歌詞排列時特別空出愛心的形狀，比屋頂的天線還厲害，用文字排成愛你的形狀。

世上真有魔法能讓文字感應到彼得潘的愛心自動讓開嗎 ? 可以的，NSTextContainer 的 property exclusionPaths 型別是 [UIBezierPath]，文字排列時遇到 [UIBezierPath] 描繪的路徑將像膽小鬼般自動讓開，接下來就讓我們一步步實現七夕告白的愛心形狀歌詞吧。

exclusionPaths | Apple Developer Documentation

在 view controller 的 viewDidLoad 設定 text view

override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
               
        textView.text = """
            喜歡妳 給我妳的外衣 讓我像躲在妳身體裡 喜歡妳 借我妳的梳子 讓我用柔軟頭髮吻妳 喜歡妳 車窗上的霧氣 彷彿是妳的愛在呼吸 喜歡妳 那微笑的眼睛 連日落也看作唇印 我喜歡這樣跟著妳 隨便妳帶我到哪裡 妳的臉 慢慢貼近 明天也慢慢地慢慢清晰 我喜歡妳愛我的心 輕觸我每根手指感應 我知道 它在訴說著妳承諾言語
            """
        let path = createPath()
        textView.textContainer.exclusionPaths = [path]       
}

text view 的 property textContainer 型別是 NSTextContainer，NSTextContainer 的 property exclusionPaths 可控制文字的排列。 exclusionPaths 的型別是 [UIBezierPath]，文字排列時遇到 [UIBezierPath] 描述的路徑將像膽小鬼般自動讓開。由於 createPath 產生的 UIBezierPath 描繪著愛心的形狀，因此歌詞排列時將聰明地空出愛心的形狀，願天下有情人終成眷屬。

One More Thing，文繞圖效果

利用 exclusionPaths 讓文字自動讓路的效果，我們也可以做出常見的文繞圖效果，讓文字遇到圖片佔據的長方形區塊時自動讓開。

override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
               
        textView.text = """
            喜歡妳 給我妳的外衣 讓我像躲在妳身體裡 喜歡妳 借我妳的梳子 讓我用柔軟頭髮吻妳 喜歡妳 車窗上的霧氣 彷彿是妳的愛在呼吸 喜歡妳 那微笑的眼睛 連日落也看作唇印 我喜歡這樣跟著妳 隨便妳帶我到哪裡 妳的臉 慢慢貼近 明天也慢慢地慢慢清晰 我喜歡妳愛我的心 輕觸我每根手指感應 我知道 它在訴說著妳承諾言語
            """
        
        let cat1ImageView = UIImageView(image: UIImage(named: "cat1"))
        textView.addSubview(cat1ImageView)
        let cat2ImageView = UIImageView(image: UIImage(named: "cat2"))
        textView.addSubview(cat2ImageView)
        cat2ImageView.frame.origin = CGPoint(x: 100, y: 200)
        
        let cat1Path = UIBezierPath(rect: cat1ImageView.frame)
        let cat2Path = UIBezierPath(rect: cat2ImageView.frame)
        textView.textContainer.exclusionPaths = [cat1Path, cat2Path]
       
        
}
