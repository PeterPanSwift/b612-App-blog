AI 解題

聽說 AI 滿會寫程式的，而且彼得潘有訂閱 GPT 4，既然錢都花了，不如請他解看看吧。如果他解不出來，下個月就把他 fire。

ps: 為了讓 AI 寫出新手容易看懂的解答，彼得潘請 AI 先不要使用 map function 和 List Comprehension。

用 python 寫出可以上傳到程式解題系統的程式，符合以下條件。
1. 用 input 讀取輸入的文字，將文字變成數字的 array 時使用 for loop，不要用到 map 和 List Comprehension。
2. 不要定義 function。
3. 將程式寫在 code block 裡。

Content
Eva的家庭作業裏有很多數列填空練習。填空練習的要求是：已知數列的前四項，填出第五項。因 為已經知道這些數列只可能是等差或等比數列，她決定寫一個程式來完成這些練習。

Input
第一行是數列的數目t（0 <= t <= 20）。 以下每行均包含四個整數，表示數列的前四項。 約定數列的前五項均為不大於105的自然數，等比數列的比值也是自然數。

Output
對輸入的每個數列，輸出它的前五項。

Sample Input #1
2
1 2 3 4
1 2 4 8

Sample Output #1
1 2 3 4 5
1 2 4 8 16

GPT 4 的回答。

t = int(input())
sequences = []

for _ in range(t):
    seq = input().split()
    num_seq = []
    for s in seq:
        num_seq.append(int(s))
    sequences.append(num_seq)

for seq in sequences:
    if seq[1] - seq[0] == seq[2] - seq[1]:
        print(seq[0], seq[1], seq[2], seq[3], seq[3] + (seq[1] - seq[0]))
    elif seq[1] // seq[0] == seq[2] // seq[1]:
        print(seq[0], seq[1], seq[2], seq[3], seq[3] * (seq[1] // seq[0]))
