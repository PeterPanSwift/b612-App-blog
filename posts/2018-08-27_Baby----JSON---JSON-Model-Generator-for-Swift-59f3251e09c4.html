Baby 也懂 JSON 的 JSON Model Generator for Swift

對 iOS App 開發新手來說，串接 API 和解析 JSON 是件有點難度的事情，就算完全理解原理，實際開發時因為需要將大量的 JSON 欄位轉換成程式，滿容易出現大小寫不合或打錯字的問題。

不過江湖上有個 Baby 很狂妄，他說解析 JSON 很簡單，只要給他 JSON 字串，他就能幫我們生成 swift 的 Model 檔。他就是 App Store 上的 CuteBaby !

安裝

1 打開 terminal，切換到剛剛下載的 Baby 專案資料夾。

2 輸入 bash install.sh

執行

輸入 baby -i JSONFilePath
比方當 test.json 檔的路徑在桌面時 ，指令如下:

baby -i ~/Desktop/test.json

輸入 baby -i ~/Desktop/test.json 後產生的結果:

複製結果
baby -i ~/Desktop/test.json | pbcopy

之後用 cmd + v 即可將結果貼到 swift 檔。

將結果生成 swift 檔
baby -i ~/Desktop/test.json >  MyModel.swift
查詢 baby 有哪些功能
baby -h
指定產生的 Model 型別名字。

預設產生的 model 型別為 MyModel，可用參數 — model-name 改名。

baby -i ~/Desktop/test.json --model-name StoreItems

改名並產生 swift 檔

baby -i ~/Desktop/test.json --model-name StoreItems > StoreItems.swift
產生 Codable 的 model 型別

參數 — codable

baby -i ~/Desktop/test.json --model-name StoreItems --codable > StoreItems.swift
產生的 Model 型別以 struct 還是 class 定義

預設以 struct 定義，可用參數 — model-type 設定。

baby -i ~/Desktop/test.json --model-type class
