處理綁定 optional 選項的 SwiftUI Picker

開發 iOS App 時當我們想從多個選項做選擇時，通常會用 Picker 實現，例如以下程式用 Picker 選擇小王子的好朋友。

import SwiftUI

enum LittlePrinceAnimal: String, CaseIterable, Identifiable {
    case fox
    case snake
    case sheep
    case elephant
    var id: Self { self }
}

struct ContentView: View {
    @State private var selectedFriend: LittlePrinceAnimal = .fox
    
    var body: some View {
        NavigationStack {
            VStack {
                Image(selectedFriend.rawValue)
                    .resizable()
                    .scaledToFit()
                Picker("小王子的好朋友", selection: $selectedFriend) {
                    ForEach(LittlePrinceAnimal.allCases) { animal in
                        Text(animal.rawValue)
                    }
                }
                .pickerStyle(.segmented)
                .padding()
            }
            .navigationTitle("小王子的好朋友")
        }
    }
}
SwiftUI 選擇項目的 Picker & 我要選李白
SwiftUI 的 picker 方便我們選擇項目，而且它不只能呈現文字，想以客製的圖文顯示也不是問題，甚至還能變身成 Segmented Control。接下來就讓我們好好認識它跟實現我要選李白的功能吧。medium.com

然而有時我們會希望選項是 optional，成為非必填的項目，比方小王子可能覺得寧缺勿濫，沒有朋友也能一個人堅強地活下去，因此 selectedFriend 可以是 nil。

struct ContentView: View {
    @State private var selectedFriend: LittlePrinceAnimal?
    
    var body: some View {
        NavigationStack {
            VStack {
                Image(selectedFriend?.rawValue ?? "chooseAnimal")
                    .resizable()
                    .scaledToFit()
                Picker("小王子的好朋友", selection: $selectedFriend) {
                    ForEach(LittlePrinceAnimal.allCases) { animal in
                        Text(animal.rawValue)
                    }
                }
                .pickerStyle(.segmented)
                .padding()
            }
            .navigationTitle("小王子的好朋友")
        }
    }
}

當 Picker 綁定的 selection 是 optional 時，此時奇怪的事發生了。不管我們點選哪個選項，Picker 都毫無反應。

這是因為當 Picker 綁定的項目是 optional 時，選項的 tag 值型別也必須是 optional，所以我們要另外對每個選項呼叫 tag modifier 設定 Optional 型別的 tag 值。

tag(_:) | Apple Developer Documentation
Sets the unique tag value of this view.developer.apple.com

SwiftUI 官方文件也貼心地提到 Picker 處理 optional 的方法。

Use an optional value for the selection input parameter. For that to work, you need to explicitly cast the tag modifier’s input as Optional to match. For an example of this, see tag(_:).

Picker | Apple Developer Documentation
A control for selecting from a set of mutually exclusive values.developer.apple.com

以下我們示範三種將 tag 值設為 optional 的方法。

  1. 利用 as 轉型。

2. 設定 Optional.some & Optional.none。

3. 用 init(nilLiteral:) & init(_:) 產生設定 optional 。

  • 利用 as 轉型。
struct ContentView: View {
    @State private var selectedFriend: LittlePrinceAnimal?
    
    var body: some View {
        NavigationStack {
            VStack {
                Image(selectedFriend?.rawValue ?? "chooseAnimal")
                    .resizable()
                    .scaledToFit()
                Picker("小王子的好朋友", selection: $selectedFriend) {
                    Text("None")
                        .tag(nil as LittlePrinceAnimal?)
                    ForEach(LittlePrinceAnimal.allCases) { animal in
                        Text(animal.rawValue)
                            .tag(animal)
                    }
                }
                .pickerStyle(.segmented)
                .padding()
            }
            .navigationTitle("小王子的好朋友")
        }
    }
}
  • 設定 Optional.some & Optional.none。
struct ContentView: View {
    @State private var selectedFriend: LittlePrinceAnimal?
    
    var body: some View {
        NavigationStack {
            VStack {
                Image(selectedFriend?.rawValue ?? "chooseAnimal")
                    .resizable()
                    .scaledToFit()
                Picker("小王子的好朋友", selection: $selectedFriend) {
                    Text("None")
                        .tag(Optional<LittlePrinceAnimal>.none)
                    ForEach(LittlePrinceAnimal.allCases) { animal in
                        Text(animal.rawValue)
                            .tag(Optional.some(animal))
                    }
                }
                .pickerStyle(.segmented)
                .padding()
            }
            .navigationTitle("小王子的好朋友")
        }
    }
}
  • 用 init(nilLiteral:) & init(_:) 產生設定 optional 。
struct ContentView: View {
    @State private var selectedFriend: LittlePrinceAnimal?
    
    var body: some View {
        NavigationStack {
            VStack {
                Image(selectedFriend?.rawValue ?? "chooseAnimal")
                    .resizable()
                    .scaledToFit()
                Picker("小王子的好朋友", selection: $selectedFriend) {
                    Text("None")
                        .tag(Optional<LittlePrinceAnimal>(nilLiteral: ()))
                    ForEach(LittlePrinceAnimal.allCases) { animal in
                        Text(animal.rawValue)
                            .tag(Optional(animal))
                    }
                }
                .pickerStyle(.segmented)
                .padding()
            }
            .navigationTitle("小王子的好朋友")
        }
    }
}
