參考模仿學長姐的 iOS App 作業，感謝同學們精心繪製的圖片

想要學會開發 iOS App，就像華仔唱的，練習是很重要的。不過彼得潘承認，我們時常會偷懶，尤其當自己沒有美術天份，只能做出醜醜的 App 時，更是興趣缺缺。其實不只寫作業如此，之前彼得潘在公司上班開發 App 時，如果美術設計的畫面很漂亮，自己也會更有動力，迫不及待想將 App 完成。當看到最後酷炫的成品時，自己也會超有成就感 !

經過了幾屆 彼得潘班 和 文組生班 後，不少學生由於身上流著美術血液，發表的作業成品都超棒的，App 裡還有自己精心繪製的圖片。經過和畢業學生討論後，感謝他們願意提供自己的作業讓學弟妹模仿練習，到時候學生寫作業時，可參考以下方式練習作業 :

  1. 下載學長姐在 GitHub 上的作業，採用專案裡的圖片模仿練習，實作一模一樣的 App，或是發揮創意改編。
  2. App 作業完成後， 發表在 Medium 時，要說明參考學長姐 xxx 的作業，並附上對方作業的 Medium 網址。

以下彼得潘舉 Yaya 同學的擇偶條件 App 為例，說明如何取得專案裡的圖片:

  1. 參考 Yaya 同學擇偶條件 App 作業的 Medium 文章。
鴨鴨 The One App
終於要把APP介面跟swift code結合在一起惹medium.com

2. 從文章裡找到作業的 GitHub 連結，前往作業的 GitHub 頁面。

nthuyaya/161111yayatheone
161111yayatheone - @IBOutlet & @IBActiongithub.com

3. 點選綠色的 Clone or download 按鈕後，點選 Download Zip 下載。

專案的 Zip 下載後，解壓縮後將變成專案的資料夾。

4. 將同學專案中 Assets.xcassets 裡的圖片複製到自己的專案。

(1）同時打開自己和同學的專案。(雙擊專案資料夾裡的 xcodeproj 檔可開啟專案。)

(2) 兩個專案都點選 Assets.xcassets，切換到圖片頁面。

(3) 全選同學專案 Assets.xcassets 裡的所有圖片 ( cmd + a ) ，然後拖曳到自己專案的 Assets.xcassets 裡。
