protocol 的條件限制


只有 class 能遵從 protocol

protocol Idol: AnyObject {
    func sing()
}

只有繼承某某類別的能遵從 protocol

class Animal {
    
}
protocol Idol: Animal {
    func sing()
}

另一種寫法。

protocol Idol where Self: Animal {
    func sing()
}
