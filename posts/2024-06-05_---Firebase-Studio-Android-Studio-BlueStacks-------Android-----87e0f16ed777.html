使用 Firebase Studio、Android Studio、BlueStacks 全螢幕顯示 Android 模擬器

開發 Flutter Android App 時，我們可以選擇用 Firebase Studio、VS Code、Android Studio 開發。不過如果想用全螢幕展示 Android App，比方在重要的期末發表會 demo，記得要用 Firebase Studio 或 Android Studio 執行 Android App，因為 VS Code 將如下圖所示，Android 模擬器無法單獨佔滿整個螢幕。

使用 Firebase Studio

在顯示模擬器的空白畫布上按出右鍵選單，點選 Open Frame in New Window，觸發瀏覽器新開視窗顯示 Android 模擬器

點選視窗左上的放大按鈕將視窗放大到全螢幕。

使用 Android Studio

  • 方法 1: 點選下圖紅色框框的 Window 按鈕。
  • 方法 2: 從上方裝置名字的右鍵選單點選 View Mode > Window。

點選左上的放大按鈕將視窗放大到全螢幕。
