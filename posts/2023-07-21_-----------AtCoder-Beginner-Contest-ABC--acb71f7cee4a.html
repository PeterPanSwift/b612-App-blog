適合程式初學者練功的 AtCoder Beginner Contest(ABC)

我們小時候學英文時，都是從簡單的 ABC 開始學起。那麼在學習程式解題時，有沒有類似 ABC 這樣適合新手的題目呢 ?

有的，它的簡寫剛好也是 ABC，全名是 AtCoder Beginner Contest。AtCoder 是日本的網站，定期舉辦不同程度的比賽，其中 AtCoder Beginner Contest 是專為新手推出的比賽，上面的題目較為簡單，比較不會嚇跑新手，很適合拿來提升自信心。

AtCoder
AtCoder is a programming contest site for anyone from beginners to experts. We hold weekly programming contests online.atcoder.jp

有興趣的朋友可參考網站參加最近舉辦的比賽。只想練習程式，不想參加比賽的話，也可參考之前比賽的題目練習。

以下彼得潘簡單介紹練習 ABC 題目的重點。

  • 查詢 ABC 比賽的題目。
  • 題目的點數(Point Status)。
  • 查詢點數 100 的題目。
  • 點數 100 的題目 — Order Something Else。
  • 上傳程式到 AtCoder，請他測試是否正確。

我們的目標是 ABC 比賽的題目，因此請點選下圖的 ABC。

此時網頁將由新到舊顯示歷屆的 ABC 比賽資訊。

題目的點數(Point Status)

ABC 比賽的題目有 8 題，難度由 A 到 H 慢慢增加。

每個題目的右下方有個數字，代表題目的點數(points)，數字愈高題目愈難。點數 300 以下是較簡單的題目，新手建議先嘗試 A，B，C 的題目，剛好對應點數 100，200 和 300。

查詢點數 100 的題目

想要柿子挑軟的吃，先練習點數 100 的題目嗎 ? 沒問題，請點選上方的 List 切換成列表模式。

從 Point Status 可選擇感興趣的點數。

選擇點數 100 後，下方的 Problem List 將顯示所有點數 100 的題目。

點數 100 的題目 — Order Something Else

點數 100 的題目通常不會太難，以下為其中一個例子，2023–07–15 ABC 比賽的第一題，Order Something Else。

上傳程式到 AtCoder，請他測試是否正確

AtCoder 支援各種程式語言，因此我們可以用 Swift，Python 等程式語言解題，然後將程式上傳到 AtCoder，請它測試程式是否正確。

AtCoder 是大好人，他很願意免費測試程式，不過他只幫認識的人測試，因此我們要註冊帳號登入後才能上傳程式。

請先點選右上的 Sign Up 註冊帳號。

登入後問題頁面的下方將出現程式的輸入框。我們可從 Language 選擇語言，在 Source Code 輸入程式，或點選 Open File 上傳程式，最後再點選 Submit 上傳到 AtCoder。

除了從問題頁面上傳，我們也可以切換到 Submit 分頁上傳程式。

上傳後會跳到 Results 頁面顯示結果，下圖為彼得潘上傳的結果，從 Status 欄位可觀看結果，綠色的 AC 表示正確，黃色的 WA 表示程式有問題。
