在 SwiftUI alert 的 actions 顯示 TextField、SecureField、Link

SwiftUI alert 的 actions 能顯示的元件有限，大部分的人都知道它能顯示 Button & Text，那還有其它元件嗎 ?

有的，從 iOS 16 開始，它還可以顯示 TextField、SecureField 和 Link 呢。

顯示 TextField ＆ SecureField

struct ContentView: View {
    @State private var isPresented = false
    @State private var account = ""
    @State private var password = ""

    var body: some View {
        Button("登入") {
            isPresented = true
        }
        .buttonStyle(.borderedProminent)
        .alert("登入", isPresented: $isPresented) {
            TextField("Account", text: $account)
            SecureField("Password", text: $password)
            Button(role: .confirm) {
                
            }
            Button(role: .cancel) {
                
            }
        }
    }
}

顯示 Link

點選後打開 Safari 顯示網頁。

struct ContentView: View {
    @State private var isPresented = false
   
    var body: some View {
        Button("Go") {
            isPresented = true
        }
        .buttonStyle(.borderedProminent)
        .alert("彼得潘的相關連結", isPresented: $isPresented) {
            Link("Blog", destination: URL(string: "https://medium.com/@apppeterpan")!)
            Link("Website", destination: URL(string: "https://apppeterpan.mystrikingly.com")!)
        }
    }
}
