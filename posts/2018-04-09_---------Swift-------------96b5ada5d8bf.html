就像年輕不要留白，Swift 程式裡的名字不要有空白

寫 Swift 程式時，我們會使用各式各樣的名字，比方變數名，常數名，function 名，型別名等。

不管你是自己發明名字，或是使用別人發明的名字，有一點請記得，在 Swift 的世界，就像年輕不要留白，名字也是不要有空白，因此以下 function 名字將產生可怕的紅色錯誤。

func listen to music() {
   print("聽音樂")
}

應該要把空白拿掉，變成

func listenToMusic() {
   print("聽音樂")
}
