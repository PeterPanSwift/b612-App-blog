利用 Search Documentation for Selected Text 快速查詢完整說明

開發 iOS App 時，我們時常需要查詢 type, method 或 property 的相關說明，不過如果想快速查到最詳細的說明，建議採用以下連結提到的方法一，利用 Search Documentation for Selected Text 打開 Documentation 視窗。(快速鍵 cmd + option + ctrl + / ）

在 Xcode 查詢 type, method , property 相關說明的五個方法
方法 1medium.com

以在 Xcode 的 playground 撰寫 swift 程式為例，它的說明會比較不完整，比方下圖的 addSubview 很神秘，按住 option 點擊後，它只肯透露它的 function 宣告。

相反的，利用 Search Documentation for Selected Text 打開 Documentation 視窗，然後再點選 UIView 的 addSubview，將能看到最完整的說明。
