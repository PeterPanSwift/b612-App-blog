影響升級 beta 版 macOS 的 Lockdown mode(封閉模式)

最近有同學想升級 beta 版的 macOS，但是他打開 System Settings 後卻找不到 Beta Updates 的選項。

升級安裝 macOS beta
從 macOS 13.4 Ventura 開始，升級 beta 版的 macOS 變得更容易了，我們可以直接從 System Settings App 升級 beta 版的作業系統。medium.com

彼得潘研究一下後，發現原來是 Lockdown Mode(封閉模式) 造成的。當我們將 Lockdown Mode 打開時，Mac 會變得更安全，但是也多了一些限制，其中一個限制正是無法升級 beta 版的 macOS。

因此除非有特別需求，建議平常還是關閉 Lockdown Mode，才不會造成日常使用上的不便。

關於「封閉模式」 - Apple 支援 (台灣)
「封閉模式」可幫助保護裝置防範極其罕見且高度精密的網路攻擊。support.apple.com
