App Icon 不見的紅色錯誤

App Icon 是 App 的臉，使用者在 App Store 第一眼看到的就是 App Icon，如果 icon 不好看，大概這輩子就不會被下載了。

因為 Icon 是如此地重要，當你不小心把它刪除時，Xcode 將馬上產生以下的錯誤訊息，不讓我們生出羞於見人的 App。

None of the input catalogs contained a matching stickers icon set, app icon set, or icon stack named  "AppIcon".

解決的方法很簡單，只要重新生成 App Icon 即可，步驟如下:

  1. 點選 Project navigator 裡的 Assets。

2. 點選左下的 +，點選 iOS > New iOS App Icon。

AppIcon 順利生成，當 Xcode 重新編譯後，討人厭的紅色錯誤將煙消雲散，不再出現 !
