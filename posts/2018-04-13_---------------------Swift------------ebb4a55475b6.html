同樣的符號，在不同的地方有不同的意義 — Swift 程式初學者的常見問題

在我們撰寫程式時，常會用到一些有著特別意義的符號，比方冒號，驚嘆號，問號等。不過請注意，這些符號在不同的地方使用時，將有不同的意義和作用，就像同一個英文單字，在不同的地方，會有不同的意義，比方以下兩個 miss 分別代表不同的意思。

代表想念的 miss。
I know how you miss your ex-girlfriend.
代表沒有打中的 miss。
The hunter fired at the deer but missed it.

在 Swift 裡也常看到類似的情況，比方以下兩個冒號的例子:

例子 1
var cuteBaby: String = "peter"
例子 2
class Baby {
   var age = 1
}

class MagicBaby: Baby {
   var magic = "魔音穿腦"
}

在例子 1，var cuteBaby: String 的冒號代表變數 cuteBaby 的型別是字串，只能儲存字串，不能儲存字串以外的東西。

在例子 2，class MagicBaby: Baby 的冒號代表我們新定義的類別 MagicBaby 繼承 Baby，將得到 Baby 裡定義的東西，比方它的 age。
