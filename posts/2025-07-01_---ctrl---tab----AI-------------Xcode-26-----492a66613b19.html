利用 ctrl + tab 輸入 AI 提示的下個程式片段 — Xcode 26 新功能

從 Xcode 26 開始，當我們不想對 AI 提示的程式照單全收時，可以利用 ctrl + tab 放慢腳步，只輸入下個提示的程式片段。

舉個例子，以下定義 Baby 型別時，AI 和彼得潘心有靈犀，猜出我想宣告 height。不過 AI 猜錯型別了，我想要的是 Double，不是 Int。

此時我們可按住 ctrl 選取 ght。

接著按下 tab，完成 height 的輸入。

然後再補上型別 : Double。

struct Baby {
    let height: Double
}

當我們想在 ( ) 裡添加傳入的參數時，在某些情況 ctrl + tab 也幫助很大，例如以下 VStack 的例子。

struct ContentView: View {
    var body: some View {
        VStack {
            Text("我最喜歡的棒球隊")
            Text("統一7-ELEVEn獅")
                .font(.largeTitle)
        }
    }
}

我們想讓兩行文字對齊左邊，因此要在 VStack 添加參數 alignment。在舊版的 Xcode，我們必須辛苦地一字一字輸入 alignment。

有了 ctrl + tab，我們可以用以下方式輸入。

在 VStack 後輸入 (，然後按住 ctrl 選取 alignment。

按下 tab 輸入 alignment，然後輸入 :。

按下 tab 輸入 AI 提示的 .leading)。
