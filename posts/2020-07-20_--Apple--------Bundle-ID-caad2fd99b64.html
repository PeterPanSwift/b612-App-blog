從 Apple 開發網站產生 Bundle ID

連到 Apple 開發網站

https://developer.apple.com/account

點選 Certificates, Identifiers & Profiles

點選側邊欄的 Identifiers

點選 + 新增 ID

有許多不同的 ID 可以新增，現在我們想要的是 App 的 Bundle ID，因此請選擇 App IDs 後按 Continue。

選擇 App 後按 Continue

有 App & App Clip 兩種 type，在此我們選擇 App。

輸入 App 的 Description & Bundle ID 後點選 Continue

點選 Register 建立 App 的 Bundle ID
