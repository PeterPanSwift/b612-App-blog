Xcode 12 調整 navigator 內容大小的 navigator size

從 Xcode 12 開始，不只程式碼可以調整大小，左邊 navigator 區塊的文字 icon 大小也能改變，方法如下:

調整 Xcode navigator 內容大小的 navigator size

Xcode 不只程式碼可以調整大小，左邊 navigator 區塊的文字 icon 大小也能改變，方法如下。

點選 Xcode > Settings

切換到 General 頁面，從 Navigator Size 可調整 navigator 內容的大小

預設為 System，可另外改為 Small，Medium & Large。

比方選擇 Large 後，Navigator 裡的 icon & 文字都變大了。

若是採用 System，navigator 的內容大小將遵從 Systems App 的設定。

打開 Mac 的 Settings App，搜尋 icon size 後點選 Sidebar icons，從 Sidebar icon Size 調整大小。
