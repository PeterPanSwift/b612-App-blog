方便讀取 bundle 資源的 #bundle

開發 iOS App 時，我們時常會從 App、Swift package 或 framework 讀取 bundle 裡的資源，像是圖片、音樂、影片、翻譯的文字等。

比方在 App 裡我們常透過 Bundle.main 讀取 App 本身資料夾的資料，例如以下例子。

let url = Bundle.main.url(forResource: "powerup", withExtension: "mp3")

在 Swift package 則透過 Bundle.module 讀取 package 的資料，例如以下例子。

let url = Bundle.module.url(forResource: "powerup", withExtension: "mp3")

從 Xcode 26 開始，我們可以更簡單地存取 bundle 裡的資源，透過新發明的 macro #bundle。

不管是在 App 、Swift package 或 framework，我們都可利用 #bundle 存取資料。

let url = #bundle.url(forResource: "powerup", withExtension: "mp3")

在 App 裡以上程式將讀取 App 裡的 powerup.mp3，在 Swift package 將讀取 package 裡的 powerup.mp3。

以下是使用 #bundle 的其它例子。

  • 文字翻譯。
String(localized:"Game Over.",
       bundle: #bundle,
       comment: "Text for game over banner.")
  • 載入 xib。
class ViewController: UIViewController {

    override func viewDidLoad() {
        super.viewDidLoad()
        if let array = #bundle.loadNibNamed("LoadingView", owner: nil),
            let loadingView = array.first as? UIView {
            loadingView.frame = CGRect(x: 100, y: 100, width: 300   , height: 300)
            view.addSubview(loadingView)
        }
    }
}
