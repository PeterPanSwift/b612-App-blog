let name = "Hello Kitty"

let birthplace = "英國倫敦"

let intro = "大家好！我的名字是\(name)，本名是\(realName)。我出生於\(birthday)，地點在\(birthplace)。我的身高是\(heightInApples)個蘋果高，體重是\(weightInApples)個蘋果重，血型是\(bloodType)型。我喜歡\(hobbies)。我的家人有\(family)。很高興認識大家！"

Dart 版

prompt: 請用 Dart 的 string interpolation 介紹 Hello Kitty。

void main() {
  String name = 'Hello Kitty';
  String realName = '凱蒂·懷特';
  String birthday = '1974年11月1日';
  String birthplace = '英國倫敦';
  int heightInApples = 5;
  int weightInApples = 3;
  String bloodType = 'A';
  String hobbies = '烘焙餅乾、音樂、閱讀、吃媽媽的蘋果派';
  String family = '爸爸喬治、媽媽瑪麗、雙胞胎妹妹米咪';

  String intro =
      '大家好！我的名字是$name，本名是$realName。我出生於$birthday，地點在$birthplace。我的身高是$heightInApples個蘋果高，體重是$weightInApples個蘋果重，血型是$bloodType型。我喜歡$hobbies。我的家人有$family。很高興認識大家！';
  print(intro);
}

intro 的結果。

'大家好！我的名字是Hello Kitty，本名是凱蒂·懷特。我出生於1974年11月1日，地點在英國倫敦。我的身高是5個蘋果高，體重是3個蘋果重，血型是A型。我喜歡烘焙餅乾、音樂、閱讀、吃媽媽的蘋果派。我的家人有爸爸喬治、媽媽瑪麗、雙胞胎妹妹米咪。很高興認識大家！'
