iPad Swift Playgrounds 開發 iOS App 初體驗

iPad 有著大大的螢幕和接近電腦的效能，所以很多人常夢想有天能在 iPad 上帥氣地開發 App，可惜目前 Xcode 只能在 Mac 上執行。

現在夢想終於可以實現了! 從 Swift Playgrounds 4 開始，我們可以在 iPad 上開發並上架 iOS App。(ps: Swift Playgrounds 也有 Mac 版，想要快速打字的可以使用 Mac 版)

建立 App 專案

點選 New Playground。

點選 App。

ps: Mac 版可直接點選 Create New App。

專案建立後將自動開啟專案，接下來讓我們開始在 iPad 上寫 code 吧。

輸入程式

  • 方便的自動完成
  • 上一步 / 下一步 (undo / redo)

鍵盤左上的向左箭頭是上一步，向右箭頭是下一步。

  • 點選程式後的相關選單

比方 Copy，Paste & Help。

點選 Help 可查詢說明。

  • 從 Library 加東西

點選右上的 + 打開 Library，從 Library 視窗加入 UI 元件，modifier，圖片 & 顏色。

加入 UI 元件。

加入 modifier。

加入 SF Symbol。

加入顏色。

預覽 & 操作 App 畫面(preview)

右邊的區塊為 App 的 preview 畫面，它不只是呈現 App 的樣子，還能直接點選操作，就像一個真正的 App。

點選紅色框框的按鈕可打開 / 關閉 preview。

如果覺得 preview 畫面太小，也可點選紅色框框的按鈕，讓 preview 變成全螢幕。

點選 App Preview 將出現選單，有以下三個選項:

  • 重新啟動 preview(Restart Preview)。
  • 暫停 preview(Pause Preview)。
  • 刪除 preview data & 重新啟動(Delete Preview Data and Restart)。

當 App 有不只一個畫面時，若左邊區塊顯示的是某個畫面的程式，preview 將可切換分頁。第一個小圓點的分頁將顯示整個 App，方便我們直接操作 App。

第二個小圓點的分頁則顯示左邊程式對應的頁面。

顯示 console，查看 preview 操作時列印的訊息

點選下圖橘色框框的按鈕可顯示 / 關閉 console，查看 preview 操作時列印的訊息。比方下圖點選 preview 的畫面將印出 tap，因此 console 可看到 tap 印出。

執行 App

點選三角形執行 App。

App 執行時畫面將帥氣地佔滿螢幕。

點選上方的三個小圓點後，下方將出現視窗選單。

點選 project 視窗可回到專案，繼續修改程式。(ps: 值得注意的，此時 App 還在執行，並沒有被終止）

比方我們將 Hello, world 的文字改成橘色。

接著點選下圖紅色框框的執行按鈕時，將有 Update & Show App 和 Stop App 兩個選項。Update & Show App 將套用我們更新的程式，然後回到剛剛的 App 畫面繼續執行，Stop App 則會終止剛剛執行的 App。

當 App 執行時，我們也可以點選右上的 Swift 圖示。

此時將出現以下操作的視窗。

若我們將上圖的 Show Console 開關打開，將可在左下看到顯示 / 關閉 console 的按鈕。

點選按鈕後，Console 將顯示在下方，比方從下圖可看到 print 列印的內容。

搜尋過濾 console 的訊息

點選 console 視窗右上的按鈕可搜尋過濾 console 的訊息。

顯示專案的檔案清單

點選下圖紅色框框的按鈕可打開 / 關閉專案的檔案清單。

專案的檔案清單。

搜尋程式

從檔案清單上方的搜尋框可搜尋程式。

搜尋程式裡出現 await 的地方。

新增 Swift 檔

點選紅色框框的 +，然後選擇 Swift File。

輸入 Swift 檔的名字。

定義自訂的 SwiftUI 型別

從 Swift Playgrounds 快速新增自訂的 SwiftUI 型別
Swift Playgrounds 不像 Xcode 那麼強大，大部分的 PK 都輸給 Xcode，不過最近彼得潘玩了一下後，發現它在新增自訂的 SwiftUI 型別時滿方便的。medium.com

刪除檔案和修改檔名

長按檔案清單裡的檔案名字，選擇 Delete 或 Rename。

新增資料夾

點選紅色框框的 +，然後選擇 Folder。

輸入資料夾的名字。

在此我們輸入 Model。

將檔案加到資料夾下

  • 方法 1: 長按資料夾名字，然後點選 Add Swift File。
  • 方法 2: 長按檔案清單裡的檔案名字，然後將它拖曳到資料夾下。

將 iPad Photos 相簿的照片加入 App

點選紅色框框的 +，然後選擇 Photo。

選擇 iPad Photos 相簿裡的照片。

加入圖片後，將生成 Assets 區塊，圖片檔將長在 Assets 下。

長按圖片檔名後，點選 Rename 可修改名字。

我們加入兩張圖片，分別命名為 ice cream & Peter。

有了 Assets 後，我們也可以長按 Assets，然後點選 Add from Photos 加入相簿的照片。

在程式裡利用 Image 顯示圖片。

將 Files App 裡的圖片加入 App

我們也可以將 Files App 裡的圖片加入 App。

  • 方法 1: 長按檔案清單裡的 Assets，然後點選 Add from Files。

顯示 Files App 裡的檔案清單。

  • 方法 2: 點選紅色框框的 +，然後選擇 Insert from。

加入第三方套件(Swift Package)

點選紅色框框的 +，然後選擇 Swift Package。

輸入套件的網址。

在此我們輸入套件幸運轉盤的網址。

https://github.com/sameersyd/FortuneWheel

找到套件後，點選右上的 Add to App Playground。

成功加入套件後，將長出 Packages 區塊。從下圖可看到套件 FortuneWheel 長在 Packages 下。

在 ContentView 使用套件 FortuneWheel。

設定 App 的名稱，主要顏色，App Icon & 功能

點選 App Settings 將跳出 App 的設定視窗。

  • 設定 App 的名稱

點選 Name，進入名稱設定頁面。

輸入名稱。

  • 設定 App 主要的顏色(Accent Color)

從 Accent Color 設定 App 主要的顏色。

比方選擇紅色後，目前顯示的 tab 上的圖文會變成紅色。

  • 設定 App Icon

方法 1: 使用 Apple 提供的圖片(Placeholder)

選擇 Placeholder，從下方的圖片清單選擇喜歡的圖片，icon 的背景顏色將採用 Accent Color。

方法 2: 自訂圖片(Custom)

加入自訂的圖片當 icon。勾選 Custom 後，可透過三個方法加入圖片。

  • Import from Files。
  • Import from Photos。
  • Paste from Clipboard。
  • 設定 App 的功能(Capabilities)

點選 Capabilities，設定 App 的功能。

點選 Add Capability。

選擇想要的功能，比方加入拍照的權限，點選 Camera。

輸入請求拍照權限的文字。

上架 iOS App

從 iPad 的 Swift Playgrounds 上架 iOS App
上架 iOS App 再也不是 Xcode 的專利了，現在我們也可以用 iPad 的 Swift Playgrounds 上架 iOS App，將 App 上傳到 App Store Connect。medium.com

其它功能

點選下圖紅色框框的按鈕可看到其它功能，比下以下幾點 :

  • 小 A 大 A: 點選小 A 縮小程式文字的大小，點選大 A 增加大小。
  • Comment Selection: 將目前選取的程式變注解，在前方加上 \\。
