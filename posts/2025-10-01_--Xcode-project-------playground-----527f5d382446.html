在 Xcode project 新增執行 #playground 的檔案

最近彼得潘看了 Apple 官方的教學影片，發現除了在 Project 原本的檔案加入 #Playground，另外建立 Playgrounds 資料夾，然後在下面新增 swift 檔來輸入 #Playground 也是不錯的方法。

如下圖所示，Apple 在 Playgrounds 資料夾下新增 Playground.swift。

然後在其中輸入 #Playground 實驗 LanguageModelSession 的相關功能。

#Playground {
    let instructions = """
        Your job is to create an itinerary for the user.
        Each day needs an activity, hotel and restaurant.
        
        Always include a title, a short description, and a day-by-day plan.
        """
    
    let session = LanguageModelSession(instructions: instructions)
    let response = try await session.respond(to: "Generate a 3-day itinerary to Paris.")
}

模仿 Apple 的架構，我們也可在專案採用類似的寫法。比方以下彼得潘在 Playground.swift 測試計算 BMI 的程式。

import Playgrounds

#Playground {
    func calculateBMI(weight: Double, height: Double) -> Double {
        return weight / (height * height)
    }
    let bmi = calculateBMI(weight: 60, height: 1.8)
}
