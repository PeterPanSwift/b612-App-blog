如何將 Flutter 專案資料夾壓縮成 zip

常常有學生問彼得潘問題，需要將專案程式碼傳給彼得潘參考。除了分享 Github 連結，另一個較簡單的方法是將專案資料夾壓縮成 zip 後傳給彼得潘，步驟如下。

刪除專案資料夾下的 build 資料夾

將專案資料夾壓縮成 zip 前，記得先刪除專案資料夾下的 build 資料夾，才不會光傳檔給彼得潘就花了大量時間。

為什麼呢 ? 因為當我們編譯專案程式執行 App 時，它將產生大量的資料存在 build 資料夾。如下圖所示，當我們將 Flutter 預設建立的專案執行在模擬器上後，生成的 build 資料夾竟有三百多 MB ! 它只不過是個簡單的計數 App，想不到竟如此肥大，真是 App 不可貌相。

刪除 build 資料夾有三種方法。

  • 方法 1: 手動刪除 build 資料夾。
  • 方法 2: 從 IDE 執行 Clean Project，叫出 Command Palette 執行 Flutter: Clean Project。
  • 方法 3: 從 IDE 執行 Clean Project，在 Terminal 輸入 flutter clean。

將專案資料夾壓縮成 zip

  • 方法 1: 從專案資料夾的右鍵選單點選壓縮資料夾。
  • 方法 2: 從 IDE 的 Terminal 輸入以下指令壓縮專案資料夾。
 zip -r ../project.zip .

它將生成 project.zip，跟專案資料夾在同一層。
