定義 Swift iOS App 的資料(model)型別

開發 iOS App 時我們常常會新增檔案定義資料的型別，以下我們以定義 struct Book 為例說明相關的步驟。

  • 新增 Empty File。

方法 1: 從 Quick Actions 點選 Empty File。

方法 2: 從 Project navigator 的右鍵選單點選 New Empty File。

方法 3: 從 Project navigator 左下的 + 點選 New Empty File。

方法 4: 從 Xcode menu 點選 File > New > Empty File。

  • 輸入檔名。

通常檔名即是我們要定義的型別名字。

  • 定義型別。

在剛剛生成的 Book.swift 定義 struct Book。

struct Book {
    let name: String
    let author: String
    let coverImage: String
    let description: String
}
