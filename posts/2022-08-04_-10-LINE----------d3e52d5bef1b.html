#10 LINE 聊天室的背景特效

模仿 LINE App，輸入特別的關鍵字時顯示背景特效。

範例: 輸入情人節，七夕，煙火時顯示可愛的煙火特效。

功能和畫面需求

  • 輸入關鍵字時顯示特效。
  • 聊天訊息裡的關鍵字會變色加底線，點擊時會顯示特效。
  • 用 CAEmitterLayer 或 SpriteKit Particle File 實現背景特效。

方法1: 使用 SpriteKit Particle File & SKView

比方聊天室畫面以 table view 實現，我們可將顯示特效的 SKView 設為 table view 的 backgroundView。

let skView = SKView(frame: view.frame)
tableView.backgroundView = skView
let scene = SKScene(size: skView.frame.size)
scene.anchorPoint = CGPoint(x: 0.5, y: 1)
let emitterNode = SKEmitterNode(fileNamed: "MyParticle")
scene.addChild(emitterNode!)
skView.presentScene(scene)

方法2: 使用 CAEmitterLayer

let snowEmitterCell = CAEmitterCell()
snowEmitterCell.contents = UIImage(named: "peter")?.cgImage
snowEmitterCell.birthRate = 1
snowEmitterCell.lifetime = 20
snowEmitterCell.scale = 0.5
snowEmitterCell.velocity = 100
snowEmitterCell.yAcceleration = 30
snowEmitterCell.scale = 0.5
snowEmitterCell.scaleRange = 0.3
snowEmitterCell.scaleSpeed = -0.02
snowEmitterCell.spin = 0.5
snowEmitterCell.spinRange = 1
snowEmitterCell.emissionRange = .pi
        
let snowEmitterLayer = CAEmitterLayer()
snowEmitterLayer.emitterPosition = CGPoint(x: view.bounds.width / 2, y: -50)
snowEmitterLayer.emitterSize = CGSize(width: view.bounds.width, height: 0)
snowEmitterLayer.emitterShape = .line
snowEmitterLayer.emitterCells = [snowEmitterCell]
tableView.backgroundView = UIView()
tableView.backgroundView?.layer.addSublayer(snowEmitterLayer)
