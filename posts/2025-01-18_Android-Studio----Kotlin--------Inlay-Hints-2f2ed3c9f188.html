Android Studio 設定 Kotlin 參數名提示的 Inlay Hints

呼叫 Kotlin 程式的 function 時，我們可以省略參數名，也可以包含參數名，例如以下計算 BMI 的 calculateBMI function。

fun calculateBMI(height: Double, weight: Double): Double {
    val heightInMeters = height / 100.0
    return weight / (heightInMeters * heightInMeters)
}
calculateBMI(height = 175.0, weight = 70.0)
calculateBMI(175.0, 70.0)

不過當我們省略參數名時，Android Studio 也會貼心地提示參數名，幫助我們更容易理解參數的意思，比方下圖的 height: 和 weight:。

參數名的提示跟 Inlay Hints(嵌入提示)有關，Android Studio 預設開啟此功能，若想關閉可參考以下方法。

從 Find Action(cmd + shift + a) 叫出 Inlay Hints 的設定頁面。

參數名的提示和 Parameter names 有關。

取消 Parameter names 的勾選將不再出現參數名的提示。

Parameter names 也可針對程式語言調整，目前可設定四種語言。
