開發 SwiftUI / Flutter App 時，我們時常將元件切出來變成獨立的型別，現在有了 AI 的幫忙，我們可以請它一步實現在新檔定義型別和 preview 的功能呢。

以下我們以環球影城的 SwiftUI 程式為例示範。

  • 請 AI 將選取的 view 切出來 & 取名。

  • Xcode。

選取程式。

叫出 Coding Tools。

輸入 extract subview。

AI 成功切出 TitleTextView，確認無誤的話按下 X，若要回到之前的程式則可點選返回的箭頭。

  • VS Code。

選取 Text("日本環球影城")。

請 GitHub Copilot extract subview。

GitHub Copilot 成功 extract subview，定義新的型別 TitleView。

  • ChatGPT App。

ChatGPT App 也可以幫我們將選取的程式區塊 extract subview。

請 AI 將選取的 view 切出來、取名、搬到新檔、定義 preview

當我們 extract subview 時，有時會想實現以下 4 個任務。

  • extract subview。
  • 幫型別取個適合的名字。
  • 將型別的定義搬到新檔。
  • 定義型別的 preview。

雖然是簡單的任務，但是一步步做還是有點辛苦。透過 AI 我們可以一個 prompt 搞定。

選取想切出的程式區塊後，輸入以下 prompt。

extract subview & move to new file
  • Xcode。

AI 成功切出 ParkTitleView 並將它搬到獨立的檔案。

  • VS Code。

AI 將新定義的型別 HeaderView 搬到新增的 HeaderView.swift，而且連 preview 也幫我們寫好了。

import SwiftUI

struct HeaderView: View {
    var title: String
    
    var body: some View {
        Text(title)
            .font(.system(size: 34, weight: .bold, design: .rounded))
            .foregroundStyle(.white)
            .shadow(radius: 5)
    }
}

#Preview {
    HeaderView(title: "日本環球影城")
        .background(Color.black)
}
