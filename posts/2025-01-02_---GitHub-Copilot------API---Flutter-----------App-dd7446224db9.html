使用 GitHub Copilot 開發串接 API 的 Flutter 網頁 ～ 比方抽獎 App

Flutter 可以開發任何人都能從瀏覽器使用的網頁 App，因此很適合用來開發一些日常生活的應用，而當我們在 App 裡串接網路 API 時，它將具備新增資料和顯示網路資料等進階功能，實現抽獎、訂飲料、每週電影等進階的應用。

最近彼得潘的朋友 Yii 出了一本不錯的書【Flutter 勇者之書】，因此接下來我們將以抽書為例，示範用 GitHub Copilot 快速開發一個串接 API 的 Flutter 抽獎 App。

在 AirTable 建立抽獎者名單的 table

為了儲存抽獎者名單，我們將串接 AirTable API，將資料存在 AirTable。

利用 Airtable 製作 iOS App 連結的後台資料
Airtable 擁有強大的資料表單功能，如下圖所示，美美的表格可以在瞬間完成，而且欄位裡可輸入多種類型的資料，比方文字，時間，圖片，多選的選項等都不是問題。medium.com

以下彼得潘在 AirTable 建立儲存抽獎者名單的 table，欄位有名字和 email。

取得 Airtable API 建立資料和讀取資料的 curl 指令

從以下連結取得 Airtable API 建立資料和讀取資料的 curl 指令。

https://airtable.com/developers/web/api/introduction
  • 取得抽獎者清單的 curl 指令和回傳的 JSON 資料。
curl "https://api.airtable.com/v0/appK5tCbiH3ysKST3/flutter?maxRecords=3&view=Grid%20view" \
  -H "Authorization: Bearer YOUR_SECRET_API_TOKEN"

EXAMPLE RESPONSE

{
    "records": [
        {
            "id": "recsLYBRpsoU3Gf8y",
            "createdTime": "2024-12-31T17:52:13.000Z",
            "fields": {
                "email": "peter@test.com",
                "Name": "peter"
            }
        }
    ],
    "offset": "itrImGRQXC8qWnjfS/recsLYBRpsoU3Gf8y"
}
  • 新增抽獎者的 curl 指令和回傳的 JSON 資料。
curl -X POST https://api.airtable.com/v0/appK5tCbiH3ysKST3/flutter \
  -H "Authorization: Bearer YOUR_SECRET_API_TOKEN" \
  -H "Content-Type: application/json" \
  --data '{
  "records": [
    {
      "fields": {
        "Name": "peter",
        "email": "peter@test.com"
      }
    },
    {
      "fields": {
        "Name": "peter",
        "email": "peter@test.com"
      }
    }
  ]
}'

EXAMPLE RESPONSE

{
    "records": [
        {
            "id": "recsLYBRpsoU3Gf8y",
            "createdTime": "2024-12-31T17:52:13.000Z",
            "fields": {
                "email": "peter@test.com",
                "Name": "peter"
            }
        },
        {
            "id": "recsLYBRpsoU3Gf8y",
            "createdTime": "2024-12-31T17:52:13.000Z",
            "fields": {
                "email": "peter@test.com",
                "Name": "peter"
            }
        }
    ]
}

使用 AI 開發抽獎 App

使用 GitHub Copilot 的 Copilot Edit 功能請 AI 修改 Flutter 專案，prompt 如下。(ps: 彼得潘事先將書本相關圖片傳到網路上，因此有三張圖片的網址。Bearer 後的 YOUR_SECRET_API_TOKEN 請換成真正的 token )

請修改 main.dart 寫一個"Flutter 勇者之書"的抽獎 App，功能如下。
1. 想抽獎的人可輸入名字和 email，資料存在 AirTable
2. 點選抽獎按鈕時從名單裡抽出得獎者。
3. 畫面上有三張書的圖片，圖片連結如下。

建資料的 AirTable API 如下。

curl -X POST https://api.airtable.com/v0/appK5tCbiH3ysKST3/flutter \
  -H "Authorization: Bearer YOUR_SECRET_API_TOKEN" \
  -H "Content-Type: application/json" \
  --data '{
  "records": [
    {
      "fields": {
        "Name": "peter",
        "email": "peter@test.com"
      }
    },
    {
      "fields": {
        "Name": "peter",
        "email": "peter@test.com"
      }
    }
  ]
}'
