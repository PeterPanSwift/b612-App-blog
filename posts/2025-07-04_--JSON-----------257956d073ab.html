將 JSON 變成容易看懂的表格

最近彼得潘家教同學利用 AI Vibe Coding 開發 App 時，發現一個重要的需求，如何讓一般人快速看懂 API 回傳的 JSON 格式資料 ?

對某些同學來說，Vibe Coding 的目的是利用 AI 做出成品，是否理解 JSON 並不重要。因此彼得潘想到不如請 AI 做個網頁，方便同學將 JSON 變成易懂的表格。

以下彼得潘利用 Firebase Studio 製作 B612 星球小王子風格的 JSON Table Viewer，然後部署到 Vercel。

B612 Star-Chart Viewer
Explore your data, just as the Little Prince explores the stars.json-table-viewer-mu.vercel.app

它包含以下功能。

  • 將上傳的 JSON 檔的 array 變成表格。

程式將自動找出 JSON 裡的 array，將它變成表格，滑動時 row 的行號和 column 的標題將固定不動。

如果找不到 array，我們可另外點選下圖的 Select a data path to display 輸入 array 所在的路徑。

以下我們選擇 feed.results。

成功顯示 feed.results 的 array 內容。

  • JSON 找不到 array 時，將第一層 { } 的 key 當 table 的欄位。

由於沒有 array，表格只顯示一筆資料。

  • 表格裡的資料呈現。

圖片網址將顯示縮圖，時間將顯示一般人熟悉的格式。

  • 表格裡圖片和網址的點擊。

點擊圖片將開啟新分頁顯示圖片，點擊圖片以外的網址將開啟新分頁顯示內容。

  • 表格裡 { } 和 [ ] 資料的點擊。

當表格呈現的資料是 { } 或 [ ] 時，點擊將開啟新分頁顯示可展開合併的樹狀圖。

點擊後顯示 JSON Object Viewer，可點擊 { } 和 [ ] 展開合併。

  • 排序。

點選欄位名字排序。

由小到大排序。

由大到小排序。

  • 搜尋過濾。

在 Filter 框框輸入搜尋過濾的關鍵字。

顯示包含劉德華的資料。
