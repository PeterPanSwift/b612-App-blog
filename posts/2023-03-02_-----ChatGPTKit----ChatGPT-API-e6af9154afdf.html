搭配套件 ChatGPTKit 串接 ChatGPT API

ChatGPT API 才剛推出幾小時，馬上就有熱心的大大開發方便 Swift 串接的 ChatGPTKit。

相較於從 ChatGPT 網頁操作，串接 ChatGPT API 的好處是它有較高的機率回答完整的程式，目前測試優化 200 行內的程式通常可以完整回答。

以下彼得潘一步步示範如何搭配 ChatGPTKit 串接 ChatGPT API。

連到 OpenAI API 網站，點選右上的 Log in 登入

OpenAI API
We're releasing an API for accessing new AI models developed by OpenAI.openai.com

點選 API

點選左邊測邊欄的 API keys

建立 API key

點選 Create new secret key。

點選 Create secret key。

複製 API key

記得一定要複製 API key，因為之後無法再取得曾經建立的 API key。

建立專案，在專案裡加入套件 ChatGPTKit

點選 Xcode menu 的 File > Add Packages。

在右上的輸入框輸入套件網址。

https://github.com/heysaik/ChatGPTKit
GitHub - heysaik/ChatGPTKit: A Swift Package to communicate with the ChatGPT API from OpenAI
A very simple Swifty API to use ChatGPT from OpenAI. Please let me know if you find any errors. In your Xcode project…github.com

ChatGPTKit 成功加入，出現在 Package Dependencies 下。

串接的程式範例

以下用 SwiftUI 製作輸入問題和顯示答案的頁面，請將 API key 換成自己帳號的 API key。

import SwiftUI
import ChatGPTKit

struct ContentView: View {
    @State private var question = ""
    @State private var answer = ""

    var body: some View {
        VStack {
            TextEditor(text: $question)
                .frame(height: 100)
                .border(.primary, width: 2)
            
            ScrollView {
                Text(answer)
            }
            
            HStack {
                Button("Chat") {
                    Task {
                        try await chat()
                    }
                }
                Button("Clear") {
                    question = ""
                    answer = ""
                }
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
       
    }
        
    func chat() async throws {
        let chattyGPT = ChatGPTKit(apiKey: "123456")
        let history = [Message(role: .user, content: question)]
        switch try await chattyGPT.performCompletions(messages: history) {
        case .success(let response):
            let firstResponse = response.choices[0]
            answer = firstResponse.message.content
            UIPasteboard.general.string = answer
            print(answer)
        case .failure(let error):
            print(error)
        }
    }
}

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
    }
}
