利用 StringTransform(ICU string transform) 做文字轉換 — 拼音，注音 & 繁簡轉換

利用 StringTransform(ICU string transform) 我們可以對文字做許多特別的處理轉換，以下讓我們看看幾種常見的中文應用。

  • 取得文字的拼音。
  • 取得護照名字的拼音。
  • 取得文字的注音。
  • 繁簡轉換。

取得文字的拼音

let name = "金城武"
let newName = name.applyingTransform(.toLatin, reverse: false)

呼叫 applyingTransform 做轉換，applyingTransform 的宣告如下，參數 transform 控制做哪一種轉換。在此我們傳入 StringTransform 的型別屬性 toLatin，得到金城武對應的拼音。參數 reverse 控制是否做相反的轉換，若是傳入 true，表示要從拼音轉換成另一種格式。並不是所有的轉換都能做相反的轉換，比方剛剛的程式傳入 true 將得到 nil。

func applyingTransform(
    _ transform: StringTransform,
    reverse: Bool
) -> String?

結果

jīn chéng wǔ

取得護照名字的拼音

let name = "金城武"
let newName = name.applyingTransform(.toLatin, reverse: false)?.applyingTransform(.stripCombiningMarks, reverse: false)?.uppercased()

為了取得護照名字的拼音，我們可以呼叫多次 applyingTransform 做多次轉換，先用 toLatin 得到拼音，然後用 stripCombiningMarks 移除音調，最後再呼叫 uppercased 變大寫。

結果

JIN CHENG WU

取得文字的注音

let message = "與你相遇好幸運"
let transform = StringTransform("Bopomofo")
let newMessage = message.applyingTransform(.toLatin, reverse: false)?.applyingTransform(transform, reverse: false)

轉換成注音必須做兩次轉換，先轉換成拼音，再轉換成注音。不過StringTransform 沒有定義轉換成注音的型別屬性，因此我們必須自己以 StringTransform("Bopomofo") 生成轉換成注音的 StringTransform。(ps: Bopomofo 念起來就是注音的 ㄅㄆㄇㄈ）

結果

ㄩˇ ㄋㄧˇ ㄒㄧㄤ ㄩˋ ㄏㄠˇ ㄒㄧㄥˋ ㄩㄣˋ

除了剛剛的寫法，以下兩種寫法也可以。生成 StringTransform 時可傳入多個轉換的字串，中間以分號隔開，因此 Latin;Bopomofo 表示要先轉成拼音，再轉成注音。

let message = "與你相遇好幸運"
let transform = StringTransform("Latin;Bopomofo")
let newMessage = message.applyingTransform(transform, reverse: false)

Han-Latin;Latin-Bopomofo 表示先從中文轉成拼音，再從拼音轉成注音。

let message = "與你相遇好幸運"
let transform = StringTransform("Han-Latin;Latin-Bopomofo")
let newMessage = message.applyingTransform(transform, reverse: false)

繁簡轉換

  • 繁體轉簡體。
let message = "與你相遇好幸運"
let transform = StringTransform("Hant-Hans")
let newMessage = message.applyingTransform(transform, reverse: false)

答案

与你相遇好幸运
  • 簡體轉繁體。
let message = "与你相遇好幸运"
let transform = StringTransform("Hans-Hant")
let newMessage = message.applyingTransform(transform, reverse: false)

答案

與你相遇好幸運
