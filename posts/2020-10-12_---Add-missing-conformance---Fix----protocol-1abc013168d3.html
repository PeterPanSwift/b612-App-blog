利用 Add missing conformance 的 Fix 遵從 protocol

開發 iOS App 時，我們時常設定 delegate 跟 data source，不過為了有資格當 delegate 跟 data source，我們必須先遵從對應的 protocol，比方以下例子的 ViewController 沒有遵從 protocol UIImagePickerControllerDelegate 和 UINavigationControllerDelegate，因此產生紅色錯誤。

Cannot assign value of type 'ViewController' to type '(UIImagePickerControllerDelegate & UINavigationControllerDelegate)?'

我們可以手動輸入，讓 ViewController 遵從 protocol UIImagePickerControllerDelegate 和 UINavigationControllerDelegate，不過點選上圖錯誤裡的 Fix 會更簡單，它會幫我們在 class ViewController 後加上 UIImagePickerControllerDelegate & UINavigationControllerDelegate。

class ViewController: UIViewController, UIImagePickerControllerDelegate & UINavigationControllerDelegate {

ps: 遵從多個 protocol 也可以用透號分隔。

class ViewController: UIViewController, UIImagePickerControllerDelegate, UINavigationControllerDelegate {
