設定需要多行程式計算的 property 初始值

定義 struct 或 property 時，我們可以在宣告 property 時設定它的初始值。

struct Baby {
    var name = "peter"
    var age = 18
}

不過有時初始值需要經過多行程式的計算，無法一行搞定呢 ? 此時我們可透過以下兩種方法設定 property 的內容。

  • 自訂 init。
  • 呼叫 closure。

自訂 init

struct Baby {
    var name: String
    var age: Int
    var skills: [String]
    
    init(name: String, age: Int) {
        self.name = name
        self.age = age
        var zodiacs = ["鼠", "牛", "虎", "兔", "龍", "蛇", "馬", "羊", "猴", "雞", "狗", "豬"]
        skills = [String]()
        while skills.count != 3 {
            let zodiac = zodiacs.randomElement()!
            let number = Int.random(in: 1...100)
            skills.append("降\(zodiac)\(number)掌")
        }
    }
}

var cuteBaby = Baby(name: "peter", age: 18)
print(cuteBaby.skills)

結果。

["降蛇46掌", "降羊4掌", "降馬60掌"]

呼叫 closure

struct Baby {
    var name: String
    var age: Int
    var skills = {
        var skills = [String]()
        var zodiacs = ["鼠", "牛", "虎", "兔", "龍", "蛇", "馬", "羊", "猴", "雞", "狗", "豬"]
        while skills.count != 3 {
            let zodiac = zodiacs.randomElement()!
            let number = Int.random(in: 1...100)
            skills.append("降\(zodiac)\(number)掌")
        }
        return skills
    }()
}

var cuteBaby = Baby(name: "peter", age: 18)
print(cuteBaby.skills)

結果。

["降鼠62掌", "降羊16掌", "降兔95掌"]

若是想讓程式的效能更好，我們也可以在剛剛的 skills 前加上 lazy，讓 closure 等到程式第一次存取 skills 時才執行。

struct Baby {
    var name: String
    var age: Int
    lazy var skills = {
        var skills = [String]()
        var zodiacs = ["鼠", "牛", "虎", "兔", "龍", "蛇", "馬", "羊", "猴", "雞", "狗", "豬"]
        while skills.count != 3 {
            let zodiac = zodiacs.randomElement()!
            let number = Int.random(in: 1...100)
            skills.append("降\(zodiac)\(number)掌")
        }
        return skills
    }()
}
