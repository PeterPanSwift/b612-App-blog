以下彼得潘一人分飾兩角，以自己的兩個帳號 PeterPanSwift 和 AppPeterPan 為例說明如何使用 Zed IDE 多人編輯程式、語音聊天和螢幕分享。

首先 PeterPanSwift 用 Zed 打開 Xcode 專案。(ps: Zed 可打開任何的資料夾，因此 swiftpm 也可以。)

點選 Zed 視窗左下的 Collab Panel 按鈕打開協作面板。

點選 Contacts 右邊的 + 加人。

輸入對方的 Zed 帳號名字搜尋。由於 Zed 必須使用 GitHub 帳號登入，Zed 帳號名將和 GitHub 帳號名一致。

以下 PeterPanSwift 選擇邀請他的知己 AppPeterPan。

此時 AppPeterPan 打開 Zed IDE 的 Collab Panel 將看到 PeterPanSwift 出現在 Requests 下方。PeterPanSwift 是 AppPeterPan 的知己，所以他開心地按下勾勾接受邀請。

於是 PeterPanSwift 在 Contacts 的 Online 區塊看到 AppPeterPan 上線了。

PeterPanSwift 從 AppPeterPan 的右鍵選單點選 Call AppPeterPan，此動作將打電話並分享專案給 AppPeterPan 。

AppPeterPan 收到了專案共享的邀請，開心地按下 Accept。

現在 AppPeterPan 將可和 PeterPanSwift 語音聊天，而且還可從 Zed 視窗觀看和操作對方的專案。

雙方都能在畫面上看到對方的操作和修改，比方當 AppPeterPan 將 Hello 改成 hi 時，PeterPanSwift 將即時看到修改，而且看到游標是代表 AppPeterPan 的橘色。

值得注意的，Zed 預設的自動存檔是關閉的，所以要按下 cmd + s 才能存檔。若想修改後自動存檔，可在 command palette 點選 zed: open settings。

加入以下程式設定停止編輯 1 秒後自動存檔。

{
  "autosave": {
    "after_delay": {
      "milliseconds": 1000
    }
  }
}

除了分享專案，Zed 還可將螢幕畫面分享給對方看，點選右上方的 Share Screen 按鈕即可分享螢幕。

當 PeterPanSwift 分享螢幕後，AppPeterPan 的畫面上方出現切換分頁的標籤，點選可切換顯示專案程式或螢幕畫面。

除了以上介紹的功能，Zed 還可以建立 chaneel 和邀請別人加入 channel 一起編輯程式跟聊天，相關說明可參考以下連結。
