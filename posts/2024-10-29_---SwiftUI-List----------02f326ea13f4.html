設定 SwiftUI List 的背景顏色或圖片

SwiftUI List 預設會搭配淺灰色的背景(dark mode 為黑色)，以下我們介紹如何設定它的背景顏色或圖片。

設定 SwiftUI List 的背景顏色或圖片

SwiftUI List 預設會搭配淺灰色的背景(dark mode 為黑色)，以下我們介紹如何設定它的背景顏色或圖片。

struct ContentView: View {
    let lyrics = [
        "右手晴天",
        "左手雨天",
        "就讓彩虹 擦亮我的臉"
    ]
    
    var body: some View {
        NavigationStack {
            List(lyrics, id: \.self) { lyric in
                HStack {
                    Text(lyric)
                    Spacer()
                    Image(systemName: "heart")
                }
            }
            .navigationTitle("乒乓男孩主題曲")
        }
    }
}

設定 List 的背景顏色

若想設定 List 的背景，必須呼叫 scrollContentBackground(.hidden) 或 listStyle(.plain) 取消 List 預設的背景。

  • scrollContentBackground(.hidden)
struct ContentView: View {
    let lyrics = [
        "右手晴天",
        "左手雨天",
        "就讓彩虹 擦亮我的臉"
    ]
    
    var body: some View {
        NavigationStack {
            List(lyrics, id: \.self) { lyric in
                HStack {
                    Text(lyric)
                    Spacer()
                    Image(systemName: "heart")
                }
            }
            .scrollContentBackground(.hidden)
            .background(.pink)
            .navigationTitle("乒乓男孩主題曲")
        }
    }
}
  • .listStyle(.plain)
struct ContentView: View {
    let lyrics = [
        "右手晴天",
        "左手雨天",
        "就讓彩虹 擦亮我的臉"
    ]

設定 List 的背景圖片

透過 modifier background 加入背景圖。

struct ContentView: View {
    let lyrics = [
        "右手晴天",
        "左手雨天",
        "就讓彩虹 擦亮我的臉"
    ]
    
    var body: some View {
        NavigationStack {
            List(lyrics, id: \.self) { lyric in
                HStack {
                    Text(lyric)
                    Spacer()
                    Image(systemName: "heart")
                }
            }
            .scrollContentBackground(.hidden)
            .background {
                Image(.pic)
                    .resizable()
                    .scaledToFill()
                    .blur(radius: 5)
                    .ignoresSafeArea()
            }
            .navigationTitle("乒乓男孩主題曲")
        }
    }
}

row 的背景預設為白色(dark mode 為黑色)，可用 listRowBackground(Color.clear) 改成透明。

struct ContentView: View {
    let lyrics = [
        "右手晴天",
        "左手雨天",
        "就讓彩虹 擦亮我的臉"
    ]
    
    var body: some View {
        NavigationStack {
            List(lyrics, id: \.self) { lyric in
                HStack {
                    Text(lyric)
                    Spacer()
                    Image(systemName: "heart")
                }
                .listRowBackground(Color.clear)
                .listRowSeparator(.hidden)
            }
            .scrollContentBackground(.hidden)
            .background {
                Image(.pic)
                    .resizable()
                    .scaledToFill()
                    .blur(radius: 5)
                    .ignoresSafeArea()
            }
            .navigationTitle("乒乓男孩主題曲")
        }
    }
}
