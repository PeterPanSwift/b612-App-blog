開發 iOS App 遊戲時，我們時常使用附檔名 sks 的 SpriteKit Scene 設計遊戲畫面。

建立新專案

將 controller 的 view 類別改成 SKView

新增設計遊戲畫面的 SpriteKit Scene (sks)

SceneKit Scene File 的附檔名是 sks。

自訂繼承 SKScene 的類別

SKScene 負責呈現遊戲的畫面，因此我們自訂繼承 SKScene 的類別 GameScene，之後可以在 GameScene 裡撰寫遊戲的功能。

import SpriteKit
class GameScene: SKScene {
}

將 sks 檔的 scene 物件類別從 SKScene 改成 GameScene

顯示 scene

import UIKit
import GameplayKit
class ViewController: UIViewController {
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        if let scene = GKScene(fileNamed: "GameScene"),
           let sceneNode = scene.rootNode as? GameScene {
            
            sceneNode.scaleMode = .aspectFill
            
            if let view = self.view as! SKView? {
                view.presentScene(sceneNode)
            }
        }
        
    }
    
}
