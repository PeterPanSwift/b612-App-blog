iOS App 連線本機後台的網址設定

從模擬器連線

網址設為 localhost 或 Mac 目前的 IP。

class ViewController: UIViewController {
    @IBOutlet weak var imageView: UIImageView!
    
    func fetchImage() {
        let url = URL(string: "http://localhost/peter.png")!
        URLSession.shared.dataTask(with: url) { data, response, error in
            if let data {
                DispatchQueue.main.async {
                    self.imageView.image = UIImage(data: data)
                }
            }
        }.resume()
    }
    
    override func viewDidLoad() {
        super.viewDidLoad()
        
        fetchImage()
    }
}

ps: 在 iOS 14 之前(包含 iOS 14)的版本設為 localhost 會有問題，必須額外設定 Allows Local Networking。

從實機連線

輸入 Mac 目前的 IP，iPhone & Mac 必須在同一個區域網路。

let url = URL(string: "http://192.168.1.105/peter.png")!
