將元件加到預覽畫面

我們希望在文字與你相遇好幸運的下方顯示小幸運的下一句歌詞，可我已失去為你淚流滿面的權利。

  • 將 preview 切換到 Selectable 模式

從 Xcode 14 開始，若要從 Preview 畫面加入元件，請先點選下圖的箭頭，切換到 Selectable 模式。

  • 點選右上方的 + 打開 Library 視窗。(快速鍵 cmd + shift + L)

Library 視窗的第一個 icon 分頁 Views library 有許多可加到畫面的元件，有興趣的朋友可參考以下連結的介紹。

  • 將 Text 拖曳到預覽畫面的文字與你相遇好幸運的下方。

此時下方還會出現貼心的提示文字，Add Text to a new Vertical Stack along with existing Text。

從右邊的預覽畫面，我們看到文字順利地出現在與你相遇好幸運下方，而且左邊的 Swift 程式也同步更新，長出了新的 Text，並且生成垂直排列元件的 VStack 容納兩個 Text。

雖然 Xcode 很聰明，但是它還不會讀心術，因此我們要勤勞一點，自己在 Text 的藍色框框裡輸入文字。

var body: some View {
   VStack {
      Text("與你相遇好幸運")
      Text("可我已失去為你淚流滿面的權利")
   }
}

將圖片加到預覽畫面

將美麗的圖片加到預覽畫面時，我們可先拖曳專門呈現圖片的 Image 元件，然後再手動輸入圖片的名字。

但其實有更快的方法。我們可切換到第 4 個 icon Media library 分頁，直接拖曳圖片到預覽畫面上。

將元件加到程式

我們也可以將 Library 視窗裡的元件拖曳到程式裡想要它出現的位置。如下圖所示，我們將 Text 元件拖曳到 Text("與你相遇好幸運") 下方。

不過此時 Xcode 沒那麼聰明，不會自動增添 VStack 或 HStack。其實這不能怪它，因為根據兩行 Text 的線索，就算福爾摩斯也推理不出我們想要水平還是垂直排列。

若我們想用 VStack 容納 Text，我們可先按住 cmd 鍵，然後按滑鼠左鍵點選程式裡的第一個 Text，選擇 Embed in VStack。

VStack {
   Text("與你相遇好幸運")
}
Text("Placeholder")

為了讓第二個 Text 也在 VStack 的 { } 裡，我們可選取第二個 Text 後，將它拖曳到 VStack 的 { } 裡。

以 VStack 容納 2 個 Text 後，此時右邊的預覽畫面也會即時更新，顯示垂直排列的文字。

利用 Embed in 加入元件

剛剛我們示範了利用 Embed in 加入 VStack，SwiftUI 還有很多元件也可以用 embed in 加入，詳情可參考以下連結。

利用 Embed in 加入 SwiftUI 元件
SwiftUI 有一些特別的元件，專門以 { } 容納其它的元件，比方 HStack，VStack，List 等，Xcode 貼心地在 Code Actions 選單提供 Embed in 功能，方便我們快速加入這些元件。medium.com

讓 Library 視窗保持在畫面上

當我們從 Library 視窗拖曳東西到預覽畫面或程式時，Library 視窗會自動消失，因為它以為我們不再需要它。若想讓它繼續保持在畫面上，方便繼續拖曳下一個東西，我們可按住 option 再點擊右上方的 +，此時 Library 視窗將保持在畫面上，除非我們點擊左上角的 X 關閉它。
