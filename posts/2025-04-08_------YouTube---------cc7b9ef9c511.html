從程式抓取 YouTube 影片的封面圖片

YouTube 影片的封面圖片有固定的網址格式，將https://i.ytimg.com/vi/影片id/ 加上下表的圖片名字和 .jpg 即為圖片網址。

以小王子的 YouTube 影片 The Little Prince : A worldwide phenomenon 為例。

它的網址是 https://www.youtube.com/watch?v=xHG1cs5IyDc，因此影片 id 是 xHG1cs5IyDc，1280 * 720 的圖片網址為 https://i.ytimg.com/vi/xHG1cs5IyDc/maxresdefault.jpg。

以下我們分別示範用 SwiftUI、Flutter、UIKit 顯示小王子圖片。

  • SwiftUI
AsyncImage(url: URL(string: "https://i.ytimg.com/vi/xHG1cs5IyDc/maxresdefault.jpg"))
  • Flutter
Image.network('https://i.ytimg.com/vi/xHG1cs5IyDc/maxresdefault.jpg'),
  • UIKit

使用套件 Kingfisher。

import Kingfisher
import UIKit

class ViewController: UIViewController {

    @IBOutlet weak var imageView: UIImageView!

    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.

        imageView.kf.setImage(with: URL(string: "https://i.ytimg.com/vi/xHG1cs5IyDc/maxresdefault.jpg"))
    }
}
