#247 利用下一個 / 上一個 button 切換內容

功能需求

  • 使用 SwiftUI / UIKit / Flutter / Compose 開發 App，利用下一個 / 上一個 button 切換內容。
  • 點擊下一個 button 顯示下一個，點擊上一個 button 顯示上一個。
  • 3 種設計。
  1. 無限循環。

在第一頁點擊上一個將顯示最後一頁的內容，在最後一頁點擊下一個將顯示第一頁的內容。

2. 控制 button 是否顯示。

在第一頁時顯示下一個 button，隱藏上一個 button。在最後一頁時顯示上一個 button，隱藏下一個 button。

3. 控制 button 是否可點擊。

不可點擊的 button 顏色會比較淡。在第一頁時上一個 button 不可點擊，在最後一頁時下一個 button 不可點擊。
