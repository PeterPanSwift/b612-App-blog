搜尋研究 Apple 官方文件的教學 — Article，Tutorial & Sample Code

學習 iOS App 開發時，彼得潘喜歡找尋各式各樣有趣的學習資源。在這麼多資源裡，彼得潘最愛的還是 Apple 自己寫的官方文件，因為它有著最新也最正確的寫法。

Apple 官方文件最主要的部分是 API 的介紹，說明了各種型別，property 和 method 的用法。不過若想看較完整的教學，官方文件還包含了以下三大類的教學。

  • Article (教學文章)。
  • Tutorial (實作完整 App 的教學)。
  • Sample Code (範例程式)。

以下我們一步步介紹如何搜尋這些精彩的 Article，Tutorial & Sample Code，分成從 Xcode 操作和網頁瀏覽兩種方法。

打開官方的文件

  • Xcode

點選 Xcode menu 的 Help > Developer Documentation。

  • 網頁

連到 Apple 開發網站的 Technologies 頁面。

Apple Developer Documentation
Edit descriptiondeveloper.apple.com

搜尋 Articles，Tutorials，Sample Code

  • Xcode

從左邊下方的 Filter 框框輸入 art，然後選擇 Type 下的 Article。

此時可看到滿滿的教學文章，以下我們點選 SwiftUI 下的 Declaring a custom view。

我們也可以加上關鍵字，比方輸入 gesture，搜尋 gesture 的相關文章。

接著我們試試輸入 sam，然後選擇 Type 下的 Sample Code。

此時可看到滿滿的範例程式，以下我們點選 SwiftUI 下的 Food Truck: Building a SwiftUI multiplatform app，點選 Download 可下載範例。

接著我們試試輸入 tut，然後選擇 Type 下的 Tutorial。

此時可看到滿滿的 App 實作教學。

以下我們點選 Exploring SwiftUI Sample Apps 下的 Choose Your Own Story，點選 Project files 可下載範例。

  • 網頁

點選有興趣的主題，比方點選 UIKit。

從左邊下方的 Filter 框框選擇搜尋的分類。

點選 Articles 查看全部的文章，以下我們點選 About App Development with UIKit。

我們也可以加上關鍵字，比方輸入 touch，搜尋 touch 的相關文章。

點選 Sample Code 查看全部的範例程式，以下我們點選 UIKit Catalog: Creating and Customizing Views and Controls，點選 Download 可下載範例。

點選 Tutorials 查看全部的教程。
