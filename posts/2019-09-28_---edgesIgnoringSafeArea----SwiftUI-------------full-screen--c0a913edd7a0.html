透過 ignoresSafeArea 或 frame 設定 SwiftUI 全螢幕的背景顏色或圖片(full screen)

為了怕東西被 iPhone 邊緣的圓角切到和時間電量檔到，SwiftUI 畫面裡的元件預設會被安置在 safe area 內，因此如下圖所示，黃色區塊不會延伸到 iPhone 頂部和底部的圓角區塊。(ps: 範例的文字是 Penny 的不想歌詞)

ZStack {
   Color.yellow
   Text("總覺得沒人瞭解吧 所以不回答 麻木跟著別人流行的步伐 總覺得沒人理會吧 所以不說話 三更半夜精疲力盡 不想爭論 不想回家")
}

雖然它預設不想延伸，但我們還是可以強迫它延伸，透過呼叫 ignoringSafeArea 忽略 safe area，實現全螢幕的背景顏色或圖片。

透過 ignoresSafeArea 設定全螢幕背景顏色

ignoresSafeArea(_:edges:) | Apple Developer Documentation
Expands the safe area of a view.developer.apple.com
  • 方法 1: ignoresSafeArea 加在 Color。
ZStack {
   Color.yellow
       .ignoresSafeArea()
   Text("總覺得沒人瞭解吧 所以不回答 麻木跟著別人流行的步伐 總覺得沒人理會吧 所以不說話 三更半夜精疲力盡 不想爭論 不想回家")
}
  • 方法 2: ignoresSafeArea 加在 ZStack。
ZStack {
    Color.yellow
        Text("總覺得沒人瞭解吧 所以不回答 麻木跟著別人流行的步伐 總覺得沒人理會吧 所以不說話 三更半夜精疲力盡 不想爭論 不想回家")
}
.ignoresSafeArea()

透過 frame 設定全螢幕背景顏色

frame 的 maxWidth & maxHeight 傳入 .infinity，讓 ZStack 的大小成為可用空間的最大值，因此 ZStack 將變成全螢幕的大小。

ZStack {
    Text("總覺得沒人瞭解吧 所以不回答 麻木跟著別人流行的步伐 總覺得沒人理會吧 所以不說話 三更半夜精疲力盡 不想爭論 不想回家")
}
.frame(maxWidth: .infinity, maxHeight: .infinity)
.background(.yellow)

設定全螢幕造成的遮蔽問題

當我們將某個 view 變成全螢幕時需特別注意是否會造成遮蔽問題。比方以下程式將 ZStack 變成全螢幕，因此裡面加了 Spacer 的 VStack 也會變成全螢幕的高度，造成 Text 跟瀏海重疊。

struct ContentView: View {
    var body: some View {
        ZStack {
            Color.yellow
            VStack {
                Text("總覺得沒人瞭解吧 所以不回答 麻木跟著別人流行的步伐 總覺得沒人理會吧 所以不說話 三更半夜精疲力盡 不想爭論 不想回家")
                Spacer()
            }
        }
        .ignoresSafeArea()
    }
}

改成將 Color 設為全螢幕則不會有遮蔽問題。

struct ContentView: View {
    var body: some View {
        ZStack {
            Color.yellow
                .ignoresSafeArea()
            VStack {
                Text("總覺得沒人瞭解吧 所以不回答 麻木跟著別人流行的步伐 總覺得沒人理會吧 所以不說話 三更半夜精疲力盡 不想爭論 不想回家")
                Spacer()
            }
        }
    }
}

設定全螢幕背景圖片

呼叫

frame(minWidth: 0, maxWidth: .infinity, minHeight: 0, maxHeight: .infinity) ，限制圖片的寬高範圍在 0 ~ 可用空間的寬高之間，因此圖片的寬高將等於螢幕寬高。

struct ContentView: View {
    var body: some View {
        ZStack(alignment: .bottomTrailing) {
            Image(.prince)
               .resizable()
               .scaledToFill()
               .frame(minWidth: 0,  maxWidth: .infinity, minHeight: 0,  maxHeight: .infinity)
               .opacity(0.5)
               .ignoresSafeArea()
            Text("小王子")
                .font(.largeTitle)
        }
    }
}

如果拿掉 frame(minWidth: 0, maxWidth: .infinity, minHeight: 0, maxHeight: .infinity) ，scaledToFill 將讓圖片的寬或高超出螢幕，因此無法看到對齊右下的 Text 小王子。

struct ContentView: View {
    var body: some View {
        ZStack(alignment: .bottomTrailing) {
            Image(.prince)
               .resizable()
               .scaledToFill()
               .opacity(0.5)
               .ignoresSafeArea()
            Text("小王子")
                .font(.largeTitle)
        }
    }
}
