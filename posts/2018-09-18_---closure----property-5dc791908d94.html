利用 closure 設定 property

當 property 的內容需要經過複雜的程式計算才能設定時，我們習慣在 init 裡設定，但其實也可以在宣告 property 時利用呼叫 closure 的 { } 程式指定內容，例如以下例子:

例子1: 產生登入的 button

class ViewController: UIViewController {

   let loginButton = {
      let button = UIButton(type: .custom)
      button.translatesAutoresizingMaskIntoConstraints = false
      button.setTitle("login", for: .normal)
      button.setTitleColor(.orange, for: .normal)
      button.addTarget(self, action: #selector(login), for: .touchUpInside)
      return button
   }()

   @objc func login() {
   }

利用 closure 指定 property 內容時，有幾個小地方要注意。

(1) closure 的 { } 後要加 () 才會執行 closure 的程式。

如果忘了加 ( )，意思將變成 property 儲存 closure 的程式碼，而不是 closure 執行後回傳的結果。

由於 loginButton 已經在宣告時透過 closure 生成，所以我們可在 viewDidLoad 裡將它加到畫面上，設定它的 auto layout 條件。

override func viewDidLoad() {
   super.viewDidLoad()
   view.addSubview(loginButton)
   loginButton.centerXAnchor.constraint(equalTo: view.centerXAnchor).isActive = true
   loginButton.centerYAnchor.constraint(equalTo:    view.centerYAnchor).isActive = true
}

(2) 不能在 closure 裡存取其它的 property 或呼叫方法，除非加上 lazy。

因為此時東西還未生成，所以其它 property 的內容也還沒設定好，因此我們不能在 closure 裡存取其它的 property 或呼叫方法。

class ViewController: UIViewController {
   var buttonTitleColor = UIColor.orange
   let loginButton = {
      let button = UIButton(type: .custom)
      button.translatesAutoresizingMaskIntoConstraints = false
      button.setTitle("login", for: .normal)
      button.setTitleColor(buttonTitleColor, for: .normal)
      button.addTarget(self, action: #selector(login), for: .touchUpInside)
      return button
   }()

比方以上例子，當我們在 closure 裡存取另一個 property buttonTitleColor 時，將產生錯誤。

倘若真的需要在 closure 裡存取另一個 property，則可將 property 改成以 lazy var 宣告。加了 lazy 的 property 會在東西已經生成，property 初次存取時才設定。

因此剛剛的例子，當 view controller 生成，loginButton 在 viewDidLoad 裡初次存取時才會執行生成 button 的 closure 程式，此時 buttonTitleColor 已被初始為橘色，可以放心地存取。

var buttonTitleColor = UIColor.orange
lazy var loginButton = {
   let button = UIButton(type: .custom)
   button.translatesAutoresizingMaskIntoConstraints = false
   button.setTitle("login", for: .normal)
   button.setTitleColor(buttonTitleColor, for: .normal)
   button.addTarget(self, action: #selector(login), for: .touchUpInside)
   return button
}()

利用型別 [Bool] 的 boardColors 儲存西洋棋的格子。true 代表黑色，false 代表白色。

struct Chessboard {
   let boardColors = {
      var temporaryBoard = [Bool]()
      var isBlack = false
      for i in 1...8 {
       
         for j in 1...8 {
            temporaryBoard.append(isBlack)
            isBlack = !isBlack
         }
         isBlack = !isBlack
      }
      return temporaryBoard
   }()
}
