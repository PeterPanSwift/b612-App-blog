!= (不等於) 的左右記得加空白

當 != 的左右不留一點空白，完全跟旁邊的東西黏在一起時，可是會惹人討厭，產生紅色錯誤的。

從錯誤訊息出現關鍵字 optional，我們了解 Swift 誤以為此處的 ! 是讀取 optional ，難怪會出錯。

因此 ，請記得 != 的左右要加空白，才不會產生天大的誤會。

var grade = 100
if grade != 100 {
print("沒考到一百分，無顏面對江東父老")
}
