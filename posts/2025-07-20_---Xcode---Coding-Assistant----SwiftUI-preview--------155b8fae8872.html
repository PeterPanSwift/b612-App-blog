使用 Xcode 的 Coding Assistant 生成 SwiftUI preview 需要的假資料

SwiftUI preview 顯示畫面時常常需要搭配一些假資料，以下我們示範使用 Xcode 的 Coding Assistant 生成 preview 需要的假資料。

以下我們用顯示可愛動物的 AnimalRow 說明。

import SwiftUI

struct AnimalRow: View {
    let name: String
    let emoji: String
    let description: String
    
    var body: some View {
        VStack(alignment: .leading, spacing: 8) {
            HStack {
                Text(emoji)
                    .font(.largeTitle)
                Text(name)
                    .font(.title)
                    .bold()
            }
            Text(description)
                .font(.body)
        }
        .padding()
    }
}

#Preview {
    AnimalRow()
}

我們加了三個 property，因此產生錯誤 Missing arguments。

請 AI 幫忙在 AnimalRow 輸入資料有三種方法。

  • 方法 1: 點選 Generate，AI 自動生成需傳入的參數，以下 AI 生成了叢林之王獅子。
  • 方法 2: 從 AnimalRow 叫出 Coding Tools，請 AI 生成指定的資料。

請 AI 生成企鵝的相關資料，description 包含 100 字的中文。

  • 方法 3: 選取 Preview 裡的 Animal( ) 後，從 Coding Assistant 請 AI 生成企鵝的相關資料，description 包含 100 字的中文。

當 AnimalRow 顯示的資料型別另外定義在 Animal.swift 時，AI 也能聰明地在 Preview 生成假資料。

比方以下分別為 Animal.swift & AnimalRow.swift 的內容。

struct Animal {
    let name: String
    let emoji: String
    let description: String
}
import SwiftUI

struct AnimalRow: View {
    let animal: Animal
    
    var body: some View {
        VStack(alignment: .leading, spacing: 8) {
            HStack {
                Text(animal.emoji)
                    .font(.largeTitle)
                Text(animal.name)
                    .font(.title)
                    .bold()
            }
            Text(animal.description)
                .font(.body)
        }
        .padding()
    }
}

#Preview {
    AnimalRow()
}

從 AnimalRow 叫出 Coding Tools，請 AI 生成企鵝的相關資料。

我們也常在資料的型別裡以 static 定義資料，以下我們在 Animal 生成 preview 需要的假資料，從 struct Animal 叫出 Coding Tools，輸入 create sample data, 企鵝，100 字的中文 description。

請 AI 生成包含多筆資料的 array 也不是問題，以下我們從 struct SampleData 叫出 Coding Tools，輸入宣告 animals 儲存 5 種動物，型別為 [Animal]。
