沒有無限循環的分頁效果

Page-Based App template 將實現 1 ~ 12 月的分頁瀏覽 App，不過它並不會無限循環。

從 1 月往右滑，無法滑到 12 月。

從 12 月往左滑，無法滑到 1 月。

修改 pageViewController(_:viewControllerBefore:) & pageViewController(_:viewControllerAfter:)

修改 ModelController.swift 裡控制頁面切換的 pageViewController(_:viewControllerBefore:) & pageViewController(_:viewControllerAfter:)。

func pageViewController(_ pageViewController: UIPageViewController, viewControllerBefore viewController: UIViewController) -> UIViewController? {
        var index = indexOfViewController(viewController as! DataViewController)
        if index == NSNotFound {
            return nil
        }
        index = (index - 1 + pageData.count) % pageData.count
        return viewControllerAtIndex(index, storyboard: viewController.storyboard!)
}
func pageViewController(_ pageViewController: UIPageViewController, viewControllerAfter viewController: UIViewController) -> UIViewController? {
        var index = indexOfViewController(viewController as! DataViewController)
        if index == NSNotFound {
            return nil
        }
        index = (index + 1) % pageData.count
        return viewControllerAtIndex(index, storyboard: viewController.storyboard!)
}

說明

  • 瀏覽上一頁

當使用者往右滑瀏覽上一頁時，將觸發 pageViewController(_:viewControllerBefore:)，我們透過 index = (index — 1 + pageData.count) % pageData.count 計算上一頁的 index，由於加上 pageData.count，因此可以確保結果不會變成負數。

  • 瀏覽下一頁

當使用者往左滑瀏覽下一頁時，將觸發 pageViewController(_:viewControllerAfter:)，我們透過 index = (index + 1) % pageData.count 計算下一頁的 index。

執行 App

從 12 月往左滑，滑到下一頁的 1 月。

從 1 月往右滑，滑到上一頁的 12 月。
