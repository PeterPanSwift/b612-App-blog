偵測 iPhone 晃動 (shake)

在 controller 裡覆寫以下 function

override func motionEnded(_ motion: UIEvent.EventSubtype, with event: UIEvent?) {
   if motion == .motionShake {
      print("shake")
   }
}

在模擬器測試

從 Simulator 的 menu 點選 Hardware > Shake Gesture。
