One more thing，從 Xcode 加入 Firebase 套件

AI 會幫 Flutter 專案加入 Firebase 套件，但是它卻不太會幫 Xcode 專案加套件，因此我們得自己從 Xcode 使用 SPM 加入 Firebase 套件。

GitHub - firebase/firebase-ios-sdk: Firebase SDK for Apple App Development
Firebase SDK for Apple App Development. Contribute to firebase/firebase-ios-sdk development by creating an account on…github.com

加入 Firebase 套件時請選擇 FirebaseCore，將它的 Add to Target 選到 App，至於列表裡的其它 library 可依 App 的功能需要加入，比方註冊登入功能需加入 FirebaseAuth。

若原本只加入 Firebase 的某些 library，之後想加入 Firebase 其它的 library，可參考以下連結。
