利用 Select Column & Move Line 搬移 SwiftUI 程式

Xcode 有很多好用的快速鍵，最近彼得潘在開發 SwiftUI App 時，發現風水輪流轉，有些之前少用的快速鍵在 SwiftUI 裡變得更重要了。

SwiftUI 的程式有許多一層包一層的寫法，因此時常會用到 Select Column & Move Line 的快速鍵，接下來就讓我們以實際例子說明吧。

將多行程式 embed 在 Group 裡

  • 在 VStack 裡顯示劉德華的經典歌曲。

華仔的經典歌很多，很輕易就超過十首。當我們在 VStack 裡放了 11 個 Text 後，將超過 @ViewBuilder 的 10 個元件個數上限，因此將產生問題。

ViewBuilder & Extra argument in call
撰寫 SwiftUI 程式時，我們時常看到類似 VStack 這樣的程式，{ } 裡寫了多行產生 UI 元件的程式。medium.com
  • 利用 Group 將多個元件組合在一起。

此問題的一種解法是利用 Group 將多個元件組合在一起，讓 VStack 裡的元件數量不會超過 10 個。我們按住 cmd 鍵，然後點選 Text("練習")，從選單裡選擇 Group。

Text("練習")順利地被包在 Group 裡。

struct ContentView: View {
    
    var body: some View {
        VStack {
            Group {
                Text("練習")
            }
            Text("冰雨")
            Text("謝謝你的愛")
            Text("我不夠愛你")
            Text("來生緣")
            Text("當我遇上你")
            Text("寧願我傷心")
            Text("小情人")
            Text("峰迴路轉")
            Text("十七歲")
            Text("等你忘了我是誰")
 
        }
    }
}
  • 利用 Select Column & Move Line 將多個 Text 移到 Group 裡。

我們的目的是在 Group 裡包好幾個 Text，而不是只包一個練習呀。假設我們想將"冰雨"，"謝謝你的愛"，"我不夠愛你"，"來生緣" 也加到 Group 裡。

請先讓滑鼠游標停在"冰雨"前，然後利用 Select Column Down 的快速鍵 ctrl + shift + 向下的方向鍵 選取 "冰雨"，"謝謝你的愛"，"我不夠愛你"，"來生緣"四行程式。

接著利用 Move Line Up 的快速鍵 cmd + option + [ 將剛剛選取的四行程式上移一行，即可將它們移民到 Group 的 { } 裡。(ps: 若需要上移不只一行 ，只要多按幾次 cmd + option + [ 即可，按一百次就可以上移 100 行)

struct ContentView: View {
    
    var body: some View {
        VStack {
            Group {
                Text("練習")
                Text("冰雨")
                Text("謝謝你的愛")
                Text("我不夠愛你")
                Text("來生緣")
            }
            Text("當我遇上你")
            Text("寧願我傷心")
            Text("小情人")
            Text("峰迴路轉")
            Text("十七歲")
            Text("等你忘了我是誰")
 
        }
    }
}

ps: 除了使用 Select Column Down 選取，我們也可以用滑鼠選取想搬移的程式，然後再按快速鍵搬移。

調整 modifier 的順序

SwiftUI 的 modifier 順序十分重要，順序不對將造成奇怪的畫面或是紅色的 compile 錯誤，因此我們常在調整 modifier 的順序，比方以下程式 的 rotationEffect 在 background 之前，因此變成先旋轉再加上背景顏色，造成文字被切到。

Text("不說出口的第三個生日願望，並不會比較容易實現，而是，這個願望，就算不能實現，你也不會因此感到受傷。")
        .foregroundColor(.white)
        .rotationEffect(.degrees(10))
        .background(LinearGradient(gradient: Gradient(colors: [Color.red, Color.blue]), startPoint: .leading, endPoint: .trailing))

我們應該要先 background 再 rotationEffect，讓它先加背景顏色再旋轉。因此我們可將滑鼠游標停在 rotationEffect 上，然後點選 move line down 的快速鍵 cmd + option + ] 讓它下移一行。

Text("不說出口的第三個生日願望，並不會比較容易實現，而是，這個願望，就算不能實現，你也不會因此感到受傷。")
            .foregroundColor(.white)
            .background(LinearGradient(gradient: Gradient(colors: [Color.red, Color.blue]), startPoint: .leading, endPoint: .trailing))
            .rotationEffect(.degrees(10))
