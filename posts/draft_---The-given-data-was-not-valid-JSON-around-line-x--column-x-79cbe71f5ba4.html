解析 The given data was not valid JSON，around line x, column x

當我們使用 JSONDecoder 解析 JSON 時，時常會遇到 Decodable 型別造成的解碼失敗問題。

不過在檢查 Decodable 型別是否有問題時，檢查先檢查 JSONDecoder 印出的解碼錯誤。

import Foundation

let data = """
["笑傲江湖",
"射雕英雄傳", a"神雕俠侶"]
""".data(using: .utf8)!

do {
    let array = try JSONDecoder().decode([String].self, from: data)
} catch {
    print(error)
}

記得先印出抓到的資料，檢查它是否是格式正確的 JSON。若是錯

dataCorrupted(Swift.DecodingError.Context(codingPath: [], debugDescription: "The given data was not valid JSON.", underlyingError: Optional(Error Domain=NSCocoaErrorDomain Code=3840 "Invalid value around line 2, column 19." UserInfo={NSDebugDescription=Invalid value around line 2, column 19., NSJSONSerializationErrorIndex=36})))
