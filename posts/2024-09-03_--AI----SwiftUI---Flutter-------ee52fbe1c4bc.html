請 AI 美化 SwiftUI / Flutter 設計的畫面

最近彼得潘發現 AI 也能幫我們美化 SwiftUI / Flutter 設計的 App 畫面，以下我們以 Zed AI 為例，請它美化 SwiftUI 設計的角色畫面 CharacterView。

Zed IDE 的基本操作和 AI 功能初體驗
最近 Zed IDE 推出了強大的 AI 功能，我們可以從 IDE 選擇喜好的 AI 助手協助開發程式，比方 Claude、GPT、Gemini、llama 等。這些 AI 大部分都要搭配 API key 跟付費才能使用，不過目前 Zed…medium.com

ps: 目前新的 AI 都有直接修改專案程式的功能，有興趣的朋友也可以試試用 Xcode、GitHub Copilot、ChatGPT App、Windsurf、Cursor 美化畫面。

以下為原版的程式。

import SwiftUI

struct CharacterView: View {
    let avatar: ImageResource
    let name: String
    
    var body: some View {
        VStack {
            Image(avatar)
                .resizable()
                .scaledToFill()
                .frame(width: 100, height: 100)
                .clipShape(.circle)
                .shadow(color: .primary.opacity(0.7), radius: 20)
            Text(name)
        }
    }
}

#Preview {
    CharacterView(avatar: .prince, name: "小王子")
}

請 AI 將畫面變得更漂亮。

AI 做了以下改進。

1. 增加了頭像的大小，使其更加突出。
2. 在頭像周圍添加白色邊框，增加了層次感。
3. 調整了陰影效果，使其看起來更自然。
4. 增加了名字的字體大小和粗細，使其更加醒目。
5. 為整個 View 添加了一個圓角矩形背景，並添加了輕微的陰影效果。

import SwiftUI

struct CharacterView: View {
    let avatar: ImageResource
    let name: String
    
    var body: some View {
        VStack(spacing: 12) {
            Image(avatar)
                .resizable()
                .scaledToFill()
                .frame(width: 120, height: 120)
                .clipShape(.circle)
                .overlay {
                    Circle()
                        .stroke(.white, lineWidth: 4)
                }
                .shadow(color: .black.opacity(0.3), radius: 10, x: 0, y: 5)
            
            Text(name)
                .font(.title2)
                .fontWeight(.semibold)
                .foregroundStyle(.primary)
        }
        .padding()
        .background {
            RoundedRectangle(cornerRadius: 20)
                .fill(.white.opacity(0.8))
                .shadow(color: .black.opacity(0.1), radius: 15, x: 0, y: 5)
        }
    }
}

#Preview {
    CharacterView(avatar: .prince, name: "小王子")
}

請 AI 將畫面變得更為活潑可愛。

AI 做了以下改進。

1. 添加了一個淡黃色的圓形背景，並使其輕微放大縮小。
2. 為頭像添加了輕微的3D旋轉效果。
3. 使用了更圓潤的字體，並添加了陰影效果。
4. 將背景改為薄荷色，並添加了粉色邊框。
5. 使用了更鮮艷的顏色，如紫色和粉色。
6. 添加了簡單的動畫效果，使整個視圖看起來更加生動。

import SwiftUI

struct CharacterView: View {
    let avatar: ImageResource
    let name: String
    @State private var isAnimating = false
    
    var body: some View {
        VStack(spacing: 15) {
            ZStack {
                Circle()
                    .fill(.yellow.opacity(0.3))
                    .frame(width: 140, height: 140)
                    .scaleEffect(isAnimating ? 1.1 : 1.0)
                
                Image(avatar)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 120, height: 120)
                    .clipShape(.circle)
                    .overlay {
                        Circle()
                            .stroke(.white, lineWidth: 4)
                            .shadow(color: .black.opacity(0.1), radius: 5)
                    }
                    .rotation3DEffect(.degrees(isAnimating ? 5 : 0), axis: (x: 0, y: 1, z: 0))
            }
            .shadow(color: .pink.opacity(0.3), radius: 10, x: 0, y: 5)
            
            Text(name)
                .font(.system(size: 24, weight: .bold, design: .rounded))
                .foregroundStyle(.purple)
                .shadow(color: .pink.opacity(0.5), radius: 2, x: 0, y: 2)
        }
        .padding(20)
        .background {
            RoundedRectangle(cornerRadius: 25)
                .fill(.mint.opacity(0.2))
                .overlay {
                    RoundedRectangle(cornerRadius: 25)
                        .stroke(.pink.opacity(0.5), lineWidth: 2)
                }
        }
        .shadow(color: .gray.opacity(0.2), radius: 15, x: 0, y: 10)
        .onAppear {
            withAnimation(.easeInOut(duration: 1.5).repeatForever(autoreverses: true)) {
                isAnimating = true
            }
        }
    }
}

#Preview {
    CharacterView(avatar: .prince, name: "小王子")
}
