SwiftUI 讓 view 變圓角的 clipShape & background(_:in:fillStyle:)

利用 clipShape 設定圓角

利用 clipShape(_:style:) 將 view 裁切成某個形狀。

  • 圓角矩形。
struct ContentView: View {
    var body: some View {
        Image(.peter)
            .resizable()
            .scaledToFill()
            .frame(width: 200, height: 200)
            .clipShape(.rect(cornerRadius: 20))
    }
}
struct ContentView: View {
    var body: some View {
        Text("""
十年生死兩茫茫！不思量，自難忘。
千里孤墳，無處話淒涼。
縱使相逢應不識，塵滿面，鬢如霜。
夜來幽夢忽還鄉，小軒窗，正梳妝。
相顧無言，惟有淚千行。
料得年年斷腸處，明月夜，短松岡。
""")
        .padding()
        .background(.yellow)
        .clipShape(.rect(cornerRadius: 20))
    }
}
  • 膠囊形。
struct ContentView: View {
    var body: some View {
        Image(.peter)
            .resizable()
            .scaledToFill()
            .frame(width: 200, height: 100)
            .clipShape(.capsule)
    }
}
  • 圓形。

  • 圓角矩形。

    var body: some View {

struct ContentView: View {
    var body: some View {
        Text("""
