#209 串接萌典 API 開發字典 App

最近發現 g0v 開發了很萌的中文詞典萌典，萌典是一個基於台灣教育部《重編國語辭典修訂本》的開放資料字典平台，收錄了十六萬筆國語、兩萬筆臺語、一萬四千筆客語條目。

萌典 - 教育部國語、臺語、客語辭典民間版
共收錄十六萬筆國語、兩萬筆臺語、一萬四千筆客語條目，每個字詞都可以輕按連到說明，並提供 Android 及 iOS 離線…www.moedict.tw
‎萌典App
Read reviews, compare customer ratings, see screenshots and learn more about 萌典App. Download 萌典App and enjoy it on your…apps.apple.com

萌典也提供了免費串接的 API，提供七個查詢資料的端點，方便我們利用它開發有趣的 App。

詳細的資訊可參考官方的 GitHub 連結。

GitHub - g0v/moedict-webkit: 萌典網站
