保護使用者隱私的 App Tracking Transparency

從 iOS 14.5 開始，Apple 要求當 App 有追蹤使用者在其它公司 App 和網站的記錄，比方顯示個性化的廣告時， 必須徵求使用者的同意，顯示以下畫面的 alert。

alert 訊息顯示 allow xxx to track your activity across other companies apps and websites (要允許 xxx 追蹤你在其他公司 App 和網站的記錄嗎)。

接下來我們將示範透過 framework App Tracking Transparency 顯示以上 alert 的文字內容。

切換到 App 的 Info 頁面

在 Info 頁面新增 Privacy — Tracking Usage Description，輸入說明文字

比方輸入以下文字。

your data will be used to deliver personalized ads to you

呼叫 ATTrackingManager 的 requestTrackingAuthorization 顯示請求權限的 alert

記得要先 import AppTrackingTransparency。

ATTrackingManager.requestTrackingAuthorization { status in
            switch status {
            case .notDetermined:
                break
            case .restricted:
                break
            case .denied:
                break
            case .authorized:
                break
            @unknown default:
                break
            }
}

完整範例

import SwiftUI
import AppTrackingTransparency
struct ContentView: View {
    
    func requestTracking() {
        ATTrackingManager.requestTrackingAuthorization { status in
            switch status {
            case .notDetermined:
                break
            case .restricted:
                break
            case .denied:
                break
            case .authorized:
                break
            @unknown default:
                break
            }
        }
    }
    var body: some View {
        Image("cs193p")
            .resizable(resizingMode: .stretch)
            .aspectRatio(contentMode: .fit)
            .onAppear(perform: {
                requestTracking()
            })
    }
}
