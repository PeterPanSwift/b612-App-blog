在 Xcode playground 加入客製字型

將字體檔 ttf 或 otf 加到 playground 的 Resources 下

playground 程式範例

import UIKit
let fontURL = Bundle.main.url(forResource: "crayon_1-1", withExtension: "ttf")! as CFURL
CTFontManagerRegisterFontsForURL(fontURL, .process, nil)
let label = UILabel()
label.font = UIFont(name: "Nagurigaki-Crayon", size: 40)
label.text = "全集中呼吸"
label.textColor = .orange
label.sizeToFit()
label

說明

let fontURL = Bundle.main.url(forResource: "crayon_1-1", withExtension: "ttf")! as CFURL
CTFontManagerRegisterFontsForURL(fontURL, .process, nil)

利用 Bundle.main.url 取得字型檔的 url，然後呼叫 CTFontManagerRegisterFontsForURL 註冊字體，之後即可在程式裡使用此字體。

CTFontManagerRegisterFontsForURL(_:_:_:) | Apple Developer Documentation
