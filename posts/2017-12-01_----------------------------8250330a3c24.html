懶得打兩個 &，不如用逗號串接 ，判斷多個條件同時成立

使用 if 或 while 判斷多個條件同時成立時，我們習慣搭配 &&，例如以下例子

var age = 18
var weight = 45
if age < 30 && weight < 60 {
    print("原來你是我最想留住的幸運")
}

不過如果你很懶，連兩個 & 都懶得打，沒關係，你可以用小小的一個逗號串接 !

var age = 18
var weight = 45
if age < 30, weight < 60 {
    print("原來你是我最想留住的幸運")
}
