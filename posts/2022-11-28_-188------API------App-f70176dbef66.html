#188 串接電商 API 開發電商 App

最近在 twitter 看到 azamsharp 大大使用有趣的電商 API，因此研究了一下，發現網路上有許多免費的電商 API，方便我們練習開發電商 App。有興趣的朋友可從以下 API 挑個喜歡的，開始製作賺不了錢的電商 App。(不過說不定可以幫助你找到工作成為 App 工程師)

  • Platzi Fake Store API (推薦)
  • Rainforest API — Amazon product (推薦)
  • BlueCart API — Walmart product (推薦)
  • Fake Store API
  • Fake E-commerce API
  • eCommerce Rest API

Platzi Fake Store API (推薦)

Platzi Fake Store API
platzi Fake Store APIfakeapi.platzi.com

特點

  • product，user，category 三種資料類型。
  • 新增 / 更新 / 刪除 product & category。
  • 註冊登入。
  • 分頁功能。
  • 圖片上傳。
