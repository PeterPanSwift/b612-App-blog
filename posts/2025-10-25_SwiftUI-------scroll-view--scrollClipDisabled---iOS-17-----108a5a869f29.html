SwiftUI 讓內容超過 scroll view 的scrollClipDisabled — iOS 17 新功能

開發 iOS App 時，我們常常會利用 scroll view 讓畫面可以捲動，內容再多也裝得下。

比方以下例子利用 scroll view 顯示冰島男孩寫程式的照片集。

struct ContentView: View {

    var body: some View {
        ScrollView(.horizontal) {
            HStack(spacing: 20) {
                ForEach(1..<5, id: \.self) { number in
                    Image("iceland\(number)")
                        .resizable()
                        .scaledToFill()
                        .frame(width: 150, height: 150)
                        .clipped()
                }
            }
        }
    }
}

當 scroll view 顯示內容時，預設超出 scroll view 的範圍會被切除，不過有時我們會希望內容完整顯示。

此時我們可利用 modifier scrollClipDisabled (iOS 17 限定)實現，以下我們示範兩個例子。

scrollClipDisabled(_:) | Apple Developer Documentation
Sets whether a scroll view clips its content to its bounds.developer.apple.com
  • 保留被切除的陰影跟旋轉區塊。

當圖片加了陰影和旋轉後，預設它會被 ScrollView 切除，因此看不到上下的陰影和旋轉的區塊。

struct ContentView: View {
    
    var body: some View {
        ScrollView(.horizontal) {
            HStack(spacing: 20) {
                ForEach(1..<5, id: \.self) { number in
                    Image("iceland\(number)")
                        .resizable()
                        .scaledToFill()
                        .frame(width: 150, height: 150)
                        .clipped()
                        .rotationEffect(.degrees(5))
                        .shadow(color: .blue.opacity(0.8), radius: 20)
                }
            }
        }
    }
}

從 ScrollView 呼叫 scrollClipDisabled，圖片的陰影和旋轉區塊可以完整顯示了 !

struct ContentView: View {
    
    var body: some View {
        ScrollView(.horizontal) {
            HStack(spacing: 20) {
                ForEach(1..<5, id: \.self) { number in
                    Image("iceland\(number)")
                        .resizable()
                        .scaledToFill()
                        .frame(width: 150, height: 150)
                        .clipped()
                        .rotationEffect(.degrees(5))
                        .shadow(color: .blue.opacity(0.8), radius: 20)
                }
            }
        }
        .scrollClipDisabled()
    }
}
  • 讓 scroll view 的右邊不會因為 padding 留白。

當畫面加了 padding 時，畫面的邊緣會多出空白區塊，不過在呈現水平捲動的內容時，通常我們會希望它可以往右延伸到螢幕的邊界，右邊不要留白。

struct ContentView: View {

    var body: some View {
        VStack {
            Text("冰島大冒險，在十個地方用 MacBook 寫程式")
                .font(.largeTitle.bold())
            ScrollView(.horizontal) {
                HStack(spacing: 20) {
                    ForEach(1..<5, id: \.self) { number in
                        Image("iceland\(number)")
                            .resizable()
                            .scaledToFill()
                            .frame(width: 150, height: 150)
                            .clipped()
                    }
                }
            }
        }
        .padding()
    }
}

加了 scrollClipDisabled，scroll view 的右邊不再留白了。

struct ContentView: View {

    var body: some View {
