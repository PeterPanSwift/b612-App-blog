讓 curl 抓取的 JSON 整齊縮排的 json_pp

開發 iOS App 時我們時常使用 curl 或 PostMan 測試後台的 API，比起需要另外安裝的 PostMan，curl 的好處是可以直接打開 Terminal 測試。不過它預設顯示的 JSON 資料卻有點慘不忍睹，一般人的肉眼很難看懂 JSON 的內容。

比方以下我們以 curl 串接聖鬥士星矢的 API 抓取星矢的資料，畫面如下，能夠一眼看懂 JSON 的應該智商有 180 以上。

Saint Seiya API
The characters! The cloths! The saints! Saint Seiya APIdiegochagas.com

那如果我們智商只有 170，沒有 180 呢 ?

沒關係，我們可以用 json_pp 指令將 curl 抓取的 JSON 變成整齊縮排。(ps: json_pp 前的 | 是將 curl 的結果輸出給 json_pp 指令的意思)

curl https://saint-seiya-api.herokuapp.com/api/character/379 | json_pp

有了 json_pp，curl 抓取的 JSON 變得比較平易近人了 ~
