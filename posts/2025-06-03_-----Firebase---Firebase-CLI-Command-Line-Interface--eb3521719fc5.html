方便管理 Firebase 的 Firebase CLI(Command Line Interface)

透過 Firebase CLI 我們可以方便地從 Terminal 下指令管理 Firebase 專案和部署網站。以下彼得潘簡單介紹如何在 macOS & Windows 安裝 Firebase CLI。

Firebase CLI reference | Firebase Documentation
Build AI-powered mobile and web apps and features with the Gemini and Imagen models using Firebase AI Logicfirebase.google.com

Mac

從 Terminal 輸入以下指令安裝 Firebase CLI。

curl -sL https://firebase.tools | bash

在 Terminal 輸入以下指令登入 Firebase。

firebase login

i Firebase optionally collects CLI and Emulator Suite usage and error reporting information to help improve our products. Data is collected in accordance with Google’s privacy policy (https://policies.google.com/privacy) and is not used to identify you.

? Allow Firebase to collect CLI and Emulator Suite usage and error reporting information? (Y/n)

開啟網頁登入 Google 帳號。

登入成功後回到 Terminal，畫面上顯示

✔ Success! Logged in as iosappneverland@gmail.com

系統將記得登入的狀態，之後再輸入 firebase login 將顯示已登入的訊息。

firebase login

Already logged in as iosappneverland@gmail.com

之後若要登出，可輸入以下指令。

firebase logout

Windows

從以下連結下載 nvm-setup.exe。

Releases · coreybutler/nvm-windows
A node.js version management utility for Windows. Ironically written in Go. - Releases · coreybutler/nvm-windowsgithub.com

點擊 nvm-setup.exe 安裝 nvm。

在 Terminal 輸入以下指令安裝最新版的 NodeJS。

nvm install latest

使用剛剛安裝的 NodeJS。

nvm use latest

安裝 firebase-tools。

npm.cmd install -g firebase-tools

ps: 輸入 npm install -g firebase-tools 會出現錯誤 npm.ps1 cannot be loaded because running scripts is disabled on this system.，其中一種解法是改成 npm.cmd。

有興趣的朋友也可參考以下官方完整的安裝說明。

Firebase CLI reference | Firebase Documentation
Build AI-powered mobile and web apps and features with the Gemini and Imagen models using Firebase AI Logicfirebase.google.co

在 Terminal 輸入以下指令登入 Firebase。

firebase.cmd login

登入成功後回到 Terminal，畫面上顯示

✔ Success! Logged in as iosappneverland@gmail.com

系統將記得登入的狀態，下次在 Terminal 執行 firebase.cmd login 時將顯示 Already logged in。
