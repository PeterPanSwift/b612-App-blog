調整 UISegmentedControl 元件的字型

想調整 UISegmentedControl 元件的字型大小，不能從 Interface Builder，只能從程式控制。

@IBOutlet weak var segmentedControl: UISegmentedControl!

override func viewDidLoad() {
    super.viewDidLoad()
    segmentedControl.setTitleTextAttributes([.font: UIFont.systemFont(ofSize: 40) ], for: .normal)
}

透過 function setTitleTextAttributes(_:for:)，我們可以控制 UISegmentedControl 的字型，文字顏色等。

setTitleTextAttributes(_:for:) - UISegmentedControl | Apple Developer Documentation
The attributes dictionary can specify the font, text color, text shadow color, and text shadow offset for the title in…developer.apple.com

不過有一點要特別注意，如果字體很大，UISegmentedControl 採用預設高度的話，顯示時將如下圖所示，文字超過 UISegmentedControl 的框框。

因此，請記得幫 UISegmentedControl 加上高度的條件，讓它長高，才能裝得下文字。

ps: 在 Interface Builder，UISegmentedControl 的高度只能用 Auto Layout 調整，無法拖曳縮放或編輯 Size Inspector 的 Height。
