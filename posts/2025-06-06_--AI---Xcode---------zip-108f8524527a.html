請 AI 將 Xcode 專案程式壓縮成 zip

常常有同學問彼得潘問題，需要將專案程式傳給彼得潘參考。除了分享 GitHub 連結，另一個較簡單的方法是將專案資料夾壓縮成 zip 後傳給彼得潘。

不過時常有同學壓縮 zip 時壓錯檔案，而且手動壓縮也滿辛苦的。身在 AI 橫行的年代，彼得潘想到可以請 AI 幫忙，程式有問題時請 AI 將專案壓縮成 zip 和傳給彼得潘。

ps: 目前 AI 還不會用 LINE 傳檔。如果想立即從 LINE 傳給彼得潘，請自己打開 LINE 傳送 AI 壓縮的 zip 檔。

以下我們使用 GitHub Copilot for Xcode 說明。

請 AI 壓縮專案

用 Xcode 打開專案，打開 GitHub Copilot for Xcode，切換成 Agent 模式。

輸入以下 prompt。

zip project

點選 Continue。

專案成功壓縮成 zip 檔。

請 AI 寄信

彼得潘在 Mac 實測後，目前請 AI 寄信比較可行的做法是用 AppleScript 控制 Mail App 寄信。

use AppleScript to send zip file to apppeterpan@gmail.com via Mail

VS Code 第一次打開 Mail App 時會出現以下視窗，請點選 Allow。

Mail App 成功寄出包含 zip 的檔案。
