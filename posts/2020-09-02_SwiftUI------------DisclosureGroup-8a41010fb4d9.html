SwiftUI 擅長展開 & 折疊的 DisclosureGroup

使用 iOS App 時，我們時常看到可以展開折疊的頁面設計。Apple 在 iOS 14 推出了專門實現展開(expand) & 折疊(collapse)功能的 DisclosureGroup，接下來就讓我們以彼得潘喜歡的推理小說家為例看它表演吧。

DisclosureGroup | Apple Developer Documentation
A view that shows or hides another content view, based on the state of a disclosure control.developer.apple.com

標題為文字的 DisclosureGroup

struct ContentView: View {
    let authors = ["伊坂幸太郎", "乙一", "東野圭吾", "湊佳苗", "島田莊司", "宮部美幸"]
    
    var body: some View {
        DisclosureGroup("日本推理作家") {
            ForEach(authors, id: \.self) { author in
                Text(author)
            }
            
        }
        .padding()
    }
}

建立 DisclosureGroup 時採用以下的 init，參數 label 傳入標題顯示的文字，參數 content 傳入展開時顯示的內容。

init<S>(_ label: S, @ViewBuilder content: @escaping () -> Content) where S : StringProtocol

若想讓展開的內容靠左對齊，則可使用 VStack 搭配 alignment 裝 DisclosureGroup 展開的內容。

struct ContentView: View {
    let authors = ["伊坂幸太郎", "乙一", "東野圭吾", "湊佳苗", "島田莊司", "宮部美幸"]
    
    var body: some View {
        DisclosureGroup("日本推理作家") {
            VStack(alignment: .leading) {
                ForEach(authors, id: \.self) { author in
                    Text(author)
                }
            }
        }
        .padding()
    }
}

以 view 當標題

DisclosureGroup 的標題不一定是文字，我們可以傳入任何的 view，以下例子以包含圖文的 Label 當標題。

struct ContentView: View {
    let authors = ["伊坂幸太郎", "乙一", "東野圭吾", "湊佳苗", "島田莊司", "宮部美幸"]
    
    var body: some View {
        DisclosureGroup {
            VStack(alignment: .leading) {
                ForEach(authors, id: \.self) { author in
                    Text(author)
                }
            }
        } label: {
            Label("日本推理作家", systemImage: "text.book.closed.fill")
        }
        .padding()
    }
}

建立 DisclosureGroup 時採用以下的 init。

init(content: @escaping () -> Content, label: () -> Label)

設定 DisclosureGroup 是否展開

DisclosureGroup 預設會害羞地折疊起來，不過我們也可以透過參數 isExpanded 控制它預設的展開狀態，比方以下例子的 DisclosureGroup 一開始是展開的。

struct ContentView: View {
    @State private var isExpanded = true
    let authors = ["伊坂幸太郎", "乙一", "東野圭吾", "湊佳苗", "島田莊司", "宮部美幸"]
    
    var body: some View {
        DisclosureGroup(isExpanded: $isExpanded) {
            VStack(alignment: .leading) {
                ForEach(authors, id: \.self) { author in
                    Text(author)
                }
            }
        } label: {
            Label("日本推理作家", systemImage: "text.book.closed.fill")
        }
        .padding()
    }
}

多層的 DisclosureGroup

DisclosureGroup 可以多層，我們可在 DisclosureGroup 裡包 DisclosureGroup，比方第一層的 DisclosureGroup 展開將顯示作家清單，點選作家將展開第二層的 DisclosureGroup，顯示他的作品清單。

struct Author {
    let name: String
    let books: [String]
}
struct ContentView: View {
    let authors = [
        Author(name: "伊坂幸太郎", books: ["孩子們", "家鴨與野鴨的投幣式置物櫃", "重力小丑", "奧杜邦的祈禱"]),
        Author(name: "乙一", books: ["夏天、煙火、我的屍體", "GOTH斷掌事件", "暗黑童話"])
    ]
    
    var body: some View {
        DisclosureGroup {
            VStack(alignment: .leading) {
                ForEach(authors, id: \.name) { author in
                    DisclosureGroup(author.name) {
                        VStack(alignment: .leading) {
                            ForEach(author.books, id: \.self) { book  in
                                Text(book)
                            }
                        }
                    }
                }
            }
            .padding()
        } label: {
            Label("日本推理作家", systemImage: "text.book.closed.fill")
        }
        .padding()
    }
}

在 List / Form 裡使用 DisclosureGroup

List / Form 很適合搭配 DisclosureGroup，比方以下例子的 List 可展開顯示日本 & 英國的推理作家。

struct ContentView: View {
    let japanAuthors = ["伊坂幸太郎", "乙一", "東野圭吾", "湊佳苗", "島田莊司", "宮部美幸"]
    let ukAuthors = ["克莉絲蒂", "柯南·道爾", "狄克森·卡爾", "G·K·切斯特頓", "多蘿西·L·塞耶斯", "約瑟芬·鐵伊"]
    
    var body: some View {
        List{
            DisclosureGroup("日本推理作家") {
                ForEach(japanAuthors, id: \.self) { author in
                    Text(author)
                }
            }
            DisclosureGroup("英國推理作家") {
                ForEach(ukAuthors, id: \.self) { author in
                    Text(author)
                }
                
            }
        }
    }
}
