SwiftUI ForEach 搭配 array 的自動完成問題

Xcode 提供了貼心的自動完成，ForEach 搭配 array 會出現如下圖的自動完成，ForEach(books) { book。

struct Book: Identifiable {
    let id = UUID()
    let name: String
    let author: String
}

而且從 Xcode 16 開始，predictive code completion 將提供更強大的自動完成。

Xcode 聰明的 AI 自動完成 predictive code completion — Xcode 16 新功能
從 Xcode 16 開始，Xcode 推出神奇的 AI 自動完成 predictive code completion，它能聰明地預測提示我們可能輸入的程式，以下我們一步步介紹和示範它的各種用法。medium.com

不過如果沒有 predictive code completion，請注意以下 ForEach 自動完成的限制。

  • array 的最後一個成員不能加逗號。
  • array 的成員是自訂型別的 struct 才有自動完成。

array 的最後一個成員不能加逗號

最後一個成員只不過加了小小的逗號，自動完成就消失了。

let books = [
    Book(name: "小王子", author: " 安東尼‧聖修伯里"),
    Book(name: "彼得潘", author: "詹姆斯‧巴利"),
]

array 的成員是自訂型別的 struct 才有自動完成

  • array 的成員是 class 定義的物件時沒自動完成。
class Book: Identifiable {
    let id = UUID()
    let name: String
    let author: String
    
    init(name: String, author: String) {
        self.name = name
        self.author = author
    }
}
  • 字串、數字的 array 沒有自動完成。
  • enum array 沒有自動完成。
enum Animal: CaseIterable {
    case cat, dog, fox, rabbit
}
  • 內建型別沒有自動完成。
