解決 SPM 的套件版本問題

當我們下載編譯搭配 SPM 套件的 Xcode 專案時，有時會出現以下的紅色錯誤。

Package Loading Group
: The source control operation was canceled.
x-xcode-log://0C0568D9-598D-4E26-8315-2D959418DF4F Missing package product 'Alamofire', please fix package resolution errors before building

別緊張，遇到 SPM 的套件版本問題，大部分都可透過 File > Swift Packages > Resolve Package Versions 解決。

Resolve Package Versions 後如果還有問題，可以再試試 Product > Build(cmd + B) ，通常問題都能迎刃而解。
