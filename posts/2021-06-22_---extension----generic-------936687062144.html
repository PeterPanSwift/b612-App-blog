利用 extension 擴充 generic 型別的寫法

假設我們定義了 generic 型別 LittlePrince，型別代號是 T。

struct LittlePrince<T> {
    var pet: T?
}

利用 extension 擴充 generic 型別

小王子將得到 function eat，不管他養了哪一種寵物。

extension LittlePrince {
    func eat() {
        
    }
}

限制型別代號遵從某個 protocol 或繼承某個類別

只有養到可愛寵物時，小王子才能得到 function play。

protocol Cute {
   func dance()
}
extension LittlePrince where T: Cute {
    func play() {
        
    }
}

限制型別代號是某個型別

只有養到貓時，小王子才能得到 function play。

struct Cat {
    func 喵喵喵() {
        
    }
}
extension LittlePrince where T == Cat {
    
    func sleep() {
        
    }
}

除了剛剛的寫法，我們也可以用 typealias 定義新的型別名稱，然後將 extension 加在它身上。

typealias LittlePrinceWithCat = LittlePrince<Cat>
extension LittlePrinceWithCat {
    
    func sleep() {
        
    }
}
