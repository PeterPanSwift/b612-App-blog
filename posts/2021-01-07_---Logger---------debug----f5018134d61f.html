利用 Logger 印出包含時間的 debug 訊息

Apple 在 iOS 14 推出更方便列印訊息的 Logger 物件，它有許多比 print 厲害的地方，有興趣的可參考以下 Apple 官方的連結。

Apple Developer Documentation
Edit descriptiondeveloper.apple.com

在此彼得潘想分享它跟 print 一個最明顯的差異，它印出訊息時將包含時間資訊，因此若想知道訊息何時印出，Logger 將是不錯的選擇。

import UIKit
import os

class ViewController: UIViewController {
  let logger = Logger()
   
  override func viewDidLoad() {
        super.viewDidLoad()
        logger.log("viewDidLoad")
        print("viewDidLoad")
  }
}

import os 後才能使用 Logger。產生 Logger 物件後，呼叫它的 log 即可列印。

結果

logger 印出的訊息包含時間，print 印出的訊息不包含時間。
