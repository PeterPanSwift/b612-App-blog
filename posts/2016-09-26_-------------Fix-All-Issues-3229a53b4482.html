一次修正所有警告錯誤 ! Fix All Issues

程式設計師最討厭兩種顏色，紅色錯誤和黃色警告。一般來說，他們都很有耐心，願意將錯誤警告一個個清除地一乾二凈 ! 不過如果你是一秒幾十萬上下的 App 工程師，一定不能錯過 Xcode 的 Fix All Issues ，一鍵修正大法 ! 接下來，就讓我們看看實際的操作步驟。

1 以下程式碼很不幸地有許多紅色錯誤，但也很幸運地，Xcode 都可以幫我們修正，所以它顯示帶著白色圓心的紅色圓圈。

比較笨的方法，是自己一個個手動修正。稍微聰明一點的，可點選程式碼旁的黃色三角形警告或紅色圓形錯誤後，點選 Fix 修正。

最聰明的方法，其實藏在 Editor 選單裡。點選 Editor -> Fix All Issues，即可將所有警告錯誤，殺得片甲不留 ! (快速鍵 cmd + option + ctrl + f )
