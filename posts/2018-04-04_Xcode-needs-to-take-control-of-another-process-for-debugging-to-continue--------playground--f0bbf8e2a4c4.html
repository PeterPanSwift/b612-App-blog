Xcode needs to take control of another process for debugging to continue — 無法使用 playground 或模擬器的權限問題

當 Mac 上有多個帳號時，如果以不是 Admin 權限的帳號登入，在使用 Xcode 的 playground 或模擬器時，往往會出現以下視窗，要求我們輸入帳號密碼。

此時我們必須輸入在 Developer Tools group 下的使用者的帳號密碼。( Enter the name and password of a user in the “Developer Tools” group to allow this)

預設只有 Admin 權限的使用者屬於 Developer Tools group，所以沒有 Admin 權限的使用者將被排擠，你的帳號密碼是沒有用的 !

沒關係，只要送禮物給 Admin 權限的使用者，請他把你的帳號加到 Developer Tools group，即可順利開發 Swift App，不會再出現剛剛的視窗。

什麼 ? 他不知道怎麼把你加到 Developer Tools group ? 好吧，除了送他巧克力，也把指令一起貼給他吧，只要請他在 terminal 輸入以下指令即可，最後的 wendy 為想新增到 Developer Tools group 的使用者的帳號。

sudo dscl . append /Groups/_developer GroupMembership wendy
