Swift type property，type method ＆ enum 的型別省略

使用 Swift 的 type property，type method ＆ enum 時，型別省略可以讓我們的手指輕鬆不少，少打一些字。

比方以下是沒有型別省略的寫法。

let color = UIColor.red
let number = Int.random(in: 1...6)
enum Pet {
    case cat, dog, rabbit
}
let myPet = Pet.cat

當我們從型別 A 存取 type property，type method & enum case 時，若是已知是型別 A，而且程式執行的結果也是型別 A，則可省略型別。

因此剛剛的程式可改寫成型別省略的版本。

let color: UIColor = .red
let number: Int = .random(in: 1...6)
enum Pet {
    case cat, dog, rabbit
}
let myPet: Pet = .cat

例子裡的 color，number & myPet 已知型別是 UIColor，Int & Pet，而 UIColor.red，Int.random(in: 1…6) & Pet.cat 得到的資料型別也是 UIColor，Int & Pet，所以型別可以省略。

若是未知型別時，省略型別將產生問題。

在設定 property 或傳入參數時，也常使用型別省略，比方以下例子。

  • 設定 property backgroundColor，已知型別為 UIColor。
let redView = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
redView.backgroundColor = .red
  • 傳入參數 rotationAngle，已知型別為 CGFloat。
let redView = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
redView.transform = CGAffineTransform(rotationAngle: .pi / 4)
