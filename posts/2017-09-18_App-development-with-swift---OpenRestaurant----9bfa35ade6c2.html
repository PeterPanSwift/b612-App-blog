App development with swift 的 OpenRestaurant 問題

app development with swift 第五章的 Guided Project 將實作出串接後台，讀取上傳資料的功能。串接後台不難，難的是我們必須先有一個後台程式。還好，Apple 早幫我們寫好了後台 App，只要在 Mac 上點兩下 OpenRestaurant App 啟動即可。不過令人難過的，它似乎有 bug，當我們的 App 連線到 http://localhost:8090/categories 時，只能得到一個空的 array。

還好，網路上的大大幫我們寫了一份 OpenRestaurant.py，只要將它下載放在目錄Restaurant 下，在 terminal 執行 python OpenRestaurant.py 即可。

ps:

1 OpenRestaurant.py 只是一個讀取 menu.json 和處理 get, post API 的後台程式，有興趣的朋友也可以另外用 ruby, swift, php 等程式語言自己實作。

2. 可以另外修改 json，比方設定圖片，改變後台回傳的資料。
