請 Germini 寫 Python 程式 & 執行程式，然後將程式輸出到 Colab

最近玩了一下 Google Germini，發現它跟 GPT 一樣，都是厲害的程式高手，可以幫我們撰寫程式。而且當他寫的是 Python 時，它還有以下厲害的地方。

  • 執行程式的能力。
  • 將程式輸出到 Colab 或 Replit。

以下彼得潘以 zerojudge 上的題目為例，請 Germini 一邊寫 Python 解題，一邊展示它厲害的地方。

題目

a053. Sagit's 計分程式 - 高中生程式解題系統
sagit 是一位高中電腦老師，這學期正在教學生寫C++程式。他的評分標準是依照每一位學生在 ZeroJudge…zerojudge.tw

請 Germini 寫程式解題 & 執行程式

請 Germini 解題，我們將網頁上的題目原封不動地複製貼上。

Germini 的回答如下。

當然可以，以下是 Python 程式：

def score(n):
  """
  計算學生在 ZeroJudge 系統上解出的題數的得分。

  Args:
    n: 學生在 ZeroJudge 系統上解出的題數。

  Returns:
    學生的得分。
  """
  if n <= 10:
    return n * 6
  elif n <= 20:
    return 60 + (n - 10) * 2
  elif n <= 40:
    return 80 + (n - 20) * 1
  else:
    return 100

if __name__ == "__main__":
  n = int(input("請輸入學生在 ZeroJudge 系統上解出的題數："))
  print(score(n))

為了檢查程式是否正確，Germini 還會執行程式比對結果。如下圖所示，score(10) 等於 60，score(40) 等於 100，答案跟題目的 Sample Output 一模一樣。

將 Python 輸出到 Colab 或 Replit

Germini 也可以把 Python 程式輸出到 Colab 或 Replit，方法如下。

點選下方的 export(輸出) 按鈕。

點選 Export to Colab。

點選 Open Colab。

Germini 貼心地在 Colab 建立新的筆記本，內容為剛剛回答的 Python 程式，方便我們之後在 Colab 進行測試和修改。
