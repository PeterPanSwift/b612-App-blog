生活中的 if (如果)實例練習

模仿以下 Swift if 的格式，從生活中的例子找靈感，在 Medium 發表創作, 寫愈多愈有趣愈好，比方如果下雨，我要走到馬路上淋雨。如果遇到虎克船長，我要把他抓起來餵鱷魚。

if 你是我的傳說 {
   print("走到你的面前")
   print("拿出彼得潘的書，假裝文青看書")
   print("等你搭訕")
}

AI 寫的其它例子。

if 今天沒有準時收到薪水 {
    print("打電話給老闆")
    print("問他是不是忘記匯款")
    print("如果他說沒錢,就要求支票或現金")
}
if 在路上看到一隻可愛的小狗 {
    print("慢慢走近牠")
    print("蹲下來對牠微笑")
    print("看牠的反應,如果友善就試著摸摸牠")
}
if 發現冰箱裡的蛋糕不見了 {
    print("問問家人有沒有人偷吃")
    print("如果大家都說沒有")
    print("只好認為是蛋糕精靈偷吃的")
}
if 跟朋友出去卻忘記帶錢包 {
    print("尷尬地告訴朋友")
    print("請朋友先幫忙付錢")
    print("回家後馬上轉帳還錢給朋友")
}
if 班上最受歡迎的人跟你打招呼 {
    print("揉揉眼睛確定不是在做夢")
    print("驚喜地回應他的招呼")
    print("努力壓抑內心的激動,裝作若無其事")
}
