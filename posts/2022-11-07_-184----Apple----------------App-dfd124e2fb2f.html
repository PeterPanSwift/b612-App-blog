#184 利用 Apple 的甜甜圈範例素材製作有點甜的 App

最近研究了一下 Apple 的 Food Truck 範例，發現它的甜甜圈超可愛的。而且它的甜甜圈由三層圖片組成，我們可以任意打造自己喜歡的甜甜圈。

正在學習開發 iOS App 但是苦無素材嗎 ? 沒關係，我們可以使用 Apple 精心準備的素材，一邊吃甜甜圈，一邊開心地開發有點甜的 App。

在專案裡加入甜甜圈相關的圖片程式

點選以下連結的 Download 下載 Apple 的範例程式 Food Truck。

Food Truck: Building a SwiftUI multiplatform app | Apple Developer Documentation
Create a single codebase and app target for Mac, iPad, and iPhone.developer.apple.com

甜甜圈相關的圖片程式在資料夾 FoodTruckKit。請將下圖的 FoodTruckKit 拖曳到自己專案的 Project navigator 下。

套件 FoodTruckKit 成功出現在 Project navigator 下。

換到 TARGET App General 頁面的 Frameworks, Libraries, and Embedded Content 區塊，點選 + 加入 FoodTruckKit。

大功告成，之後我們可以開心地在 App 裡使用甜甜圈的相關圖片和程式了。

加入生成 UIImage 的 function

若是使用 UIKit & storyboard 開發 App，請在 Ingredient.swift 加入以下生成 UIImage 的 function。(ps: 使用 SwiftUI 開發 App 可跳過此步驟。)

public extension Ingredient {
    func uiImage(thumbnail: Bool) -> UIImage? {
        UIImage(named: "\(Self.imageAssetPrefix)/\(imageAssetName)-\(thumbnail ? "thumb" : "full")", in: .module, with: nil)
    }
}

FoodTruckKit 的甜甜圈圖片

FoodTruckKit 的甜甜圈圖片藏在 FoodTruckKit > Sources > Assets 裡，分成 dough(麵團)，glaze(糖皮) & topping(配料)三層。

dough(麵團)的可愛圖片。

glaze(糖皮)的可愛圖片。

topping(配料)的可愛圖片。

之後在程式裡，我們只要將 dough(麵團)，glaze(糖皮) & topping(配料)三層圖片疊起來，即可做出可愛的甜甜圈。

如何從程式讀取甜甜圈的相關資料

從 Donut.Dough.all，Donut.Glaze.all & Donut.Topping.all 可分別得到 dough，glaze & topping 的 array。在 Donut.Dough，Donut.Glaze & Donut.Topping 下則以型別屬性宣告每一種 dough，glaze & topping，比方 Donut.Glaze.blueberry 代表藍莓的糖皮。

利用 Ingredient 裡定義的 function image(thumbnail:) 或 uiImage(thumbnail:) 可生成 dough，glaze & topping 對應的圖片，比方 Donut.Glaze.blueberry.image(thumbnail: false) 可生成 Image，Donut.Glaze.blueberry.uiImage(thumbnail: false) 可生成 UIImage

如果有選擇障礙，懶得自己設計甜甜圈，我們也可從 Donut 的型別屬性讀取預先設計好的甜甜圈。

比方 blackRaspberry 的組成如下。

static let blackRaspberry = Donut(
    id: 14,
    name: String(localized: "Black Raspberry", bundle: .module, comment: "A donut-flavor name."),
    dough: .plain,
    glaze: .chocolate,
    topping: .blueberryDrizzle
)

從 Donut.all 可得到事先設計好的甜甜圈 array。

static let all = [
    classic, blueberryFrosted, strawberryDrizzle, cosmos, strawberrySprinkles, lemonChocolate, rainbow, picnicBasket, figureSkater,
    powderedChocolate, powderedStrawberry, custard, superLemon, fireZest, blackRaspberry, daytime, nighttime
]

產生甜甜圈圖片的程式範例

  • SwiftUI
import SwiftUI
import FoodTruckKit

struct ContentView: View {
    
    var body: some View {
        ZStack {
            Donut.Dough.plain.image(thumbnail: false)
                .resizable()
            Donut.Glaze.blueberry.image(thumbnail: false)
                .resizable()
            Donut.Topping.blueberryDrizzle.image(thumbnail: false)
                .resizable()
        }
        .scaledToFit()
        .padding()
    }
}

  • UIKit

Interface Builder。

假設上圖三個 image view 的 outlet 分別為 doughImageView，glazeImageView & toppingImageView，我們可透過以下程式設定圖片。

import UIKit
import FoodTruckKit

class ViewController: UIViewController {
    @IBOutlet weak var doughImageView: UIImageView!
    @IBOutlet weak var glazeImageView: UIImageView!
    @IBOutlet weak var toppingImageView: UIImageView!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        doughImageView.image = Donut.Dough.plain.uiImage(thumbnail: false)
        glazeImageView.image = Donut.Glaze.blueberry.uiImage(thumbnail: false)
        toppingImageView.image = Donut.Topping.blueberryDrizzle.uiImage(thumbnail: false)
    }
}
import UIKit
import FoodTruckKit

class ViewController: UIViewController {
    @IBOutlet weak var doughImageView: UIImageView!
    @IBOutlet weak var glazeImageView: UIImageView!
    @IBOutlet weak var toppingImageView: UIImageView!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        doughImageView.image = Donut.blackRaspberry.dough.uiImage(thumbnail: false)
        glazeImageView.image = Donut.blackRaspberry.glaze?.uiImage(thumbnail: false)
        toppingImageView.image = Donut.blackRaspberry.topping?.uiImage(thumbnail: false)
    }
}
