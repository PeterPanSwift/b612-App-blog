請 AI 將包含中文 key 的 JSON 定義成 Codable 型別

最近有同學遇到特別的 JSON，它的 key 寫成中文。雖然我們較容易看懂 JSON，但是為了方便程式開發，我們定義 JSON 的 Codable 型別時常會另外將 key 轉換成英文。

https://data.coa.gov.tw/Service/OpenData/TransService.aspx?UnitId=QcbUEzN6E6DL

將中文 key 轉換成英文不難，只要利用 coding keys 定義對應的關係，不過有兩個滿辛苦的地方。

  • 要一個個寫出中文和英文的對應。
  • 有些單字的英文要另外查，比方彼得潘就不知道狂犬病的英文。

現在有了方便的 AI，這些麻煩事都可以請他幫忙了。以下彼得潘請 AI(GPT 4) 定義 JSON 的 Codable 型別，他貼心地把中文翻成英文，連 coding keys 的對應都幫我們寫好。

  • 給 AI 的 prompt。

struct Animal: Codable {
    let animalID: Int
    let areaNumber: String
    let cityCode: Int
    let shelterCode: Int
    let location: String
    let type: String
    let breed: String
    let gender: String
    let size: String
    let color: String
    let age: String
    let neutered: String
    let rabiesVaccine: String
    let foundLocation: String
    let webTitle: String
    let status: String
    let note: String
    let otherDescription: String
    let adoptionStartTime: String
    let adoptionEndTime: String
    let dataUpdateTime: String
    let dataCreateTime: String
    let shelterName: String
    let pictureUrl: String
    let modificationTime: String
    let updateDataTime: String
    let address: String
    let phone: String

    enum CodingKeys: String, CodingKey {
        case animalID = "動物的流水編號"
        case areaNumber = "動物的區域編號"
        case cityCode = "動物所屬縣市代碼"
        case shelterCode = "動物所屬收容所代碼"
        case location = "動物的實際所在地"
        case type = "動物的類型"
        case breed = "動物品種"
        case gender = "動物性別"
        case size = "動物體型"
        case color = "動物毛色"
        case age = "動物年紀"
        case neutered = "是否絕育"
        case rabiesVaccine = "是否施打狂犬病疫苗"
        case foundLocation = "動物尋獲地"
        case webTitle = "動物網頁標題"
        case status = "動物狀態"
        case note = "資料備註"
        case otherDescription = "其他說明"
        case adoptionStartTime = "開放認養時間(起)"
        case adoptionEndTime = "開放認養時間(迄)"
        case dataUpdateTime = "動物資料異動時間"
        case dataCreateTime = "動物資料建立時間"
        case shelterName = "動物所屬收容所名稱"
        case pictureUrl = "圖片名稱"
        case modificationTime = "異動時間"
        case updateDataTime = "資料更新時間"
        case address = "地址"
        case phone = "連絡電話"
    }
}
