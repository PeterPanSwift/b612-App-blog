GitHub 的 iOS App 程式

想研究 iNaturalist iOS App 程式的朋友，也可以下載 GitHub 的 INaturalistIOS 專案。

GitHub - inaturalist/INaturalistIOS: iNaturalist iOS app
INaturalistIOS is the official iOS app for submitting data to iNaturalist.org. We use cocoapods for dependencies…github.com

下載後請進行以下步驟完成專案的設定。

  • 在 terminal 輸入 pod install 安裝套件。
  • 打開專案後，加入從 Firebase 下載的 GoogleService-Info.plist。
  • 將 config.h.example 複製一份到資料夾 INaturalistIOS 下，命名為 config.h。

config.h 的內容如下。可以先不做修改，採用預設值即可執行 App。

#ifndef iNaturalist_config_h
#define iNaturalist_config_h

// iNat oauth client id & secret
#define INatClientID        @"INat-client-ID"
#define INatClientSecret    @"INat-client-secret"

// Google oauth client id
#define GoogleClientId      @"Google-client-ID"

#endif

若想申請自己的 client id & secret，可參考以下文章 Authentication 的說明。

完成以上專案設定後，iNaturalist 的專案畫面如下。

專案的程式以 Objective-C 為主，加上一點點的 Swift 程式。

畫面的設計包含 storyboard，xib & 從程式的 NSLayoutAnchor 設定 auto layout。

API 的串接使用 URLSession，JSON 的解析利用 NSJSONSerialization & 第三方套件 Mantle。

- (void)extractObjectsFromData:(NSData *)data classMapping:(Class)ClassForMapping handler:(INatAPIFetchCompletionCountHandler)done {
    NSError *error;
    id json = [NSJSONSerialization JSONObjectWithData:data options:NSJSONReadingAllowFragments error:&error];
    if (error) {
        dispatch_async(dispatch_get_main_queue(), ^{
            done(nil, 0, error);
        });
    } else {
        NSMutableArray *results = [NSMutableArray array];
        for (NSDictionary *resultJSON in json) {
            NSError *error;
            MTLModel *result = [MTLJSONAdapter modelOfClass:ClassForMapping
                                         fromJSONDictionary:resultJSON
                                                      error:&error];
            
            if (result) {
                [results addObject:result];
            } else {
                // skip this one
                NSLog(@"MANTLE ERROR: %@", error);
            }
        }
        
        dispatch_async(dispatch_get_main_queue(), ^{
            done([NSArray arrayWithArray:results], [results count], nil);
        });
    }
}
