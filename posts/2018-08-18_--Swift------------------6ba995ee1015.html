從 Swift 的屬性和參數型別判斷內容要填什麼

在寫 Swift 程式時我們時常需要設定東西的屬性，呼叫 function 時傳入參數，或是生出某個型別的東西。但是該如何知道填什麼，要如何將東西生成呢 ? 在找 Google 大神幫忙前，可先嘗試以下幾個方法:

1 當型別是 Int, Double, Float, CGFLoat，String 時，給它數字和字串。

例子:

(1) 型別 String

let utterance = AVSpeechUtterance(string: "好懷念那夏天，我們被雷雨嚇得狂叫過大街")

(2) 型別 Int。

segmentedControl.selectedSegmentIndex = 2
2 當型別是 Bool 時，給它 true 或 false。
isloveMeSwitch.isOn = false
3 其它型別的東西，一般會用型別名 + () 生成。

例子: 型別 UIColor。

view.backgroundColor = UIColor(red: 120/255, green: 3/255, blue: 1/255, alpha: 1)

輸入型別名 + ( 後，可看到很多選項，可傳入不同的參數生出東西。

4 當型別是 optional (加 ? 或 ! ) 時，可以給它 nil 或此型別的東西。
label.text = "不管在別人眼裡有多麼瘋癲，有多麼不體面"
label.text = nil
5 當型別是 nested type ，長得像 A.B.C 的樣子時，常常可以用 . 搭配自動完成輸入。

例子: 型別 UIControl.State

button.setTitle("Havana", for: .normal)

normal 是以 static 宣告的變數，在這裡可以寫成 UIControl.normal，或是省略 UIControl，寫 .normal 就好。

public static var normal: UIControl.State { get }
6 試試輸入 .，如果有自動完成提示，表示可從清單裡選擇。

例子: 型別 NSTextAlignment

label.textAlignment = .center

型別 NSTextAlignment 以 enum 定義，指定它的內容時可用 NSTextAlignment.center，或是省略 NSTextAlignment，寫 .center 就好。

7 輸入型別名. + 屬性或方法生出東西。

例子:

let session = URLSession.shared
let dic = try? JSONSerialization.jsonObject(with: data)
8 當型別出現 -> 時，表示它是 function 型別，要以 { } 傳入我們想執行的程式碼(closure)。

例子: (Data?, URLResponse?, Error?) -> Void

建議如上圖參數型別為藍色框框時按下 enter，讓 Xcode 幫我們自動輸入 { } 的部分程式。

依序輸入 ( ) 裡的參數名和程式碼。

URLSession.shared.dataTask(with: url) { data, response, error in
   if let data {
      print(data)
   }
}
