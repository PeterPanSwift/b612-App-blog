SwiftUI ScrollView 上下捲動的內容常會搭配 frame(maxWidth: .infinity)

當我們使用 SwiftUI ScrollView 設計上下捲動的頁面時，有時要注意一下 ScrollView 的寬度問題。

以下我們用顯示日職球隊的 ScrollView 說明。

struct ContentView: View {
    let npbTeams = [
        "讀賣巨人",
        "阪神虎",
        "中日龍",
        "橫濱DeNA海灣之星",
        "廣島東洋鯉魚",
        "東京養樂多燕子",
        "歐力士猛牛",
        "福岡軟銀鷹",
        "北海道日本火腿鬥士",
        "千葉羅德海洋",
        "埼玉西武獅",
        "東北樂天金鷹"
    ]
    
    var body: some View {
        ScrollView {
            VStack(spacing: 30) {
                ForEach(npbTeams, id: \.self) { team in
                    Text(team)
                        .font(.system(size: 30))
                        .padding()
                        .background(.teal, in: .capsule)
                }
            }
        }
        .background(.yellow)
    }
}

如下圖所示，ScrollView 的高度佔滿了可用空間，然而它的寬度卻是內容決定的，因此 ScrollView 沒有佔滿螢幕，造成螢幕左右兩側空白的區塊無法捲動。

通常我們會希望整個頁面都可以上下捲動，因此若遇到內容的寬度不足螢幕寬度，可在內容呼叫 frame(maxWidth: .infinity) 讓 ScrollView 的寬度等同螢幕寬度。

struct ContentView: View {
    let npbTeams = [
        "讀賣巨人",
        "阪神虎",
        "中日龍",
        "橫濱DeNA海灣之星",
        "廣島東洋鯉魚",
        "東京養樂多燕子",
        "歐力士猛牛",
        "福岡軟銀鷹",
        "北海道日本火腿鬥士",
        "千葉羅德海洋",
        "埼玉西武獅",
        "東北樂天金鷹"
    ]
    
    var body: some View {
        ScrollView {
            VStack(spacing: 30) {
                ForEach(npbTeams, id: \.self) { team in
                    Text(team)
                        .font(.system(size: 30))
                        .padding()
                        .background(.teal, in: .capsule)
                }
            }
            .frame(maxWidth: .infinity)
        }
        .background(.yellow)
    }
}

剛剛我們在 ScrollView 的內容 VStack 呼叫 frame(maxWidth: .infinity)，這是比較建議的做法，若直接從 ScrollView 呼叫 frame(maxWidth: .infinity)，將有不太一樣的效果。

var body: some View {
    ScrollView {
        VStack(spacing: 30) {
            ForEach(npbTeams, id: \.self) { team in
                Text(team)
                    .font(.system(size: 30))
                    .padding()
                    .background(.teal, in: .capsule)
            }
        }
    }
    .frame(maxWidth: .infinity)
    .background(.yellow)
}

App 畫面如下，從下圖的 scroll bar 可看出捲動的區塊只是 VStack 本身寬度裡的區塊，螢幕兩邊還是不能捲動。
