搭配 storyboard reference 時傳資料

當我們使用 segue 切換頁面時，可透過 IBSegueAction 或 prepare 傳資料給下一頁的 view controller。那如果 segue 連到下一頁是透過 storyboard reference 呢 ? 一樣可以 ! 基本上你可以把 storyboard reference 當隱形人，把 segue 當成連到 view controller 那樣傳資料。

接下來讓我們以漫威影集為例說明吧。

storyboard 的設定

有兩個 storyboard，Main.storyboard & Video.storyboard，Main.storyboard 裡有個 ViewController，Video.storyboard 裡有個 VideoViewController。

ViewController 有三個 button，顯示汪達幻視，洛基 & 鷹眼。點選 button 將跳到 Video.storyboard 的 VideoViewController，顯示影集的封面圖片。

從 ViewController 連 segue 到 storyboard reference，之後點選 button 時以 performSegue 切換頁面。

Segue ID 設為 showVideo。

storyboard reference 的 Storyboard 欄位設為 Video，表示連到 Video.storyboard。

在 Video.storyboard 加入 image view，顯示帥氣的漫威影集封面。

點選 button 觸發的 IBAction

利用參數 sender 傳入影集的名稱。在此我們採用 iOS 15 的 configuration 設定 button 標題，因此使用 sender.configuration?.title 取得 button 的文字。

@IBAction func showVideo(_ sender: UIButton) {
        
    performSegue(withIdentifier: "showVideo", sender: sender.configuration?.title ?? "")
}

顯示影集封面圖的 VideoViewController

在 viewDidLoad 利用 ViewController 傳送的 video 名字設定圖片。

class VideoViewController: UIViewController {
    var video = ""
    @IBOutlet weak var imageView: UIImageView!

    override func viewDidLoad() {
        super.viewDidLoad()
        imageView.image = UIImage(named: video)
    }

以下我們分別示範 IBSegueAction & prepare 兩種不同的方法。

方法 1: IBSegueAction

從 Main.storyboard 的 segue 連 IBSegueAction 到 ViewController。在最後一個欄位選擇 Sender，因為我們將從 sender 取得 performSegue 傳來的影集名稱。

在 ViewController 定義 function showVideoViewController。

@IBSegueAction func showVideoViewController(_ coder: NSCoder, sender: Any?) -> UIViewController? {
        let controller = VideoViewController(coder: coder)
        if let name = sender as? String {
            controller?.video = name
        }
        return controller
}

方法 2: prepare

在 ViewController 定義 function prepare。

override func prepare(for segue: UIStoryboardSegue, sender: Any?) {
        if let videoViewController = segue.destination as? VideoViewController,
           let name = sender as? String {
            videoViewController.video = name
        }
}

結果

點選 button 顯示對應的封面圖片。
