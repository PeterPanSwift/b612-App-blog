使用 AI 開發串接 Firebase 的 SwiftUI / UIKit / Flutter App
開發串接 Firebase 的 App 需要進行許多設定，步驟有點複雜，因此接下來我們就交給 AI 吧，請 AI 完成 App 串接 Firebase 的相關程式和設定。medium.com

打開 Firebase project 的帳密登入功能

連到 Firebase 的專案頁面，切換到 Build > Authentication > Sign-in method 頁面，點選 Email/Password。

打開 Email/Password 的開關。

Email/Password 成功開啟，狀態變成 Enabled。

啟用 Firebase project 的 Cloud Firestore

使用者註冊時除了輸入帳密，通常還會輸入其它資訊，比方名字，生日等，因此我們利用 Cloud Firestore 儲存這些資訊。

連到 Firebase 的專案頁面，切換到 Build > Firestore Database，點選 Create database。

點選 Standard edition。

Location 選擇 Taiwan。

設定使用者存取資料的權限，可先選擇方便測試的 test mode。

之後若想修改權限可參考以下連結。

AI 成功完成串接 Firebase 的帳號註冊功能，以下分別為 SwiftUI & Flutter App 的畫面。

  • SwiftUI App
  • Flutter App

從 App 註冊帳號後可從 Firebase 專案的 Authentication / Users 頁面看到新增的使用者。

從 Firestore Database 頁面可看到 user 的相關資訊，document 的 ID 將對應 Authentication / Users 裡使用者的 User UID。
