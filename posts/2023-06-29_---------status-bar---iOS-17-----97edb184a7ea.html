自動隨背景變色的 status bar — iOS 17 新功能

iOS App 上方顯示時間電量的 status bar 有白色和黑色兩種選擇，開發 App 時由於不同的畫面有不同的背景，我們常常需要依據畫面的顏色從程式調整 status bar 的顏色。

從 iOS 17 開始，終於不用這麼麻煩了。iOS 17 的 status bar 會聰明地自動隨背景變色，時間電量不管在哪種背景都能清楚顯示，而且還可以左邊的時間跟右邊的電量顯示不同顏色。

以下我們做個簡單的實驗，將畫面分割成左右兩半，利用 TimelineView 讓畫面每 5 秒鐘隨機變色。

struct ContentView: View {
    var body: some View {
        
        TimelineView(.periodic(from: .now, by: 5)) { context in
            HStack(spacing: 0) {
                ForEach(0..<2) { _ in
                    Color(red: .random(in: 0...1), green: .random(in: 0...1), blue: .random(in: 0...1))
                }
            }
     
        }
        .ignoresSafeArea()
    }
}

執行 App 後，左右兩半的顏色每 5 秒隨機變色時， status bar 會立即更新顯示適合的顏色。

如下圖所示，左邊明亮的藍色背景搭配黑色的時間，右邊深色的背景搭配白色的電量。

左右都是深色背景時，status bar 更新成白色。

左右都是淺色背景時，status bar 更新成黑色。
