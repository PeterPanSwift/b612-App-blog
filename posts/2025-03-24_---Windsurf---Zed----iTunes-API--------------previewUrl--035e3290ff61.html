使用 Windsurf / Zed 找出 iTunes API 歌曲 / 影片的預覽網址(previewUrl)

Windsurf / Zed 內建方便的連網抓資料功能，以下彼得潘以偶像戴佩妮的歌曲勢在必行為例，示範透過它們找出 iTunes API 歌曲 / 影片的預覽網址。

使用 Windsurf

  • 下載 Windsurf。
Windsurf Editor by Codeium
Tomorrow's editor, today. Windsurf Editor is the first AI agent-powered IDE that keeps developers in the flow…codeium.com
  • 登入帳號。
  • 選擇 Chat & AI model Cascade Base。

免費版的功能即可抓取網路資料，請選擇 Chat & AI model Cascade Base。

  • 請 AI 取得歌曲的預覽網址。

比方想抓取戴佩妮歌曲勢在必行的資訊，API 的網址為 https://itunes.apple.com/search?term=勢在必行+戴佩妮&media=music&country=tw。

找出戴佩妮的勢在必行的 previewUrl
https://itunes.apple.com/search?term=勢在必行+戴佩妮&media=music&country=tw

結果。

成功取得 Penny 勢在必行的試聽網址 https://audio-ssl.itunes.apple.com/itunes-assets/AudioPreview221/v4/92/17/31/921731e0-d123-b5f9-a8d0-b73ea14b1275/mzaf_15112140117265056641.plus.aac.p.m4a。

除了預覽網址，我們還可以請 AI 找出專輯的名稱和縮圖。

找出戴佩妮的勢在必行的 previewUrl、collectionName、artworkUrl100
https://itunes.apple.com/search?term=勢在必行+戴佩妮&media=music&country=tw

artworkUrl100 的圖片可改大小，比方 1024 * 1024 的網址如下。

https://is1-ssl.mzstatic.com/image/thumb/Music211/v4/cd/a9/1d/cda91d38-6d33-8a78-95ad-4e50f7926f74/4712592250584.jpg/1024x1024bb.jpg

使用 Zed

  • 下載 Zed。
Zed - The editor for what's next
Zed is a high-performance, multiplayer code editor from the creators of Atom and Tree-sitter.zed.dev
  • 啟動 Zed，點選視窗右上方的 Sign in 登入 GitHub 帳號。
  • 打開 AI 的聊天視窗。

點選右下的星星圖案。

  • AI model 選擇 Claude 3.7。
  • 請 AI 對網址進行編碼。
url query params encode, just give me result
https://itunes.apple.com/search?term=勢在必行+戴佩妮&media=music&country=tw

結果。

The encoded URL is:

```
https://itunes.apple.com/search?term=%E5%8B%A2%E5%9C%A8%E5%BF%85%E8%A1%8C+%E6%88%B4%E4%BD%A9%E5%A6%AE&media=music&country=tw
```
  • 複製產生的網址，輸入以下指令按 enter。
/fetch https://itunes.apple.com/search?term=%E5%8B%A2%E5%9C%A8%E5%BF%85%E8%A1%8C+%E6%88%B4%E4%BD%A9%E5%A6%AE&media=music&country=tw
  • 取得 previewUrl。
找出戴佩妮的勢在必行的 previewUrl

結果。

previewUrl 是:

```

```
