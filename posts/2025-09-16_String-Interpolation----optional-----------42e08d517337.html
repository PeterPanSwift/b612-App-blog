String Interpolation 讀取 optional 時可以搭配預設值了

從 Swift 6.2 開始，String Interpolation 讀取 optional 時可以搭配預設值了。

當 String Interpolation 直接在 \( ) 裡傳入 optional 時，生成的字串會多出 Optional( )。

var age: Int? = 18
print("我今年\(age)歲")
我今年Optional(18)歲

因此我們通常會用 ??、optional binding 或 ! 讓 \( ) 裡的內容變成非 optional。

print("我今年\(age ?? 1)歲")

若想要 optional 無值時也能顯示預設的文字，看來 ?? 是不錯的方法，但是 ?? 後的預設值必須跟 optional 同一個型別，因此像以下例子，想要預設值為 " ? " 就行不通了，因為 age 的型別是 Int?。

var age: Int? = 18
print("我今年\(age ?? " ? ")歲")

在 Swift 6.2 有了 Default Value in String Interpolation，剛剛的問題終於可以迎刃而解，在 \( ) 裡我們可透過參數 default 設定預設的字串。

因此想要 age 無值時顯示 1 或 ? 都不是問題。

var age: Int? 
print("我今年\(age, default: "1")歲")
var age: Int? 
print("我今年\(age, default: " ? ")歲")
