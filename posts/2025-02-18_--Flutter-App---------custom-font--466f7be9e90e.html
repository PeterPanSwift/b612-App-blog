為 Flutter App 加入客製字型 (custom font)

覺得 Flutter App 內建的中文字型不夠可愛嗎 ? 沒關係，我們可另外加入字型檔，將 Flutter App 顯示的文字變成我們喜愛的字型。

可加到專案裡的字體檔主要有 ttf ，otf & ttc 三種格式，其中 ttc 比較容易有問題，因此建議選擇 ttf 或 otf。接下來我們將從日本的字體網站下載可愛的粉筆字體，在 iOS App 上顯示可愛的粉筆字 !

手書きフォント「殴り書きクレヨン」のダウンロード｜すもももじ
無料のクレヨン手書きフォント「殴り書きクレヨン」のダウンロードができます。font.sumomo.ne.jp
  • 取得字體檔。
  • 將字體檔加到專案。
  • 在 pubspec.yaml 設定字體檔的路徑。
  • 在 Flutter 程式使用客製字體。

取得字體檔

從網站點選紅色按鈕，點選下載可愛的粉筆字體。

將字體檔加到專案

在 Flutter 專案新增存放字體檔的資料夾。以下我們新增資料夾 fonts。

按下 cmd + c 複製字體檔 crayon_1–1.tiff，然後在 VS Code 點選資料夾 fonts，按下 cmd + v 貼上。

在 pubspec.yaml 設定字體檔的路徑

從 pubspec.yaml 找到 fonts 區塊，從 fonts 設定字體檔的 family 名字和路徑，family 名字可自訂。

 fonts:
   - family: crayon
     fonts:
       - asset: fonts/crayon_1-1.ttf

ps: 如果字體檔還包含粗體、斜體等多種字體，可參考以下連結設定 style & weight。

Use a custom font
How to use custom fonts.docs.flutter.dev

在 Flutter 程式使用客製字體

從 TextStyle 的 fontFamily 傳入 pubspec.yaml 裡設定的 family 名字。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Text('彼得', style: TextStyle(fontFamily: 'crayon', fontSize: 50)),
      ),
    );
  }
}
