執行 App，省略 app build 時間的 Run Without Building(cmd+ctrl+r)

開發 iOS App 時，我們通常點選左上角的三角形(快速鍵 cmd + r)將 App 裝到模擬器上測試。

不過當專案程式很龐大時，比方以下的 Telegram 專案，往往點個三角形要等待很久，因為 cmd + r 會先花時間 build App，然後再裝到模擬器上。

如果程式沒有修改，不需要重新 compile，可以點選 Xcode menu 的 Product > Perform Action > Run Without Building (快速鍵 cmd + ctrl + r ) ，將 App 直接裝到模擬器上，省掉漫長的 app build 時間。
