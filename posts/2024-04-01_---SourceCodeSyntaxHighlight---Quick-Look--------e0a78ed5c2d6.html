安裝 SourceCodeSyntaxHighlight 讓 Quick Look 的程式變彩色

macOS 的 Quick Look 十分方便，它讓我們在 Finder 上按空白鍵一鍵顯示檔案內容，不過遇到程式檔時它卻有一些小小的缺點。

  • 顯示的程式是黑白的。
  • 有些程式檔無法顯示。

Swift、Python 等程式可以，不過 Dart 程式卻不行。

研究一下後，彼得潘發現 Quick Look extension SourceCodeSyntaxHighlight 完美地解決剛剛的問題，它讓程式在 Quick Look 變彩色，而且 Dart 等原本不支援的程式也能順利顯示。

連到 SourceCodeSyntaxHighlight 的 GitHub

安裝 SourceCodeSyntaxHighlight

參考 GitHub 上的說明用 brew 安裝，或從以下連結下載 SourceCodeSyntaxHighlight。

Releases · sbarex/SourceCodeSyntaxHighlight
Quick Look extension for highlight source code files on macOS 10.15 and later. - Releases ·…github.com

以下彼得潘點選 Syntax.Highlight.zip。

將下載的 Syntax Highlight 移到 Applications 下。

啟動 SourceCodeSyntaxHighlight

從右鍵選單點選 Open 啟動。

啟動後將看到 Syntax Highlight 的設定視窗，裡面有許多不錯的樣式可選擇。

成功啟動後， Syntax Highlight 將出現在 System Settings > Extensions > Quick Look 下的 extension 清單。

如下圖所示，當 Syntax Highlight Quick Look Extension 勾選時，即表示 Quick Look 可以顯示彩色程式。

從 Quick Look 顯示彩色程式

有了外掛 Syntax Highlight，Quick Look 的程式不再是黑白了，終於能一覽美美的彩色 Swift 程式。

顯示 Dart 程式也不是問題。
