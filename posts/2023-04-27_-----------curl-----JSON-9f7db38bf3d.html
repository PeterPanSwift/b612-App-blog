以整齊的縮排彩色顯示 curl 抓到的 JSON

當我們在 Mac 的 terminal 輸入 curl 指令時，顯示的 JSON 資料預設會是不易閱讀的無縮排文字，而且文字全是一樣的顏色。

curl "https://itunes.apple.com/search?term=愛情&media=music&country=tw"

有沒有辦法讓它變成整齊的縮排顯示，而且內容還會用不同顏色區分呢 ? 感謝 GPT 4 的幫忙，它教彼得潘三種搭配 python 技術實現的方法。

  • 方法1: 輸入 curl 指令 ＆ python 的相關指令。
  • 方法2: 輸入 curl 指令 & 自訂的 script。
  • 方法3: 執行自訂的 python 程式檔，傳入 API 網址。

方法1: 輸入 curl 指令 ＆ python 的相關指令。

curl "https://itunes.apple.com/search?term=愛情&media=music&country=tw" | python3 -c "import sys, json; data=json.load(sys.stdin.buffer); print(json.dumps(data, ensure_ascii=False, indent=2))" | pygmentize -l json

利用 python3 -c "import sys, json; data=json.load(sys.stdin.buffer); print(json.dumps(data, ensure_ascii=False, indent=2))" 縮排 JSON，pygmentize -l json 將 JSON 變彩色。

ps: 如果無法執行 pygmentize，請先輸入以下指令安裝 pygments。

pip3 install pygments

方法2: 輸入 curl 指令 & 自訂的 script

新增 script 檔 JSONFormatter。

#!/bin/zsh

python3 -c "import sys, json; from pygments import highlight; from pygments.lexers import JsonLexer; from pygments.formatters import TerminalFormatter; data=json.load(sys.stdin.buffer); formatted_json=json.dumps(data, ensure_ascii=False, indent=2); colorful_json=highlight(formatted_json, JsonLexer(), TerminalFormatter()); print(colorful_json)"

設定 JSONFormatter 的執行權限。

chmod +x JSONFormatter

將 JSONFormatter 搬到 /usr/local/bin，方便之後在 terminal 的任何路徑下輸入 JSONFormatter。

sudo mv JSONFormatter /usr/local/bin/

輸入 curl 指令，然後串接 JSONFormatter，將 JSON 變成整齊的縮排彩色顯示。

curl "https://itunes.apple.com/search?term=愛情&media=music&country=tw" | JSONFormatter

方法3: 執行自訂的 python 程式檔，傳入 API 網址

新增 fetch_and_print_json.py，內容如下。

import sys
import json
import requests
from pygments import highlight
from pygments.lexers import JsonLexer
from pygments.formatters import TerminalFormatter

url = sys.argv[1]

response = requests.get(url)
data = response.json()

formatted_json = json.dumps(data, ensure_ascii=False, indent=2)
colorful_json = highlight(formatted_json, JsonLexer(), TerminalFormatter())

print(colorful_json)

執行時傳入 API 的網址。

python3 fetch_and_print_json.py "https://itunes.apple.com/search?term=愛 情&media=music&country=tw"

縮排 JSON 的其它方法

我們也可以用 python3 -m json.tool 縮排 JSON，不過它會有中文亂碼的問題。例如以下例子。

curl "https://itunes.apple.com/search?term=愛情&media=music&country=tw" | python3 -m json.tool
