還原 Xcode Coding Assistant 改壞的程式

當我們使用 Xcode 的 Coding Assistant 修改程式時，常常會遇到 AI 寫錯，我們想要還原程式的情況。

Coding Assistant 本身提供了幾種還原的方法。

  • 方法 1: 點選下圖的 Revert 取消這次的修改。
  • 方法 2: 點選右上的時間。

滑動 Slider，選擇還原的時間點後點選 Restore，從 changes history 選擇還原點。

如果想要更確保能隨時回到某個正確的程式版本，建議再搭配 git 進行版本管理，相關說明可參考以下連結。
