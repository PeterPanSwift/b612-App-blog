如何讓 iOS App 畫面擁有背景圖片

在 Storyboard 設定 view 的 Background 欄位即可擁有漂亮的背景顏色，令人難過的，它卻沒有一個設定背景圖片的 BackgroundImage 欄位。

想要擁有漂亮的背景圖片，其實還是做得到，只是過程比較曲折些。接下來就讓我們學習兩種設定的方法:

方法1: 從 storyboard ，在 controller 的 view 上加入佔滿畫面的 image view，讓它在最底層。

方法2: 在 controller 的 function viewDidLoad 裡設定 view 的 backgroundColor。

UIColor 有個特別的 init，init(patternImage image: UIImage)，能讓我們將圖片變成顏色。

override func viewDidLoad() {
   super.viewDidLoad()
   self.view.backgroundColor = UIColor(patternImage: UIImage(named: "background")!)
}
