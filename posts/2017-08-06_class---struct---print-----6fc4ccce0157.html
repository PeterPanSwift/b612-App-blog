class 和 struct 的 print 大不同

class 和 struct 是很像的東西，但我們也知道，它一定是有差異的，畢竟 Swift 沒有必須發明兩個一樣的東西。至於它們差別在哪，一般常聽到的是以下三點:

  1. struct 不能繼承。
  2. struct 有 memberwize initializer。
  3. struct 是 value type，class 是 event type。

不過今天我們來聊聊 print。當 print 在列印 struct 和 class 生成的資料時，可是有很大的不同。讓我們先看看 struct 的例子 :

從下圖我們可看出 print 列印 struct 資料時很貼心，將 Baby 的名字年齡完整地印出，讓我們看得一清二楚。

相反的，當我們列印 class 資料時，它變得十分神秘，只願意透露它的型別是 Baby。如果想知道寶寶的名字年齡，得另外讀取屬性 name & age。

因此，如果單純以方便我們列印了解資料的內容 PK 的話，struct 是比 class 優秀的 !
