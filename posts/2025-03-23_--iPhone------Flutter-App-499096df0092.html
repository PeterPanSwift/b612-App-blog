從 iPhone 桌面啟動 Flutter App

當我們從 IDE 將 Flutter App 安裝啟動於 iPhone 時，它將生成方便 debug 的 App，因此 Console 會顯示訊息 Launching … in debug mode。

然而 debug 版的 App 在 iPhone 將無法從桌面點擊啟動。若想點擊 icon 啟動 App，我們必須生成 release 版的 App。方法有兩種。

  • 從 Terminal 輸入指令。
  • 在 launch.json 加入 release 版的設定。

ps: release 版的 App 不方便 debug，少了修改程式更新 App 的 hot reload 功能。

從 Terminal 輸入指令

flutter run --release

選擇執行 App 的裝置。

在 launch.json 加入 release 版的設定

如果覺得從 Terminal 輸入指令有點麻煩，我們也可在 launch.json 加入 release 版的設定，之後即可一鍵啟動 release 版的 App。

從 Command Palette 點選 Add Configuration。

Add Configuration 將在專案下新增資料夾 .vscode & 檔案 launch.json。

launch.json 內容如下，自動包含三種模式，我們要的 release mode 也在其中。

{
    // Use IntelliSense to learn about possible attributes.
    // Hover to view descriptions of existing attributes.
    // For more information, visit: https://go.microsoft.com/fwlink/?linkid=830387
    "version": "0.2.0",
    "configurations": [
        {
            "name": "flutter_application_1",
            "request": "launch",
            "type": "dart"
        },
        {
            "name": "flutter_application_1 (profile mode)",
            "request": "launch",
            "type": "dart",
            "flutterMode": "profile"
        },
        {
            "name": "flutter_application_1 (release mode)",
            "request": "launch",
            "type": "dart",
            "flutterMode": "release"
        }
    ]
}

加了剛剛的設定後，我們有兩種方法啟動 release 版的 App。

  • 方法 1。

從側邊欄選擇 Run and Debug 分頁，點選下圖的框框選擇 release mode。

點選下圖的三角形。

  • 方法 2: 從 Command Palette 點選 Run App in Release Mode。
