SwiftUI 設計線條樣式的 StrokeStyle

SwiftUI 的形狀可以利用 stroke 畫出線條，比方以下程式利用 Path 畫出帥氣的勾勾圖案。

SwiftUI 設計線條樣式的 StrokeStyle

SwiftUI 的形狀可以利用 stroke 畫出線條，比方以下程式畫出帥氣的長方形線條。

struct ContentView: View {
    var body: some View {
        Rectangle()
            .stroke(.blue, lineWidth: 10)
            .frame(width: 300, height: 300)
    }
}

stroke 的能力其實不僅於此，它可透過 StrokeStyle 調整線條的樣式，接下來就讓我們好好認識它吧。

設定線條樣式的 StrokeStyle

呼叫 stroke 時，可在參數 style 傳入 StrokeStyle。

StrokeStyle 十分厲害，它最多可傳入六個參數。

StrokeStyle 的 init 宣告如下，除了熟悉的 lineWidth，以下我們介紹常用的 lineCap，lineJoin & dash。

init(lineWidth: CGFloat = 1, lineCap: CGLineCap = .butt, lineJoin: CGLineJoin = .miter, miterLimit: CGFloat = 10, dash: [CGFloat] = [CGFloat](), dashPhase: CGFloat = 0)
init(lineWidth:lineCap:lineJoin:miterLimit:dash:dashPhase:) | Apple Developer Documentation
Creates a new stroke style from the given components.developer.apple.com

控制線條尾端形狀的 lineCap

lineCap 的型別是 CGLineCap，控制著線條尾端的形狀，有 butt，round，square 三個選項，預設是 butt。

為了能看到線條尾端的形狀，我們利用 trim 將長方形變成只有兩邊的線條。當我們傳入 .round 時，線條的兩個端點都變成圓形。

struct ContentView: View {
    var body: some View {
        Rectangle()
            .trim(from: 0, to: 0.5)
            .stroke(.blue, style: StrokeStyle(lineWidth: 10, lineCap: .round))
            .frame(width: 300, height: 300)
    }
}

butt & square 尾端的形狀都是方的，差別在 square畫的線會超出起點跟終點。

struct ContentView: View {
    var body: some View {
        ZStack {
            Path { path in
                path.move(to: CGPoint(x: 30, y: 100))
                path.addLine(to: CGPoint(x: 300, y: 100))
            }
            .stroke(style: StrokeStyle(lineWidth: 10, lineCap: .butt))
            
            Path { path in
                path.move(to: CGPoint(x: 30, y: 130))
                path.addLine(to: CGPoint(x: 300, y: 130))
            }
            .stroke(style: StrokeStyle(lineWidth: 10, lineCap: .square))
        }
        
    }
}
lineCap | Apple Developer Documentation
Specifies the line cap style for the shape's path.developer.apple.com

控制線條連接樣式的 lineJoin

lineJoin 的型別是 CGLineJoin，控制著線條連接的樣式，有 miter，round，bevel 三個選項，預設是 meter。

struct ContentView: View {
    var body: some View {
        Rectangle()
            .stroke(.blue, style: StrokeStyle(lineWidth: 10, lineJoin: .miter))
            .frame(width: 300, height: 300)
    }
}

當我們傳入 .round 時，連接的樣式變成圓形。

.stroke(.blue, style: StrokeStyle(lineWidth: 10, lineJoin: .round))

當我們傳入 .bevel 時，連接的樣式如下。

.stroke(.blue, style: StrokeStyle(lineWidth: 10, lineJoin: .bevel))

關於 miter，round，bevel 的差異，可參考下圖 Apple 官方的示意圖。

控制虛線樣式的 dash

參數 dash 可實現虛線的效果，它的型別是 [CGFloat]，array 裡的數字控制虛線 & 間隔的長度。

struct ContentView: View {
    var body: some View {
        Rectangle()
            .stroke(.blue, style: StrokeStyle(lineWidth: 10, dash: [3]))
            .frame(width: 300, height: 300)
    }
}

下圖為 dash 設為 [3] 的虛線，當 array 裡只有一個數字時，表示虛線跟間隔的長度一樣。

下圖為 dash 設為 [15] 的虛線，當 array 裡只有一個數字時，表示虛線跟間隔的長度一樣。

.stroke(.blue, style: StrokeStyle(lineWidth: 10, dash: [15]))

當 array dash 裡傳入多個數字時，將改變虛線跟間隔的長度，比方 [10, 3] 表示虛線長度 10，間隔 3。

.stroke(.blue, style: StrokeStyle(lineWidth: 10, dash: [10, 3]))

[15, 2, 2, 4] 表示順序依序為

  1. 虛線長度 15
  2. 間隔 2
  3. 虛線長度 2
  4. 間隔 4
.stroke(.blue, style: StrokeStyle(lineWidth: 10, dash: [15, 2, 2, 4]))

關於 dash 更多的細節，可參考下圖 Apple 官方的示意圖。

lineDashPattern | Apple Developer Documentation
