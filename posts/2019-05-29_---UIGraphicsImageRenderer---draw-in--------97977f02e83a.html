利用 UIGraphicsImageRenderer & draw(in:) 縮小圖片

當我們利用 UIImagePickerController 拍照時，得到的照片大小為 3024 * 4032 pixel，然而一般我們的 App 畫面或是上傳到後台的圖片並不需要那麼大，因此我們時常會將拍照得到的照片縮小。

新版做法(iOS 15 以上)

縮小圖片的 preparingThumbnail
開發 iOS App 時，有時我們並不需要那麼大的圖片，若能將圖片縮小，將可大幅減少 App 使用的記憶體，或是加快圖片上傳到後台的時間。medium.com

舊版做法

接下來就讓我們透過以下的程式說明如何將圖片縮小吧。function resizeImage 將回傳縮小後的 UIImage，圖片比例和原來一樣，但寬度縮小為 640 pixel。

func resizeImage(image: UIImage, width: CGFloat) -> UIImage {
        let size = CGSize(width: width, height:
            image.size.height * width / image.size.width)
        let renderer = UIGraphicsImageRenderer(size: size)
        let newImage = renderer.image { context in
            image.draw(in: renderer.format.bounds)
        }
        return newImage
}
let smallImage = resizeImage(image: image, width: 320)

說明

我們希望將圖片縮小，變成寬度 320 points 的圖片。

計算縮小後的圖片尺寸。

我們希望圖片縮小後仍能維持比例，因此可利用以下程式計算尺寸。

let size = CGSize(width: width, height:
            image.size.height * width / image.size.width)
生成 UIGraphicsImageRenderer 物件，指定繪製的圖片大小。
let renderer = UIGraphicsImageRenderer(size: size)
利用 UIGraphicsImageRenderer 的 function image(actions:) 產生縮小的 UIImage。
let newImage = renderer.image { context in
   image.draw(in: renderer.format.bounds)
}

function image(actions:) 的宣告如下

func image(actions: (UIGraphicsImageRendererContext) -> Void) -> UIImage
image(actions:) | Apple Developer Documentation
Creates an image from a set of drawing instructions.developer.apple.com

我們在參數 actions 的 closure 裡呼叫 UIImage 的 draw(in:) 繪製圖片。

func draw(in rect: CGRect)
draw(in:) | Apple Developer Documentation
Draws the entire image in the specified rectangle, scaling it as necessary to fit.developer.apple.com

當我們呼叫 draw(in:) 繪製圖片時，它將在我們指定的長方形 rect 裡繪製圖片。在此我們傳入的 rect 參數為 renderer.format.bounds， 它的origin 為 (0,0)，size 則為當初生成 UIGraphicsImageRenderer 指定的 size。因此 draw(in:) 會將原本的大圖縮小繪製，填滿 UIGraphicsImageRenderer 的長方形畫布。

resizeImage(image: image, width: 320) 產生的圖片寬度將由 iPhone 決定，比方 iPhone 11 Pro 是 3x，所以將是 960 pixel，iPhone 11 是 2x，所以是 640 pixel。

拍照後將圖片縮小

在 imagePickerController(_:didFinishPickingMediaWithInfo:) 裡呼叫剛剛定義的 function resizeImage(image:) 縮小圖片。

func imagePickerController(_ picker: UIImagePickerController, didFinishPickingMediaWithInfo info: [UIImagePickerController.InfoKey : Any]) {
   if let image = info[.originalImage] as? UIImage {
      let smallImage = resizeImage(image: image)
   }
   dismiss(animated: true)
}

若想將縮小後的圖片儲存在 documents 資料夾，可參考以下程式碼。

let documentsDirectory = URL.documentsDirectory
let url = documentsDirectory.appendingPathComponent("pic").appendingPathExtension("jpg")
try? smallImage.jpegData(compressionQuality: 0.8)?.write(to: url)
