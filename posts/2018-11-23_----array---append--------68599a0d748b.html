為什麼 array 的 append 有紅線呢 ?

想要將東西加到 array 裡，只要呼叫 function append 即可，不過卻有朋友遇到問題，輸入程式時發現自動完成選單的 append 出現不能輸入的紅線。

var gifts = ["iMac Pro", "Macbook Pro"]
var newGifts = gifts.ap

為什麼會出現紅線呢 ? 其實這是聰明的 Xcode 提醒我們在這裡不該使用 append，因為 append 不會回傳東西，所以我們不該用 newGifts 儲存 append 回傳的資料。

如果我們不管 Xcode 的好心警告，硬要輸入 append 呢 ? 其實是可以的，只是沒意義，因為 function append 不會回傳東西，此時 newGifts 的內容會變成 ()。( 因為 () 等於 Void )

var newGifts = gifts.append("iPhone")
