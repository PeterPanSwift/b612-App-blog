新版做法

利用 Xcode 的 Format File with swift-format 縮排程式。

利用 Xcode 的 Format File with swift-format 縮排程式 — Xcode 16 新功能
Xcode 從 16 開始內建縮排程式的工具 swift-format，現在縮排程式變得更加容易，按下快速鍵 ctrl + shift + i 即可用 Format File with swift-format 縮排整個檔案。medium.com

舊版做法

假設原本的程式如下圖，縮排雜亂無章，就像彼得潘的書桌。

  • 點選 cmd + a 全選檔案裡的程式(Edit > Select All)。
  • 點選 ctrl + i 縮排(Editor > Structure > Re-Indent)。

程式完美地縮排，希望未來也能有類似的科技幫忙彼得潘整理書桌。

ps: 從 Quick Actions 選單也可以點選 Select All 和 Re-Indent。
