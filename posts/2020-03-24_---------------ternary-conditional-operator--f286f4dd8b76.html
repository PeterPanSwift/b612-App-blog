使用 ?: 指定二選一的內容(ternary conditional operator)

開發 SwiftUI App 時，我們時常使用 ? : ( ternary conditional operator) 設定內容或呼叫 modifier，它可以讓程式變得更精簡易懂。

使用 ?: 指定二選一的內容(ternary conditional operator)

? : ( ternary conditional operator) 是一個很古老但 iOS App 開發者不見得常用的語法。不過風水輪流轉，在 SwiftUI 的世界它將幫我們簡化 if else 冗長的程式碼，是個幫我們將程式變得更精簡易懂的超級小幫手呢。

接下來我們先看看沒有 ?: 時使用 if else 的做法，然後再好好見識 ?: 的威力。

用 if else 判斷顯示實心 / 空心的星星

以 SwiftUI 實現下圖 Medium App 文章紅色框框標示的文章資訊。

struct ArticleInfoView: View {
    let isStarFill: Bool
    
    var body: some View {
        HStack {
            Image(.peter)
                .resizable()
                .scaledToFill()
                .frame(width: 50, height: 50)
                .clipShape(.circle)
            Text("Peter Pan")
            Text("March 16")
            Text("3 min read")
            if isStarFill {
                Image(systemName: "star.fill")
            } else {
                Image(systemName: "star")
            }
            
        }
    }
}

#Preview(traits: .sizeThatFitsLayout) {
    ArticleInfoView(isStarFill: true)
}
#Preview(traits: .sizeThatFitsLayout) {
    ArticleInfoView(isStarFill: false)
}

medium 的設計是有的文章資訊會顯示星星，有的不會顯示，我們將它改成顯示實心和空心的星星，利用 if else 判斷 isStarFill 顯示不同的圖片。

用 ?: 判斷顯示實心 / 空心的星星

改成 ?: 的寫法，會比 if else 簡單很多，原本好幾行的程式，現在只要一行 !

? 的左邊判斷結果為 true 或 false，當結果為 true 時，內容將為 ? 右邊的內容，結果為 false 時，內容將為 : 右邊的內容。因此 isStarFill 為 true 時將顯示圖片 star.fill，isStarFill 為 false 時將顯示圖片 star。

Image(systemName: isStarFill ? "star.fill" : "star")
struct ArticleInfoView: View {
    let isStarFill: Bool
    
    var body: some View {
        HStack {
            Image(.peter)
                .resizable()
                .scaledToFill()
                .frame(width: 50, height: 50)
                .clipShape(.circle)
            Text("Peter Pan")
            Text("March 16")
            Text("3 min read")
            Image(systemName: isStarFill ? "star.fill" : "star")
        }
    }
}

呼叫 modifier 時利用 ?: 傳入不同的內容

我們希望實心的星星是紅色，空心的星星是灰色。因此我們在呼叫 foregroundStyle 時利用 ?: 語法判斷，當 isStarFill 為 true 時傳入紅色，isStarFill 為 false 時傳入灰色。

struct ArticleInfoView: View {
    let isStarFill: Bool
    
    var body: some View {
        HStack {
            Image(.peter)
                .resizable()
                .scaledToFill()
                .frame(width: 50, height: 50)
                .clipShape(.circle)
            Text("Peter Pan")
            Text("March 16")
            Text("3 min read")
            Image(systemName: isStarFill ? "star.fill" : "star")
                .foregroundStyle(isStarFill ? .red : .gray)
        }
    }
}
