如何取得 Xcode 專案(project)的路徑

大部分的時候，我們打開 Xcode 後，可從最近開啟過的專案打開專案，因此並不需要知道專案在 Mac 擺放的路徑。不過某些時候我們還是需要知道專案的路徑，如果忘了路徑位置，可參考以下方法:

1 點選 Project navigator 裡的專案檔。

2 在右邊的 File inspector 頁面，找到 Full Path。

此時 Full Path 的內容即為專案的路徑，甚至我們還可直接點選箭頭打開 Finder，在 Finder 上顯示專案檔。

打開 Finder 後，一開始只會顯示專案檔，若想查詢專案檔所在的資料夾，可將 Finder 切換為 Columns 模式，即可找到專案檔上一層的資料夾。
