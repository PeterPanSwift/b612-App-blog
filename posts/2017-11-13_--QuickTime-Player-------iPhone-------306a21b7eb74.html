從 QuickTime Player 錄製或播放 iPhone 的操作畫面

想要錄製 iPhone 的操作畫面，但是身為邊緣人，找不到朋友幫你錄嗎 ? 又或者覺得 iPhone 的螢幕太小，想在 Mac 的大螢幕顯示 iPhone 的畫面嗎 ?

你想要的，都可以實現 ! 只要透過 Mac 內建的 App QuickTime Player。

1. 將 iPhone 接線連接到 Mac。

2. 打開 Mac 的 QuickTime Player。

3. 點選 File -> New Movie Recording。

4. 切換到 iPhone 的畫面錄影。

預設錄影將採用 Mac 的鏡頭，所以會拍到鏡頭前的自己。雖然我們很帥很美，不過現在不是自戀的時候，請點選紅色圓形按鈕旁的下拉箭頭後，選擇 Camera 下的 iPhone 切換。

如果想播放或錄製 iPhone 的聲音，記得要勾選 Microphone 下的 iPhone。

5. 切換到 iPhone 畫面後，點選紅色圓形按鈕開始錄影。

6 點選變成黑色正方形的按鈕結束錄影。

7 點選 File -> Save 存檔。

8 點選 File -> Share -> Facebook 將影片分享到Facebook。

如果在分享清單裡看不到 Facebook，千萬別生氣，只要打開 Mac 的 System Preferences App，點選 Internet Accounts，即可進一步點選 Facebook 設定帳號。一旦完成 Facebook 帳號的設定，未來即可在分享清單裡看到 Facebook 的蹤影。
