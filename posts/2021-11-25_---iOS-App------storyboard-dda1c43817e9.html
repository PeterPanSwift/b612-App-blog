設定 iOS App 主畫面的 storyboard

使用 storyboard 開發 iOS App 時，我們習慣用 Main.storyboard 的 view controller 當 App 打開後顯示的第一個畫面(經過開頭畫面後顯示的畫面)。不過當我們有多個 storyboard 時，我們也可以設定其它的 storyboard 當主畫面的 storyboard，方法如下:

我們想要 WandaVision.storyboard 裡的 view controller 當第一個畫面，想要一打開 App 就看到美麗的緋紅女巫。

切換到專案 Target 下 App 的 Info 頁面。

將 Application Scene Manifest > Scene Configuration > Application Session Role > Item 0 > Storyboard Name 設成 WandaVision，讓 WandaVision.storyboard 裡箭頭指到的 controller 成為 App 的第一個畫面。

有一招快速修改 Storyboard Name 的小技巧。我們可在 Find navigator 搜尋 main，然後點選 Info 下 main 修改 Storyboard Name。

結果

App 啟動後，美麗的緋紅女巫出現了 ~。

ps: 從 Main interface 調整第一個畫面的 storyboard 是無效的。
