自動提示跟 property 一樣的變數名稱 — Xcode 15 新功能

最近發現 Xcode 15 多了一個貼心的功能，它會自動提示跟 property 一樣的變數名稱。

以下我們以可愛的 Baby 說明，型別 Baby 的定義如下。

struct Baby {
    var name: String
    var age: Int
    var weight: Double?
}

然後我們在程式裡產生一個名字叫 peter 的可愛寶寶。

var cuteBaby = Baby(name: "peter", age: 1, weight: 10)

之後當我們想要宣告 name 儲存 cuteBaby.name 時，不管是一般的變數 / 常數宣告，或是 if let 和 guard let，Xcode 都會貼心地出現 name 的提示。

  • 宣告常數。
  • 宣告變數。
  • if let。
  • guard let。

另外此功能也有一些限制，有以下兩點要注意的地方。

  • 當 cuteBaby 宣告為 optional 時，自動完成不會加上 ?。

因此我們要自己補上問號。

let name = cuteBaby?.name
  • 只有第一層的 property 會自動提示。

比方我們在 Baby 裡加上 toy，Toy 裡有 price。

struct Toy {
    let price: Int
}

struct Baby {
    var name: String
    var age: Int
    var weight: Double?
    var toy: Toy?
}

輸入 pri 時，不會產生任何的提示，因為對 cuteBaby 來說，price 不是它的 property。
