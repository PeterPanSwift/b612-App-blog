從 Swift Playgrounds 和 Xcode 的 SwiftUI Sample Apps 學習 SwiftUI

Apple 的 SwiftUI Sample Apps 提供了豐富的 SwiftUI 範例，從 10 個以上的 App 範例教學，我們可以學到各種 SwiftUI 的相關技術，包含了畫面的製作，資料的傳遞，網路的串接，有趣的動畫和手勢，存取相簿，拍照和 sensor，機器學習等。

以下我們分成 4 個部分介紹如何從 SwiftUI Sample Apps 學習 SwiftUI。

  • 從哪裡研究 SwiftUI Sample Apps。
  • 各版本的差異。
  • 研究教學沒提到的程式。
  • 從 Xcode 研究 Swift Playgrounds 的範例。

各版本的差異

  • Mac & iPad 版 Swift Playgrounds 的差異。

iPad 版多了一些跟 sensor 有關的範例，像是 Seismometer。

  • Xcode Developer Documentation 和 Apple 開發網站的比較。

內容是一樣的，不過從 Xcode Developer Documentation 閱讀的體驗比網站好。

下載的 Project files 是 swiftpm 檔，可用 Xcode 或 Swift Playgrounds 打開。

兩個版本類似，但是裡面的程式和說明還是有一些差異。以下我們以 Choose Your Own Story 為例說明。

struct StoryPageView: View {

    let story: Story
    let pageIndex: Int

    var body: some View {
        VStack {
            ScrollView {
                Text(story[pageIndex].text)
            }
            
            ForEach(story[pageIndex].choices, id: \Choice.text) { choice in
                NavigationLink(destination: StoryPageView(story: story, pageIndex: choice.destination)) {
                    Text(choice.text)
                        .multilineTextAlignment(.leading)
                        .frame(maxWidth: .infinity, alignment: .leading)
                        .padding()
                        .background(Color.gray.opacity(0.25))
                        .cornerRadius(8)
                }
            }
        }
        .padding()
        .navigationTitle("Page \(pageIndex + 1)")
        .navigationBarTitleDisplayMode(.inline)
    }
}

從 Xcode 研究 Swift Playgrounds 的範例

比起 Swift Playgrounds，Xcode 還是強大許多，研究程式也比較方便，因此我們也可以從 Xcode 研究 Swift Playgrounds 的範例，方法如下。

  • 從 Finder 打開 Swift Playgrounds 的範例。

從清單裡找到有興趣的範例，然後從右鍵選單點選 Show in Finder。

  • 用 Xcode 開啟。

從右鍵選單點選 Open With > Xcode。
