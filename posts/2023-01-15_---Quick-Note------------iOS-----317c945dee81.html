啟動 Quick Note 快速打中文，複製貼到 iOS 模擬器

在 iOS 模擬器上只要加入注音鍵盤就可以方便地打中文，不過彼得潘平常習慣使用嘸蝦米輸入法，模擬器沒有嘸蝦米，因此彼得潘在模擬器只能可憐地用超慢的速度打著不熟的注音。

令人開心的，最近想到 macOS 的 Quick Note 可以解決這個問題，彼得潘可以用嘸蝦米在 Quick Note 打中文，然後再複製貼到 iOS 模擬器。

以下為詳細的步驟說明。

在模擬器執行 App，點選輸入框，準備在輸入框打字

啟動 Quick Note

  • 方法 1。

按住 mac 鍵盤的 Fn 鍵(地球 icon)，然後再按 q，此時 Notes App 將自動啟動，打開 New Note 視窗。

  • 方法 2。

滑鼠游標移到螢幕右下方，看到 note 後點擊它即可打開 New Note 視窗。

Start a Quick Note on Mac
If you're working in another app and want to write something down, you can easily start a Quick Note. Do one of the…support.apple.com

在 New Note 視窗打字，然後全選複製文字

彼得潘開心地在 Note 視窗裡用熟悉的嘸蝦米打字，輸入想和女朋友一起做的一百件事，然後按 cmd + a 全選，cmd + c 複製文字。

回到模擬器，按 cmd + v 貼上文字

之後若想再打字，只要再按 Fn 加 q 或以滑鼠點選 Mac 桌面上的 New Note 視窗。
