使用 Android Studio 的 Image Asset Studio 設定 App Icon

打開 Image Asset Studio。

  • 方法 1: 切換到 Resource Manager，點選 +，選擇 Image Asset。
  • 方法 2: 從 Project 檔案清單裡 res 的右鍵選單點選 New > Image Asset。

Icon type 選擇 Launcher Icons(Adaptive and Legacy)，Asset type 選擇 Image，從 Path 的按鈕選擇 icon 圖片。

在此彼得潘選擇 AI 繪製的桌球女孩 icon，大小為 1024 * 1024，格式為 png。

執行 App 後，Android 的桌面成功出現桌球女孩的 App Icon。
