Xcode playground 的 Timeline 在新版變成 Live View

在 Playground 寫 Swift 程式時，可打開右邊的 Assistant Editor，將 Assistant Editor 切換成 Live View 後，觀看 App 的畫面。

然而如果有同學研究 Apple 的電子書，研究 Swift 開發 App 入門課程第七課的 BoogieBot playground，將發現它要求我們將 Assistant Editor 切換為 Timeline。

The assistant editor should be showing the playground’s timeline. If it isn't, click the first entry in the bar at the top and choose “Timeline” so it looks like this

別擔心，不是你眼睛不好找不到 Timeline，只是在新版它的名字變成 Live View。
