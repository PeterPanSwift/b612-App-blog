利用 SwiftUI 的 transition 動畫呈現蒙娜麗莎

開發 iOS App 時，我們時常

方法 1: 設定 opacity

opacity 由 0 ~ 1，因此蒙娜麗莎將慢慢浮現在我們眼前。

struct ContentView: View {
    
    @State private var opacity: Double = 0
    
    var body: some View {
        Image("蒙娜麗莎")
            .opacity(opacity)
            .animation(.easeInOut(duration: 5))
            .onAppear {
                self.opacity = 1
        }
               
    }
}

方法 2: if 搭配 transition opacity

當元件利用 if 控制是否出現時，我們可搭配 transition 設定它出現或消失時的動畫。

struct ContentView: View {
    
    @State private var show = false
    
    var body: some View {
        VStack {
            if show {
                Image("蒙娜麗莎")
                    .transition(.opacity)
                
            }
        }
        .animation(.easeInOut(duration: 5))
        .onAppear {
            self.show = true
        }
    }
}
