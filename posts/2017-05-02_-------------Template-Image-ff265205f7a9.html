設定圖片顏色，將圖片變成 Template Image

button 或 image view 的圖片想要換顏色，其實不一定要苦苦哀求美術設計。變色這點小事，透過將圖片變成 Template Image 即可搞定 ! 方法如下。

將 asset 裡圖片的 Render As 設為 Template Image

在 Assets.xcassets 裡，將圖片的 Render As 設為 Template Image。到時候圖片呈現時，顏色將由 Tint Color 決定。

以下我們將帥氣 peter 的 Render As 設為 Template Image。

以下我們分別以 SwiftUI & UIKit(storyboard) 示範如何使用 Tint 設定圖片顏色。

SwiftUI

  • image
struct ContentView: View {
    
    var body: some View {
        Image(.peter)
            .resizable()
            .scaledToFit()
            .foregroundStyle(.pink)
    }
}
