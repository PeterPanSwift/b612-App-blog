#243 課程日期計算器

彼得潘平常主要的工作是教課，因此常常需要規劃上課的日期。最近想到不如請 AI 用 Flutter 開發一個課程日期計算器 App，部署到 Firebase 上，即刻變身成方便規劃上課日期的網站。

以下是彼得潘使用 VS Code 搭配 GitHub Copilot(Claude 3.5) 完成的課程日期計算器，有興趣的朋友也可以練習不靠 AI，自己用 SwiftUI、UIKit 或 Flutter 開發一個類似功能的 App。

course_calc
A new Flutter project.my-course-calc.web.app

功能需求

  • 從月曆選擇開始日期。
  • 輸入堂數。
  • 選擇上課日是星期幾，可多選。
  • 可排除台灣的國定假日。

國定假日透過 ruyut 大大提供的 API 抓取。

GitHub - ruyut/TaiwanCalendar: 紀錄中華民國政府行政機關辦公日曆表的 JSON 資料，內容包含日期、星期、是否放假、說明。
紀錄中華民國政府行政機關辦公日曆表的 JSON 資料，內容包含日期、星期、是否放假、說明。. Contribute to ruyut/TaiwanCalendar development by creating an account on…github.com

API 的格式為

https://cdn.jsdelivr.net/gh/ruyut/TaiwanCalendar/data/年份.json

比方以下為 2025 國定假日的 API。

https://cdn.jsdelivr.net/gh/ruyut/TaiwanCalendar/data/2025.json
  • 依據以上條件顯示課程日期清單。
  • 顯示課程日期裡排除的國定假日。
  • 選擇開始日期後，清除之前選擇的上課日，並將上課日更新成開始日對應的星期幾。比方選擇 2025/3/10，上課日更新為星期一。
  • 課程日期包含多個年份時，要抓取多個年份的國家假日。比方開始日為 2024/10/3，堂數 30，上課日為週四時，排除的假日為 2024/10/10、2025/1/30、2025/4/03。
