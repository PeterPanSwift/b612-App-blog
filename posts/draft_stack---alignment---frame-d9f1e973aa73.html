stack 的 alignment 和 frame


stack 裡的東西預設置中於 frame ，可從 frame 的參數 alignment 調整。

struct ContentView: View {
    var body: some View {
        VStack(alignment: .trailing) {
            Image(systemName: "globe")
                .imageScale(.large)
                .foregroundStyle(.tint)
            Text("Hello, world!")
        }
        .frame(width: 300, height: 300, alignment: .bottom)
        .background(.yellow)
    }
}
