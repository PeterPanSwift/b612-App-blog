使用 Icon Composer 製作 Liquid Glass 風格的 App Icon — Xcode 26 新功能

Apple 推出了全新的 Liquid Glass 設計，因此 App Icon 也需要全面更新。為了幫助我們更容易製作多圖層 Liquid Glass 風格的 App Icon，Apple 推出了專門製作 App Icon 的 App Icon Composer。

以下彼得潘算簡單介紹 Icon Composer 的基本操作。

準備 Icon 顯示的圖層圖片

Liquid Glass 風格的 App Icon 通常由多個圖層組成，以下我們從 FLATICON 網站下載火箭和星星的 SVG 檔。

ps: 使用 Icon Composer 製作 Icon 時，常用的圖片格式是 SVG & PNG。

啟動 Icon Composer

Xcode 從 26 開始內建 Icon Composer，從 Xcode 的 Quick Actions 點選 Open Developer Tool > Icon Composer。

點選 File > New。

製作 App Icon

將火箭圖片拖曳到圓角矩形上。

我們可用多個 Group 打造多個圖層，點選左下的 + 可新增 Group。

將星星圖片拖曳到第一個 Group。

Group 由下而上堆疊圖層，因此第一個 Group 的星星會疊在第二個 Group 的火箭上。

點選 Group 和圖片可調整 Group 和圖片的細節。

以下是 Group 的設定欄位。

以下是圖片的設定欄位。點選側邊欄的 star，從 Layout 欄位調整星星的位置和大小。

ps: 從畫布上用拖曳的方式也可移動星星。

點選側邊欄的 rocket-lunch，從 Fill 欄位將火箭改成漸層色。

點選側邊欄的 Icon，從 Fill 欄位調整背景顏色。

點選右下的 Dark icon 預覽 Dark 風格的 Icon。

點選右下的 Mono icon 預覽 Mono 風格的 icon。

點選上方的背景圖 icon 可設定背景圖， 預覽 App Icon 搭配背景圖的呈現效果。

設定好 App Icon 後，按下 cmd + s 儲存，以下我們取名為 FirstAppIcon。

FirstAppIcon 的附檔名為 icon。

將 Icon 加到 Xcode 專案

將剛剛生成的 FirstAppIcon 加到 Xcode 專案。先按 cmd + c 複製 FirstAppIcon，然後在 Xcode 的 Project navigator 按下 cmd + v 貼上。

在 App General 頁面的 App Icon 欄位輸入 icon 檔的檔名，不需要包含附檔名。

ps: 使用 Icon 檔設定 App Icon 後，Assets 的 AppIcon 不用再放圖片。

測試 App Icon

執行 App，桌面出現了我們使用 Icon Composer 精心設計的火箭星星 icon。

Clear 風格。

Tinted 風格。

輸出 App Icon 的 png 檔

Icon Composer 也可輸出 png 格式的 icon。

點選 menu 的 File > Export。

調整 Icon 的相關設定後，點選 Export 輸出。
