使用 jq 指令解析 JSON

開發 App 時我們常常用 curl 串接 API 測試抓到的 JSON 資料，然而 JSON 裡密密麻麻的內容中，我們通常只對某些欄位有興趣。

最近彼得潘在跟 AI 聊天的過程中，學到透過 jq 指令可以方便地截取出 JSON 裡感興趣的欄位。以下我們串接一些實際的 API 說明 jq 的用法。

jq
jq is a lightweight and flexible command-line JSON processorjqlang.org

幸運餅乾 API

https://api.viewbits.com/v1/fortunecookie?mode=random

回傳的 JSON 範例。

{
  "text": "A routine task will turn into an enchanting adventure.",
  "source": "www.best-ever-cookie-collection.com",
  "url": "https://www.best-ever-cookie-collection.com/fortune-cookie-sayings.html",
  "numbers": "81, 68, 67, 70, 34, 43",
  "html": "<blockquote>A routine task will turn into an enchanting adventure.<small>Lucky Numbers: 81, 68, 67, 70, 34, 43</small><footer>Source: <a href=\"https://www.best-ever-cookie-collection.com/fortune-cookie-sayings.html\">www.best-ever-cookie-collection.com</a></footer></blockquote>"
}

利用 jq 將 JSON 變成彩色整齊的縮排。

curl "https://api.viewbits.com/v1/fortunecookie?mode=random" | jq

結果。

{
  "text": "It doesn’t get easier. You get better.",
  "source": "www.abenteuerfreundschaft.de",
  "url": "https://www.abenteuerfreundschaft.de/en/fortune-cookie-sayings/",
  "numbers": "71, 80, 31, 4, 8, 89",
  "html": "<blockquote>It doesn’t get easier. You get better.<small>Lucky Numbers: 71, 80, 31, 4, 8, 89</small><footer>Source: <a href=\"https://www.abenteuerfreundschaft.de/en/fortune-cookie-sayings/\">www.abenteuerfreundschaft.de</a></footer></blockquote>"
}

截取某個欄位，比方利用 jq '.text' 取得 JSON 第一層 key text 的內容。

curl "https://api.viewbits.com/v1/fortunecookie?mode=random" | jq '.text'

結果

"Procrastination is art of keeping up with yesterday."

截取多個欄位也不是問題，比方利用 jq '.text, .numbers' 取得 JSON 第一層 key text & numbers 的內容。

curl "https://api.viewbits.com/v1/fortunecookie?mode=random" | jq '.text, .numbers'

結果。

"You can be trusted to keep a secret."
"22, 55, 99, 34, 9, 86"

輸出時若想包含 key 的名字有兩種寫法。

  • 寫法 1: jq '{text: .text, numbers: .numbers}' 。
curl "https://api.viewbits.com/v1/fortunecookie?mode=random" | jq '{text: .text, numbers: .numbers}'
  • 寫法 2: jq '. | {text, numbers}'。
curl "https://api.viewbits.com/v1/fortunecookie?mode=random" | jq '. | {text, numbers}'

結果。

{
  "text": "An alien of some sort will be appearing to you shortly.",
  "numbers": "69, 85, 12, 33, 24, 89"
}

iTunes API

https://itunes.apple.com/search?term=swift&media=music&country=tw

利用jq '.results' 取得 JSON 第一層 key results 的內容。

curl "https://itunes.apple.com/search?term=swift&media=music&country=tw" | jq '.results'

結果

利用 jq '.results[0]' 取得 array 第一筆資料的內容。

curl "https://itunes.apple.com/search?term=swift&media=music&country=tw" | jq '.results[0]'

利用jq '.results[0].trackName' 取得 array 第一筆資料的 trackName。

curl "https://itunes.apple.com/search?term=swift&media=music&country=tw" | jq '.results[0].trackName'

結果。

"Don't Waste My Time"

利用 jq '.results[0].trackName, .results[0].artistName' 取得第一筆資料的 trackName & artistName。

curl "https://itunes.apple.com/search?term=swift&media=music&country=tw" | jq '.results[0].trackName, .results[0].artistName'

結果。

"Don't Waste My Time"
"DeBarge, SMASH (PT) & Swift"

利用 jq '.results[0] | {trackName, artistName}' 取得第一筆資料的 trackName & artistName 並包含 key 的名字。

curl "https://itunes.apple.com/search?term=swift&media=music&country=tw" | jq '.results[0] | {trackName, artistName}'

結果。

{
  "trackName": "Don't Waste My Time",
  "artistName": "DeBarge, SMASH (PT) & Swift"
}

利用 '.results[:3][] | .trackName' 取得前三筆資料的 trackName。

curl "https://itunes.apple.com/search?term=swift&media=music&country=tw" | jq '.results[:3][] | .trackName'

結果。

"Don't Waste My Time"
"T Style"
"Devil Emoji"

jq 指令還可以搭配 select & test 找出 JSON 裡特定的資料，比方利用 jq '.results[] | select(.trackName | test("love"; "i"))' 找出 trackName 包含 love 的歌，test("love"; "i") 的 i 表示不分大小寫。

curl "https://itunes.apple.com/search?term=swift&media=music&country=tw" | jq '.results[] | select(.trackName | test("love"; "i"))'

GitHub API

https://api.github.com/users/twostraws/followers

API 回傳的資料是 followers 的 array，利用 jq '.[0]' 取得 array 的第一筆資料。

curl "https://api.github.com/users/twostraws/followers" | jq '.[0]'
