儲存任何型別的資料

型別宣告為 Object 或 dynamic。Dart 的所有東西都是物件，數字跟字串也是物件。

Object & dynamic 的差異

dynamic 宣告的變數可存取任何方法或屬性，Object 宣告的變數只能存取Object 的方法或屬性，要轉型才能存取特定型別的方法或屬性。

final list & constant list 的差別。

final 的 list 可以改變，只是不能重新 assign。

void main() { 
  final finalList = [1, 2, 3];
  finalList.add(4);
  print(finalList);
}

constant 的 list 不可以改變，而且也不能重新 assign。

void main() { 
  const constList = [1, 2, 3];
  constList.add(4);
}

以下的 list 也是不可以改變，而且也不能重新 assign。

void main() { 
  final list = const [1, 2, 3];
  list.add(4);
}

constructor

  • default contructor

使用類別名當 constructor 的名字，然後在 ( ) 裡用 this.屬性名將傳入的參數存到屬性。(this.屬性名) 的寫法稱為 Initializing Formals。

以下例子裡，Baby(this.name) 會將傳入的參數存到 Baby 的 property name，因此 Baby('peter') 會讓 name 等於 peter。

class Baby {
  String name;

  Baby(this.name);
}

void main() {
  var cuteBaby = Baby('peter');
  print(cuteBaby.name);
}

若想在物件生成時執行一些程式，也可在 constructor 加上 { }，寫在 { } 裡。

class Baby {
  String name;

  Baby(this.name) {
    print('create baby $name');
  }
}

值得注意的， Dart 要求執行 constructor { } 裡的程式前，所有的 property 都要初始化完畢。因此以下寫法會有問題，顯示錯誤 Non-nullable instance field ‘name’ must be initialized。

class Baby {
  String name;

  Baby(String name) {
    this.name = name;
  }
}

以上問題的詳細說明可參考以下連結。

解法 1: property 一開始有初始值。

當類別有繼承時，子類別的 constructor 也要初始父類別的 property，我們可在 constrctor 的 ( ) 裡用 super.屬性名將傳入的參數存到父類別的屬性。

以下例子裡， SuperBaby(super.name, this.magic) 的 super.name 會將傳入的參數存到 SuperBaby 的 property name，因此 SuperBaby('peter', '點石成金') 會讓 name 等於 peter。

    Baby(this.name);
}

   String magic;
  
   SuperBaby(super.name, this.magic);
}

void main() {
  var cuteBaby = SuperBaby('peter', '點石成金');
  print(cuteBaby.name);
}

Initializer List

我們也可以利用 Initializer List 初始 property，它的語法是在 constructor 之後接冒號，然後撰寫初始 property 的程式，若要初始多個 property 則可以逗號分隔。

以下 Student 的 : average = (english + math) / 2,
 maxScore = max(english, math) 即為 Initializer List 的例子。

import 'dart:math';

class Student {
  double english;
  double math;
  double average;
  double maxScore;

  Student({required this.english, required this.math})
      : average = (english + math) / 2,
        maxScore = max(english, math);
}
 
void main() {
   var cuteStudent = Student(english: 80, math: 100);
   print(cuteStudent.average);
}

我們也可以在 Initializer List 初始父類別的 property，類似以下的 super('可愛的$name')。，因此 SuperBaby('peter', '點石成金') 會讓 name 等於可愛的peter。


class Baby {
    String name;
    
    Baby(this.name);
}

class SuperBaby extends Baby {
   String magic;
  
   SuperBaby(String name, this.magic):
      super('可愛的$name');
}

void main() {
  var cuteBaby = SuperBaby('peter', '點石成金');
  print(cuteBaby.name);
}

named constructors(多個 constructor)

以類別名( )定義的 constructor 只能有一個，因此若想要有多個 constructor，則要搭配 named constructors。

named constructors 的格式是類別名.constructor名。以剛剛的 Baby 為例，我們用 Baby.withDefaultAge(this.name) : age = 1 定義 named constructor，之後再用 Baby.withDefaultAge('Peter') 生成 1 歲的 Peter 寶寶。

class Baby {
  String name;
  int age;

  Baby(this.name, this.age);

  Baby.withDefaultAge(this.name) : age = 1;
}

 
void main() {
   var cuteBaby = Baby.withDefaultAge('peter');
   print(cuteBaby.age);
}

forward constructor

calling the main constructor from the named constructor

class Baby {
  String name;
  int age;

  Baby(this.name, this.age);
  
  Baby.peter(): this('peter', 1);
  Baby.wendy(): this('wendy', 1);
}

void main() {
   var cuteBaby = Baby.peter();
   print(cuteBaby.name);
}

const constructor

若希望生成的物件不能修改，我們可在 constructor 前加上 const，並將 property 設為 final。


class Baby {
  final String name;

  const Baby(this.name);
}

void main() {
  var cuteBaby1 = const Baby('peter');
  print(cuteBaby1.name);
  const cuteBaby2 = Baby('peter');
  print(cuteBaby2.name);
}
