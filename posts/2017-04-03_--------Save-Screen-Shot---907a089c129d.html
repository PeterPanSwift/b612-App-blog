問題:

讓模擬器閃退的 Save Screen Shot !

問題:

我什麼都不要，只是想要螢幕截圖，記錄自己精心製作的 App 畫面，但是只要點選模擬器的 File > Save Screen Shot，模擬器立即閃退，螢幕截圖也因此消失無蹤。

解答:

原來一切都是 Xcode 的 bug。Apple 自己在 Xcode 的 Release Notes 承認了。如果你的 OS X 是 EI Capitan，有可能有這個問題。解決的方法很簡單，你只要升級到 macOS Sierra，即可順利地螢幕截圖。

如果因為某些難言之隱，暫時不能升級到 macOS Sierra，還有一個解決的方法。只要在 Terminal 輸入以下指令，一樣可以實現螢幕截圖。

xcrun simctl io booted screenshot app.jpg
