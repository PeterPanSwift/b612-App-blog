如何查詢型別遵從哪些 protocol

iOS SDK 內建的型別往往有很多功能，遵從了很多 protocol，我們要如何知道它遵從了哪些 protocol 呢 ?

讓我們用以下的 URL 為例說明。

struct ContentView: View {
    let url = URL(string: "https://www.youtube.com/watch?v=0-4mm0e2h44")!
    
    var body: some View {
        VStack {
            Link(destination: url) {
                Text("退後")
            }
            Text("""
    我知道你我都沒有錯
    只是忘了怎麼退後
    信誓旦旦給了承諾
    卻被時間撲了空
""")
        }
        
    }
}

按住 option 鍵後點選 URL，顯示它的說明視窗

滑到說明視窗的底部，點選 Open in Developer Documentation

此時將打開 URL 型別的詳細說明頁面。

點選右側清單的 Relationships

從 Relationships 區塊可看到 URL 遵從了 6 個 protocol。
