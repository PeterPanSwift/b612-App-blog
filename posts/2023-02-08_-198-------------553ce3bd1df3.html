我們可從 Xcode 的 playground 定義 function 練習，function 的參數為 Input，然後對照 Sample 的例子檢查是否正確。

不過若想更精準地判斷程式是否正確，則可登入 CodeChef 帳號，然後在網站上輸入程式，讓 CodeChef 幫我們驗證程式。

在網站練習時記得將語言改成 Swift。

CodeChef 的 Swift 程式如何讀取 Input 跟輸出 Output

  • 範例 1: 只有一行的 Input。

用 readLine 讀取一行的 Input，用 print 輸出 Output。由於題目的 input 一定是數字的字串，因此我們可用 ! 讀取 optional 的內容。

let numberString = readLine()!
let number = Int(numberString)!
let result = number * number / 2
print(result)
  • 範例 2: 多行的 Input。

以上圖為例，迴圈將跑 3 次，inputNumbers 每一次的內容依序為 [1,2,5,2]，[1,2,1,5]，[1,1,8,8]。

let testNumberString = readLine()!
var testNumber = Int(testNumberString)!
while testNumber > 0 {
    let inputLine = readLine()!
    let inputNumbers = inputLine.components(separatedBy: .whitespaces).compactMap(Int.init)    
    
    testNumber -= 1
}
