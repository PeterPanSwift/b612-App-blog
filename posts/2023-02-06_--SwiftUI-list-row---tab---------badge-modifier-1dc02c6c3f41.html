在 SwiftUI list row & tab 顯示輔助資訊的 badge modifier

從 iOS 15 開始，我們可以利用 SwiftUI 的 badge modifier 設定 list row 右邊和 tab 右上顯示的輔助資訊。

呼叫 badge 時我們可以傳入數字，字串或是 Text，以下彼得潘搭配最近得到的超棒生日禮物 Keychron 機械鍵盤舉例說明。

設定 list row 右邊的輔助資訊

定義 List 顯示的 row。

struct KeyboardRow: View {
    let name: String
    
    var body: some View {
        HStack {
            Image(name)
                .resizable()
                .scaledToFit()
                .frame(width: 100)
            Text(name)
            Spacer()
        }
        
    }
}
  • 顯示數字。
struct KeyboardList: View {
    let keyboards = [
        "Q3 Developer",
        "Q3 碳黑",
        "Q3 海軍藍",
        "Q3 銀灰"
    ]
    
    var body: some View {
        List(keyboards, id: \.self) { keyboard in
            KeyboardRow(name: keyboard)
                .badge(3)
        }
        
    }
}
  • 顯示字串。
struct KeyboardList: View {
    let keyboards = [
        "Q3 Developer",
        "Q3 碳黑",
        "Q3 海軍藍",
        "Q3 銀灰"
    ]
    
    var body: some View {
        List(keyboards, id: \.self) { keyboard in
            KeyboardRow(name: keyboard)
                .badge("Keychron")
        }
        
    }
}
  • 顯示 Text。
struct KeyboardList: View {

    var body: some View {
        List {
            KeyboardRow(name: "Q3 Developer")
                .badge(
                    Text(">")
                )
            KeyboardRow(name: "Q3 碳黑")
                .badge(
                    Text("✓")
                )
            KeyboardRow(name: "Q3 海軍藍")
                .badge(
                    Text("ⓘ")
                )
            KeyboardRow(name: "Q3 銀灰")
                .badge(
                    Text("ⓘ>")
                )
        }
        
    }
}

Text 的好處在於可以設定樣式，比方調整顏色和字型。

struct KeyboardList: View {
  
    var body: some View {
        List {
            KeyboardRow(name: "Q3 Developer")
                .badge(
                    Text("❤︎")
                        .foregroundColor(.pink)
                        .font(.system(size: 100))
                )
            
        }
        
    }
}
  • 用 Text 顯示圖片
struct KeyboardList: View {
    let keyboards = [
        "Q3 Developer",
        "Q3 碳黑",
        "Q3 海軍藍",
        "Q3 銀灰"
    ]
    
    var body: some View {
        List(keyboards, id: \.self) { keyboard in
            KeyboardRow(name: keyboard)
                .badge(
                    Text(Image(systemName: "swift"))
                        .foregroundColor(.pink)
                        .font(.system(size: 50))
                )
        }
        
    }
}

設定 tab 右上的輔助資訊

struct ContentView: View {
    
    var body: some View {
        TabView {
            Image("gift")
                .resizable()
                .scaledToFit()
                .tabItem({
                    Label("keyboard", systemImage: "keyboard")
                })
                .badge("Q3")
            
            Image("love story")
                .resizable()
                .scaledToFit()
                .tabItem({
                    Label("music", systemImage: "music.note")
                })
                .badge("99")
        }
        
    }
}
