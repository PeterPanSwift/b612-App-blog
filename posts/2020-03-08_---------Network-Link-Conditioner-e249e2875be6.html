調整模擬器網速的 Network Link Conditioner

開發 iOS App 時，有時我們想測試 App 在不同網路速度執行的情況。改變 iPhone 的網路速度容易些，我們只要到訊號不好的深山，或是利用 Environment Overrides。

Xcode 11 方便 debug 的 Device Conditions & Environment Overrides
Xcode 11 多了許多方便的 debug 功能，現在我們可以透過 Device Conditions & Environment Overrides 測試在不同情況時 App 的運作和畫面呈現，比方測試不同的網路速度，溫度狀態和…medium.com

那麼模擬器的網路速度可以調整嗎 ? 當然也可以，只是比較麻煩，我們必須另外安裝 Network Link Conditioner，方法如下:

點選 Xcode menu 的 Open Developer Tool > More Developer Tools

登入 Apple 開發網站，進入 More Downloads 網頁

搜尋 additional，下載 Additional Tools for Xcode 15

Additional Tools for Xcode 15 包含許多幫助我們開發的小工具，Network Link Conditioner 是其中尊貴的一員。

雙擊 Additional Tools for Xcode 15 的 dmg 檔

雙擊 Hardware 下的 Network Link Conditioner.prefPane 安裝

從 Network Link Conditioner 調整網速

成功安裝 Network Link Conditioner 後，從 Mac 的 System Preferences 點選它，試試調整 Mac 的網路速度吧。

由於模擬器的網速跟 Mac 一致，所以我們調整 Mac 的網速即是調整模擬器的網速。

從 Profile 選單可選擇不同的網速，比方選擇 100% Loss 代表網路完全不通。

最後記得要將開關打開，Network Link Conditioner 設定的網速才會生效，然後要回到正常網速時再將開關關掉。

讓模擬器完全沒有網路

模擬器的網路跟 Mac 一致，所以我們只要將 Mac 斷網，比方將它的 wifi 關掉，也可以讓模擬器沒有網路。
