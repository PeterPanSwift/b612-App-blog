Open Emoji API
A free emoji API sourced directly from unicode.org — always up to date.emoji-api.com

使用 API 前記得先申請 API key。

Open Emoji 主要提供以下五種 API，相關資訊可參考文件的說明。

  • List all emojis
  • Search for emojis
  • Get a single emoji
  • Get categories
  • Get emojis in a category
