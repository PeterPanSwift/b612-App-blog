init 設定 State, StateObject 的問題

跟 State 有關，在 init 裡先給 dices 空的 array，後來再加東西無效

init 設定 State, StateObject 的問題


struct ContentView: View {
    
    @State private var dices: [Dice]

    init() {
        dices = [Dice]()
        for _ in 1...3 {
            let dice = Dice(value: .random(in: 1...6), color: Color(red: .random(in: 0...1), green: .random(in: 0...1), blue: .random(in: 0...1)))
            dices.append(dice)
        }
    }
  • 跟 State 有關，在 init 裡先給 dices 空的 array，後來再加東西無效
  • 原因跟 @State 宣告的變數是 property wrapper 有關。對 property wrapper 有興趣的，可以之後再另外研究
struct ContentView: View {
    
    @State private var dices: [Dice]

    init() {
        var dices = [Dice]()
        for _ in 1...3 {
            let dice = Dice(value: .random(in: 1...6), color: Color(red: .random(in: 0...1), green: .random(in: 0...1), blue: .random(in: 0...1)))
            dices.append(dice)
        }
        self.dices = dices
    }


struct ContentView: View {
    
    @State private var dices = {
            var dices = [Dice]()
            for _ in 1...3 {
                let dice = Dice(value: .random(in: 1...6), color: Color(red: .random(in: 0...1), green: .random(in: 0...1), blue: .random(in: 0...1)))
                dices.append(dice)
            }
            return dices
    }()
