SwiftUI 的 MeshGradient — iOS 18 新功能

SwiftUI 在 iOS 18 推出了新的漸層效果 MeshGradient，它將畫面切成一格格的網格，搭配每個網格設定的顏色和控制點，製作出美麗的漸層效果。

我們可以把繪製漸層的長方形區塊想像成以下圖片，它的座標系統為左上是 0,0，右上是 1,0，左下是 0,1、右下是 1,1。

以下我們將它切成 2 * 2 的網格，設定 4 個顏色生成 MeshGradient，程式如下。

import SwiftUI

struct ContentView: View {
    var body: some View {
        MeshGradient(width: 2, height: 2, points: [
            .init(0, 0), .init(1, 0),
            .init(0, 1), .init(1, 1),
        ], colors: [
            .red, .blue,
            .green, .yellow,
        ])
    }
}
  • 參數 width & height。

傳入 2，代表水平方向 2，垂直方向 2，因此會產生 2 * 2 的網格。

  • 參數 points。

顏色控制點的 array，這些點控制顏色的位置。網格是 2 * 2，因此我們需要傳入4 個控制點，位置分別是左上(0,0)、右上( 1,0)、左下( 0,1) 和右下(1,1)。

  • 參數 colors。

顏色的 array，網格是 2 * 2，因此我們需要傳入4 個顏色。搭配參數 points 指定的位置，代表紅色在左上，藍色在右上，綠色在左下，黃色在右下。

控制點的位置將影響生成的漸層效果，比方將 blue 的位置設為 (0.5, 0)，藍色將變成落在水平一半的位置。

import SwiftUI

struct ContentView: View {
    var body: some View {
        MeshGradient(width: 2, height: 2, points: [
            .init(0, 0), .init(0.5, 0),
            .init(0, 1), .init(1, 1),
        ], colors: [
            .red, .blue,
            .green, .yellow,
        ])
    }
}

理解 2 * 2 的網格後，以下我們再看個 3 * 3 的網格，由 9 個顏色組成的 MeshGradient。

import SwiftUI

struct ContentView: View {
    var body: some View {
        MeshGradient(width: 3, height: 3, points: [
            .init(0, 0), .init(0.5, 0), .init(1, 0),
            .init(0, 0.5), .init(0.5, 0.5), .init(1, 0.5),
            .init(0, 1), .init(0.5, 1), .init(1, 1)
        ], colors: [
            .red, .purple, .indigo,
            .orange, .white, .blue,
            .yellow, .green, .mint
        ])
    }
}

參數 width 和 height 不見得要一樣，比方以下為 3 * 2 的網格。

MeshGradient 套用在文字和圖片上也很美麗，例如以下例子。

struct ContentView: View {
    var body: some View {
        Text("愛瘋一切為蘋果的彼得潘")
            .font(.largeTitle)
            .foregroundStyle(
                MeshGradient(width: 3, height: 3, points: [
                    .init(0, 0), .init(0.5, 0), .init(1, 0),
                    .init(0, 0.5), .init(0.5, 0.5), .init(1, 0.5),
                    .init(0, 1), .init(0.5, 1), .init(1, 1)
                ], colors: [
                    .red, .purple, .indigo,
                    .orange, .white, .blue,
                    .yellow, .green, .mint
                ])
            )
    }
}

asset 的圖片也可以套用 MeshGradient，只要將它的 Render As 設為 Template Image。

設定圖片顏色，將圖片變成 Template Image
button 或 image view 的圖片想要換顏色，其實不一定要苦苦哀求美術設計。變色這點小事，透過設定 Tint Color 即可搞定 ! 方法如下:medium.com
import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Image("peter")
                .resizable()
                .scaledToFit()
                .frame(width: 200)
            Text("愛瘋一切為蘋果的彼得潘")
        }
        .foregroundStyle(
            MeshGradient(width: 3, height: 3, points: [
                .init(0, 0), .init(0.5, 0), .init(1, 0),
                .init(0, 0.5), .init(0.5, 0.5), .init(1, 0.5),
                .init(0, 1), .init(0.5, 1), .init(1, 1)
            ], colors: [
                .red, .purple, .indigo,
                .orange, .white, .blue,
                .yellow, .green, .mint
            ])
        )
    }
}
