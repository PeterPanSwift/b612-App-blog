#186 適合新手練習串接網路 API 的 Random name — 回傳字串

最近研究了一下適合新手練習網路程式的 API。由於網路程式有點複雜，像是抓取 JSON & 圖片還要學習如何解析資料，因此想找尋是否有簡單回傳字串的 API，讓新手練習抓到字串後顯示在畫面上。

#186 回傳字串的簡單 API

最近彼得潘研究了一下是否有比 JSON 格式簡單的 API，發現有一些回傳簡單字串的 API，很適合新手練習抓到字串後顯示在畫面上。

有興趣的朋友可參考以下回傳字串的簡單 API。

  • wttr.in 的天氣 API。
GitHub - chubin/wttr.in: :partly_sunny: The right way to check the weather
partly_sunny: The right way to check the weather. Contribute to chubin/wttr.in development by creating an account on…github.com

抓取台北目前的天氣。

https://wttr.in/Taipei?format=3

範例結果。

Taipei: ⛅️  +16°C
