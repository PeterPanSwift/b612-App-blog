在公司上班時，我們時常會跟後台工程師合作，開發有著註冊登入功能，透過 REST API 上傳下載資料的 iOS App。

因此想找工作轉職當 iOS 工程師的朋友，最好能練習製作一個有著這樣功能的 App。新手通常不會有幫忙寫後台程式的朋友，不過沒關係，關於資料的上傳和下載，我們可以利用 AirTable & Imgur API 實現。

那帳號的註冊登入功能呢 ? 業界的 App 通常會提供兩個 API，分別做註冊跟登入功能，註冊跟登入成功時 API 會回傳一個 token，App 之後上傳下載資料都需要將 token 帶在 request 裡，比方在 HTTP header 加入 token。

最近彼得潘研究了一下，FavQs API 剛好有提供註冊和登入的 API，而且也會在成功時回傳 token。因此我們可以結合 FavQs ，AirTable & Imgur API 製作轉職的 App 作品。

以製作類似 IG 的照片分享 App 為例，我們可透過以下方式開發 App:

  • 以 FavQs API 開發註冊登入功能，註冊登入成功時取得 token，將 token 存在 App 裡。
  • 啟動 App 時以 token 判斷使用者是否登入。
  • 以 AirTable & Imgur API 實現上傳 po 文和讀取動態牆的功能。在串接 AirTable & Imgur API 時，在 HTTP header 加入 token。(ps: 加入 token 的目的只是為了模仿未來上班工作的程式，到時串接公司的後台 API 通常會需要帶 token。在串接 AirTable & Imgur API 時，我們加入的 token 不會有任何作用，因為它是來自 FavQs API 的 token。)
  • 註冊登入的 App 畫面可以請 AI 設計。

接下來我們簡單說明 FavQs API 的使用方式。

註冊 FavQs 帳號

我們必須有 FavQs 的 API key 才能串接註冊 & 登入 API，因此請先註冊 FavQs 帳號，有了帳號才能申請 API key。

連到 FavQs API 網站，點選 Sign up。

產生 API key

連到以下網頁，點選 Generate API Key。

https://favqs.com/api_keys

複製下圖的 Key，之後串接註冊 & 登入 API 會用到。
