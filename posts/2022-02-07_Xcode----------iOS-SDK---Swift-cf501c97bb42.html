安裝需求

Xcode 26 搭配的 macOS 至少要 15.6，Xcode 16.4 搭配的 macOS 至少要 15.3。有些朋友因為 mac 比較舊，無法升級到最新版的 macOS，因此只能委屈地安裝舊版的 Xcode。從 macOS 的 wiki 可查詢 mac 能安裝的 macOS， 比方下圖顯示 2018 之後的 Macbook Pro 才能安裝 macOS 15。

macOS - Wikipedia
macOS (; previously Mac OS X and later OS X) is a proprietary graphical operating system developed and marketed by…en.wikipedia.org

iOS SDK

iOS SDK 決定 Xcode 開發 App 時能使用的功能，從下圖的 SDK 可查詢 Xcode 搭配的 iOS SDK。Xcode 26 搭配 iOS 26，因此能使用 iOS 26 SDK 的新功能。若是使用 Xcode 16，將無法使用 iOS 26 SDK 的新功能。

以 WebView 為例，它在 iOS 26 推出，因此只有搭配 iOS 26 以上 SDK 的 Xcode 能使用。

Swift

從下圖的 Swift 可查詢 Xcode 搭配的 Swift 版本，比方 Xcode 26 支援 Swift 6.2。

Swift 版本影響我們可使用的 Swift 語法。比方 async & await 是 Swift 5.5 推出的新功能，因此 Xcode 13 之後的版本才能使用。
