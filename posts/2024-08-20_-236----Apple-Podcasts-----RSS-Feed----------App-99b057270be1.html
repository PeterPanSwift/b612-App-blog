最近 Apple 推出網頁版的 Podcasts，現在我們從網頁就能聆聽喜歡的 podcast 節目。

Apple Podcasts
Browse your favorite podcast shows, genres, and hosts on Apple Podcasts. Download and listen from any Apple device or…podcasts.apple.com

網頁版的 Apple Podcasts 還帶給我們另一個好處，我們可以從網頁查出 podcast 音檔的網址，因此有興趣的朋友可以練習開發播放 podcast 的 App。

以下我們將介紹兩種查詢音檔網址的方法，先介紹從 Apple Podcasts 網站查詢單集節目的音檔，再介紹從 Podcast Feed 的 XML 查詢多集節目的音檔。

從 Apple Podcasts 網站查詢單集節目的音檔網址

我們以 podcast The Little Prince: Chapter 1 — Pictures 為例，說明如何找出它的音檔網址。

The Little Prince: Chapter 1 - Pictures
CHAPTER ONE PICTURES One day, I drew a picture of a snake with an elephant inside it. "Why?" you ask me. This is my…podcasts.apple.com

連到 podcast 的網頁，點選 Play 播放。

在網頁上叫出右鍵選單，點選 Inspect Element。

ps: 如果沒看到 Inspect Element，請先打開 Safari 的 Settings，勾選 Advanced 頁面的 Show features for web developers。

搜尋 .mp3。

找到音檔網址後，在網址上叫出右鍵選單，點選 Copy Link。

成功取得 The Little Prince: Chapter 1 — Pictures 的音檔網址。

從 podcast 的 RSS Feed 查詢多集節目的音檔網址

剛剛介紹的方法只能一次查詢一集節目的音檔，若想一次取得多集節目的音檔，則可透過 podcast 的 RSS Feed。RSS 是一種透過 XML 制定的資料格式，從 RSS Feed 我們可查詢 podcast 多集節目的詳細資訊。

以剛剛的小王子為例，它在 Apple Podcasts 網站的網址如下。

https://podcasts.apple.com/us/podcast/the-little-prince-uma-leitura-guiada-em-ingl%C3%AAs/id1586819833

從網址我們得知它的 id 是 1586819833，因此我們可用 https://itunes.apple.com/lookup?id= 加上 1586819833 查詢它的詳細資訊。

https://itunes.apple.com/lookup?id=1586819833

甚至我們還可以請 AI 寫解析 RSS Feed XML 的 pyton 程式，輸出包含音檔網址的 JSON 資料。

import xml.etree.ElementTree as ET
import json
import requests

def parse_rss_feed(rss_url):
    # Fetch the RSS feed from the URL
    response = requests.get(rss_url)
    response.raise_for_status()  # Check if the request was successful
    
    # Parse the RSS content
    root = ET.fromstring(response.content)

    # Extract audio file URLs and related information
    episodes = []

    for item in root.findall('./channel/item'):
        episode = {}
        
        title = item.find('title').text if item.find('title') is not None else "No Title"
        description = item.find('description').text if item.find('description') is not None else "No Description"
        pub_date = item.find('pubDate').text if item.find('pubDate') is not None else "No Publication Date"
        enclosure = item.find('enclosure')
        audio_url = enclosure.get('url') if enclosure is not None else "No URL"

        episode['title'] = title
        episode['description'] = description
        episode['pub_date'] = pub_date
        episode['audio_url'] = audio_url

        episodes.append(episode)

    return episodes

def save_as_json(data, output_file_path):
    with open(output_file_path, 'w') as json_file:
        json.dump(data, json_file, indent=4)

if __name__ == "__main__":
    # Replace with your RSS feed URL
    rss_url = 'https://www.inglesessencial.com/the-little-prince-uma-leitura-guiada-em-ingls?format=rss'
    
    # Parse the RSS feed
    episodes_info = parse_rss_feed(rss_url)
    
    # Output file path for JSON
    json_output_path = 'output_episodes_info.json'
    
    # Save as JSON
    save_as_json(episodes_info, json_output_path)
    
    print(f"JSON file has been created at: {json_output_path}")
