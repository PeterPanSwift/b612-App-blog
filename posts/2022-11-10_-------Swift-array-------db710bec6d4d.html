在迴圈裡修改 Swift array 成員的內容

在迴圈裡修改 array 成員的內容是開發 App 時常遇到的功能，不過最近有同學遇到問題，接下來我們就來看看同學遇到的問題和解決的方法吧。

假設情人的型別是以 struct 定義的 Lover，我們以 array lovers 儲存 2 個情人。

struct Lover {
    var name: String
    var weight: Double
    var height: Double
}

var lovers = [
    Lover(name: "Sandy", weight: 50, height: 160),
    Lover(name: "Wendy", weight: 55, height: 155),
]

五月天愛情的模樣說愛情有謎樣的魔力，也許它可以讓談戀愛的人長高 10 公分。

於是我們利用 for loop 將每個情人的身高增加 10 公分。不過很不幸，紅色錯誤出現了，它說 lover 是不能改變的常數。

錯誤的原因是因為寫在 for in 中間的 lover 預設是不能修改的常數，而且型別 Lover 是 value type，因此我們無法修改它的 property。

此問題可在 for 後加上 var 解決，讓 lover 變成可變的變數。

for var lover in lovers {
    lover.height += 10
}
print(lovers)

但是當我們將 array lovers 印出時，卻發現情人的身高完全沒變。難道是愛情已失去魔力 ?

我們要相信愛情呀，愛情還是有魔力的。問題出在 Lover 是 value type，因此我們在迴圈裡修改的是複製的情人，不是 array 裡的情人。

Swift 的 value type 和 reference type
在 Swift 裡，型別分成 value type 和 reference type。struct，enum 和 tuple 是 value type，而 class，function 和 closure 則是 reference…medium.com

要修改 array 裡的情人，有以下幾種解法。

解法 1

for i in lovers.indices {
    lovers[i].height += 10
}
print(lovers)

解法 2

for var (i, lover) in lovers.enumerated() {
    lover.height += 10
    lovers[i] = lover
}
print(lovers)

解法 3

lovers = lovers.map {
    var lover = $0
    lover.height += 10
    return lover
}
print(lovers)

解法 4

除了以上三種解法，我們也可以改用 class 定義 Lover，如此情人將變成 reference type，我們將可在 for loop 裡直接修改 array 成員的內容。

class Lover {
    var name: String
    var weight: Double
    var height: Double
    
    init(name: String, weight: Double, height: Double) {
        self.name = name
        self.weight = weight
        self.height = height
    }
}

var lovers = [
    Lover(name: "Sandy", weight: 50, height: 160),
    Lover(name: "Wendy", weight: 55, height: 155),
]
for lover in lovers {
    lover.height += 10
}
for lover in lovers {
    print(lover.name, lover.height)
}
