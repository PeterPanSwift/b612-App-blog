使用線上 IDE Firebase Studio 開發 Flutter App

Google 推出了結合 AI 的線上 IDE Firebase Studio，以後只要在有網路的地方，我們可以用任何電腦打開瀏覽器寫程式，不用再花時間裝軟體準備開發環境了。

Firebase Studio

Firebase Studio 有著類似 VS Code 的操作介面，它可以開發許多程式，比方 Flutter、Android、Go、Python、Node、Java、.NET 等，而且它還搭配 Google 的 AI Gemini 輔助我們寫程式呢。

Safari 必須先允許 third-party cookies

使用 Safari 操作 Firebase Studio 的朋友，必須先允許 third-party cookies，方法如下。

點選 Safari 的 Settings。

取消 Advanced > Block all cookies 的勾選。

取消 Privacy > Prevent cross-site tracking 的勾選。

使用 Firebase Studio

連到 Firebase Studio，點選 Try Firebase Studio 或 Get Started 登入 Google 帳號。

Firebase Studio

登入成功後，同意相關的服務條款後按 Confirm。

可以開始用 Firebase Studio寫程式了 !

開發 Flutter App

建立 Flutter App 專案。

  • 方法 1: 點選 Flutter icon。
  • 方法 2: 點選 + New Workspace。

選擇 Mobile 的 Flutter。

設定 workspace 名稱後按 Create。

成功建立 Flutter 專案，Firebase Studio 的介面跟 VS Code 類似，開發體驗就像使用線上版的 VS Code。請稍等片刻，等待專案完成初始設定後，將在右邊的 Preview 頁面看到 Android & Web 的 App 畫面。

使用 Firebase Studio 就像使用 VS Code，同樣能用快速鍵(cmd + shift + p)叫出 Command Palette 輸入指令。

同樣有方便的自動完成。

同樣有方便的 Quick Fix 選單。

修改程式後會自動 hot reload。

從下圖的選單可點選 Hot Reload、Full Reload 或 Hard Restart。

上傳檔案、圖片也很方便，有以下幾種方法。

  • 在 Explorer 區塊按 cmd + v 貼上。
  • 拖曳檔案。
  • 從右鍵選單點選 Upload。

Run and Debug App

除了從 preview 操作 App，我們也可以另外啟動模擬器執行 App。若想從 Debug Console 查看 print 列印的訊息，記得要啟動模擬器執行 App，預設的 preview 模式將無法看到 print 列印的訊息。

  • 方法 1: 點選 main.dart 後，點選 function main 上方的 Run 或 Debug。

執行 App 跟 preview 模式不同，修改程式後必須按 cmd + s 存檔才會觸發 hot reload 更新 App。

  • 方法 2: 點選上方的三角形按鈕。
  • 方法 3: 點選側邊欄的三角形按鈕，然後點選 Run and Debug。

使用 AI 功能

Firebase Studio 的另一個強大功能是 AI 程式助手，初次使用前請先點選下方的 Gemini。(快速鍵 Cmd + Shift + Space)

可以開始跟 AI 聊天了 !

點選右下的 model 箭頭可選擇 AI model。

點選下圖的箭頭可切換 Agent、Agent (Auto-run) & Ask 模式。Agent 模式可直接修改程式和執行指令，Agent (Auto-run) 則會自動執行指令，不會在執行指令前詢問我們是否執行。

在下方的框框輸入 prompt，比方請它說明 Scaffold。

切換成 Agent 模式，請 AI 寫一個打地鼠 App，點選 Update File 即可在專案套用 AI 修改的程式。

任何程式問題都可以問它，雖然 Firebase Studio 不能開發 SwiftUI，但是它也是略懂 SwiftUI。

點選 attach 按鈕可上傳圖檔或畫圖。

我們也可以按下 cmd + i 叫出 AI 聊天框框輸入 prompt。

點選 Accept 可輸入 AI 建議的程式。

我們也可以選取某段程式，然後從 Command Palette 點選 Gemini 的相關指令。

當我們輸入程式時，AI 也會馬上提供建議，按 tab 可輸入 AI 的建議。

Firebase Studio 的 workspace 上限

Firebase Studio 預設只能建立 3 個 workspace，達到上限時可點選網頁的 Join 加入 Google Developer Program，它可將上限提高到 10 個。

若想要更多的 workspace，可訂閱 Google Developer Program Plans，將 workspace 的上限提高到 30 個。

One More Thing，寫寫 Python & C

我們也可以用 Firebase Studio 寫一些簡單的 Python & C 程式，方法如下。

建立 workspace 時選擇 Misc 的 Blank Workspace。

  • Python 程式。

新增 python 檔。

請 AI 幫忙寫個計算數字 1 加到 10 的 main function。

從下方的 TERMINAL 視窗執行 python 程式。

  • C 程式。

新增 c 檔，請 AI 寫一個 main function，計算數字 1 加到 10。

完整的程式如下。

#include <stdio.h>

int main() {
  int sum = 0;
  for (int i = 1; i <= 10; i++) {
    sum += i;
  }
  printf("The sum of numbers from 1 to 10 is: %d\n", sum);
  return 0;
}

從下方的 TERMINAL 視窗用 gcc 編譯 C 程式。雖然 gcc 沒有安裝，我們還是可以從 package 清單選擇 gcc。

編譯 C 程式後將產生執行檔 a.out，輸入 ./a.out 即可執行程式。
