Xcode 的 GitHub 帳號設定

打開 Settings 視窗

從 Quick Actions 輸入框打開 Settings。

新增帳號

切換到 Source Control 頁面，點選 Add Account。

設定 GitHub 帳號

在帳號設定的選單裡，我們有多種不同的帳號可選擇，在此我們選擇 GitHub。

輸入 GitHub 的 account & token。等等，我們只知道密碼，不知道 token 呀。別擔心，請點選下圖的 Create a Token on GitHub。

點選後將開啟新增 token 的網頁。

ps: 也可以從以下連結進入新增 token 的網頁。

https://github.com/settings/tokens/new

  • 設定 token。

輸入自訂的 token 名字，在 Expiration 設定 token 過期的時間(ps: 若是想要 token 永不過期，請選擇 No expiration)，然後勾選 repo，如此才能擁有基本的權限，之後即可從 Xcode 將程式上傳到 GitHub。

點選下方的 Generate Token。

經歷千辛萬苦，終於產生 token。請將寶貝的 token 複製起來。

回到剛剛 Xcode 的畫面輸入 account & token，點選 Sign In 登入。

大功告成 ! Accounts 下順利出現 GitHub 帳號。
