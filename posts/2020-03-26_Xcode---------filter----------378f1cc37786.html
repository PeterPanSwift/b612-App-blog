Xcode 新增檔案時利用 filter 快速選取檔案類型

開發 iOS App 時，我們時常點選 cmd + n 新增檔案，然後從視窗裡選擇想要新增的檔案類型，比方 Cocoa Touch Class 或 SwiftUI View 。

如果一秒鐘幾十萬上下，想要快速選到想要的檔案類型，可以利用以下方法實現。

  • 輸入搜尋的關鍵字。

在右上方的 filter (過濾)框框輸入文字，比方輸入 ui。

  • 先按 tab 跳到下方的檔案類型區塊，然後按方向鍵選到目標後按 Enter。

在此我們按向下的方向鍵移動，從 UI Test Case Class 跳到 SwiftUI View。
