利用 table view controller 的 table view header 快速製作捲動頁面

開發 iOS App 時， scroll view ，table view 和 collection view 都能實現畫面捲動，瀏覽超出螢幕內容的功能。但如果一行程式都不想寫，只想快速做個 prototype，利用 table view controller 的 table view header，可說是最快的方法了 !

table view controller 控制的畫面是表格，本來就可以上下捲動。我們只要加入一個 view，讓它成為長在表格前面的 table view header，然後將此 view 的大小設為想要顯示的內容大小，最後再將想顯示的內容加到 view 上即可。

接下來，就讓我們一步步實做一個上下捲動的照片牆吧。

1 在 storyboard 加入 table view controller。
2 將 cell 刪除。

我們不需要 cell，果斷地將它刪除吧。

3 將 view 加到畫面上，讓它長在 table view 底下，成為 table view footer。
4. 調整 view 的高度，設為欲顯示的內容高度。
5. 在 view 上加入想顯示的內容。

加入元件時，我們還可上下捲動畫面，將元件放到想要的位置。

ps: 如果想在設計畫面時一次看到全部的內容，可參考以下連結，調整畫面的大小。
