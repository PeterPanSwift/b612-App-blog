將 SPM package 的 library 加到專案

利用 SPM 我們可以方便地安裝各種強大的第三方套件，比方下圖的 Firebase package。

使用 SPM 安裝第三方套件 — 以抓圖套件 Kingfisher 為例
開發 iOS App 時，我們有許多管理安裝第三方套件的方法，其中最為人熟知的三大套件管理天王為 CocoaPods，SPM(Swift Package Manager) & Carthage。medium.com

不過一個 package 裡可能包含多個 library，假設我們原本只加入 Firebase package 裡的 FirebaseFirestoreSwift-Beta，之後如果還想加入 Firebase 其它的 library，要怎麼加呢 ?

方法如下

  • 點選 Target 下的 App，切換到 General 分頁。
  • 點選 Frameworks, Libraries, and Embedded Content 下的 +。
  • 選擇想加入的對象，比方 FirebaseStorageSwift-Beta。

成功加入 ! 從下圖可看到 Frameworks, Libraries, and Embedded Content 新增了剛剛加入的 FirebaseStorageSwift-Beta。
