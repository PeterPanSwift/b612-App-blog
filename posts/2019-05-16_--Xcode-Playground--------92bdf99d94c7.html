在 Xcode Playground 測試動畫效果

為了製作完美的 App 動畫效果，我們往往需要微調許多動畫的細節，因此 Xcode Playground 是個不錯的動畫開發工具。我們可以在上面快速地修改程式，從 Assistant editor 看到動畫效果，等調到滿意的效果後再將程式加入 Xcode 專案。

在 Xcode Playground 測試動畫的關鍵在於 PlaygroundPage 的 liveView，比方以下例子:

import UIKit
import PlaygroundSupport
let liveView = UIView(frame: CGRect(x: 0, y: 0, width: 300, height: 300))
liveView.backgroundColor = UIColor.yellow
PlaygroundPage.current.liveView = liveView
let blueView = UIView(frame: CGRect(x: 10, y: 10, width: 50, height: 50))
blueView.backgroundColor = UIColor.blue
liveView.addSubview(blueView)
UIViewPropertyAnimator.runningPropertyAnimator(withDuration: 3, delay: 0, options: [], animations: {
   blueView.transform = CGAffineTransform(translationX: 100, y: 100)
}, completion: nil)

PlaygroundPage 有個神奇的 liveView，可以透過 Assistant editor 顯示。因此我們可將 PlaygroundPage.current.liveView 設為某個 view 或 controller，然後再打開 Assistant editor，從 Assistant editor 看到 view 或 controller 的畫面。

一般簡單的動畫只跟 view 有關，還不須動用到 controller。因此測試動畫時，最簡單的方法莫過於將 PlaygroundPage.current.liveView 設為某個固定大小的 UIView 元件，然後再將動畫的元件加到 liveView 上後，即可開始撰寫動畫的程式。

因此剛剛的程式碼裡，我們將 300 * 300 的黃色 view 設為 liveView，然後再將 blueView 加到黃色 view 上，利用動畫程式讓 blueView 移動。

let liveView = UIView(frame: CGRect(x: 0, y: 0, width: 300, height: 300))
liveView.backgroundColor = UIColor.yellow
PlaygroundPage.current.liveView = liveView
