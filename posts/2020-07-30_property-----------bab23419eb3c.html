property 設定內容的幾種方法

開發 Swift iOS App 時，我們時常設定 property 的內容，在 = 的右邊生成某個型別的資料，將它存入 property 裡。

Swift 生東西的幾種方法

開發 Swift iOS App 時，我們時常生成東西，然後將它存入 property 或當成 function 的參數傳入。

接下來我們以設定 property 的內容為例，介紹 Swift 生東西的幾種方法:

property 的型別是基本的 String，Int，Float，Bool

  • 設定字串。
let loveLabel = UILabel(frame: CGRect(x: 0, y: 0, width: 300, height: 40))
loveLabel.backgroundColor = UIColor(red: 1, green: 1, blue: 0, alpha: 1)
loveLabel.text = "在天願作比翼鳥，在地願為連理枝。"

text 的型別是 String?，因此直接存入字串。

  • 設定數字。
let slider = UISlider()
slider.value = 0.5

value 的型別是 Float，因此直接存入浮點數。

從自動完成的型別提示，以型別( ) 生成東西

  • 設定顏色。
let redView = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
redView.backgroundColor = UIColor(red: 1, green: 0, blue: 0, alpha: 1)

從自動完成的提示，我們明白 backgroundColor 的型別是 UIColor?，因此要生成型別 UIColor 的資料存入 backgroundColor。

利用型別( ) 可生成某型別的東西，因此我們在 = 的右邊輸入 UIColor。

選擇參數搭配 red，green，blue，alpha 的選項。

  • 設定圖片。
let wifiImageView = UIImageView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
wifiImageView.image = UIImage(systemName: "wifi")

image 的型別是 UIImage?，因此以 UIImage( ) 生成資料存入。

以型別 property 生東西，輸入 . 從自動完成選單選擇

許多常用的東西會被定義成型別 property，方便我們快速產生，比方產生紅色的程式是 UIColor.red，因此我們可用以下程式設定紅色。

let redView = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
redView.backgroundColor = UIColor.red

當我們從型別 A 存取型別 property 時，若是已知是型別 A，而且 property 也是型別 A 時，則可省略型別。

Swift type property，type method ＆ enum 的型別省略
使用 Swift 的 type property，type method ＆ enum 時，型別省略可以讓我們的手指輕鬆不少，少打一些字。medium.com

因此剛剛的程式有更簡單的寫法，我們可在 = 後先輸入 . ，然後從自動完成選單選擇 red。

let redView = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
redView.backgroundColor = .red

以 enum case 生東西，輸入 . 從自動完成選單選擇

有些 property 的型別以 enum 定義，因此我們存入的內容將為 enum 的某個 case。我們可在 = 後輸入 .，自動完成選單以格子 K 顯示的即為 enum 的 case。

  • 設定文字對齊方式。
let loveLabel = UILabel(frame: CGRect(x: 0, y: 0, width: 300, height: 40))
loveLabel.backgroundColor = UIColor(red: 1, green: 1, blue: 0, alpha: 1)
loveLabel.text = "在天願作比翼鳥"
loveLabel.textAlignment = .center

選擇 .center，代表文字置中。

.center 其實省略了型別 NSTextAlignment，我們也可以寫成 NSTextAlignment.center。

loveLabel.textAlignment = NSTextAlignment.center
  • 設定文字框樣式。
let textField = UITextField(frame: CGRect(x: 0, y: 0, width: 300, height: 40))
textField.borderStyle = .roundedRect

選擇 .roundedRect，設定圓角框。

以 .init 生東西

型別() 生東西等同於型別.init()，因此我們也可以用 .init 生東西存入 property。

  • 設定顏色。
let redView = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
redView.backgroundColor = UIColor.init(red: 1, green: 0, blue: 0, alpha: 1)
redView.backgroundColor = .init(red: 1, green: 0, blue: 0, alpha: 1)

以 .ini 開頭，比方輸入 .inir。

我們也可以在 . 後輸入參數名找到目標的 init，比方 .redg。

.init 其實省略了型別 UIColor，我們也可以寫成 UIColor.init。

redView.backgroundColor = UIColor.init(red: 1, green: 0, blue: 0, alpha: 1)
  • 設定中心點座標。
let redView = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
redView.backgroundColor = .init(red: 1, green: 0, blue: 0, alpha: 1)
let blueView = UIView(frame: CGRect(x: 0, y: 0, width: 50, height: 50))
blueView.backgroundColor = .init(red: 0, green: 0, blue: 1, alpha: 1)
blueView.center = .init(x: 50, y: 50)
redView.addSubview(blueView)
redView

輸入 .x。

  • 設定旋轉。
let redView = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
redView.backgroundColor = .init(red: 1, green: 0, blue: 0, alpha: 1)
let blueView = UIView(frame: CGRect(x: 0, y: 0, width: 50, height: 50))
blueView.backgroundColor = .init(red: 0, green: 0, blue: 1, alpha: 1)
blueView.center = .init(x: 50, y: 50)
blueView.transform = .init(rotationAngle: .pi / 4)
redView.addSubview(blueView)
redView

輸入 .ro。
