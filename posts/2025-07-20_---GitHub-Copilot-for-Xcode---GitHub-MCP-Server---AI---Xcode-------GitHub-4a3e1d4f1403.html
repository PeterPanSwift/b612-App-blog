安裝 GitHub Copilot for Xcode

參考以下連結安裝 GitHub Copilot for Xcode。

使用 GitHub Copilot for Xcode 幫忙寫程式
最近聽說 GitHub 推出了厲害的 AI 程式碼工具 Copilot，它可以提供不錯的程式碼建議，大幅加快程式的開發速度。medium.com

加入 GitHub MCP Server

打開 Xcode 的 SwiftUI App 專案，從 Quick Actions 點選 GitHub Copilot > Open Chat。

出現 Copilot 的聊天視窗。

切換到 Agent mode，點選 Tool Picker 按鈕設定 MCP server。

點選 MCP 頁面的 Edit Config 開啟 mcp.json。

在 servers 的 { } 加入 GitHub MCP Server，YOUR_GITHUB_PAT 換成真正的 GitHub token。

{
  "servers": {
    "github": {
      "url": "https://api.githubcopilot.com/mcp/",
      "requestInit": {
        "headers": {
          "Authorization": "Bearer YOUR_GITHUB_PAT"
        }
      }
    }
  }
}

ps: 從以下網址可建立 GitHub token，複製 token 後取代剛剛 mcp.json 的 YOUR_GITHUB_PAT。

https://github.com/settings/tokens/new

成功加入 GitHub MCP Server 後，MCP 頁面的 Available MCP Tools 出現了 MCP Server: github。

請 AI 將 Xcode 專案上傳到 GitHub

以下彼得潘將專案做了些修改，App 畫面顯示一手打 Mac，一手打桌球的 peter。

從 Quick Actions 點選 GitHub Copilot > Open Chat。

切到 Agent mode，輸入 Create GitHub repository & push project to GitHub。

AI 開始一步步執行任務，大部分的時候我們只要無腦點選 Continue 同意 AI 執行任務。

若想調整則可點選 Cancel。比方我們的 Xcode 專案在建立時已勾選 Source Control，因此不用再做 git init。

告訴 AI 我們已做過 git init。
