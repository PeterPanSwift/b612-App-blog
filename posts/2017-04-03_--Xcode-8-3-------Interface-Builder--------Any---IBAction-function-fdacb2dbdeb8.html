在 Xcode 8.3，終於可以從 Interface Builder 連到參數型別 Any 的 IBAction function

問題:

當我們從 Interface Builder 連線到程式碼，將 Connection 選為 Action，輸入名稱後，將自動生成參數型別 Any 的 IBAction function 。然而當我們將連線刪除，想再從 Interface Builder 連線到 function 時，8.3 之前的 Xcode 是不允許的。明明 function 就在面前，我們卻連不到 !

@IBAction func showBigPhoto(_ sender:Any) {
}

在 Xcode 8.3 之前，如果我們想從 Interface Builder 裡的 button 連線到程式碼裡的 function， function 的參數型別必須宣告為 UIButton，否則無法連線。

在 Xcode 8.3 ，如此奇怪的魔咒終於被解除了。當參數的型別是 Any 時，我們依然可以順利連線 !

Connecting @IBAction methods from Interface Builder to Swift code now works when the sender parameter type is Any.
