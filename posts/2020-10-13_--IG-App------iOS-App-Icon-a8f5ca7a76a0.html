像 IG App 一樣更換 iOS App Icon

最近 IG App 因為生日推出特別的彩蛋，使用者可以自由更換 App 的 Icon。這樣的功能，當然不能只有 IG 獨享，接下來讓我們一步步實現 App Icon 更換的功能。

產生多組 App Icon 圖片

準備黃色，藍色和紅色的 peter， 利用網站 App Icon Generator產生三組 App Icon 圖片，到時候使用者可選擇喜歡的 peter 當 icon。

從 App Icon Generator 產生 iOS App Icon
透過網站 App Icon Generator 的幫忙，我們只要上傳圖片，即可產生 iOS App 需要的各種 App icon 。接下來就讓我們一步步示範如何產生 App Icon。medium.com

值得注意的，使用者只能選擇 App 本身提供的圖片當 icon，不能將自己喜愛的新垣結衣變成 app icon。

將 icon 圖片加到 Xcode 專案

黃色的 peter 是預設的 App icon，將它加到 Assets。

非預設的 App Icon，可加入尺寸 120 * 120 & 180 * 180 就好。請將它們加到 project navigator，並將 120 * 120 的檔名結尾設為 @2x， 180 * 180 的檔名結尾設為 @3x。(ps: 非預設的 App Icon 一定要加到 project navigator，不能加到 Assets）

在 CFBundleAlternateIcons 下輸入以下內容。

CFBundleAlternateIcons 的 key 的名稱將是程式裡指定 icon 時採用的名稱。我們的備用 icon 有藍色和紅色 peter，因此我們設定 2 個 key，分別取名 red peter & blue peter。(名字可自訂)

red peter & blue peter 的 Type 也是 Dictionary，它們的 key 為 CFBundleIconFiles，Type 為 Array，array 的成員為 project navigator 下 icon 圖檔的名字(不包含 @2x.png & @3x.png)。

將三種顏色的 peter 圖片加到 assets

為了在 App 畫面呈現 peter 圖片給使用者選擇，我們將三種顏色的 peter 圖片加到 assets。

選擇 App Icon 的 App 頁面

struct ContentView: View {
    var body: some View {
        HStack {
            Button(action: {
                UIApplication.shared.setAlternateIconName(nil)
            }) {
                Image("peter1")
                    .resizable()
                    .scaledToFit()
            }
            Button(action: {
                UIApplication.shared.setAlternateIconName("blue peter")
            }) {
                Image("peter2")
                    .resizable()
                    .scaledToFit()
            }
            Button(action: {
                UIApplication.shared.setAlternateIconName("red peter")
            }) {
                Image("peter3")
                    .resizable()
                    .scaledToFit()
            }
        }
        .frame(height: 200)
    }
}

呼叫 UIApplication 的 function setAlternateIconName 設定 app icon，傳入 nil 時將採用 asset 的預設 icon，傳入圖片名字時則會換成對應的 icon。
