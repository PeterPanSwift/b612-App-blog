手動新增 Xcode 的快速鍵(shortcut)

Xcode 提供很多方便好用的快速鍵，不過也有一些功能的快速鍵沒有設定，需要我們自己補上，比方下圖開關 Document Outline 的功能。

接下來就讓我們學女媧補天，自己補上開關 Document Outline 的快速鍵。

打開 Preferences 視窗

點選 Xcode menu 的 Xcode > Preferences。

切換到 Key Bindings 頁面

在搜尋框輸入 outline，找到目標 Document Outline

點擊 Document Outline 的 Key 欄位

輸入快速鍵

在此我們按下 option + d

現在我們比一般人厲害了，我們可以用 option + d 快速開關 Document Outline !
