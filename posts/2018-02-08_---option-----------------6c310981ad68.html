按住 option 鍵，顯示神奇紅線查詢和微調間距

在 Interface Builder 設計 App 畫面時，如果想要查詢或微調元件間的間距，按住 option 鍵將帶給你意想不到的驚喜 !

如下圖所示，當我們點選 image view 後，將滑鼠游標移到白色 view 上，按住 option 時，紅線將標示出 image view 跟白色 view 四個邊界的距離。這時候你甚至還可微調間距，按住上下左右鍵來調整。

我們可將滑鼠游標移動到任何想要比較查詢間距的對象，如下圖所示，當滑鼠游標停在 label 小可愛上，按住 option 鍵時，紅線將標示出 image view 和 label 的間距。
