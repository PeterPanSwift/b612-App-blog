讓 Button & NavigationLink 的圖片變色的 renderingMode

當我們使用 SwiftUI 的 Button & NavigationLink 顯示 SF Symbols 時，我們可以控制圖片的顏色，例如以下星星的顏色。

struct ContentView: View {
    var body: some View {
        Button(action: {
            
        }) {
            Image(systemName: "star")
                .resizable()
                .scaledToFit()
        }
        .accentColor(.red)
    }
}

然而當我們顯示的是自訂的圖片時，它就不肯乖乖變色了。

struct ContentView: View {
    var body: some View {
        Button(action: {
            
        }) {
            Image("favorites")
                .resizable()
                .scaledToFit()
        }
        .accentColor(.red)
    }
}

利用 renderingMode 讓圖片變色

想讓圖片變色很簡單，我們只要在 Image 呼叫renderingMode(.template)。

struct ContentView: View {
    var body: some View {
        Button(action: {
            
        }) {
            Image("favorites")
                .resizable()
                .renderingMode(.template)
                .scaledToFit()
        }
        .accentColor(.red)
    }
}

同樣的，我們也可以用 renderingMode 改變 NavigationLink 的圖片顏色。

struct ContentView: View {
    var body: some View {
        
        NavigationView {
            NavigationLink(
                destination: Text("favorites"),
                label: {
                    Image("favorites")
                        .resizable()
                        .renderingMode(.template)
                        .scaledToFit()
                })
                .accentColor(.red)
            
        }
    }
}
