利用 SPM 搜尋 & 列出 GitHub 上收藏的套件

開發 iOS App 時，我們可以透過 Xcode 內建的套件管理工具 SPM 加入套件，比方點選 Xcode menu 的 File > Add Package Dependency 後，在下圖的框框輸入網路抓圖套件 Kingfisher 的 GitHub 網址。

不過加入套件還要先搜尋套件的網址實在有點麻煩，有沒有簡單一點的方法呢?

有的，我們可以輸入套件的關鍵字搜尋曾在 Xcode 搜尋的套件，或是選擇自己在 GitHub 上收藏的套件

搜尋套件

框框裡提到可以 Search。

輸入 facebook。

輸入 Firebase。

選擇 GitHub 上收藏的套件(starred repositories)

SPM 加套件的視窗將自動列出我們在 GitHub 上點選 Star 按鈕收藏的套件。

比方彼得潘收藏了 Kingfisher & Alamofire。(ps: 點選網頁右上方的 Star button，已收藏時 button 的文字會變成 Unstar)

此時 Add Package Dependency 視窗顯示時，一開始就會列出我們收藏的套件呢。
