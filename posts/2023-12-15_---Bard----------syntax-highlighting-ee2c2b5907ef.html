使用 Bard 對程式進行縮排和 syntax highlighting

使用 AI 我們可以方便地對程式進行縮排和 syntax highlighting，不過有那麼多好用的 AI，要選擇哪一個呢 ?

最近彼得潘分別測試 GPT 和 Bard 後，覺得 Bard 在這回合因為以下兩點勝出。

  • Bard 的程式可以選擇 light theme 或 dark theme。

從右上的設定選擇 Use dark theme 切換到 dark theme，選擇 Use light theme 切換到 light theme。

light theme 的程式。

dark theme 的程式。

GPT 的網頁也可以切換 light 或 dark theme，但是程式碼只能搭配 dark theme。

  • Bard syntax highlighting 支援的程式語言較多。

比方 dart 程式在 Bard 可以 syntax highlighting，在 GPT 卻不行。
