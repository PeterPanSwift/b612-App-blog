利用 Grok 和 Chrome 套件一鍵取得 App 需要的多張圖片

彼得潘平常寫程式製作 Swift iOS 和 Flutter App 範例時，常常會需要大量的可愛圖片，因此常常要苦惱如何快速找圖。

相較於 Google 搜圖，彼得潘最近發現 Grok 也是不錯的選擇，它的 Imagine 有著不錯的生圖能力，而且它不只會極速生圖，還能無限生圖，往下滑動時它將繼續生成更多圖片。

圖片的來源有了，但是一張一張下載也頗花時間，因此我們將利用 Chrome 的套件一鍵下載。

從以下連結安裝 Chrome extension Image downloader — Imageye。

Image downloader - Imageye - Chrome Web Store
Find and download all images on a web page with Image downloader.chromewebstore.google.com

接下來我們示範利用 Grok 生圖，然後利用 SwiftUI 製作可愛的桌球男孩照片牆。

連到 Grok，從 Imagine 生圖，輸入以下 prompt。

左手用 MacBook 寫程式，右手拿桌球拍打桌球的可愛男孩，日本動畫風格

Cool ! Grok 生成一系列帥氣的桌球 coding 男孩。

往下滑動生成更多圖片。

點選右上的 Image downloader — Imageye 按鈕。

Grok 生成的圖會變成 JPG，因此 Type 請選擇 JPG，然後點選 Select all。

點選 Download 下載圖片。

點選 Download。

成功取得滿滿的桌球 coding 男孩。

將圖片全選複製貼到 Xcode 專案的 assets。

撰寫 SwiftUI 的照片牆程式。

import SwiftUI 

struct LazyGridGalleryView: View {
    let columns = Array(repeating: GridItem(), count: 3)
  
    var body: some View {
        ScrollView {
            LazyVGrid(columns: columns) {
                ForEach(2...22, id: \.self) { number in
                    Image("imgi_\(number)_default")
                        .resizable()
                        .scaledToFill()
                        .frame(height: 150)
                        .clipped()
                }
            }
            .padding()
        }
    }
}
