table view cell 點選時維持 cell 裡元件的背景顏色

當我們點選表格時，cell 的背景顏色會變色，貼心地提醒使用者 cell 被點選了。但是，它卻有點做過頭了，它還同時將 cell 裡元件的背景顏色變透明。

table view cell 點選時維持 cell 裡元件的背景顏色

當我們點選表格時，cell 的背景顏色會變色，貼心地提醒使用者 cell 被點選了。但是，它卻有點做過頭了，它還同時將 cell 裡元件的背景顏色變透明。

如下圖所示，cell 點選時，cell 裡的 image view & label 因為背景顏色變透明，變成顯示 cell 點選時的背景色灰色。

解法

將 cell 的類別設成繼承 UITableViewCell 的類別，連結元件的 outlet，然後定義 function setSelected(_:animated:)。

此 function 將在 cell 被點選或取消點選時執行，因此我們只要在裡面設定元件的背景顏色，即可讓它元件永遠維持同一個背景顏色，不管 cell 是否點選。

class TableViewCell: UITableViewCell {
   @IBOutlet weak var photoImageView: UIImageView!
   @IBOutlet weak var nameLabel: UILabel!
   override func setSelected(_ selected: Bool, animated: Bool) {
      super.setSelected(selected, animated: animated)
      nameLabel.backgroundColor = UIColor.cyan
      photoImageView.backgroundColor = UIColor.cyan
   }
}
