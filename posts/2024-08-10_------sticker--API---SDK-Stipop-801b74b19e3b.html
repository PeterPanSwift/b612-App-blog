有趣的貼圖(sticker) API & SDK Stipop

最近有同學問到如何開發 iOS App 的貼圖功能，彼得潘查了一下後，發現有個現成的貼圖 API Stipop ，上面有許多可愛的貼圖，而且還提供 iOS、Android & Flutter SDK，只要花幾分鐘設定即可在 App 裡加入貼圖功能。

  • 啟動 Stipop 的 iOS App 範例。
  • 將 Stipop iOS SDK 加到自己的專案。
  • 串接 Stipop API。
  • 研究 Stipop iOS SDK。

取得 Stipop 的 API key 和 Stipop.plist

為了使用 Stipop 的貼圖，我們必須先在 Stipop 網站建立 App，取得 App 的 API key 和 iOS App 需要的 Stipop.plist。

連到 Stipop 網站。

Stipop | Stickers API and SDK | Sticker platform for creators
150,000 stickers for messages, videos, and profile icons ready to integrate into any app, created by talented artists…stipop.io

點選上圖的 Get Started，連到帳號頁面建立帳號。

建立 application。MAU 是 Monthly Active Users，代表 App 的每月活躍用戶數。

我們必須設定信用卡帳號後才能取得 App 的 API key 和 Stipop.plist，請點選 Add a new card 設定。

別擔心，設定信用卡不會扣到錢，Stipop 提供一些免費的使用額度。

點選 Add a new card。

輸入信用卡資訊。

填好信用卡資訊後，切換到 Build 頁面可複製 API Key 和下載 iOS App 需要的 Stipop.plist。

啟動 Stipop 的 iOS App 範例

下載 Stipop UI SDK for iOS。

GitHub - stipop-development/stipop-ios-sdk: 150,000+ stickers API & SDK for iOS Apps.
150,000+ stickers API & SDK for iOS Apps. Contribute to stipop-development/stipop-ios-sdk development by creating an…github.com

打開專案，加入 Stipop.plist。

啟動 App，從 App 展示的聊天頁面可輸入可愛的貼圖。

點選貼圖鍵盤上方的 search icon 可顯示貼圖的搜尋頁面。

串接 Stipop API

我們也可以串接 Stipop API，打造自己的貼圖介面。

Stipop Docs | Documentation for SDK and API
Welcome to Stipop API Documentation for chat stickers! Here’s a quick overview of what is provided through chat sticker…docs.stipop.io

Stipop 提供豐富的 API endpoints。

比方以下為貼圖搜尋 API 的串接範例。

func searchSticker(term: String, userID: String) async throws {
    let url = URL(string: "https://messenger.stipop.io/v1/search?q=\(term)&userId=\(userID)")!
    var request = URLRequest(url: url)
    request.setValue("123456", forHTTPHeaderField: "apikey")
    let (data, _) = try await URLSession.shared.data(for: request)
    if let content = String(data: data, encoding: .utf8) {
        print(content)
    }
}
