讓 enum 變成 array 的 CaseIterable & allCases — Swift 4.2 新功能

在 Swift 4.2，SE-0194 的 Derived Collection of Enum Cases 讓 enum 變得更方便了，現在我們可以把 enum 變成 array 來使用呢。

  • 想要讓 enum 型別變成 array，需要讓它遵從 protocol CaseIterable。
enum BestPhone: CaseIterable {
   case iPhone16ProMax, iPhone16Pro, iPhone16Plus, iPhone16
}
  • 利用 static property allCases 取得 array。

array 裡將包含每個 case 的資料，順序將依當初 enum 裡定義的 case 順序。

  • 因為是 array，因此我們可以使用熟悉的 array 功能。

比方以下例子，我們利用 count 取得 enum 的成員數量，利用 for in 讀取 enum 裡的每個成員。

let phones = BestPhone.allCases
print(phones.count)
for phone in phones {
   print(phone)
}
