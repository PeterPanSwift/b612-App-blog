用 IBSegueAction 傳資料給 navigation controller 連接的 controller

開發 iOS App 時，我們有時會在顯示下一頁時以 present modally 的方式呈現，就像下圖 Calendar App 點選 Edit 時由下而上顯示編輯頁面。

此時編輯頁面被包在 navigation controller 裡，我們必須將 event 從 Event Detail 頁面傳到 Edit Event 頁面。

若以 segue 切換頁面，將 event 傳到下一頁有兩種做法。

  • 定義 IBSegueAction function。
  • 定義 function prepare。

從 Event Detail 傳資料到 Edit Event 時，中間會經過 navigation controller，此時使用 IBSegueAction 傳資料會比 prepare 簡單，以下我們做個簡單的範例說明。

storyboard 的畫面設計

點選 EventDetailViewController 右上的 Edit 將顯示 navigation controller 連結的 EditEventViewController。

EditEventViewController

property event 將儲存 EventDetailViewController 傳來的資料。

class EditEventViewController: UIViewController {
    @IBOutlet weak var eventNameLabel: UILabel!
    var event: String?
    
    override func viewDidLoad() {
        super.viewDidLoad()

        eventNameLabel.text = event
    }

連結 IBSegueAction function 傳資料

點選 Edit 後，頁面將從 EventDetailViewController 切換到 EditEventViewController，透過下圖紅色框框的 segue 連到 EditEventViewController，因此我們從它拉 IBSegueAction 到 EventDetailViewController。

IBSegueAction function 的程式如下。雖然 EventDetailViewController & EditEventViewController 的中間有個電燈泡 navigation controller，不過它只是幫忙管理頁面的切換，因此從 EventDetailViewController 切換到 EditEventViewController 時，還是會觸發 EventDetailViewController 定義的 IBSegueAction function。以下程式我們將 Study SwiftUI 存入 EditEventViewController 的 event。

 @IBSegueAction func editEvent(_ coder: NSCoder) -> EditEventViewController? {
     let controller = EditEventViewController(coder: coder)
     controller?.event = "Study SwiftUI"
     return controller
}

點選 Edit 後，Study SwiftUI 成功顯示在 EditEventViewController。

剛剛的例子也可以使用 prepare 傳資料，不過這樣會麻煩許多，經過多道手續才能傳資料到下一頁。

  • 將 segue.destination 轉型成 UINavigationController。
  • 讀取 UINavigationController 的 topViewController ＆ 轉型為 EditEventViewController。
  • 將資料存入 EditEventViewController 的 event。
override func prepare(for segue: UIStoryboardSegue, sender: Any?) {
   guard let navigationController = segue.destination as? UINavigationController,
         let controller = navigationController.topViewController as? EditEventViewController else {
        return
   }
   controller.event = "Study SwiftUI"
}
