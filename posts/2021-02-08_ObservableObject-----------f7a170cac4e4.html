ObservableObject 物件的畫面更新問題

開發 SwiftUI App 時，我們可以在遵從 ObservableObject 的類別裡宣告加上 @Published 的 property。如此當 property 改變時，它將自動更新畫面。例如以下 Baby 的 age 改變時，它將觸發畫面更新。

class Baby: ObservableObject {
    @Published var age = 1
}

按下 button 時 cuteBaby 的 age + 1，因此將觸發 ContentView 更新畫面。

struct ContentView: View {
    
    @StateObject var cuteBaby = Baby()
    
    var body: some View {
        VStack {
            Image("baby")
                .resizable()
                .scaledToFit()
            Text("\(cuteBaby.age) 歲")
                .font(.largeTitle)
                .padding()
                .background(
                    Capsule()
                        .foregroundColor(.yellow)
                        
                )
            Button(action: {
                cuteBaby.age += 1
            }, label: {
                Text("一暝大一寸")
            })
            
        }
    }
}

然而若我們使用 SwiftUI 常見的 Extract Subview 功能，將剛剛程式的 Text 獨立成新的 View AgeText，此時就算按 99 下按鈕，cuteBaby 已經 100 歲，畫面還是不會更新，依然顯示 1 歲。

struct ContentView: View {
    
    @StateObject var cuteBaby = Baby()
    
    var body: some View {
        VStack {
            Image("baby")
                .resizable()
                .scaledToFit()
            AgeText(cuteBaby: cuteBaby)
            Button(action: {
                cuteBaby.age += 1
            }, label: {
                Text("一暝大一寸")
            })
            
        }
    }
}
struct AgeText: View {
    
    var cuteBaby: Baby
    
    var body: some View {
        Text("\(cuteBaby.age) 歲")
            .font(.largeTitle)
            .padding()
            .background(
                Capsule()
                    .foregroundColor(.yellow)
                
            )
    }
}

問題出在 AgeText 不知道它的畫面要更新。我們要在它的 property cuteBaby 宣告時加上 @ObservedObject，如此 cuteBaby 的 age 改變時，畫面才會更新。

struct AgeText: View {
    
    @ObservedObject var cuteBaby: Baby
    
    var body: some View {
        Text("\(cuteBaby.age) 歲")
            .font(.largeTitle)
            .padding()
            .background(
                Capsule()
                    .foregroundColor(.yellow)
                
            )
    }
}

另外如果當初 Baby 的型別是以 struct 定義，則不會有此問題，AgeText 只要宣告一般的 property cuteBaby 即可在 age 改變時自動更新。

struct Baby {
    var age = 1
}
struct ContentView: View {
    
    @State var cuteBaby = Baby()
    
    var body: some View {
        VStack {
            Image("baby")
                .resizable()
                .scaledToFit()
            AgeText(cuteBaby: cuteBaby)
            Button(action: {
                cuteBaby.age += 1
            }, label: {
                Text("一暝大一寸")
            })
            
        }
    }
}
struct AgeText: View {
    
    var cuteBaby: Baby
    
    var body: some View {
        Text("\(cuteBaby.age) 歲")
            .font(.largeTitle)
            .padding()
            .background(
                Capsule()
                    .foregroundColor(.yellow)
                
            )
    }
}
