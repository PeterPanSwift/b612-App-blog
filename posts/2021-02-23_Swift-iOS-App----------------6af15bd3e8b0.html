Swift iOS App 新手工程師的工作面試技術需求

最近彼得潘研究了台灣 iOS 工程師的工作職缺，參考 104，CakeResume，Yourator，meet.jobs 等網站列出的技術需求，整理出以下找 iOS 工作的基本條件 & 加分條件，歡迎想成為 Swift iOS App 工程師的朋友參考。

基本條件

  • Swift 程式語言
  • Xcode
  • 熟悉物件導向(object-oriented，OOP)
  • Auto Layout
  • MVC 架構
  • 熟悉常用的 iOS SDK，UIKit ＆Cocoa Touch Framework
  • 熟悉 view controller 的生命週期
  • 串接後台的 RESTful API & JSON 解析
  • 英文技術文件閱讀能力
  • 熟悉 multithread programming，比方 GCD ＆ NSOperation
  • 從 storyboard，xib(nib) & 程式製作 App 畫面(Programmatically & Interface Builder)
  • Core Animation
  • 熟悉資料儲存技術(offline data storage)
  • 第三方 API 串接，比方 Facebook 登入，Google 登入
  • Git 版本管理，熟悉 Sourcetree ＆ 程式碼托管平台，比方 GitHub，GitLab & Bitbucket。
  • 撰寫技術文件
  • 放在 GitHub 的程式作品
  • iOS App 上架經驗，有 App Store 的上架作品
  • 使用和管理第三方套件(third-party libraries)，比方 SPM，CocoaPods & Carthage
  • 團隊合作
  • 良好的溝通能力
  • 乾淨的 coding style
  • MVVM 架構。(註: 熟練 MVC 架構的同學可以再研究 MVVM，滿多公司採用 MVVM 架構。不過只會 MVC 也找得到新手的工作，滿多朋友找到的公司都可以進去再學。)

加分條件

  • 熟悉記憶體管理(ARC)，處理 memory leak 問題
  • 熟悉 iOS Human Interface Guidelines
  • MVP，Clean，VIPER 等 design pattern(設計模式)
  • 熟悉 Protocol Oriented Programming(POP)
  • CoreData
  • 熟悉 async / await
  • 多國語言 App(localization)
  • 推播經驗(push notification，APNs)
  • 效能優化(performance optimization) & Instrument
  • 熟悉各種除錯技術(debug)
  • Objective-C 程式語言
  • 熟悉自動化測試，單元測試(Unit testing)，TDD，UI Test。
  • SwiftUI
  • 熟悉 functional programming，比方 Combine，RxSwift 或 RxCocoa
  • CI/CD，比方 Fastlane，Jenkins
  • 影音串流(video streaming)
  • 藍牙串接(Bluetooth，BLE)
  • WidgetKit
  • 資工，資訊，計算機相關科系
  • Flutter
  • tvOS App 開發經驗
  • watchOS App 開發經驗
  • 資料庫相關技術，比方 SQLite，FMDB
  • XML 解析
  • React Native 經驗
  • AR / VR 開發經驗
  • GraphQL
  • Chat 功能，比方使用 WebSocket 技術
  • 註冊登入功能
  • 金流功能開發經驗，比方 IAP，Apple Pay
  • 熟悉資料結構 & 演算法(data structure & algorithm)
  • App 安全性相關經驗(security) ，熟悉編碼加解密技術
  • 熟悉用戶分析平台，比方 Google Analytics，Mixpanel
  • 英文溝通能力
  • UI / UX 設計經驗
  • 個人技術 Blog
  • 熟悉敏捷式開發(Agile)，有 Scrum 經驗
