方便 App 測試假(mock)的 JSON 資料的平台

開發 iOS App 時，我們時常需要連線到後台抓取 JSON 資料。然而有時我們會想有個回傳假的 JSON 資料的後台，比方以下情況:

  1. 不會寫後台程式，但想從後台取得自訂的 JSON 資料，比方本週電影資訊。
  2. 後台工程師寫太慢了，程式還沒寫好，我們想先有個測試的後台方便 App 開發。
  3. 網路不通，想先在本機端測試。

以上的問題，都可以透過網路上一些提供 mocking services 的平台解決。
