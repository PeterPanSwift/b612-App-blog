如何產生搭配 iPhone 不鏽鋼外框的螢幕截圖

iPhone 是世界上最美的手機，不過如果直接擷取 iPhone 的模擬器視窗，將令人有小小的遺憾，它的邊邊是黑色的。

沒關係，待會透過網站 MockUPhone 的幫忙，我們將可產生擁有高級不鏽鋼邊框的 iPhone。

1 將 App 啟動到 iPhone 模擬器。

2 點選 Simulator menu 的 File > New Screen Shot，在桌面產生螢幕截圖的檔案。

3 連到網站 MockUPhone 製作搭配 iPhone X 外框的螢幕截圖。

MockUPhone
MockUPhone is a free tool that helps you wrap app screenshots in different mobile devices. Support iPhone mockup, iPad…mockuphone.com

從 Select Device 選擇模擬器對應的 iPhone 機型，從 Add Screenshot 上傳剛剛在桌面生成的螢幕截圖。

點選 Save mockup 下載圖片。
