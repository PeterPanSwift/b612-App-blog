利用 iOS 15 & macOS 12 的 Live Text copy 影片的程式碼

學習程式時，我們時常觀看大大們的教學影片學習。有些大大會提供下載的程式範例，有些大大則希望訓練我們打字，所以我們練習時要自己辛苦地一字字輸入。

即將推出的 iOS 15 & macOS 12 內建方便的 Live Text 功能，彼得潘玩了一下，發現它很適合用來 copy 影片裡的程式。接下來讓我們試試 copy CS193p 課程精彩的程式吧。

從 iPhone 拍照影片的程式碼

彼得潘還沒有升級 macOS 12，所以將用 iPhone 拍照實驗 Live Text，不然更方便的方法應該是從 macOS 擷取圖片再搭配 Live Text。

拍攝的照片如下。

點選圖片右下角的文字辨識按鈕

點選 Select All 選取全部的文字

點選 copy 複製文字

將程式貼到 Xcode

調整排版和移除多餘的文字後(比方滑鼠游標被辨識成大寫的 I )，影片裡的程式碼完美地重現了。

import SwiftUI
struct Palette: Identifiable {
    var name: String
    var emojis: String
    var id: Int
}
class PaletteStore: ObservableObject {
    let name: String
    @Published var palettes = [Palette]()
    
    init(named name: String) {
        self.name = name
    }
}
