找出 storyboard 裡 container view 連結的 child view controller

在 storyboard 設計 App 畫面時，利用 container view 我們可以方便地加入 child view controller。不過當我們有多個 container view 時，有時會容易混淆，忘了 container view 連結到哪個 child view controller。

接下來讓我們以貓狗列表為例，說明如何快速找出 container view 連結的 child view controller。

storyboard 的畫面設計如下，ViewController 裡有兩個 container view，分別連到 CatTableViewController & DogTableViewController。

我們想找出 Dog Container View 連結的 child view controller，以下介紹兩種方法:

方法一

  • 滑鼠游標移到 Dog Container View 上，按出右鍵選單。
  • 滑鼠游標移到右鍵選單的 viewDidLoad 上，即可看到它連結的 DogTableViewController 被框選出來。

方法二

  • 點選 Dog Container View。
  • 切換到 Connections inspector 頁面，滑鼠游標移到 viewDidLoad 上，即可看到它連結的 DogTableViewController 被框選出來。
