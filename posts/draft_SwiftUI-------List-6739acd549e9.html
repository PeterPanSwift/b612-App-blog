顯示文字的表格

利用 List 呈現表格，在 init 的參數 content 傳入 closure，在 closure 裡列出每一個 row 顯示的文字。

init(@ViewBuilder content: () -> Content)

struct ContentView: View {
   var body: some View {
      List {
         Text("可惜沒如果")
         Text("一千年以後")
         Text("殺手")
         Text("不懂")
         Text("小酒窩")
         Text("記得")
         Text("她說")
         Text("愛笑的眼睛")
         Text("一個又一個")
         Text("你要的不是我")
      }
   }
}
