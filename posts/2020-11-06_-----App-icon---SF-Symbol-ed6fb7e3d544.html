Apple 提供超過 6000 個圖案的 SF Symbol 幫助我們開發 App。(SF 是 San Francisco 的縮寫，是舊金山的意思）

SF Symbol 包含五花八門的圖案， App 時常使用的圖案它都幫我們設計好了，比方垃圾桶，鈴鐺，郵件，月曆，頭像等。有了這些 Apple 精心設計的 icon，我們再也沒有做不出精美 App 的藉口，接下來就讓我們一步步認識和使用 SF Symbol 吧。

下載 SF Symbols App

iOS 內建 SF Symbols，因此不一定要安裝 SF Symbols App。不過為了方便查詢有哪些可用的 SF Symbol，建議先從以下網站下載 & 安裝 Mac 版的 SF Symbols 7 App。

設定 SF Symbol 的樣式

以下分別說明 SwiftUI & interface builder 兩種範例。

  • SwiftUI 範例

建立 Image 時，搭配參數 systemName，利用 foregroundStyle 控制顏色。

struct ContentView: View {
    var body: some View {
        Image(systemName: "paperplane")
            .resizable()
            .scaledToFit()
            .frame(width: 100, height: 100)
            .foregroundStyle(.purple)
    }
}

調整粗細

以下分別說明 SwiftUI & interface builder 兩種範例。

  • SwiftUI 範例

從 font 調整。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "paperplane")
                .font(.system(size: 100, weight: .ultraLight))
            
            Image(systemName: "paperplane")
                .font(.system(size: 100, weight: .black))
        }
        .foregroundStyle(.purple)
   }
}

從 fontWeight 調整。

struct ContentView: View {
  
    var body: some View {
        VStack {
            Image(systemName: "paperplane")
                .resizable()
                .scaledToFit()
                .frame(width: 200)
                .fontWeight(.bold)
            Image(systemName: "paperplane")
                .resizable()
                .scaledToFit()
                .frame(width: 200)
            Image(systemName: "paperplane")
                .resizable()
                .scaledToFit()
                .frame(width: 200)
                .fontWeight(.ultraLight)
        }
        .foregroundStyle(.purple)
    }
}
