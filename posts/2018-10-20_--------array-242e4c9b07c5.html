將 Swift 的數字範圍變成 array

在 Swift 裡我們可以用 ... 或 ..< 的語法表示某段數字範圍，比方 1…10 表示 1 ~ 10。那有沒有方法生成成員依序是數字 1 ~ 10 的 array 呢 ?

透過 for in 就可以做到這件事。

var numbers = [Int]()
for i in 1...10 {
   numbers.append(i)
}

但是這樣太麻煩了，其實只要一行就能搞定，Array 原本就有個 init 可以讓我們傳入數字的 range，生成包含 range 裡每個數字的 array。

let numbers = Array(1...10)
init(_:) - Array | Apple Developer Documentation
Creates an array containing the elements of a sequence.developer.apple.com

此 init 接受的參數型別其實是 Sequence，所以只要是 Sequence 都可以，比方以下程式可將 dictionary 的 values 組合起來，得到分數的 array。

let dic = ["peter": 98, "hook": 50]
let grades = Array(dic.values)

ps: 利用 function map 也可以將數字範圍變成 array，例如以下例子

let range = 1...10
let numbers = range.map { $0 }
