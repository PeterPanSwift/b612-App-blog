將 GitHub 網頁的 repository 變成方便瀏覽程式的 VS Code 介面

最近發現從 GitHub 網頁瀏覽 repository 程式時，一鍵就能將它切換成方便瀏覽程式的 VS Code 介面。

以下我們以 Apple 放在 GitHub 的範例 sample-backyard-birds 說明。

GitHub - apple/sample-backyard-birds
Contribute to apple/sample-backyard-birds development by creating an account on GitHub.github.com
  • 方法 1: 將 repository 網址裡的 github.com 改成 github.dev。(快速鍵 .)
  • 方法 2: 用 GitHub Codespaces 打開 repository。

方法 1: 將 repository 網址裡的 github.com 改成 github.dev。(快速鍵 .)

在 repository 的網頁按下快速鍵 .，或是手動將網址從 github.com 改成 github.dev。以剛剛 Apple 的 repository https://github.com/apple/sample-backyard-birds 為例，改成 https://github.dev/apple/sample-backyard-birds 即可變成 VS Code 介面。

方法 2: 用 GitHub Codespaces 打開 repository

點選 repository 網頁右上方的 Code，切換到 Codespaces 分頁，點選 Create codespace on main。

從 GitHub Codespaces 開啟 repository。
