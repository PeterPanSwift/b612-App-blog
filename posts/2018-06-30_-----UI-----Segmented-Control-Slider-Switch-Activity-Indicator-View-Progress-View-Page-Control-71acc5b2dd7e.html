研究以下 UI 元件: Segmented Control，Slider，Switch，Activity Indicator View，Progress View，Page Control，Date Picker，Visual Effect View with Blur，Color Well

Segmented Control

切換分類的元件，比方選擇性別。

研究欄位

1 Segments
2 Segment
3 Title
4 Image
5 Selected

Slider

滑動設定數值的元件，比方控制音樂的音量，螢幕的亮度等。

研究欄位

1 Value
2 Minimum
3 Maximum
4 Min Image
5 Max Image
6 Min Track
7 Max Track
8 Thumb Tint

Switch

開關，比方設定是否打開 9:30 起床的鬧鐘。

研究欄位

1 State
2 On Tint
3 Thumb Tint

Activity Indicator View

代表正在做某件事的進度條，比方資料下載中。

研究欄位

1 Style
2 Color
3 Animating

Progress View

顯示進度的進度條，比方資料下載的進度。

研究欄位

1 Style
2 Progress
3 Progress Tint
4 Track Tint
5 Progress Image
6 Track Image

Visual Effect View with Blur

毛玻璃效果(背景模糊)。

研究欄位

Blur Style

Page Control

小圓點，顯示有幾個分頁和目前在第幾個分頁。

研究欄位

1 # of Pages
2 Current
3 Hides for Single Page
4 Tint Color
5 Current Page

Date Picker

時間選擇的元件。

研究欄位

1 Preferred Style
2 Mode
3 Locale
4 Interval
5 Date
6 Minimum Date
7 Maximum Date

Color Well

選擇顏色。
