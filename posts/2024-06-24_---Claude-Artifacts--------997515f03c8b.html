使用 AI 的 Canvas 製作精美表格

從前透過 AI 我們可以製作 markdown 格式的表格，例如以下 SwiftUI & Flutter 的比較表。

markdown 格式的表格簡單易懂，可惜不夠漂亮。現在大部分的 AI 都具有 Canvas 功能，我們可以請 AI 在 Canvas 寫網頁程式並顯示預覽畫面。

以下彼得潘以 Claude 為例，一步步分享請 AI 製做表格的過程。

  • 使用 HTML & CSS。
  • 請 AI 寫自動化程式將 markdown table 變成漂亮的 HTML / CSS table。
  • 使用 SVG。
  • 使用 React。

ps: 使用 GPT 的繪圖的功能可繪製最精美的表格，不過可能會有部分文字寫錯。

  • 第四版的表格。

prompt: 請繪製成日本動漫風的可愛表格。

當然可以！我會為您創建一個日本動漫風格的可愛表格。這個設計將融入一些典型的動漫元素，如柔和的色彩、可愛的圖標和有趣的字體。我們將使用React和Tailwind CSS，並添加一些自定義的CSS來實現這種風格。
