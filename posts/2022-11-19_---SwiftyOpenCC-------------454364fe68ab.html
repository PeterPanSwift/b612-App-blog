使用 SwiftyOpenCC 轉換繁體中文和簡體中文

以蔡淳佳好聽的談心歌詞為例，示範使用 SwiftyOpenCC 轉換繁中和簡中。 以下我們分別介紹 SwiftUI & UIKit 的範例。

SwiftUI

  • 將簡中變成繁中
import SwiftUI
import OpenCC

struct ContentView: View {
    let converter = try! ChineseConverter(options: [.traditionalize])
    let lyrics = """
        告诉我 什么事情让你开心 谁让你烦心 让我来抚平
        有些话 放在心里心有灵犀 不需要原因 我就能感应
        能和知心朋友一起谈心
        不在乎主题
        感觉永远历久弥新
        我明白全世界只有你
        最珍惜我的快乐伤心
        """
    
    var body: some View {
        Text(converter.convert(lyrics))
            .font(.largeTitle)
    }
}

說明

let converter = try! ChineseConverter(options: [.traditionalize])

產生 ChineseConverter 做轉換，參數 traditionalize 表示要轉換成繁中。

converter.convert(lyrics)

將字串變成繁中。

UIKit

import UIKit
import OpenCC

class ViewController: UIViewController {
    @IBOutlet weak var label: UILabel!
    let converter = try! ChineseConverter(options: [.traditionalize])
    let lyrics = """
        告诉我 什么事情让你开心 谁让你烦心 让我来抚平
        有些话 放在心里心有灵犀 不需要原因 我就能感应
        能和知心朋友一起谈心
        不在乎主题
        感觉永远历久弥新
        我明白全世界只有你
        最珍惜我的快乐伤心
        """
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        label.text = converter.convert(lyrics)
    }
}
