利用 code 指令啟動 VS Code 開啟專案

在 Terminal 操作時，我們可以用 code 指令啟動 VS Code 開啟專案，方法如下。

安裝 VS Code 的命令列工具 code

設定 VS Code 的命令列工具後，才能在 Terminal 輸入 code 指令。若是尚未設定，請先叫出 Command Palette (cmd + shift + p)，輸入 shell，點選 Shell Command: Install 'code' command in PATH。

點選 OK。

輸入電腦的密碼。

成功安裝 VS Code 的命令列工具，指令名稱為 code。

利用 code 指令啟動 VS Code 開啟專案

比方我們先在 Terminal 用以下指令下載 Flutter SearchAnchor 的範例程式。

flutter create --sample=material.SearchAnchor.bar.1 mysample

輸入以下指令進入範例的資料夾 mysample。

cd mysample

輸入 code . ，啟動 VS Code 開啟專案。

code .
