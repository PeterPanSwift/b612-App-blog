#204 扣除假日的實際工作天數

最近彼得潘開發 iOS App 案子時遇到一個有趣的功能，覺得它滿適合出作業和當面試考題，歡迎有興趣的朋友試試。

功能需求

選擇開始日期和結束日期，扣除假日，計算實際的工作天數。

假日須考慮星期六，星期天，國家假日和連假，若遇到補假也要加回來，因為補假也要工作呀。

從政府行政機關辦公日曆表 API 可查詢放假資訊。

https://data.ntpc.gov.tw/datasets/308dcd75-6434-45bc-a95f-584da4fed251

從以上網址得知資料 id 為 308dcd75–6434–45bc-a95f-584da4fed251，因此我們可從以下網址抓取 JSON

https://data.ntpc.gov.tw/api/datasets/308dcd75-6434-45bc-a95f-584da4fed251/json?size=100&page=0

值得注意的，資料從 2013 開始由小排到大，目前測試 page=11 才能抓到 2023 的資料。若不想一頁一頁抓資料，也可以將 size 設為 2000，一次抓取全部資料。

https://data.ntpc.gov.tw/api/datasets/308dcd75-6434-45bc-a95f-584da4fed251/json?size=2000

ps: 聽說未來 API 會增加參數方便抓取某一年的資料，敬請期待。
