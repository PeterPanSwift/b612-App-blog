安裝 GitHub CLI 和登入 GitHub 帳號

參考 GitHub CLI 網頁的教學安裝 GitHub CLI。

GitHub CLI
Take GitHub to the command linecli.github.com

安裝好 GitHub CLI 後，從 Terminal 輸入指令登入 GitHub 帳號。只要登入一次，之後執行 gh 指令即可不用再輸入帳號密碼。

gh auth login
? Where do you use GitHub?  [Use arrows to move, type to filter]
> GitHub.com
  Other
? What is your preferred protocol for Git operations on this host?  [Use arrows to move, type to filter]
> HTTPS
  SSH
? Authenticate Git with your GitHub credentials? (Y/n) y

打開網頁登入 GitHub 帳號。

? How would you like to authenticate GitHub CLI?  [Use arrows to move, type to filter]
> Login with a web browser
  Paste an authentication token
! First copy your one-time code: 88CC-267C
Press Enter to open https://github.com/login/device in your browser...

登入成功，Terminal 顯示 Authentication complete。
