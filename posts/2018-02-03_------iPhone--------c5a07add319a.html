解決找不到 iPhone 模擬器的問題

當我們在 Mac 上同時安裝多個版本的 Xcode，一下子用版本 A 開發，一下子用版本 B 開發時，可能會遇到一個特別的問題。

如下圖所示，當我們將程式寫好，開心地準備將 App 安裝到模擬器時，卻發現模擬器消失了，而且是通通消失，一個都不留 !

會出現這個問題，是因為 Xcode 的 Command Line Tools 版本錯誤。你可以透過以下方法讓模擬器重出江湖 !

方法一:
  1. 點選 Xcode menu，Xcode > Preferences。
  2. 切換到 Locations 頁面。

3. 選擇正確的 Xcode 版本。比方你正在使用 Xcode 9.2 的話，請選擇 Xcode 9.2。

方法二:

從 terminal 輸入指令，切換 Xcode 的版本，在 --switch 後請輸入目前使用中的 Xcode 路徑，路徑要描述到 Xcode.app 下的 Contents/Developer 這一層。

sudo xcode-select --switch  /Applications/Xcode.app/Contents/Developer
