  • 基本版: 製作 iPhone 13 mini 尺寸的畫面，沒有使用 auto layout。

基本版: 製作 iPhone 13 mini 尺寸的畫面，沒有使用 auto layout

  • 加入 tab bar controller & navigation controller，Music 頁面以 table view controller 設計。

我們主要設計的是紅色框框代表的 Music 頁面，其它的 controller 只是為了讓 App 有 5 個 tab。

  • 設計圖的 navigation bar & tab bar 是半透明的毛玻璃效果，因此請將 tab bar & navigation bar 的 Appearances 勾選 Standard & Scroll Edge。
  • 在 navigation bar 的中間顯示 segmented control。

將 segmented control 拖曳到 table view controller 的 navigation bar 中間，設定標題為 Featured & Charts。

  • 在 navigation bar 的左邊跟右邊加入按鈕。

將 bar button item 拖曳到 table view controller 的 navigation bar 的左邊跟右邊。

將右邊 bar button item 的 Image 設為 SF Symbol list.bullet。

  • 將 table view 的 Content 設為 Static Cells。

本篇文章採用適合不會寫程式的初學者的做法，因此 Music 頁面將以 Static Cells 設計。

  • 設定 Table View 的 Selection 為 No Selection。

讓 cell 點選時不會變色。

  • 設定 table view 的 header 高度。

將 table view 的 Sections Header Estimate & Footer Estimate 設為 0，讓表格不要顯示 header & footer。

iOS 15 的 Static Cells 的 section header / footer 顯示問題
在 iOS 15 當我們使用 table view controller 搭配 Static Cells 的 table view 時，將遇到特別的 header / footer 顯示問題。medium.com
  • 用 4 個 cell 設計音樂 App 頁面。

設計圖的四個區塊將對應到 4 個 cell。

  • 將第一個 cell 的高度設為 152。
  • 輸出 Figma 設計圖裡的紫色圖片，將圖片加到 Xcode 專案的 asset。

輸出 Figma 圖片的方法可參考以下連結。

輸出 Figma 設計的 iOS App 圖片
輸出畫面裡的圖片medium.com

輸出圖片時要選到下圖的 Image/Main，才不會包含文字。文字我們將另外用 label 顯示。

  • 在第一個 cell 顯示圖片。

加入佔滿 cell 的 image view，設為剛剛加入的紫色圖片。

ps: 在 storyboard 會看到紫色圖片上方有一塊白色區塊。別擔心，當 App 在模擬器執行時，並不會有白色區塊。

App 實際執行的模樣。

  • 設定第二個 cell 的高度為 207。

點選 Figma 設計圖的第二個區塊，查詢它和紫色圖片的間距。在 Inspect 分頁時，移動滑鼠可查詢頭像和其它元件的間距。若是在 Design 或 Prototype 分頁，則須按住 option & 移動滑鼠才能查詢間距。從下圖看出我們可將第二個 cell 的高度設為 207 (14.5 + 192.83)。

  • 在第二個 cell 加入顯示 New Music 的 label。

從設計圖得知 New Music 跟左邊間距 15，跟上面間距 15。(設計圖是 14.5，我們可以採用整數就好)

點選設計圖的 New Music 後，從右邊的 Inspect 頁面查詢它的字型大小和顏色。下圖顯示字型為 SF Pro，大小為 17，顏色的十六進位是 000000。

依據以上設計圖的規格，我們在第二個 cell 加入顯示 New Music 的 label。

設定 label 顏色時，在 Hex Color 欄位輸入 000000。設定大小時，x & y 都輸入 15。

Font 設為 System 17，因為 System 就是 SF Pro。

  • 在第二個 cell 設計專輯清單。

依難度可分成基本版跟進階版的做法。

  • 基本版

專輯清單不會水平捲動。

  • 進階版

專輯清單可以水平捲動，利用以下連結介紹的 scroll view 實現。

設定 content size，實現水平捲動，上下捲動和分頁的 scroll view
