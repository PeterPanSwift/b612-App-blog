打開 Xcode 的 Coding intelligence — Xcode 26 新功能

從 Xcode 26 開始，我們終於可以出一張嘴下 prompt 請 AI 幫忙開發 App 了。

ps: 需搭配 macOS 26。

請 AI 幫忙前，我們首先要打開 Xcode 的 Coding intelligence，設定幫忙的 AI model。

從 Quick Action(cmd +shift + a)打開 Xcode 的 Settings — Intelligence 頁面。

Xcode 內建免費的 AI Model ChatGPT，請先點選 ChatGPT in Xcode 的 Turn On 啟用它。

點選 Next。

點擊 Turn On ChatGPT 可享用免費的 ChatGPT，然而它每天有使用額度的限制，不能問太多問題。若有訂閱 ChatGPT 帳號，點擊 Use ChatGPT with an Account 可享有更高的額度問更多問題。

點擊 Use ChatGPT with an Account 後再點擊 Sign In 登入 GPT 帳號。

登入成功後，ChatGPT in Xcode 的右邊將顯示登入的帳號。

Claude in Xcode 必須有訂閱 Claude 才能使用，若有訂閱可點選 Claude in Xcode，然後點擊 Sign In 登入 Claude 帳號。

點擊 Authorize。

免費的 Claude 帳號 Authorize 將無法成功，會出現錯誤 This application is not authorized for claude_free organizations。

好吧，沒關係，既然要錢彼得潘就犧牲一下，多吃幾餐泡麵訂一下 Claude 吧。用付費的 Claude 帳號登入後，Claude in Xcode 成功出現彼得潘的帳號。

除了 ChatGPT & Claude，我們也可以添加自己喜歡的 AI model。點選 Add a Model Provider 將可設定網路上或本機的 AI model。

以下我們加入 OpenAI 的 API key，URL 輸入 https://api.openai.com/v1/responses。

ps: OpenAI API Key 的取得方法可參考以下連結。

用一餐麥當勞的錢玩玩 OpenAI API
OpenAI 提供豐富的 API 讓我們開發各種結合 AI 的有趣應用，比方聊天機器人、生成圖片，寫程式等。然而天下沒有白吃的午餐，串接 OpenAI API 會消耗 token，而 token 必須花錢購買，因此要先儲值才能使用…medium.com

加入 OpenAI 後，點選 OpenAI 可看到它琳琅滿目的 AI model。

若是選擇 Locally Hosted，則可設定本機的 AI。

成功設定 AI model 後，我們將可點選下圖的星星打開 Coding Assistant，請 AI 一起寫程式開發 App。

點選下方的 Reasoning 可選擇 AI 的思考程度，High 是最聰明的 AI，不過也會花更多的時間思考。

點選左上的箭頭可選擇 AI model。

以下我們在 prompt 輸入加上漂亮的漸層背景，AI 直接修改專案的程式，幫我們加上漂亮的漸層背景。
