Xcode Swift 程式自動完成的彩色方塊格子

當我們左手寫詩，右手開心輸入 Swift 程式碼時，時常會看到自動完成的提示視窗顯示可愛的彩色方塊格子，到底格子裡的字母代表什麼呢 ?

接下來讓我們一個個解密這些可愛彩色格子的意義吧。

.

程式保留字，比方 var，let，func 等。

V

變數或常數。

{ }

事先打好的程式片段(code snippet)。

比方選擇上圖的 var — Var Declaration，自動輸入下圖的變數宣告程式。

f

呼叫 function。

I

生成東西的 init

三個堆疊的方塊

module / framework。

P

stored property(屬性)

M

method

computed property

L

local variable，比方 function / closure 的參數，或是 function / closure { } 裡宣告的變數常數。

Ｓ

struct

C

class

E

enum

K

enum 的 case。

Pr

protocol

T

typealias
