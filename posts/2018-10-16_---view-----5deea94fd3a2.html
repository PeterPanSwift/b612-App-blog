旋轉 view 的方法

利用 CGAffineTransform 的 init(rotationAngle:) 旋轉，傳入參數控制旋轉的角度。

init(rotationAngle:) - CGAffineTransform | Apple Developer Documentation
The angle, in radians, by which this matrix rotates the coordinate system axes. In iOS, a positive value specifies…developer.apple.com

範例

label.transform = CGAffineTransform(rotationAngle: .pi)

.pi 代表 180 度。( 也可以傳入 CGFloat.pi，在此我們省略 CGFloat)

因此順時針旋轉 45 度要除 4。

numberLabel.transform = CGAffineTransform(rotationAngle: .pi / 4)

如果傳入的角度為負數，表示逆時針旋轉，比方以下為逆時針旋轉 45 度的例子。

numberLabel.transform = CGAffineTransform(rotationAngle: -.pi / 4)
