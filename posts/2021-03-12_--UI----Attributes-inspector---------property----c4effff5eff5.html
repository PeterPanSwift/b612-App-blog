從 UI 元件 Attributes inspector 欄位查詢對應的 property 名稱

滑鼠游標移到欄位的輸入框上，然後稍等一兩秒，即可看到欄位的說明和 property 名稱。

  • 範例 1 : Alpha 欄位的 property 名稱是 alpha

利用 alpha 調整透明度。

alpha 的型別可在程式輸入時從自動完成選單得知，比方從下圖可看出 alpha 的型別是 CGFloat，要傳入浮點數。

  • 範例 2: Background 欄位的 property 名稱是 backgroundColor

利用 backgroundColor 調整顏色。

backgroundColor 的型別是 UIColor?。
