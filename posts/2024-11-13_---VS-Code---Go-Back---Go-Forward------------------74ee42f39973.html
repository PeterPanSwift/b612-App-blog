使用 VS Code 的 Go Back & Go Forward 跳到上一個 / 下一個瀏覽的地方

開發程式時，我們時常會跳到上一個 / 下一個瀏覽的地方，比方原本在 main.dart，跳到 app_bar.dart 裡查看 AppBar 的定義後，再跳回 main.dart。

VS Code 提供方便的 Go Back / Go Forward 指令讓我們跳到上一個 / 下一個瀏覽的地方。執行 Go Back / Go Forward 有以下幾種方法。

  • 快速鍵。
  • 從 menu 點選。
  • 從 Command Palette 點選。
  • 點選 Go Back / Go Forward 的箭頭按鈕。

快速鍵

ctrl -(Go Back) 或 ctrl shift -(Go Forward)。

從 menu 點選

從 Command Palette 點選

點選 Go Back / Go Forward 的箭頭按鈕

為了節省空間，Go Back / Go Forward 的箭頭按鈕預設並不會顯示。若想顯示 Go Back / Go Forward 的箭頭按鈕，我們必須先打開 VS Code 的 Settings。

打開 Window: Command Center 的 Show command launcher together with the window title。

如此視窗上方的 bar 即可出現 Go Back / Go Forward 的箭頭按鈕。
