設定 isIdleTimerDisabled 讓 iPhone 不再自動進入鎖定畫面

正常情況下，當使用者沒有觸碰 App 畫面一段時間後，螢幕會自動變暗，進入鎖定畫面。

不過有些特別的 App 會希望畫面一直顯示，螢幕一直亮著，即使使用者沒有觸碰畫面，比方新垣結衣的時計 App，也許你會希望螢幕一直亮著，隨時能看到結衣對你微笑。

很簡單，只要在程式裡輸入以下程式碼即可。

UIApplication.shared.isIdleTimerDisabled = true
isIdleTimerDisabled - UIApplication | Apple Developer Documentation
You should set this property only if necessary and should be sure to reset it to when the need no longer exists. Most…developer.apple.com

ps: 螢幕一直亮著將讓 iPhone 比較耗電，因此請謹慎使用此功能呀。(ps: 不過為了看到結衣，多耗一點電也是值得的啦。)
