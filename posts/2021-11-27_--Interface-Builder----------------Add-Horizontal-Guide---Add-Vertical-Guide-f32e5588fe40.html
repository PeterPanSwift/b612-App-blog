在 Interface Builder 新增水平或垂直的排版參考線: Add Horizontal Guide & Add Vertical Guide

在 Xcode 的 Interface Builder 排版元件時，我們可以利用水平或垂直的參考線幫助排版，接下來讓我們玩玩有趣的參考線吧。

點選畫布裡的 view

如下圖所示，點選畫布裡的白色長方形畫面。(ps: 記得要從畫布點選，若從左邊的元件清單(document outline)裡點選，待會將無法加入參考線)

加入參考線

點選 Editor > Guides > Add Horizontal Guide 加入水平參考線。

此時畫面多了一條水平的參考線。

拖曳線調整線的位置

線上方的數字代表線跟 view 上邊界的間距，線下方的數字代表線跟 view 下邊界的間距。

我們可加入多條參考線，比方下圖加了三條參考線。

在某個 view 加入參考線

當我們點選畫面裡的某個 view，然後點選 Add Horizontal / Vertical Guide 加入參考線時，參考線將加在 view 的 superview 上。

比方我們點選下圖的牛排圖片，然後點選 Add Horizontal Guide，此時水平參考線將加在它的 superview 黃色 view 身上。

下圖表示參考線跟黃色 view 的上邊界間距 40，參考線跟黃色 view 的下邊界間距 356。

利用參考線對齊元件

有了參考線，我們可以利用它對齊元件。

如下圖所示，當好吃的牛排圖片對齊參考線時，參考線會變成藍色，讓我們知道它已經完美對齊。

移除參考線

將參考線拖曳移出它所在的 view，即可讓它消失在世界上，就像把討厭的人推入懸崖一樣。

比方若想移除下圖的參考線，只要將它移出黃色 view。
