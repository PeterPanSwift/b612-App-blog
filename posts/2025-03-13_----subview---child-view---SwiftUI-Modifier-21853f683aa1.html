傳遞到 subview / child view 的 SwiftUI Modifier

開發 SwiftUI App 時，我們常利用 modifier 調整元件的樣式功能，比方以下例子調整圖文的字體大小，顏色和斜體。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "swift")
                .font(.title)
                .foregroundStyle(.orange)
            Text("Swift")
                .font(.title)
                .foregroundStyle(.orange)
                .italic()
            Text("The powerful programming language that’s also easy to learn.")
                .font(.title)
                .foregroundStyle(.orange)
                .italic()
        }
    }
}

剛剛的程式因為多個元件有一樣的樣式，所以有許多重覆的 modifier，像是 .foregroundStyle(.orange) 就寫了三次。

此時我們可利用 modifier 傳遞到 subview / child view 的技巧簡化程式，改從 superview / parent view 呼叫 modifier，如此 modifier 只要呼叫一次，不用在每個 subview 都呼叫一樣的 modifier。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "swift")
            Text("Swift")
            Text("The powerful programming language that’s also easy to learn.")
        }
        .font(.title)
        .foregroundStyle(.orange)
        .italic()
    }
}

當我們將 font、foregroundStyle、italic 加在 VStack 時，它們將傳遞給 VStack 的 subview，因此會作用在 Image & Text 上。

當我們想將相同樣式套用在多個 view 時，也可用不影響排版的 Group 元件包住這些 view，然後將 modifier 加在 Group 上。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "swift")
                .font(.largeTitle)
                .foregroundStyle(.orange)
            Group {
                Text("Swift")
                Text("The powerful programming language that’s also easy to learn.")
            }
            .font(.title)
            .foregroundStyle(.blue)
            .italic()
        }
    }
}
SwiftUI 將多個元件組合的 Group
SwiftUI 提供了擅長將多個元件組合的 Group，接下來讓我們認識它主要應用的 2 個地方:medium.com

並非所有 modifier 都會傳遞到 subview。比方以下加在 VStack 的 padding、background、clipShape 只會作用於 VStack，不會作用在 Image & Text。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "swift")
            Text("Swift")
            Text("The powerful programming language that's also easy to learn.")
        }
        .font(.title)
        .foregroundStyle(.orange)
        .italic()
        .padding()
        .background(.gray)
        .clipShape(.rect(cornerRadius: 10))
    }
}
