移除字串前後空白的 trimmingCharacters

開發 iOS App 時，我們一般利用 text field 讓使用者輸入文字，然而有時使用者會不小心在文字的開頭或結尾輸入空白。有沒有方法能移除這些沒用的空白呢 ?

有的，String 的 function trimmingCharacters(in:) 可以幫助我們，trim 的中文意思是修剪的意思，因此 trimmingCharacters 即代表將字串前後不要的文字剪掉的意思。

func trimmingCharacters(in set: CharacterSet) -> String

在參數 set 傳入不喜歡的文字後 ，它會回傳一個新的字串，將字串前後出現這些文字的地方移除。注意 set 的型別 CharacterSet 是集合，因此我們可以貪心地移除多個不喜歡的字元。

trimmingCharacters(in:) - NSString | Apple Developer Documentation
Returns a new string made by removing from both ends of the receiver characters contained in a given character set.developer.apple.com

比方以下例子會將字首和字尾的我移除，變成為人人，人人為。

let text = "我為人人，人人為我"
let newText = text.trimmingCharacters(in: CharacterSet(charactersIn: "我"))

因此我們只要參數傳入 CharacterSet 裡定義的 whitespaces，即可移除文字前後的空白字元。

let text = " 我為人人，人人為我 "
let newText = text.trimmingCharacters(in: .whitespaces)

如果連換行符號也想移除，則可傳入 whitespacesAndNewlines。

let text = " 我為人人，人人為我\n"
let newText = text.trimmingCharacters(in: .whitespacesAndNewlines)
