控制 Flutter 圖片的大小和縮放

最近彼得潘剛買新的 M3 MAX MacBook Pro，覺得十分地開心，因此以下我們就以 Apple 強大的 m3 Max 圖片為例，示範如何在 Flutter 控制圖片的大小和縮放。

  • 沒有設定 width 和 height。
  • 只設定 width 或 height。
  • 同時設定 width 和 height。
  • 設定 double.infinity。
  • 用 BoxFit 控制圖片如何縮放。
  • BoxFit 搭配 Alignment 設定顯示的區域。
  • 使用 AspectRatio widget 設定圖片比例。
  • 使用 SizeBox 或 Container 限制大小。

沒有設定 width 和 height

依據 parent widget 而產生兩種情況。

  • 比方 Center 包 Image 時，圖片比可用空間小時會顯示原本的大小，圖片比可用空間大時會維持比例縮放至可用空間裡適合的大小。
  • 比方 Row 包 Image 時，圖片會顯示原本的大小。

只設定 width 或 height

只設定 width 時圖片將維持比例縮放，自動計算出剛好的高度。同樣的，只設定 height 時圖片將維持比例縮放，自動計算出剛好的寬度。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Image.asset(
          'assets/m3Max.jpg',
          width: 300,
        ),
      ),
    );
  }
}

設定 width 300，因此正方形圖片的高度也會是 300。

同時設定 width 和 height

同時設定 width 和 height 時，圖片將在 width 和 height 設定的框框裡維持比例縮放且完整顯示。當框框的比例和圖片比例不一樣時，將造成框框的上下或左右留白。

以下程式 Image 的框框是 300 * 100，因此原本正方形的 m3Max 將縮放到 100 * 100 的大小。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Image.asset(
          'assets/m3Max.jpg',
          width: 300,
          height: 100,
        ),
      ),
    );
  }
}

為了看出程式裡 Image 左右留白的效果，我們可將 Image 包在 Container 裡，然後將 Container 的顏色設為紫色。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Container(
          color: Colors.deepPurple,
          child: Image.asset(
            'assets/m3Max.jpg',
            width: 300,
            height: 100,
          ),
        ),
      ),
    );
  }
}

設定 double.infinity

以下程式將 width 設成 double.infinity，圖片的寬度將是可用空間的最大寬度。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Container(
          color: Colors.deepPurple,
          child: Image.asset(
            'assets/m3Max.jpg',
            width: double.infinity,
            height: 100,
          ),
        ),
      ),
    );
  }
}

用 BoxFit 控制圖片如何縮放

Image 的參數 fit 控制圖片如何縮放。比方 BoxFit.cover 可讓圖片維持比例且置中佔滿 width 和 height 設定的框框，但超出框框的部分會被切掉。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Image.asset(
          'assets/m3Max.jpg',
          width: 300,
          height: 100,
          fit: BoxFit.cover,
        ),
      ),
    );
  }
}

如下圖所示，正方形的圖片縮放成 300 * 300 置中佔滿 300 * 100 的框框，上下超出的部分則被切掉。

BoxFit 是 enum，它還有許多實用的 case，比方以下幾種。

BoxFit enum
API docs for the BoxFit enum from the painting library, for the Dart programming language.api.flutter.dev
  • BoxFit.fitWidth。

圖片寬度的部分完整顯示，高度的部分有可能被切除。

  • BoxFit.fitHeight。

圖片高度的部分完整顯示，寬度的部分有可能被切除。

  • BoxFit.contain。

圖片將維持比例且完整顯示，因此以下程式的結果跟剛剛沒加 fit 參數的結果一樣。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Image.asset(
          'assets/m3Max.jpg',
          width: 300,
          height: 100,
          fit: BoxFit.contain,
        ),
      ),
    );
  }
}

BoxFit 搭配 Alignment 設定顯示的區域

當我們使用 BoxFit.cover、BoxFit.fitWidth 顯示圖片時，圖片預設會顯示中間的區塊，超出的地方被切除。不過有時圖片的重點不在中間區塊，此時我們可透過 Alignment 調整。

比方以下程式將對齊圖片下面的區塊，因此變成圖片的上半部被切除。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Image.asset(
          'assets/m3Max.jpg',
          width: 300,
          height: 100,
          fit: BoxFit.cover,
          alignment: Alignment.bottomCenter,
        ),
      ),
    );
  }
}

Alignment 也可以用數字表示，(0, 0) 表示中間，(-1,-1) 表示左上角，(1, 1) 表示右下角。

以下程式設定 Alignment(0, -0.1)，因此圖片顯示的區塊將從中間往下偏移一點，效果比預設的置中更好，不會看到露出一點點的 MAX。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Image.asset(
          'assets/m3Max.jpg',
          width: 300,
          height: 100,
          fit: BoxFit.cover,
          alignment: const Alignment(0, -0.1),
        ),
      ),
    );
  }
}

使用 AspectRatio widget 設定圖片比例

有時我們想要圖片隨螢幕尺寸動態調整大小，但是維持固定的比例，此時可用 AspectRatio widget 限定比例。

以下例子 aspectRatio 16 / 9 表示寬高比 16:9，圖片寬度將等於螢幕寬度減 20 pixel，高度則是圖片寬度 / 16 * 9。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Padding(
          padding: const EdgeInsets.symmetric(horizontal: 10),
          child: AspectRatio(
            aspectRatio: 16 / 9,
            child: Image.asset(
              'assets/m3Max.jpg',
              fit: BoxFit.cover,
            ),
          ),
        ),
      ),
    );
  }
}

從 Widget Inspector 可看到圖片比例的確是 16:9。

使用 SizeBox 或 Container 限制大小

  • 例子 1。
class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: SizedBox(
          width: 300,
          height: 100,
          child: Image.asset(
            'assets/m3Max.jpg',
            fit: BoxFit.cover,
          ),
        ),
      ),
    );
  }
}
  • 例子 2。
class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Container(
          constraints: const BoxConstraints.tightFor(
            width: 300,
            height: 100,
          ),
          child: Image.asset(
            'assets/m3Max.jpg',
            fit: BoxFit.cover,
          ),
        ),
      ),
    );
  }
}
