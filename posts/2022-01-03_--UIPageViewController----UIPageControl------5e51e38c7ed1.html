讓 UIPageViewController 顯示 UIPageControl(小圓點)

使用 UIPageViewController 製作水平滑動的分頁畫面時，它預設就支援UIPageControl(小圓點) 的顯示。接下來我們將以 Apple 的 Page-Based App template 為例，說明如何讓 UIPageViewController 顯示小圓點。

使用 Xcode 的 Page-Based App template 建立專案

新版的 Xcode 沒有 Page-Based App template，請參考以下連結取得 Page-Based App template。

將捲動效果改成 scroll

修改 RootViewController.swift，將 UIPageViewController 的 transitionStyle 設為 scroll。UIPageViewController 只有在 transitionStyle 是 scroll 時才能顯示小圓點，pageCurl 不會顯示小圓點。

override func viewDidLoad() {
        super.viewDidLoad()
       
        self.pageViewController = UIPageViewController(transitionStyle: .scroll, navigationOrientation: .horizontal, options: nil)

定義 UIPageViewControllerDataSource 的 function，讓 UIPageViewController 顯示小圓點

定義 UIPageViewControllerDataSource 的 presentationCount(for:) & presentationIndex(for:)，讓 UIPageViewController 顯示小圓點。

  • 設定小圓點的數量
func presentationCount(for pageViewController: UIPageViewController) -> Int {
     return pageData.count
}
  • 設定要被突顯的小圓點是第幾個

presentationIndex(for:) 回傳的數字決定第幾個小圓點要被突顯。比方回傳 1 時，第二個小圓點將被突顯。我們希望突顯的小圓點表示目前在第幾頁，因此我們利用 pageViewController.viewControllers![0] 取得目前所在頁面的 view controller，然後透過 indexOfViewController 找出它的 index。

func presentationIndex(for pageViewController: UIPageViewController) -> Int {
        return indexOfViewController(pageViewController.viewControllers![0] as! DataViewController)
}
