從 Xcode 登入 Apple ID

如果你是真心開發 iOS App，不是玩玩而已，那麼你一定要從 Xcode 登入你的 Apple ID，如此才能將 App 安裝到 iPhone 和上架 App Store。從 Xcode 登入 Apple ID 的步驟如下:

  1. 點選 Xcode > Preferences。

2. 切換到 Accounts 頁面，點選左下角的 + 。

3. 選擇 Apple ID，點選 Continue。

4. 輸入你的帳號密碼。

大功告成，Apple ID 成功登入 !
