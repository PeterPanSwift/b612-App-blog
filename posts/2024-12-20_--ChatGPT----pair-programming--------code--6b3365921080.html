跟 ChatGPT 一起 pair programming (語音聊天寫 code)

ChatGPT 的 Mac Desktop App 現在可以結合語音模式和 work with app 功能，因此跟 ChatGPT 一起聊天寫 code pair programming 的夢想終於可以實現了 ~

以下彼得潘示範如何跟 ChatGPT Pair Programming，一起用 SwiftUI 開發 BMI App。

建立 Xcode 專案，按下 option + shift + 1 打開 ChatGPT，此時它將自動配對 Xcode，因此 ChatGPT 將可讀取 Xcode 的程式。

ps: ChatGPT 配對 App 的相關說明可參考以下連結。

使用 Mac ChatGPT App 的 Work with App 讀取程式
最近 Mac ChatGPT App 推出新功能 Work with App，透過它我們可以直接讀取 App 正在編輯的程式，不用再辛苦地複製貼上程式或上傳檔案。medium.com

點選右下的聲音按鈕，啟動 ChatGPT 的聊天模式(Advanced Voice mode)。

現在我們可以開始跟 ChatGPT Pair Programming，開心地邊聊天邊寫 code 了。

以下彼得潘截取幾段跟 ChatGPT 聊天的對話。

prompt: 現在第一個參數要打什麼?

answer: 在 TextField 的第一個參數中，填入提示文字，比如 “輸入身高”。這樣就可以了。

prompt: 第二個參數要打什麼?

answer: 第二個參數填寫 $height，這樣就綁定到變數了。

prompt: 是這樣寫嗎?

answer: 沒錯，這樣寫是對的！接下來，你可以添加一個類似的 TextField 來輸入體重。

prompt: 怎麼讓 TextField 有圓角邊框 ?

answer: 要給 TextField 添加圓角邊框，可以使用 .textFieldStyle 並設置為 RoundedBorderTextFieldStyle()。這樣可以實現圓角邊框效果。
