iOS SDK 常見的型別縮寫

UI: UIKit framework

UI 代表 User Interface，定義了 UIViewController，UIButton，UILabel 等型別。(ps: UIKit 的 Kit 是套件的意思)

CG: CoreGraphics framework

CG 代表 Core Graphics，定義了 CGPoint，CGRect，CGFloat 等型別。

AV: AVKit & AVFoundation framework

AV 代表 audio & video，定義了 AVPlayer，AVSpeechUtterance 等型別。

WK: WebKit framework

WK 代表 WebKit，定義了 WKWebView 等型別。

MK: MapKit framework

MK 代表 MapKit，定義了 MKMapView，MKCoordinateRegion 等型別。

SF: SafariServices framework

SF 代表 SafariServices，定義了 SFSafariViewController 等型別。

CI: CoreImage framework

CI 代表 CoreImage，定義了 CIFilter，CIImage 等型別。

UN: UserNotifications framework

UN 代表 UserNotifications，定義了 UNUserNotificationCenter，UNMutableNotificationContent 等型別。

CB: CoreBluetooth framework

CB 代表 CoreBluetooth，定義了 CBCentralManager，CBPeripheral 等型別。
