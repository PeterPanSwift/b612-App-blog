利用 SwiftUI 的 brightness 調整圖片亮度

利用 SwiftUI View 的 function brightness(_:) 我們可以調整 view 的亮度，參數 amount 的範圍是 -1 ~ 1，大於 0 時會提升白色的亮度，小於 0 時會提升黑色的亮度。

func brightness(_ amount: Double) -> some View
brightness(_:) | Apple Developer Documentation
Brightens this view by the specified amount.developer.apple.com

接下來讓我們試試利用 Slider 調整可愛 peter 的圖片亮度吧。

struct ContentView: View {
    
    @State private var brightnessAmount: Double = 0
    
    var body: some View {
        
        VStack {
            Image(.peter)
                .resizable()
                .scaledToFit()
                .brightness(brightnessAmount)
            HStack {
                Text("亮度")
                Slider(value: $brightnessAmount, in: 0...1)
            }
        }
        .padding()
    }
}

說明

呼叫 modifier brightness 調整圖片亮度。

.brightness(brightnessAmount)

brightness 參數的範圍是 0 ~ 1，不過通常過猶不及，如下圖所示，當 brightness 等於 1 時，圖片變成了一片白，再也看不到可愛的 peter。
