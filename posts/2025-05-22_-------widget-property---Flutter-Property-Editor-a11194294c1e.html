方便修改檢視 widget property 的 Flutter Property Editor

從 Flutter 3.32 開始，從 VS Code / Android Studio 開發 Flutter App 時多了一個方便修改檢視 widget property 的工具 Flutter Property Editor !

打開 Flutter Property Editor

  • 方法 1: 從 VS Code 的 Command Palette / Android Studio 的 Actions 視窗叫出 Flutter Property Editor。
  • 方法 2: 點選側邊欄的 Flutter Property Editor 按鈕。

修改檢視 widget property

滑鼠游標點選呼叫 widget constructor 的地方即可查看和編輯 widget 的 property。

比方滑鼠游標點選 Column 後將顯示 Column 的屬性。

set 表示已從程式設定數值，default 表示採用預設值。

點選三角形可叫出選擇清單。

mainAxisAlignment 選擇 .end 後，Column 程式的 mainAxisAlignment 自動更新為 MainAxisAlignment.end。

當 App 執行時，透過 Flutter Property Editor 修改 widget property 將可即時更新 App。

滑鼠游標移到 i 上會顯示屬性的說明。

從 filter 框框可輸入文字過濾出想調整的屬性。
