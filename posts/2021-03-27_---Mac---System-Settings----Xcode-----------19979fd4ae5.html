利用 Mac 的 System Settings 刪除 Xcode 佔據大量空間的資料

使用 Xcode 開發 App 將產生許多資料，隨著時間的累積，它們佔據的空間將十分可觀，甚至可達幾十 G 的規模。

刪除這些 Xcode 產生的資料有很多方法，最簡單的方法莫過於從 macOS 的 System Settings 操作，接下來讓我們一步步清除它們吧。

從 Spotlight 點選 Storage

點選 Developer 右邊的 i

點選想刪除的資料，然後按下 Delete

Xcode Project Build Files & Device Support 佔據了好幾 G 的空間。當空間不夠時，這些都是我們可以狠心刪除的資料。

點選項目後按下 delete 刪除。
