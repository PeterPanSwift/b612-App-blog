Xcode 11 拉 outlet / action — 點選 Adjust Editor Options 按鈕

從前當我們想將 Interface Builder 的東西拉 outlet / action 到程式時，最重要的步驟莫過於點選右上角兩個圓圈的按鈕，打開顯示相關程式的 Assistant Editor (輔助編輯)畫面。

但是在 Xcode 11，右上角兩個圓圈的按鈕卻不見了 !

別緊張，我們還是可以拉 outlet / action 啦，只是變成要點選下圖紅色圈圈標示的 Adjust Editor Options 按鈕。

它可以控制 editor 視窗顯示的內容，點選它後將顯示可設定的選項。主要有以下三種選擇:

  • Show Editor Only
  • Preview
  • Assistant

所以答案很清楚了，想拉 outlet / action 時，只要選擇 Assistant (快速鍵 cmd + option + control + enter)，即可顯示我們熟悉的 Assistant Editor (輔助編輯)畫面。
