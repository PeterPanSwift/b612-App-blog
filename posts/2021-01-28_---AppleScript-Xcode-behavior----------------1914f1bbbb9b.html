利用 AppleScript，Xcode behavior & 自訂快速鍵同時全選跟縮排

最近彼得潘在寫 SwiftUI App 時，時常使用全選跟縮排的快速鍵，cmd+a & ctrl + i。然而要分兩個步驟還是有點麻煩，能不能自訂一組新的快速鍵，結合全選跟縮排的功能呢 ?

當然可以，只要結合 AppleScript 指令，Xcode behavior & 自訂快速鍵。接下來就讓我們一步一腳印完成同時全選跟縮排的懶人功能。

啟動 TextEdit，建立新檔案

File > New。

格式改成 plain text

Format > Make Plain Text。

輸入以下 AppleScript 指令控制 Xcode Select All & Re-Indent

#!/bin/sh
osascript <<END
tell application "System Events"
   tell process "Xcode"
      set frontmost to true
      click menu item "Select All" of menu "Edit" of menu bar 1
      click menu item "Re-Indent" of menu of menu item "Structure" of menu "Editor" of menu bar 1
   end tell
end tell
END

以上指令對應 Xcode menu 的 Edit > Select All & Editor > Structure > Re-Indent。

存檔為 SelectAllAndReIndent.sh

將 SelectAllAndReIndent.sh 變成可執行的檔案

打開 Terminal App，輸入以下指令。(假設 SelectAllAndReIndent.sh 存在 Desktop 資料夾下)

cd Desktop
chmod +x SelectAllAndReIndent.sh

現在 SelectAllAndReIndent.sh 變成可執行的檔案，接下來我們將新增自訂的 Xcode behavior，讓 Xcode 執行它。

打開 Xcode 的 Behaviors 視窗

從 Xcode 的 menu 選擇 Xcode > Behaviors > Edit Behaviors。

點選左下的 + 新增 Behavior

輸入名稱 Select all and reindent & 勾選下方的 Run

點選 Run 旁的 Choose Script，選擇 SelectAllAndReIndent.sh

大功告成 ! 現在我們可以測試一鍵全選和縮排程式的功能了。

使用 Xcode Behavior 全選和縮排

原本縮排慘不忍睹的程式如下。

點選 Xcode > Behaviors > Select all and reindent。

若是第一次要求 Xcode 控制 System Events，此時將跳出以下視窗，點選 OK 即可。(剛剛的 SelectAllAndReIndent.sh 透過 System Events 控制 Xcode 執行全選和縮排)

接著請再點選 Xcode > Behaviors > Select all and reindent。不過令人難過的，還是有問題。

別灰心，只剩最後一步了。請點選 Open System Preferences，將 Security & Privacy 下 Privacy > Accessibility 的 Xcode 勾選。

現在終於沒問題了。點選 Xcode > Behaviors > Select all and reindent，程式順利地全選和縮排 !

設定快速鍵

不過點選 Xcode behavior 還不夠快，因為要使用滑鼠，聽說厲害的大神都是用快速鍵，接下來請參考以下連結，幫 Select all and reindent 加入快速鍵 cmd + ctrl + a。

自訂 App menu command 快速鍵
快速鍵可以大幅提高我們做事的效率，因此很多 App 都為常用功能定義了快速鍵。然後App 不太可能為每個功能定義快速鍵，不過沒關係，只要出現在 App menu 的 command，我們都可以幫它加上快速鍵。medium.com

Cool，現在可以用 cmd + ctrl + a 全選跟縮排了。
