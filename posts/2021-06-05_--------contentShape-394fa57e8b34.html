控制點選範圍的 contentShape

利用 contentShape 我們可以控制點選的範圍，接下來讓我們看看以下幾種應用的例子。

控制點選範圍的 contentShape

利用 contentShape 我們可以控制點選的範圍，接下來讓我們看看以下幾種應用的例子。

  • 設定只有圖案可點選，空白地方不能點選。
  • 設定空白地方也可點選。
  • 加大點選範圍。

設定只有圖案可點選，空白地方不能點選

原本圓形以外的空白區塊也可點選。

struct ContentView: View {
    var body: some View {
        Image(.fish)
            .resizable()
            .scaledToFill()
            .frame(width: 300, height: 300)
            .clipShape(.circle)
            .onTapGesture {
                print("tap")
            }
    }
}

我們想要只有圓形區塊可以點選，加入 modifier contentShape(.circle)，設定圓形範圍是點選區塊。

struct ContentView: View {
    var body: some View {
        Image(.fish)
            .resizable()
            .scaledToFill()
            .frame(width: 300, height: 300)
            .clipShape(.circle)
            .contentShape(.circle)
            .onTapGesture {
                print("tap")
            }
    }
}

設定空白地方也可點選

原本只有圖片可點選，框框裡的空白區塊不能點選。

struct ContentView: View {
    var body: some View {
        Image(.peter)
            .foregroundStyle( .red)
            .frame(width: 300, height: 300)
            .border(Color.black, width: 5)
            .onTapGesture {
                print("tap")
            }
    }
}

我們希望整個框框都可點選，利用 contentShape(.rect) 設定長方形範圍是點選區塊。

struct ContentView: View {
    var body: some View {
        Image(.peter)
            .foregroundStyle( .red)
            .frame(width: 300, height: 300)
            .border(Color.black, width: 5)
            .contentShape(.rect)
            .onTapGesture {
                print("tap")
            }
    }
}

加大點選範圍

原本只有點我可以點選。

struct ContentView: View {
    var body: some View {
        Text("點我")
            .onTapGesture {
                print("點我")
            }
    }
}

加大點選範圍，讓文字附近 100 * 100 的區塊都可點選。

struct ContentView: View {
    var body: some View {
        Text("點我")
            .frame(width: 100, height: 100)
            .contentShape(.rect)
            .onTapGesture {
                print("點我")
            }
    }
}
