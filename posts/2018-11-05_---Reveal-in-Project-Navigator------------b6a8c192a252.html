利用 Reveal in Project Navigator 展開資料夾和選取檔案

在 Xcode 裡我們時常利用 Open Quickly (cmd + shift + o) 找東西，快速在 Editor Area 開啟相關的檔案。

比方如下圖所示，找到 CharacteristicCell.xib 後，Editor Area 顯示 xib 的畫面。

然而有時我們想知道此檔案在 Project Navigator 的位置，或是希望它在 Project Navigator 裡被選取。

其中一種做法是從上方的 jump bar 查詢檔案的路徑。從以下三張圖，我們可知 CharacteristicCell.xib 長在 HMCatalog 的 Homes 的 Accessories 下，於是可從 Project Navigator 一層層展開相關的資料夾。

然而當資料夾很多層時，此方法將變得十分麻煩。除非你像楊過一樣有耐心，願意等小龍女十六年。不然建議你採用另一個方法。

只要從 menu 點選 Navigate > Reveal in Project Navigator (cmd + shift + J )，即可自動展開 Project Navigator 下的相關資料夾並選取檔案。
