Xcode 12.2 的 menu 多了方便的 Create Preview 功能，幫我們自動生成顯示 SwiftUI preview 畫面的型別。接下來就讓我們看看如何操作吧。

Xcode 12.2 的 Create Preview

Xcode 12.2 的 menu 多了方便的 Create Preview 功能，幫我們自動生成顯示 SwiftUI preview 畫面的型別。接下來就讓我們看看如何操作吧。

定義 SwiftUI view 的型別 ContentView

import SwiftUI
struct ContentView: View {
    var body: some View {
        Image("神隱少女")
            .resizable()
            .scaledToFit()
    }
}

點選 Xcode menu 的 Editor > Create Preview

此時將自動產生遵從 protocol PreviewProvider 的型別 ContentView_Previews，它的變數 previews 代表右邊顯示的預覽畫面。

不過它沒那麼聰明，它不明白我們想顯示什麼，所以我們得在變數 previews 的 { } 裡產生想顯示的畫面，比方 ContentView()。

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
    }
}
