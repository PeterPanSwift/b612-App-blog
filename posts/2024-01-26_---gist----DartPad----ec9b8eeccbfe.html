利用 gist 分享 DartPad 程式

在 DartPad 練習 Flutter 跟 Dart 程式時，我們可以利用 GitHub 的 gist 分享程式，讓別人方便地從 DartPad 載入、修改和執行程式。

以下是分享的逐步解說。

連到 GitHub 的 gist 網站

Discover gists
GitHub Gist: instantly share code, notes, and snippets.gist.github.com

建立 gist

輸入 Dart 程式，取名為 main.dart，點選右下的 Create secret gist。

ps: 也可以點選 Create public gist。不管是 Private Gist 或 Public Gist，都能產生分享程式給別人的 DartPad 連結，差別只在 gist 本身是否公開。

DartPad
An online Dart editor with support for console and Flutter apps.dartpad.dev

取得 gist 的 id

成功建立 gist 後，從網址列可看到 gist 的網址，/ 後的亂碼即為 gist 的 id，比方下圖的 81c56849193ca7330e0f5c20bbdd76dd。

https://gist.github.com/PeterPanSwift/81c56849193ca7330e0f5c20bbdd76dd

從 DartPad 顯示 gist 的程式

在以下網址的 = 後加上 gist id 即可從 DartPad 顯示 gist 的程式。

https://dartpad.dev/?id=

因此從瀏覽器連到以下網址將載入剛剛生成的 gist 程式。

https://dartpad.dev/?id=81c56849193ca7330e0f5c20bbdd76dd

之後任何人連到此連結都會自動從 DartPad 網站載入 gist 裡寫好的程式，而且他們也可自由修改，不會影響到 gist 原本的程式。
