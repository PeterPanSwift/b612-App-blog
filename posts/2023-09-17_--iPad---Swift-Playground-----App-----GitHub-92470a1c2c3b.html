將 iPad 上 Swift Playground 開發的 App 上傳到 GitHub

透過 Swift Playground，我們在 iPad 上也能開發 iOS App，不過它目前尚未支援 git 版本管理和上傳到 GitHub 的功能。

因此以下彼得潘將示範如何快速將 Swift Playground 開發的 App 上傳到 GitHub。

ps: 此方法不考慮版本管理，我們的目標只是將 App 的程式傳到 GitHub 。

取得 App 的 swiftpm 檔

在 iPad Swift Playgrounds 建立的 App 是附檔名為 swiftpm 的檔案。swiftpm 檔其實是個資料夾，裡面包含了程式碼，圖片，和各種組成 App 的檔案。

它預設會存在 iCloud drive 的 Swift Playground 資料夾，因此我們可從 iCloud Drive 取得 swiftpm 檔。以下分別說明 Mac & Windows 的操作步驟。

ps: 若檔案沒有存在 iCloud，可將 App 的 swiftpm 檔直接以 Share > Mail 的方法寄給自己。

  • 使用 Mac。

打開 Finder，點選 iCloud Drive 下的 Swift Playground。

從 Swift Playground 找出要上傳的 App 檔案。

  • 使用 Windows。

從以下連結登入 iCloud Drive

iCloud.com
Sign in to iCloud to access your photos, videos, documents, notes, contacts, and more. Use your Apple ID or create a…www.icloud.com

登入後從 Recents 分頁或 Browse 分頁的 Swift Playground 下載 App 的 swiftpm 檔。

將 swiftpm 檔上傳到 GitHub

登入 GitHub，在 GitHub 建立 repository，然後將 swiftpm 檔上傳到 GitHub。

點選 uploading an existing file。

上傳 swiftpm 檔，上傳後它會自動被壓縮成 zip 檔。

GitHub repository 成功出現我們剛剛上傳的檔案。
