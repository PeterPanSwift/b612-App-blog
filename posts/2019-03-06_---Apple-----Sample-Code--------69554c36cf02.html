搜尋 Apple 官方的 Sample Code (範例程式)

Apple 官方的 Sample Code (範例程式) 往往是我們學習新功能的最佳參考對象，畢竟 Apple 出品，品質保證。但是我們要怎麼找到這些 Sample Code 呢 ?

從 Xcode 10.2 開始，我們從 Developer Documentation 即可快速找到 Sample Code，方法如下:

Documentation for symbols can be filtered by SDK availability, introduced version, and deprecation. Documentation can also be filtered to show only articles or sample code. For example, you can filter the documentation to show all of the sample code in a framework like UIKit.
Xcode 10.2 Beta 4 Release Notes
Xcode 10.2 beta 4 includes SDKs for iOS 12.2, watchOS 5.2, macOS 10.14.4, and tvOS 12.2. Xcode 10.2 supports on-device…developer.apple.com
1 點選 Xcode menu 的 Help > Developer Documentation。
2 在 Developer Documentation 左下角的 filter 框框輸入 sample code。

Xcode 很貼心，只要輸入 sam，它將顯示 Sample Code 的提示。

點選 Sample Code 後，左邊的 navigator 區塊將顯示 Sample Code 的相關資料。

不過並非全部的資料都是 Sample Code，只有 Icon 圖案為 { } 的才是 Sample Code。

比方我們點選 UIKitCatalog: Creating and Customizing Views and Controls，看到它的相關說明，點選 Download 按鈕即可下載它的 Xcode 專案。

開啟下載的 UIKitCatalog 專案。

如果想搜尋某個主題的 sample code，也可以在 filter 框框裡輸入相關的文字，比方 sample code Scollection 將找到 collection view 相關的 sample code。
