列出參數排列組合的自動完成 — Xcode 15 新功能

Xcode 15 的自動完成變得更方便了 ! 當參數因為預設值變成可省略時，我們在輸入時可以按向右鍵請它列出所有可能的排列組合。

以下我們分成兩種情況討論。

  • 多個參數有預設值。
  • 只有一個參數有預設值。

多個參數有預設值

以 function add 為例，參數 number3 & number4 都有預設值。

func add(number1: Int, number2: Int, number3: Int = 0, number4: Int = 0) -> Int {
    return number1 + number2 + number3 + number4
}

當我們輸入 add 時，預設的選項只有必填的 number1 & number2。此時它的旁邊會出現 >，表示我們可以按向右鍵展開。

展開後它將列出所有可能的排列組合，箭頭也變成了向下的 v。

展開時我們還可以輸入參數的關鍵字過濾。比方輸入 3，只顯示包含參數 number3 的選項。

只有一個參數有預設值

當只有一個參數有預設值時，由於選項不多，Xcode 會貼心地自動展開，不用另外按向右鍵展開。

func add(number1: Int, number2: Int, number3: Int = 0) -> Int {
    
    return number1 + number2 + number3
}
