Dart 3.10 的新功能: Dot shorthands

Dart 3.10 推出方便的新功能 Dot shorthands，從 Dart 3.10 & Flutter 3.38 開始，當型別已知時，我們可以省略型別，直接輸入 .xxx 即可。

以常見的 Column widget 為例，原本為了設定 alignment & size，我們必須輸入冗長的 MainAxisAlignment.end、MainAxisSize.min、CrossAxisAlignment.start。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.end,
          mainAxisSize: MainAxisSize.min,
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            Text('彼得潘'),
            Text('小王子'),
            Text('愛麗絲夢遊仙境'),
          ],
        ),
      ),
    );
  }
}

有了 Dot shorthands，因為 Dart 知道參數 mainAxisAlignment、mainAxisSize、crossAxisAlignment 的型別是 MainAxisAlignment、MainAxisSize、CrossAxisAlignment，因此我們不用再打一次，輸入 .end、.min、.start 即可。

Column(
  mainAxisAlignment: .end,
  mainAxisSize: .min,
  crossAxisAlignment: .start,
  children: [
    Text('彼得潘'),
    Text('小王子'),
    Text('愛麗絲夢遊仙境'),
  ],
)

剛剛的例子我們省略了 enum 型別，以下我們再看到省略 named constructor 型別的例子。

以 padding 為例，從前我們常常用 EdgeInsets.all 產生 padding。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Padding(
        padding: const EdgeInsets.all(8.0),
        child: Column(
          mainAxisAlignment: .end,
          mainAxisSize: .min,
          crossAxisAlignment: .start,
          children: [
            Text('彼得潘'),
            Text('小王子'),
            Text('愛麗絲夢遊仙境'),
          ],
        ),
      ),
    );
  }
}

現在我們可以省略型別 EdgeInsets，只要寫 .all 即可。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Padding(
        padding: const .all(8.0),
        child: Column(
          mainAxisAlignment: .end,
          mainAxisSize: .min,
          crossAxisAlignment: .start,
          children: [
            Text('彼得潘'),
            Text('小王子'),
            Text('愛麗絲夢遊仙境'),
          ],
        ),
      ),
    );
  }
}
