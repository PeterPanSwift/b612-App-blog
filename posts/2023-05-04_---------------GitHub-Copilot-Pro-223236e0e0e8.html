註冊 GitHub 帳號，設定 Profile

註冊 GitHub 帳號，然後設定 GitHub 的 Profile，填寫個人名字並在 Bio 做簡單的介紹，之後 GitHub 在驗證身份時會比對 Profile 的名字跟證明文件的名字是否一致。

驗證老師 / 學生身份

連到以下連結，點選 Start an application。

Build software better, together
GitHub is where people build software. More than 150 million people use GitHub to discover, fork, and contribute to…github.com

選擇老師 / 學生身份和學校，輸入學校 email，然後點選 Share Location。如果 GitHub 帳號不是 edu 結尾的學校信箱，可點選 Add it here 新增學校信箱。

ps: 在電腦操作時 Share Location 可能出現錯誤 Error getting location，建議改用手機操作。

點選 Continue 後，進入身份證明的頁面，可選擇上傳教師 / 學生證或其它證明身份的文件。

彼得潘上傳海洋大學的聘書。

等待 GitHub 的審核，才等了幾分鐘，還來不及喝杯咖啡，重新整理網頁發現狀態已變成 Approved。

取得免費使用 GitHub Copilot Pro 的權限

使用 GitHub Copilot 寫程式

現在可以開始使用 copilot 寫程式了。以下彼得潘請 copilot 用 SwiftUI 寫一個鬼故事 App。

  • 使用 Copilot 的 agent mode 寫程式。

AI 的回答。

App 畫面。

  • 輸入 Copilot 的提示程式。

先寫注解，然後按 enter 後 copilot 會提示可輸入的程式。

按 tab 可完成輸入。

如果提示消失，可按 Trigger Inline Suggestion 的快速鍵option 加 \ 觸發提示再次出現。

ps: 如果不想整段輸入，只想輸入部分內容，我們還有以下兩種選項。

  • Accept Word: 輸入一個單字，快速鍵 cmd + 向右鍵。
  • Accept Line: 輸入一行，預設無快速鍵，以下是彼得潘另外設定的 cmd + ctrl + 英文的 l。

struct StoryView: View {
    var story: Story
    var body: some View {
        VStack {
            Image(story.imageName)
                .resizable()
                .scaledToFit()
            Text(story.content)
                .padding()
        }
        .navigationBarTitle(story.title)
    }
}

struct Story: Identifiable {
    var id = UUID()
    var title: String
    var content: String
    var imageName: String
}

Premium Models 的額度

Premium Models 的 request 有數量的限制， GitHub Copilot 一個月可下 50 個 request，GitHub Copilot Pro 一個月可下 300 個。若需要更多的 request 則須額外付錢。
