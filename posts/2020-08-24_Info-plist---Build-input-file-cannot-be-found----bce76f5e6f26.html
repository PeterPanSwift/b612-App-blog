Info.plist 的 Build input file cannot be found 錯誤

有朋友修改 Xcode 專案下的資料夾名稱後發生可怕的慘劇。如下圖所示，資料夾原本叫 Demo。

改名為 Demo2。

此時 App 完全跑不起來，出現以下紅色錯誤。

Build input file cannot be found: '/Users/shih-yingpan/Desktop/Demo/Demo/Info.plist'

問題發生的原因出在資料夾名稱變成 Demo2，因此我們必須修改 Info.plist 的路徑，它原本是 Demo/Info.plist，應該要更新成 Demo2/Info.plist。

解法

  • 切換到 TARGETS App 的 Build Settings 分頁，找到 Info.plist File。
  • 將 Demo/Info.plist 更新成 Demo2/Info.plist。
