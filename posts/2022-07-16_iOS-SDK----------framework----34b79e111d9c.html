iOS SDK 的型別開頭述說著 framework 縮寫

iOS SDK 定義了各式各樣的型別，幫助我們快速開發 iOS App 需要的功能。這些型別名稱的開頭通常包含著定義它們的 framework 縮寫，以下我們簡單介紹一些常見的例子。

UI

來自 UIKit framework，比方 UIView，UIImageView，UIImage，UIColor，UIButton，UILabel，UIViewController， UITableViewController 等。

CG

來自 Core Graphics，比方 CGRect，CGFloat，CGPoint，CGSize，CGColor，CGAffineTransform。

CA

來自 Core Animation，比方 CALayer，CAShapeLayer，CAGradientLayer，CAAnimation，CATransition。

AV

來自 AVFoundation，比方 AVPlayer，AVSpeechSynthesizer。

MK

來自 MapKit，比方 MKMapView，MKCoordinateRegion，MKPointAnnotation。

SF

來自 SafariServices，比方 SFSafariViewController。
