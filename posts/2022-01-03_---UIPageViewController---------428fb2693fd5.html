控制 UIPageViewController 切換到指定頁面

UIKit 的 UIPageViewController 可以方便地做出 App 的分頁效果，使用者可以左滑到下一頁，右滑到上一頁。

然而有時我們會想從程式控制 UIPageViewController 切換到指定頁面，比方以下例子:

  • 自動輪播，UIPageViewController 每隔幾秒自動切換到下一頁。

範例: LINE App 的 Today 頁面，每隔幾秒自動切換到下一個精彩的電影消息。

  • 使用者點選分頁的標題，快速切換到某個分頁。

範例: LINE App 的購物頁面，點選通路門市快速切換到對應頁面。

接下我們將以 Apple 的 Page-Based App template 為例，示範如何控制 UIPageViewController 切換到指定頁面。

使用 Xcode 的 Page-Based App template 建立專案

新版的 Xcode 沒有 Page-Based App template，請參考以下連結取得 Page-Based App template。

將捲動效果改成 scroll

修改 RootViewController.swift，將 UIPageViewController 的 transitionStyle 設為 scroll。

override func viewDidLoad() {
        super.viewDidLoad()
       
        self.pageViewController = UIPageViewController(transitionStyle: .scroll, navigationOrientation: .horizontal, options: nil)

加入 navigation controller，在 navigation bar 的右上方加入 text field

之後使用者將在 text field 輸入月份的數字，然後按下 return 鍵切換到指定月份的頁面。

控制 UIPageViewController 切換到指定頁面

輸入月份數字後，按下 return 鍵切換到指定頁面

@IBAction func returnTapped(_ sender: UITextField) {
        if let month = Int(sender.text!),
           (1...12).contains(month) {
            let newIndex = month - 1
            let originalIndex = modelController.indexOfViewController(pageViewController!.viewControllers![0] as! DataViewController)
            let controller = modelController.viewControllerAtIndex(newIndex, storyboard: storyboard!)
            if newIndex > originalIndex {
                pageViewController?.setViewControllers([controller!], direction: .forward, animated: true, completion: nil)
            } else {
                        pageViewController?.setViewControllers([controller!], direction: .reverse, animated: true, completion: nil)
            }
        }
}

說明

呼叫 UIPageViewController 的 setViewControllers(_:direction:animated:completion:)，傳入要顯示的 view controller。值得注意的，我們必須判斷準備顯示的頁面是之前的頁面還是之後的頁面，如此才能決定捲動的方法是 forward 或 reverse。

執行 App

在 text field 輸入 5，快速跳到 5 月的頁面。
