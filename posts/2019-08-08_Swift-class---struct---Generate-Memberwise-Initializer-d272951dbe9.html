Swift class & struct 的 Generate Memberwise Initializer

撰寫 Swift 程式時為了方便建立某個型別的東西，我們時常會為型別定義 initializer。如果是 struct 的話會簡單一些，因為它可以自己生成 memberwise initializer。

可惜 class 沒這個功能，不過沒關係，Xcode 捨不得我們辛苦，它願意幫我們自動生成 class 的 memberwise initializer。

ps: Xcode 14 有更簡單的方法

memberwise init 的自動完成 — Xcode 14 新功能
從 Xcode 14 開始，Xcode 可以自動幫我們生成包含每個 property 的 memberwise init ，方法如下:medium.com

class 的 Generate Memberwise Initializer

假設 class Song 的定義如下，

class Song {
   var name: String
   var singer: String
}

將滑鼠游標移到 class 上，按住 cmd 鍵後按滑鼠左鍵，從選單裡點選 Generate Memberwise Initializer。

此時 Song 的 memberwise initializer 將自動生成，我們不用再自己辛苦輸入 init 了。 (ps: 如果需要其它特別的 init ，還是得自己定義。)

class Song {
   internal init(name: String, singer: String) {
      self.name = name
      self.singer = singer
   }
   var name: String
   var singer: String
}

struct 的 Generate Memberwise Initializer

struct 自動就會產生 memberwise initializer，不過當我們自訂 init 後，它就不會幫我們生成 memberwise initializer。

struct Baby {
    let name: String
    let age: Int
    
    init(name: String) {
        self.age = 1
        self.name = name
    }
}

如果還是希望有 memberwise initializer，我們也可以將滑鼠游標移到 struct 上，按住 cmd 鍵後按滑鼠左鍵，從選單裡點選 Generate Memberwise Initializer。

struct Baby {
    internal init(name: String, age: Int) {
        self.name = name
        self.age = age
    }
    
    let name: String
    let age: Int
    
    init(name: String) {
        self.age = 1
        self.name = name
    }
}

ps: Generate Memberwise Initializer 只能在 project 使用，playground 沒有提供。
