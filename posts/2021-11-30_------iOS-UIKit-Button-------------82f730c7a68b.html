從程式設定和讀取 iOS UIKit Button 的文字，圖片 & 樣式

從 iOS 15 開始，UIKit 的 button 有新的設定方法，以下我們分別介紹新版跟舊版的 button 寫法。

iOS 15 以上(包含 15)

iOS 15 button 介紹

iOS 15 更容易客製的 UIButton — plain，gray，tinted & filled 四種樣式
iOS 15 的 UIButton 進化了 ! 現在它有 Plain，Gray，Tinted & Filled 四種樣式可選擇，可以做更多的客製，比方調整圖片的位置，圓角的弧度等。medium.com

讀取 button 的文字

從 button 的 configuration?.title 讀取。

button.configuration?.title

設定文字 & 文字顏色。

從 Configuration 的 attributedTitle & baseForegroundColor 設定文字 & 顏色。

var configuration = UIButton.Configuration.filled()
var title = AttributedString("Login")
title.font = UIFont(name: "AmericanTypewriter-Bold", size: 30)
configuration.attributedTitle = title
configuration.baseForegroundColor = .yellow
let action = UIAction { action in
    print("button tap")
}
let button = UIButton(configuration: configuration, primaryAction: action)
button.frame = CGRect(x: 100, y: 100, width: 140, height: 70)

button 的四種樣式

override func viewDidLoad() {
        super.viewDidLoad()
        
        let configurations: [UIButton.Configuration] = [
            .plain(),
            .tinted(),
            .gray(),
            .filled(),
        ]
        for i in configurations.indices {
            var configuration = configurations[i]
            var title = AttributedString("Login")
            title.font = UIFont(name: "AmericanTypewriter-Bold", size: 30)
            configuration.attributedTitle = title
            configuration.baseForegroundColor = .yellow
            let action = UIAction { action in
                print("button tap")
            }
            let button = UIButton(configuration: configuration, primaryAction: action)
            button.frame = CGRect(x: 100, y: 80*(i+1), width: 140, height: 70)
            view.addSubview(button)
        }
        
}

設定圖片

設定 Configuration 的 image。 值得注意的，圖片顯示時會維持原本的大小，因此如果圖片太大，圖片將超出 button 的框框。若是希望圖片依 button 尺寸調整大小，請將圖片設成背景圖片。

var configuration = UIButton.Configuration.plain()
configuration.image = UIImage(named: "peter")
let action = UIAction { action in
    print("button tap")
}
let button = UIButton(configuration: configuration, primaryAction: action)
button.frame = CGRect(x: 100, y: 100, width: 100, height: 160)

設定圖文並排 & 圖片位置

從 Configuration 的 imagePlacement 設定圖片位置。

var configuration = UIButton.Configuration.filled()
var title = AttributedString("Login")
title.font = UIFont(name: "AmericanTypewriter-Bold", size: 30)
configuration.attributedTitle = title
configuration.image = UIImage(systemName: "star.fill")
configuration.baseForegroundColor = .yellow
configuration.imagePlacement = .top
let action = UIAction { action in
    print("button tap")
}
let button = UIButton(configuration: configuration, primaryAction: action)
button.frame = CGRect(x: 100, y: 100, width: 140, height: 70)

設定背景圖片，圖片縮放模式，背景顏色，邊框顏色，邊框寬度 & 圓角樣式

configuration.background 的型別是 UIBackgroundConfiguration，從它的 image，imageContentMode，backgroundColor，strokeColor & strokeWidth 設定背景圖片，圖片縮放模式，背景顏色，邊框顏色 & 邊框寬度。Configuration 的 cornerStyle 則可控制圓角樣式。

var configuration = UIButton.Configuration.filled()
configuration.background.image = UIImage(named: "peter")
configuration.background.imageContentMode = .scaleAspectFill
configuration.background.backgroundColor = .orange
configuration.background.strokeColor = .red
configuration.background.strokeWidth = 6
configuration.cornerStyle = .capsule
let action = UIAction { action in
    print("button tap")
}
let button = UIButton(configuration: configuration, primaryAction: action)
button.frame = CGRect(x: 100, y: 100, width: 200, height: 200)
view.addSubview(button)

取得 button 的文字，圖片 & 背景圖片

讀取 button.configuration?.title, button.configuration?.image & button.configuration?.background.image。

var configuration = UIButton.Configuration.filled()
var title = AttributedString("Login")
title.font = UIFont(name: "AmericanTypewriter-Bold", size: 30)
configuration.attributedTitle = title
configuration.baseForegroundColor = .yellow
configuration.image = UIImage(named: "peter")
configuration.background.image = UIImage(named: "grass")
configuration.background.imageContentMode = .scaleAspectFill
configuration.background.backgroundColor = .orange
configuration.background.strokeColor = .red
configuration.background.strokeWidth = 6
configuration.cornerStyle = .capsule
let action = UIAction { action in
    print("button tap")
}
let button = UIButton(configuration: configuration, primaryAction: action)
button.frame = CGRect(x: 100, y: 100, width: 200, height: 200)
print(button.configuration?.title, button.configuration?.image, button.configuration?.background.image)

舊版

若是從 Interface Builder 加入 button，當 Style 為 Default 時即為舊版的 button。

讀取 button 的文字

從 button 的 currentTitle。

button.currentTitle

設定文字，圖片 & 背景圖片

呼叫 UIButton 的 setTitle，setImage & setBackgroundImage 設定文字，圖片 & 背景圖片，button.imageView?.contentMode 設定圖片的縮放模式(不是背景圖片)。

let button = UIButton(frame: CGRect(x: 100, y: 100, width: 200, height: 200))
button.setTitle("Login", for: .normal)
button.setTitleColor(.yellow, for: .normal)
button.titleLabel?.font = UIFont(name: "AmericanTypewriter-Bold", size: 30)
button.setImage(UIImage(named: "peter"), for: .normal)
button.imageView?.contentMode = .scaleAspectFill
button.setBackgroundImage(UIImage(named: "grass"), for: .normal)

取得 button 的文字，圖片 & 背景圖片

讀取 UIButton 的 currentTitle，currentImage & currentBackgroundImage。

let button = UIButton(frame: CGRect(x: 100, y: 100, width: 200, height: 200))
button.setTitle("Login", for: .normal)
button.setTitleColor(.yellow, for: .normal)
button.titleLabel?.font = UIFont(name: "AmericanTypewriter-Bold", size: 30)
button.setImage(UIImage(named: "peter"), for: .normal)

print(button.currentTitle, button.currentImage, button.currentBackgroundImage)
