SwiftUI 的 view value & view identity

view value 不等於 view identity。view value 指的是每次畫面更新，body 回傳的 view。

當 PurrDecibelView 被建立和出現後，SwiftUI 設定它的 identity。之後當畫面更新時，view value 會改變，但 view identity 不變。PurrDecibelView 在以下兩種情況會結束生命。

  • 從畫面上移除。
  • 改變 identity，比方改變 id。

State & StateObject 的資料將一直存在，直到 view 的生命結束。
