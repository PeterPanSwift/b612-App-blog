請 IDE 的 AI 修正程式錯誤 — 以 Xcode、VS Code、Android Studio 為例

內建 AI 的 IDE 可以幫我們寫程式，不過 AI 也常常寫出 bug。俗話說得好，自己的 bug 自己修，所以我們當然也可以要求 AI 修正 bug。

以下我們介紹 Xcode、VS Code、Android Studio 請 AI 修正程式錯誤的方法。

Xcode Coding Assistant

  • 方法 1。

點擊程式的紅色錯誤。

點擊 Generate Fix for Issue 的 Generate。

  • 方法 2。

從 Issue navigator 的錯誤點選右鍵選單的 Generate Fix for Issue。

AI 成功修正錯誤。

GitHub Copilot for Xcode

在 GitHub Copilot 的輸入框輸入 fix error 和錯誤訊息。

VS Code

請先安裝 Xcode-Build-Server，讓 VS Code 能正確判斷 Swift 程式的相關錯誤。

使用 VS Code 搭配 Xcode-Build-Server 開發 Xcode 專案的程式
VS Code 只要安裝 Swift extension 即可開發 Swift 程式。不過如果使用 VS Code 打開 Xcode 建立的專案資料夾，卻會發現以下問題。medium.com
  • GitHub Copilot。

方法 1: 滑鼠游標移到紅色錯誤上，點擊 Fix using Copilot。

方法 2: 從 Problems 區塊的錯誤點選右鍵選單的 Fix using Copilot。

AI 修正錯誤後，點選 Accept 接受 AI 的修改。

  • 使用 Codex 或 Claude Code 修正錯誤。

Android Studio

選取有問題的程式，請 Agent mode 的 Gemini fix bug。

點選 Accept Change 接受 AI 的修改。
