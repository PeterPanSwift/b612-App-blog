從Xcode七步建立Apple ID

古有曹植七步成詩，今有Xcode七步建立Apple ID。想不到就連建立 Apple ID 這點小事，都可以在Xcode完成 !

  1. 點選Xcode -> Preferences。

2. 切換到Accounts頁面，點選左下角的 + 後，點選 Add Apple ID。

3. 點選 Create Apple ID。

4. 輸入出生年月日，男人的年齡是秘密，所以在這裡彼得潘不方便公開。

5. 填寫email，密碼等帳號相關資訊。

6. 設定三個安全提示問題和答案。

大功即將告成，請到Apple ID的信箱收信完成最後的驗證步驟。

7. 點選Apple ID驗證信裡的Verify Now。

驗證成功，順利建立Apple ID !
