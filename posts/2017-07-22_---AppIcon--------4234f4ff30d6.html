少了 AppIcon 的我該怎麼辦

當我們建立 Xcode 專案後，一開始 Assets.xcassets 即為我們貼心地準備好 AppIcon，方便我們加入 App 的 icon 圖片。

不過，如果我們不小心將 Assets.xcassets 的 AppIcon 刪除，將產生很可怕的紅色錯誤，連 App 都無法生成 !
None of the input catalogs contained a matching stickers icon set or app icon set named “AppIcon”.

解法:

在 Assets.xcassets 的左邊區塊按右鍵，從選單裡點選 App Icons & Launch Images > New iOS App Icon，重新產生 AppIcon。
