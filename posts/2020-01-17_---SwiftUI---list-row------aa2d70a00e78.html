設定 SwiftUI 的 list row 點選功能

當我們在 iOS App 的畫面上呈現表格時，時常會加入點選 cell 的功能，比方點選下圖 Music App 表格上的怎樣時，可以聽到 Penny 好聽的歌聲。

那麼在 SwiftUI 的 List 要如何設定表格點選的功能呢 ? 如果想點選 row 切換頁面可透過 NavigationLink，但如果不是切換頁面呢 ? 比方剛剛 Music App 的播放音樂功能。

利用 NavigationView & NavigationLink 切換頁面
SwiftUI 利用 NavigationView 管理切換多層頁面，它會在畫面的上方長出一條 navigation bar，bar 的左上角則有 back 返回的按鈕，類似以下的 Music App 畫面。medium.com

接下來彼得潘將以偶像 Penny 的經典情歌為例，示範 button & tap gesture 兩種實現表格點選功能的方法。

struct ContentView: View {
    var songs = ["怎樣", "你要的愛", "街角的祝福", "單身潛逃", "野薔薇", "賊", "防...空洞", "愛瘋了", "光著我的腳丫子"]
    var body: some View {
       
        List(songs, id: \.self) { (song)  in
            HStack {
                Text(song)
                Spacer()
                Image(systemName: "icloud.and.arrow.down")
                    .onTapGesture {
                        print("download \(song)")
                }
                
            }
        }
    }
}

透過以上程式，我們以表格呈現 Penny 的經典情歌，這些歌都很好聽，可惜只能看不能聽，點選表格時毫無反應，也完全不會有點選變色的效果。

方法1: 利用 button 當 row

List(songs, id: \.self) { (song)  in
            
    Button(action: {
        print("play \(song)")
    }, label: {
        HStack {
           Text(song)
           Spacer()
           Image(systemName: "icloud.and.arrow.down")
              .onTapGesture {
                   print("download \(song)")
           }
        }
    })          
}

利用 Button 當 List 的 row，表格點選時順利觸發 button 的 action，印出歌曲的名稱，而且還有點選變色的效果。

方法2: 利用 tap gesture

List(songs, id: \.self) { (song)  in
    HStack {
        Text(song)
        Spacer()
        Image(systemName: "icloud.and.arrow.down")
            .onTapGesture {
                print("download \(song)")
        }
    }
    .onTapGesture {
        print("play \(song)")
    }
}

在代表 row 的 HStack 上呼叫 onTapGesture，設定 row 被點選時印出歌曲名稱。

當我們點選表格上的某首情歌時，Console 順利地印出歌名。

但是等等，當我們點選歌名和下載按鈕中間的空白處時，它卻毫無反應 !

這是因為空白處是 HStack 沒有顏色的區塊，當我們點選 HStack 無顏色的區塊時，它是不會偵測到 tap gesture 的。此問題可透過呼叫 contentShape 解決。

List(songs, id: \.self) { (song)  in
   HStack {
      Text(song)
      Spacer()
      Image(systemName: "icloud.and.arrow.down")
         .onTapGesture {
             print("download \(song)")
      }
   }
   .contentShape(Rectangle())
   .onTapGesture {
      print("play \(song)")
   }
}

contentShape 控制可點選的區塊，因此從 HStack 呼叫 contentShape(Rectangle()) 將讓整個 HStack 可點選。
