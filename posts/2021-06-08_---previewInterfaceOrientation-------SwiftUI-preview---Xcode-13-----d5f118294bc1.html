利用 previewInterfaceOrientation 設定橫向的 SwiftUI preview — Xcode 13 新功能

接下來我們將示範利用 previewInterfaceOrientation將以下程式的直向預覽畫面變成橫向。(ps: 此功能必須搭配 Xcode 13 & iOS 15 以上版本）

利用 previewInterfaceOrientation 設定橫向的 SwiftUI preview — Xcode 13 新功能

接下來我們將示範利用 previewInterfaceOrientation 將以下程式的直向預覽畫面變成橫向。

struct ContentView: View {
    var body: some View {
        Image("wwdc")
            .resizable()
            .scaledToFit()
    }
}
struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
    }
}

設定橫向(landscape)的 preview

方法 1: 點選 preview 上方的旋轉按鈕

Cool，現在 preview 畫面變橫向了。

而且按三角形啟動後還是可以維持橫向。

此時 preview 的程式也新增了 modifier previewInterfaceOrientation。

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
           .previewInterfaceOrientation(.landscapeLeft)
    }
}

ps: 值得注意的，iOS 15 以上才支援 previewInterfaceOrientation，所以 iOS 15 以下版本將出現紅色錯誤。

從 TARGETS App 的 General 頁面的 Development Info 可設定 App 支援的最低 iOS 版本。

方法 2: 從 Attributes inspector 設定 Orientation

點選 preview 的 ContentView。

從 Attributes inspector 設定 Orientation。

方法 3: 從程式直接輸入 previewInterfaceOrientation

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
            .previewInterfaceOrientation(.landscapeLeft)
    }
}

讓舊版 Xcode 的 preview 橫向

Xcode 13 之前的版本無法使用 previewInterfaceOrientation，不過我們可以利用 previewLayout 設定固定尺寸，比方設定 iPhone 13 橫向的寬高 844 * 390。

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
            .previewLayout(.fixed(width: 844, height: 390))
        
    }
}

ps: 此方法有個小小的缺點，點選 preview 的三角形執行後它將自動變回直向，因此測試 App 執行後的畫面必須透過模擬器。
