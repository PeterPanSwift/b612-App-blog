Zed IDE 的基本操作和 AI 功能初體驗

最近 Zed IDE 推出了強大的 AI 功能，我們可以從 IDE 選擇喜好的 AI 助手協助開發程式，比方 Claude、GPT、Gemini、llama 等。這些 AI 大部分都要搭配 API key 跟付費才能使用，不過目前 Zed 免費提供 Claude 3.7 Sonnet，只要登入帳號即可享用。

以下彼得潘簡單分享 Zed IDE 的基本操作和 AI 相關功能。

下載安裝 Zed

連到 Zed 網站。

Zed - The editor for what's next
Zed is a high-performance, multiplayer code editor from the creators of Atom and Tree-sitter.zed.dev

點選 Download now。

點選 macOS 的 Download now。

下載後雙擊 dmg 檔安裝。

將 Zed 移到 Applications 資料夾。

啟動 Zed

安裝程式語言的 extension

Zed IDE 支援多種程式語言，以下彼得潘安裝 swift 的 extension，如此才會有 swift 的 syntax highlight 和自動完成。

打開程式專案

從 Command Palette(cmd + shift + p) 點選 workspace: open 或點選 Menu 的 File > Open。

以下為 SwiftUI 專案開啟的畫面。

了解 Zed IDE 的基本操作後，接下來我們將重點放在好玩的 AI 上。

Assistant
The Assistant is a powerful tool that integrates large language models into your development workflow. This section…zed.dev

登入帳號

為了免費使用 Zed AI 提供的 Claude 3.7 Sonnet，我們必須先登入帳號。

點選視窗右上方的 Sign in。

使用 GitHub 帳號登入。

打開 Assistant Panel

登入帳號後，我們可在 Assistant Panel 跟 AI 聊天。打開 Assistant Panel 的方法有三種。

  • 快速鍵 cmd + r。
  • 點選右下的 Assistant Panel。
  • 從 command palette 點選 toggle right dock。

選擇 AI model

從 Assistant Panel 下方點選 model 名字可切換 AI model，在此我們選擇可免費使用的 Claude 3.7 Sonnet Zed。

從 Assistant Panel 聊天

現在我們可以開始跟 AI 聊天了。在 You 的下方輸入 prompt，然後點選右下的 Send 發送(快速鍵 cmd + enter)。

第一次發送前要先同意 Terms of Service(服務條款)。

AI 用 SwiftUI 寫出猜數字 App。

複製 AI 生成的程式

AI 生成的程式會放在 ``` 區塊裡，點擊 ``` 區塊的任一處，按 cmd + k 後放開，再按 c 即可複製整段程式。

ps: 快速鍵對應的指令是 assistant: copy code。

選取程式碼問問題

選取程式碼後按 cmd + shift + > 或點選 Add Context 後選擇 selection， 此時 Assistant Panel 將顯示選取的區塊，例如下圖顯示的 ContentView.swift, Lines 12 to 17。
