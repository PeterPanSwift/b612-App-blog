讓 SwiftUI List / Form row 裡的元件可以分別點選

SwiftUI 的 List / Form 很適合呈現清單頁面，然而若每筆資料的長方形區塊(row)想有不同的點擊區域，則需要額外處理。

以下我們用經典童書清單的畫面為例說明。

import Foundation

struct Book: Identifiable {
    let id = UUID()
    let name: String
    let author: String
}
struct ContentView: View {
    let classicChildrenBooks = [
        Book(name: "彼得潘", author: "詹姆斯·巴利"),
        Book(name: "小王子", author: "安東尼·德聖修伯里"),
        Book(name: "愛麗絲夢遊仙境", author: "路易斯·卡羅"),
        Book(name: "綠野仙蹤", author: "萊曼·弗蘭克·鮑姆"),
        Book(name: "木偶奇遇記", author: "卡洛·科洛迪"),
    ]

    var body: some View {
        List {
            ForEach(classicChildrenBooks) { book in
                HStack {
                    Text(book.name)
                    Spacer()
                    Button {
                        print("delete \(book.name)")
                    } label: {
                        Image(systemName: "trash")
                    }
                    Button {
                        print("share \(book.name)")
                    } label: {
                        Image(systemName: "square.and.arrow.up")
                    }
                }
            }
        }
    }
}

List & Form 中的按鈕預設會觸發整個 row 的點擊事件，因此點選 row 的區塊，或是點選 row 裡的按鈕時，都會造成整個 row 被點選並觸發 row 裡每個按鈕的 action。

比方點選彼得潘的分享按鈕時，它會觸發整個 row 的點擊，因此也觸發了刪除按鈕的點選。

為了讓 List / Form row 裡的元件可以分別點選，我們有三種解法。

  • 設定 button 的 buttonStyle。
  • 不要用 button，改以 onTapGesture 偵測點選。
  • 不要用 List / Form，改用 ScrollView。

設定 button 的 buttonStyle

在 button 上設定 buttonStyle 將讓 button 點選時只觸發 button 自己的 action，任何的 buttonStyle 都可以，因此 plain、borderless、bordered、
borderedProminent 皆可。

以剛剛的程式為例，我們可將 buttonStyle 加到 HStack，它將套用到 HStack 裡的多個 button。

var body: some View {
    List {
        ForEach(classicChildrenBooks) { book in
            HStack {
                Text(book.name)
                Spacer()
                Button {
                    print("delete \(book.name)")
                } label: {
                    Image(systemName: "trash")
                        .foregroundStyle(Color.accentColor)
                }                
                Button {
                    print("share \(book.name)")
                } label: {
                    Image(systemName: "square.and.arrow.up")
                        .foregroundStyle(Color.accentColor)
                }
            }
            .buttonStyle(.plain)
        }
    }
}

我們也可以在 row 裡的每個 button 加上 buttonStyle。

var body: some View {
    List {
        ForEach(classicChildrenBooks) { book in
            HStack {
                Text(book.name)
                Spacer()
                Button {
                    print("delete \(book.name)")
                } label: {
                    Image(systemName: "trash")
                        .foregroundStyle(Color.accentColor)
                }
                .buttonStyle(.plain)
                Button {
                    print("share \(book.name)")
                } label: {
                    Image(systemName: "square.and.arrow.up")
                        .foregroundStyle(Color.accentColor)
                }
                .buttonStyle(.plain)
            }
        }
    }
}

不要用 button，改以 onTapGesture 偵測點選

只有 button 會觸發整個 row 的點擊，因此我們也可以放棄 button，改用 onTapGesture 偵測點選。

var body: some View {
    List {
        ForEach(classicChildrenBooks) { book in
            HStack {
                Text(book.name)
                Spacer()
                Image(systemName: "trash")
                    .foregroundStyle(Color.accentColor)
                    .onTapGesture {
                        print("delete \(book.name)")
                    }
                Image(systemName: "square.and.arrow.up")
                    .foregroundStyle(Color.accentColor)
                    .onTapGesture {
                        print("share \(book.name)")
                    }
            }
        }
    }
}

不要用 List / Form，改用 ScrollView

List & Form 中的按鈕預設會觸發整個 row 的點擊，因此改用 ScrollView 也可以避免問題。

var body: some View {
    ScrollView {
        ForEach(classicChildrenBooks) { book in
            HStack {
                Text(book.name)
                Spacer()
                Button {
                    print("delete \(book.name)")
                } label: {
                    Image(systemName: "trash")
                        .foregroundStyle(Color.accentColor)
                }                
                Button {
                    print("share \(book.name)")
                } label: {
                    Image(systemName: "square.and.arrow.up")
                        .foregroundStyle(Color.accentColor)
                }
            }
        }
    }
}
