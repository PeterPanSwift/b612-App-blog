設定 Theme Color 調整 Xcode 的顏色

Xcode App 介面預設是藍色，如果不喜歡藍色，我們也可以改變它的顏色。

透過設定 Mac System Settings 的 Theme Color，我們可以設定 Mac 系統的預設顏色，它將影響 Mac 上安裝的每一個 App，因此 Xcode 的顏色也會變成 Theme Color。

點選左上的 Apple Logo，點選 System Settings

點選 Appearance

選擇 Theme 下的 Color

在此彼得潘選擇活潑的橘色。

Cool，現在 Xcode 變成橘色了。使用跟別人不同顏色的 Xcode，感覺開發 App 的功力也變得更厲害了。
