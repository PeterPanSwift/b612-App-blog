讀取 outlet 變數的屬性

透過連結 storyboard 的元件到程式碼產生 outlet，我們可以很方便地從程式碼存取另一個世界，鬼的世界，哦，不是啦，是存取 storyboard 畫面上的元件內容。

然而對初學者來說，有一點他們在讀取 outlet 變數內容時容易犯的錯誤，忘了要讀取 outlet 的屬性。讓我們看看以下例子，畫面上星座的 text field 連結到 outlet starTextField :

我們想要判斷使用者輸入的星座是不是世界上最棒的水瓶座，於是很直覺的用 starTextField == “水瓶座” 來比較，然而這樣子卻是錯的 !

@IBOutlet weak var starTextField: UITextField!
@IBAction func buttonPressed(_ sender: Any) {
    if starTextField == "水瓶座" {
        print("真愛")
    } else {
        print("假愛")
    }
}

要記得，starTextField 代表 text field，所以上圖的錯誤訊息告訴我們，UITextField 不能和 String 比較。

我們真正要的應該是 text field 裡的文字。我們必須研究它的型別 ，找出讓我們得到文字的屬性或方法。就好像彼得潘代表一個人，但是你只看人看不出什麼，只知道他很帥，你必須再另外詢問，才知道他的身高，體重，薪水等。

text field 也跟人一樣，它有很多屬性，比方 font 是字體，textColor 是文字顏色，在這裡我們需要的是 String 型別的 text 屬性，它才代表 text field 的文字內容。

因此，剛剛的例子，只要改用 starTextField.text 來跟 “水瓶座” PK 比較就 ok 了 !

@IBAction func buttonPressed(_ sender: Any) {
    if starTextField.text == "水瓶座" {
        print("真愛")
    } else {
        print("假愛")
    }
}
