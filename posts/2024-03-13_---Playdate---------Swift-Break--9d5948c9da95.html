看了文章的介紹，彼得潘也很想買個 Playdate，不過沒關係，Playdate 也有方便開發者測試的 SDK 和模擬器，以下彼得潘依照 Apple 範例的教學，將範例的打磚塊遊戲(Swift Break)執行在 Playdate 模擬器玩玩。

下載和安裝 Playdate SDK

Develop for Playdate

下載和安裝 Swift nightly toolchain

Swift.org
Swift is a general-purpose programming language built using a modern approach to safety, performance, and software…www.swift.org

點選下圖 Xcode 的 Universal。

取得 Swift toolchain 的名字

等下編譯程式時要設定 Swift toolchain，因此我們必須先取得 Swift toolchain 的名字。

在 Terminal 輸入以下指令取得 Swift toolchain 的名字，路徑格式為 /Library/Developer/Toolchains/swift-xxx/Info.plist，swift-xxx 請換成剛剛裝好 Swift toolchain 後在 Toolchains 下生成的資料夾。

plutil -extract CFBundleIdentifier raw -o - /Library/Developer/Toolchains/swift-DEVELOPMENT-SNAPSHOT-2024-03-07-a.xctoolchain/Info.plist

彼得潘取得的名字如下。

org.swift.59202403071a

下載 Swift Playdate Examples

GitHub - apple/swift-playdate-examples: A technical demonstration of Embedded Swift running on…
A technical demonstration of Embedded Swift running on Playdate by Panic - apple/swift-playdate-examplesgithub.com

在 Terminal 輸入以下指令下載 Swift Playdate Examples。

git clone https://github.com/apple/swift-playdate-examples.git

在 swift.mk 加入 Swift toolchain 的名字

為了編譯程式，請先編輯 swift-playdate-examples/Examples 下的 swift.mk，加入 Swift toolchain 的名字。

如下圖所示，在 ifeq ($(TOOLCHAINS),) 前加入 TOOLCHAINS := "org.swift.59202403071a"，org.swift.59202403071a 是前面取得的 toolchain 名字。

使用 Xcode 編譯打磚塊遊戲

請使用 Xcode 15.3 以上的版本，雙擊 swift-playdate-examples/Examples/SwiftBreak 下的 Package.swift。

設定 Xcode 使用從 Swift 官網下載的 toolchain。

點選三角形編譯程式，出現 Operation not permitted 的權限問題。

打開 sh 的 Full Disk Access 權限

為了解決權限問題，我們必須打開 sh 的 Full Disk Access 權限，步驟如下。

打開 System Settings，進入 Privacy & Security 的 Full Disk Access 頁面。

點選 +。

進入硬碟的資料夾。

按 cmd + shift + . 顯示隱藏檔，雙擊 bin。

點選 sh。

成功編譯程式和執行 App

打開 sh 的 Full Disk Access 權限後，我們終於可以成功點選三角形編譯程式了。

Xcode 打開 Playdate Simulator 執行遊戲前會先出現一連串的權限視窗，請先一一點選 Allow 同意。

Playdate Simulator 的相關視窗出現了
