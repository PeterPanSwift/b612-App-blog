如何將 Xcode 專案資料夾壓縮成 zip

常常有學生問彼得潘問題，需要將專案程式碼傳給彼得潘參考。除了分享 Github 連結，另一個較簡單的方法是將專案資料夾壓縮成 zip 後傳給彼得潘，步驟如下。

找到專案的資料夾。

從 Xcode 的 Project navigator 找到專案的藍色 icon，在上面按右鍵顯示選單，然後點選 Show in Finder。

此時 Mac 的 Finder 視窗將顯示專案檔。

將 Finder 的顯示改成直欄

點選下圖紅色框框的按鈕。

選擇直欄。

此時 Finder 將同時顯示專案檔上一層的清單。如下圖所示，Demo.xcodeproj 上一層的 Demo 為專案的資料夾，它將是我們準備壓縮成 zip 的對象。

將專案的資料夾壓縮成 zip

點選專案的資料夾，從右鍵選單點選壓縮，壓縮產生 zip 檔。
