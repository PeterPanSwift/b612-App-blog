SwiftUI navigation bar 的背景顏色

當 SwiftUI App 顯示 navigation bar 時，它預設是透明的，因此當以下程式顯示全螢幕的背景圖片時，它並不會被 navigation bar 遮到。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(.peter)
                .resizable()
                .scaledToFill()
                .ignoresSafeArea()
                .navigationTitle("冰島 MacBook Boy")
        }
    }
}

有時我們會誤以為 navigation bar 是白色的，其實那是因為畫面的背景色是白色，navigation bar 本身還是透明的。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(.peter)
                .resizable()
                .scaledToFit()
                .navigationTitle("冰島 MacBook Boy")
        }
    }
}

當 App 為 Dark mode 時，畫面的背景色變成了黑色，navigation bar 本身還是透明的。

若想要讓 navigation bar 有顏色，則須透過 toolbarBackgroundVisibility(_:for:) & toolbarBackground(_:for:)。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(.peter)
                .resizable()
                .scaledToFill()
                .ignoresSafeArea()
                .navigationTitle("冰島 MacBook Boy")
                .toolbarBackgroundVisibility(.visible, for: .navigationBar)
                .toolbarBackground(.green, for: .navigationBar)
        }
    }
}

toolbarBackgroundVisibility(.visible, for: .navigationBar) 讓 navigation bar 顯示，toolbarBackground(.green, for: .navigationBar) 讓 navigation bar 變成綠色。不過 navigation bar 天生帶點透明，因此下圖仍能看到綠色 bar 底下的圖片。

當畫面沒有特別設定背景色或背景圖時，將可更清楚看到 navigation bar 本身的顏色。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(.peter)
                .resizable()
                .scaledToFit()
                .navigationTitle("冰島 MacBook Boy")
                .toolbarBackgroundVisibility(.visible, for: .navigationBar)
                .toolbarBackground(.green, for: .navigationBar)
        }
    }
}
