設定 App 文字兩端對齊的 justify — UIKit、Flutter & Jetpack Compose

App 顯示多行文字時，通常預設會類似下圖的排版，邊界會參差不齊(尤其是右邊)。

若是希望顯示多行文字時能兩邊對齊，可設定文字對齊的方式為 justify。如下圖所示，為了兩邊對齊，單字間的空白間距也會跟著調整，有些間距較大，有些間距較小。

以下我們分別介紹 UIKit、Flutter 和 Jetpack Compose 實現的方法。

UIKit

  • 從 storyboard 設定。

點選 label 元件紅色框框的 justify 按鈕。

  • 從程式設定 label 的 textAlignment 為 justified。
let label = UILabel(frame: CGRect(x: 10, y: 50, width: 300, height: 100))
label.text = "And now here is my secret, a very simple secret: It is only with the heart that one can see rightly; what is essential is invisible to the eye."
label.numberOfLines = 0 // 支援多行文字
label.textAlignment = .justified

Flutter

設定 Text 的 textAlign 為 TextAlign.justify。

Text(
    'And now here is my secret, a very simple secret: It is only with the heart that one can see rightly; what is essential is invisible to the eye.',
    style: TextStyle(fontSize: 30),
    textAlign: TextAlign.justify,
)

JetPact Compose

設定 Text 的 textAlign 為 TextAlign.Justify。

Text(
    text = "And now here is my secret, a very simple secret: It is only with the heart that one can see rightly; what is essential is invisible to the eye.",
    fontSize = 30.sp,
    textAlign = TextAlign.Justify,
)

為什麼叫 justify

Claude 的說明。

"Justify" 這個詞在文字排版中表示「兩端對齊」的原因主要與這個詞的本意和歷史有關：

1. 詞源解釋：
"Justify" 源自拉丁文 "justificare"，意為「使正確」或「使整齊」。其中 "justus" 意為「正確的、公正的」，"facere" 則是「使成為」的意思。

2. 印刷歷史：
- 在早期活字印刷時代，印刷工人需要手動調整每行文字間的間距，使文字能夠精確地對齊左右邊界
- 這個過程被稱為 "justification"，因為它使文字變得「正確」且「整齊」

3. 視覺效果：
- Justify 排版讓文字左右兩側形成整齊的直線
- 這種對齊方式給人一種正式、整潔和專業的感覺，就像文字被「校正」(justified)過一樣

4. 專業用語：
在印刷和排版領域，"justify" 這個詞已經成為了專業術語，表示文字的兩端對齊處理。

所以，"justify" 用來表示文字左右對齊，是因為這個詞本身就有「使整齊、使正確」的含義，非常符合這種排版方式的特點和目的。
