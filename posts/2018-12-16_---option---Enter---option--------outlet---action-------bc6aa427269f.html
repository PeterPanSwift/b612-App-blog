利用 option + Enter 或 option + 點擊顯示 outlet / action 連線的檔案

拉 outlet / action 時，我們習慣先打開 Assistant Editor，然後再讓 Assistant Editor 的 Automatic 自動幫我們找到相關的檔案。

連結 IBAction 和 IBOutlet 的步驟說明
以 Yaya 同學的小費 App 為例medium.com

不過條條大路通羅馬，接下來我們將認識另一種拉 outlet / action 的方法，以拉 table view cell 裡元件的 outlet 為例。

  • 打開 storyboard，找到要拉 outlet 的 table view cell 畫面。
  • 利用 option + Enter 或 option + 點擊，讓 Assistant Editor 顯示 outlet / action 連線的檔案。

方法1: 打開 Open Quickly 視窗(快速鍵 cmd + shift + o)，輸入 cell 類別的關鍵字 songc 後，找到 cell 的類別檔 SongTableViewCell.swift，然後按下 option + Enter。

方法2: 按住 option 點擊 Project navigator 下的 SongTableViewCell.swift。

Assistant Editor 成功顯示 SongTableViewCell.swift，現在我們可以順利地拉 cell 裡元件的 outlet。
