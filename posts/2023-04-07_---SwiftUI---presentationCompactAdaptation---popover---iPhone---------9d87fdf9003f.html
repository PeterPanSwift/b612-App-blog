利用 SwiftUI 的 presentationCompactAdaptation & popover 在 iPhone 上顯示彈出視窗

從 iOS 16.4 開始，SwiftUI App 在 iPhone 上可以利用 presentationCompactAdaptation & popover 顯示 iPad 上常見的彈出視窗。

以下我們以可愛狐狸為例，點選按鈕後會顯示 FoxView，看到認真寫程式的可愛狐狸。

struct ContentView: View {
    @State private var show = false
    
    var body: some View {
        Button("show") {
            show = true
        }
        .font(.largeTitle)
        .popover(isPresented: $show) {
            FoxView()
        }
    }
}

struct FoxView: View {
    var body: some View {
        VStack {
            Image("fox")
                .resizable()
                .scaledToFit()
                .frame(minWidth: 0, maxWidth: .infinity)
            Text("認真寫程式的可愛狐狸")
                .foregroundColor(.white)
        }
        .padding()
        .background(.black.gradient)
    }
}

我們呼叫 popover 顯示 FoxView，因此狐狸畫面會以彈出視窗的效果呈現。

在 iOS 16.4 之前，popover 只有在 iPad 會以彈出視窗顯示，在 iPhone 還是會以 sheet 或 full screen 的方式顯示。

從 16.4 開始，我們可以在顯示新頁面時呼叫 presentationCompactAdaptation，傳入 .none 要求它在 compact size 時維持以彈出視窗的方式呈現，不要做特別的調整。相較於 iPad，iPhone 在直向跟橫向時都會至少有一邊是 compact size，因此 presentationCompactAdaptation(.none) 將讓 iPhone 以彈出視窗的方式顯示新頁面。

struct ContentView: View {
    @State private var show = false
    
    var body: some View {
        Button("show") {
            show = true
        }
        .font(.largeTitle)
        .popover(isPresented: $show) {
            FoxView()
                .presentationCompactAdaptation(.none)
        }
    }
}

我們也可以調整 popover 的大小，比方設定它的寬高為 300 * 200。

struct ContentView: View {
    @State private var show = false
    
    var body: some View {
        Button("show") {
            show = true
        }
        .font(.largeTitle)
        .popover(isPresented: $show) {
            FoxView()
                .frame(width: 300, height: 200)
                .presentationCompactAdaptation(.none)

        }
    }
}

除了 presentationCompactAdaptation(_:)，若想對水平和垂直的 compact size 做不同處理，則可參考 presentationCompactAdaptation(horizontal:vertical:)。
