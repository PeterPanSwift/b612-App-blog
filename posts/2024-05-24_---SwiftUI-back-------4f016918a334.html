設定 SwiftUI back 按鈕的文字

當我們利用 NavigationStack & NavigationLink 切換頁面時，左上方的 back 按鈕預設會依據以下規則顯示文字。

  • 前一頁有標題時，back 按鈕的文字為前一頁的標題。
import SwiftUI

struct ContentView: View {
    var body: some View {
        NavigationStack {
            NavigationLink("小王子和猩猩的故事") {
                StoryView()
            }
            .font(.largeTitle)
            .navigationTitle("小王子")
        }
    }
}
import SwiftUI

struct StoryView: View {
    var body: some View {
        VStack {
            Image(.小王子和猩猩)
                .resizable()
                .scaledToFit()
            Text("""
一踏上這個星球，小王子就聽到了樹上傳來的吱吱聲和喧鬧聲。他抬頭一看，發現樹上有一隻大猩猩，正盯著他看。這隻大猩猩有著濃密的黑色毛髮和溫柔的眼神，它沒有任何敵意，反而對這個來自遠方的小訪客充滿了好奇。

「你好啊，我是小王子，來自B612星球。」小王子微笑著向大猩猩打招呼。

「你好，我是卡洛，我是這顆星球的守護者。你來這裡有什麼事嗎？」卡洛友善地回應道。
""")
        }
    }
}
  • 前一頁無標題時，back 按鈕的文字為 Back。

若想修改 back 按鈕的文字，目前 SwiftUI 尚未提供對應的 modifier，因此必須透過比較麻煩的兩個步驟實現。

  • 隱藏原本的 back 按鈕。
  • 在 navigation bar 的左上方加入自訂的 back 按鈕。

範例。

import SwiftUI

struct StoryView: View {
    @Environment(\.dismiss) var dismiss
    
    var body: some View {
        VStack {
            Image(.小王子和猩猩)
                .resizable()
                .scaledToFit()
            Text("""
一踏上這個星球，小王子就聽到了樹上傳來的吱吱聲和喧鬧聲。他抬頭一看，發現樹上有一隻大猩猩，正盯著他看。這隻大猩猩有著濃密的黑色毛髮和溫柔的眼神，它沒有任何敵意，反而對這個來自遠方的小訪客充滿了好奇。

「你好啊，我是小王子，來自B612星球。」小王子微笑著向大猩猩打招呼。

「你好，我是卡洛，我是這顆星球的守護者。你來這裡有什麼事嗎？」卡洛友善地回應道。
""")
        }
        .navigationBarBackButtonHidden()
        .toolbar(content: {
            ToolbarItem(placement: .topBarLeading) {
                Button(action: {
                    dismiss()
                }, label: {
                    HStack(spacing: 5) {
                        Image(systemName: "chevron.backward")
                            .fontWeight(.semibold)
                        Text("回B612星球")
                    }
                })
            }
        })
    }
}

#Preview {

    }
}

說明。

  • 隱藏原本的 back 按鈕。
.navigationBarBackButtonHidden()
  • 在 navigation bar 的左上方加入自訂的 back 按鈕。

button 設計的方法有很多種，以下彼得潘用 HStack 搭配 spacing 5 和 SF Symbol chevron.backward 實現，因為它呈現的樣子最接近內建的 back 按鈕。

.toolbar(content: {
    ToolbarItem(placement: .topBarLeading) {
        Button(action: {
            dismiss()
        }, label: {
            HStack(spacing: 5) {
                Image(systemName: "chevron.backward")
                    .fontWeight(.semibold)
                Text("回B612星球")
            }
        })
    }
})
