從程式控制 UIKit UI 元件的加入，移除，隱藏，階層調整 & 讀取相關聯的元件

使用 UIKit 開發 iOS App 時，畫面上看到的元件主要有 UIView & CALayer 兩種，接下來我們將說明如何從程式控制這些 UI 元件的加入，移除，隱藏，階層調整 & 讀取相關聯的元件。

加入元件

UIView 的 addSubview

@IBAction func buttonTap(_ sender: Any) {
    let orangeView = UIView(frame: CGRect(x: 100, y: 100, width: 100, height: 100))
    orangeView.backgroundColor = .orange
    view.addSubview(orangeView)
}

其它繼承 UIView 的元件，比方 UILabel & UIImageView 也是一樣的方法。

利用 UIView 的 addSubview 將 view 加到另一個 view 上
從程式製作 App 畫面時，我們時常使用 addSubview。不過接下來先讓我們以比較直覺易懂的 storyboard 說明，然後再來研究程式的做法。medium.com

UIView 的 insertSubview(_:at:)，insertSubview(_:aboveSubview:) & insertSubview(_:belowSubview:)

指定新加入的 view 要在哪一層，以下例子指定 blueView 在 orangeView 下面。

@IBAction func buttonTap(_ sender: Any) {
     let orangeView = UIView(frame: CGRect(x: 100, y: 100, width: 100, height: 100))
     orangeView.backgroundColor = .orange
     view.addSubview(orangeView)
        
     let blueView = UIView(frame: CGRect(x: 130, y: 130, width: 100, height: 100))
     blueView.backgroundColor = .blue
     view.insertSubview(blueView, belowSubview: orangeView)
}

CALayer 的 addSublayer

@IBAction func buttonTap(_ sender: Any) {
    let redLayer = CALayer()
    redLayer.frame = CGRect(x: 100, y: 100, width: 100, height: 100)
    redLayer.backgroundColor = CGColor(red: 1, green: 0, blue: 0, alpha: 1)
    view.layer.addSublayer(redLayer)
}

其它繼承 CALayer 的元件，比方 CAShapeLayer & CAGradientLayer 也是一樣的方法。

CALayer 的 insertSublayer(_:at:)，insertSublayer(_:above:) & insertSublayer(_:below:)

指定新加入的 layer 要在哪一層，以下例子指定 blueLayer 在 redLayer 下面。

@IBAction func buttonTap(_ sender: Any) {
    let redLayer = CALayer()
    redLayer.frame = CGRect(x: 100, y: 100, width: 100, height: 100)
    redLayer.backgroundColor = CGColor(red: 1, green: 0, blue: 0, alpha: 1)
    view.layer.addSublayer(redLayer)
        
    let blueLayer = CALayer()
    blueLayer.frame = CGRect(x: 130, y: 130, width: 100, height: 100)
    blueLayer.backgroundColor = CGColor(red: 0, green: 0, blue: 1, alpha: 1)
    view.layer.insertSublayer(blueLayer, below: redLayer)
}

隱藏

UIView & CALayer 的 property isHidden

isHidden 為 true 時元件將隱藏。

class ViewController: UIViewController {
   @IBOutlet weak var imageView: UIImageView!

   @IBAction func buttonTap(_ sender: Any) {
        
        imageView.isHidden = true
    }
    
}

將內容真變假，假變真的 toggle function

呼叫 Bool 型別的 function toggle，它將讓 true 變 false，false 變 true。

class ViewController: UIViewController {
    @IBOutlet weak var imageView: UIImageView!

    @IBAction func buttonTap(_ sender: Any) {
        
        imageView.isHidden.toggle()
    }
    
}
將內容真變假，假變真的 toggle function — Swift 4.2

移除元件

UIView 的 removeFromSuperview

class ViewController: UIViewController {
    
    var squareViews = [UIView]()
        
    @IBAction func add(_ sender: Any) {
        
        let squareView = UIView(frame: CGRect(x: CGFloat.random(in: 0...200), y: CGFloat.random(in: 0...200), width: 50, height: 50))
        squareView.backgroundColor = UIColor(red: .random(in: 0...1), green: .random(in: 0...1), blue: .random(in: 0...1), alpha: 1)
        view.addSubview(squareView)
        squareViews.append(squareView)
    }
    
    @IBAction func remove(_ sender: Any) {
        if let squareView = squareViews.first {
            squareView.removeFromSuperview()
            squareViews.removeFirst()
        }
    }
    
}

CALayer 的 removeFromSuperlayer

class ViewController: UIViewController {
    
    var squareLayers = [CALayer]()
        
    @IBAction func add(_ sender: Any) {
        
        let squareLayer = CALayer()
        squareLayer.frame = CGRect(x: .random(in: 0...200), y: .random(in: 0...200), width: 50, height: 50)
        squareLayer.backgroundColor = CGColor(red: .random(in: 0...1), green: .random(in: 0...1), blue: .random(in: 0...1), alpha: 1)
        view.layer.addSublayer(squareLayer)
        squareLayers.append(squareLayer)
    }
    
    @IBAction func remove(_ sender: Any) {
        if let squareLayer = squareLayers.first {
            squareLayer.removeFromSuperlayer()
            squareLayers.removeFirst()
        }
    }
    
}

階層調整

UIView 的 bringSubviewToFront & sendSubviewToBack

以下程式呼叫 bringSubviewToFront，將 orangeView 變到最上層。

class ViewController: UIViewController {
    
    
    @IBAction func buttonTap(_ sender: Any) {
        
        let orangeView = UIView(frame: CGRect(x: 100, y: 100, width: 100, height: 100))
        orangeView.backgroundColor = .orange
        view.addSubview(orangeView)
        
        let blueView = UIView(frame: CGRect(x: 130, y: 130, width: 100, height: 100))
        blueView.backgroundColor = .blue
        view.addSubview(blueView)
        
        view.bringSubviewToFront(orangeView)
    }
    
}

讀取相關聯的元件

UIView 透過 superview ， subviews & viewWithTag，CALayer 透過 superlayer & sublayers。
