自動完成失靈，還有 Snippets library 可以輸入

Xcode 的自動完成很方便，不過偏偏有些情況會失靈。比方以下程式我們想在 scheduledTimer 的 closure 裡輸入 for，但是 for 的 code snippet 卻不會出現。

Timer.scheduledTimer(withTimeInterval: 1, repeats: true) { (_) in
     for
}

簡單的 for 當然難不倒我們，不過如果懶得打字，我們也可以點選右上方的 + 叫出 Library 視窗。(快速鍵 cmd + shift + l )

然後切換到 Snippets library 視窗，找到 for 的 code snippet，雙擊完成 for in 的輸入。
