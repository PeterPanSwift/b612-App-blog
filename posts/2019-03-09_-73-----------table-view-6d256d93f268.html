#73 包含左右滑動區塊的 table view

為了顯示更多的內容，許多 iOS App 會設計上下捲動的頁面，而且在頁面裡的某些區塊還可以左右滑動，就像以下例子裡，我們可以水平滑動瀏覽 Marvel 的漫畫，Medium 的 Topic，以及 Music 的專輯。

這次的謎題，就讓我們實作出包含左右滑動區塊的 table view 吧。

1 無程式版 (適合還不會寫程式的初學者)

使用 static cells 的 table view controller，在其中的某些 cell 加入可水平捲動的 scroll view。

設定 content size，實現水平捲動，上下捲動和分頁的 scroll view
對剛學會從 storyboard 製作 App 畫面，但還不會 Auto Layout 和寫程式的朋友，利用 table view controller 的 Static Cells…medium.com

ps: 另一種做法是在上下捲動的 scroll view 裡加入水平捲動的 scroll view。

2 程式版

版本1: 使用 table view 呈現頁面，在 table view cell 裡加入 collection view。

版本2: 使用 table view 呈現頁面，在 table view cell 裡加入 scroll view。

版本3: 使用 collection view 呈現頁面，在 collection view cell 裡加入 collection view。
