SwiftUI 文字 Button 的兩種寫法跟差異

當我們使用 SwiftUI Button 顯示文字時，有兩種常見的做法，一種是利用 init(_:action:)，一種是利用 init(action:label:)。雖然他們產生的 button 長得一模一樣，但是在點擊時卻會有一些不同的地方。

以下我們用一個綠底藍字的 button 為例說明。

  • init(_:action:)
struct ContentView: View {
    var body: some View {
        Button("button") {
            print("button tap")
        }
        .frame(width: 200, height: 50)
        .background(.green)
    }
}
  • init(action:label:)
struct ContentView: View {
    var body: some View {
        Button {
            print("button tap")
        } label: {
            Text("button")
                .frame(width: 200, height: 50)
                .background(.green)
        }
    }
}

使用 init(_:action:) 產生 button 時，點擊時只有文字會變色，背景顏色不會變色，而且必須點擊到文字才能偵測到點擊的動作，點擊文字外的背景區塊是沒反應的。

使用 init(action:label:) 產生 button 時，點擊時文字和背景都會變色，點擊到整個背景區塊都能偵測到點擊的動作。

另外若是用 buttonStyle 實現有背景色的 button，使用 init(_:action:) 產生的 button 即使用 frame 改了大小，button 的背景色範圍和點擊範圍也不會受影響。

struct ContentView: View {
    var body: some View {
        Button("button") {
            print("button tap")
        }
        .frame(width: 200, height: 50)
        .buttonStyle(.borderedProminent)
    }
}

使用 init(action:label:) 產生的 button 搭配 buttonStyle 時，加在 Text 的 frame 才能影響 button 的背景色範圍和點擊範圍。

struct ContentView: View {
    var body: some View {
        Button {
            print("button tap")
        } label: {
            Text("button")
                .frame(width: 200, height: 50)
        }
        .buttonStyle(.borderedProminent)
    }
}
