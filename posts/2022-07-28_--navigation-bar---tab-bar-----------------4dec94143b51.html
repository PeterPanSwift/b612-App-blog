讓 navigation bar 或 tab bar 捲動跟不捲動時都維持同一種樣式

從 iOS 26 開始 navigation bar & tab bar 將搭配 liquid glass，因此 Apple 建議盡量不要客製樣式。

TN3106: Customizing the appearance of UINavigationBar | Apple Developer Documentation
Adopt UINavigationBarAppearance for a navigation bar background color that's consistent on iOS 13 through 18.developer.apple.com

以下為舊版的寫法。

iOS App 的上方跟下方時常會顯示 navigation bar & tab bar，預設的設計是捲動的內容跟 bar 重疊時 bar 的顏色為毛玻璃，沒重疊時則變成透明的 bar。比方以下 Music App 的 navigation bar & Shortcuts App 的 tab bar。

不過有時我們會希望 navigation bar 或 tab bar 維持同一種樣式，不管畫面是否捲動，不管畫面的內容是否跟 bar 重疊。

對 navigation bar & tab bar 來說，standardAppearance & scrollEdgeAppearance 控制著 bar 的樣式。捲動的內容跟 bar 重疊時，採用 standardAppearance。捲動的內容跟 bar 沒重疊時，採用 scrollEdgeAppearance。我們想要統一樣式，因此關鍵在設定 standardAppearance & scrollEdgeAppearance 使用同一種樣式。

ps: iOS 15 之後跟之前的版本有些差異，詳細的規則可參考下圖。

接下來我們將以彼得潘冒險抱石的照片為例，說明如何實現 navigation bar 或 tab bar 維持同一種樣式，分成以下三種 case:

  • 維持毛玻璃，透過 configureWithDefaultBackground 實現。
  • 維持某個顏色，透過 configureWithOpaqueBackground 實現。
  • 維持透明，透過 configureWithTransparentBackground 實現。

當我們沒做任何調整時，預設的效果如下，捲動的內容跟 bar 重疊時 bar 的顏色為毛玻璃，沒重疊時則變成透明的 bar :

維持毛玻璃

  • 方法 1: 從 Interface Builder 設定

勾選 tab bar & navigation bar 的 Standard & Scroll Edge，將 Standard Appearance & Scroll Edge Appearance 設為同一種 Blur Style & Background，在此我們將 Blur Style 設為 Default，Background 設為 Default，讓 tab bar & navigation bar 維持毛玻璃。

tab bar & navigation bar 永遠維持毛玻璃。

  • 方法 2: 從程式設定，讓 App 的 navigation bar & tab bar 永遠維持毛玻璃

在 AppDelegate 的 function application(_:didFinishLaunchingWithOptions:) 設定，毛玻璃的關鍵在呼叫 configureWithDefaultBackground。

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    // Override point for customization after application launch.
    
    let navigationBarAppearance = UINavigationBarAppearance()       navigationBarAppearance.configureWithDefaultBackground()
    UINavigationBar.appearance().standardAppearance = navigationBarAppearance
    UINavigationBar.appearance().scrollEdgeAppearance = navigationBarAppearance
    
    let tabBarAppearance = UITabBarAppearance()
    tabBarAppearance.configureWithDefaultBackground()
    UITabBar.appearance().standardAppearance = tabBarAppearance
    UITabBar.appearance().scrollEdgeAppearance = tabBarAppearance
    
    return true
}
  • 方法 3: 從程式設定，讓某個 navigation bar & tab bar 永遠維持毛玻璃

比方 App 顯示的第一個畫面為 ViewController，在它的 viewDidLoad 設定 navigation bar & tab bar 的樣式。毛玻璃的關鍵在呼叫 configureWithDefaultBackground。

class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        let navigationBarAppearance = UINavigationBarAppearance()      navigationBarAppearance.configureWithDefaultBackground()
        navigationController?.navigationBar.standardAppearance = navigationBarAppearance       navigationController?.navigationBar.scrollEdgeAppearance = navigationBarAppearance
        
        let tabBarAppearance = UITabBarAppearance()
        tabBarAppearance.configureWithDefaultBackground()
        tabBarController?.tabBar.standardAppearance = tabBarAppearance
        tabBarController?.tabBar.scrollEdgeAppearance = tabBarAppearance
    }    
}

維持某個顏色

  • 方法 1: 從 Interface Builder 設定

勾選 tab bar & navigation bar 的 Standard & Scroll Edge，將 Standard Appearance & Scroll Edge Appearance 設為同一種 Blur Style & Background，在此我們將 Blur Style 設為 Default，Background 設為黃色，讓 tab bar & navigation bar 維持黃色。

tab bar & navigation bar 永遠維持黃色。

  • 方法 2: 從程式設定，讓 App 的 navigation bar & tab bar 永遠維持某個顏色

在 AppDelegate 的 function application(_:didFinishLaunchingWithOptions:) 設定，背景顏色的關鍵在呼叫 configureWithOpaqueBackground & 設定 backgroundColor。

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    // Override point for customization after application launch.
    
    let navigationBarAppearance = UINavigationBarAppearance()
    navigationBarAppearance.configureWithOpaqueBackground()
    navigationBarAppearance.backgroundColor = .systemYellow
    UINavigationBar.appearance().standardAppearance = navigationBarAppearance
    UINavigationBar.appearance().scrollEdgeAppearance = navigationBarAppearance
    
    let tabBarAppearance = UITabBarAppearance()
    tabBarAppearance.configureWithOpaqueBackground()
    tabBarAppearance.backgroundColor = .systemYellow
    UITabBar.appearance().standardAppearance = tabBarAppearance
    UITabBar.appearance().scrollEdgeAppearance = tabBarAppearance
    return true
}
  • 方法 3: 從程式設定，讓某個 navigation bar & tab bar 永遠維持某個顏色

比方 App 顯示的第一個畫面為 ViewController，在它的 viewDidLoad 設定 navigation bar & tab bar 的樣式。背景顏色的關鍵在呼叫 configureWithOpaqueBackground & 設定 backgroundColor。

override func viewDidLoad() {
    super.viewDidLoad()
    // Do any additional setup after loading the view.
    
    let navigationBarAppearance = UINavigationBarAppearance()
    navigationBarAppearance.configureWithOpaqueBackground()
    navigationBarAppearance.backgroundColor = .systemYellow
    navigationController?.navigationBar.standardAppearance = navigationBarAppearance
    navigationController?.navigationBar.scrollEdgeAppearance = navigationBarAppearance
    
    let tabBarAppearance = UITabBarAppearance()
    tabBarAppearance.configureWithOpaqueBackground()
    tabBarAppearance.backgroundColor = .systemYellow
    tabBarController?.tabBar.standardAppearance = tabBarAppearance
    tabBarController?.tabBar.scrollEdgeAppearance = tabBarAppearance
}

維持透明

  • App 的 navigation bar & tab bar 永遠維持透明。

在 AppDelegate 的 function application(_:didFinishLaunchingWithOptions:) 設定，透明的關鍵在呼叫 configureWithTransparentBackground。

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    // Override point for customization after application launch.
    
    let navigationBarAppearance = UINavigationBarAppearance()
    navigationBarAppearance.configureWithTransparentBackground()
    UINavigationBar.appearance().standardAppearance = navigationBarAppearance
    UINavigationBar.appearance().scrollEdgeAppearance = navigationBarAppearance
    
    let tabBarAppearance = UITabBarAppearance()
    tabBarAppearance.configureWithTransparentBackground()
    UITabBar.appearance().standardAppearance = tabBarAppearance
    UITabBar.appearance().scrollEdgeAppearance = tabBarAppearance
    
    return true
}
  • 設定某個 navigation bar & tab bar 永遠維持透明。

比方 App 顯示的第一個畫面為 ViewController，在它的 viewDidLoad 設定 navigation bar & tab bar 的樣式，透明的關鍵在呼叫 configureWithTransparentBackground。

class ViewController: UIViewController {
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        let navigationBarAppearance = UINavigationBarAppearance()
        navigationBarAppearance.configureWithTransparentBackground()
        navigationController?.navigationBar.standardAppearance = navigationBarAppearance
        navigationController?.navigationBar.scrollEdgeAppearance = navigationBarAppearance
        
        let tabBarAppearance = UITabBarAppearance()
        tabBarAppearance.configureWithTransparentBackground()
        tabBarController?.tabBar.standardAppearance = tabBarAppearance
        tabBarController?.tabBar.scrollEdgeAppearance = tabBarAppearance
        
    }
}
