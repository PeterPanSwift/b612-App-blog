SwiftUI 設定畫面 property 的幾種寫法

宣告 property 時指定內容

SwiftUI 設定畫面 property 的幾種寫法

  • 宣告 property 時指定內容
  • 宣告 property 時執行 closure 設定內容
  • 在 View 的 onAppear 設定內容
  • 在 View 的 init 設定內容
  • 產生 View 時傳入參數設定 property

以下我們在 App 畫面上顯示可愛的圓形數字為例說明。

假設資料的型別為 NumberData。

struct NumberData: Identifiable {
    let id = UUID()
    let value: Int
}

宣告 property 時指定內容

struct ContentView: View {
    @State private var numberData = NumberData(value: Int.random(in: 1...50))
    
    var body: some View {
        Image(systemName: "\(numberData.value).circle")
            .resizable()
            .scaledToFit()
    }
}

宣告 property 時執行 closure 設定內容

有時候 property 的內容比較複雜，必須經過一些程式運算，此時我們可以透過執行 closure 設定內容。

struct ContentView: View {
    @State private var numberDatas: [NumberData] = {
        var numberDatas = [NumberData]()
        for _ in 1...10 {
            let numberData = NumberData(value: Int.random(in: 1...10))
            numberDatas.append(numberData)
            
        }
        return numberDatas
    }()
    
    var body: some View {
        VStack {
            ForEach(numberDatas) { numberData in
                Image(systemName: "\(numberData.value).circle")
                    .resizable()
                    .scaledToFit()
            }
        }
    }
}

在 View 的 onAppear 設定內容

先將 numberDatas 初始為空的 array，然後在畫面出現時再設定 array 的內容。

struct ContentView: View {
    @State private var numberDatas = [NumberData]()
    
    var body: some View {
        VStack {
            ForEach(numberDatas) { numberData in
                Image(systemName: "\(numberData.value).circle")
                    .resizable()
                    .scaledToFit()
            }
        }
        .onAppear {
            for _ in 1...10 {
                let numberData = NumberData(value: Int.random(in: 1...10))
                numberDatas.append(numberData)
                
            }
        }
    }
}

在 View 的 init 設定內容

當我們以 ContentView() 生成畫面時，它會觸發 init，因此我們可在 init 裡設定 property 的內容。值得注意的，由於 State property 是 property wrapper，因此我們在 init 裡必須透過 _numberDatas 初始 numberDatas 的內容。

ps: property wrapper 可參考以下連結的說明。

Properties associate values with a particular class, structure, or enumeration. Stored properties store constant and…docs.swift.org
struct ContentView: View {
    @State private var numberDatas: [NumberData]
    
    init() {
        var array = [NumberData]()
        for _ in 1...10 {
            let numberData = NumberData(value: Int.random(in: 1...10))
            array.append(numberData)
            
        }
        _numberDatas = State(initialValue: array)
    }
    
    var body: some View {
        VStack {
            ForEach(numberDatas) { numberData in
                Image(systemName: "\(numberData.value).circle")
                    .resizable()
                    .scaledToFit()
            }
        }
    }
}

產生 View 時透過參數設定 property

以下程式在產生 ContentView 時傳入 numberData。不過 State property 通常會設為 private，因此比較少會採用此做法。

struct ContentView: View {
    @State var numberData: NumberData
    
    var body: some View {
        Image(systemName: "\(numberData.value).circle")
            .resizable()
            .scaledToFit()
    }
}
struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        let numberData = NumberData(value: Int.random(in: 1...10))
        return ContentView(numberData: numberData)
    }
}
