利用 Shortcuts 實現在 Finder 用 Xcode & VS Code 開啟專案資料夾

Mac 的 Finder 預設沒有用 App 開啟資料夾的功能，不過彼得潘因為開發程式的關係，常常需要用 Xcode 或 VS Code 開啟專案資料夾，因此以下彼得潘透過 Shortcuts App 實現在 Finder 用 Xcode & VS Code 開啟專案資料夾。

  • 用 Xcode 開啟專案資料夾。
  • 用 VS Code 開啟專案資料夾。

用 Xcode 開啟專案資料夾

  • 新增 Quick Actions。

啟動 Shortcuts App，點選 Quick Actions，然後點選右上的 +。

  • 設定名稱。

輸入 Open Folder from Xcode。

  • 在右邊的搜尋框輸入關鍵字加入相關動作。

加入 Get Contents of Folder 和 Filter Files。

點選 Filter 右邊的 Add Filter 按鈕，設定 File Extension is xcodeproj。

加入 Open File。

  • 設定 Finder 的 Quick Action。

我們想在 Finder 操作 Open Folder from Xcode，因此請點選右上的 I 按鈕 ，切換到 Details 分頁，勾選 Use as Quick Action 下方的 Finder。

  • 設定快速鍵。

我們也可以設定 Open Folder from Xcode 的快速鍵，從 Details 頁面點選 Add Keyboard Shortcut。

  • 用 Xcode 開啟 Finder 的專案資料夾。

現在我們可以從 Finder 用 Xcode 開啟專案資料夾了! 比方從下圖 Demo 資料夾的右鍵選單點選 Quick Actions > Open Folder from Xcode。

專案順利地從 Xcode 啟動。

  • 選擇啟動的 App

若想讓使用者選擇啟動的 App，可勾選下圖的 Show Open in Menu。

之後從 Finder 點選 Open Folder from Xcode 時會先出現下圖的 App 選擇視窗。

用 VS Code 開啟專案資料夾

設定的方法跟剛剛類似，而且更簡單，動作清單裡只需加入 Open File。

點選 Open 動作右邊的 Default App，選擇 Visual Studio Code。

現在我們可以從 Finder 用 VS Code 開啟專案資料夾了! 比方從下圖 demo 資料夾的右鍵選單點選 Quick Actions > Open Folder from VS Code。
