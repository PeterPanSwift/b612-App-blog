請身兼工程師和 UI 設計師的 Agent mode ChatGPT 開發 App

開發一個完整的 App 通常要包含程式和圖片，因此很難一句話就請 AI 生成完整的 App。

不過世上 AI 那麼多，還是有例外，Agent mode 的 ChatGPT 已具有開發完整 App 的能力。它可以完成各種任務，上網查資料、寫程式、生成圖片、下載圖片都難不倒它。

以下彼得潘請身兼工程師和 UI 設計師的 ChatGPT 開發精美的 iOS App，請它同時寫程式和生成圖片。

prompt: 請用 SwiftUI 製作小王子風格的剪刀石頭布 App，App 需要的圖片請用 AI 繪製

從 GPT 即時的預覽畫面可看到它精心繪製的圖片。

GPT 花了六分鐘完成 App，包含 swift 程式檔和 png 圖檔，壓縮成 zip 讓我們下載。

將 swift 和 png 檔加到 iOS App 的 Xcode 專案。
