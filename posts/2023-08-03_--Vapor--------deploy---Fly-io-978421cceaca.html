連到 Fly.io 註冊帳號

Deploy app servers close to your users · Fly
Install flyctl on GNU/Linux $ curl -L https://fly.io/install.sh | sh > Ship a Docker image $ fly launch > Run it on…fly.io

點選右上的 Sign Up。

在 Mac 上安裝 flyctl

打開 Terminal，輸入以下指令。

brew install flyctl

ps: 記得先安裝 Homebrew 才能用 brew 安裝 flyctl。

Homebrew
The Missing Package Manager for macOS (or Linux).brew.sh

登入 flyctl

打開 Terminal，輸入以下指令。

flyctl auth login

此時會自動打開網頁，請點選下圖紅色框框的 Add credit card 設定信用卡。

不用太擔心，Fly.io 提供一些免費的額度，使用的量超過額度才要付錢。

設定好信用卡後，再點選 Continue as xxx 的按鈕。

現在再回到 Terminal，成功出現 successfully logged in 的訊息。

將 Vapor 專案部署到 Fly.io 和啟動 server

打開 Terminal，切換到 Vapor 專案資料夾下，輸入以下指令。

fly launch

app name & region 可先採用預設的設定，直接按 enter。

我們先不設定 database，因此 Postgresql database 和 Upstash Redis database 都輸入 N。

Would you like to deploy now 輸入 y，將 Vapor 專案部署到 Fly.io。

稍待片刻後，成功出現下圖的訊息，現在世界上任何人的電腦和手機都可從網址 https://xxx.fly.dev 連到我們的 server 了 ! (ps: 為了怕流量爆掉，https://xxx.fly.dev 是假的網址)

現在任何人都可自由存取我們 Vapor 專案的 API。比方彼得潘在剛剛的專案裡寫了一個隨機名字 API，路徑是 randname，因此從 https://xxx.fly.dev/randname 可得到隨機的名字。
