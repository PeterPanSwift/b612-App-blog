請 AI 製作程式詞彙表(glossary)，用一句話說明程式概念

學程式時利用詞彙表(glossary)復習加深印象是個不錯的方法，當自己看到程式的關鍵字能用一句話說明，不只能代表自己已經熟練理解，而且面試回答時也能有不錯的表現。

Apple 和 Paul 的 Glossary of Common Swift Terms 都整理了不錯的 Swift & App 開發詞彙表。

不過詞彙表當然是愈多愈好，看到各家詞彙表對同一個關鍵字的不同說明，也能幫助我們更加熟練理解。最近彼得潘想到請教 AI 也是不錯的方法，AI 精通世界上所有的知識，任何程式相關的關鍵字或技術，我們都可以請它用一句話說明。

以下彼得潘以 Swift 為例，請 AI 用一句話說明 string interpolation、optional 和 optional binding。

Swift 的 string interpolation

  • 用一句英文說明 swift 的 string interpolation

string interpolation allows you to construct a new String by including existing variables, constants, literals, or expressions within a string literal, making it easy to create dynamic and customized text.

  • 用一句可愛的英文說明 swift 的 string interpolation

string interpolation is like a friendly chat with your code, where you can mix your words (strings) with little bits of magic (variables or expressions) to create a whole new delightful message!

  • 用一句中文說明 swift 的 string interpolation

Swift 的字符串插值就像是在句子中穿插小秘密，讓你能夠輕鬆地將變數或表達式嵌入到文字中，創造出充滿個性的信息。

  • 用一句可愛的中文說明 swift 的 string interpolation

字串插值就像是在一段故事裡，輕鬆地插入自己的角色一樣，讓整個故事更加生動有趣。

Swift 的 optional

  • 用一句英文說明 swift 的 optional

an optional is a type that can hold either a value or ‘nil’ to indicate the absence of a value.

  • 用一句可愛的英文說明 swift 的 optional

optional is like a wrapped gift box — it might hold a delightful surprise inside or it could be empty, but you’ll never know until you unwrap it!

  • 用一句中文說明 swift 的 optional

可選型別（optional）就像是一個問號盒子，裡面可能有值，也可能是空的，你只有在打開它的時候才會知道。

  • 用一句可愛的中文說明 swift 的 optional

可選型別（optional）就像一顆神秘的糖果，你不嘗試就不會知道它是甜的還是空的！

Swift 的 optional binding

  • 用一句英文說明 swift 的 optional binding

optional binding is a technique used to safely unwrap an optional, checking if it contains a value and, if so, making that value available as a temporary constant or variable.

  • 用一句可愛的英文說明 swift 的 optional binding

Optional binding in Swift is like fishing in a pond of surprises — you cast your line hoping to catch a fish (a value), and if you’re lucky, you get to keep it for a while!

  • 用一句中文說明 swift 的 optional binding

optional binding 是一種安全地處理可能為 nil 的選擇值的方式，它允許我們條件性地解包 optional，並在 optional 有值時執行一段代碼。

  • 用一句可愛的中文說明 swift 的 optional binding

Swift 的 optional binding 就像是打開一個裝滿驚喜的禮物盒，如果裡面有禮物（值存在），我們就開心地使用它；如果是空的，我們就安全地略過，不會有任何麻煩哦！
