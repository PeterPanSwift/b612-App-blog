指定 App Playground(.swiftpm) 的 App 方向

開發 iOS App 時，App 預設可以執行直向或橫向，不過我們也可以指定它只執行某個方向。

Xcode 建立的 iOS 專案可以輕鬆地從 General 分頁設定方向，然而從 Xcode 或 Swift Playgrounds 建立的 App Playground 想控制方向卻要額外費一番工夫。

控制 App 支援方向的 orientation
設計 iOS App 時，我們可以控制 App 畫面要以直向或橫向顯示。比方下圖的 Photos App 兩種都支援，可以用直向跟橫向顯示照片。medium.com

假設我們用 App Playground 撰寫的程式如下。

import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Image("penguin")
                .resizable()
                .scaledToFit()
            Text("A cheerful penguin coding on a MacBook in a cozy setting.")
                .font(.title)
        }
    }
}

預設 App 可以直向也可以橫向。

以下我們一步步示範如何指定方向，目標讓 App Playground(.swiftpm) 只能橫向。

從 Mac 的 Finder 點選 swiftpm 檔，從右鍵選單點選 Show Package Contents

雙擊 Package.swift

設定 supportedInterfaceOrientations

Package.swift 有許多 App 的相關設定，請找到其中的 supportedInterfaceOrientations 區塊。supportedInterfaceOrientations 預設包含 portrait & landscape，因此 App 會支援直向跟橫向。

彼得潘想要 App 只能橫向，將它改成只有 .landscapeRight 和 .landscapeLeft。

supportedInterfaceOrientations: [
    .landscapeRight,
    .landscapeLeft,
]

重新啟動 App。現在 App 只能橫向，轉成直向時畫面不會跟著旋轉。

ps: Preview 還是可以直向，方向的設定只影響在實機或模擬器執行的 App。
