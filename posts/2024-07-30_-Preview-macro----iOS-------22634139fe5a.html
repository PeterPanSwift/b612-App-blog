#Preview macro 搭配 iOS 的版本問題

透過 #Preview macro 我們可以方便地預覽 App 畫面，不過當專案的 Minimum Deployments 小於 iOS 17 時，在某些情況會遇到版本問題。

  • 問題 1: 顯示 UIKit 的元件。
  • 問題 2: 搭配 traits 參數。

iOS 17 以上才支援在 #Preview { } 顯示 UIKit 的元件和搭配參數 traits，因此修正的方法很簡單，在 #Preview 前加入 @available(iOS 17.0, *)即可修正問題。

@available(iOS 17.0, *)
#Preview {
    let storyboard = UIStoryboard(name: "Main", bundle: nil)
    return storyboard.instantiateInitialViewController()!
}
@available(iOS 17.0, *)
#Preview(traits: .sizeThatFitsLayout) {
    ContentView()
}
