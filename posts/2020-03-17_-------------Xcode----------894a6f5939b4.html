只要打 {, [, (, “，不用自己打 }, ], ),”

程式裡很多東西是成對的，比方 { }，[ ]，( ) & ""。貼心的 Xcode 會幫我們輸入另一半，所以我們只要打左邊的 { ，[ ，( & "，Xcode 會幫我們完成右邊的另一半，{ }，[ ]，( ) & "" 。

在參數的輸入框按下 enter 鍵

在參數的藍色框框按下 enter 鍵，加快程式開發速度
當我們撰寫 Swift 程式時，時常需要在呼叫 function 或生成東西的 ( ) 裡傳入參數。當我們輸入參數時，若能在某些時候在藍色框框按下 enter 鍵，將大大加快開發的速度。medium.com

按 esc 叫出自動完成選單

當自動完成選單消失時，我們可按 esc 叫醒它。

按 esc 顯示參數提示

比方我們在生成 UIColor 時，忘了輸入參數 alpha。

let color = UIColor(red: 1, green: 0, blue: 0)

此時我們可輸入逗號，然後按 esc 顯示參數提示。

多多點選 { } 圖示的 code snippet

初學者容易忘了程式語法。沒關係，Xcode 早為我們準備好常用語法的 code snippet。比方想輸入 if else 時，我們只要打 ife，自動完成選單將列出顯示 { } 圖示(code snippet) 的 ifelse，點選它即可完成 if else 的輸入。

Xcode 幫我們輸入了 if else 的重點程式，接下來在框框裡輸入內容的小事就由我們接手吧。
