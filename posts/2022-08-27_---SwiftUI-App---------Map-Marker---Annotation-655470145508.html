開發 SwiftUI App 的地圖: 使用 Map、Marker & Annotation

開發 SwiftUI iOS App 時，我們可以利用 MapKit 的 Map 顯示地圖。接下來我們將學習在 SwiftUI App 加入以下常見的地圖功能:

  • 顯示地圖。
  • 設定地圖顯示的範圍。
  • 加入氣球形狀的標記 — Marker。
  • 加入客製樣子的標記 — Annotation。

顯示地圖

import MapKit，使用 Map 生成地圖。

import SwiftUI
import MapKit

struct ContentView: View {
    var body: some View {
        Map()
    }
}

若想在 preview 或模擬器上縮放地圖，可按住 option 鍵後拖曳觸控板。

查詢地點的經緯度

我們希望在地圖的中心點顯示英國的大笨鐘，因此我們需要在程式裡輸入它的經緯度。別擔心，透過 Apple Map，我們可以輕易取得位置的經緯度，方法如下:

利用 Mac 的 Maps App 查詢經緯度的兩個方法
以查詢 apple 101 為例。medium.com

從 Apple Map 查詢的大笨鐘(Big Ben)經緯度是 51.500702, -0.124562。

設定地圖顯示的範圍

Map 的 initialPosition 控制地圖顯示的範圍。以下程式利用 MKCoordinateRegion 的 center 設定地圖中心點的經緯度，latitudinalMeters 和 latitudinalMeters 設定經緯度的範圍。

import SwiftUI
import MapKit

struct ContentView: View {
    var body: some View {
        Map(initialPosition: .region(region)) 
    }
    
    private var region: MKCoordinateRegion {
        MKCoordinateRegion(center: CLLocationCoordinate2D(latitude: 51.500702, longitude: -0.124562), latitudinalMeters: 1000, longitudinalMeters: 1000)
    }
}

加入氣球形狀的標記 — Marker

使用 Marker 生成氣球形狀的標記，傳入顯示的名字和經緯度。

import SwiftUI
import MapKit

struct ContentView: View {
    var body: some View {
        Map {
            Marker("Big Ben", coordinate: CLLocationCoordinate2D(latitude: 51.500702, longitude: -0.124562))
        }
    }
}

加了標記後，地圖會將聰明地將顯示的範圍縮放到能顯示標記的大小。由於我們只有一個標記，標記將出現在地圖的中心。

為了方便輸入經緯度，我們也可以事先定義 CLLocationCoordinate2D 的 extension，在裡面宣告標記的經緯度。

extension CLLocationCoordinate2D {
    static let bigBen = CLLocationCoordinate2D(latitude: 51.500702, longitude: -0.124562)
    static let cambridgeUniversity = CLLocationCoordinate2D(latitude: 52.2054, longitude: 0.1132)
}

struct ContentView: View {
    var body: some View {
        Map {
            Marker("Big Ben", coordinate: .bigBen)
            Marker("Cambridge University", coordinate: .cambridgeUniversity)
        }
    }
}

我們也可以結合 initialPosition 和 Marker，同時設定地圖顯示的範圍和加入標記。

import SwiftUI
import MapKit

extension CLLocationCoordinate2D {
    static let bigBen = CLLocationCoordinate2D(latitude: 51.500702, longitude: -0.124562)
    static let cambridgeUniversity = CLLocationCoordinate2D(latitude: 52.2054, longitude: 0.1132)
}

struct ContentView: View {
    var body: some View {
        Map(initialPosition: .region(region)) {
            Marker("Big Ben", coordinate: .bigBen)
            Marker("Cambridge University", coordinate: .cambridgeUniversity)
        }
    }
    
    private var region: MKCoordinateRegion {
        MKCoordinateRegion(center: CLLocationCoordinate2D(latitude: 51.500702, longitude: -0.124562), latitudinalMeters: 1000, longitudinalMeters: 1000)
    }
}

加入客製樣子的標記 — Annotation

使用 Annotation 可以自己設計標記的樣子，以下是顯示 SF Symbol 的標記。

extension CLLocationCoordinate2D {
    static let bigBen = CLLocationCoordinate2D(latitude: 51.500702, longitude: -0.124562)
    static let cambridgeUniversity = CLLocationCoordinate2D(latitude: 52.2054, longitude: 0.1132)
}

struct ContentView: View {
    var body: some View {
        Map {
            Annotation("Big Ben", coordinate: .bigBen) {
                Image(systemName: "clock")
                    .padding()
                    .background {
                        Circle()
                            .foregroundStyle(.yellow)
                    }
            }
            Annotation("Cambridge University", coordinate: .cambridgeUniversity) {
                Image(systemName: "graduationcap")
                    .padding()
                    .background {
                        Circle()
                            .foregroundStyle(.yellow)
                    }
            }
        }
    }
}

以下是顯示 asset 圖片的標記。

extension CLLocationCoordinate2D {
    static let bigBen = CLLocationCoordinate2D(latitude: 51.500702, longitude: -0.124562)
    static let cambridgeUniversity = CLLocationCoordinate2D(latitude: 52.2054, longitude: 0.1132)
}

struct ContentView: View {
    var body: some View {
        Map {
            Annotation("Big Ben", coordinate: .bigBen) {
                Image(.bigBen)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 50, height: 50)
                    .clipShape(.circle)
            }
            Annotation("Cambridge University", coordinate: .cambridgeUniversity) {
                Image(.cambridgeUniversity)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 50, height: 50)
                    .clipShape(.circle)
            }
        }
    }
}

AI 範例: 東京迪士尼樂園介紹

prompt: 請用 SwiftUI 設計一個東京迪士尼的介紹頁面，包含圓片，地圖和簡介，使用 Map 搭配 initialPosition MapCameraPosition.region 和 Marker。

import SwiftUI
import MapKit

struct DisneylandTokyoView: View {
    // 設定迪士尼樂園的座標
    private let disneyLocation = CLLocationCoordinate2D(
        latitude: 35.6329,
        longitude: 139.8804
    )
    
    // 設定地圖顯示區域
    private var region: MKCoordinateRegion {
        MKCoordinateRegion(
            center: disneyLocation,
            span: MKCoordinateSpan(latitudeDelta: 0.01, longitudeDelta: 0.01)
        )
    }
    
    var body: some View {
        ScrollView {
            VStack(spacing: 20) {
                // 圓形圖片
                Image(.disney) // 請確保有此圖片資源
                    .resizable()
                    .scaledToFill()
                    .frame(width: 200, height: 200)
                    .clipShape(.circle)
                    .overlay {
                        Circle()
                            .stroke(.blue, lineWidth: 4)
                    }
                    .shadow(radius: 7)
                
                // 標題
                Text("東京迪士尼樂園")
                    .font(.title)
                    .fontWeight(.bold)
                
                // 地圖
                Map(initialPosition: .region(region)) {
                    Marker("東京迪士尼樂園", coordinate: disneyLocation)
                }
                .frame(height: 300)
                .clipShape(.rect(cornerRadius: 20))
                .padding(.horizontal)
                
                // 簡介
                VStack(alignment: .leading, spacing: 10) {
                    Text("關於東京迪士尼")
                        .font(.title2)
                        .fontWeight(.bold)
                    
                    Text("""
                        東京迪士尼樂園（Tokyo Disneyland）於1983年4月15日開園，是日本第一個迪士尼主題樂園。

                        園區分為七大主題區：
                        • 世界市集
                        • 探險樂園
                        • 西部樂園
                        • 幻想樂園
                        • 卡通城
                        • 明日樂園
                        • 環形花園

                        這裡不僅有經典的迪士尼遊樂設施，還有獨特的日式服務與美食，每年吸引數百萬遊客前來遊玩。
                        """)
                        .padding()
                        .background(.gray.opacity(0.1))
                        .clipShape(.rect(cornerRadius: 10))
                }
                .padding(.horizontal)
            }
            .padding(.vertical)
        }
    }
}
