使用舊版 Xcode 開啟新版 Xcode 建立的專案

使用舊版的 Xcode 可以開啟新版 Xcode 建立的專案嗎 ?

這跟專案檔的設定有關，有可能不行。如下圖所示，當我們使用 Xcode 13 開啟 Xcode 14 建立的專案時，將遇到以下錯誤。

The project at '/Users/peterpan/Desktop/Demo/Demo.xcodeproj' cannot be opened because it is in a future Xcode project file format. Adjust the project format using a compatible version of Xcode to allow it to be opened by this version of Xcode.

解法: 調降 Project Format 的版本

用 Xcode 14 打開專案，點選專案檔。

切換到 File inspector 分頁。它的 Project Format 預設是 Xcode 14.0-compatible，難怪 Xcode 13 無法開啟。

將它降版為 Xcode 13.0-compatible 即可。
