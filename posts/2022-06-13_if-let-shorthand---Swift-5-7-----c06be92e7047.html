if let shorthand — Swift 5.7 新功能

從 Swift 5.7(搭配 Xcode 14)開始，optional binding 有更簡單的寫法了 !

當我們寫 optional binding 時，時常會將 if let 的常數名取名跟 optional 一樣，比方以下的 if let girlFriend = girlFriend。

var girlFriend: String? = "Wendy"
if let girlFriend = girlFriend {
   print(girlFriend)
}

由於我們時常這樣寫，所以 Swift 推出了縮寫的版本，我們可以拿掉後面的 = & optional，只要寫 if let girlFriend 就好。

var girlFriend: String? = "Wendy"
if let girlFriend {
   print(girlFriend)
}

同樣的概念也可以作用在 guard let，while let，if var，guard var & while var，比方以下例子:

func test() {
    var girlFriend: String? = "Wendy"
    guard let girlFriend else { return }
    print(girlFriend)
}
