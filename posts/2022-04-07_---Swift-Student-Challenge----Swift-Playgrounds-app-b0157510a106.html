Swift Student Challenge 參賽的作品需求

彼得潘已經不是學生，無法參加 Swift Student Challenge，不過還是滿好奇怎麼樣的 App 才能參加比賽。根據 Apple 的比賽文件，作品需滿足以下條件。

  • 以 app playground(.swiftpm)開發。
  • 使用 Mac 的 Xcode 或 iPad / Mac 的 Swift Playgrounds 開發。
  • swiftpm 壓縮成 zip 上傳，檔案的大小不能超過 25 MB。
  • 開發單機版可實現的功能，不能透過網路抓取資料。
  • 畫面的文字內容以英文撰寫。

有興趣的朋友可再參考以下詳細的作品需求和參加條件。

Eligibility - Swift Student Challenge
Make sure to review the current eligibility conditions to confirm that you're qualified to participate.developer.apple.com

建議開發的方向和技術

  • 建議用 SwiftUI 開發。(ps: 雖然也有少數入選的作品用 UIKit)
  • 建議搭配生動的動畫跟音樂音效(AVFoundation)。
  • 主題最好是知識類的 App，學習到某種知識。
  • 搭配有趣的小遊戲。
  • 包含教學導覽頁面。
  • 可以考慮設計 iPad 介面，方便展示更多東西。

做到以上幾點就滿有機會入選，如果有時間或興趣研究其它技術，可考慮以下幾點。

  • iOS 18 的新技術，比方 Apple Intelligence。
  • AR / VR 相關技術，比方 ARKit、RealityKit、 SceneKit(製作 3D 畫面)。
  • VisionKit，辨識資訊，比方用相機辨識人臉。
  • SpriteKit，方便創作 2D 遊戲和動畫效果。
  • CoreML，Machine Learning 的相關技術。
  • PencilKit，Apple Pencil 相關技術。

使用 Mac 的 Xcode 開發 App Playground，搭配 SwiftUI 或 UIKit

  • 建立專案時選擇 iOS 下的 App Playground。
  • 專案預設以 SwiftUI 開發。

若是你本來就熟悉 SwiftUI，恭喜你，現在可以馬上開始用心愛的 SwiftUI 開發 App。

  • 若想改用 UIKit 開發，則須做以下修改。

假設我們在專案裡加入了 Main.storyboard & ViewController.swift。

下圖 controller 的類別為 ViewController，Storyboard ID 為 ViewController。

我們必須將 view controller 包裝成 SwiftUI view 才能在 SwiftUI App 裡顯示，請參考以下連結的說明，將 view controller 包裝成 SwiftUI view。

在 SwiftUI App 加入 view controller & 從 SwiftUI 畫面切換到 Storyboard 畫面
SwiftUI 有很多很棒很炫的功能，幫助我們更容易地開發 iOS App，不過因為它才剛出生，所以一些傳統 iOS App 常用的 UI 元件還不支援，比方它缺少以下這些元件:medium.com
import SwiftUI

struct ViewControllerView: UIViewControllerRepresentable {
    
    func makeUIViewController(context: Context) -> ViewController {
        let storyboard = UIStoryboard(name: "Main", bundle: nil)
        let controller = storyboard.instantiateViewController(identifier: "\(ViewController.self)") as! ViewController
        return controller
    }
    
    func updateUIViewController(_ uiViewController: ViewController, context: Context) {
        
    }
    
    typealias UIViewControllerType = ViewController
    
    
}

將剛剛定義的 ViewControllerView 設為 App 的第一個畫面。

import SwiftUI

@main
struct MyApp: App {
    var body: some Scene {
        WindowGroup {
            ViewControllerView()
        }
    }
}

使用 iPad 或 Mac 的 Swift Playgrounds 開發 app，搭配 SwiftUI 或 UIKit

參考以下連結的說明。

iPad Swift Playgrounds 開發 iOS App 初體驗
iPad 有著大大的螢幕和接近電腦的效能，所以很多人常夢想有天能在 iPad 上帥氣地開發 App，可惜目前 Xcode 只能在 Mac 上執行。medium.com

值得注意的，Swift Playgrounds 建立的 App 專案預設採用 SwiftUI 開發，若想改用 UIKit，可參考前面提到的方法。不過此時畫面只能使用程式開發，不能搭配 storyboard 跟 xib，因為我們不能在 Swift Playgrounds 裡編輯 storyboard 跟 xib。
