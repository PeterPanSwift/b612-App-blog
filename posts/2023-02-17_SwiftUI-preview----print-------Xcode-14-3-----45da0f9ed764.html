SwiftUI preview 支援 print 了 ! — Xcode 14.3 新功能

從 Xcode 14.3 開始，我們在 SwiftUI preview 也能看到 print 列印的訊息了 !

import SwiftUI

struct ContentView: View {
    @State private var count = 0
    
    var body: some View {
        VStack {
            Image("loveU3000")
                .resizable()
                .scaledToFit()
            Button("❤️ You") {
                count += 1
                print("Love You \(count)")
            }
            .buttonStyle(.borderedProminent)
            Text("\(count)")
        }
        .font(.largeTitle)
    }
}

#Preview {
    ContentView()
}

下方的 Console 現在多了一個 Previews 分頁。比方我們點選 preview 的 love you 按鈕，看到 Canvas 分頁顯示 print 列印的 Love You 1, Love You 2, …。

當 App 安裝到摸擬器或實機測試時，print 列印的結果則會在 Executable 分頁。
