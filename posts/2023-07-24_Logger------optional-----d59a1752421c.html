Logger 無法列印 optional 的資料

當我們使用 OSLog 的 Logger 列印資訊時，必須特別注意 optional 的問題。Logger 無法列印 optional 的資料，因此以下程式會產生錯誤，錯誤訊息顯示 No exact matches in call to instance method ‘appendInterpolation’。

我們必須取出 optional 的內容後，才能用 Logger 印出。剛剛的程式有以下幾種解法。

  • 使用 Nil Coalescing Operator。
let logger = Logger()
let name: String? = "peter"
logger.log("我的名字是\(name ?? "")")
  • 使用 if let 或 guard let。
let logger = Logger()
let name: String? = "peter"
if let name {
    logger.log("我的名字是\(name)")
}
  • 使用 force unwrap。(確定資料一定有值時)
let logger = Logger()
let name: String? = "peter"
logger.log("我的名字是\(name!)")
