#156 參考 raywenderlich 的 Make An App Like Instagram 製作 App 專題

當我們剛學會基本的 iOS App 開發技術，想練習製作一個完整的 App 作品時，常常會因為以下幾點非程式的原因卡關:

  • 沒有靈感，想不到要做什麼。
  • 沒有藝術細胞，只會設計醜醜的畫面。
  • 不懂如何從無到有設計一個包含完整流程和功能的 App。

其實如果上班當 iOS App 工程師，通常不會有這些問題，因為會有同事專門規劃 App 的流程和功能，也有美術負責設計精美的畫面。

但是如果沒同事也沒朋友，只有自己一個人呢 ?

沒關係，最近彼得潘研究了 Ray Wenderlich 的 How To Make An App Like Instagram in iOS，發現它不只是很棒的課程，還可以完美解決剛剛的問題。

How To Make An App Like Instagram in iOS
Have you ever wanted to build an app like Instagram, from architecture to coding to the back-end server, but weren't…www.raywenderlich.com

How To Make An App Like Instagram in iOS 教我們從頭開發一個包含基本功能的 IG App，使用 SwiftUI 開發 iOS App，後台搭配基於 Swift 的框架 Kitura。

有興趣的朋友可進一步研究它的課程。就算沒有看課程，我們也可以從 GitHub 下載它的範例，參考它精美的 Sketch 設計檔開發一個完整的 App 作品，步驟如下:

安裝字體檔 CoolStoryregular.ttf

IG App 裡使用到字體 CoolStoryregular，因此我們必須在 Mac 先安裝 CoolStoryregular，待會打開 App 的設計檔時才能正常顯示。

雙擊 video-insta-materials-versions-1.0 > 14-login-and-signup > resources 下的 CoolStoryregular.ttf 安裝。

開啟 Petstagram Final.sketch

開啟 video-insta-materials-versions-1.0 > 06-ui-design > resources 下的 Petstagram Final.sketch。Petstagram Final.sketch 是 IG App 的設計檔，記得先安裝 Sketch App 才能開啟它。

Sketch
The design platform used by over one million people - from freelancers, to the world's largest teams.www.sketch.com

Petstagram Final.sketch 裡主要分成 UX Flow & UI Flow。從 UX Flow 我們可以了解 App 的流程，知道開發時如何串接不同的頁面。而 UI Flow 則展示每個頁面的模樣，因此製作 App 頁面時要參考 UI Flow。

  • 登入註冊相關的 UX Flow
  • 登入註冊相關的 UI Flow。
  • 主功能的 UX Flow

包含了主畫面的動態牆，留言列表，拍照 & 個人頁面等。

  • 主功能的 UI Flow

包含了 Home，Comment，Like，Search，Take Photo ＆ Search。

將 Sketch 設計檔輸出到 Zeplin

製作 App 頁面時，我們必須了解畫面的細節，比方元件的大小，位置，顏色等。因此我們先將 Sketch 設計檔輸出到 Zeplin，它可以幫忙標示畫面的各個細節。

  • iOS App

使用 SwiftUI 或 UIKit。

  • 後台

自己寫後台，或是第三方的平台，像是 Firebase，Airtable & Imgur。
