設定 SwiftUI & UIKit App navigation bar 的副標題 — iOS 26 新功能

從 iOS 26 開始，我們也可以在 navigation bar 顯示副標題了。以下分別介紹 SwiftUI & UIKit App 的做法。

SwiftUI App

透過 modifier navigationSubtitle 設定副標題。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(.peter)
                .resizable()
                .scaledToFill()
                .ignoresSafeArea()
                .navigationTitle("Iceland Boy")
                .navigationSubtitle("Coding with MacBook Pro")
        }
    }
}

當透過 navigationBarTitleDisplayMode(.inline) 將標題縮小時，標題和副標題將顯示在 navigation bar 的中間。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(.peter)
                .resizable()
                .scaledToFill()
                .ignoresSafeArea()
                .navigationTitle("Iceland Boy")
                .navigationSubtitle("Coding with MacBook Pro")
                .navigationBarTitleDisplayMode(.inline)
        }
    }
}

UIKit App

透過 navigationItem 的 subtitle 設定副標題。

navigationItem.title = "Iceland Boy"
navigationItem.subtitle = "Coding with MacBook Pro"
