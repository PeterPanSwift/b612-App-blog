Xcode 的提示輸入框有兩種

當我們在 Xcode 寫程式時，長方形的提示輸入框提醒我們要填入東西，不過如果你仔細看，將發現它有兩種不同的呈現呢。

第一種: 長方形的提示文字只有參數型別。

我們呼叫 function 時，需要保留參數名字和冒號。

func eat(name: String) {

}
eat(name: "牛排")

第二種: 長方形的提示文字有參數名字和參數型別。

因為原本 function 定義時，外部參數名字設為 _ ，因此呼叫 function 時可省略參數名字。

func eat(_ name: String) {

}
eat("牛排")
