#81 製作 Google 搜尋 App — 搭配 Google Custom Search JSON API

Custom Search JSON API | Custom Search | Google Developers
The Custom Search JSON API lets you develop websites and applications to retrieve and display search results from…developers.google.com

搭配 Google Custom Search JSON API，我們可以自己製作搜尋宇宙萬物的 Google App。如果另外再指定搜尋的網站，甚至還可以開發呈現 IG，Youtube，FB 內容的 App !

  • 使用者可輸入搜尋的文字。
  • 解析 Google Custom Search JSON API 回傳的 JSON 資料，以 table view 或 collection view 呈現搜尋結果。

API 網址範例:

參數 cx & key 如何取得等下會說明。

https://www.googleapis.com/customsearch/v1?cx=abc&key=def&q=小幸運
  • cell 可顯示許多有趣的資訊，比方縮圖，標題，描述等。

參考 JSON 範例

  • 點選 cell 顯示對應的網頁內容。
  • 指定搜尋的網站，比方搜尋 Youtube，IG，FB 粉絲團。

搜尋 Youtube 的網址範例:

q 設定戴佩妮，siteSearch 設定網址 youtube.com

https://www.googleapis.com/customsearch/v1?cx=abc&key=def&q=戴佩妮&siteSearch=youtube.com

搜尋 IG 的網址範例:

q 設定 *，siteSearch 設定 JJ 的網址 www.instagram.com/jjlin/p

https://www.googleapis.com/customsearch/v1?cx=abc&key=def&q=*&siteSearch=www.instagram.com/jjlin/p

搜尋 FB 粉絲團的網址範例:

q 設定 *，siteSearch 設定彼得潘的貼文網址 www.facebook.com/iphone.peterpan/posts

https://www.googleapis.com/customsearch/v1?cx=abc&key=def&q=*&siteSearch=www.facebook.com/iphone.peterpan/posts
  • 滑到底自動載入更多資料。(或是點選底部的 More Result)

API 網址範例:

從 JSON nextPage 的 startIndex 取得載入更多資料時， start 參數應設定的數值。

https://www.googleapis.com/customsearch/v1?cx=abc&key=def&q=小幸運&start=11

如何取得 search engine ID & API key

想要串接 Google Custom Search JSON API，我們必須在網址裡夾帶 search engine ID & API key，因此接下來先讓我們看看如何取得這兩項資訊。

1 建立 Custom Search Engine

想得到 search engine ID 必須先建立 Custom Search Engine。我們連到 Custom Search Engine control panel 建立 Search Engine。

https://cse.google.com/cse/all

點選 Add 建立 Search Engine。在 Sites to search 欄位輸入搜尋的網站，比方我們輸入 *.com & *.tw 表示想搜尋網址裡有 .com & .tw 的網頁。

2 取得 Search Engine ID

有了 search engine 後，我們可從以下網頁點選它，查看它的 Search Engine ID。

https://cse.google.com/cse/all

3 取得 API key。

從以下網頁點選 GET A KEY。

Custom Search JSON API | Custom Search | Google Developers
The Custom Search JSON API lets you develop websites and applications to retrieve and display search results from…developers.google.com

點選 Create a new project 或某個之前建立的 project。

得到 API KEY !

串接 Google Custom Search JSON API

有了 search engine ID & API key，我們將可採用以下網址搜尋 Google 上比全世界人口還多的豐富資料。

https://www.googleapis.com/customsearch/v1?cx=006414095369450443889:m2aush7jofk&key=abcde&q=wwdc

此網址至少要包含以下三個參數:

  • key: API key
  • cx: search engine ID
  • q: 搜尋的文字

除了這三個必須參數，我們還可以添加其它參數進行更進階的搜尋，詳細的參數列表和回傳的 JSON 格式可參考以下連結:

CSE: list | Custom Search | Google Developers
method returns metadata about the search performed, metadata about the custom search engine used for the search, and…developers.google.com

ps: Google Custom Search JSON API 可以免費使用，不過一天只能下 100 個，想做更多搜尋請另外付錢給 Google。
