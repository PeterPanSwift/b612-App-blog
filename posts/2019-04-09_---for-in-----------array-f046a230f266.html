其實不用這麼麻煩，利用 for in 迴圈的語法，剛剛的程式可以簡化成這樣。

var images = [UIImage]()
for number in 1...59 {
  let image = UIImage(named: String(format: "%02d", number))!
  images.append(image)
}
let animatedImage = UIImage.animatedImage(with: images, duration: 1)

說明

(1) 產生空的 array，array 裡儲存的東西型別為 UIImage。

var images = [UIImage]()

(2) 利用 for in 迴圈讓 { } 裡的程式跑 59 次。

for number in 1...59 {
}

for in 迴圈 { } 裡的程式可以重覆執行，由於 in 後的 1...59 代表數字 1 到 59，有 59 個數字，所以 { } 裡的程式會跑 59 次。

(3) 產生 UIImage ，加它加到 array 裡。

let image = UIImage(named: String(format: "%02d", number))!

剛剛 for in 迴圈 { } 裡的程式第一次執行時，number 將是數字 1，第二次是數字 2，其它以此類推。由於原本圖片的名字是從 01, 02 一直到 59，因此我們利用 String(format: “%02d”, number) 將數字變成字串，而且讓只有個位數的數字前面加上 0，比方 1 變成 01，2 變成 02。

(4) 將 UIImage 物件加到 array 裡。

images.append(image)

改成 for in 迴圈的語法後，我們再也不用擔心圖片太多了。就算圖片從 59 張變成 100 張，也只要將 for number in 1…59 改成 for number in 1…100 。
