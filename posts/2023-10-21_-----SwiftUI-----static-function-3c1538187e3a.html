快速生成 SwiftUI 形狀的 static function

開發 SwiftUI 時，我們時常會用 clipShape 將元件裁切成特定形狀。例如以下裁切成圓形和圓角矩形的例子。

  • 圓形。
struct ContentView: View {
    var body: some View {
        Image(.littlePrince)
            .resizable()
            .scaledToFill()
            .frame(width: 300, height: 300)
            .clipShape(Circle())
    }
}
  • 圓角矩形。
struct ContentView: View {
    var body: some View {
        Image(.littlePrince)
            .resizable()
            .scaledToFill()
            .frame(width: 300, height: 300)
            .clipShape(RoundedRectangle(cornerRadius: 25))
    }
}

雖然傳入 Circle() & RoundedRectangle(cornerRadius: 25) 也滿簡單的，不過 SwiftUI 現在有更簡單的寫法。它增添了快速生成常用形狀的 static function，而且還能相容到 iOS 13 呢。

以剛剛的圓形和圓角矩形為例，現在我們可以輸入更簡潔的 .circle() 和 .rect(cornerRadius: 25)。

  • 圓形。
struct ContentView: View {
    var body: some View {
        Image(.littlePrince)
            .resizable()
            .scaledToFill()
            .frame(width: 300, height: 300)
            .clipShape(.circle)
    }
}
  • 圓角矩形。
struct ContentView: View {
    var body: some View {
        Image(.littlePrince)
            .resizable()
            .scaledToFill()
            .frame(width: 300, height: 300)
            .clipShape(.rect(cornerRadius: 25))
    }
}
