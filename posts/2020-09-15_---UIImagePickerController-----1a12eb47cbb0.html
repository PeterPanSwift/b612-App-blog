使用 UIImagePickerController 選影片

透過 UIImagePickerController 我們可以拍照和從相簿選照片，不過預設它只能拍照跟選照片，若想錄影或選影片，必須設定 mediaTypes。

mediaTypes | Apple Developer Documentation
An array that indicates the media types to access by the media picker controller.developer.apple.com

設定 mediaTypes

@IBAction func buttonTapped(_ sender: Any) {
    let controller = UIImagePickerController()
    controller.sourceType = .photoLibrary
    if let mediaTypes = UIImagePickerController.availableMediaTypes(for: .photoLibrary) {
        controller.mediaTypes = mediaTypes
    }
    controller.delegate = self
    present(controller, animated: true)
}

利用 UIImagePickerController.availableMediaTypes(for: .photoLibrary) 取得相簿支援的 media types，因此會包含照片和影片。

若想限制只能選擇影片，我們可將 mediaTypes 設為 [kUTTypeMovie as String]。

controller.mediaTypes = [kUTTypeMovie as String]

ps: 記得先 import CoreServices，若要存取圖片可使用 kUTTypeImage。

取得影片的 URL

extension ViewController: UIImagePickerControllerDelegate, UINavigationControllerDelegate {
    
    func imagePickerController(_ picker: UIImagePickerController, didFinishPickingMediaWithInfo info: [UIImagePickerController.InfoKey : Any]) {
      
        guard let url = info[.mediaURL] as? URL else {
           return
        }
       
    }
}

從 info[.mediaURL] 取得影片的 URL。
