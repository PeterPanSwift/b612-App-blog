Swift array & dictionary 的排版 & trailing comma

開發 iOS App 時，我們時常用 array 或 dictionary 儲存大量的資料，比方以下儲存兩個深度問題的 array qnas。

struct Qna {
    let question: String
    let answer: String
}

let qnas = [Qna(question: "世界上最棒的星座", answer: "水瓶座")
, Qna(question: "彼得潘的另一個名字", answer: "小王子")]

不過當 array 或 dictionary 有很多資料時，這樣的排版將讓我們不易閱讀。

每筆資料單獨一行

建議將 array 或 dictionary 尊貴的 [ & ] 單獨一行，然後每筆資料也單獨一行，保持安全距離，這樣將能更方便一眼看清裡頭的內容。

let qnas = [
    Qna(question: "世界上最棒的星座", answer: "水瓶座"),
    Qna(question: "彼得潘的另一個名字", answer: "小王子")
]

trailing comma(dangling comma)

最後一筆資料的後面也可以加上逗號(comma)，如此未來在 array 裡添加資料時，不用再花時間先補上逗號。

let qnas = [
    Qna(question: "世界上最棒的星座", answer: "水瓶座"),
    Qna(question: "彼得潘的另一個名字", answer: "小王子"),
]
