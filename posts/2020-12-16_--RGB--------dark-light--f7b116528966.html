從 RGB 判斷顏色深淺(dark/light)

開發 iOS App 時，有時我們希望文字顏色能依背景顏色的深淺動態調整。比方以下 Apple SwiftUI 的範例，當表格 row 是淺色時，顯示黑色文字。當表格 row 是深色時，顯示白色文字。

不過要如何判斷顏色的深淺呢?

答案就藏在 Apple 的範例程式。下載 Apple 以下連結的範例，在它的 extension Color 裡定義了判斷顏色深淺的 function isLightColor。

Apple Developer Documentation
Edit descriptiondeveloper.apple.com
func isLightColor(red: CGFloat, green: CGFloat, blue: CGFloat) -> Bool {
    
    let lightRed = red > 0.65
    let lightGreen = green > 0.65
    let lightBlue = blue > 0.65
    
    let lightness = [lightRed, lightGreen, lightBlue].reduce(0) { $1 ? $0 + 1 : $0 }
    return lightness >= 2
}
