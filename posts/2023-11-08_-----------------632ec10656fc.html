適合寫程式的字體和有趣的連字效果

身為程式設計師，我們每天要盯著螢幕看著密密麻麻的程式碼，所以選擇一個適合寫程式的字體也是很重要的。

最近發現有許多針對寫程式推出的字體，它們通常有以下特點。

  1. 等寬性(Monospaced): 每個字佔相同的水平空間，有利於程式的對齊和閱讀。
  2. 辨識度高: 能輕易辨識容易混淆的字元，例如數字 0 和字母 O，小寫的 l和數字 1，大寫的 I 和小寫的 L。
  3. 連字功能(ligatures): 將常見的程式符號組合，比方將 != 轉換成單一的符號，讓程式更為整潔。

以下彼得潘以免費的 Fira Coda、Google Sans Code 和付費的 Comic Code 字體為例，在 Xcode 上示範套用的效果。

  • 新增 Xcode 的 Theme，套用新字體。

從 Quick Actions 點選 Edit Themes。

點選 Theme 的名字叫出選單。

點選 Manage Themes。

點選左下的 + 新增 Theme。

新增 Theme 時必須先選擇某個樣版，以下我們選擇 Presentation Large (Dark)。

產生的 Theme 名字為 Presentation Large (Dark) 2，點選右邊的 i 可以改名。

改名為 Fira Code。

按 cmd + a 全選 Source Editor 裡的選項，然後點擊下圖的框框調整字體。

選擇 Fira Code Regular。
