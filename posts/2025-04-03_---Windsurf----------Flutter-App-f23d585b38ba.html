使用 Windsurf 快速部署網頁版的 Flutter App

最近 Windsurf 推出了 App Deploy 功能，我們可以快速將網站部署到 Netlify，馬上變身成可公開連線的網站 。(ps: Netlify 是免費部署靜態網站的平台)

App Deploys
Docs for Windsurf (formerly Codeium): your modern coding superpowers.docs.windsurf.com

以下彼得潘以 Flutter 開發的網頁版 App 為例示範。

使用 Windsurf 打開 Flutter App 的專案資料夾，然後在 Cascade Write mode 輸入部署到 Netlify 的 prompt。

Deploy this project to Netlify

依據 AI 的指示點選 Accept 執行指令 flutter build web 生成網頁。

若是沒有 Netlify 帳號，請先在 Netlify 網站註冊帳號(https://app.netlify.com/signup)。

點選 Accept 執行指令 npm install -g netlify-cli 生成安裝 Netlify CLI。若是不能執行 npm，記得先安裝 Node.js。

在 Mac 安裝 Node.js & NPM
安裝 Node.js 後就會同時安裝 NPM，因此以下我們將示範如何在 Mac 安裝 Node.js。medium.com

ps: 若是遇到權限問題請先遵照指示修改權限。

點選 Accept 執行指令 netlify login登入 netlify。

登入成功後，若是 Cascade 沒有自動繼續，請另外輸入 continue 觸發它執行部署的最後一步，執行指令 netlify deploy --prod。

在 Terminal 選擇 Create & Configure a new site。

輸入 Site name。

部署成功 ! Website URL 顯示的 https://peter-counter.netlify.app 即為網站的網址。
