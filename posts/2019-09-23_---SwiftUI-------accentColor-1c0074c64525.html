影響 SwiftUI 元件顏色的 accentColor

SwiftUI 利用 accentColor 控制一些常用的操作元件和容器的顏色，比方以下例子:

Tab

var body: some View {
   TabView {
      SongList()
         .tabItem {
            Image(systemName: "music.house.fill")
            Text("情歌")
      }
      ContentView()
         .tabItem {
            Image(systemName: "info.circle.fill") 
            Text("About")
      }
   }
   .accentColor(.orange)
}

Slider & Button

VStack {
   Slider(value: .constant(0.5))
      .accentColor(.purple)
   Button("button") {
   }
   .accentColor(.green)
}

當 accentColor 作用於像 stack，tab 之類的元器容件時，它將同步影響容器裡面元件的 accentColor，因此以下程式當我們設定 VStack 的 accentColor 是紅色時，它裡面的 Slider & Button 也會變成紅色。

VStack {
   Slider(value: .constant(0.5))
   Button("button") {
   }
}
.accentColor(.red)
