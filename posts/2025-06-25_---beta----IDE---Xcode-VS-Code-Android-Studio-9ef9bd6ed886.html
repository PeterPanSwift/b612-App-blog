安裝 beta 版的 IDE — Xcode、VS Code、Android Studio

beta 版的 IDE 比正式版的 IDE 多了不少新功能，有興趣冒險嚐鮮的朋友也可以考慮安裝玩玩，而且 beta & 正式版的 IDE 是可以並存的，所以可依據不同的需求使用適合的 IDE。

以下彼得潘介紹三種常用 IDE 的 beta 版安裝方法，Xcode beta、VS Code Insider & Android Studio Preview。

Xcode beta

連到 Xcode 的 Resources 網頁。

Resources - Xcode - Apple Developer
Find Xcode downloads, tools, documentation, tutorials, videos, and more.developer.apple.com

點選 View available Xcode betas。

點選 Applications 下的 Xcode 26.1 beta，Mac 若是 Apple 晶片請選擇 Apple silicon，Intel 晶片選擇 Universal。

此時會開始下載 xip。

下載完成後雙擊 xip 檔解壓縮得到 Xcode-beta App，將 Xcode-beta 搬移到 Applications。

VS Code Insider

連到 VS Code 網站。

Visual Studio Code - Code Editing. Redefined
Visual Studio Code redefines AI-powered coding with GitHub Copilot for building and debugging modern web and cloud…code.visualstudio.com

點選 Insider edition。

下載適合的版本，彼得潘的電腦是 Apple 晶片的 Mac，因此選擇 Apple silicon 的 zip。

解壓縮得到綠色的 Visual Studio Code — Insiders，將它搬移到 Applications。

Android Studio Preview

連到 Android Studio 網站。

Download Android Studio & App Tools - Android Developers
Android Studio provides app builders with an integrated development environment (IDE) optimized for Android apps…developer.android.com

點選 Preview release。

下載 Beta 或 Canary 版的 Android Studio。

將 Android Studio Preview 搬移到 Applications。
