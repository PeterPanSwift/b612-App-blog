Open with Xcode 都去哪兒了 ?

Xcode 和 GitHub 深度整合，當我們連到 GitHub 的 Xcode 專案頁面，點選綠色的 Clone or download 按鈕時，將出現貼心的 Open with Xcode，方便我們下載後直接開啟它的 Xcode 專案。

但是，卻有人因為人品問題，看不到 Open with Xcode !

別想太多，GitHub 不是神，不可能知道你的人品的。你只要滿足以下兩個條件，即可看到 Open with Xcode。

  1. 在 Github 的網站，登入 Github 帳號。

未登入時，右上角將顯示 Sign in 和 Sign up。

登入帳號後，右上角將顯示你美麗帥氣的大頭照。

2. 曾在 Xcode 設定 GitHub 帳號。

如果你未曾設定過，可依照以下步驟設定。
