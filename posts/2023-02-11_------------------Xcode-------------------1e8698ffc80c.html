以下我們一個個示範各種不同的例子。

{ } 的例子

選取以下程式，然後輸入 {。

Xcode 自動輸入 }。

struct ContentView: View {
    
    var body: some View {
        
        {
            Text("hello")
            Text("Peter")
        }
        
    }
}

此時游標停在 { 前，我們接著輸入 VStack。

struct ContentView: View {
    
    var body: some View {
        
        VStack {
            Text("hello")
            Text("Peter")
        }
        
    }
}
