macOS App 的網路連線問題

最近開發 macOS App 時發現無法串接網路上的 API 抓取資料，比方以下透過 URLSession 抓取 MOPCON22 彼得潘簡介的 JSON，結果卻一無所獲，完全抓不到 !

let url = URL(string: "https://mopcon.org/api/2022/speaker/214")!
do {
    let (data, _) = try await URLSession.shared.data(from: url)
    print(data)
} catch {
    print(error)
}

Xcode 印出許多錯誤訊息，比較重要的有以下兩個錯誤。

  • 錯誤 1
networkd_settings_read_from_file Sandbox is preventing this process from reading networkd settings file at "/Library/Preferences/com.apple.networkd.plist", please add an exception
  • 錯誤 2
Error Domain=NSURLErrorDomain Code=-1003 "A server with the specified hostname could not be found." UserInfo={_kCFStreamErrorCodeKey=-72000, NSUnderlyingError=0x600003f9e700 {Error Domain=kCFErrorDomainCFNetwork Code=-1003 "(null)" UserInfo={_NSURLErrorNWPathKey=satisfied (Path is satisfied), interface: en7, ipv4, ipv6, dns, expensive, constrained, _kCFStreamErrorCodeKey=-72000, _kCFStreamErrorDomainKey=10}}, _NSURLErrorFailingURLSessionTaskErrorKey=LocalDataTask <5BD15161-C2E5-44DF-ABC7-23891C31ECBE>.<1>, _NSURLErrorRelatedURLSessionTaskErrorKey=(
    "LocalDataTask <5BD15161-C2E5-44DF-ABC7-23891C31ECBE>.<1>"
), NSLocalizedDescription=A server with the specified hostname could not be found., NSErrorFailingURLStringKey=https://mopcon.org/api/2022/speaker/214, NSErrorFailingURLKey=https://mopcon.org/api/2022/speaker/214, _kCFStreamErrorDomainKey=10}

為什麼不能連線呢？這是因為 macOS App 很重視安全，所以預設不讓我們接觸網路，畢竟網路上藏著許多可怕的東西。

不過想要的話，我們還是可以解除封印，抓取網路上的資料。

切換到 App 的 Signing & Capabilities 頁面，找到 App Sandbox 區塊，勾選 Outgoing Connections (Client) 即可讓 macOS App 串接網路上的 API。
