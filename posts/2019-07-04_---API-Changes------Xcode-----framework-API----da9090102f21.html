利用 API Changes 瀏覽不同 Xcode 版本的 framework API 差異

不斷改版的 Xcode 為我們帶來更厲害的 App 功能，也讓我們更方便開發 iOS App。不過我們要如何快速了解新版 Xcode 多了哪些功能呢 ? 一般最快的方法是閱讀 Xcode 的 Release Note。

Xcode 11 Beta 3 Release Notes
Xcode 11 includes SDKs for iOS 13, macOS Catalina 10.15, watchOS 6, and tvOS 13. Xcode 11 supports on-device debugging…developer.apple.com

如果想了解更多細節，比方 framework 多了哪些新的 API，或是想了解 beta 版的差異，想知道 Xcode 11 beta 3 跟 beta 2 有什麼差別，那就要靠 Apple Developer Documentation 網站上顯示 API Changes 的功能了，步驟如下:

連到 Apple Developer Documentation 網站

Apple Developer Documentation
Browse the latest developer documentation including API reference, articles, and sample code.developer.apple.com

點選右上角的 API Changes 旁的 Show

瀏覽不同 Xcode 版本的 framework API 差異

此時網頁上將顯示不同 Xcode 版本間，Swift & 各種 framework API 的變化。

有三種可能的差異，Modified(修改)，Added(新增) 和 Deprecated(過時刪除)，從網頁的左上角可看到它們分別代表的圖示和變化的數量。

如下圖所示，當顯示 Xcode 10.2 & Xcode 11 beta 3 的差異時，可看到 SwiftUI 是新增的 framework，而 Swift，UIKit 等原本存在的 framework 則有了修改。

從網頁右上角我們可選擇想要比較的版本。接下來我們試試將它改成比較 Xcode 11 beta 2 & beta 3。

Views and Controls 顯示它有 18 個修改和 5 個新增的地方。
