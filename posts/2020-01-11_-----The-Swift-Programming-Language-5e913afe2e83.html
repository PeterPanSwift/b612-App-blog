如何搜尋 The Swift Programming Language

想學習了解 Swift，Apple 官方的 The Swift Programming Language 是最棒的電子書。不過如果不懂某個語法，有什麼方法能快速跳到書裡介紹相關語法的段落呢 ?

由於 The Swift Programming Language 分成線上網頁跟 Books App 電子書版，因此以下我們分別介紹各自搜尋的方法。

搜尋線上的電子書網頁

The Swift Programming Language 的線上網頁如下。

About Swift - The Swift Programming Language (Swift 5.1)
Swift is a fantastic way to write software, whether it's for phones, desktops, servers, or anything else that runs…docs.swift.org

很可惜的，網頁上沒有提供搜尋的功能。

不過沒關係，我們可以找 Google 幫忙。

比方我們想查詢 dynamicMemberLookup，因此我們在 Google 輸入dynamicMemberLookup，然後利用 site 設定從 The Swift Programming Language 的網址 docs.swift.org/swift-book/ 搜尋。

dynamicMemberLookup site:docs.swift.org/swift-book/

如下圖所示，我們很快地找到 dynamicMemberLookup 在書裡出現的地方。

點選第一個連結 Attributes — The Swift Programming Language (Swift 5.1)，然後利用 cmd + f 搜尋 dynamicMemberLookup，馬上就能跳到網頁裡介紹 dynamicMemberLookup 的段落。

搜尋 Books App 的電子書

我們也可以從 Books App 下載 The Swift Programming Language。

‎The Swift Programming Language (Swift 5.1)
‎Swift is a programming language for creating iOS, macOS, watchOS, and tvOS apps. Swift builds on the best of C and…books.apple.com

比網站方便許多，Books App 本身即提供搜尋功能，點選右上方的放大鏡即可搜尋 dynamicMemberLookup。
