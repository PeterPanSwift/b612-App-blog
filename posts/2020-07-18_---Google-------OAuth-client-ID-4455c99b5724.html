取得 Google 登入需要的 OAuth client ID

串接 Google 登入時，我們需要搭配 OAuth client ID。接下來彼得潘將示範如何取得 OAuth client ID。

連到 Google Developers Console

Google Cloud Platform
Google Cloud Platform lets you build, deploy, and scale applications, websites, and services on the same infrastructure…console.developers.google.com

建立 Project

點選 CREATE PROJECT。

輸入專案名稱後點選 CREATE。

設定 OAuth consent screen

切換到 OAuth consent screen 頁面。

選擇 External 後按 CREATE。

輸入 Application name 後按 Save。

建立 Credentials

切換到 Credentials 頁面。

點選 CREATE CREDENTIALS。

選擇 OAuth client ID。

選擇 iOS。

輸入 Bundle ID 後按 CREATE。

取得 Client ID
