使用 Notion AI 提升程式可讀性

最近筆記軟體 Notion 推出強大的 AI 功能，它也像 ChatGTP 一樣可以幫我們寫程式。以下我們將介紹利用 Notion Web Clipper 將 GitHub 上的程式加入 Notion，然後使用 Notion AI 提升程式可讀性。

  • 註冊 Notion 帳號。
  • 安裝 Notion Web Clipper。
  • 將 GitHub 上的程式加到 Notion。
  • 使用 Notion AI 提升程式可讀性。

ps: Notion AI 有時會給錯的程式，不過它的英文不錯，因此可以請它幫忙優化程式裡 type / property / function / variable 的命名。

註冊 Notion 帳號

Notion AI
Notion AI is available to all users. You can prompt AI using your space bar, highlighting text and selecting “Ask AI”…www.notion.so

成功註冊後，我們即可免費使用 AI 功能。不過它有次數限制，達到上限後會出現以下畫面，必須付費才能繼續使用，月費的方案是一個月 10 美金。

安裝 Notion Web Clipper

Notion Web Clipper 是瀏覽器的外掛，它可以幫我們快速將網頁加到 Notion，待會我們將利用它將 GitHub 的程式加到 Notion，然後使用 Notion AI 提升程式可讀性。

Notion Web Clipper for Chrome, Safari, Firefox, and mobile
Save any web page on Notion. Annotate and organize to your heartʼs content.www.notion.so

以下彼得潘點選 Install for Safari。

點選 GET 下載。

下載好後點選 OPEN。

點選 Quit and Open Safari Extensions Preferences。

勾選 Safari Extensions 頁面的 Notion Web Clipper。

成功安裝後，瀏覽器的網址列將出現 Notion Web Clipper 的 N icon，點擊即可將網頁加到 Notion。

將 GitHub 上的程式加到 Notion

以下彼得潘連到某個學生 GitHub 上的程式作業，科幻電影選擇題的 iOS App。下圖為 App 專案裡的 ViewController.swift。不過此頁面還包含了程式以外的資訊，我們只想將程式加入 Notion，因此請點選下圖的 Raw，顯示程式的原始檔。

點選 Raw 後網頁乾淨許多，只剩下單純的程式碼。

點擊網址列旁的 N，將網頁內容加入 Notion。（ps: 點選 N 前記得先登入 Notion，如此它才能將資料存入我們的 Notion 帳號。)

若是第一次點擊 N，會先出現以下視窗。請先點擊 Close popup，然後再點擊 N。

點擊 Always Allow on Every Website。

完成以上設定後，之後只要點擊 N 即可將網頁加入 Notion。

從下圖的紅色框框選擇網頁要存入的筆記。

彼得潘選擇下圖的 Swift 程式優化。

點選 Save page。

點選 Open in Notion 打開存入的 Notion 頁面。

Cool，ViewController.swift 的程式碼成功顯示在 Notion 筆記裡。

使用 Notion AI 提升程式可讀性

點選程式碼右上的 …。

點選 Ask AI。

輸入問題。彼得潘想提升程式可讀性，優化程式的命名，因此輸入以下問題。

Make the code more readable and follow Swift's naming conventions. Please keep the strings in the program and do not change them to English.

Notion AI 果然不是蓋的，很快地幫我們寫出改善後的程式。不過它可能無法一次寫完，此時我們可點選 Continue writing 麻煩它繼續寫。

當 AI 繼續時，有時會額外輸入 _Continued from previous message_，之後記得要將它移除.

AI 回答完後，我們可點選 Insert below，將回答顯示在原本的程式下方，或是點選 Replace selection 取代原本的程式。
