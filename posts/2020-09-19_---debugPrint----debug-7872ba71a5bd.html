利用 debugPrint 幫助 debug

撰寫 Swift 程式時我們習慣利用 print 列印東西，幫助我們 debug。其實 print 還有另一個叫 debugPrint 好兄弟。debugPrint 人如其名，它專精 debug，因此有時可以印出更方便我們 debug 的資訊。

Apple Developer Documentation
Edit descriptiondeveloper.apple.com

接下來讓我們看看 debugPrint 幾個應用的實例吧。

列印 escape character，比方換行符號

let lyrices = "我愛妳這樣深\n日月星辰可為證"
debugPrint(lyrices)

使用 debugPrint 可以清楚看到 escape character \n，相反的，使用 print 印列時無法看到 \n。

列印字串

let number = "207"
debugPrint(number)

使用 debugPrint 可以看到 207 是字串，使用 print 只會看到數字，看不出 207 是字串。
