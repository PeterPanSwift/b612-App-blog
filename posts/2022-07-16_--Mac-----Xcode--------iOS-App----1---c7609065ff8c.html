學習開發 iOS App 就像爬山，要經歷很多很多步。不過沒關係，只要勇敢踏出第一步，然後一步一步前進，總有一天會抵達目的地。

以下我們分別介紹正式版跟 beta 版的安裝方法。

安裝 Xcode

  • 正式版

從 Mac 的 App Store 下載，是較穩定的版本，問題較少，不過缺少 beta 版酷炫的新功能。

點選 Download Xcode from the Mac App Store。

點選 Get 安裝 Xcode。

  • beta 版

點選 View available Xcode betas。

點選 Applications 下的 Xcode 26.2 beta，Mac 若是 Apple 晶片請選擇 Apple silicon，Intel 晶片選擇 Universal。

下載完成後雙擊下載的 xip 檔解壓縮得到 Xcode-beta App，將 Xcode 搬移到 Applications 資料夾。

啟動 Xcode，安裝 iOS simulator(模擬器)

啟動 Xcode 有三個方法。

  • 方法 1: 從 App Store 安裝的話，可以點選 Open 啟動 Xcode。
  • 方法 2: 點選螢幕右上的放大鏡(快速鍵 cmd + 空白)，輸入 xcode 搜尋，然後點選 Xcode 啟動 App。
  • 方法 3: 從 Apps 啟動 Xcode。

啟動 Xcode 後點選 Agree。

勾選 macOS、iOS 和 Predictive Code Completion Model，點選 Download & Install。

安裝 iOS Simulator 和 Predictive Code Completion Model。

出現 Xcode 的 Welcome 視窗，點選 Create New Project 開始我們的 App 開發旅程。

ps: 如果 iOS Simulator 安裝失敗，可參考以下連結安裝。

新增 Apple 平台(Platforms)的模擬器
Xcode 提供各種平台(platforms)的模擬器方便我們測試 App，不過為了節省空間，iOS、watchOS、tvOS & visionOS 模擬器都要另外下載安裝。apppeterpan.medium.com
