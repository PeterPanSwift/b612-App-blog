使用 map 搭配 KeyPath 印出 array 成員的某個 property

開發 Swift 程式時，為了研究問題，有時我們會印出 array 成員的某個 property。

struct Baby {
    var name: String
    var age: Int
}

let babies = [
    Baby(name: "Peter", age: 1),
    Baby(name: "John", age: 2),
    Baby(name: "Mary", age: 3),
]

以 array babies 為例，若想知道每個 baby 的 name，一般我們會用 for 或 forEach 的方式列印。

for baby in babies {
    print(baby.name)
}

babies.forEach { print($0.name) }

不過若想用更精簡的寫法，則可使用 map 搭配 KeyPath 的寫法。

print(babies.map(\.name))
Swift 方便存取 property 的 KeyPath & KeyPath as Functions
Swift 的 KeyPath 可以指定某個 property 或 subscript，方便我們存取資料的 property 或 subscript，接下來就讓我們以 Song 為例，看看它特別的地方吧。medium.com

結果。

["Peter", "John", "Mary"]

KeyPath 也可以串聯，因此印出每個 baby 的 toy 的 name 可以用 print(babies.map(\.toy.name))。

struct Baby {
    var name: String
    var age: Int
    var toy: Toy
}

struct Toy {
    var name: String
}

let babies = [
    Baby(name: "Peter", age: 1, toy: Toy(name: "Ball")),
    Baby(name: "John", age: 2, toy: Toy(name: "Train")),
    Baby(name: "Mary", age: 3, toy: Toy(name: "Robot")),
]

print(babies.map(\.toy.name))

結果。

["Ball", "Train", "Robot"]

如果再複雜一點，想印出每個 baby 的每個 toy 的 name，可以用 print(babies.flatMap(\.toys).map(\.name))，使用 flatMap 的目的是取得所有 toy，得到的資料型別是 [Toy]，若是使用 map(\.toys)，得到的型別會是 [[Toy]]。

struct Baby {
    var name: String
    var age: Int
    var toys: [Toy]
}

struct Toy {
    var name: String
}

let babies = [
    Baby(name: "Peter", age: 1, toys: [
        Toy(name: "Ball"),
        Toy(name: "Car"),
    ]),
    Baby(name: "John", age: 2, toys: [
        Toy(name: "Train"),
    ]),
    Baby(name: "Mary", age: 3, toys: [
        Toy(name: "Robot"),
    ]),
]

print(babies.flatMap(\.toys).map(\.name))

結果。

["Ball", "Car", "Train", "Robot"]
