查詢 macOS & Xcode 版本和 Mac 機型的方法

查詢 macOS 版本

  • 從 About This Mac。

點選左上的 App Icon，點選 About This Mac。

  • 從 Terminal 輸入 sw_vers。

打開 Terminal App。

輸入 sw_vers。

範例結果。

ProductName:  macOS
ProductVersion:  26.2
BuildVersion:  25C5031i
  • 從 Spotlight 點選 About This Mac。
  • 從 System Settings 的 About 頁面。

查詢 Xcode 版本

  • 從 Xcode menu 點選 Xcode > About Xcode。
  • 從 Terminal 輸入 xcodebuild -version。

範例結果。

Xcode 26.2
Build version 17C5013i
  • 從 Xcode 啟動後顯示的視窗。

查詢 Mac 機型

  • 從 About This Mac。

點選左上的 App Icon，點選 About This Mac。

  • 從 Terminal 輸入 system_profiler SPHardwareDataType | grep "Chip"。

範例結果。

Chip: Apple M3 Max

同時查詢 macOS & Xcode 版本和 Mac 機型

從 Terminal 輸入 sw_vers && xcodebuild -version && system_profiler SPHardwareDataType | grep "Chip"。

範例結果。

ProductName:  macOS
ProductVersion:  26.2
BuildVersion:  25C5031i
Xcode 26.2
Build version 17C5013i
      Chip: Apple M3 Max
