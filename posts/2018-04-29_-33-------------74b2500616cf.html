#33 顯示時間的正妹帥哥時計

目的: 學習 Date 和 Timer。

搭配正妹帥哥或自己喜歡的圖片，在畫面上顯示時間。

基本功能:

1 點選按鈕顯示時間。

2 每秒更新時間。

時間轉字串的程式參考

let now = Date()
let formatter = DateFormatter()
formatter.dateFormat = "HH:mm\nyyyy/MMM/dd E"
let time = formatter.string(from: now)

定時觸發 function 的方法。

利用 Timer 的 scheduledTimer(withTimeInterval:repeats:block:)。

scheduledTimer(withTimeInterval:repeats:block:) — Timer | Apple Developer Documentation
Creates a timer and schedules it on the current run loop in the default mode.developer.apple.com

程式範例參考

Timer.scheduledTimer(withTimeInterval: 1, repeats: true) { (_) in
   print("每秒印一次")
