macOS 叫出右鍵選單(context menu)的快速鍵 control + return

從 macOS 15 開始，不喜歡操作觸控板和滑鼠的朋友終於可以用鍵盤的快速鍵叫出右鍵選單了 !

叫出右鍵選單的快速鍵是 control + return，以下兩種開發 iOS App 常見的操作皆可完全從鍵盤操作。

  • SwiftUI 的 embed。

點選 Text，按下 control + return 叫出右鍵選單點選 Embed in VStack。

VStack {
    Text("draw me a sheep")
}
  • 新增檔案。

點選 Project navigator 的檔案，按下 control + return 叫出右鍵選單點選 New File from Template。

聊完了叫出右鍵選單的快速鍵，最後要感謝一下 AI，是 AI 告訴彼得潘的，而且 AI 連 control + return 的出處都知道，Apple 在 WWDC24 影片 What’s new in AppKit 的 8:44 ~ 9:10 提到。

若想調整右鍵選單的快速鍵，方法如下。

從 System Settings 搜尋 keyboard sho，點選側邊欄的 Keyboard shortcuts。

點選側邊欄的 Keyboard，雙擊 Show contextual menu 右邊的按鍵符號輸入新的快速鍵。
