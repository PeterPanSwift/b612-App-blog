利用 spacing 小於 0 的 stack view 產生元件重疊效果

利用 stack view，我們可以輕易地讓多個元件水平或垂直排列，並設定 spacing 讓元件間保持一定間距，比方以下金庸作品的呈現。

正常情況下，代表間距的 Spacing 都是大於等於 0，那如果小於 0 呢 ? 負數的間距會怎樣呢 ?

就讓我們勇敢試試吧。當間距設為 -20 時，將產生元件重疊的特別效果。如下圖所示，射鵰和‎神雕重疊了 20 point 的寬度，而笑傲也不甘寂寞地和射鵰重疊了 20 point 呢。
