利用 SwiftUI 的 background(_:in:fillStyle:) 設定背景，用顏色、漸層、圖片填滿形狀

開發 iOS App 時，我們時常會替元件加上某種樣式的背景，比方背景為黃色圓角的按鈕。從 iOS 15 開始，利用 modifier background(_:in:fillStyle:) 可以更方便地設定用顏色、漸層、圖片填滿形狀的背景。

以下我們利用 background(_:in:fillStyle:) 實現 4 種常見的元件背景。

  • 用顏色填滿形狀。
  • 用漸層填滿形狀。
  • 用圖片填滿形狀。
  • 自訂形狀。

用顏色填滿形狀

background(.yellow, in: .rect(cornerRadius: 10)) 設定背景為黃色，形狀是圓角矩形，background(.yellow, in: .capsule) 設定背景為膠囊形。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(.tame)
                .resizable()
                .scaledToFit()
            Text("請馴養我\nPlease tame me")
                .font(.largeTitle)
                .padding()
                .background(.yellow, in: .rect(cornerRadius: 10))
            Label("請馴養我", systemImage: "star")
                .font(.largeTitle)
                .padding()
                .background(.yellow, in: .capsule)
        }
    }
}

用圖片填滿形狀

呼叫 .image(Image(.pattern)) 生成型別 ImagePaint 的圖片，它的宣告如下。

static func image(
    _ image: Image,
    sourceRect: CGRect = CGRect(x: 0, y: 0, width: 1, height: 1),
    scale: CGFloat = 1
) -> ImagePaint
image(_:sourceRect:scale:) | Apple Developer Documentation
A shape style that fills a shape by repeating a region of an image.developer.apple.com
struct ContentView: View {
    var body: some View {
        VStack {
            Image(.tame)
                .resizable()
                .scaledToFit()
            Text("請馴養我\nPlease tame me")
                .font(.largeTitle)
                .padding()
                .background(.image(Image(.pattern)), in: .rect(cornerRadius: 10))
        }
    }
}

圖片填滿形狀時會以原本的大小填滿，所以原圖比較大時將只顯示一部分，超出的地方被切掉。若想圖片顯示更多區域，可用參數 scale 縮放圖片，比方 0.5 表示圖片變成一半的大小。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(.tame)
                .resizable()
                .scaledToFit()
            Text("請馴養我\nPlease tame me")
                .font(.largeTitle)
                .padding()
                .background(.image(Image(.pattern), scale: 0.5), in: .rect(cornerRadius: 10))
        }
    }
}

自訂形狀

剛剛的例子都是搭配內建的常用形狀，不過我們也可以自己定義型別。以下例子用 IrregularShape 定義不規則的五邊形。

struct IrregularShape: Shape {
    func path(in rect: CGRect) -> Path {
        var path = Path()
        path.move(to: CGPoint(x: 0, y: 0))
        path.addLine(to: CGPoint(x: rect.width, y: 0))
        path.addLine(to: CGPoint(x: rect.width, y: rect.height * 1))
        path.addLine(to: CGPoint(x: rect.width * 0.5, y: rect.height))
        path.addLine(to: CGPoint(x: 0, y: rect.height * 0.8))
        path.closeSubpath()
        return path
    }
}

struct ContentView: View {
    var body: some View {
        VStack {
            Image(.tame)
                .resizable()
                .scaledToFit()
            Text("請馴養我\nPlease tame me")
                .font(.largeTitle)
                .padding()
                .background(.yellow, in: IrregularShape())
        }
    }
}

若想在呼叫 background(_:in:fillStyle:) 時更簡單地的傳入形狀，可用 extension 定義型別屬性，傳入時只要打 .irregularShape 即可。

struct IrregularShape: Shape {
    func path(in rect: CGRect) -> Path {
        var path = Path()
        path.move(to: CGPoint(x: 0, y: 0))
        path.addLine(to: CGPoint(x: rect.width, y: 0))
        path.addLine(to: CGPoint(x: rect.width, y: rect.height * 1))
        path.addLine(to: CGPoint(x: rect.width * 0.5, y: rect.height))
        path.addLine(to: CGPoint(x: 0, y: rect.height * 0.8))
        path.closeSubpath()
        return path
    }
}

extension Shape where Self == IrregularShape {
    static var irregularShape: IrregularShape {
        return IrregularShape()
    }
}

struct ContentView: View {
    var body: some View {
        VStack {
            Image(.tame)
                .resizable()
                .scaledToFit()
            Text("請馴養我\nPlease tame me")
                .font(.largeTitle)
                .padding()
                .background(.yellow, in: .irregularShape)
        }
    }
}
