利用 SwiftUI scaleEffect(_:anchor:) 的 anchor 控制元件放大時固定的位置

SwiftUI 的 scaleEffect 可以方便地將元件變大，例如以下例子按下變大吧會將圖片放大 3 倍，按下還原吧會將圖片還原。

struct ContentView: View {
    @State private var scale: Double = 1
    
    var body: some View {
        VStack {
            Image(.coding)
                .resizable()
                .scaledToFit()
                .frame(width: 100)
                .scaleEffect(scale)
            HStack {
                Button("變大吧") {
                    scale = 3
                }
                Button("還原吧") {
                    scale = 1
                }
            }
            .buttonStyle(.borderedProminent)
        }
    }
}

不過放大時元件預設將維持它的中心點，因此如下圖所示，放大後的圖片跟按鈕重疊了。

其實 scaleEffect 還有一個 anchor 參數可控制元件放大時固定的位置。以下例子 scaleEffect(scale, anchor: .bottom) 將讓元件固定下方的位置，因此放大時圖片不再和按鈕重疊。

struct ContentView: View {
    @State private var scale: Double = 1
    
    var body: some View {
        VStack {
            Image(.coding)
                .resizable()
                .scaledToFit()
                .frame(width: 100)
                .scaleEffect(scale, anchor: .bottom)
            HStack {
                Button("變大吧") {
                    scale = 3
                }
                Button("還原吧") {
                    scale = 1
                }
            }
            .buttonStyle(.borderedProminent)
        }
    }
}

透過 anchor 還可生成特別的動畫效果，以下例子的 anchor bottom 將讓圖片變大由下而上生成。

struct ContentView: View {
    @State private var scale: Double = 1
    
    var body: some View {
        VStack {
            Image(.coding)
                .resizable()
                .scaledToFit()
                .frame(width: 100)
                .scaleEffect(scale, anchor: .bottom)
                .animation(.easeInOut(duration: 3), value: scale)
            HStack {
                Button("變大吧") {
                    scale = 3
                }
                Button("還原吧") {
                    scale = 1
                }
            }
            .buttonStyle(.borderedProminent)
        }
    }
}
