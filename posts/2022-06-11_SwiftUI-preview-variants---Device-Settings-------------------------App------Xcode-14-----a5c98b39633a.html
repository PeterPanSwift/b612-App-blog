SwiftUI preview variants & Device Settings: 方便查看不同顏色，方向 & 文字大小設定下的 App 畫面 — Xcode 14 新功能

從 Xcode 14 開始，利用全新的 SwiftUI preview variants & Device Settings 功能，我們可以方便地查看不同顏色，方向 & 文字大小設定下的 App 畫面。

以下就讓我們以 Penny 經典的第一張專輯主打歌習慣這樣為例說明吧。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(.習慣這樣)
                .clipShape(.circle)
                .shadow(color: .primary, radius: 10)
            Text("習慣這樣的距離\n習慣這樣的你的背影\n習慣這樣 這樣的你\n習慣這樣")
                .font(.title)
                .bold()
                .foregroundStyle(.orange.gradient)
        }
    }
}

設定 SwiftUI preview variants

點擊下方的 Variants 按鈕。

可選擇 Color Scheme Variants，Orientation Variants & Dynamic Type Variants。

Color Scheme Variants

顯示 light mode & dark mode 的 App 畫面。

點擊畫面後還能讓畫面變成主角放大顯示，比方點選 dark mode 畫面的結果如下。

Orientation Variants

顯示 Portrait(直向) & Landscape(橫向) 的 App 畫面。

Dynamic Type Variants

顯示依據 Dynamic Type 調整文字大小的 App 畫面，從最小的 X Small 到最大的 AX 5。

調整顏色，方向 & 文字大小的 Device Settings

從 Device Settings 也可以調整顏色，方向 & 文字大小的設定，而且還能結合剛剛的 Variants。

以下我們試試觀看搭配 dark mode 的 Dynamic Type Variants。

先將 Variants 選擇 Dynamic Type Variants，然後點選 Device Settings。

打開 Color Scheme 的開關，選擇 Dark appearance。

Cool，我們成功看到了 dark mode 下各種文字大小的 App 畫面。
