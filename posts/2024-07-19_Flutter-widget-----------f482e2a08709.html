Flutter widget 設定邊框的各種方法

開發 Flutter App 畫面時，我們時常看到元件加上邊框的設計，以下我們分別介紹 Button、TextField 和其它 widget 設定邊框的方法。

Button 的邊框

OutlinedButton 預設是膠囊形邊框。

OutlinedButton(
  onPressed: () {},
  child: const Text('button'),
)

利用 RoundedRectangleBorder 調整 OutlinedButton 的圓角弧度，BorderSide 調整邊框的寬度和顏色。

OutlinedButton(
     onPressed: () {},
     style: OutlinedButton.styleFrom(
         shape: RoundedRectangleBorder(
             borderRadius: BorderRadius.circular(10),
         ),
         side: const BorderSide(
             width: 5,
             color: Colors.orange,
         ),
     ),
     child: const Text('button'),
)

TextField 的邊框

InputDecoration 的 border 設定 OutlineInputBorder。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return const Scaffold(
      body: Center(
        child: Padding(
          padding: EdgeInsets.all(8.0),
          child: TextField(
            decoration: InputDecoration(
              hintText: 'name',
              border: OutlineInputBorder(),
            ),
          ),
        ),
      ),
    );
  }
}

利用 OutlineInputBorder 的 borderRadius 調整圓角弧度。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return const Scaffold(
      body: Center(
        child: Padding(
          padding: EdgeInsets.all(8.0),
          child: TextField(
            decoration: InputDecoration(
              hintText: 'name',
              border: OutlineInputBorder(
                borderRadius: BorderRadius.all(
                  Radius.circular(20),
                ),
              ),
            ),
          ),
        ),
      ),
    );
  }
}

其它 widget 的邊框

利用 Container 搭配 BoxDecoration 的 border 實現邊框。

  • 矩形邊框。
Container(
  decoration: BoxDecoration(
    border: Border.all(
      color: Colors.black,
      width: 4,
    ),
  ),
  padding: const EdgeInsets.all(10),
  child: const Text('Hello SwiftUI'),
)
  • 圓角矩形邊框。
Container(
  decoration: BoxDecoration(
    border: Border.all(
      color: Colors.black,
      width: 4,
    ),
    borderRadius: BorderRadius.circular(10),
  ),
  padding: const EdgeInsets.all(10),
  child: const Text('Hello SwiftUI'),
)
class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Padding(
          padding: const EdgeInsets.all(8.0),
          child: Container(
            clipBehavior: Clip.antiAlias,
            decoration: BoxDecoration(
              border: Border.all(
                color: Colors.black,
                width: 10,
              ),
              borderRadius: BorderRadius.circular(10),
            ),
            child: Image.network(
                'https://m.media-amazon.com/images/M/MV5BZTZlZWM0MTEtMWFkZC00MDU4LTgwZTgtNDM1YjQwZWZjYzdhXkEyXkFqcGdeQXVyMDM2NDM2MQ@@._V1_.jpg'),
          ),
        ),
      ),
    );
  }
}
  • 圓形邊框。
Container(
  alignment: Alignment.center,
  width: 150,
  height: 150,
  decoration: BoxDecoration(
    shape: BoxShape.circle,
    border: Border.all(
      color: Colors.black,
      width: 4,
    ),
  ),
  padding: const EdgeInsets.all(10),
  child: const Text('Hello SwiftUI'),
)

除了搭配 BoxDecoration 的 border 實現邊框，Container 也可以用 padding 做出邊框效果，例如以下例子。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Padding(
          padding: const EdgeInsets.all(8.0),
          child: Container(
            color: Colors.black,
            padding: const EdgeInsets.all(10),
            child: Image.network(
                'https://m.media-amazon.com/images/M/MV5BZTZlZWM0MTEtMWFkZC00MDU4LTgwZTgtNDM1YjQwZWZjYzdhXkEyXkFqcGdeQXVyMDM2NDM2MQ@@._V1_.jpg'),
          ),
        ),
      ),
    );
  }
}
