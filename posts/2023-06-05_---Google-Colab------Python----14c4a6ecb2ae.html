如果說 programiz 是舊愛，那麼 Google Colab(Colaboratory) 就是彼得潘的新歡。最近認識她後，彼得潘發現它有許多方便和好用的地方，以下讓我們一步步認識她吧。

連到 Colab 網站

如果尚未登入 Google 帳號，請先點選右上的 Sign in 登入。

新增神奇的 Notebook

在 Colab 網站我們可以打開神奇的 notebook(筆記本)，它比只能寫字畫畫的筆記本強多了，它可以拿來寫程式和執行程式。

連到網站後，會先跳出以下視窗，方便我們快速開啟之前建立的 notebook 或新增 notebook。

點選右下的 New notebook，建立我們人生第一本 Python 程式 notebook。

ps: 從上方選單的 File > New notebook 也可以新增 notebook。

設定 Notebook 的名字

點選上方的檔名可以幫 notebook 取個好聽的名字。

撰寫程式和執行程式

Notebook 主要由一個個框框(cell) 組成，一開始它會幫我們建立第一個 cell。

我們可以在 cell 裡輸入程式，然後按三角形或快速鍵(cmd + enter 或 ctrl + enter)執行，結果將顯示在下方。

在 cell 裡可輸入多行程式，輸入一百行也不是問題，比方以下計算平均分數的例子。

我們也可將程式分散在不同的 cell。點選上方的 + Code 可新增輸入程式的 cell(快速鍵是 cmd+M+B 或 ctrl+M+B)。

cell 間可互助合作，可以使用其它 cell 定義的程式。下圖第二個 cell 使用第一個 cell 的 math 和 english 計算平均分數。值得注意的，定義 math 和 english 的 cell 記得要先執行，這樣其它的 cell 執行時才能認得 math 和 english。

我們也可以一次執行全部的 cell，點選 Menu 的 Runtime > Run all。

刪除 cell 則可點選 cell 後點擊右上方選單的垃圾桶按鈕。

陪寫程式的可愛動物

寫程式常會有孤單寂寞冷的副作用。沒關係，Colab 貼心地提供可愛動物陪我們寫 code。呼喚可愛動物現身的方法如下。

點選右上方的設定按鈕。

切換到 Miscellaneous 分頁，選擇喜歡的小動物，目前有 Corgi(柯基犬)，Kitty(小貓) 和 Crab(螃蟹) 三種。

彼得潘打開 Kitty 開關後，現在寫程式再也不孤單了，可愛的小貓一直在上方走來走去陪著我們。

打程式的火花效果

點選右上方的設定按鈕，切換到 Miscellaneous 分頁，然後選擇 Many power(use with caution)。

內建許多常用的函式庫

除了寫簡單的程式，我們還可透過 Colab 開發複雜的功能，因為 Colab 幫我們裝好許多常用的函式庫，我們可以直接使用它們，實現網路爬蟲，讀取 Google 表單產生圖表，機器學習等功能。

比方以下例子我們在 Colab 執行爬蟲程式，印出 KKBOX 上戴佩妮歌詞後果自負的 JSON-LD 區塊。

加入呈現文字圖片的 text cell

除了用 code cell 寫程式，我們也可以加入 text cell，幫程式加上說明的文字圖片，將 notebook 變成一本精美的筆記本或教學文件。

點選 + Text 可新增 text cell，以下我們幫 GPT 4 用心改編的歌詞加上相關說明。

text cell 支援 markdown 語法，我們可以自由地輸入文字，圖片，連結，html 和 LaTeX。

編輯完後按 shift+enter 或 ctrl+enter 會儲存並離開編輯模式，一本精美的筆記本誕生了，癡心癡情癡狂癡迷都是為 Swift ~

分享 notebook

Colab 就像 Google 的其它服務一樣，提供了方便的分享功能。

我們可以點選右上的 Share 分享連結，朋友點選連結後不只能觀看內容，還可以修改程式做各種有趣的實驗。
