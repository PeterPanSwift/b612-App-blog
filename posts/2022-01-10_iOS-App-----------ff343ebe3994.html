iOS App 設定部分圓角的方法

開發 iOS App 時，我們時常會設定長方形的圓角。比起四個角都有圓角，只有部分圓角，比方左上和右下，則會複雜一點。

以下我們分別介紹 SwiftUI & UIKit 實現部分圓角的方法，目標實現左上 & 右下有圓角的圖片。

SwiftUI 範例

  • 利用 UnevenRoundedRectangle。(iOS 16 之後的版本)
struct ContentView: View {
    var body: some View {
        Image(.spiderMan)
            .resizable()
            .scaledToFill()
            .frame(width: 350, height: 280)
            .clipShape(
                UnevenRoundedRectangle(topLeadingRadius: 30, bottomTrailingRadius: 30)
            )
    }
}

使用 clipShape 將圖片裁切成特定形狀，用 UnevenRoundedRectangle 產生部分圓角的長方形。

以下是產生 UnevenRoundedRectangle 的另一種寫法，傳入 RectangleCornerRadii 設定圓角的位置。

struct ContentView: View {
    var body: some View {
        Image(.spiderMan)
            .resizable()
            .scaledToFill()
            .frame(width: 350, height: 280)
            .clipShape(
                UnevenRoundedRectangle(cornerRadii: RectangleCornerRadii(topLeading: 30, bottomTrailing: 30))
            )
    }
}

ps: 除了使用 clipShape 裁切形狀，我們也可以用 mask 遮罩成特定形狀。

struct ContentView: View {
    var body: some View {
        Image(.spiderMan)
            .resizable()
            .scaledToFill()
            .frame(width: 350, height: 280)
            .mask({
                UnevenRoundedRectangle(topLeadingRadius: 30, bottomTrailingRadius: 30)
            })
           
    }
}
  • 利用 UIBezierPath。(iOS 16 之前的版本)
struct RoundedCornerShape: Shape {
    let corner: UIRectCorner
    let radius: Double
    
    func path(in rect: CGRect) -> Path {
        let bezierPath = UIBezierPath(roundedRect: rect, byRoundingCorners: corner, cornerRadii: CGSize(width: radius, height: radius))
        return Path(bezierPath.cgPath)
    }    
}

struct ContentView: View {
    
    var body: some View {
        Image(.spiderMan)
            .resizable()
            .scaledToFill()
            .frame(width: 350, height: 280)
            .clipShape(RoundedCornerShape(corner: [.topLeft, .bottomRight], radius: 30))
    }
}

說明

  • 型別 RoundedCornerShape 定義部分圓角的長方形，產生 UIBezierPath 時傳入的參數 byRoundingCorners 控制圓角的位置。
  • 呼叫 clipShape，傳入 RoundedCornerShape 將 view 切成部分圓角。

利用 maskedCorners 設定部分圓角
想幫 iOS App 的元件設定圓角並不是太難的事，一般來說，只要以下兩行程式即可搞定，cornerRadius 設定圓角弧度，clipsToBounds 讓元件可以被切成圓角。medium.com
  • 方法 2: 將 mask 設成 CAShapeLayer

定義顯示部分圓角的 RoundedCornerImageView。

class RoundedCornerImageView: UIImageView {
    var corners: UIRectCorner
    var radius: Double
    
    init(image: UIImage?, corners: UIRectCorner, radius: Double) {
        self.corners = corners
        self.radius = radius
        super.init(image: image)
    }
    
    required init?(coder: NSCoder) {
        corners = .allCorners
        radius = 10
        super.init(coder: coder)
    }
  
    
    override func layoutSubviews() {
        super.layoutSubviews()
        
        let bezierPath =  UIBezierPath(roundedRect: bounds, byRoundingCorners: corners, cornerRadii: CGSize(width: radius, height: radius))
        let shapeLayer = CAShapeLayer()
        shapeLayer.path = bezierPath.cgPath
        layer.mask = shapeLayer
    }
}

從 storyboard 加入 image view，類別設為 RoundedCornerImageView。

由於 init?(coder:) 裡將 corners 設為 allCorners，radius 設為 10，因此加入 storyboard 的圖片預設將在四個頂點都有 radius 10 的圓角。

我們可從程式修改，比方將圖片連結 outlet imageView，然後在 viewDidLoad 設定 corners & radius。

class ViewController: UIViewController {
  @IBOutlet weak var imageView: RoundedCornerImageView!

  override func viewDidLoad() {
        super.viewDidLoad()
        
        imageView.corners = [.topLeft, .bottomRight]
        imageView.radius = 30
    }
}

我們也可以直接從程式生成 RoundedCornerImageView，搭配它的 init(image:corners:radius:)。

class ViewController: UIViewController {
   override func viewDidLoad() {
        super.viewDidLoad()
        
        let imageView = RoundedCornerImageView(image: UIImage(named: "spider man"), corners: [.topLeft, .bottomRight], radius: 30)
        imageView.contentMode = .scaleAspectFill
        imageView.translatesAutoresizingMaskIntoConstraints = false
        view.addSubview(imageView)
        imageView.widthAnchor.constraint(equalToConstant: 350).isActive = true
        imageView.heightAnchor.constraint(equalToConstant: 280).isActive = true
        imageView.centerXAnchor.constraint(equalTo: view.safeAreaLayoutGuide.centerXAnchor).isActive = true
        imageView.centerYAnchor.constraint(equalTo: view.safeAreaLayoutGuide.centerYAnchor).isActive = true
    }
}
