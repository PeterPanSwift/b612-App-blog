從 Xcode 搜尋下載 Sample Code

學習 iOS App 開發時，Apple 官方的 sample code 是滿值得參考的對象，畢竟是 Apple 寫的，一定比民間的大神可靠。接下來就讓我們學著從 Xcode 搜尋下載 sample code 吧。

1 點選 Documentation and API Reference，打開 Documentation 視窗。(或按快速鍵 cmd + shift + 數字0 )

2 點選左邊區塊的第二個 icon ，切換到 guide and sample code 頁面。

3 找到自己感興趣的 sample code 後，點選右邊頁面的 Open Project 即可下載專案。

下載完成後還會貼心地自動從 Xcode 打開專案呢 ! 專案預設將下載到 Downloads 資料夾下。
