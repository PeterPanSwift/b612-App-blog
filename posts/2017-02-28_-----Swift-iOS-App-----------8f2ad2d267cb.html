如何繳交 Swift iOS App 作業 ~ 大學生篇

繳交作業前的準備:
透過 FB 或 LINE 告訴我你的 Medium 帳號和本名，
名字超級無敵重要，
這樣我才能打分數呀。
( 彼得潘的FB: https://www.facebook.com/deeplove.pan
彼得潘的 LINE ID: deeplovepeterpan )

如何得知 Medium 帳號 ?
(1) 登入 Medium ，點選右上角的頭像照片後，點選 profile。

(2) profile 頁面網址裡 @ 開頭的字串即為你的帳號，比方以下例子，帳號為 crazyloveiphone。你可以傳給我帳號，也可以傳給我這串網址。

之後當你完成作業，即可透過在 Medium 發表文章的方式繳交作業。

繳交作業的步驟:

  1. 完成作業，在 Medium 發表文章，文章的 tag 要包含作業的名稱。文章內容至少包含以下四點:
    (1) App 操作的動畫影片 gif
    (2) GitHub 連結。( GitHub 上的專案頁面需附上 App 操作 gif 或圖片)
    (3) 特製的 App 畫面截圖。
    (4) 文字說明 (字數不限)
    (5) 重點程式碼講解。(如果有程式的話)

2. 文章發表後，點擊文章頁面下方的點點點按鈕，然後點選 Add to publication。( 注意: 要你自己寫的文章才能點選 Add to publication )

3. 勾選你參加課程的 publication，比方 台大 CS+X iOS App 程式設計。如果沒看到，表示我還沒有把你加入，你還沒有權限，請另外跟我聯絡。

5. 點擊 Submit story 完成上傳。

6. 此時我將收到你投稿的文章。針對你的作品，我可能會按讚或留言給你意見。如下圖為學生繳交的作業清單:

8. 想跟我討論作業，或是作業遇到問題，也歡迎透過 FB 或 LINE 跟我討論。
