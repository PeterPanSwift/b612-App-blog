利用 Xcode 的 Search package collections 搜尋找不到的套件

開發 iOS App 時，我們時常利用 import 加入套件。最近彼得潘發現當 import 的套件找不到時，Xcode 還會自動幫我們搜尋套件呢。

比方以下我們 import DotLottie，由於還沒加入 DotLottie 的 package，因此顯示 No such module。

點選紅色錯誤後出現 Search package collections 的提示。

點擊 Search 即可打開 Add Package 的視窗，而且 Xcode 還貼心地在搜尋框填入 DotLottie 搜尋套件。
