MaterialApp vs Scaffold
MaterialApp 和 Scaffold 是 Flutter 中的兩個重要的 widget，都與 Material Design 有關，但它們的功能和用途有所不同。下面是它們的主要差別：chat.openai.com

解法: 用 Scaffold 或 Material 包著 Text。

  • Scaffold
void main() {
  runApp(
    MaterialApp(
      home: Scaffold(
        body: Center(
          child: Text('I Am Rich'),
        ),
      ),
    ),
  );
}
  • Material
void main() {
  runApp(
    MaterialApp(
      home: Material(
        child: Center(
          child: Text('I Am Rich'),
        ),
      ),
    ),
  );
}

在 Flutter App 顯示專案裡的圖片
利用 Flutter 的 Image widget 我們可以方便地顯示專案裡的圖片。以下我們一步步示範如何將圖片加入 Flutter 專案，然後利用 Image widget 顯示圖片。apppeterpan.medium.com
