利用 SwiftUI 的 presentationDetents 和 presentationCornerRadius 控制頁面的高度和圓角弧度

當我們使用 SwiftUI 的 sheet 顯示新頁面時，預設它會顯示接近全螢幕的高度，背後露出一小塊前一頁的內容，例如以下例子 AlbumView 顯示 Penny 好聽的新專輯雙生火焰。

struct ContentView: View {
    @State private var showAlbumView = false
    
    var body: some View {
        Button("雙生火焰") {
            showAlbumView = true
        }
        .sheet(isPresented: $showAlbumView) {
            AlbumView()
        }
    }
}
struct AlbumView: View {
    var body: some View {
        ZStack {
            Color.black
                .ignoresSafeArea()
            VStack {
                Image(.twinFlame)
                    .resizable()
                    .scaledToFit()
                Text(
                    """
                    你啊 是我鏡像的靈魂嗎
                    總是反映出熟悉的想法 莫名又複雜
                    害怕 但還是忍不住一起玩耍
                    一分為二的確定在誰那
                    相愛相殺都注定要牽掛
                    """
                )
                .foregroundStyle(.white)
            }
        }
    }
}

從 iOS 16 開始，我們可以彈性地利用 presentationDetents 控制頁面的高度，以下我們示範各種常見的應用。

  • 指定螢幕一半的高度。
  • 指定某個螢幕比例的高度。
  • 指定高度的數字。
  • 指定多種高度。

指定螢幕一半的高度

presentationDetents 的宣告如下，參數的型別為 Set。

func presentationDetents(_ detents: Set<PresentationDetent>) -> some View
presentationDetents(_:) | Apple Developer Documentation
Sets the available detents for the enclosing sheet.developer.apple.com

當我們傳入 [.medium] 時，頁面將顯示螢幕一半的高度。

struct ContentView: View {
    @State private var showAlbumView = false
    
    var body: some View {
        Button("雙生火焰") {
            showAlbumView = true
        }
        .sheet(isPresented: $showAlbumView) {
            AlbumView()
                .presentationDetents([.medium])
        }
    }
}

按空白處或下拉可關掉 AlbumView 回到上一頁。

指定某個螢幕比例的高度

以 fraction 指定某個螢幕比例的高度，比方 .fraction(0.8) 表示 80% 的螢幕高度。

struct ContentView: View {
    @State private var showAlbumView = false
    
    var body: some View {
        Button("雙生火焰") {
            showAlbumView = true
        }
        .sheet(isPresented: $showAlbumView) {
            AlbumView()
                .presentationDetents([.fraction(0.8)])
        }
    }
}

指定高度的數字

以 height 指定高度的數字。

struct ContentView: View {
    @State private var showAlbumView = false
    
    var body: some View {
        Button("雙生火焰") {
            showAlbumView = true
        }
        .sheet(isPresented: $showAlbumView) {
            AlbumView()
                .presentationDetents([.height(250)])
        }
    }
}

指定多種高度

presentationDetents 的參數型別是 Set，因此我們可傳入多種高度，讓使用者自己拖曳想要的高度。

struct ContentView: View {
    @State private var showAlbumView = false
    
    var body: some View {
        Button("雙生火焰") {
            showAlbumView = true
        }
        .sheet(isPresented: $showAlbumView) {
            AlbumView()
                .presentationDetents([.medium, .large])
        }
    }
}

比方以 [.medium, .large] 顯示 AlbumView 時，它將先顯示最小的高度，所以它會先顯示 medium 的高度。

使用者可自己拖曳調整高度。剛剛的例子傳入 [.medium, .large]，因此從一半高度往上拖曳時將變成顯示 large 的高度。

什麼是 Detent ?

presentationDetents 的 detent 在日常生活代表機械卡點的意思，以轉動風扇旋鈕為例，我們會聽到「喀、喀」的聲音，那就是每個檔位之間的 detent。

因此 presentationDetents 的 detents 可以想成我們可以傳入多個「固定點」指定頁面的高度，比方傳入 [.medium, .large] 表示頁面顯示時可停留在兩種高度，一半高度的 medium 和接近全螢幕的 large。

One more thing: 調整頁面的圓角弧度

透過 presentationCornerRadius 控制頁面的圓角弧度。

presentationCornerRadius(_:) | Apple Developer Documentation
Requests that the presentation have a specific corner radius.developer.apple.com
struct ContentView: View {
    @State private var showAlbumView = false
    
    var body: some View {
        Button("雙生火焰") {
            showAlbumView = true
        }
        .sheet(isPresented: $showAlbumView) {
            AlbumView()
                .presentationDetents([.medium])
                .presentationCornerRadius(50)
        }
    }
}
