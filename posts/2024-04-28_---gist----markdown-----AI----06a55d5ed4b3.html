使用 gist 存放 markdown 格式的 AI 回答

現在 AI 已成為我們日常生活中不可或缺的好幫手，彼得潘平常在程式的開發、學習和教學常常跟 AI 請益，AI 也會給予我許多不錯的回答。

彼得潘想將 AI 這些優質的回答整理在一個地方，除了自己方便查詢，也能讓大家 Google 搜尋程式問題時快速找到不錯的回答。

目前想到免費的 gist 是個不錯的選擇，尤其現在最強的 3 個 AI GPT、Claude 和 Gemini 都以 markdown 格式回答，而 gist 剛好也特別適合存放和呈現 markdown。

以下彼得潘以 Claude 為例，示範如何使用 gist 存放 markdown 格式的 AI 回答。

複製 AI 的回答

AI 回答後，點選下方的 copy 複製 AI 的回答，複製的內容將是 markdown 格式。

連到 gist

記得要先登入 Github 帳號。

Discover gists
GitHub Gist: instantly share code, notes, and snippets.gist.github.com

貼上 AI 的回答

如下圖所示，剛剛複製的 AI 回答果然是 markdown 格式。

在 Filename 的框框輸入以 .md 結尾的檔名

markdown 的副檔名是 md，輸入 .md 結尾的檔名後，gist 即可美美地呈現 markdown 的內容。

點選 Preview 可預覽 markdown 呈現的模樣，點選 Edit new file 可編輯 markdown。

輸入 gist 的描述

建立 gist

選擇 Create secret gist 或 Create public gist，public 表示任何人都可以看到，也能被搜尋引擎找到，secret 表示只有取得 gist 網址的人能看到。

在此彼得潘點選 Create public gist，讓它成為公開的 gist。

分享 gist 網址

https://gist.github.com/PeterPanSwift/a89ce578a2f022c8b497490cfc010e7c

在 Blog 加入 gist

Blog 也很適合加入 gist 分享的內容，比方彼得潘在 Medium 貼上 gist 網址按 enter 後，它自動美美地呈現 markdown 的完整內容。
