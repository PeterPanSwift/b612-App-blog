SwiftUI 辨識 gesture(手勢)的三種寫法

開發 SwiftUI App 時，透過 gesture modifier，我們可以在 view 上添加 gesture，快速做出手勢辨識的功能。

寫法 1: 在 gesture( ) 裡生成傳入的 Gesture

我們可以在 gesture( ) 裡生成傳入的 Gesture。比方我們生成辨識拖曳手勢的 DragGesture。在拖曳過程中，它會觸發 onChanged modifier。當拖曳結束時，它會觸發 onChanged modifier。從參數 value 我們可讀取拖曳的相關資訊，比方拖曳的距離。

struct DragView: View {
    
    var body: some View {
        Image("peter")
            .gesture(
                DragGesture()
                    .onChanged({ value in
                        print("change", value)
                    })
                    .onEnded({ value in
                        print("end", value)
                    })
            )
    }
}

不過這樣的寫法將讓程式變得凌亂，程式會有多層的縮排，因此通常我們會採用另外兩種寫法。

寫法 2: 宣告 computed property 或 method 產生 gesture

  • 宣告產生 gesture 的 computed property
struct DragView: View {
    
    var dragGesture: some Gesture {
        DragGesture()
            .onChanged({ value in
                print(value)
            })
            .onEnded({ value in
                print(value)
            })
    }
    
    var body: some View {
        Image("peter")
            .gesture(dragGesture)
    }
}
  • 宣告產生 gesture 的 method
struct DragView: View {
    
    func dragGesture() -> some Gesture {
        DragGesture()
            .onChanged({ value in
                print(value)
            })
            .onEnded({ value in
                print(value)
            })
    }
    
    var body: some View {
        Image("peter")
            .gesture(dragGesture())
    }
}

寫法 3: 在 body 裡宣告儲存 gesture 的常數

struct DragView: View {
    
    var body: some View {
        let dragGesture = DragGesture()
            .onChanged({ value in
                print(value)
            })
            .onEnded({ value in
                print(value)
            })
        
        return Image("peter")
            .gesture(dragGesture)
    }
}
