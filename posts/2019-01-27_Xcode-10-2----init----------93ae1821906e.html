Xcode 10.2 修正 init 的自動完成問題了

開發 iOS App 時，我們時常用型別( ) 生東西，比方以下例子。

struct Song {
   var name: String
   var singer: String
}
let song = Song(name: "真的傻", singer: "徐佳瑩")

然而在專案裡寫程式時，當我們將 Song 單獨定義在 Song.swift，然後在 view controller 裡輸入 Song( 時，自動完成有時會失靈，明明已經打完 Song(，自動完成的提示卻還不明白我們的心意，不明白我們想生出一首感人的情歌。

在舊版的 Xcode，我們有兩種解法，一種是利用 .init 輸入。

另一種則是第一次先不靠自動完成，自己辛苦地輸入。之後在同一個檔案裡再次輸入 Song( 時 ，自動完成將恢復正常，順利找到 Song 的 init。

不過這些都是過去式了 ! 新版的 Xcode 10.2 修正了這個問題，現在我們第一次輸入 Song( 生東西時就可以看到令人感動的 init 自動完成了。
