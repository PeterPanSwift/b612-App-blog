使用 shadow offset 實現 SwiftUI / UIKit / Flutter 的立體文字和元件

設計 App 畫面時，我們常常會透過右下方的陰影讓文字和元件看起來更為立體，以下我們介紹如何透過 shadow offset 實現 SwiftUI / UIKit / Flutter 的立體文字和元件。

SwiftUI

透過 shadow modifier 實現陰影立體文字效果，參數 color 設定陰影的顏色，radius 設定陰影模糊的程度，x 和 y 設定陰影的偏移量。x 和 y 設為 2 將讓陰影在文字的右下方，模擬光源從左上方照射的效果，讓文字看起來更立體。

shadow(color:radius:x:y:) | Apple Developer Documentation
Adds a shadow to this view.developer.apple.com
import SwiftUI

struct ContentView: View {
    var body: some View {
        Text("重要的東西是眼睛看不見的")
            .font(.system(size: 32, weight: .bold))
            .foregroundStyle(.white)
            .shadow(color: .black, radius: 1, x: 2, y: 2)
    }
}

運用 shadow offset 可以讓任何元件變立體，以下例子讓圖片變立體。

import SwiftUI

struct ContentView: View {
    var body: some View {
        Image(.peter)
            .shadow(color: .black, radius: 1, x: 5, y: 5)
    }
}

設定 label 的 Shadow 顏色和 Shadow offset。Shadow offset 設定陰影的偏移量，Width 和 Height 設為 2 將讓陰影在文字的右下方，模擬光源從左上方照射的效果，讓文字看起來更立體。

設定 image view 的 layer.shadowOpacity 和 layer.shadowOffset。layer.shadowOpacity 控制陰影的不透明度，範圍 0 ~ 1，數字愈大愈清楚。layer.shadowOffset 設定陰影的偏移量，第一個數字是寬度，第二個數字是高度，{5, 5} 將讓陰影在元件的右下方，模擬光源從左上方照射的效果，讓元件看起來更立體。

執行 App 才能看到 Runtime Attributes 設定的效果。

設定 label 的 shadowColor & shadowOffset。

class ViewController: UIViewController {

    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        let label = UILabel()
        label.text = "重要的東西是眼睛看不見的"
        label.font = .boldSystemFont(ofSize: 32)
        label.textColor = .white
        label.sizeToFit()
        label.center = view.center
        label.shadowColor = .black
        label.shadowOffset = CGSize(width: 2, height: 2)
        view.addSubview(label)
    }
}

設定 image view 的 layer.shadowOpacity 和 layer.shadowOffset。

class ViewController: UIViewController {

    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        let imageView = UIImageView(image: .peter)
        imageView.center = view.center
        imageView.layer.shadowOpacity = 0.8
        imageView.layer.shadowOffset = CGSize(width: 5, height: 5)
        view.addSubview(imageView)
    }

}

Flutter

透過 TextStyle 的 shadows 實現陰影立體文字效果，參數 blurRadius 設定陰影模糊的程度，offset 設定陰影的偏移量。Offset 的第一個參數是 dx，第二個參數是 dy，Offset(2, 2) 將讓陰影在文字的右下方，模擬光源從左上方照射的效果，讓文字看起來更立體。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return const Scaffold(
      body: Center(
        child: Text(
          '重要的東西是眼睛看不見的',
          style: TextStyle(
            fontSize: 32,
            fontWeight: FontWeight.bold,
            color: Colors.white,
            shadows: [
              Shadow(
                color: Colors.black,
                offset: Offset(2, 2),
                blurRadius: 1,
              ),
            ],
          ),
        ),
      ),
    );
  }
}

使用 Container 搭配 BoxDecoration，設定 BoxShadow 的 offset 實現右下方的陰影。

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Container(
          padding: const EdgeInsets.all(8.0),
          decoration: BoxDecoration(
            color: Colors.blue,
            borderRadius: BorderRadius.circular(10),
            boxShadow: const [
              BoxShadow(
                blurRadius: 1,
                offset: Offset(5, 5),
              ),
            ],
          ),
          child: const Row(
            mainAxisSize: MainAxisSize.min,
            children: [
              Icon(Icons.star),
              Text('小王子: 你，就只有你，擁有會笑的星星'),
            ],
          ),
        ),
      ),
    );
  }
}
