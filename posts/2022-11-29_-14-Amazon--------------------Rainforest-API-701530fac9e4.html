#14 Amazon 的商品搜尋、詳情 & 評論 — 串接 Rainforest API

模仿 Amazon App，串接 Rainforest API，製作 Amazon 的商品搜尋，商品詳情 & 評論清單。

‎Amazon - Shopping made easy
‎Comprehensive Quickly search, get product details, read reviews and purchase millions of products available from…apps.apple.com

申請 API key

為了串接 Rainforest API 抓取 Amazon 的商品資訊，請先申請 Rainforest 的 API key。

Rainforest API — Product Data API
Rainforest API is the real-time Amazon Product Data API you’ve been looking for. No manual rules or web-scraper…www.rainforestapi.com

功能和畫面需求

  • 在 search bar 輸入關鍵字，搜尋 Amazon 上的相關商品。
  • 串接 Rainforest API 搜尋 Amazon 的商品。
Rainforest API - Product Data API
Rainforest API is the real-time Amazon Product Data API you've been looking for. No manual rules or web-scraper…www.rainforestapi.com

範例:

透過參數 type=search 和 search_term=swiftui 抓取 SwiftUI 的相關商品，api_key 請換成自己帳號的 api key。

https://api.rainforestapi.com/request?api_key=123456&type=search&search_term=swiftui&sort_by=most_recent&amazon_domain=amazon.com

回傳的 JSON 資料。

  • 點選商品清單的商品，切換到商品詳情頁面。
  • 串接 Rainforest API 抓取 Amazon 的商品詳細資訊。
Rainforest API - Product Data API
Rainforest API is the real-time Amazon Product Data API you've been looking for. No manual rules or web-scraper…www.rainforestapi.com

範例:

透過參數 type=product 和 asin=B0B1QKRFGS 抓取商品 An iOS Developer’s Guide to SwiftUI 的詳細資訊。api_key 請換成自己帳號的 api key，asin 可由商品搜尋回傳的 JSON 取得。

https://api.rainforestapi.com/request?api_key=123456&type=product&asin=B0B1QKRFGS&amazon_domain=amazon.com

回傳的 JSON 資料。

  • 商品的評論清單。
  • 串接 Rainforest API 抓取商品的評論 & 評分。
Rainforest API - Product Data API
Rainforest API is the real-time Amazon Product Data API you've been looking for. No manual rules or web-scraper…www.rainforestapi.com

範例:

透過參數 type=reviews 和 asin=1950325164 抓取商品 SwiftUI Apprentice 的評論 & 評分。api_key 請換成自己帳號的 api key，asin 可由商品搜尋回傳的 JSON 取得。

https://api.rainforestapi.com/request?api_key=123456&type=reviews&amazon_domain=amazon.com&asin=1950325164&sort_by=most_recent
