當我們在拉 outlet / action 時，大部分的時候右邊的 Assistant Editor 都會聰明地顯示對應的程式檔，不過有時它沒那麼聰明，顯示的不是我們想要的檔案，比方以下例子:

  • 想拉打卡 button 的 action 到 controller 類別，但是右邊卻顯示 UIResponder.h。
  • 想拉 cell 裡 label 的 outlet 到 cell 類別，但是右邊卻顯示 LocationTableViewController.swift。

此時我們可調整 Assistant Editor ，讓它顯示我們想要的檔案。調整的方法有以下三種:

方法一

點選右邊的 < > 切換，< > 裡的數字表示可切換的檔案數量。比方下圖表示有 3 個檔案可切換，我們可點選 < > 切換。

ps: 若是 cell 裡的元件想拉 outlet 到 cell 類別，記得要點選 cell 下的元件。點選 cell 的話，Assistant Editor 只會顯示 controller 類別。

方法二

點選 Automatic 旁的檔案名稱，然後再從檔案選單裡選擇想要的檔案。

方法三

利用 option + Enter 或 option + 點擊顯示 outlet / action 連線的檔案。
