使用和自訂儲存好用程式的 Xcode code snippet

Xcode 的 code snippet 貼心提供許多好用的程式片段，讓我們飛速打出一大串程式，享受隔壁同事看過來的崇拜眼神。接下來就讓我們好好認識如何使用和自訂 code snippet 吧。

使用 Code Snippet

以定義 unwind segue 連結的 function 為例，我們可以從 Snippets library 找到 Unwind Segue Method，將它加到程式裡。

喜歡打字的朋友也可以輸入特殊的關鍵字 unwi，再從選單裡選擇 unwind segue method。如下圖所示，灰色 { } 的 icon 表示它是 code snippet。

如下圖所示，Completion 告訴我們打出 unwind segue method 的神秘咒語是 unwind。(其實還滿好猜的，一點都不神秘)

讓我們再回到 Xcode 幫我們輸入的 code snippet。通常 code snippet 裡會有需要使用者輸入的藍色框框，比方下圖 unwindTo 後接的名字。

在此我們輸入 unwindToHome，因為此 unwind segue 將回到 HomeViewController。

自訂 Code Snippet

除了使用 Xcode 內建的 code snippet，我們還可以自己發明新的 code snippert。

  • Create Code Snippet
DispatchQueue.main.asyncAfter(deadline: .now() + .seconds(1)) {
                    
}

例如呼叫 DispatchQueue 的 asyncAfter 可設定程式在幾秒之後執行，我們懶得自己打一大串程式，因此將它變成 code snippt。

通常 code snippet 裡會有需要使用者輸入的藍色框框，在此我們希望使用者輸入秒數，也就是 .seconds( ) 裡的數字。

這些需要使用者輸入的框框稱為 placeholder，我們可以用 <# #> 設定 placeholder，<# #> 裡的文字將成為框框顯示的提示文字。

在此我們輸入 <# second #>。

DispatchQueue.main.asyncAfter(deadline: .now() + .seconds(<# second #>)) {
                    
}

選取想變成 code snippet 的程式，然後從右鍵選單點選 Create Code Snippet。

  • 設定 Code Snippet

將 code snippet 取名為 After Second，設定 Completion 欄位，將輸入的快速鍵設為 aftersecond。

  • 使用 Code Snippet

有了剛剛發明的 after second snippet，我們只要輸入 aft，即可快速產生相關的程式碼。
