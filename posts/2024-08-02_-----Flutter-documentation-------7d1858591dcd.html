Flutter documentation
Get started with Flutter. Widgets, examples, updates, and API docs to help you write your first Flutter app.docs.flutter.dev

以下我們一步步說明如何下載 Flutter documentation 和研究其中的範例程式。

下載或 git clone GitHub flutter/website

GitHub - flutter/website: Flutter documentation web site
Flutter documentation web site. Contribute to flutter/website development by creating an account on GitHub.github.com

下載後從 examples 資料夾可看到 Flutter documentation 網站上的範例程式。
