特殊大小的 UIKit 元件

iOS SDK 的 UIKit 提供許多方便好用的 UI 元件，大部分的元件都很隨和，任我們隨意控制它長高變胖，但還是有些元件比較不聽話，需要特別的處理才願意乖乖地改變大小。

比方以下元件特別頑固，無法用 Auto Layout 條件或 frame 設定大小，但可從 transform 控制。

UISwitch

demoSwitch.transform = CGAffineTransform(scaleX: 2, y: 3)

UISlider

demoSlider.transform = CGAffineTransform(scaleX: 1, y: 3)

UIProgressView

progressView.transform = CGAffineTransform(scaleX: 1, y: 3)

UIActivityIndicatorView

activityIndicatorView.transform = CGAffineTransform(scaleX: 3, y: 3)

UIPageControl

pageControl.transform = CGAffineTransform(scaleX: 3, y: 3)

UIStepper

stepper.transform = CGAffineTransform(scaleX: 1, y: 3)

以下元件比較好一點，除了可從 transform 控制，也可用 Auto Layout 或 frame 設定。(ps: 但無法在 Interface Builder 拖曳縮放或從 Size Inspector 修改高度)

UISegmentedControl
