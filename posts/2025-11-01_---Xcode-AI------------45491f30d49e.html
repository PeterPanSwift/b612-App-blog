使用 Xcode AI 修正遺失或多出的 }

初學者剛開始學習程式時，常常會遇到遺失或多出的 } 造成問題。比方以下例子。

  • Expected '}' in struct。

遺失的 } 造成問題。

struct ContentView: View {
    var body: some View {
        VStack {
            HStack {
                Image(.tableTennis)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 100, height: 100)
                    .clipShape(.circle)
                Text("桌球")
                    .font(.largeTitle.bold())
                    .foregroundStyle(.orange.gradient)
                Spacer()
            }
            HStack {
                Image(.baseball)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 100, height: 100)
                    .clipShape(.circle)
                Text("棒球")
                    .font(.largeTitle.bold())
                    .foregroundStyle(.orange.gradient)
                Spacer()
        }
        .padding()
    }
}
