檢視 SwiftUI 元件的大小和位置 — 從 Xcode 的 preview 或加入暫時的 border

最近彼得潘研讀 Apple 的教學文章 Inspecting view layout，發現 Apple 介紹兩個不錯的方法幫助我們檢視 SwiftUI 元件的大小和位置。

Inspecting view layout | Apple Developer Documentation
Determine the position and extent of a view using Xcode previews or by adding temporary borders.developer.apple.com
  • 方法 1: 從 Xcode 的 preview 顯示元件的框框。
  • 方法 2: 加入暫時的 border 顯示元件的框框。

方法 1: 從 Xcode 的 preview 顯示元件的框框

範例程式如下。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(.love)
                .resizable()
                .scaledToFit()
                .frame(width: 200)
            Text("I Love You to the Moon and Back")
                .font(.largeTitle)
        }
        .padding()
    }
}

點選 preview 左下的箭頭，切換到 Selectable 模式。

當 preview 在 Selectable 模式時，點選 preview 的元件或點選程式都能讓 preview 顯示對應元件的框框。

如下圖所示，點選 preview 的 I Love You to the Moon and Back 後，出現了 Text 的框框。

點選程式的 VStack 時，preview 也顯示了 VStack 的框框。

方法 2: 加入暫時的 border 顯示元件的框框

如果想不用點選就能顯示元件的大小位置，則可加入暫時的 border 顯示元件的框框，之後再將 border 移除。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(.love)
                .resizable()
                .scaledToFit()
                .frame(width: 200)
            Text("I Love You to the Moon and Back")
                .font(.largeTitle)
                .border(.black)
        }
        .padding()
        .border(Color.red)
    }
}
