修改 UI 元件的 action event

當我們在 storyboard 上連結 IBAction，生成元件的特定事件(event)觸發的 function 時，我們還可以自由選擇綁定的事件。

不過大部分的時候我們都不需修改，因為預設的事件就是我們最常用的，大概只有 text field 比較可能要修改。因此接下來我們就以 text field 為例，看看如何設定 action 的 event，以及 text field 裡不同的 event 將在何時觸發吧。

方法一

從 storyboard 拉 text filed 的 action 到程式。如下圖所示，點選 Editing Did End 後將出現事件的選單，有許多不同的事件可選擇。

方法二

點選 text field ，切換到 Connection inspector 後，點選 Sent Events 下事件的圓圈拉線到 Assistant editor 裡的程式。

text field 裡不同的 event 介紹:

Editing Did End

function 將在使用者結束 text filed 的文字輸入，游標離開 text field 時觸發。

Did End On Exit

function 將在使用者按下 return 鍵時觸發，可幫我們實現按下 return 鍵收鍵盤的功能。

連結 UITextField 的 Did End On Exit event，按下 return 鍵收鍵盤
開發 iOS App 時，我們時常想在 text field 的鍵盤按下 return 鍵時收鍵盤。想實現這樣的功能有很多方法，不過最簡單的應該是連結 text field 的 action，並將 Event 選為 Did End On…medium.com
Editing Changed

function 將在使用者輸入文字，text field 文字改變時觸發。

Editing Did Begin

function 將在使用者點選 text field，游標出現在 text field 裡，鍵盤出現時觸發。
