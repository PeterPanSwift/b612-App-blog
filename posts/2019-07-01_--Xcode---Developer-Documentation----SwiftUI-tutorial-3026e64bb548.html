從 Xcode 的 Developer Documentation 學習 SwiftUI tutorial

Apple 官方網站提供了超棒的 SwiftUI 教學，帶著我們一步步利用 SwiftUI 製作 iOS App。不過除了從 Safari 連到網站學習，我們也可以打開 Xcode 的 Developer Documentation 直接學習呢。

https://developer.apple.com/tutorials/swiftui/tutorials

點選 Xcode 的 menu Help > Developer Documentation。(快速鍵 cmd + shift + 0)

在左下角的 filter 框框輸入 tuto 後，點選 Tutorial 的提示。

此時將看到 Tutorial 的相關文章。如下圖所示，三個正方形的 icon 表示 Tutorial。

點選 Learn to Make Apps with SwiftUI。

在 Learn to Make Apps with SwiftUI 裡包含了一系列豐富的 tutorial 教學。

研究 tutorial

比方我們點選 tutorial Building Lists and Navigation。

為了更舒適地學習 SwiftUI tutorial，我們可點選左上角的 full screen & hide navigator 按鈕，享受全螢幕的 SwiftUI 教學。

點選 Project Files 的 icon 可下載範例程式。

跟著 tutorial 一步步學習。如下圖所示，當我們捲動頁面，從 Step 1 到 Step 2 時，右邊的畫面也會聰明地更新，比方從 Landmark.swift 變成 landmark.json，左邊的藍色長條則說明目前進行到的步驟。

當右邊顯示程式時，它還會貼心地將每個步驟的相關程式以藍色背景突顯出來。

有些步驟則可在右邊看到 App 的 preview 畫面。

我們可點選右上角的箭頭控制是否顯示 preview 畫面，比方想仔細閱讀程式碼時可將 preview 藏起來。

tutorial 小測驗

tutorial 的最後 Apple 還提供了貼心的小測驗，有興趣的朋友可挑戰看看。
