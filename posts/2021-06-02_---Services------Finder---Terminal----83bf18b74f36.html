利用 Services 將路徑從 Finder 或 Terminal 打開

開發 iOS App 時，有時我們會想查詢檔案的路徑，比方 Bundle 或 Document 裡的檔案路徑，然後在 Finder 顯示檔案或從 Terminal 切換到檔案的資料夾下。

常見的方法是用 print 印出路徑，然後再將路徑貼到 Terminal，不過其實利用 Mac 的 Services 我們可以更快將路徑從 Finder 或 Terminal 打開。

假設我們透過以下程式印出路徑。

if let url = Bundle.main.url(forResource: "cat", withExtension: "mp4") {
   print(url)
}

印出的路徑如下。

file:///Users/peterpan/Library/Developer/CoreSimulator/Devices/1C18CD63-4868-4810-BFA9-A5EF8A8BC594/data/Containers/Bundle/Application/0F7A6226-838B-4C85-97EE-7D052003E7A2/Demo.app/cat.mp4

選取路徑後(從 /Users 的地方開始選取)，從右鍵選單點選 Services > Show in Finder 即可打開 Finder 顯示檔案。

點選 Services > New Terminal at Folder 則可打開 Terminal，自動切換到路徑的資料夾下。
