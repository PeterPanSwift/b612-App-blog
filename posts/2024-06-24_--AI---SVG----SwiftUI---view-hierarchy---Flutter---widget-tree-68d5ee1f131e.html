請 AI 用 SVG 生成 SwiftUI 的 view hierarchy 和 Flutter 的 widget tree

請 AI 用 SVG 製做圖表的效果挺不錯的，比方請 AI 用 SVG 製做彩色版的 SwiftUI view hierarchy 和 Flutter widget tree。

以下我們分別示範幾個 SwiftUI & Fluter 的例子。

SwiftUI 的 view hierarchy

  • 範例 1。
struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "swift")
                .foregroundStyle(.orange)
            AppleView()
        }
    }
}

struct AppleView: View {
    var body: some View {
        HStack {
            Image(systemName: "apple.logo")
            Text("Apple")
                .font(.largeTitle)
        }
    }
}

        NavigationStack {
            List(books, id: \.self) { book in

        }
    }
}

Flutter 的 widget tree

  • 範例 1。
class MyApp extends StatelessWidget {
  const MyApp({super.key});

  // This widget is the root of your application.
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(seedColor: Colors.deepPurple),

      home: const MyHomePage(),
    );
  }
}

  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return const Scaffold(
      body: Center(
        child: Row(
          children: [
            Text('I love'),
            Text(
              'SwiftUI',
              style: TextStyle(
                fontSize: 30,
              ),
            ),
          ],
        ),
      ),
    );
  }
