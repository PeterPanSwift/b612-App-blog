利用 sizeToFit & sizeThatFits 讓 label 的大小剛好顯示文字

以情歌王子林隆璇的我愛妳這樣深為例。

顯示單行文字

let label = UILabel()
label.text = "我愛妳這樣深，日月星辰可為證"
label.backgroundColor = .yellow
label.font = .systemFont(ofSize: 30)
label.sizeToFit()
label

呼叫 sizeToFit，label 的寬度剛好等於文字的長度。

顯示多行文字

let label = UILabel()
label.text = "我愛妳這樣深\n日月星辰可為證"
label.backgroundColor = .yellow
label.font = .systemFont(ofSize: 30)
label.numberOfLines = 0
label.sizeToFit()
label

呼叫 sizeToFit，label 的高度剛好等於多行文字的高度。

顯示多行文字 & 指定寬度

let label = UILabel()
label.text = "我愛妳這樣深，日月星辰可為證，一朝決心不再為妳等，情難滅霜雪難分"
label.backgroundColor = .yellow
label.font = .systemFont(ofSize: 30)
label.numberOfLines = 0
let size = label.sizeThatFits(CGSize(width: 200, height: 0))
label.frame.size = size
label

呼叫 sizeThatFits，傳入 CGSize(width: 200, height: 0)，計算 label 寬度 200 時容納文字的理想尺寸。如下圖所示，算出的寬高為 182 和 215，因此我們將 label 設為寬 182 高 215。
