加入 child view controller — 從 storyboard & 程式 

開發較複雜的 App 畫面時，view controller 常會因此要處理過多的功能，造成 controller 的程式變得十分龐大。

為了減輕 controller 的負擔，其中一個方法是將畫面分成不同區塊，各區塊有不同的 view controller 負責。

以下圖為例，controller A 的畫面上有橘色 view 和藍色 view，橘色 view 由 controller B 控制，藍色 view 由 controller C 控制。對 controller A 來說，幫忙它組合出畫面的 B 和 C 就像它的小孩，因此我們會說 B 和 C 是 A 的 child view controller。

了解 child view controller 的概念後，接下來我們分別介紹兩種加入 child view controller 的方法。

  • 從 storyboard 加入 child view controller。
  • 從程式加入 child view controller。

從 storyboard 加入 child view controller

  • 將 Container View 加到某個 view 上。

當我們將 Container View 加到 view 上時，Container View 將自動連到另一個 view controller，此 controller 將成為 Container View 所在 controller 的 child view controller。

App 執行時，Container View 的區塊將顯示它連到的 controller 畫面，比方我們將 Container View 連到的 controller 設為黃色。

App 執行後將看到 Container View 的區塊變成黃色。

我們也可以自己指定 child view controller。先刪除 Container View 預設連到的 controller，然後將 Container View 連到某個 controller。

以下我們將 Container View 連到 table view controller，連線時 segue 請選擇 Embed。

從程式加入 child view controller

假設 child view controller 的類別為 ImageViewController。

class ImageViewController: UIViewController {

    override func viewDidLoad() {
        super.viewDidLoad()

        // Do any additional setup after loading the view.
        view.backgroundColor = .yellow
    }
}

從 ViewController 加入 child view controller 的程式如下。

class ViewController: UIViewController {

    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        let imageViewController = ImageViewController()
        addChild(imageViewController)
        view.addSubview(imageViewController.view)
        let imageViewControllerView = imageViewController.view
        imageViewControllerView?.translatesAutoresizingMaskIntoConstraints = false
        imageViewControllerView?.topAnchor.constraint(equalTo: view.safeAreaLayoutGuide.topAnchor).isActive = true
        imageViewControllerView?.leadingAnchor.constraint(equalTo: view.safeAreaLayoutGuide.leadingAnchor).isActive = true
        imageViewControllerView?.trailingAnchor.constraint(equalTo: view.safeAreaLayoutGuide.trailingAnchor).isActive = true
        imageViewControllerView?.heightAnchor.constraint(equalToConstant: 300).isActive = true
        imageViewController.didMove(toParent: self)
    }

}

從程式加入 child view controller 依序有 4 個步驟。

  • 呼叫 addChild(_:) 加入 child view controller。
addChild(imageViewController)
  • 加入 child view controller 的 view。
view.addSubview(imageViewController.view)
  • 設定 child view controller 的 view 的 auto layout 條件。
let imageViewControllerView = imageViewController.view
imageViewControllerView?.translatesAutoresizingMaskIntoConstraints = false
imageViewControllerView?.topAnchor.constraint(equalTo: view.safeAreaLayoutGuide.topAnchor).isActive = true
imageViewControllerView?.leadingAnchor.constraint(equalTo: view.safeAreaLayoutGuide.leadingAnchor).isActive = true
imageViewControllerView?.trailingAnchor.constraint(equalTo: view.safeAreaLayoutGuide.trailingAnchor).isActive = true
imageViewControllerView?.heightAnchor.constraint(equalToConstant: 300).isActive = true
  • 呼叫 didMove(toParent:)，
imageViewController.didMove(toParent: self)
