利用 Add Documentation 快速產生文件說明 — Xcode 11 新功能

開發 iOS App 時，我們會寫很多很多的 function。這些 function 如果可以加上文件說明，將讓未來的自己或別人閱讀我們程式時更容易理解。

然而幫 function 加上文件說明是件麻煩的事情，要用特別的注解格式為每個參數輸入說明，因此我們常常懶得加。在 Xcode 11 現在有方法可以自動輸入這些注解，再也沒有藉口偷懶了。

  1. 將滑鼠游標移到 function 名字上，然後按住 cmd 鍵後按滑鼠左鍵。
  2. 從選單裡選擇 Add Documentation。

此時 Xcode 將貼心地幫我們輸入相關說明的注解，我們只要在框框裡填入說明就大功告成了 !

/// eat
/// - Parameter name: food name
/// - Parameter price: food price
func eat(name: String, price: Int) {
}

現在 function eat 有了文件說明，之後只要在 eat 上按住 option 再點選滑鼠左鍵，即可看到它的相關說明。

ps: 如果嫌剛剛的步驟不夠快，還有更快的方法。以滑鼠點選 function 後，按快速鍵 cmd + option + / 也可以加入 Documentation。
