利用 Preview 快速調整多張圖片的尺寸和轉檔

有同學問彼得潘，有沒有方法可以快速調整多張圖片的尺寸和轉檔，而不用一張一張修改呢 ?

當然可以，而且使用 Mac 內建的 preview 就能做到。接下來我們就以 iPhone 拍照的 HEIC 檔為例，示範如何將原本 3024 * 4032 的 HEIC 圖片快速轉換成 400 * 533 的 JPG 檔。

利用 AirDrop 將照片傳到 Mac。

從 Finder 選取多張照片後雙擊觸控板，從 Preview 打開圖片

點選 Edit > Select All 選取全部的圖片

點選 Tools > Adjust Size 修改圖片尺寸

設定圖片的尺寸

比方單位選擇 pixels，width 設為 400，此時高度將自動依圖片比例換算成 533。

點選 Edit > Select All 選取全部的圖片

點選 File > Export Selected Images 輸出縮小後的圖片

點選左下的 Options 按鈕設定圖片格式，比方選擇 JPEG。

點選右下的 Choose 按鈕完成圖片的輸出。

Cool，原本 3024 * 4032 的 HEIC 圖片全都轉換成 400 * 533 的 JPG 檔。
