#211 打造吸睛的 About 頁面

選擇喜歡的主題打造吸睛的 About 頁面。

範例
  • 自我介紹、履歷。
  • 介紹其它感興趣的人事物，比方動物、電影、動漫角色、美食、小說、商品、遊戲、景點、偶像等。
使用技術

以下技術三選一。

  • SwiftUI。
  • 從程式使用 UIKit 和 auto layout。
  • Flutter。
大學生的 SwiftUI 作業需求。
  1. 一定要使用的元件。

VStack、HStack、ZStack、Text、Image、ScrollView。

2. 圖片用到 asset 跟 SF Symbol。

3. 使用多個 modifier。

4. 畫面可以上下捲動，搭配 ScrollView。

5. 定義多個 view 的型別。

6. 使用 property 客製自訂的 SwiftUI view。

#229 使用 property 客製 SwiftUI view

  • 設計兩種畫面，一般模式和 Dark mode。
  • iOS 26 的 liquid glass 效果。
  • SF Symbol animation。

  • Text 搭配 Markdown 語法實現點擊文字打開連結的功能。

  • 將圖片裁切成自訂的形狀。

大學生的 Flutter 作業需求。
  1. 可選擇做 iOS App、Android App、Web App 或 Windows App。
  2. 一定要使用的 widget

Text、Image、Icon、Column、Row、Color、Stack、 SingleChildScrollView

3. 定義多個 stateless widget 類別

4. 畫面可以上下捲動，搭配 SingleChildScrollView。

5. 其它可以考慮使用的 widget

Container、Padding、 SizeBox、Expanded、Positioned、AppBar、SafeArea、Transform、Divider、CircleAvatar、ClipRRect、Align、Wrap、Gradient、BoxDecoration、Table。

6. 進階功能。

自訂 widget constructor、搭配自訂的資料型別、包含多個檔案、使用 collection for 顯示 List 的資料。
