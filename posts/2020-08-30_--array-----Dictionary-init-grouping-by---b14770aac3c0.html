將 array 分類的 Dictionary init(grouping:by:)

開發 iOS App 時，有時我們會想將 array 裡的資料分類，將它分成多個 array，比方以下 array tvShows 的電視影集，我們希望依類別拆成不同的 array，方便之後分別存取喜劇，恐怖 ＆解謎的影片清單。

enum Genre {
    case comedy, horror, mystery
}

struct TVShow {
    let name: String
    let genre: Genre
}

let tvShows = [
    TVShow(name: "The Big Bang Theory", genre: .comedy),
    TVShow(name: "Stranger Things", genre: .horror),
    TVShow(name: "Friends", genre: .comedy),
    TVShow(name: "Sherlock", genre: .mystery),
    TVShow(name: "The X-Files", genre: .mystery),
]

透過 Dictionary 的 init(grouping:by:)，我們可將 array 裡的資料分類，變成一個 Dictionary，然後以 key 存取對應的 array。

init(grouping:by:) | Apple Developer Documentation
Creates a new dictionary whose keys are the groupings returned by the given closure and whose values are arrays of the…developer.apple.com
let tvDic = Dictionary(grouping: tvShows) {
    $0.genre
}

以上程式將以影集的類別當 key，因此 tvDic 的型別將為 [Genre : [TVShow]]。

array 裡的影集順利地依類別 comedy, horror, mystery 分成三個 array，分別存在 dictionary 的三個 key 裡，以 dump(tvDic) 印出的內容如下:
