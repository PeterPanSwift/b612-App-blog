SwiftUI NavigationStack 的 UI 調整

預設白跟黑的 navigation bar

原本的 navigation bar 預設是白色跟黑色背景(dark mode)。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(._100Years)
                .resizable()
                .scaledToFill()
                .frame(minWidth: 0, maxWidth: .infinity)
                .navigationTitle("Agatha Christie")
        }
    }
}

透明的 navigation bar

從 NavigationStack 裡的畫面呼叫 ignoresSafeArea， 畫面將延伸到 navigation bar 底下。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(._100Years)
                .resizable()
                .scaledToFill()
                .frame(minWidth: 0, maxWidth: .infinity)
                .ignoresSafeArea()
                .navigationTitle("Agatha Christie")
        }
    }
}

設定文字的顏色和大小

利用 toolbar modifier 可客製 navigation bar 的內容。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(._100Years)
                .resizable()
                .scaledToFill()
                .frame(minWidth: 0, maxWidth: .infinity)
                .toolbar(content: {
                    ToolbarItem(placement: .principal) {
                        Text("Agatha Christie")
                            .font(.largeTitle)
                            .foregroundStyle(.red)
                    }
                })
        }
    }
}

在 toolbar 裡傳入 ToolbarItem，參數 placement 傳入 principal 讓元件出現在 bar 的中間，{ } 裡傳入 Text 設定顯示的文字。

利用 navigationBarTitleDisplayMode 讓 navigation bar 的高度縮短。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(._100Years)
                .resizable()
                .scaledToFill()
                .frame(minWidth: 0, maxWidth: .infinity)
                .toolbar(content: {
                    ToolbarItem(placement: .principal) {
                        Text("Agatha Christie")
                            .font(.largeTitle)
                            .foregroundStyle(.red)
                    }
                })
                .navigationBarTitleDisplayMode(.inline)
        }
    }
}

設定客製的 navigation bar 內容，比方圖片和文字

在 ToolbarItem 裡傳入包含圖文的 HStack。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(._100Years)
                .resizable()
                .scaledToFill()
                .frame(minWidth: 0, maxWidth: .infinity)
                .toolbar(content: {
                    ToolbarItem(placement: .principal) {
                        HStack {
                            Image(.peter)
                            Text("Peter")
                                .font(.largeTitle)
                        }
                        .offset(y: 30)
                    }
                })
                .navigationBarTitleDisplayMode(.inline)
        }
    }
}

設定 bar 的顏色

利用 toolbarBackground modifier 設定 navigation bar 的顏色跟是否顯示。(iOS 16 以上)

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(._100Years)
                .resizable()
                .scaledToFill()
                .frame(minWidth: 0, maxWidth: .infinity)
                .navigationTitle("Agatha Christie")
                .toolbarBackground(.visible, for: .navigationBar)
                .toolbarBackground(.red, for: .navigationBar)
        }
    }
}
