從程式讀取 Info.plist 的內容，比方 version & API key

有些時候我們會想從程式讀取 Info.plist 的內容，比方 App 的版本，某個 API key 等。

在程式裡我們可以透過 key 從 Info.plist 讀取對應的 value，接下來就讓我們以 APIKey 為例說明吧。

方法1:

利用 Bundle.main 呼叫 function object(forInfoDictionaryKey:) 讀取 Info.plist 的內容。由於 Info.plist 的內容可以是多種型別，因此 object(forInfoDictionaryKey:) 回傳的資料型別是 Any?，我們還要透過 as? 將它轉型成資料真正的型別。

object(forInfoDictionaryKey:) - Bundle | Apple Developer Documentation
Edit descriptiondeveloper.apple.com
if let apiKey = Bundle.main.object(forInfoDictionaryKey: "APIKey") as? String {
   print(apiKey)
}

方法2:

利用 Bundle.main 的 property infoDictionary 讀取 Info.plist 的內容。

infoDictionary - Bundle | Apple Developer Documentation
Edit descriptiondeveloper.apple.com
if let apiKey = Bundle.main.infoDictionary?["APIKey"] as? String {
   print(apiKey)
}

Info.plist 裡包含了 App 的 Version & Build 資訊，因此我們也可以利用剛剛學到的方法讀取。

不過原本 Info 頁面顯示的 key 並非 Version & Build 真正的 key，我們必須在 Info 頁面按右鍵，從右鍵選單點選 Show Row Keys/Values 後才能看到真正的 key。

如下圖所示，version 的 key 為 CFBundleShortVersionString，build 的 key 為 CFBundleVersion。

因此，我們可透過 CFBundleShortVersionString & CFBundleVersion 得到 App 的 version & build。

if let version = Bundle.main.object(forInfoDictionaryKey: "CFBundleShortVersionString") as? String {
print(version)
}
if let build = Bundle.main.object(forInfoDictionaryKey: "CFBundleVersion") as? String {
   print(build)
}
