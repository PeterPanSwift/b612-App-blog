back 按鈕長按顯示 menu，返回之前的任一頁 — iOS 14 新功能

從 iOS 14 開始，我們可以在 navigation bar 的 back button 長按，它將顯示之前頁面的 menu 清單，方便我們點選返回任何一頁，再也不用辛苦地按很多次 back 了。

back 按鈕長按顯示 menu，返回之前的任一頁 — iOS 14 新功能

從 iOS 14 開始，我們可以在 navigation bar 的 back button 長按，它將顯示之前頁面的 menu 清單，方便我們點選返回任何一頁，再也不用辛苦地按很多次 back 了。

開發 App 時，只要使用 SwiftUI 的 NavigationStack 或 UIKit 的 UINavigationController 即有此功能。

SwiftUI

struct FirstView: View {
    var body: some View {
        NavigationStack {
            NavigationLink {
                SecondView()
            } label: {
                Image(systemName: "1.circle")
                    .resizable()
                    .scaledToFit()
            }
            .navigationTitle("1")
        }
    }
}

struct SecondView: View {
    var body: some View {
        NavigationLink {
            ThirdView()
        } label: {
            Image(systemName: "2.circle")
                .resizable()
                .scaledToFit()
        }
        .navigationTitle("2")
    }
}

struct ThirdView: View {
    var body: some View {
        Image(systemName: "3.circle")
            .resizable()
            .scaledToFit()
            .navigationTitle("3")
    }
}
