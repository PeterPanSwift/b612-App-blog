上架 App，請用心用愛寫 App Description ~ app reject #1

有朋友上架 App 時遇到了問題，Apple 說他的 App 不夠完整，Performance: App Completeness 有問題。

the application description does not provide information about the app’s content and functionality.

這位朋友的 App 是個幫助你化學學測考高分 App，它的 App Description 原本長這樣，只有短短十個字 !

別以為 Apple 評審只看美美的 App 畫面截圖，他們跟彼得潘一樣，都是文青，也很注重文筆。所以 Description 請清楚描述你的 App 功能，讓任何人一看就懂。雖然我們都知道使用者不見得有耐心看，就好像我們寄給心儀女生的情書從未被拆開一樣，但還是請用心用愛寫清楚，就算這世上沒有其他人看一眼，至少 Apple 的評審大大會看到你的用心。

By 彼得潘的 iOS App Neverland on October 4, 2017.
