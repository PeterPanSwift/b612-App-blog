UISearchBar 的樣式調整

search bar 的預設樣式

消除 search bar 的背景顏色

讓下圖的白色區塊變透明。

方法 1: Search Style 設為 Minimal

  • 從 Interface Builder
  • 從程式
class ViewController: UIViewController {
    @IBOutlet weak var searchBar: UISearchBar! {
        didSet {
            searchBar.searchBarStyle = .minimal
        }
    }

方法 2: backgroundImage 設為看不到的圖片

class ViewController: UIViewController {
    @IBOutlet weak var searchBar: UISearchBar! {
        didSet {
            searchBar.backgroundImage = UIImage()
        }
    }

調整 search 框框的顏色

修改下圖灰色區塊的顏色。

修改 search bar 的 searchTextField.backgroundColor。

  • 從 User Defined Runtime Attributes。
  • 從程式
class ViewController: UIViewController {
    @IBOutlet weak var searchBar: UISearchBar! {
        didSet {
            searchBar.searchTextField.backgroundColor = .systemGreen
        }
    }

調整 search bar 的背景顏色

透過 backgroundColor 設定背景顏色。記得要將 searchBarStyle 設為 minimal，背景顏色才能顯示。

  • 從 Interface Builder
  • 從程式
class ViewController: UIViewController {
    @IBOutlet weak var searchBar: UISearchBar! {
        didSet {
            searchBar.searchBarStyle = .minimal
            searchBar.backgroundColor = .systemGreen
            searchBar.searchTextField.backgroundColor = .white
        }
    }
