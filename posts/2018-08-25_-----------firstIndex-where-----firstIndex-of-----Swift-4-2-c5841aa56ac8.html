想精簡程式的朋友，也可以採用省略參數的 $0 寫法。

let index = songs.firstIndex {
    $0.name.hasPrefix("一")
}

3 firstIndex(of:)

回傳 array 裡第一個和參數 element 一樣的成員的編號。

我們會在 function 裡傳入參數 element，然後它會找出和參數一樣的成員，因此它有個特別的限制，它必須能利用 == 判斷東西一樣。

在 Swift 裡，只有遵從 protocol Equatable，定義 function == 的型別才能比較是否相等 (相關的說明可參考以下連結)。

自動幫遵從 Equatable 的 struct & enum 定義 == function — Swift 4.1
在 Swift 的世界，並不是任何東西都可以用 == 判斷相等，比方以下我們自訂的 Man 就不行。medium.com

因此在剛剛的例子，原本 array songs 無法呼叫 function firstIndex(of:) ，因為 Song 無法比較相等。

一旦我們讓 Song 遵從 protocol Equatable，讓它可以比較相等， array song 就可以呼叫 firstIndex(of:)，找出和傳入參數一樣的成員的編號。

struct Song: Equatable {
   var name: String
   var singer: String
}
let song1 = Song(name: "一次就好", singer: "楊宗緯")
let song2 = Song(name: "兩次就好", singer: "楊宗緯")
var songs = [Song]()
songs.append(song1)
songs.append(song2)
let index = songs.firstIndex(of: song1)

答案: 0

當我們想知道某個東西是 array 裡的第幾個成員時常會用到 firstIndex(of:)，比方在表格頁面時，先找到東西在 array 的位置，然後更新相關的 cell。
