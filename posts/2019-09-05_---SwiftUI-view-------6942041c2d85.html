新增 SwiftUI view 的各種方法

點選 Xcode menu 的 File > New > File。(快速鍵 cmd + n)

新增 SwiftUI view 的各種方法

  • 從 Xcode 新增檔案，選擇 SwiftUI View。
  • 手動定義 view & Extract to File。
  • 請 AI 建立。
  • 從 Swift Playgrounds 新增檔案，輸入 view。

從 Xcode 新增檔案，選擇 SwiftUI View

  • 新增檔案。

方法 1: 從 Quick Actions 選擇 New > File from Template。(快速鍵 cmd + n)

方法 2: 點選 Xcode menu 的 File > New > File from Template。

方法 3: Project navigator 的右鍵選單點選 New File from Template。

  • 選擇 User Interface 下的 SwiftUI View。
  • 輸入 view 的型別名。(同時也是檔名)

如下圖所示，生成了 EditPhotoView.swift，裡面定義了 struct EditPhotoView，預設的內容是我們熟悉的 Hello World!。

//
//  EditPhotoView.swift
//  Demo
//
//  Created by Peter Pan on 2023/9/22.
//

import SwiftUI

struct EditPhotoView: View {
    var body: some View {
        Text("Hello, World!")
    }
}

#Preview {
    EditPhotoView()
}

手動定義 view & Extract to File

輸入 { } ，點選 { } View。

產生定義 View 的 struct 和 preview。

輸入內容。

struct EditPhotoView: View {
    var body: some View {
        Text("Hello, World!")
    }
}

#Preview {
    EditPhotoView()
}

若想將 View 的定義搬到獨立的檔案，可再另外透過 Extract to File。

Extract Subview 後將 view 型別搬到新檔的 Extract to File — Xcode 16 新功能
開發 SwiftUI App 時，我們常會用 Extract Subview 將 subview 變成獨立的型別，增加程式的可讀性，方便未來重覆使用。medium.com

請 AI 建立

比方在 Xcode 的 Coding Assistant 輸入以下 prompt。

new SwiftUI view LoginView with Text peter
import SwiftUI

struct LoginView: View {
    var body: some View {
        Text("peter")
    }
}

#Preview {
    LoginView()
}

從 Swift Playgrounds 新增檔案，輸入 view

  • 點選 + 新增檔案。
  • 名字改名為 view 的型別。
  • 輸入 vi，點選下圖的 View。

Swift Playgrounds 自動填入定義 view 和 preview 的相關程式。
