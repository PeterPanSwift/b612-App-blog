利用 SwiftUI 的 TabView 搭配 page 實現分頁水平滑動瀏覽

利用 iOS 14 的 page 我們可以將顯示 tab 分頁的 TabView 變成水平滑動的分頁照片瀏覽。接下來就讓我們看它表演吧。

利用 SwiftUI 的 TabView 搭配 page 實現分頁水平滑動瀏覽

利用 tabViewStyle 的 page，我們可以將顯示 tab 分頁的 TabView 變成分頁水平滑動瀏覽，而且還有顯示目前所在分頁的小圓點，接下來就讓我們看它表演吧。

原本顯示 tab 分頁的 TabView

struct ContentView: View {
    var body: some View {
        TabView {
            Tab("teddybear", systemImage: "teddybear.fill") {
                Image(systemName: "teddybear.fill")
                    .resizable()
                    .scaledToFit()
            }
            Tab("swift", systemImage: "swift") {
                Image(systemName: "swift")
                    .resizable()
                    .scaledToFit()
            }
        }
    }
}

實現分頁照片瀏覽的 page

從 TabView 呼叫 modifier tabViewStyle(_:)，傳入 .page。

struct ContentView: View {
    var body: some View {
        TabView {
            Tab("teddybear", systemImage: "teddybear.fill") {
                Image(systemName: "teddybear.fill")
                    .resizable()
                    .scaledToFit()
            }
            Tab("swift", systemImage: "swift") {
                Image(systemName: "swift")
                    .resizable()
                    .scaledToFit()
            }
        }
        .tabViewStyle(.page)
    }
}

畫面變成水平滑動的分頁，而且下方的 tab bar 也消失了。

由於不需要顯示 tab 的 icon 和文字，因此我們也不再需要 Tab，剛剛的程式可以省略 Tab，TabView { } 裡直接加入每一頁呈現的內容即可。

struct ContentView: View {
    var body: some View {
        TabView {
            Image(systemName: "teddybear.fill")
                .resizable()
                .scaledToFit()
            Image(systemName: "swift")
                .resizable()
                .scaledToFit()
        }
        .tabViewStyle(.page)
    }
}

顯示目前所在分頁的小圓點

看起來的確滿炫的，但是下方的小圓點怎麼不見了呢 ?

因為小圓點預設是白色的，所以當畫面背景也是白色時，我們就算有火眼金睛也看不到它。當我們將 App 變成 dark mode 時，即可看到想念的小圓點。

若是 Tab 沒省略，小圓點還會自動變成 tab 的 icon 圖示呢。

struct ContentView: View {
    var body: some View {
        TabView {
            Tab("teddybear", systemImage: "teddybear.fill") {
                Image(systemName: "teddybear.fill")
                    .resizable()
                    .scaledToFit()
            }
            Tab("swift", systemImage: "swift") {
                Image(systemName: "swift")
                    .resizable()
                    .scaledToFit()
            }
        }
        .tabViewStyle(.page)
    }
}

讓小圓點顯示背景的 indexViewStyle

為了讓小圓點在白色背景也能清楚呈現，我們可在 TabView 呼叫 modifier indexViewStyle(_:)，傳入參數 always 讓它有一塊長條圓角的背景。

struct ContentView: View {
    var body: some View {
        TabView {
            Image(systemName: "teddybear.fill")
                .resizable()
                .scaledToFit()
            Image(systemName: "swift")
                .resizable()
                .scaledToFit()
        }
        .tabViewStyle(.page)
        .indexViewStyle(.page(backgroundDisplayMode: .always))
    }
}

不想看到小圓點

如果不想看到小圓點，我們可在生成 .page 時傳入參數 .never，never 將讓小圓點消失不見。

struct ContentView: View {
    var body: some View {
        TabView {
            Image(systemName: "teddybear.fill")
                .resizable()
                .scaledToFit()
            Image(systemName: "swift")
                .resizable()
                .scaledToFit()
        }
        .tabViewStyle(.page(indexDisplayMode: .never))
    }
}

指定分頁照片的大小

我們也可以利用 frame 設定 TabView 的大小，讓它不是佔滿整個螢幕，比方以下例子 TabView 的高度設為 300。

struct ContentView: View {
    var body: some View {
        VStack {
            TabView {
                Image(systemName: "teddybear.fill")
                    .resizable()
                    .scaledToFit()
                Image(systemName: "swift")
                    .resizable()
                    .scaledToFit()
            }
            .tabViewStyle(.page)
            .frame(height: 300)
            
            Text("其它內容")
                .font(.system(size: 100))
        }
    }
}

TabView 顯示 array 的資料

水平滑動的 TabView 很適合用來顯示 array 的大量資料。以下

struct ContentView: View {
    let animals = [
        "teddybear.fill",
        "swift",
        "hare.fill",
        "dog.fill",
        "cat.fill",
    ]
    var body: some View {
        TabView {
            ForEach(animals, id: \.self) { animal in
                Image(systemName: animal)
                    .resizable()
                    .scaledToFit()
                    .padding()
            }

        }
        .tabViewStyle(.page)
        .indexViewStyle(.page(backgroundDisplayMode: .always))
    }
}
