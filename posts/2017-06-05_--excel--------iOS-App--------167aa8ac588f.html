將 excel 檔輸出成方便 iOS App 讀取的文字檔

以下以醒吾科技大學的藏書為例子。

定案書單 — 醒吾科技大學

1 打開 excel 檔。

2 點選 File > Save As。

3 將 excel 檔輸出成方便 iOS App 讀取的文字檔，比方 UTF-16 Unicode Text (.txt)

158618870.txt 的內容

4. 將 txt 最前面和最後面多餘的文字刪除，只留下 App 需要的書籍資訊。

5. 讀取 txt 檔，解析內容，印出書籍名稱。書籍名稱在第三欄，所以要從 array 的編號 2 讀取。

let url = Bundle.main.url(forResource: "158618870", withExtension: "txt")
let content = try! String(contentsOf: url!)
let lineArray = content.components(separatedBy: "\n")
for line in lineArray {
   let bookInfoArray = line.components(separatedBy: "\t")
   print(bookInfoArray[2])
}
