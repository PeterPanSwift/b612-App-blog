連結 UITextField 的 Did End On Exit event，按下 return 鍵收鍵盤

開發 iOS App 時，我們時常想在 text field 的鍵盤按下 return 鍵時收鍵盤。想實現這樣的功能有很多方法，不過最簡單的應該是連結 text field 的 action，並將 Event 選為 Did End On Exit。

@IBAction func closeKeyboard(_ sender: Any) {

}

至於 @IBAction function { } 裡的程式呢 ? 什麼都不用寫，此時按下 return 鍵，鍵盤就會神奇地收起來了 !

如果想從程式設定也可以，範例如下:

@objc func closeKeyboard() {
}

override func viewDidLoad() {
   super.viewDidLoad()
   starTextField.addTarget(self, action: #selector(closeKeyboard), for: .editingDidEndOnExit)
}
