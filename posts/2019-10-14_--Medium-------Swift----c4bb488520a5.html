在 Medium 貼上彩色的 Swift 程式

利用 Medium 撰寫程式相關的技術文章時，我們時常需要貼上程式碼。但是要如何在文章裡顯示彩色的 Swift 程式呢 ?

我們可透過以下步驟實現:

  • 生成灰色背景區塊。
  • 語言選擇 Swift。
  • 將程式貼到灰色背景區塊。

生成灰色背景區塊

我們將用灰色背景區塊顯示程式，讓程式跟文章裡的其它文字有明顯區隔。

生成灰色背景區塊的方法有兩種:

  • 方法 1

點選 +。

選擇 code block。

code block 將生成下圖的灰色背景區塊。

  • 方法 2

在文章裡新的一行的開頭輸入 ``` ，輸入到第三個 ` 時將自動生成灰色背景區塊。

此步驟最難的地方在於如何輸入 `，在 Mac 的鍵盤上，` 在數字 1 的左邊。另外輸入 ``` 的位置很重要，只有在新的一行的開頭輸入才能生成灰色背景區塊。

將程式貼到灰色背景區塊

身為工程師，我們最喜歡 copy paste 了。以下我們從 Xcode 選取程式後按下 cmd + c 複製程式。

在灰色背景區塊按 cmd + v 貼上程式。

struct ContentView: View {
    
    var body: some View {
        
        VStack {
            Text("彼得潘作品集")
                .font(.largeTitle)
            Image("books")
                .resizable()
                .scaledToFit()
        }
    }
}
