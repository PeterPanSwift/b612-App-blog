從本機執行 AI 寫的 React Web App

當我們請 GPT、Claude、Gemini、Grok 等 AI 開發 App 時，它們通常都會預設開發 React Web App，而且可生成能讓任何人連線瀏覽的網頁。

不過有時我們只想在自己的本機執行 App，此時我們可下載 AI 撰寫的 tsx 檔，從本機啟動 React Web App。

以下彼得潘分別以 GPT & Claude 開發冰島去哪玩 App 說明。

GPT 5

開發一個冰島去哪玩 React Web App，點擊按鈕隨機一個冰島景點的圖文介紹，從 wiki 取得漂亮的冰島景點照片

點選 Run code 可執行 App。

GPT 生成的 App 網址。

Iceland Explorer React App (app
ChatGPT helps you get answers, find inspiration and be more productive. It is free to use and easy to try. Just ask and…chatgpt.com

點選 Download 下載程式。

剛剛 GPT 的程式其實用到 TypeScript 語法，請將附檔名改成 tsx，變成 iceland_explorer_react_app_app.tsx。

ps: 通常 AI 開發的 React Web App 都是以 TypeScript 撰寫。

建立

npm create vite@latest iceland -- --template react-ts


將 tsx 檔加到 iceland/src 下。

查看 tsx 檔的 import。

import { useEffect, useMemo, useState } from "react";

不需要另外安裝其它套件，在 iceland 下輸入指令。

npm install


查詢 tsx 檔 export 的地方。

export default function App() {


修改 iceland/src 下的 main.tsx，將 import App 改成從 iceland_explorer_react_app_app.jsx。

import App from './iceland_explorer_react_app_app.tsx'

在 iceland 資料夾下執行

npm run dev

Claude

開發一個冰島去哪玩 React Web App，點擊按鈕隨機一個冰島景點的介紹





npm create vite@latest iceland -- --template react-ts


cd iceland


查看 tsx 檔的 import。

import React, { useState } from 'react';
import { MapPin, Camera, Snowflake, Mountain } from 'lucide-react';


npm install lucide-react


將 tsx 檔加到 src 下。



查詢 tsx 檔最後 export 的名字。

export default IcelandTravelApp;


修改 iceland/src 下的 main.tsx。

import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import './index.css'
import IcelandTravelApp from './iceland-travel-app.tsx'

createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <IcelandTravelApp />
  </StrictMode>,
)


修改 src 下的 App.tsx。


import { useState } from 'react'
import reactLogo from './assets/react.svg'
import viteLogo from '/vite.svg'
import './App.css'
import IcelandTravelApp from './iceland-travel-app'

function App() {
  return <IcelandTravelApp />
}

export default App
