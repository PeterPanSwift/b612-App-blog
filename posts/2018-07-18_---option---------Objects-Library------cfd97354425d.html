按住 option 拖曳元件後，讓 Objects Library 一直開啟

在 Xcode 10，Objects Library 一般會長得像下圖，當我們將元件拖曳出來後，Objects Library 就消失了。

不過當我們想拖曳多個元件時，這樣的操作實在太辛苦了，需要不斷重新打開 Objects Library。

還好，在 Xcode 10 beta 4，Objects Library 變得更方便了。現在想拖曳多個元件到畫面上，可以先按住 option 拖曳第一個元件後，讓 Objects Library 的上方長出一條 bar，然後再放心地放開 option 鍵。

之後除非點選左上角的紅色 X 按鈕將它關掉，否則 Objects Library 將一直陰魂不散地開著，方便我們繼續拖曳其它的元件。
