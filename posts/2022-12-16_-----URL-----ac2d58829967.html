以 Platzi Fake Store API 為例，以下介紹三種組合產生 URL 的方法。

  • 呼叫 appending
  • 利用 init?(string: String, relativeTo url: URL?)
  • 使用 URLComponents

呼叫 appending

let baseURL = URL(string: "https://api.escuelajs.co")!
let url = baseURL.appending(path: "api/v1/categories")

利用 init?(string: String, relativeTo url: URL?)

let baseURL = URL(string: "https://api.escuelajs.co")!
let url = URL(string: "api/v1/categories", relativeTo: baseURL)

使用 URLComponents

var components = URLComponents()
components.scheme = "https"
components.host = "api.escuelajs.co"
components.path = "/api/v1/categories"
let url = components.url
