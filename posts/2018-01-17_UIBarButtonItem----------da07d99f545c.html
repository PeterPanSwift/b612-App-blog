UIBarButtonItem 點選後的顏色問題

當我們利用 segue 將 navigation bar 上的 button 連到下一個 controller 時，現在會出現一個十分特別的顏色問題，它會變成很像不能點選的樣子欺騙我們。

如下圖所示，UIBarButtonItem Add 連線到下一個 controller。

當我們點選 Add，進到下個頁面再返回時，原本的 Add 顏色會變淡，就好像它不能點選一樣。

千萬別被它的外表騙了，它還是可以點選的。如果希望 UIBarButtonItem 的文字顏色正常，目前可用以下的解法解決:

在 button Add 所在的 controller ，定義 function viewWillAppear(_:)，加入以下程式碼

override func viewWillAppear(_ animated: Bool) {
   super.viewWillAppear(animated)
   navigationItem.rightBarButtonItem?.isEnabled = false
   navigationItem.rightBarButtonItem?.isEnabled = true
}

說明:

先將 UIBarButtonItem 設為 disabled ，再將它設為 enabled。
