使用 Android Studio 加入 Compose App 的第三方套件

開發 Jetpact Compose App 時，我們時常會安裝別人寫好的 library 快速實現某個功能。以下我們以顯示網路圖片的套件 Coil 為例，一步步示範如何在 Compose App 上加入第三方套件。

使用 Android Studio 加入 Compose App 的第三方套件

開發 Jetpact Compose App 時，我們時常會安裝別人寫好的 library 快速實現某個功能。以下我們以顯示網路圖片的套件 Coil 為例，一步步示範如何在 Compose App 上加入第三方套件。

  • 找到套件的 groupId & artifactId。
  • 利用 Android Studio 的 Add Dependency 加入套件。
  • 使用套件。

找到套件的 groupId & artifactId

從套件的說明頁面找到套件的 groupId & artifactId。

以下為套件 Coil 的 GitHub 頁面。

GitHub - coil-kt/coil: Image loading for Android and Compose Multiplatform.
Image loading for Android and Compose Multiplatform. - coil-kt/coilgithub.com

從 implementation 區塊得知安裝套件 Coil 需加入 2 個 library，implementation( ) 裡的字串格式為 groupId:artifactId:version，因此套件的資訊如下。

  • 套件 1: groupId 是 io.coil-kt.coil3，artifactId 是 coil-compose，版本是 3.0.4。
  • 套件 2: groupId 是 io.coil-kt.coil3，artifactId 是 coil-network-okhttp，版本是 3.0.4。

利用 Android Studio 的 Add Dependency 加入套件

從 Find Action 點選 Project Structure。

切換到 Dependencies > app 頁面。

點選 +，選擇 Library Dependency。

輸入套件的相關資訊搜尋，輸入groupId:artifactId將可精準地找到套件，因此我們輸入 io.coil-kt.coil3:coil-compose。

成功找到套件後，在 Step 2 選擇 implementation，然後點選 OK。

用同樣的方法加入套件 io.coil-kt.coil3:coil-network-okhttp。

點選 OK 加入套件。

此時 Android Studio 將幫我們將套件 coil-compose & coil-network-okhttp 的相關資訊填入 libs.versions.toml & build.gradle.kts，然後開始安裝套件，畫面上顯示 Gradle project sync in progress。

libs.versions.toml 的 versions 區塊加了 coilCompose = "3.0.4" 和 coilNetworkOkhttp = "3.0.4"，libraries 區塊加了 coil-compose = { group = "io.coil-kt.coil3", name = "coil-compose", version.ref = "coilCompose" } 和 coil-network-okhttp = { group = "io.coil-kt.coil3", name = "coil-network-okhttp", version.ref = "coilNetworkOkhttp" }。

使用套件

由於 Coil 的功能是下載顯示網路圖片，因此 App 必須有存取網路的權限。請在 AndroidManifest.xml 設定網路權限，加入 <uses-permission android:name="android.permission.INTERNET" />。

在 App 程式裡 import coil3.compose.AsyncImage 後，即可用 AsyncImage 搭配網址顯示網路圖片。

class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        enableEdgeToEdge()
        setContent {
            DemoTheme {
                Scaffold(modifier = Modifier.fillMaxSize()) { innerPadding ->
                    AsyncImage(
                        model = "https://developer.apple.com/assets/elements/icons/swiftui/swiftui-96x96_2x.png",
                        contentDescription = null,
                        modifier = Modifier.fillMaxSize().padding(innerPadding)
                    )
                }
            }
        }
    }
}
