從 GitHub 網站使用 GitHub Copilot 問問題

現在從 GitHub 網站就能使用 GitHub Copilot 了。

GitHub Copilot now available in github.com for Copilot Individual and Copilot Business plans
With this public preview, we're unlocking the context of your code and collaborators-and taking the next step in…github.blog

GitHub 網站的 GitHub Copilot 可進行以下功能。

  • 詢問程式問題。
  • 詢問檔案內容的的程式問題。
  • 詢問整個 repository 的程式問題。

詢問程式問題

登入 GitHub 網站後，從上方可找到 Chat with Copilot 按鈕。

點擊即可打開 Copilot 的聊天視窗。

任何程式問題都可以問，比方請 AI 用 SwiftUI 寫個猜數字 App。

很可惜地 AI 拒絕回答程式以外的問題。

詢問檔案內容的的程式問題

瀏覽某個檔案時，右上方會出現 Ask Copilot 的按鈕。

點擊即可打開 Copilot 視窗詢問檔案內容的的程式問題。

交作業的同學甚至可請 AI 幫程式打分數，建議改進的地方。

幫程式打分數，列出優缺點和改進的地方，滿分 100 分

詢問整個 repository 的程式問題

我們也可以針對整個 repository 詢問程式問題。連到 repository 後，點擊右上方的 Chat with Copilot 問問題。

若想詢問關於 repository 的相關問題，最好點選下圖的 Index button，幫助 AI 更了解 repository 的程式內容。

index 完成後會顯示 Indexed for improved understanding and accuracy。

完成 index 後，我們可以對整個 repository 詢問程式問題。

雖然 index 好好多多，不過有幾個要注意的小地方。

  • index 有數量限制，個人的 Copilot 訂閱者只能 index 5 個 repository。
  • 只能 index 自己有 write 權限的 repository。
