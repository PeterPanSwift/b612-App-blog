以 static 和 class 宣告型別方法的差異

類別的型別方法也可以被繼承覆寫，但是當你用 static 宣告時，是不能覆寫的。如果想把被覆寫的開關打開，請用 class 宣告。

用 class 宣告的型別方法，才能順利覆寫，定義超級寶寶的最大年齡為 1000 歲。

class Baby {
    class func getMaxAge() -> Int {
        return 100
    }
}
class SuperBaby: Baby {
    override class func getMaxAge() -> Int {
        return 1000
    }
}
SuperBaby.getMaxAge()
