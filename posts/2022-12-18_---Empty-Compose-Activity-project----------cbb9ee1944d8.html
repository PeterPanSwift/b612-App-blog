建立 Empty Compose Activity project & 顯示預覽畫面

啟動 Android Studio，選擇 Projects > New Project

選擇 template，選擇 Phone and Tablet 的 Empty Compose Activity

設定專案資訊

點選 Finish 後，有可能會出現以下畫面，表示 Android Studio 正在安裝 Android SDK。

專案畫面

成功建立專案後，畫面如下。

此時 Android Studio 還在進行一些專案的設定，從下方可看到它正在做的事情，點選 Show all 可看到它正在進行的事項清單。

視窗的右邊一開始會顯示 What’s New 說明 Android Studio 的新功能，稍微看一下後，點選 — 關掉它，把空間留給重要的程式跟 App 畫面。

關掉 What’s New 後的畫面如下。

專案設定完成

當專案設定完成後，點選下方的 Event Log 可看到 Gradle sync finished。

點選右上的 Split，同時顯示程式和程式對應的預覽畫面(preview)

點選 Build & Refresh

預覽畫面出現了
