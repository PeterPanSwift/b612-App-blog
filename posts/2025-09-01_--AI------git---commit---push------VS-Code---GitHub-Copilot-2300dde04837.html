請 AI 幫忙進行 git 的 commit & push — 使用 VS Code 的 GitHub Copilot

開發 App 時，透過 git 進行專案的版本管理是很重要的一件事。但是對於 git 不太了解怎麼辦吧 ? 沒關係，我們可以請 AI 幫忙，請它進行 git 的 commit & push。

以下我們使用 VS Code 的 GitHub Copilot 示範，使用 Codex 或 Claude Code 也是類似的方法。

當程式修改到某個想儲存的時間點時，請 AI 進行 commit 記錄目前的程式。不同於會覆蓋之前內容的存檔，每次 commit 儲存的版本都會完整保留，未來我們將可隨時切換到某個 commit 時間點的程式版本。

AI 成功完成 commit，而且還聰明地分析程式描述詳細的 commit 訊息 Change background color to green。

打開 Source Control 分頁，看到剛剛完成的 commit。

點選 Publish Branch 將專案程式上傳到 GitHub。

選擇將 GitHub 上的專案設為 private 或 public。

之後每當程式修改到某個想儲存的時間點時，我們可以直接請 AI 進行 commit & push，記錄完後直接上傳到 GitHub。

commit & push

若想打開 GitHub 的專案網頁，請先點選左下的分支圖示。

然後點選 remote branches 旁的 GitHub icon。
