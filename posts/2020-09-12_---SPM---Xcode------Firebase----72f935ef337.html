利用 SPM 在 Xcode 專案加入 Firebase 套件

Firebase 從 6.31.0 版開始，可以從 Xcode 內建的套件管理工具 SPM 安裝了 ! (須搭配 Xcode 12)

利用 SPM 在 Xcode 專案加入 Firebase 套件

對使用 SPM 安裝第三方套件還不熟的朋友可參考以下連結，接下來就讓我們試試利用 SPM 安裝 Firebase 套件吧。

使用 SPM 安裝第三方套件 — 以抓圖套件 Kingfisher 為例
開發 iOS App 時，我們有許多管理安裝第三方套件的方法，其中最為人熟知的三大套件管理天王為 CocoaPods，SPM(Swift Package Manager) & Carthage。medium.com

以下為新版 Xcode 的示範操作，使用舊版 Xcode 的同學請參考以上連結的方法 2 操作。

點選 Xcode menu 的 File > Add Packages

輸入套件的網址

https://github.com/firebase/firebase-ios-sdk
GitHub - firebase/firebase-ios-sdk: Firebase iOS SDK
This repository contains all Apple platform Firebase SDK source except FirebaseAnalytics. Firebase is an app…github.com

點選 Add Package。

勾選要安裝的套件，以下我們勾選比較常用到的三個套件:

  • FirebaseAuth

帳號登入功能。

  • FirebaseFirestoreSwift

存取 Firestore 資料庫。

  • FirebaseStorage

存取 Storage 上的檔案。

成功安裝後的畫面如下，它將聰明地安裝其它相關的套件，因此在 Package Dependencies 下可看到一長串的套件。
