使用 environment modifier 調整 UI 的相關設定

使用 environment modifier 我們可以方便地調整 UI 的相關設定，比方文字的對齊，文字的行距，是否為 dark mode 等，而且它會作用在施加的元件以及元件的 descendant view(child)上，只要設定一次，即可影響多個畫面。接下來就讓我們以實際的例子說明吧。

Apple Developer Documentation
Edit descriptiondeveloper.apple.com

各種常見的 environment 設定

設定文字的對齊為置中，行距為 10，顏色為 dark mode，字體大小為 largeTitle。從以下例子我們看到 environment 作用在 VStack，然而它將影響 VStack 包含的所有元件。

struct ContentView: View {
    
    var body: some View {
        VStack {
            Text("鬼滅之刃\n最帥的水柱")
                .background(Color.yellow)
            Button(action: {
            }, label: {
                Text("顯示答案")
            })
        }
        .environment(\.multilineTextAlignment, .center)
        .environment(\.lineSpacing, 10)
        .environment(\.colorScheme, .dark)
        .environment(\.font, .largeTitle)
        
    }
}

當 environment 作用在 NavigationView

它將影響 NavigationView 管理的 view，比方以下例子利用 NavigationLink 切換畫面，ContentView & HeroView 都會受到 environment 的影響。

@main
struct DemoApp: App {
    var body: some Scene {
        WindowGroup {
            NavigationView {
                ContentView()
            }
            .environment(\.multilineTextAlignment, .center)
            .environment(\.lineSpacing, 10)
            .environment(\.colorScheme, .dark)
            .environment(\.font, .largeTitle)
        }
    }
}
struct ContentView: View {
    
    var body: some View {
        VStack {
            Text("鬼滅之刃\n最帥的水柱")
                .background(Color.yellow)
            NavigationLink(
                destination: HeroView(),
                label: {
                    Text("顯示答案")
                })
        }
        
    }
}
struct HeroView: View {
    var body: some View {
        VStack {
            Text("富岡義勇\n生日2月8日\n水瓶座")
                .background(Color.yellow)
            Image("富岡義勇")
                .resizable()
                .scaledToFill()
        }
    }
}

若是 environment 作用在 ContentView，然後以 NavigationLink 切換到 HeroView，此時 HeroView 將不受影響。

呼叫 sheet 或 fullScreenCover 切換畫面時，新的頁面也會受 environment 影響

從 ContentView 呼叫 environment，利用 sheet 顯示的 HeroView 也受到影響。

@main
struct DemoApp: App {
    var body: some Scene {
        WindowGroup {
            NavigationView {
                ContentView()
                    .environment(\.multilineTextAlignment, .center)
                    .environment(\.lineSpacing, 10)
                    .environment(\.colorScheme, .dark)
                    .environment(\.font, .largeTitle)
            }
            
        }
    }
}
struct ContentView: View {
    @State private var show = false
    
    var body: some View {
        VStack {
            Text("鬼滅之刃\n最帥的水柱")
                .background(Color.yellow)
            Button(action: {
                show = true
            }, label: {
                Text("顯示答案")
            })
        }
        .sheet(isPresented: $show, content: {
            HeroView()
        })
        
    }
}
struct HeroView: View {
    var body: some View {
        VStack {
            Text("富岡義勇\n生日2月8日\n水瓶座")
                .background(Color.yellow)
            Image("富岡義勇")
                .resizable()
                .scaledToFill()
        }
    }
}
