抓取影片和圖片的 pixabay API

連到 pixabay 網站，註冊一個帳號

https://pixabay.com

點選 Explore，點選 API

查看 API 文件

點選 Get Started(View API documentation)。

查詢 API key

在 API 的文件頁面往下滑動，從下圖的 key 查到自己的 API key。

從 PostMan 測試 API

以搜尋影片為例。

Pixabay API
Welcome to the Pixabay API documentation. Our API is a RESTful interface for searching and retrieving free images and…pixabay.com

網址是 https://pixabay.com/api/videos/，參數如下。

在 PostMan 輸入以下網址。(ps: key 當然是假的呀，不能讓你知道，請換成自己申請的 API key)

https://pixabay.com/api/videos?q=cat&page=1&per_page=10&key=123
