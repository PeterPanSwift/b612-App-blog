memberwise init 的自動完成 — Xcode 14 新功能

從 Xcode 14 開始，Xcode 可以自動幫我們生成包含每個 property 的 memberwise init ，方法如下:

memberwise init 的自動完成

Xcode 可以自動幫我們生成包含每個 property 的 memberwise init ，方法如下。

輸入 init 後，從選單選擇 memberwise init，Xcode 將幫我們自動生成。

struct Baby {
    let name: String
    let age: Int

    init(name: String, age: Int) {
        self.name = name
        self.age = age
    }
}

剛剛的 Baby 例子是用 struct 定義，用 class 也會有一樣的效果。

若是 property 有預設值，生成的 memberwise init 也會聰明地在參數裡填入預設值。

struct Baby {
    let name: String
    var age = 3

    init(name: String, age: Int = 3) {
        self.name = name
        self.age = age
    }
}

當搭配 predictive code completion 時，Xcode 也會提供另一種 init 的提示。
