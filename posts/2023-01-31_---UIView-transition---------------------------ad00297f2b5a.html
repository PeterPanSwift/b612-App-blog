利用 UIView 的 transition(with:duration:options:animations:completion:) 可以快速實現 App 裡常見的圖片轉換動畫，以下我們介紹幾種有趣的應用。

利用 UIView.transition 製作圖片轉換動畫 — 淡入淡出，翻轉，無限循環播放

利用 UIView 的 transition(with:duration:options:animations:completion:) 可以快速實現 App 裡常見的圖片轉換動畫，以下我們介紹幾種有趣的應用。

  • 淡入淡出。
  • 翻轉(翻牌)。
  • 無限循環播放。
transition(with:duration:options:animations:completion:) | Apple Developer Documentation
Creates a transition animation for the specified container view.developer.apple.com

storyboard 的畫面如下，圖片的 outlet 是 imageView，按鈕點選會觸發 tap。

淡入淡出

@IBAction func tap(_ sender: Any) {
    UIView.transition(with: imageView, duration: 1, options: .transitionCrossDissolve) {
        self.imageView.image = UIImage(named: "blue peter")
    }
}

function transition 的宣告如下。

class func transition(
    with view: UIView,
    duration: TimeInterval,
    options: UIView.AnimationOptions = [],
    animations: (() -> Void)?,
    completion: ((Bool) -> Void)? = nil
)

參數說明

  • with: 動畫作用的 view，在此我們傳入 imageView。
  • duration: 動畫的時間，以秒為單位。
  • options: 動畫的效果，transitionCrossDissolve 表示淡入淡出。
  • animations: 設定動畫要改變的內容，在此我們將紅彼得改成藍彼得。
  • completion: 動畫完成後要做的事情。

以上程式將實現紅彼得以淡入淡出的方式變成藍彼得，動畫時間為一秒鐘。

翻轉(翻牌)

@IBAction func tap(_ sender: Any) {
    UIView.transition(with: imageView, duration: 1, options: [.transitionFlipFromLeft]) {
        self.imageView.image = UIImage(named: "blue peter")
    }
｝

傳入 transitionFlipFromLeft 實現翻轉效果，flip 系列有transitionFlipFromLeft，transitionFlipFromRight，transitionFlipFromTop，transitionFlipFromBottom 可選擇。

無限循環播放

@IBAction func tap(_ sender: Any) {        
    UIView.transition(with: imageView, duration: 0.1, options: [.transitionCrossDissolve, .repeat, .autoreverse]) {
        self.imageView.image = UIImage(named: "blue peter")
    }
 }

options 可以用 [ ] 設定多個動畫效果，我們傳入 [.transitionCrossDissolve, .repeat, .autoreverse]，repeat 讓動畫重覆播放，紅彼得變成藍彼得後，autoreverse 讓藍彼得以動畫變回紅彼得，而不是瞬間變成紅彼得。
