最近學生跟彼得潘說他可以免費使用 GitHub Copilot，因為他有學生身份。因此彼得潘也查了一下，發現老師也有這樣的福利耶。medium.com

我們可以在 GitHub 網頁，GitHub App 和各種知名的 IDE 使用 GitHub Copilot 寫程式。

  • 在 Xcode 使用 GitHub Copilot。
使用 GitHub Copilot for Xcode 幫忙寫程式
最近聽說 GitHub 推出了厲害的 AI 程式碼工具 Copilot，它可以提供不錯的程式碼建議，大幅加快程式的開發速度。medium.com
  • 在 VS Code 使用 GitHub Copilot。
  1. 點選 VS Code 上方的 Copilot 頭像。

2. 點選 Continue with GitHub 或 Continue with Google。

點選後將打開登入 GitHub 帳號的網頁。

成功登入 GitHub 後，回到 VS Code，此時畫面將顯示 Build with agent mode，可以開始在 VS Code 請 AI 寫 code 了。

從左下的按鈕可切換 Agent / Ask / Edit，建議選擇較厲害的 Agent 模式。Agent 和 Edit 模式皆可直接修改專案程式，然而 Agent 還多了自動執行指令和搜索研究專案檔案的功能。

可選擇不同的 AI model。
