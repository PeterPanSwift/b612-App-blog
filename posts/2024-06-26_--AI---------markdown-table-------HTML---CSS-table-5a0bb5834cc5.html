請 AI 寫自動化程式將 markdown table 變成漂亮的 HTML / CSS table

透過 AI 我們可以輕易製作簡單易懂的表格，不過它通常預設是 markdown 格式，因此畫面比較陽春單調。

最近彼得潘研究一下後，發現 AI 一暝大一吋，它現在已能製作精美表格，只要指定它使用特定格式，比方 HTML / CSS、SVG 或 React。

使用 Claude Artifacts 製作精美表格
從前透過 AI 我們可以製作 makrdwn 格式的表格，例如以下 SwiftUI & Flutter 的比較表。medium.com

不過每次下 prompt 麻煩 AI 製作表格也挺費時的，而且每次做出的風格都不一樣，無法保證品質。

因此彼得潘想到可以請 AI 用 HTML / CSS 設計幾款漂亮的表格，然後再請它寫自動化程式將 markdown table 變成套用樣式的 HTML / CSS 表格。
