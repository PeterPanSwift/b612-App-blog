Android Studio 的 Find Action 快速鍵衝突問題

在 Android Studio 我們時常利用 Find Action 的快速鍵 cmd + shift + a 輸入指令，不過最近彼得潘按下 cmd + shift + a 遇到了 Shortcuts conflicts 的問題，原來它跟 macOS 內建的快速鍵起了衝突。

點選 Modify Shortcuts 後看到 Find Actions 的快速鍵有衝突。

甚至它還會在按下 cmd + shift + a 時跳出 Terminal 視窗，無法正常地輸入指令。

解決的方法是關掉 macOS 內建快速鍵的 cmd + shift + a。

打開 System Settings App，點選 Keyboard 下的 Keyboard shortcuts。

點選 Services，展開 Text 下的項目。

取消 Search man Page in Terminal 的勾選。
