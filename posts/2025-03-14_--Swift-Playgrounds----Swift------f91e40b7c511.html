從 Swift Playgrounds 練習 Swift 程式語法

Swift Playgrounds 除了能開發 SwiftUI App，也能練習一些基礎的 Swift 程式語法。方法如下。

從 Swift Playgrounds 練習 Swift 程式語法

Swift Playgrounds 除了能開發 SwiftUI App，也能練習 Swift 程式語法，主要的差別在建立 Playground 時的選擇。開發 SwiftUI App 要選擇 New App，練習程式語法則選擇 New Book。

建立 Book

  • iPad

點選 New Playground。

點選 Book。

  • Mac

從 Menu 點選 File > New Book。

寫程式 & 執行程式

在剛剛建立的檔案寫程式，點選 Run My Code 執行程式。

執行後每行程式的結果會顯示在右邊，123 表示結果是數字。

若結果是字串則會顯示 abc。

123 和 abc 只告訴我們結果的類型，必須點選後才能看到結果。下圖顯示 sum 的內容為 3。

若想更快看到結果，可點選 Add Viewer，之後每次點選 Run My Code 時結果將直接顯示在程式下方。
