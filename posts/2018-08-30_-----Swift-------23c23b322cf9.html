一行宣告 Swift 的變數常數

在寫 Swift 程式時，我們習慣一行宣告一個變數(常數)，類似以下的例子:

struct Baby {
   var name: String
   var weight: Double
   var height: Double
}

如果你懶得打那麼多 var，也可以全部寫在同一行，只要一個 var，不同變數間用逗號分開。

struct Baby {
   var name: String, weight: Double, height: Double
}

當多個變數是同一個型別時，也可以偷懶，只寫一次型別。例如以下例子，var weight, height: Double，先將多個變數的名字寫完，最後再加上冒號和型別。

struct Baby {
   var name: String
   var weight, height: Double
}
