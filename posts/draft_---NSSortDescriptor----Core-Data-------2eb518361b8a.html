利用 NSSortDescriptor 設定 Core Data 的排序條件


let request = Location.fetchRequest()
request.predicate = NSPredicate(format: "name CONTAINS[c] %@", "apple")
request.sortDescriptors = [
   NSSortDescriptor(keyPath: \Location.time, ascending: false)
]
