在 Medium 建立收藏文章的 List

最近 Medium 推出了方便收藏文章的 List 功能，我們可以建立各種不同的 List，比方 SwiftUI 密技 & UIKit 密技，然後將自己或別人的寫的 medium 文章加到 List。

以下我們來看看如何使用 List 吧。

建立 List

  • 方法 1

點選左邊的 Lists 按鈕，切換到 Lists 頁面。

點選右上的 New list。

輸入 list 的名稱，說明和是否隱藏。

  • 方法 2

點選文章下方的 Save 按鈕，然後點選 Create New list。

輸入 list 的名稱，說明和是否隱藏。點選 Create 建立後，剛剛的文章將自動被加到 list 裡。

將文章加到 List

點選文章下方的 Save 按鈕。

勾選想加入的 list，可勾選多個。

將文章從 List 中移除

點選文章下方的 Save 按鈕，然後取消 list 的勾選。

調整 List 中文章的順序

切換到 Lists 頁面，然後點選想調整的 List。

點選右上方的三個三圓點，然後點選 Reorder Items。

上下拖曳清單裡的文章調整順序。
