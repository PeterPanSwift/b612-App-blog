一鍵複製貼上選取的程式段落(duplicate) — Xcode 14 新功能

從 Xcode 14 開始，我們終於可以一鍵複製貼上選取的程式段落了。

以下我們以戴佩妮新專輯好聽的歌曲後果自負說明。

彼得潘覺得後果自負歌詞我多想學你一樣恍恍惚惚 為愛而不得如泣如訴寫得超棒的，想要讓它出現兩次。

struct ContentView: View {
    var body: some View {
        VStack {
            Image("被動的觀眾")
                .resizable()
                .scaledToFit()
            Text("我多想學你一樣恍恍惚惚 為愛而不得如泣如訴")
        }
    }
}

將滑鼠游標移到 Text 歌詞上。

或是選取整行 Text 歌詞。

點選 Xcode menu 的 Duplicate，或是按快速鍵 cmd + d。

Cool，Xcode 成功地複製貼上 Text("我多想學你一樣恍恍惚惚 為愛而不得如泣如訴")。

我們也可以一次複製貼上多行程式，比方選取下圖的 4 行 Text。

然後按 cmd + d 複製貼上。

值得注意的，我們剛剛選取多行文字時，特別多出下方空白的一行。

它雖然是什麼都沒有的空白，但卻很重要。為什麼呢 ? 讓我們試試只選取四行 Text，沒有多出下方空白的一行。

按 cmd + d 複製貼上後，將如下圖所示，貼上的 Text 直接長在原本 Text 的後面，造成紅色錯誤，變成我們要麻煩地自己手動將 Text 換行。
