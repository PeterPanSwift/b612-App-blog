讓 Xcode 的 editor 最大化，隱藏 navigator，inspector，debug area

開發 iOS App 時，Xcode 的四大區塊 Editor，Navigator，Inspector & Debug Area 各司其職，協助我們開發 App。

不過寫程式時我們主要都在 Editor 區塊操作，此時若想讓寫程式的空間最大化，也可以考慮將暫時用不到的 navigator，inspector，debug area 隱藏。

隱藏 navigator

快速鍵 cmd + 0。

其它方法: 點選下圖紅色框框區塊。

隱藏 inspector

快速鍵 cmd + option + 0。其實滿好記的，隱藏左邊的 navigator 是 cmd + 0，隱藏右邊的 inspector 是 cmd + option + 0，差一個 option。

其它方法: 點選下圖紅色框框區塊。

隱藏 debug area

快速鍵 cmd + shift + y。

若是懶得手動隱藏 debug area，也可以另外設定停止 App 時自動隱藏 debug area，相關說明可參考以下連結。

停止 App 時自動隱藏 debug area
