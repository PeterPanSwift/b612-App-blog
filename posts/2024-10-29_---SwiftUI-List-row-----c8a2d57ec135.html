客製 SwiftUI List row 的形狀

當我們用 SwiftUI 的 List 顯示清單時，row 預設會是簡單的白色背景(dark mode 是黑色背景)，就像以下用 List 顯示的乒乓男孩主題曲接住我歌詞。

struct ContentView: View {
    let lyrics = [
        "右手晴天",
        "左手雨天",
        "就讓彩虹 擦亮我的臉"
    ]

    var body: some View {
        NavigationStack {
            List(lyrics, id: \.self) { lyric in
                HStack {
                    Text(lyric)
                    Spacer()
                    Image(systemName: "heart")
                }
            }
            .navigationTitle("乒乓男孩主題曲")
        }
    }
}

若想要 row 有特別的形狀，常見的方法有兩種。

  • 方法 1: 利用 background modifier 設定 row 的背景形狀。
  • 方法 2: 利用 listRowBackground 設定 row 的背景形狀。

方法 1

利用 listRowBackground(Color.clear) 讓 row 的背景變成透明，background modifier 設定 row 的背景形狀。若想設定 List 的背景，必須呼叫 scrollContentBackground(.hidden) 或 listStyle(.plain) 取消 List 預設的背景。

利用 SwiftUI 的 scrollContentBackground 取消 List / Form 預設的背景
當我們使用 SwiftUI 的 List / Form 設計 App 畫面時，它會預設有背景顏色，就像以下可愛 emoji 動物清單的例子。medium.com
struct ContentView: View {
    let lyrics = [
        "右手晴天",
        "左手雨天",
        "就讓彩虹 擦亮我的臉"
    ]

    var body: some View {
        NavigationStack {
            List(lyrics, id: \.self) { lyric in
                HStack {
                    Text(lyric)
                    Spacer()
                    Image(systemName: "heart")
                }
                .padding()
                .background {
                    UnevenRoundedRectangle(bottomTrailingRadius: 20)
                        .foregroundStyle(.yellow)
                }
                .listRowBackground(Color.clear)
                .listRowSeparator(.hidden)
            }
            .scrollContentBackground(.hidden)
            .background(.pink)
            .navigationTitle("乒乓男孩主題曲")
        }
    }
}

如果沒有將 listRowBackground 設為透明，row 的背景預設將為白色(在 dark mode 為黑色)。

方法 2

listRowBackground 設定 row 的背景形狀。

struct ContentView: View {
    let lyrics = [
        "右手晴天",
        "左手雨天",
        "就讓彩虹 擦亮我的臉"
    ]
    
    var body: some View {
        NavigationStack {
            List(lyrics, id: \.self) { lyric in
                HStack {
                    Text(lyric)
                    Spacer()
                    Image(systemName: "heart")
                }
                .padding()
                .listRowBackground(
                    UnevenRoundedRectangle(bottomTrailingRadius: 20)
                        .foregroundStyle(.yellow)
                        .padding()
                )
                .listRowSeparator(.hidden)
            }
            .scrollContentBackground(.hidden)
            .background(.pink)
            .navigationTitle("乒乓男孩主題曲")
        }
    }
}
