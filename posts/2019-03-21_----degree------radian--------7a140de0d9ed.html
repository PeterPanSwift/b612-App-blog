將角度(degree)變成弧度(radian) 的三種方法

開發 iOS App 時，我們時常使用弧度(radian)，但是我們熟悉的卻是角度(degree)，因此時常需要將角度轉換成弧度。

弧度 pi 等於角度 180 度，因此想要換算其實不難。不過轉換的方法很多，有些方法我們甚至不用自己計算呢。接下來就讓我們看看角度轉弧度的三種方法。

方法一: 利用單位轉換的 Measurement
let radian = Measurement(value: 30, unit: UnitAngle.degrees).converted(to: UnitAngle.radians)

從 radian.value 我們可取得數值，不過它的型別是 Double，很多 iOS SDK 要求的 radian 型別都是 CGFloat，所以記得要轉換成 CGFloat。

let view = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
let blueView = UIView(frame: CGRect(x: 30, y: 30, width: 30, height: 30))
view.addSubview(blueView)
blueView.backgroundColor = UIColor.blue
let radian = Measurement(value: 30, unit: UnitAngle.degrees).converted(to: UnitAngle.radians)
blueView.transform = CGAffineTransform(rotationAngle: CGFloat(radian.value))

方法二: 利用 CGFloat.pi 計算

CGFloat.pi 等於 180 度，因此先將它除以 180，得到角度一度的弧度後，即可乘上角度取得弧度。

let radian = CGFloat.pi / 180 * 30
方法三: 利用 GLKit framework 的 function
import GLKit
let radian = GLKMathDegreesToRadians(30)
