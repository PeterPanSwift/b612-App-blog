利用 frame 或 Grid / GridRow 解決 SwiftUI Slider 排列不整齊的問題

開發 SwiftUI iOS App 時，我們常會用 Slider 調整數值，並在 Slider 的左邊顯示文字說明 Slider 的用途。

不過當畫面上同時有多個 Slider，而且左邊文字的長度不一致時，將造成 Slider 排列不整齊的問題。

上圖的程式範例如下。

import SwiftUI

struct ContentView: View {
    @State private var red = 1.0
    @State private var green = 1.0
    @State private var blue = 1.0
    @State private var opacity = 1.0

    var body: some View {
        VStack {
            Image(.lion)
                .resizable()
                .scaledToFill()
                .frame(width: 300, height: 300)
                .background(Color(red: red, green: green, blue: blue, opacity: opacity))
                .clipShape(.circle)
            Form {
                ColorSlider(title: "Red", color: .red, value: $red)
                ColorSlider(title: "Green", color: .green, value: $green)
                ColorSlider(title: "Blue", color: .blue, value: $blue)
                ColorSlider(title: "Opacity", color: .yellow, value: $opacity)
            }
        }
    }
}

struct ColorSlider: View {
    let title: String
    let color: Color
    @Binding var value: Double
    
    var body: some View {
        HStack {
            Text(title)
            Slider(value: $value)
                .tint(color)
        }
    }
}

此問題有許多解法，以下我們介紹兩種解法。

  • 使用 frame 設定文字固定寬度。
  • 使用 Grid 和 GridRow 排版。

使用 frame 設定文字固定寬度

使用 frame 設定文字固定寬度並將 alignment 設為 leading，讓文字統一寬度，因此 Slider 也會跟著有一樣的寬度。

struct SliderRow: View {
    let title: String
    let color: Color
    @Binding var value: Double
    
    var body: some View {
        HStack {
            Text(title)
                .frame(width: 60, alignment: .leading)
            Slider(value: $value)
                .tint(color)
        }
    }
}

使用 Grid 和 GridRow 排版

Grid 和 GridRow 適合建立表格般的排版，讓多行排列的元件在水平方向對齊。我們可用它實現兩個 column，第一個 column 塞文字，第二個 column 塞 Slider，如此 Slider 即可有著一樣的寬度。

struct ContentView: View {
    @State private var red = 1.0
    @State private var green = 1.0
    @State private var blue = 1.0
    @State private var opacity = 1.0

    var body: some View {
        VStack {
            Image(.lion)
                .resizable()
                .scaledToFill()
                .frame(width: 300, height: 300)
                .background(Color(red: red, green: green, blue: blue, opacity: opacity))
                .clipShape(.circle)
            Form {
                Grid(alignment: .leading) {
                    SliderRow(title: "Red", color: .red, value: $red)
                    SliderRow(title: "Green", color: .green, value: $green)
                    SliderRow(title: "Blue", color: .blue, value: $blue)
                    SliderRow(title: "Opacity", color: .yellow, value: $opacity)
                }
            }
        }
    }
}

struct SliderRow: View {
    let title: String
    let color: Color
    @Binding var value: Double
    
    var body: some View {
        GridRow {
            Text(title)
            Slider(value: $value)
                .tint(color)
        }
    }
}

值得注意的，為了讓 cell 裡的文字靠左對齊，Grid 的 alignment 要從預設的置中改成 leading。
