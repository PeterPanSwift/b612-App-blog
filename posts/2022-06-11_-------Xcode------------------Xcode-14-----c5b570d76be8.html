輸入 } 時 Xcode 會自動縮排 { } 裡的程式 — Xcode 14 新功能

從 Xcode 14 開始，當我們輸入 } 時， Xcode 會貼心地幫我們自動縮排 { } 裡的程式，我們再也不用手動按 ctrl + i 縮排了。

範例 1: 加入 if

我們只想在對方是真愛時印出你給我的一切 純屬意外，因此我們在 print 前輸入 if isTrueLove {。

func test() {
    let isTrueLove = Bool.random()
    if isTrueLove {
    print("你給我的一切 純屬意外")
}

然後我們將游標移到第 16 行，準備在第 16 行輸入 }。

輸入 } 後神奇的事發生了 ! if { } 裡的內容自動向右縮排。

func test() {
    let isTrueLove = Bool.random()
    if isTrueLove {
        print("你給我的一切 純屬意外")
    }
}

範例 2: 加入 VStack

我們想將 Text 包在 VStack 裡，因此我們在 Text 前輸入 VStack {。

struct ContentView: View {
    var body: some View {
        VStack {
        Text("你給我的一切 純屬意外")
            .font(.largeTitle)
        
    }
}

然後我們將游標移到第 15 行，準備在第 15 行輸入 }。

輸入 } 後神奇的事發生了 ! VStack { } 裡的內容自動向右縮排。

struct ContentView: View {
    var body: some View {
        VStack {
            Text("你給我的一切 純屬意外")
                .font(.largeTitle)
        }
    }
}
