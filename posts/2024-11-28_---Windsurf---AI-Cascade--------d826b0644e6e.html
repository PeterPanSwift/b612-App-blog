使用 Windsurf 的 AI Cascade 修改專案程式

現在 IDE 的 AI 功能愈來愈厲害，許多 IDE 的 AI 都可以分析整個專案，直接對專案的多個檔案進行修改。

使用 Windsurf 的 AI Cascade 修改專案程式

現在 IDE 的 AI 功能愈來愈厲害，許多 IDE 的 AI 都可以分析整個專案，直接對專案的多個檔案進行修改。

以下彼得潘以 IDE Windsurf 為例，示範利用 Windsurf 的 AI Cascad 修改 Xcode 開發的 SwiftUI App 專案，請 Cascad 替推理小說作家增加國家欄位並在畫面上顯示國家資訊。

import Foundation

struct Author: Identifiable, Hashable {
    let id = UUID()
    let name: String
    let image: String
    let books: [Book]
}

下載安裝 Windsurf

連到 Windsurf 網頁，點選 Download the Windsurf Editor。

Windsurf Editor by Codeium
Tomorrow’s editor, today. Windsurf Editor is the first AI agent-powered IDE that keeps developers in the flow…codeium.com

將 Windsurf 移到 Applications。

啟動 Windsurf，點選 Get started。

點選 Start fresh 進行相關設定，或是點選 Import from VS Code / Cursor 加入 VS Code / Cursor 的設定。

選擇喜歡的鍵盤快速鍵(keybindings)設定。

選擇喜歡的配色樣式(theme)。

使用 Windsurf 必須搭配 Codeium 帳號，在此彼得潘點選 Sign up 註冊帳號。

成功註冊帳號後，點選 Allow 打開 Windsurf App。

使用 AI Cascade 修改專案程式

打開 Windsurf App，點選 Open Folder 打開專案的資料夾。

以下彼得潘打開 Xcode 專案的資料夾。

從右邊的輸入框跟 AI Cascade 聊天。以下我們請它幫 Author 增加 property country。

ps: Write 的開關記得打開，它表示 AI 將直接對程式進行修改。

Cascade 聰明地分析資料夾的內容後找出需調整的檔案。以下 Cascade 找到 Author.swift & SampleData.swift 並進行修改。

我們可檢視 Author.swift & SampleData.swift 的改動，選擇接受改動或拒絕改動。

Cascade 在 struct Author 增添 country。

Cascade 在 SampleData 進行的修改更令人驚豔，它不只幫資料加入 country，而且連填寫的內容也完全正確。Cascade 不只會寫程式，對推理小說也略懂，它也知道克莉絲蒂是英國作家，伊坂幸太郎是日本作家。

成功增添 Author 的 country 後，接著讓我們調整畫面，在作家列表頁面顯示國家資訊。

請 Cascade 在 AuthorList 顯示 country。

Cascade 聰明地找出 AuthorList 利用 AuthorRow 顯示作家資訊，因此它修改 AuthorRow 顯示國家資訊。
