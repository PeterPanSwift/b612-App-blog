SwiftUI 的 preview 文字 ... 問題

在 Xcode 11.4 的 SwiftUI preview 觀看 App 畫面時，有時會出現多行文字變 ... 的顯示問題。例如以下顯示 Penny 歌詞之間的例子。

struct ContentView: View {
    var body: some View {
        Text("""
誰的咖啡讓你覺得有點苦澀 誰的眼神 顯得不捨
他的出現到底屬於什麼顏色 黑的白的 都有可能
在往返之間 我已漸漸 失去了平衡點
在雙唇之間 不經意透露 一種寓言
你的擁抱讓我變得有點羞澀 你的忌妒 我哭笑不得
我的疑惑來自你所有的譴責 我叛逆任性 你能不能負荷
在往返之間 我已漸漸 失去了平衡點
在雙唇之間 不經意透露 一種寓言
在去留之間 微笑再見 我卻留下眼淚
在我們之間 缺乏語言 你可否聽見
我站在站在城市的邊緣 不斷不斷不斷的眷戀 墮落但淒美
站在站在幸福的對面 不斷不斷自我的催眠 錯過也很完美
在你發現之前 我也許已經 故意的疏遠
在你醒來之前 請做好準備 隨時崩潰
""")
        .padding()
        .background(Color.yellow)
        
    }
}

其實 App 在模擬器或實機執行時是沒問題的，文字可以完整地顯示 。因此有時遇到 preview 文字顯示不完整的問題時，記得要從模擬器測試，也許只是 Xcode preview 的 bug。
