利用 SwiftUI 的 scaledToFit 顯示完整圖片

製作 iOS App 畫面時，有時我們會想要圖片維持比例地佔滿可用空間，而且整張圖都能完整顯示，比方常見的照片瀏覽頁面，在 SwiftUI 有個很簡單的方法可實現這樣的效果。

我們針對 Image 呼叫 scaledToFit，命令它維持比例地佔滿可用空間完整顯示。

struct ContentView: View {
    var body: some View {
        Image(.star)
            .resizable()
            .scaledToFit()
    }
}

如下圖所示，此時無敵星星爆米花桶圖片的寬度等於螢幕寬度，但是為了維持比例，所以上下留白。

當然，它不一定是上下留白，如下圖所示，當手機轉成橫向時(landscape)，變成左右留白。

當 frame 只指定寬度(高度)時，scaledToFit 將自動依圖片比例計算高度(寬度)。

struct ContentView: View {
    var body: some View {
        Image(.star)
            .resizable()
            .scaledToFit()
            .frame(width: 300)
    }
}

使用 List 製作表格頁面時我們也可以使用這樣的技巧，讓 row 裡的圖片完整顯示並且維持比例地佔滿空間 。

struct ContentView: View {
    var body: some View {
        List {
            HStack {
                Image(.star)
                    .resizable()
                    .scaledToFit()
                Text("無敵星星爆米花桶")
            }
            HStack {
                Image(.usj)
                    .resizable()
                    .scaledToFit()
                Text("環球影城")
            }
        }
    }
}

如下圖所示，圖片會盡可能地佔滿可用空間，不過它並不會填滿螢幕的寬度，因為 List 預設會讓 Row 裡的內容和左右邊界有間距。
