請 AI 寫程式解題並測試結果是否正確

我們現在不只能請 GPT 寫程式，還可以請他執行 Python 程式，實現一些方便的應用。

以下我們示範其中一種應用，請 GPT 寫程式解題，然後測試結果是否正確。

ps: 看來很適合程式考試作弊(誤)。

我們以 zerojudge 上的題目 Sagit’s 計分程式為例。

a053. Sagit's 計分程式 - 高中生程式解題系統
sagit 是一位高中電腦老師，這學期正在教學生寫C++程式。他的評分標準是依照每一位學生在 ZeroJudge…zerojudge.tw

請 GPT 寫 python 程式解題，使用 input 讀取輸入的資料。然後用 Sample Input 輸入資料，比對 Sample Output 檢查程式是否正確。

GPT 的回答如下，它不只貼心寫出 Python 程式並執行程式，比對 Sample Output 檢查程式是否正確。

產生的 python 程式如下。

def calculate_score(N):
    if N <= 10:
        score = N * 6
    elif N <= 20:
        score = 10 * 6 + (N - 10) * 2
    elif N <= 40:
        score = 10 * 6 + 10 * 2 + (N - 20) * 1
    else:
        score = 100
    return score

# Test the function with sample inputs
print(calculate_score(10))  # Expected output: 60
print(calculate_score(40))  # Expected output: 100
