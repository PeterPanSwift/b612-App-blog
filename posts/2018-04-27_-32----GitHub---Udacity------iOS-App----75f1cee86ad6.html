#32 研究 GitHub 上 Udacity 學生們的 iOS App 程式

目的: 研究 GitHub 上 Udacity 學生們的 iOS App 程式，開發 Udacity iOS App 課程的 4 個 App。

Become an iOS Developer | Udacity
Learn iOS app development and become an iOS developer with our iOS Developer Nanodegree. Use Swift programming language…www.udacity.com

線上課程網站 Udacity 推出的 Become an iOS Developer 滿不錯的，提供 免費的課程影片，從基本的 UI 畫面打造到網路串接跟資料儲存，教學生開發 4 個完整的 App。如果對 iOS App 開發已有一定的了解，不想再重頭看影片，也可以直接參考 GitHub 上 Udacity 學生們的 iOS App 專案練功。

  1. 下載學生的專案，執行 App 後參考它的操作，試著自己開發一模一樣的 App。
  2. 研究學生在 GitHub 的程式如何實作出 App 的功能。
  3. 修改學生的程式，擴充 App 的功能。

比方以下同學們都已完成課程的 4 個 App，可參考研究他們的程式，或是另外在 GitHub 搜尋。

rajanikantdeshmukh/iOS-Nanodegree-Projects
iOS-Nanodegree-Projects - Master repository of projects for Udacity's iOS Nanodegree.github.com
paterik (Patrick Paechnatz)

deborahepadilla (Deborah E. Padilla)

如何搜尋

1 在 GitHub 網站上搜尋 Udacity 課程的 App 名字加上 Udacity。4 個 App 的名字如下
(1) Pitch Perfect
(2) MemeMe
(3) On the Map
(4) Virtual Tourist

比方你可搜尋 udacity On the Map

2 語言選擇 Swift，因為我們只對 Swift 寫的版本有興趣。

3 可從右上角的 Sort 選擇排序，比方 Most stars 是以星星數排序，Recently updated 將先顯示新的專案。

注意最好選擇新一點的專案，搭配的 Swift ，Xcode 和 iOS SDK 才會比較新，比較不會有問題。
