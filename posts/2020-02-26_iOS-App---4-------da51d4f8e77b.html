iOS App 的 4 種系統字體

設計 iOS App 畫面時，文字的字體是其中很重要的一環，感謝 Apple 早為我們準備了許多字體任君選擇。若是你跟彼得潘一樣有選擇障礙，那也不用擔心，不知如何選擇的話就選使用者熟悉的系統字體吧。

Apple 提供的系統字體有以下幾種，iOS App 可使用的有 SF Pro，SF Pro Rounded，SF Mono & New York。開發 App 時，若我們沒有特別設定，預設將採用 SF Pro。(ps: SF Compact 適用於 watchOS App)

接下來我們分別介紹如何在 SwiftUI & UIKit 使用 Font.Design & UIFontDescriptor.SystemDesign 設定系統字體。

  • 預設採用 SF Pro

生成 Font 時傳入 text style，或是以 system(size:) 生成 Font 時傳入字體大小。

VStack {
    Text("Apple")
        .font(.largeTitle)
    Text("Apple")
        .font(.system(size: 50))
}
.padding()
  • 以 Font.Design 指定系統字體

以 Font.Design 指定採用的系統字體，.default 對應 SF Pro，.rounded 對應 SF Pro Rounded，.monospaced 對應 SF Mono，.serif 則對應 New York。

例子1。

用 function system 產生 Font 時傳入字體的大小，weight(粗細) & design(型別為 Font.Design)。

VStack {
    HStack {
        Text("SF Pro")
        Text("Apple")
            .font(.system(size: 100, weight: .regular, design: .default))
    }
    HStack {
        Text("SF Pro Rounded")
        Text("Apple")
            .font(.system(size: 100, weight: .regular, design: .rounded))
    }
    HStack {
        Text("SF Mono")
        Text("Apple")
            .font(.system(size: 100, weight: .regular, design: .monospaced))
    }
    HStack {
        Text("New York")
        Text("Apple")
            .font(.system(size: 100, weight: .regular, design: .serif))
    }
}

例子2。

用 function system 產生 Font 時傳入字體的 text style & design(型別為 Font.Design)。

VStack {
    HStack {
        Text("SF Pro")
        Text("Apple")
            .font(.system(.largeTitle, design: .default))
    }
    HStack {
        Text("SF Pro Rounded")
        Text("Apple")
            .font(.system(.largeTitle, design: .rounded))
    }
    HStack {
        Text("SF Mono")
        Text("Apple")
            .font(.system(.largeTitle, design: .monospaced))
    }
    HStack {
        Text("New York")
        Text("Apple")
            .font(.system(.largeTitle, design: .serif))
    }
}

UIKit & system font

  • 預設採用 SF Pro

方法1: 從 Interface Builder。

設定 Font 時選擇 System 或 Text Styles。

方法2: 從程式。

呼叫 UIFont 的 systemFont(ofSize:) 或 preferredFont(forTextStyle:)。

let font1 = UIFont.systemFont(ofSize: 24)
let font2 = UIFont.preferredFont(forTextStyle: .largeTitle)
  • 以 UIFontDescriptor.SystemDesign 指定系統字體

以 UIFontDescriptor.SystemDesign 選擇想要的系統字體，跟剛剛 SwiftUI 的例子一樣，選項有 .default，.monospaced，.rounded & .serif 四種。不過它的程式比 SwiftUI 複雜，我們必須先產生 UIFontDescriptor，然後再呼叫它的 function withDesign(_:)，最後在生成 UIFont 時再傳入 UIFontDescriptor。

if let fontDescriptor = UIFontDescriptor.preferredFontDescriptor(withTextStyle: .largeTitle).withDesign(.rounded) {
    let font = UIFont(descriptor: fontDescriptor, size: 0)
}

值得注意的，以 init(descriptor:size:) 生成 UIFont 時，size 傳入 0 生成的字體大小將為 UIFontDescriptor 設定的 text style，size 大於 0 時字體大小則以 size 的數字為準，比方以下例子的 100。

if let fontDescriptor = UIFontDescriptor.preferredFontDescriptor(withTextStyle: .largeTitle).withDesign(.rounded) {
    let font = UIFont(descriptor: fontDescriptor, size: 100)
}

ps: 如果想要產生固定大小的 SF Mono，也可以採用以下比較精簡的寫法，呼叫 function monospacedSystemFont(ofSize:weight:) & monospacedDigitSystemFont(ofSize:weight:)。

let font1 = UIFont.monospacedSystemFont(ofSize: 20, weight: .bold)
let font2 = UIFont.monospacedDigitSystemFont(ofSize: 20, weight: .bold)
