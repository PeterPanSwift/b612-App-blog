沒有畫面的 Xcode

有朋友遇到一個奇怪的問題。他說他明明點了 Xcode 的 App Icon，但卻什麼都看不到，就像利綺唱的，永遠有多遠我看不見。

其實 Xcode 有成功啟動，如下圖所示，上方的 menu 顯示著 Xcode 的 menu。它只是沒有顯示視窗，所以我們什麼都看不到。

讓 Xcode 顯示視窗有以下兩種方法

方法 1 : 點選 Xcode menu 的 Window > Welcome to Xcode

Welcome to Xcode 視窗的畫面如下，建議勾選下方的 Show this window when Xcode launches，如此可保證每次點選到 Xcode App icon 都會顯示此視窗。

方法 2 : 點選 Xcode menu 的 File > New > Project 或 Playground
