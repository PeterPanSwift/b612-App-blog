#147 串接 Imgur API，開發包含按讚、留言、follow 等社群功能的 App

練習開發 iOS App 時，實作一個包含按讚、留言、follow 等社群功能的 App 是個滿不錯的目標，而且也可以當個不錯的轉職作品。

不過要實現社群功能有很大一部分需要搭配後台，如果不會寫後台，或是沒朋友，找不到合作的後台工程師朋友呢 ?

最近研究了一下，照片社群網站 Imgur 提供豐富的 API，幾乎包含了常見的社群功能，有興趣的朋友不妨考慮串接 Imgur API 開發一個社群 App。

Imgur API
image](https://i.imgur.com/n744BL9.png) ## API Status Status for the API can be found at…apidocs.imgur.com

基本功能

  • 搜尋照片，顯示照片牆。
  • OAuth 2.0 登入。
  • 上傳照片。
  • 對照片按讚(Favorite)。
  • 對照片留言(comment)。
  • 自己和別人的個人頁面。

包含名字，大頭照，上傳的照片清單，按讚的照片清單等。

  • follow user。

進階功能

Imgur 不只支援照片，也支援影片，因此我們也可以在 App 裡加入影片功能。

  • 上傳影片。
  • 顯示影片牆。
