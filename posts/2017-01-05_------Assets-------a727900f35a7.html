將圖片加到 Assets 的多種方法

方法一: 複製貼上

  • 當圖片是網路上的檔案時。
將網頁上的圖片複製貼上到 Xcode 的 Assets
當我們在網頁上看到不錯的圖片時，利用複製貼上可以快速將圖片加到 Xcode 專案的 Assets。medium.com

加入圖片後，在名字上點擊或按 return 鍵可改名。

  • 當圖片是 Mac 裡的檔案時。

比方我們想將 Finder 裡可愛的 peter 圖片 20200104br.png 加到 Assets，點選它後按下 cmd + c 複製。(ps: 也可以一次複製多張圖片)

進入 Xcode 專案的 Assets 頁面，點擊下圖的紅色框框。

按下 cmd + v 貼上，圖片順利地出現在 Assets 裡。

  • 當圖片以 Preview App 打開時。

按下 cmd + c 複製。

切換到 Xcode 專案的 Assets 頁面，按下 cmd + v 貼上。

  • 當圖片在 Mac Photos App 時

從 Photos App 點選圖片後，按下 cmd + c 複製。

切換到 Xcode 專案的 Assets 頁面，按下 cmd + v 貼上。

方法二: 拖曳圖片

當圖片是 Mac 裡的檔案時，我們可從 Finder 將它拖曳到下圖 Assets 的紅色框框裡。(ps: 也可一次拖曳多張圖片)

方法三: 點選 import 加入圖片

點選 Assets，從 Quick Actions 點選 Add New Asset > Import。

從 Finder 視窗選擇想加入的圖片。(可多選)

透過資料夾管理 Xcode asset 的圖片和顏色
開發 App 時，我們會在 Xcode asset 存放 App 需要的素材，像是圖片和顏色，不過當素材很多時，asset 裡的東西將變得十分凌亂，此時可透過資料夾分類管理。medium.com
