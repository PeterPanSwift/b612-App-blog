請 AI 翻譯 Swift Programming Language Book

彼得潘最近試了一下，請 AI 對 Swift Programming Language Book 的章節全文翻譯，發現效果挺不錯的，有興趣的朋友也可以試試。

以下我們用 Zed IDE 的 AI 進行翻譯，由於 Swift Programming Language Book 網站的格式比較特別，Zed 無法抓取網頁內容，我們改抓取 Swift Book 章節在 GitHub 的 md 檔。

Documentation

連到 Swift Book 的 GitHub

GitHub - swiftlang/swift-book: The Swift Programming Language book
The Swift Programming Language book. Contribute to swiftlang/swift-book development by creating an account on GitHub.github.com

從 TSPL.docc 下找尋感興趣的內容。

以下我們想翻譯 LanguageGuide 的 Closures.md。

swift-book/TSPL.docc/LanguageGuide/Closures.md at main · swiftlang/swift-book
The Swift Programming Language book. Contribute to swiftlang/swift-book development by creating an account on GitHub.github.com

點選 Raw 查詢 Closures.md 的檔案網址。

Closures.md 的網址是 https://raw.githubusercontent.com/swiftlang/swift-book/main/TSPL.docc/LanguageGuide/Closures.md。

使用 Zed IDE 的 fetch 指令抓取 Closures.md，然後請 Zed AI 將它翻成台灣中文。

AI 的翻譯結果。
