cmd + xxx 的 Xcode 快速鍵

Xcode 很多的快速鍵都是以 cmd + xxx 組合，接下來讓我們看看一些 cmd + xxx 組合的快速鍵。

  • build & run app
cmd + r
  • stop app
cmd + .
  • build app
cmd + b
  • new file
cmd + n
  • toggle comments (//)
cmd + /
  • Undo
cmd + z
  • search 目前編輯檔案的內容
cmd + f
  • find next
cmd + g
  • shift left (將程式左移)
cmd + [
  • shift right(將程式右移)
cmd + ]
  • 跳到檔案的第一行
cmd + ↑
  • 跳到檔案的最後一行
cmd + ↓
  • 跳到行首
cmd + ←
  • 跳到行尾
cmd + →
  • 跳到某一行
cmd + l
  • 打開 Preferences 視窗
cmd + ,
  • 清空 console
cmd + k
  • 開關 navigator
cmd + 0
