SwiftUI 顯示網頁的 WebView & WebPage

從 iOS 26 開始，SwiftUI App 可以直接用原生的 WebView 在 App 內顯示網頁。

顯示彼得潘的 blog 超簡單的 ! 只要 import WebKit，即可使用搭配 URL 的 WebView 顯示網頁。

import SwiftUI
import WebKit

struct ContentView: View {
    var body: some View {
        WebView(url: URL(string: "https://apppeterpan.medium.com"))
    }
}

若想對網頁進行更多操作，比方呼叫 JavaScript，則可搭配 WebPage。以下我們在點擊按鈕時透過 WebPage 的 load(_:) 動態載入彼得潘和文組班的 App 課程網頁。

import SwiftUI
import WebKit

struct ContentView: View {
    let webPage = WebPage()
    
    var body: some View {
        VStack {
            WebView(webPage)
                .clipShape(.rect(cornerRadius: 20))
                .shadow(radius: 10)
                .padding()
            Group {
                Button {
                    webPage.load(URLRequest(url: URL(string: "https://www.apppeterpan.com")!))
                } label: {
                    Text("彼得潘的 Swift iOS App 程式設計入門")
                        .frame(maxWidth: .infinity)
                }
                Button {
                    webPage.load(URLRequest(url: URL(string: "https://learniosfromzero.mystrikingly.com")!))
                } label: {
                    Text("文組生的 iOS SwiftUI App 程式設計入門")
                        .frame(maxWidth: .infinity)
                }
            }
            .buttonStyle(.glass)
            .padding(.horizontal, 20)

        }
       
    }
}

以下例子先用 for try await _ in webpage.load 等待網頁載好，然後透過 WebPage 的 callJavaScript 執行 JavaScript，將網頁的背景顏色改成黃色。

import SwiftUI
import WebKit

struct ContentView: View {
    let webpage = WebPage()

    var body: some View {
        WebView(webpage)
            .task {
                do {
                    for try await _ in webpage.load(
                        URL(string: "https://www.google.com")
                    ) {
                    }
                    let js = """
                        const style = document.createElement('style');
                        style.textContent = 'html, body { background: yellow !important; }';
                        document.head.appendChild(style);
                        """
                    try await webpage.callJavaScript(js)
                } catch {
                    print(error)
                }
            }
    }
}

以上只是 WebView & WebPage 最基本跟常用的功能介紹，有興趣的朋友可進一步參考官方文件探索進階功能。
