調整 SwiftUI Image 顯示的區塊

當我們使用 SwiftUI Image 顯示圖片時，時常會用 scaledToFill & frame 設定圖片維持比例填滿指定的寬高，然後再用 clipped 切除超過 frame 的區塊。

用以下 AI 繪製的小王子狐狸圖片為例，clipped 在切除時將預設保留圖片中間的區塊，切除上下或左右超出的區塊。

struct ContentView: View {
    var body: some View {
        Image(.prince)
            .resizable()
            .scaledToFill()
            .frame(width: 200, height: 400)
            .clipped()
    }
}

沒有 clipped 時，超過 200 * 400(藍色框框)的區塊仍然會顯示。

從上圖的藍色框框可看出圖片預設會置中於框框裡 ，因此 clipped 切除時將切除左右超出的區塊。

若想調整圖片 clip 後保留的區塊，可透過以下方法。

  • 設定 frame 的 alignment。
  • 設定 offset。
  • 利用 CGImage 的 cropping 指定圖片顯示的區塊。

設定 frame 的 alignment

frame 的 alignment 預設為 .center，因此圖片預設置中於 frame 的框框。只要調整 alignment，即可控制 clip 時保留的區塊。

以下例子將 alignment 設為 .leading，因此圖片的左邊將對齊框框的左邊，clipped 將保留圖片左半邊 200 points 的寬度 (假設系統的語言設定是由左到右的語言)。

struct ContentView: View {
    var body: some View {
        Image(.prince)
            .resizable()
            .scaledToFill()
            .frame(width: 200, height: 400, alignment: .leading)
            .clipped()
    }
}

設定 offset

利用 offset 偏移圖片，x -70 將讓圖片往左移動 70 points。

struct ContentView: View {
    var body: some View {
        Image(.prince)
            .resizable()
            .scaledToFill()
            .frame(width: 200, height: 400, alignment: .topLeading)
            .offset(x: -70)
            .clipped()
    }
}

利用 CGImage 的 cropping 指定圖片顯示的區塊

若想精細地指定圖片顯示的區塊，則可利用 CGImage 的 cropping function。

以下例子將顯示圖片左下 1/4 的區塊。

import SwiftUI

extension UIImage {
    func crop(rect: CGRect) -> UIImage {
        guard let cgImage = self.cgImage?.cropping(to: rect) else {
            return self
        }
        return UIImage(cgImage: cgImage)
    }
}

struct ContentView: View {
    var body: some View {
        let uiImage = UIImage.prince
        let halfWidth = uiImage.size.width / 2
        let halfHeight = uiImage.size.height / 2
        let cropRect = CGRect(
            x: 0,
            y: halfHeight,
            width: halfWidth,
            height: halfHeight
        )
        Image(uiImage: uiImage.crop(rect: cropRect))
            .resizable()
            .scaledToFit()
            .frame(width: 300)
    }
}
