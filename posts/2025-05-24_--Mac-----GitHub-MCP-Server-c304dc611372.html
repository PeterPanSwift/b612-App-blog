在 Mac 上設定 GitHub MCP Server

最近 GitHub 推出了 GitHub MCP Server，方便我們透過 AI 執行 GitHub 的相關操作。

以下我們介紹 Mac 上設定 GitHub MCP Server 的相關步驟。

連到 GitHub MCP Server 網頁，點選 Remote GitHub MCP Server 下 VS Code 的 Install Server。

GitHub — github/github-mcp-server: GitHub’s official MCP Server
GitHub’s official MCP Server. Contribute to github/github-mcp-server development by creating an account on GitHub.github.com

ps: 其它 IDE 的操作可參考連結的說明。

從跳出的 VS Code 選單點選 Install Server。

點選 Allow 進行 GitHub 帳號認證的動作。

從打開的網頁點選 Continue 登入 GitHub 帳號。

點選 Authorize Visual-Studio-Code。

完成認證後，點選 Allow 回到 VS Code。

成功安裝 GitHub MCP Server，VS Code 的 settings.json 裡 MCP 的設定區塊多了 github 的設定。

現在我們將 Copilot 設為 Agent 模式時，將可透過 GitHub MCP Server 進行 GitHub 的相關操作，點選 Select Tools 按鈕可看到 MCP Server: github 的相關工具。
