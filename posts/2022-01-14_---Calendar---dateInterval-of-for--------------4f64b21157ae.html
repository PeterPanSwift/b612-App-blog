利用 Calendar 的 dateInterval(of:for:) 取得某單位的時間區間

開發 iOS App 時，有時我們會想取得某單位的開始時間或結束時間，比方現在時間是 2022/1/14 9:56 AM，我們想取得的時間是以整點為單位，因此想要的是 2022/1/14 9:00 AM。

透過 Calendar 的 dateInterval(of:for:)，我們可以取得某單位的時間區間，實現剛剛提到的功能，接下來讓我們看看幾個不同的例子。

以小時為單位

let now = Date.now
if let dateInterval = Calendar.current.dateInterval(of: .hour, for: now) {
    dateInterval.start
    dateInterval.end
}

以天為單位

取得今天開始跟結束的時間。

let now = Date.now
if let dateInterval = Calendar.current.dateInterval(of: .day, for: now) {
    dateInterval.start
    dateInterval.end
}

ps: 取得今天開始的時間也可透過 startOfDay(for:)。

let startOfDay = Calendar.current.startOfDay(for: now)

以星期為單位

取得星期開始跟結束的時間。

let now = Date.now
if let dateInterval = Calendar.current.dateInterval(of: .weekOfMonth, for: now) {
    dateInterval.start
    dateInterval.end
}

以下例子的 1/9 & 1/16 都是星期天。

以月為單位

取得月份開始跟結束的時間。

let now = Date.now
if let dateInterval = Calendar.current.dateInterval(of: .month, for: now) {
    dateInterval.start
    dateInterval.end
}
