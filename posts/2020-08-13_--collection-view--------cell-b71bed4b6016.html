讓 collection view 捲動到某一個 cell

在 iOS App 上我們時常看到類似下圖的照片牆設計，點選小圖的照片後切換到下一頁顯示大圖，然後在大圖頁面左右滑動瀏覽其它圖片。

讓 collection view 捲動到某一個 cell

在 iOS App 上我們時常看到類似下圖的照片牆設計，點選小圖的照片後切換到下一頁顯示大圖，然後在大圖頁面左右滑動瀏覽其它圖片。

collection view 很適合用來實現大圖的分頁照片瀏覽。在剛剛的例子，當我們點選第五個小圖進到大圖頁面時，collection view 應該要顯示第五個 cell 的大圖，而非顯示第一個 cell 的大圖。

透過呼叫 collection view 的 function scrollToItem(at:at:animated:)，我們可控制它捲動到某一個 cell。

func scrollToItem(at indexPath: IndexPath, 
               at scrollPosition: 
               UICollectionView.ScrollPosition, 
               animated: Bool)

相關程式範例如下:

class FullPhotoCollectionViewController: UICollectionViewController {
    
    var index: Int!
    var isShow = false
    
    override func viewDidLayoutSubviews() {
        super.viewDidLayoutSubviews()
        if isShow == false {
            collectionView.scrollToItem(at: IndexPath(item: index, section: 0), at: .left, animated: false)
            isShow = true
        }
    }

說明:

index 代表要顯示的 cell 編號(編號從 0 開始)，前一頁顯示小圖的 controller 會將 index 的數字傳給 FullPhotoCollectionViewController，讓 FullPhotoCollectionViewController 知道要顯示哪一個 cell。值得注意的，我們要在 viewDidLayoutSubviews 呼叫 scrollToItem(at:at:animated:)，若是在 viewDidLoad 呼叫可能會出問題。另外由於 viewDidLayoutSubviews 可能被多次觸發，所以我們利用 isShow 判斷，只在第一次呼叫時捲動 collection view。

ps: 若發現大圖顯示時位置有點偏移，可在大圖的 controller 取消 Adjust Scroll View Insets 的勾選。
