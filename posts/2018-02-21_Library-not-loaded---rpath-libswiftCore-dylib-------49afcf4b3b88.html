Library not loaded: @rpath/libswiftCore.dylib 的閃退問題

從 Xcode 將 App 安裝到 iPhone 時，發現它馬上閃退，出現以下的錯誤訊息:

Library not loaded: @rpath/libswiftCore.dylib 的閃退問題

從 Xcode 將 App 安裝到 iPhone 時，發現它馬上閃退，出現以下的錯誤訊息:

dyld: Library not loaded: @rpath/libswiftCore.dylib
Referenced from: /var/containers/Bundle/Application/1A07316C-21A1-4A78-A34F-E15A6199FB49/Demo.app/Demo
Reason: no suitable image found.  Did find:
/private/var/containers/Bundle/Application/1A07316C-21A1-4A78-A34F-E15A6199FB49/Demo.app/Frameworks/libswiftCore.dylib: code signature invalid for '/private/var/containers/Bundle/Application/1A07316C-21A1-4A78-A34F-E15A6199FB49/Demo.app/Frameworks/libswiftCore.dylib'

別緊張，這不是你的錯。只要點選 Xcode menu 的 Product > Clean，將東西清掉，然後重新 build 安裝到 iPhone，即可順利執行。
