解析 menu

可不可的 menu。

輸入 /fetch 指令，在 fetch 後加上網址。

按下 enter 後變成 @ fetch，表示已抓到資料。

點選 @ fetch 可展開查看抓到的資料。

解析 GitHub 的程式

Apple 的範例 sample-backyard-birds。

GitHub - apple/sample-backyard-birds
Contribute to apple/sample-backyard-birds development by creating an account on GitHub.github.com
這段代碼定義了一個名為 `BackyardList` 的 SwiftUI View 結構體,用於顯示後院列表。主要功能包括:

1. 使用 @Query 從 SwiftData 中獲取並排序後院數據。

2. 接受 isSubscribed、backyardLimit 和 onOfferSelection 作為初始化參數。

3. 根據訂閱狀態決定是否顯示優惠卡片。

4. 使用 List 顯示後院列表,每個項目是一個 NavigationLink。

5. 每個後院項目顯示為 BackyardViewport,並在上方覆蓋顯示後院名稱。

6. 提供了滑動操作來關閉優惠卡片。

7. 使用條件渲染來控制優惠卡片的顯示。

8. 在 watchOS 上使用輪播樣式的列表。

整體而言,這個視圖提供了一個靈活的後院列表界面,可以根據用戶的訂閱狀態和其他條件來調整顯示內容。它結合了 SwiftUI、SwiftData 和 StoreKit 等框架的功能,以創建一個交互式的列表視圖。

fetch 指令的限制

fetch 指令雖然強大，但也有一些限制。

  • 有些網頁無法抓取。

無法取得統一獅 YouTube 頁面的影片清單。

  • 抓不到圖片的網址。

目前 fetch 似乎只能抓到網頁的文字資料，無法取得網頁裡的圖片網址。

以 Amazon 網站查詢 SwiftUI 相關商品為例。

AI 可以解析出商品的名字，作者和價錢，但是無法取得圖片網址。
