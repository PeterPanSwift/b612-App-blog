請 AI 幫忙進行 git 的 commit & push — 使用 GitHub Copilot for Xcode

開發 App 時，透過 git 進行專案的版本管理是很重要的一件事。但是對於 git 不太了解怎麼辦吧 ? 沒關係，我們可以請 AI 幫忙，請它進行 git 的 commit & push。

以下我們使用 GitHub Copilot for Xcode 示範。

ps: GitHub Copilot for Xcode 的安裝說明可參考以下連結。

使用 GitHub Copilot for Xcode 幫忙寫程式
最近聽說 GitHub 推出了厲害的 AI 程式碼工具 Copilot，它可以提供不錯的程式碼建議，大幅加快程式的開發速度。medium.com

當程式修改到某個想儲存的時間點時，請 AI 進行 commit 記錄目前的程式。不同於會覆蓋之前內容的存檔，每次 commit 儲存的版本都會完整保留，未來我們將可隨時切換到某個 commit 時間點的程式版本。

AI 成功完成 commit，而且還聰明地分析程式描述詳細的 commit 訊息 Add red background that fills entire screen to ContentView。

打開 Source Control navigator，看到剛剛完成的 commit。

從右鍵選單點選 New remote 將專案程式上傳到 GitHub。

ps: 如果還沒在 Xcode 設定 GitHub 帳號，請在上傳到 GitHub 前先設定。

Xcode 的 GitHub 帳號設定
Xcode 的 GitHub 帳號設定medium.com

Visibility 選擇 Public，點選 Create 建立。

ps: 除了自己手動將程式上傳到 GitHub，我們也可以請 AI 透過 GitHub MCP Server 上傳，相關說明可參考以下連結。

透過 GitHub Copilot for Xcode 請 AI 將 Xcode 專案上傳到 GitHub
透過 GitHub Copilot for Xcode & GitHub MCP Server 我們可以從 Xcode 控制 AI 執行 GitHub 的相關操作，以下彼得潘以 Xcode 專案為例，示範從 Xcode 出一張嘴叫 AI…medium.com

之後每當程式修改到某個想儲存的時間點時，我們可以直接請 AI 進行 commit & push，記錄完後直接上傳到 GitHub。

commit & push
