SwiftUI preview 的 Failed to build the scheme 問題

只要專案裡有一個地方出現紅色錯誤，SwiftUI 將無法顯示 preview。

比方以下程式原本正常顯示好書彼得潘的 Swift 程式設計入門的圖片。

import SwiftUI

struct ContentView: View {
    var body: some View {
        Image("book")
            .resizable()
            .scaledToFit()
    }
}

#Preview {
    
    ContentView()
    
}

然而只要專案裡有個地方出現紅色錯誤，SwiftUI 的 preview 將無法顯示。以下我們故意讓 SecondView.swift 的程式少了一個 )，造成紅色錯誤。

此時完全無錯的 ContentView 將無法顯示 preview，顯示錯誤 Failed to build the scheme xxx，因為它被其它地方的錯誤拖累了。

因此之後在 Preview 看到 Failed to build the scheme 時，記得先到 Issue navigator 查詢是否有其它錯誤。

有錯時也可直接從右鍵選單點選 Generate Fix for Issue 請 AI 幫忙修正。
