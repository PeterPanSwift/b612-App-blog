SwiftUI 讓 view 變圓形的各種方法

開發 iOS App 時，我們習慣設定 cornerRadius 讓東西變圓形，不過 SwiftUI 其實有很多方法讓東西變圓形，而且還比 cornerRadius 直覺，不用再手動計算 cornerRadius 的數字呢。

SwiftUI 讓 view 變圓形的各種方法

  • 方法 1: clipShape 搭配 circle。
  • 方法 2: mask 搭配 circle。
  • 方法 3: 使用 background(_:in:fillStyle:) 設定圓形背景。
  • 方法 4: cornerRadius (deprecated)。

clipShape 搭配 circle

struct ContentView: View {
   var body: some View {
       Image(.peter)
         .resizable()
         .scaledToFill()
         .frame(width: 200, height: 200)
         .clipShape(.circle)
   }
}

利用 clipShape(_:style:) 將 view 裁切成某個形狀，傳入 .circle 或 Circle() 裁切成圓形。

func clipShape<S>(_ shape: S, style: FillStyle = FillStyle()) -> some View where S : Shape
clipShape(_:style:) | Apple Developer Documentation
Sets a clipping shape for this view.developer.apple.com
struct ContentView: View {
   var body: some View {
      Image(.peter)
         .resizable()
         .scaledToFill()
         .frame(width: 200, height: 200)
         .clipShape(.circle)
         .overlay {
             Circle()
                 .stroke(.white, lineWidth: 4)
         }
         .shadow(radius: 10)
   }
}

mask 搭配 circle

struct ContentView: View {
   var body: some View {
       Image(.peter)
         .resizable()
         .scaledToFill()
         .frame(width: 200, height: 200)
         .mask {
             Circle()
         }
   }
}

利用 mask(_:) 將 view 遮罩成某個形狀，傳入 Circle() 將 view 遮罩成圓形。

mask(alignment:_:) | Apple Developer Documentation
Masks this view using the alpha channel of the given view.developer.apple.com

使用 background(_:in:fillStyle:) 設定圓形背景

struct ContentView: View {
    var body: some View {
        Text("請馴養我\nPlease tame me")
            .font(.largeTitle)
            .frame(width: 200, height: 200)
            .background(.yellow, in: .circle)
    }
}
利用 SwiftUI 的 background(_:in:fillStyle:) 設定背景，用顏色、漸層、圖片填滿形狀

cornerRadius (deprecated)

struct ContentView: View {
    var body: some View {
        Image(.peter)
            .resizable()
            .scaledToFill()
            .frame(width: 200, height: 200)
            .cornerRadius(100)
    }
}
