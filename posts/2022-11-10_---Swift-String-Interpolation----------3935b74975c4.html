輸入 Swift String Interpolation \( ) 的方法

開發 iOS App 時，我們時常使用到 String Interpolation \( )。最近發現 Xcode 超貼心的，現在我們在 \( 裡輸入內容後，Xcode 會自動幫我們補上 )，不用再自己打 ) 了 !

以下我們實際來實驗看看吧。

  • 在 " "裡輸入 \( 。
let name = "Sandy"
let lyrics = "愛你就打 I love \("
  • 輸入 n，Xcode 自動補上 )。

選擇 name 按 enter，完成輸入。

let name = "Sandy"
let lyrics = "愛你就打 I love \(name)"

ps: 歌詞參考華仔有趣的 I MISS YOU。
