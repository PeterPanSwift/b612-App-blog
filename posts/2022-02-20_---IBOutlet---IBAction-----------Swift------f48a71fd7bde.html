拉完 IBOutlet / IBAction 後，快速切到連線的 Swift 程式編輯

開發 iOS App 時，拉完 IBOutlet / IBAction 後，我們的下一步通常是切到連線的 swift 程式，讓它出現在主畫面的編輯視窗再繼續編輯。

最近研究了一下，發現有以下幾種可行的方法。

  • 利用 Open in Tab 快速鍵 cmd + option + o。
  • 從 Quick Actions 輸入 tab，選擇 Open in Tab。
  • 利用 Open in 快速鍵，cmd + option + shift + < + enter。
  • 從 Project Navigator 點選連線的 swift 檔。
  • 利用 Open Quickly 的快速鍵，cmd + shift + o
  • 利用 Reveal in Project Navigator 的快速鍵，cmd + shift + j

利用 Open in Tab 快速鍵 cmd + option + o

點選右邊 assistant editor 的 swift 檔。

按下 Open in Tab 的快速鍵 cmd + option + o。

從 Quick Actions 輸入 tab，選擇 Open in Tab

點選右邊 assistant editor 的 swift 檔。

按 cmd + shift + a 叫出 Quick Actions，輸入 tab，然後選擇 Open in Tab。

利用 Open in 快速鍵，cmd + option + shift + < + enter

點選右邊 assistant editor 的 swift 檔。

按下 Open in 的快速鍵 + enter，因此是按下 cmd + option + shift + < + enter。

從 Project Navigator 點選連線的 swift 檔。

若檔案太多，也可先從 Project Navigator 下的 filter 框框輸入關鍵字過濾。

利用 Open Quickly 的快速鍵，cmd + shift + o

按下 Open Quickly 的快速鍵 cmd + shift + o，輸入檔名關鍵字後按 enter，比方輸入 remli 找到 ReminderListViewController 後按 enter。

利用 Reveal in Project Navigator 的快速鍵，cmd + shift + j

點選右邊 assistant editor 的 swift 檔，然後按下 cmd + shift + j，讓檔案在 Project Navigator 裡被選取，最後再用滑鼠點選被選取的檔案。
