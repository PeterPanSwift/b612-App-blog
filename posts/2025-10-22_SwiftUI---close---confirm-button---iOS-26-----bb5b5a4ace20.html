SwiftUI 的 close & confirm button — iOS 26 新功能

SwiftUI 在 iOS 26 新增了常用的 close & confirm button，我們不用再辛苦地用 SF Symbol(比方 xmark.circle.fill & checkmark.circle.fill) 或 asset 圖片製作關閉和完成的 button 了。

  • close button。

使用 Button(role: .close) { } 生成 close button。

struct AnswerView: View {
    @Environment(\.dismiss) var dismiss
    
    var body: some View {
        ZStack {
            Image(.pic)
                .resizable()
                .scaledToFill()
                .frame(minWidth: 0, maxWidth: .infinity)
                .ignoresSafeArea()
                .blur(radius: 15)
            Text("7 個星球")
                .font(.largeTitle)
        }
        .overlay(alignment: .topTrailing, content: {
            Button(role: .close) {
                dismiss()
            }
            .padding()
        })
        .foregroundStyle(.white)
    }
}

button 預設會顯示文字，透過 labelStyle 可調整顯示圖片或文字。

只顯示圖片。

Button(role: .close) {
    dismiss()
}
.labelStyle(.iconOnly)

只顯示文字。

Button(role: .close) {
    dismiss()
}
.labelStyle(.titleOnly)

顯示圖片和文字。

Button(role: .close) {
    dismiss()
}
.labelStyle(.titleAndIcon)

close button 圖案的樣式則可透過 symbolVariant 調整，比方 .circle.fill 設定填滿圓形的樣式。

Button(role: .close) {
    dismiss()
}
.labelStyle(.iconOnly)
.symbolVariant(.circle.fill)

利用 buttonStyle 可進一步調整 button 的樣式。

Button(role: .close) {
    dismiss()
}
.labelStyle(.iconOnly)
.buttonStyle(.glass)
Button(role: .close) {
    dismiss()
}
.labelStyle(.iconOnly)
.buttonStyle(.glassProminent)

如果頁面有搭配 NavigationStack，透過 toolbar 設定 close button 時預設將顯示圓形填滿的圖案。

struct AnswerView: View {
    @Environment(\.dismiss) var dismiss
    
    var body: some View {
        NavigationStack {
            ZStack {
                Image(.pic)
                    .resizable()
                    .scaledToFill()
                    .frame(minWidth: 0, maxWidth: .infinity)
                    .ignoresSafeArea()
                    .blur(radius: 15)
                Text("7 個星球")
                    .font(.largeTitle)
            }
            .toolbar(content: {
                Button(role: .close) {
                    dismiss()
                }
            })
            .foregroundStyle(.white)
        }
    }
}

透過 tint 可設定 toolbar 的 button 顏色。

.toolbar(content: {
    Button(role: .close) {
        dismiss()
    }
    .tint(.orange)
})
  • confirm button。

寫法類似 close button，以下為一些參考的範例。

Button(role: .confirm) {
    
}

圖案為勾勾。

Button(role: .confirm) {
    
}
.labelStyle(.iconOnly)

toolbar 的 confirm button 預設為圓形填滿的勾勾。

.toolbar(content: {
    Button(role: .confirm) {
        
    }
})

透過 tint 可設定圓形背景的顏色。

.toolbar(content: {
    Button(role: .confirm) {
        dismiss()
    }
    .tint(.orange)
})
