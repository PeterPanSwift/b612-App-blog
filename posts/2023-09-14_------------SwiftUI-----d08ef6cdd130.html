如何快速將多個元件包在 SwiftUI 容器裡

開發 SwiftUI App 時，我們有 2 個方法可以快速將多個元件包在 SwiftUI 的容器裡。

  • 方法 1: 輸入型別名稱 {，然後再輸入 }。
  • 方法 2: 選取多個元件後輸入 {。

以下我們假設有兩個 Text 元件，目標將它包在 VStack 裡。

Text("航海王")
Text("草帽海賊團")

方法 1: 輸入型別名稱 {，然後再輸入 }

輸入 VStack {。

然將游標移到第 15 行，準備在第 15 行輸入 }。

輸入 } 後神奇的事發生了 ! VStack { } 裡的內容自動向右縮排。

struct ContentView: View {
    var body: some View {
        VStack {
            Text("航海王")
            Text("草帽海賊團")
        }
    }
}

方法 2: 選取多個元件後輸入 {

選取兩個 Text。

輸入 {，Xcode 自動幫我們輸入對應的 }。

在 { 前輸入 VStack。

struct ContentView: View {
    var body: some View {
        VStack {
            Text("航海王")
            Text("草帽海賊團")
        }
    }
}
