升級安裝 macOS beta

從 macOS 13.4 Ventura 開始，升級 beta 版的 macOS 變得更容易了，我們可以直接從 System Settings App 升級 beta 版的作業系統。

以下我們以升級macOS Tahoe 26 beta 為例說明。

打開 System Settings App，搜尋 sof，點選 Software Update。

點選 Beta Updates 右邊的 ⓘ。

Beta Updates 選擇 macOS Tahoe 26 Developer Beta。

點選 macOS Beta 右邊的 Upgrade Now。

點選 Agree。

輸入 Mac 的密碼。

開始下載安裝 macOS 26 Beta。

成功安裝重開機後，全新的 macOS 26 桌面出現眼前。

打開 About This Mac 確認一下，作業系統真的變成了 macOS 26。
