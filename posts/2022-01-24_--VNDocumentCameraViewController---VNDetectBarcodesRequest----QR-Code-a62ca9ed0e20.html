使用VNDocumentCameraViewController & VNDetectBarcodesRequest 掃描 QR Code

開發 iOS App 時，我們時常需要加入掃描 QR Code 的功能。從 iOS 13 開始，Apple 提供了專門掃描圖片的 controller VNDocumentCameraViewController，開發掃描功能變得簡單許多。

接下來我們將一步步示範如何使用VNDocumentCameraViewController & VNDetectBarcodesRequest 掃描 QR Code。

在 storyboard 設計畫面，加入點擊開始掃描的 button

連結按鈕的 IBAction

@IBAction func scan(_ sender: Any) {
      
}

加入請求存取相機的權限

為了掃描 QR Code，我們必須徵求使用者的同意才能存取相機。在 TARGETS App 的 Info 頁面加入 Privacy — Camera Usage Description，設定請求相機權限的文字。

加入 framework Vision & VisionKit

顯示掃描畫面的 VNDocumentCameraViewController 來自 VisionKit，檢查 QR Code 的 VNDetectBarcodesRequest 來自 Vision。

import AVFoundation
import Vision
import VisionKit

顯示掃描圖片的 VNDocumentCameraViewController ，設定它的 delegate

顯示 VNDocumentCameraViewController。

@IBAction func scan(_ sender: Any) {
    let documentCameraViewController = VNDocumentCameraViewController()
    documentCameraViewController.delegate = self
    present(documentCameraViewController, animated: true)
}

讓 ViewController 遵從 protocol VNDocumentCameraViewControllerDelegate。

extension ViewController: VNDocumentCameraViewControllerDelegate {
    
   
}

讀取掃描的圖片 & 停止掃描

我們想在使用者按下 Save 時讀取掃描的圖片 & 停止掃描，因此我們定義 protocol VNDocumentCameraViewControllerDelegate 的 documentCameraViewController(_:didFinishWith:)，它將在使用者按下 Save 時觸發。

extension ViewController: VNDocumentCameraViewControllerDelegate {
    func documentCameraViewController(_ controller: VNDocumentCameraViewController, didFinishWith scan: VNDocumentCameraScan) {
        let image = scan.imageOfPage(at: scan.pageCount - 1)
        print(image)
        dismiss(animated: true, completion: nil)
    }
}

從參數 scan 的 imageOfPage(at:) 可取得掃描的圖片，型別為 UIImage。我們利用 scan.imageOfPage(at: scan.pageCount — 1) 取得最後一張掃描的圖片，然後呼叫 dismiss 停止掃描 & 關閉掃描畫面。

使用 VNImageRequestHandler & VNDetectBarcodesRequest 解析圖片裡的 QR Code

我們的目標是圖片裡的 QR Code，因此以下我們定義 function processImage(image:)，使用 VNImageRequestHandler & VNDetectBarcodesRequest 解析圖片裡的 QR Code 。

func processImage(image: UIImage) {
    guard let cgImage = image.cgImage else {
        print("Failed to get cgimage from input image")
        return
    }
    let handler = VNImageRequestHandler(cgImage: cgImage)
    let request = VNDetectBarcodesRequest { request, error in
        if let observation = request.results?.first as? VNBarcodeObservation,
           observation.symbology == .qr {
            print(observation.payloadStringValue ?? "")
        }
    }
    do {
        try handler.perform([request])
    } catch {
        print(error)
    }
}

說明

  • 產生 VNImageRequestHandler
guard let cgImage = image.cgImage else {
   print("Failed to get cgimage from input image")
   return
}
let handler = VNImageRequestHandler(cgImage: cgImage)

VNImageRequestHandler 可以幫我們解析圖片，不過它只能解析 CGImage 或 CIImage，因此我們必須先用 image.cgImage 讀取型別 CGImage 的圖片。

  • 找出圖片的 QR Code 內容
let request = VNDetectBarcodesRequest { request, error in
   if let observation = request.results?.first as? VNBarcodeObservation,
      observation.symbology == .qr {
          print(observation.payloadStringValue ?? "")
   }
}
do {
   try handler.perform([request])
} catch {
   print(error)
}

VNImageRequestHandler 十分強大，可以解析圖片裡的各種內容，不過我們必須先告訴它要解析什麼。在此我們產生 VNDetectBarcodesRequest，然後呼叫 perform 時傳入 request，告訴它我們要解析的是 barcode。(ps: barcode 包含很多種，除了 QR Code 外，ean8 & ean13 也是常見的 barcode。)

產生 VNDetectBarcodesRequest 時搭配的 init 如下，傳入的 closure 將在檢查到 barcode 時執行。

init(completionHandler: VNRequestCompletionHandler? = nil)

VNRequestCompletionHandler 的定義如下，我們可從參數 VNRequest 的 property results 讀取檢查到的資料。

public typealias VNRequestCompletionHandler = (VNRequest, Error?) -> Void

results 的型別為 [VNObservation]?，在此我們找出它的第一筆，將它轉型為 VNBarcodeObservation，然後檢查它是否是 QR Code。若以上檢查都成立，即可從 payloadStringValue 讀取 QR Code 的內容。

if let observation = request.results?.first as? VNBarcodeObservation,
   observation.symbology == .qr {
          print(observation.payloadStringValue ?? "")
}

呼叫 processImage(image:) 解析圖片

extension ViewController: VNDocumentCameraViewControllerDelegate {
    func documentCameraViewController(_ controller: VNDocumentCameraViewController, didFinishWith scan: VNDocumentCameraScan) {
        let image = scan.imageOfPage(at: scan.pageCount - 1)
        processImage(image: image)
        dismiss(animated: true, completion: nil)
    }
}
