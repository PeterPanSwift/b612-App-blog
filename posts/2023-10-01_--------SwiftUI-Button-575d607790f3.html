方便顯示圖片的 SwiftUI Button

從 iOS 14 開始，SwiftUI 提供更方便的 init 創建包含圖文的 Button。

以下我們示範幾種常見的例子。

顯示 SF Symbol (iOS 14 以上)

struct ContentView: View {
    var body: some View {
        Button("play", systemImage: "play.fill") {
            print("play")
        }
    }
}

呼叫 font 調整文字和圖片的大小。

struct ContentView: View {
    var body: some View {
        Button("play", systemImage: "play.fill") {
            print("play")
        }
        .font(.system(size: 50))
    }
}

呼叫 buttonStyle & tint 設定 button 的樣式。

struct ContentView: View {
    var body: some View {
        Button("play", systemImage: "play.fill") {
            print("play")
        }
        .font(.system(size: 50))
        .buttonStyle(.borderedProminent)
        .tint(.orange)
    }
}

顯示 asset 的圖片 (iOS 17 以上)

  • 寫法 1
struct ContentView: View {
    var body: some View {
        Button("peter", image: .peter) {
            print("peter")
        }
    }
}
  • 寫法 2
struct ContentView: View {
    var body: some View {
        Button("peter", image: ImageResource(name: "peter", bundle: .main)) {
            print("peter")
        }
    }
}

呼叫 font 可調整文字的大小，不過不能改變圖片的大小。

struct ContentView: View {
    var body: some View {
        Button("peter", image: .peter) {
            print("peter")
        }
        .font(.system(size: 50))
    }
}

只顯示圖片

呼叫 labelStyle，傳入 .iconOnly。

struct ContentView: View {
    var body: some View {
        Button("play", systemImage: "play.fill") {
            print("play")
        }
        .font(.system(size: 50))
        .labelStyle(.iconOnly)
    }
}

在 navigation bar 上顯示 button

只會顯示圖片，文字不會顯示。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Text("Hello SwiftUI")
                .toolbar(content: { 
                    Button("Add", systemImage: "plus") {
                        
                    }
                })
        }
    }
}
