storyboard copy paste controller 的三個方法

方法 1 從畫布選取 controller 的長方形框框後複製貼上

從畫布選取 controller 的長方形框框後，按 cmd +d 複製貼上。

方法 2 按住 option，然後將 Document Outline 裡的 view controller 拖曳到畫布上

按住 option，然後將 Document Outline 裡的 view controller 拖曳到畫布上，出現綠色的 + 圖示後放開。

方法 3 點選 controller ，然後按 cmd + c & cmd + v 複製貼上

在 Document Outline 點選 controller，然後按 cmd +c & cmd + v 複製貼上。

若是點選畫布裡的 controller 按 cmd + c 也可以，不過此時必須先點選畫布空白的地方，然後再按 cmd + v 才能貼上。
