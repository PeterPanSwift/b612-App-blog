利用快速鍵移動到行尾跟新增一行

打完一行呼叫 function 的程式時，為了輸入下一行程式，我們通常要按向右鍵或 ctrl + e，讓游標移到 ) 的右邊，然後再按 enter 鍵，繼續在下一行寫程式。

不過向右鍵跟 ctrl + e 都有點難按。因此接下來彼得潘將自訂 shift + enter 的快速鍵， 實現快速移動到行尾跟新增一行的功能。

打開設定 Key Bindings 的 IDETextKeyBindingSet.plist

Xcode 的 Key Bindings 可設定操作的快速鍵，我們現在想新增移動到行尾跟新增一行的功能，因此我們必須新增一個 key binding。

請在 terminal 輸入以下指令，打開設定 Key Bindings 的 IDETextKeyBindingSet.plist。

open "/Applications/Xcode.app/Contents/Frameworks/IDEKit.framework/Resources/IDETextKeyBindingSet.plist"

新增移動到行尾跟新增一行的 key binding

在 Insertions and Indentations 下新增 key binding，key 輸入 Move to End and Insert Newline，Value 輸入 moveToEndOfParagraph:, insertNewline:。指令 moveToEndOfParagraph: 將讓游標移動到行尾，insertNewline: 將新增一行，因此結合即可同時移動到行尾跟新增一行。

打開 Xcode 的 Key Bindings 頁面，設定 Move to End and Insert Newline 的快速鍵

設定快速鍵為 shift + enter。

不過 shift + enter 會跟 Run Playground to Current Line的快速鍵衝突，因此我們將 Run Playground to Current Line 的快速鍵改成 fn + shift + enter。

Cool ! 現在我們打完一行呼叫 function 的程式時，只要按下 shift + enter，即可繼續在下一行寫程式。
