幫 VS Code 資料夾和檔案換上美麗 icon 的 Material Icon Theme

VS Code 專案預設的模樣如下，看起來有點單調，資料夾名字旁只有箭頭，沒有任何區分的圖案。

沒關係，我們很快就可以將它從醜小鴨變天鵝，VS Code 有許多幫資料夾和檔案換上美麗 icon 的 File Icon Theme，待會我們將為它裝上其中最有名的 Material Icon Theme。

Material Icon Theme - Visual Studio Marketplace
Extension for Visual Studio Code - Material Design Icons for Visual Studio Codemarketplace.visualstudio.com

打開 VS Code 的 Extensions 分頁，找到 Material Icon Theme 後點選 Install 安裝。

裝好後點選 Set File Icon Theme。

點選 Material Icon Theme。

VS Code 成功變身 ! 專案的資料夾和檔案都換上了新衣，穿上容易識別的美麗 icon。

之後若想修改 File Icon Theme，可從 menu 點選 Settings > Theme > File Icon Theme。

從已安裝的清單選擇。

或是點選 Install Additional File Icon Themes 找尋其它有趣的 Icon Theme。
