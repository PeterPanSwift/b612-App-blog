下載 GitHub repository 下的資料夾

我們時常下載 GitHub 上大大們寫的優質程式，有時為了學習，有時為了抄襲。 不過有些大大的 GitHub repository 十分龐大，比方知名 iOS 開發網站 use your loaf 的程式範例全都放在同一個 repository。

有時我們只想下載 repository 下某個資料夾的程式，比方以下關於 Container View Controller 的程式範例，但是網頁上卻找不到下載的按鈕。

kharrison/CodeExamples
Code Examples. Contribute to kharrison/CodeExamples development by creating an account on GitHub.github.com

別找了，GitHub 網站並不支援下載 repository 下的資料夾，只能下載整個 repository。看來我們還是找其它幫手幫忙吧，接下來我們就請網站 DownGit 登場。

1 連到網站 DownGit。

https://minhaskamal.github.io/DownGit

2 複製剛剛 GitHub 網站 Container View Controller 程式範例的網址，將它貼在 DownGit 的輸入框。
3 點選 Download 按鈕。

Cool ! 我們成功下載 GitHub repository 下的資料夾了 ~
