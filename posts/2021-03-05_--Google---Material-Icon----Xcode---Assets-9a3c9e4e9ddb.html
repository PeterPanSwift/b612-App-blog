Apple 貼心地提供 SF Symbol 讓我們使用兩千多種的 icon 圖片，不過圖片永遠沒有夠用的一天，因此接下來讓我們從 Google 那裡拿一些 icon 來用吧。

連到 Google 的 Material Icon 網站

Google 的 Material Icon 網站提供了大量的 icon 圖片。

Google Fonts
Making the web more beautiful, fast, and open through great typographyfonts.google.com

從網站找尋並點選喜歡的 icon

在此彼得潘點選可愛的 Face。

點選後右半邊將出現 icon 相關的設定視窗。

切換到 iOS 分頁，點選 Download 下載

顏色的部分有黑白兩色可選擇。

點選 Download 下載，得到圖片的 zip 壓縮檔。

解壓縮後的資料夾如下，它有很多不同的 size。

將 icon 的 imageset 加到 Xcode 的 Assets

比方我們想要大小 36 pt 的圖片，因此複製下圖的 outline_face_black_36pt.imageset ( cmd + c)。

然後再貼到 Xcode 的 Assets 即可享用。(cmd + v)

SwiftUI 測試

struct ContentView: View {
    
    
    var body: some View {
       
        Image("outline_face_black_36pt")
            .resizable()
            .scaledToFit()
            .frame(width: 36, height: 36)
    }
    
}
