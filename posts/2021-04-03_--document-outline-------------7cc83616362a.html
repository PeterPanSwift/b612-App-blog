從 document outline 設定方便搜尋的元件名稱

從 Interface Builder 我們可以方便地設計 App 畫面，比方下圖的藍色 view & 黃色 view。但當畫面上的元件一多時，找尋某個元件往往變得十分困難。

從 document outline 設定方便搜尋的元件名稱

從 Interface Builder 我們可以方便地設計 App 畫面，比方下圖的藍色 view & 黃色 view。但當畫面上的元件一多時，找尋某個元件往往變得十分困難。

解法

從 document outline 幫畫面上的元件取個容易辨識的名字，之後即可快速找到元件，方法如下:

點兩下想設定名字的元件，讓它的名字變成輸入框

輸入名字

在此我們輸入 blue view。

元件的名字也可從 Identity inspector 頁面的 Label 欄位設定。

在下方的 Filter 框框搜尋

有了名字，之後我們可在 document outline 下方的 Filter 框框輸入文字搜尋，比方輸入 blue 即可找到 blue view。

在 Find Navigator 搜尋

在 Find Navigator 搜尋時也會包含 document outline 裡元件的名字。

元件自動改名

除了我們自己設定名字，元件也會因為以下幾種情況自動改名。

依據元件的內容改名

label & button 的名字變成顯示的文字，image view 的名字變成圖片的名字。

依據元件的 outlet 改名

元件有連 outlet 時，名字將變成 outlet 的名字。

@IBOutlet weak var peterImageView: UIImageView!

依據元件的 class 名字改名

下圖的 view controller class 設為 HomeViewController，因此名字變成 Home View Controller。
