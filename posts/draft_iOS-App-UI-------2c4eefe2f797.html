iOS App UI 元件點將錄

滾滾長江東逝水，浪花淘盡英雄，是非成敗轉頭空，青山依舊在，幾度夕陽紅。

三國有很多英雄人物，小時候彼得潘很喜歡看三國的英雄點將錄，深深為每位英雄豪傑的有趣故事著迷。不過我們今天要把主題從古代拉回現代，好好來認識二十一世紀的英雄，幫助我們打造美麗 iOS App 畫面的 iOS App UI 元件點將錄。

View 
  • 最基本的 UI 元件，一般顯示為長方形區塊。
  • 具有 UI 元件的基本特性，例如背景顏色。
  • View 本身可當成容器，在它身上再添加其它的 UI 元件。
  • 可改造成其它形狀，甚至增添圓角或陰影效果。
