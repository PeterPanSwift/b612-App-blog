將東西存到變數常數的型別問題

在寫程式時，我們時常需要將東西存到變數(常數)裡。請記得 Swift 是個專一的人，他就像彼得潘一樣，小時候喜歡女生，這輩子就只會喜歡女生。因此像以下的程式，變數 age 一開始儲存 18，就代表它只能儲存型別整數的東西，這輩子只喜歡整數。你想要給它字串，它是不會接受的。

此時的錯誤訊息顯示 Cannot assign value of type ‘String’ to type ‘Int’，assign 的中文是指定的意思，在程式裡 = 就是 assign，因為我們把 = 右邊的東西指定存到左邊的變數或常數裡。因此剛剛錯誤訊息的意思是它不能將型別 String 的東西存到型別 Int 的 age。
