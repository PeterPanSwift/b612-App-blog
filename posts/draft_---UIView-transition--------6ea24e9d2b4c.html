利用 UIView.transition 實現翻牌動畫

在 Assets 加入圖片

在 Assets 加入三張圖片，back 是牌的背面，翻到正面後可能會出現 ace 或 king。

在 storyboard 加入顯示牌的 image view

預設顯示牌背，並將 User Interaction Enabled 勾選，如此 image view 點選才能有反應。

dd

class CardImageView: UIImageView {
    
    var rank = ""
    
    var isFaceUp = false {
        didSet {
            
            image = UIImage(named: isFaceUp ? rank : "back")
        }
    }
}
