幫助程式老師寫範例的小幫手 GitHub Copilot

彼得潘平常主要的工作是程式教學，因此常會在教課時邊寫程式邊講解，或是花許多時間寫教學文章。

不過寫程式範例常要花時間發想有趣的例子跟輸入內容，有沒有加速的方法呢 ?

有的 ! 最近彼得潘發現 GitHub Copilot 是程式老師不可多得的好幫手。有了它教學時寫程式範例變得又快、又簡單、又好玩。

以下我們示範幾種寫程式範例時常見的例子。

  • 生成 array。
  • 生成字串。
  • 定義型別。
  • 生成 dictionary。
  • 依序使用 array 成員。
  • 生成經緯度。

生成 array

從 books 生成 array 的內容。

從型別 [String] 生成字串 array。

從 fairyTaleBooks 生成童話故事的 array。

從 fairyTaleBooks 和彼得潘生成中文的童話故事 array。

當程式定義了以下 Book 型別時，自動生成包含 title 和 author 的 Book array。

struct Book {
    let title: String
    let author: String
}

從注解生成 array。

生成字串

輸入字串再也不怕打錯字。

從明月生成蘇東坡的水調歌頭。

從無間道生成電影簡介。

從 Oceans apart day after day 生成下一句歌詞。

從 Dreams do come true, 生成彼得潘接下來的名言。

從 quote 和 sta 生成 Steve Jobs 的名言。

從 Peter Pan 生成相關的名言。

從 once 生成故事。

定義型別

從 struct Book 生成 property。

從 rel 生成 property releaseData。

生成 dictionary

從 "name": "彼得潘"生成 "author": "J.M. Barrie"。

從 peterPanBookDic 生成彼得潘故事的 dictionary。

依序使用 array 成員

從已經打好的 littlePrinceQuotes[0] 生成下一個要讀取的 littlePrinceQuotes[1]、littlePrinceQuotes[2]。

生成經緯度

從 sunMoonLakeRegion 生成日月潭的經緯度。
