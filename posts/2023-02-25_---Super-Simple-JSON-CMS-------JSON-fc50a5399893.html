利用 Super Simple JSON CMS 存放和更新 JSON

開發 iOS App 時，我們時常會串接 API 抓取 JSON 資料，然後顯示在 App 畫面上。

通常這些 API 會由帥氣的後台工程師開發，不過如果是一些簡單的 JSON，我們也可以不寫程式產生，然後將它存放在雲端定時更新。

以下我們以 Super Simple JSON CMS 為例，說明如何利用它存放電影清單 JSON 並產生 API。

連到 Super Simple JSON CMS

Super Simple JSON CMS 免費提供存放 JSON 跟產生 API 的服務。

website
Online JSON editor for non-devs, automatically exposed through our speedy API. Perfect for your simple site/app when…simplejsoncms.com

點選 New document 新增 JSON，輸入 JSON 內容後按 Save

取得 API 網址

產生的 API 網址如下，zya2ekn3i9p 是資料的 ID。

https://simplejsoncms.com/api/zya2ekn3i9p

為了怕別人亂改 JSON，建議在 Add a password 輸入密碼後按 Save，確保之後只有我們能修改。

更新 JSON

電影每個星期會更新，沒人看的片只能傷心地下片，因此我們也需要定期更新 JSON。

連到以下網址更新修改之前儲存的 JSON。

https://simplejsoncms.com/editor/資料 ID

以剛剛的電影 JSON 為例，它的 ID 是 zya2ekn3i9p，因此網址如下。

https://simplejsoncms.com/editor/zya2ekn3i9p

修改後要在 Password protected 輸入正確密碼才能按 Save 儲存。
