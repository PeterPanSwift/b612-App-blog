調整 SwiftUI navigation bar 的標題樣式

navigation bar 預設的標題為黑色。(Dark mode 為白色)

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(.peter)
                .resizable()
                .scaledToFit()
                .navigationTitle("Iceland Boy")
        }
    }
}

若想改變標題的顏色或字體，則須利用 modifier toolbar(content:)。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(.peter)
                .resizable()
                .scaledToFit()
                .toolbar {
                    ToolbarItem(placement: .title) {
                        Text("Iceland Boy")
                            .foregroundStyle(.purple)
                            .font(.system(size: 50, weight: .semibold, design: .rounded))
                    }
                }
        }
    }
}

在 toolbar 的 { } 裡傳入 ToolbarItem，placement 設為 .title 讓 ToolbarItem 顯示在 navigation bar 的中間。ToolbarItem 顯示的內容為 Text，因此我們可自由添加各種 modifier 調整文字的樣式。

以下為另一個例子，我們讓 navigation bar 也變成特定的顏色。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            Image(.peter)
                .resizable()
                .scaledToFit()
                .toolbar {
                    ToolbarItem(placement: .title) {
                        Text("Iceland Boy")
                            .foregroundStyle(.purple)
                            .font(.system(size: 50, weight: .semibold, design: .rounded))
                    }
                }
                .toolbarBackgroundVisibility(.visible, for: .navigationBar)
                .toolbarBackground(.green, for: .navigationBar)
        }
    }
}
