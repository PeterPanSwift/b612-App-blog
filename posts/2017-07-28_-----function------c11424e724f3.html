生活中的 function 實例練習

模仿以下 Swift 定義和呼叫 function 的格式，從生活中的例子找靈感，在 Medium 發表創作，寫愈多愈有趣愈好，比方寫作業，刷牙，搭訕女生，如何追到 Taylor Swift 都可以拆解成多個步驟，用 function 來定義。

條件： function 的 { } 內至少要包含三行程式。

func 買珍珠奶茶給女朋友喝()  {
    print("出門")
    print("走到對面的飲料店")
    print("跟店員點一杯去冰三分糖的珍珠奶茶")
    print("付錢給店員")
    print("走回家")
    print("將飲料拿給女朋友")
}
