Provide Author Information to Create Git Repositories 錯誤

建立 Xcode 專案時，如果之前沒有設定過 author information，將出現以下錯誤訊息。

只要輸入名字和 email，然後點選 Save即可修正問題。

不過如果不小心按到 Cancel，忘了修正問題，我們還可以開發 iOS App 嗎 ?

其實是可以的，只是當我們要做版本管理，要進行 commit 動作時會出問題。因此為了讓專案可以順利 commit，此時我們要打開 Xcode 的 Settings 視窗。

切換到 Source Control 分頁，點選 Git settings。

在 Git Settings 頁面輸入 Author name & Author email。
