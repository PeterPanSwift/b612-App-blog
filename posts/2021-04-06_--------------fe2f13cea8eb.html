SwiftUI 實現圓形圖片邊框的三種方法

  • 在 Image 上用 overlay 添加用 stroke 或 strokeBorder 畫的圓框。
  • 設定 Image 的 background 是 Circle。
  • 用 ZStack 裝 Circle & Image。

在 Image 上用 overlay 添加用 stroke 或 strokeBorder 畫的圓框

struct ContentView: View {
    var body: some View {
        VStack(spacing: 50) {
            Image(.star)
                .resizable()
                .scaledToFill()
                .frame(width: 200, height: 200)
                .clipShape(.circle)
                .overlay {
                    Circle()
                        .stroke(.yellow, lineWidth: 20)
                }
            Image(.star)
                .resizable()
                .scaledToFill()
                .frame(width: 200, height: 200)
                .clipShape(.circle)
                .overlay {
                    Circle()
                        .strokeBorder(.yellow, lineWidth: 20)
                }
        }
    }
}

設定 Image 的 background 是 Circle

struct ContentView: View {
    var body: some View {
        Image(.star)
            .resizable()
            .scaledToFill()
            .frame(width: 200, height: 200)
            .clipShape(.circle)
            .background {
                Circle()
                    .padding(-10)
                    .foregroundStyle(.yellow)
            }
    }
}

padding 傳入負數讓背景的圓形變得比圓片大，成為圖片的邊框。

用 ZStack 裝 Circle & Image

struct ContentView: View {
    var body: some View {
        ZStack {
            Circle()
                .padding(-10)
                .foregroundStyle(.yellow)
            Image(.star)
                .resizable()
                .scaledToFill()
                .clipShape(.circle)
        }
        .frame(width: 200, height: 200)
    }
}
