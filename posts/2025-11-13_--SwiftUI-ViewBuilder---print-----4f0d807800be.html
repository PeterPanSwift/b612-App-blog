在 SwiftUI ViewBuilder 加 print 的方法

寫程式時 print 是簡單卻實用的 debug 技巧，不過在製作 SwiftUI App 時，ViewBuilder 的 closure 卻有許多限制，只能寫一些特定的語法，像是生成 View & 宣告變數常數，而像 print 這樣獨立一行呼叫 function 的程式是不能存在 ViewBuilder 的。

SwiftUI ViewBuilder 可以寫的程式
撰寫 SwiftUI 程式時，我們時常看到類似 VStack 這樣的程式，{ } 裡寫了多行產生 UI 元件的程式。medium.com

如果真的想在 ViewBuilder 加 print debug，其實有個小技巧。別忘了 ViewBuilder 可以宣告變數常數，因此只要將 print 包裝成常數宣告即可，例如以下的 let _ = print("VStack")。

struct ContentView: View {
    var body: some View {
        VStack {
            let _ = print("VStack")
            Image(systemName: "globe")
                .imageScale(.large)
                .foregroundStyle(.tint)
            Text("Hello, world!")
        }
        .padding()
    }
}

以下我們再看個實際的倒數功能例子。為了檢查 TimelineView ViewBuilder { } 裡計算的過程，加上 let _ = print(context.date, elapsed, remaining) 印出每次觸發的時間和計算的結果。

struct ContentView: View {
    let countdownSeconds = 10
    let startTime = Date()
    
    var body: some View {
        TimelineView(.periodic(from: startTime, by: 1)) { context in
            let elapsed = context.date.timeIntervalSince(startTime)
            let remaining = countdownSeconds - Int(elapsed)
            let _ = print(context.date, elapsed, remaining)
            if remaining > 0 {
                Text("\(remaining)")
                    .font(.largeTitle.monospacedDigit())
            } else {
                Image(systemName: "die.face.1.fill")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 120)
            }
        }
    }
}
