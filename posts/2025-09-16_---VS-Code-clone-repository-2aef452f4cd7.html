使用 VS Code clone repository

使用 VS Code clone repository 有三種方法。

  • 從 Welcome 視窗 Clone Git Repository。
  • 從 Quick Actions 點選 Git: Clone。
  • 請 AI clone 跟打開專案。

以下我們用彼得潘在 GitHub 的 repository Demo 為例示範。

GitHub - PeterPanSwift/Demo
Contribute to PeterPanSwift/Demo development by creating an account on GitHub.github.com
  • 從 Welcome 視窗點選 Clone Git Repository 或從 Quick Actions 點選 Git: Clone。

在 Clone from Github 的輸入框輸入 repository Demo 的網址。

選擇儲存 repository 的位置。

打開 repository。

  • 請 AI clone 跟打開專案。
