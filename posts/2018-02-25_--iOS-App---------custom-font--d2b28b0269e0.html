為 iOS App 加入客製字型 (custom font)

覺得 iOS App 內建的中文字型不夠可愛嗎 ? 沒關係，我們可另外加入字型檔，將 iOS App 顯示的文字變成我們喜愛的字型。

可加到專案裡的字體檔主要有 ttf ，otf & ttc 三種格式，其中 ttc 比較容易有問題，因此建議選擇 ttf 或 otf。接下來我們將從日本的字體網站下載可愛的粉筆字體，在 iOS App 上顯示可愛的粉筆字 !

手書きフォント「殴り書きクレヨン」のダウンロード｜すもももじ
無料のクレヨン手書きフォント「殴り書きクレヨン」のダウンロードができます。font.sumomo.ne.jp

滑鼠游標移到網站的紅色按鈕，點選下載可愛的粉筆字體

將字體檔 crayon_1–1.tiff 加到專案裡

按下 cmd + c 複製字體檔 crayon_1–1.tiff，然後在 Xcode 的 Project navigator 按下 cmd + v 貼上。

在 Info 頁面設定 Fonts provided by application

切換到 Info 頁面，從右鍵選單點選 Add Row。

新增 Fonts provided by application。

在它下方的 Item 0 輸入 crayon_1–1.ttf。

查詢字體的名字

在程式設定字體時必須使用字體的 PostScript name，在 Interface Builder 設定字體時則使用 Family name。如果不知道字體的名字，可參考以下方法。

方法 1: 利用指令 otfinfo。

在 terminal 輸入 otfinfo -i 加檔名。

otfinfo -i crayon_1-1.ttf

結果。

Family:              Nagurigaki Crayon
Subfamily:           Regular
Full name:           Nagurigaki Crayon
PostScript name:     Nagurigaki-Crayon
Version:             Version 1.10
Unique ID:           Nagurigaki Crayon
Trademark:           Nagurigaki Crayon
Copyright:           (C) 2015 Do-Font
Vendor ID:           Take
Permissions:         Editable

方法 2: 利用 Font Book。

如何利用 Font Book 知道字型的名字
想在 iOS App 裡使用特別的字型並不難，只要能將字體的 otf (Open Type Font)或 ttf(True Type Font) 檔加到專案裡，並在 App 的 info 頁面做設定。medium.com

方法 3: 從程式列出所有的字型名字

for family in UIFont.familyNames.sorted() {
    let names = UIFont.fontNames(forFamilyName: family)
    print("Family: \(family) Font names: \(names)")
}

在 App 裡使用字體

  • 從 SwiftUI 的程式
Text("With great power, comes great responsibility")
   .font(.custom("Nagurigaki-Crayon", size: 30))
  • 從 Interface Builder。

從 Font 選字體時，我們另外加入的字體會出現在最上方。

ps: 有時 Xcode 會有 bug，在 Interface Builder 找不到另外加入的字體，不過還是可以從程式設定。

  • 從 UIKit 的程式。

假設 label 儲存 UILabel 物件。

label.font = UIFont(name: "Nagurigaki-Crayon", size: 30)
