點選 Interface Builder Class 旁的箭頭快速跳到 Swift 程式

開發 iOS App 時，我們時常需要在 Interface Builder 和 Swift 程式之間切換，一下子在 Interface Builder 設計畫面，一下子在 Swift 檔開發程式。

當我們在 Interface Builder 設計完畫面時，有方法能點選某個元件，快速跳到它的 Swift 程式開發功能嗎 ?

當然有，跳過去的按紐就在一個神秘的箭頭上。接下來讓我們以 Apple 的 Recipes App 為例說明。

Apple Developer Documentation
Edit descriptiondeveloper.apple.com

打開 Recipes App 專案，切換到 RecipeEditor.storyboard，點選下圖的 controller 物件

我們想跳到 controller 的 Swift 程式，開發此頁面的相關功能。

切換到右邊 Identity inspector 頁面，點選 Class 右邊的箭頭

Class 欄位將顯示物件的類別，點選右邊的箭頭可跳到類別的 swift 檔。

Cool，不用一秒鐘的時間，我們就跳到了 RecipeEditorViewController.swift。
