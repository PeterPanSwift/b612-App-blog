讓 VS Code 新建的 Flutter Project 從新的 Window 開啟

透過 VS Code Command Palette 的 Flutter: New Project 我們可以快速建立全新的 Flutter 專案。不過它有個小小的缺點，它預設會在原本的視窗打開新專案，因此若原本視窗正在編輯另一個專案，它將被完全取代。

彼得潘想要在某個視窗編輯專案時，還能在新視窗開啟新建的專案。查了一下後，實現的方法有兩種。

  • 方法 1: 設定 Open Folders In New Window。

打開 VS Code 的 Open Settings (UI)。

將 Open Folders In New Window 設為 on。

之後新建立的 Flutter 專案將用新視窗打開。

  • 方法 2: 先開新視窗再建專案。

從 Command Palette 點選 New Window 開新視窗。

此時建立 Flutter 專案時將從新視窗打開。
