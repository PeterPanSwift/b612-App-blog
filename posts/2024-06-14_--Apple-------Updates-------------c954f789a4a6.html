從 Apple 開發網站的 Updates 頁面研究最新的開發技術

每年 WWDC Apple 都會發表大量的新技術，我們要怎麼跟上 Apple 的腳步，如何知道有哪些新功能和 API 呢 ?

別擔心，Apple 的開發網站的 Updates 頁面早為我們分門別類各種技術更新的文件，方便我們快速查詢相關的更新。

Updates | Apple Developer Documentation
View major documentation updates and highlights from WWDC, browse ongoing updates from a set of framework releases over…developer.apple.com

Updates 分類的方式有兩種，依每年的 WWDC 分類或依技術主題分類。

比方你對 WWDC24 的新技術亮點有興趣，可瀏覽 WWDC24 的 Update 頁面。

WWDC24 | Apple Developer Documentation
Highlights of new technologies introduced at WWDC24.developer.apple.com

對某項技術主題有興趣，可瀏覽它的 Update 頁面，例如以下 Xcode、SwiftUI & UIKit 的例子。

Xcode updates | Apple Developer Documentation
Learn about important changes to Xcode.developer.apple.com
SwiftUI updates | Apple Developer Documentation
Learn about important changes to SwiftUI.developer.apple.com
UIKit updates | Apple Developer Documentation
Learn about important changes to UIKit.developer.apple.com
