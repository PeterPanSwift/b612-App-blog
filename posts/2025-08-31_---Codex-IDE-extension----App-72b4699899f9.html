使用 Codex IDE extension 開發 App

Codex 是 OpenAI 推出的程式 agent，只要訂閱 ChatGPT 即可享用。使用 Codex 可透過以下三種方式。

  • 從 IDE 執行 Codex。
  • 從 Terminal 執行 Codex CLI。
  • 從 ChatGPT 網頁執行 Codex。

目前 Codex 支援的 IDE 主要有 VS Code、Cursor & Windsurf，以下我們以 VS Code 為例，說明如何在 VS Code 使用 Codex 開發 App。

安裝 Codex 的 extension，全名是 Codex — OpenAI’s coding agent。

Codex - OpenAI's coding agent - Visual Studio Marketplace
Extension for Visual Studio Code - One agent for everywhere you code-included in ChatGPT Plus, Pro, Team, Edu, and…marketplace.visualstudio.com

打開 Codex 視窗。

  • 方法 1: 點選視窗上方的 Codex icon。
  • 方法 2: 從 Command Palette 打開 Codex 視窗，點選 View: Show Codex 或 Open Codex。

點選 Sign in with ChatGPT。

點選 Open。

登入 ChatGPT。

登入成功後回到 VS Code，點選 Next 看完 Codex 的教學。

可以開始下 prompt 請 Codex 寫 code 了。

從 switch mode 可選擇 Chat、Agent 和 Agent(full access) 模式。若想存取網路和編輯專案外的檔案，記得選擇 Agent(full access)。

從 Select model 可選擇 AI model。

點選 + > Add image 可上傳圖片。

以下我們請 Codex 製作一個介紹冰島的畫面。

製作一個介紹冰島的畫面，顯示一張網路上找到的漂亮冰島圖片

Agent 模式的 Codex 會直接修改專案的程式，如果不喜歡 Codex 的修改，點選 Undo 可還原之前的程式。
