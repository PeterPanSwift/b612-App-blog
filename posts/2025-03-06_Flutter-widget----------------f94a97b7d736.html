Flutter widget 的參數型別常常和參數名字一樣

開發 Flutter App 時，我們常在產生 widget 時透過參數設定 widget 的內容。當遇到不熟的參數時，我們如何知道參數可傳入的值有哪些呢 ?

有一個小小的技巧，widget 的參數型別常常和參數名字一樣，因此我們可以先輸入參數名，再透過 . 或 ( ) 生成資料傳入。

以下我們看看幾個例子。

  • Text 的 textAlign，型別為 TextAlign。(ps: Text 的文字內容為電影秋水依人的主題曲)
Text(
  "Mon amour, je t'attendrai toute ma vie\nReste près de moi, reviens, je t'en supplie",
  textAlign: TextAlign.center,
)
  • TextStyle 的 fontWeight、color ，型別分別為 FontWeight、Color。
Text(
  "Mon amour, je t'attendrai toute ma vie\nReste près de moi, reviens, je t'en supplie",
  style: TextStyle(
    fontWeight: FontWeight.bold,
    color: Color.from(alpha: 1, red: 1, green: 0, blue: 0),
  ),
)
  • Column 的 mainAxisAlignment、mainAxisSize、crossAxisAlignment，型別分別為 MainAxisAlignment、MainAxisSize、CrossAxisAlignment。
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  mainAxisSize: MainAxisSize.min,
  crossAxisAlignment: CrossAxisAlignment.start,
  children: [
    Text('秋水依人'),
    Text('真善美'),
    Text('媽媽咪呀'),
    Text('樂來越愛你'),
    Text('萬花嬉春'),
  ],
)

有時參數型別和參數名字並不完全一樣，但也滿類似的，比方 Image 參數 fit 的型別是 BoxFit。

Image.network(
  'https://upload.wikimedia.org/wikipedia/commons/a/a3/Aptenodytes_forsteri_-Snow_Hill_Island%2C_Antarctica_-adults_and_juvenile-8.jpg',
  width: 300,
  height: 300,
  fit: BoxFit.cover,
)
