在 Windows 上準備 Flutter App 的開發環境

透過 Google 推出的 Flutter framework 我們可以更方便地開發跨平台 App，開發出在 iOS、Android、Web、macOS、Windows 和 Linus 上都能執行的 App。

開發 Flutter App 前，我們必須先準備好開發環境。之前彼得潘介紹過如何在 Mac 準備 Flutter App 的開發環境。

在 Mac 上準備 Flutter App 的開發環境
透過 Google 推出的 Flutter framework 我們可以更方便地開發跨平台 App，開發出在 iOS、Android、Web、macOS、Windows 和 Linus 上都能執行的 App。medium.com

以下我們改以 Windows 為例，介紹如何在 Windows 準備 Flutter App 的開發環境。

  • 安裝 git for windows。
  • 安裝提供 Android 模擬器的 Android Studio。
  • 安裝開發程式的 IDE VS Code。
  • 安裝 VS Code 的 Flutter plugin。
  • 從 VS Code 安裝 Flutter SDK 和設定 PATH。

ps:

  1. Flutter 網站上有最新的安裝教學，有興趣的朋友可參考以下連結。

2. 懶得花時間準備開發環境的朋友，也可考慮從 Google 的線上 IDE Firebase Studio 或 Zapp 開發 Flutter App。

安裝提供 Android 模擬器的 Android Studio

Android Studio 是開發 Android App 的軟體，之後開發 Flutter App 時我們將利用 Android Studio 的 Android 模擬器執行 Android App。

ps: Windows 上開發 Flutter App 時無法測試 iOS，因為提供 iOS 模擬器的 Xcode 只能在 Mac 安裝。

安裝開發程式的 IDE VS Code

在 Windows 上撰寫 Flutter App 有許多 IDE 可選擇，比方 VS Code、Android Studio 和 IntelliJ IDEA。

以下我們安裝 VS Code。

Visual Studio Code - Code Editing. Redefined

安裝 VS Code 的 Flutter plugin

為了更方便開發 Flutter App，我們最好在 IDE 上安裝 Flutter plugin，它將提供更好的自動完成和除錯功能，而且也能方便啟動 Android 模擬器執行 Flutter App。

請從 Flutter 的 Extensions 分頁搜尋 flutter，點選 Install 安裝。

從 VS Code 安裝 Flutter SDK 和設定 PATH

啟動 VS Code 後打開 Command Palette(ctrl + shift + p)，輸入 flutter，點選 Flutter: New Project。

此時會出現找不到 Flutter SDK 的錯誤，別擔心，點選 Download SDK 即可下載 SDK。

選擇儲存 SDK 的路徑，記得 SDK 的路徑不能有空白或特殊字元(最好全英文，中文也是特殊字元)。

此時會開始下載 SDK，請稍等片刻，等待是值得的。

安裝好時會出現 Add SDK to PATH，點選會自動設定 PATH。

ps:

  • 若要手動設定 PATH，請從 VS Code 的 Terminal 或命令提示字元輸入這行加入 flutter SDK 的路徑，C:\flutter 換成 flutter SDK 儲存的路徑。設定後必須重新啟動 VS Code / 命令提示字元才會生效。
setx Path "%Path%;C:\flutter\bin"
  • 如果以上方法安裝失敗，還有第二個方法，請參考網頁的說明下載 Flutter SDK 的 zip 檔安裝和設定 PATH。

測試是否已準備好開發 Flutter App 的環境

打開命令提示字元(command prompt)，輸入以下指令，它會檢查電腦是否已準備好開發 Flutter App 的環境。

flutter doctor

結果如下，彼得潘的電腦成功通過大部分的測試，表示我們可以開心地用 Flutter 開發 Android App 了。(ps: Android toolchain 和 Visual Studio — desktop windows apps 的問題可先暫時不管。)

ps: 若想看到更完整的檢測報告，可再加上參數 -v。

flutter doctor -v
