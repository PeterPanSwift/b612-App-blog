顯示縮圖的 image 欄位 & 方便加入 image view 的 Image library— Xcode 11 新功能

在 Xcode 11，當我們想在 Interface Builder 設定畫面上的圖片時，多了以下 2 個方便的功能。

顯示縮圖的 image 欄位

當我們從 image 欄位選擇圖片時，再也不用像從前那樣瞎子摸象，事先記起來圖片的名字了。如下圖所示，現在我們可以看到圖片的縮圖，可以看個清楚再做選擇。

選好後 Image 欄位除了顯示名字，還會貼心地顯示縮圖。

我們可以跟舊版的 Xcode 比較一下。如下圖所示，舊版 Xcode 的 Image 欄位完全不會顯示縮圖。

方便加入 image view 的 Image library

想加入顯示某張圖片的 image view，現在最快的方法應該是從 Image library 將圖片拖曳到畫面上，方法如下:

1. 點選右上角的 + 打開 Library 視窗。
2. 從 Image library 選擇圖片拖曳到畫面上。

點選第二個 icon 切換到 Image library 分頁，然後選擇圖片後即可將它拖曳到 App 畫面上。
