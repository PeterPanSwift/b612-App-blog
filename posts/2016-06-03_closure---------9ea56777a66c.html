closure 參數的自動完成

很多 Swift 初學者不喜歡 closure，因為覺得輸入很麻煩，很容易打錯字。其實我們都誤會她了，輸入 closure 一點也不麻煩，比方以下例子:

範例 :

func playMusic(album: (String, String) -> String) {
   print(album("周杰倫", "晴天"))
}
  1. 輸入 play，從提示清單裡選擇 playMusic，按下enter。

2. 當 album 參數為藍色提示區塊時，按下 enter，Xcode 將聰明地幫我們自動輸入 closure 的相關資訊。( Xcode 的自動完成會幫我們挑選適合的格式，比方在這個例子，它選擇以 trailing closure 的格式輸入，因為傳入的 closure 是最後一個參數。)

3. 輸入 closure 第一個參數的名字 singer 後，按下 tab，游標和藍色提示區塊移動到第2個參數。(參數的名字我們要自己取，可自由取一 個你喜歡的名字。)

(2) 輸入第二個參數的名字 song 後，按下 tab，游標移動到 code 的藍色提示區塊。

4. 輸入 closure 裡執行的程式碼。

因此，輸入 closure 參數時請先按下 enter，讓 Xcode 幫我們自動完成，然後再利用 tab 鍵移動，依序輸入參數名和程式碼，即可讓我們不動滑鼠地完成 closure 的輸入。
