利用 Mac Finder 的 tag 管理檔案

Mac Finder 的 tag 可以幫助我們標記檔案，方便瀏覽不同資料夾下相關的檔案。剛好彼得潘最近想把教學，book，iPhone_project 資料夾下常用的檔案利用 tag 管理，方便之後瀏覽，接下來就讓我們一步步認識使用 tag 吧。

管理 tag

點選 Finder > Preferences。

切換到 Tags 分頁。如下圖所示，有多種顏色的 tag 可選擇。彼得潘想要將橘色 tag 變成 Swift，因此點選 Orange，將它改名為 Swift。

彼得潘覺得 Mac 裡重要的只有 Swift，沒有別的了，因此勾選 Swift，取消其它顏色，讓 Finder 的側邊欄只會顯示橘色 tag。

為檔案標記 tag

比方點選 SwiftUI Basics for Beginners.pdf，然後點選右上方的 tag 按鈕，從 tag 清單裡選擇 Swift。

標記 tag 後，檔案旁將出現可愛的橘色小圓點。

我們也可以標記整個資料夾，比方下圖的 iOS_Apprentice_v8.1.0。

當檔案開啟時，我們也可以加上 tag。如下圖所示，點選下拉箭頭後，點選 Tags 輸入框，然後選擇 Swift。

列出某個 tag 的檔案

我們可從 Finder 的側邊欄點選 Tags 下的 Swift，列出所有標記 Swift 的檔案。

我們也可以點選下圖的按鈕，然後選擇 Tags，讓 Finder 以 tag 分類檔案清單。
