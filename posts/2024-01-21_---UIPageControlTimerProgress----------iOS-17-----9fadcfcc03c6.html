使用 UIPageControlTimerProgress 無限輪播內容 — iOS 17 新功能

開發 iOS App 時，我們通常會用 Timer 實現內容輪播功能，不過最近彼得潘發現 iOS 17 有更簡單的方法。

iOS 17 的 UIPageControlTimerProgress 會每隔幾秒觸發 page control 切換小圓點，因此我們可以結合 UIPageControl 和 UIPageControlTimerProgress 實現內容輪播。

以下我們簡單做個星座圖片輪播的 App 說明，目標每隔 3 秒鐘切換圖片。

使用 UIPageControlTimerProgress 實現輪播的方法有兩種。

  • 寫法 1: 搭配 pageControl 的 valueChanged action。
  • 寫法 2: 搭配 UIPageControlTimerProgress 的 delegate。

  • 設定 UIPageControlTimerProgress。

產生每 3 秒切換小圓點的 UIPageControlTimerProgress。

override func viewDidLoad() {
    super.viewDidLoad()
    // Do any additional setup after loading the view.
    
    let timerProgress = UIPageControlTimerProgress(preferredDuration: 3)
    pageControl.progress = timerProgress
    timerProgress.resetsToInitialPageAfterEnd = true
    timerProgress.resumeTimer()
}

說明

let timerProgress = UIPageControlTimerProgress(preferredDuration: 3)
pageControl.progress = timerProgress

產生每 3 秒切換小圓點的 UIPageControlTimerProgress，將它設為 page control 的 progress。

timerProgress.resetsToInitialPageAfterEnd = true

我們希望輪播無限循環，不要到最後一個就停止，因此將 resetsToInitialPageAfterEnd 設為 true。如此當 page control 的 currentPage 是最後一個小圓點時，下次 Timer 觸發將回到第一個小圓點。

timerProgress.resumeTimer()

啟動 timer。要記得呼叫 resumeTimer，UIPageControlTimerProgress 才會定時切換小圓點。

  • 定義 page control 的 IBAction signPageControlValueChanged。

UIPageControlTimerProgress 每 3 秒切換小圓點時將觸發signPageControlValueChanged，因此我們在裡面設定小圓點切換時要做的事，在此我們改變顯示的圖片。

@IBAction func signPageControlValueChanged(_ sender: UIPageControl) {
    signImageView.image = UIImage(named: signs[sender.currentPage])
}

完整程式。

class ViewController: UIViewController {
    let signs = ["水瓶", "雙子", "天秤"]
    @IBOutlet weak var pageControl: UIPageControl!
    @IBOutlet weak var signImageView: UIImageView!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        let timerProgress = UIPageControlTimerProgress(preferredDuration: 3)
        pageControl.progress = timerProgress
        timerProgress.resetsToInitialPageAfterEnd = true
        timerProgress.resumeTimer()
    }
    
    @IBAction func signPageControlValueChanged(_ sender: UIPageControl) {
        signImageView.image = UIImage(named: signs[sender.currentPage])
    }
}

Cool ! 執行 App 後，我們成功看到星座圖片療癒地每三秒鐘換圖。

寫法 2: 搭配 UIPageControlTimerProgress 的 delegate

設定 UIPageControlTimerProgress 的 delegate，定義 pageControlTimerProgress(_:shouldAdvanceToPage:)。

class ViewController: UIViewController {
    let signs = ["水瓶", "雙子", "天秤"]
    @IBOutlet weak var pageControl: UIPageControl!
    @IBOutlet weak var signImageView: UIImageView!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        let timerProgress = UIPageControlTimerProgress(preferredDuration: 3)
        pageControl.progress = timerProgress
        timerProgress.resetsToInitialPageAfterEnd = true
        timerProgress.resumeTimer()
        timerProgress.currentProgress = 1
        timerProgress.delegate = self
    }
}

extension ViewController: UIPageControlTimerProgressDelegate {
    func pageControlTimerProgress(_ progress: UIPageControlTimerProgress, shouldAdvanceToPage page: Int) -> Bool {
        signImageView.image = UIImage(named: signs[page])
        return true
    }
}

pageControlTimerProgress(_:shouldAdvanceToPage:) 將在 timer 觸發，小圓點準備切換時呼叫，參數 page 代表下個變色的小圓點。因此我們利用 page 設定下個顯示的圖片，然後回傳 true 讓小圓點跳到下一個。
