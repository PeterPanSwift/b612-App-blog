為了在 VS Code 開發 Swift 程式，必須安裝 Swift extension。

從 Command Palette 點選 Swift: Create New Project。

template 選擇 Executable。

成功建立專案後，點選 Sources 下的 main.swift。

程式解題通常要讀取使用者輸入的資料，Swift 可用 readLine() function 讀取使用者輸入的文字。

func readLine(strippingNewline: Bool = true) -> String?
readLine(strippingNewline:) | Apple Developer Documentation
Returns a string read from standard input through the end of the current line or until EOF is reached.developer.apple.com

針對剛剛的分數計算題目，以下我們提供兩種解法。

寫法 1。

程式解題的題目通常可以假設輸入的資料是正確的格式，所以可以放心使用 force unwrap。

let name = readLine()!
let studentNumber = readLine()!
let chineseGrade = Int(readLine()!)!
let computerGrade = Int(readLine()!)!
let codeGrade = Int(readLine()!)!

let totalGrade = chineseGrade + computerGrade + codeGrade
let average = totalGrade / 3
print("Name:\(name)")
print("Id:\(studentNumber)")
print("Total:\(totalGrade)")
print("Average:\(average)")

使用 optional binding。

  • 執行程式。

方法 1: 從 Command Palette 點選 Swift: Run Build。

方法 2: 從 menu 點選 Run > Start Debugging。

方法 3: 點選側邊欄的三角形，然後點選上方的三角形。

  • 從 terminal 輸入資料。

依序輸入同學的資料後按 enter 後即可看到列印的結果。

  • 無限執行程式。

剛剛的程式只能計算一位同學的分數，重新執行才能計算另一位同學的分數。有沒有方法讓它無限執行計算多位同學的分數呢 ?

當然可以，只要將剛剛的程式包在 while true { } 裡即可。

while true {
    let name = readLine()!
    let studentNumber = readLine()!
    let chineseGrade = Int(readLine()!)!
    let computerGrade = Int(readLine()!)!
    let codeGrade = Int(readLine()!)!

    let totalGrade = chineseGrade + computerGrade + codeGrade
    let average = totalGrade / 3

    print("Name:\(name)")
    print("Id:\(studentNumber)")
    print("Total:\(totalGrade)")
    print("Average:\(average)")
}

  • 請 AI 解題。

目前 AI 已經十分聰明，難度不要太高的題目它都能迎刃而解。

import Foundation

// Read student name (one character)
let name = readLine() ?? ""

// Read student ID
let idString = readLine() ?? "0"
let id = Int(idString) ?? 0

// Read three subject scores
let chineseScoreString = readLine() ?? "0"
let chineseScore = Int(chineseScoreString) ?? 0

let csIntroScoreString = readLine() ?? "0"
let csIntroScore = Int(csIntroScoreString) ?? 0

let programmingScoreString = readLine() ?? "0"
let programmingScore = Int(programmingScoreString) ?? 0

// Calculate total and average scores
let totalScore = chineseScore + csIntroScore + programmingScore
let averageScore = totalScore / 3

// Print results without spaces after colons to match required format
print("Name:\(name)")
print("Id:\(id)")
print("Total:\(totalScore)")
print("Average:\(averageScore)")
