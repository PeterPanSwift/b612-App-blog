UITableView 可以讓它不要捲動，讓它的大小剛好等於內容的大小嗎


是的，可以讓 `UITableView` 根據內容的大小自適應高度並且不允許捲動。你可以按照以下步驟來實現這個效果：

1. **禁用捲動**：將 `UITableView` 的 `isScrollEnabled` 屬性設置為 `false`。

2. **調整高度**：根據內容來調整 `UITableView` 的高度。可以計算 `UITableView` 內容的總高度，並將其設置為 `UITableView` 的高度。

這裡有一個範例程式碼，展示了如何在 Swift 中實現這個功能：

```swift
import UIKit

class ViewController: UIViewController, UITableViewDataSource, UITableViewDelegate {
    
    let tableView = UITableView()
    let data = ["Item 1", "Item 2", "Item 3", "Item 4", "Item 5"]

    override func viewDidLoad() {
        super.viewDidLoad()
        
        tableView.dataSource = self
        tableView.delegate = self
        tableView.isScrollEnabled = false
        tableView.translatesAutoresizingMaskIntoConstraints = false
        view.addSubview(tableView)
        
        NSLayoutConstraint.activate([
            tableView.topAnchor.constraint(equalTo: view.topAnchor, constant: 20),
            tableView.leadingAnchor.constraint(equalTo: view.leadingAnchor, constant: 20),
            tableView.trailingAnchor.constraint(equalTo: view.trailingAnchor, constant: -20),
            tableView.heightAnchor.constraint(equalToConstant: tableView.contentSize.height)
        ])
        
        // Register a simple UITableViewCell
        tableView.register(UITableViewCell.self, forCellReuseIdentifier: "cell")
    }

    override func viewDidLayoutSubviews() {
        super.viewDidLayoutSubviews()
        updateTableViewHeight()
    }
    
    func updateTableViewHeight() {
        tableView.layoutIfNeeded()
        tableView.heightAnchor.constraint(equalToConstant: tableView.contentSize.height).isActive = true
    }
    
    // UITableViewDataSource methods
    func tableView(_ tableView: UITableView, numberOfRowsInSection section: Int) -> Int {
        return data.count
    }

    func tableView(_ tableView: UITableView, cellForRowAt indexPath: IndexPath) -> UITableViewCell {
        let cell = tableView.dequeueReusableCell(withIdentifier: "cell", for: indexPath)
        cell.textLabel?.text = data[indexPath.row]
        return cell
    }
}
```

### 注意事項
1. **初次加載時的高度計算**：在 `viewDidLayoutSubviews` 中調用 `updateTableViewHeight()` 以確保在視圖佈局後更新 `UITableView` 的高度。
2. **自適應高度**：可以在 `updateTableViewHeight()` 方法中調整 `UITableView` 的高度約束，以匹配其內容大小。

這樣就可以實現一個不捲動並且大小自適應內容的 `UITableView`。
