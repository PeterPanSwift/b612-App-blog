結合 AI 開發全端軟體的 Firebase Studio

Google 推出了結合 AI 開發全端軟體(full stack)的 Firebase Studio，現在前端、後端和手機 App 都可從瀏覽器連到 Firebase Studio 開發，不用再花時間辛苦地在自己的電腦安裝開發軟體。

以下彼得潘簡單示範 Firebase Studio 幾個主要的功能。

使用 Firebase Studio

連到 Firebase Studio。

Firebase Studio
Firebase Studio is an entirely web-based workspace for full-stack application development, complete with the latest…firebase.studio

點選 Try Firebase Studio。

勾選同意 Terms of Service & Android SDK Terms and Conditions，然後點選 Confirm。

開發 Flutter App

點選 Flutter icon 建立 Flutter App 專案。

開發 Android App

點選 Android Studio Cloud icon 建立 Android 專案。

ps: 模擬器初次執行時要花較久時間，waiting for build to finish 跑很久是正常的。如果一直卡在 waiting for build to finish，可試試點選上方的三角形啟動模擬器。

生成網頁 App

透過 Prototype an app with AI 可以出一張嘴下 prompt 生成網頁 App。

prompt: 可愛的二十一點 App，搭配 B612 星球的小王子主題。

AI 仔細思量後列出它規劃的 App 功能，點選右上方的編輯按鈕可修改，點選右下的 Prototype this App 將開始寫程式生成 App。

如果出現 Enter your Gemini API key 的框框，請點選 Auto-generate 自動生成 API key。

生成的網頁出現在左半邊，可以開始玩可愛的 Le Petit Blackjack !

點選下圖紅色框框的按鈕可分享網頁的 preview 連結和 QR Code。

若想讓連結變成人人皆可存取記得將它變成 Public。
