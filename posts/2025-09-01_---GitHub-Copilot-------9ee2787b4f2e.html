還原 GitHub Copilot 改壞的程式

當我們使用 VS Code 的 GitHub Copilot 修改程式時，常常會遇到 AI 寫錯，我們想要還原程式的情況。

GitHub Copilot 本身提供了幾種還原的方法

  • 點選下圖的 Restore 取消這次的修改。
  • 在虛線上點選 Restore Checkpoint 回到虛線處的程式版本。
  • 點選 …，點選 Undo Last Request 回到修改前的程式。

如果想要更確保能隨時回到某個正確的程式版本，建議再搭配 git 進行版本管理，相關說明可參考以下連結。
