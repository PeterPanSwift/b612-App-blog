Android Studio 修改程式即時更新 App 的 Live Edit 功能

Android Studio 提供修改程式即時更新 App 的 Live Edit 功能，現在我們修改程式時，將可看到執行中的 App 即時更新，不用再麻煩地重新啟動 App。

以預設的 Empty Activity 專案為例，我們將顯示的文字從 Android 改成 iOS，透過 Live Edit 觸發 App 即時更新。

class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        enableEdgeToEdge()
        setContent {
            MyApplicationTheme {
                Scaffold(modifier = Modifier.fillMaxSize()) { innerPadding ->
                    Greeting(
                        name = "iOS",
                        modifier = Modifier.padding(innerPadding)
                    )
                }
            }
        }
    }
}

觸發 Live Edit 更新 App 的方法有三種，可從 Find Action 打開 Settings 的 Live Edit 設定。

  • Push Edits Automatically。

修改程式時立即更新。

  • Push Edits Manually。

按下快速鍵cmd + '觸發更新。

  • Push Edits Manually on Save。

存檔時觸發更新。(快速鍵是 cmd + s)

Live Edit 十分方便，尤其當我們調整 Jetpack Compose App 的 UI 時可以立即看到更新的畫面。不過它也有一些限制，有些修改還是要重啟 App 才能生效，相關說明可參考以下連結。
