Show this window when Xcode launches

當我們啟動 Xcode 時，一開始會先顯示歡迎視窗，方便我們建立新專案或開啟之前的專案。

不過如果我們不小心按到左上角的 X 將它關掉，或是更慘地，將下方的 Show this window when Xcode launches 取消勾選，當我們下次啟動 Xcode 時，Xcode 將不再歡迎我們，不肯秀出歡迎視窗。

想要討 Xcode 歡心，讓歡迎視窗再度出現嗎 ?

不需要買 999 朵玫瑰給她，只要點選 Window > Welcome to Xcode 即可。

當歡迎視窗出現時，記得將滑鼠游標移至視窗的底部，出現 Show this window when Xcode launches 後將它勾選，這樣下次打開 Xcode 時，就能重見開頭視窗了。
