方便從 Firebase Studio啟動專案的 Open in Firebase Studio

最近 Firebase Studio 推出 Open in Firebase Studio button，我們可以幫 GitHub 專案添加從 Firebase Studio 啟動專案的連結，方便對專案有興趣的使用者打開 Firebase Studio 查看程式，不用另外花時間下載專案和安裝開發軟體。

Add an "Open in Firebase Studio" button
With Firebase Studio, you can simplify the steps for someone new to your codebase to set up their development…firebase.google.com

以下彼得潘以原本在 VS Code 開發的 Flutter App 專案為例說明。

以下是我們上傳到 GitHub 的專案網址。

https://github.com/PeterPanSwift/peter_profile
GitHub - PeterPanSwift/peter_profile
Contribute to PeterPanSwift/peter_profile development by creating an account on GitHub.github.com

將 https://studio.firebase.google.com/import?url= 和專案的 git 網址組合即可生成從 Firebase Studio啟動專案的連結。

因此結合彼得潘剛剛的 GitHub 專案網址後連結如下。

https://studio.firebase.google.com/import?url=https://github.com/PeterPanSwift/peter_profile

點選連結將打開 Firebase Studio，出現 Import project 視窗，若是 Flutter 專案記得要勾選 This is a Flutter app。

ps: 從 Firebase Studio 網站點選 Import repo 也可將 GitHub 專案匯入，出現上圖 Import project 的視窗。

準備專案的開發環境。

點選下圖的 Install & Yes。

我們的專案是 Flutter，因此要點選 Install 安裝 Dart-Code extension。

點選 Yes 設定 IDX configuration file(dev.nix)，然後點選下圖的 Take me there。

點選 Rebuild Environment。

幾乎不費吹灰之力，Firebase Studio 成功加入 GitHub 的專案，現在我們可以開心開發 Flutter App 了 !

我們還可以將剛剛的連結包裝成 button，加到 GitHub 專案的 README ，方法如下。

連到以下連結，滑到 Add an “Open in Firebase Studio” button 區塊。

Add an "Open in Firebase Studio" button
With Firebase Studio, you can simplify the steps for someone new to your codebase to set up their development…firebase.google.com

輸入 GitHub 專案的網址，從 Label、Size、Color、Format 調整 button 的樣式。

複製 button 對應的 html。

<a href="https://studio.firebase.google.com/import?url=https%3A%2F%2Fgithub.com%2FPeterPanSwift%2Fpeter_profile">
  <img
    height="32"
    alt="Open in Firebase Studio"
    src="https://cdn.firebasestudio.dev/btn/open_bright_32.svg">
</a>

編輯 GitHub 專案的 README，貼上複製的 html。

大功告成 ! README 出現了美美的 Open in Firebase Studio button，點選就能從 Firebase Studio 啟動專案。
