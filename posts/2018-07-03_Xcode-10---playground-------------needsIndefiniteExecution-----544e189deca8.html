Xcode 10 的 playground 抓取網路資料不用再設定 needsIndefiniteExecution 了 !

從前我們在 Playground 執行某些特別動作，比方抓取網路 JSON 資料，利用 AVSpeechSynthesizer 唸出某段告白的話，或是定時執行某段程式的 timer 時，必須額外加入以下程式碼

import PlaygroundSupport
PlaygroundPage.current.needsIndefiniteExecution = true

如此 Playground 才能一直執行，不會執行到最後一行就結束執行，來不及執行之後才會被觸發的程式。

但在 Xcode 10，再也沒有這個問題了 ! playground 再也不會執行到最後一行就結束，方便我們在 playground 測試任何需要等待額外時間才會執行的程式。

let url = URL(string: "https://itunes.apple.com/search?term=swift&media=music")
let task = URLSession.shared.dataTask(with: url!) { (data, response, error) in
   if let content = String(data: data!, encoding: .utf8) {
      print(content)
   }
}
task.resume()
