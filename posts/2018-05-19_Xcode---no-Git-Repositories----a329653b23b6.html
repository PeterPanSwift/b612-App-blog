Xcode 的 no Git Repositories 問題

如果一開始在 Xcode 建立專案時忘了勾選 Create Git repository on my Mac，專案將沒有 git 版本管理的功能。如下圖所示，Source Control 的 Changes 分頁將顯示 no Git Repositories，無法將專案上傳到 GitHub。

沒關係，我們隨時可以幫專案加上 git 版本管理，方法如下:

  1. 點選 Quick Actions 的 New Git Repository。

2 點選右下角的 Create。

經過剛剛的步驟，專案將再度擁有版本管理的功能。切換到 Source Control 的 Changes 分頁， 變成顯示訊息 No Changes。
