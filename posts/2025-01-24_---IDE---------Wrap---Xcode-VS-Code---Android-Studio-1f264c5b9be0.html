設定 IDE 程式自動換行的 Wrap — Xcode、VS Code & Android Studio

開發程式時，我們有時會寫出很長很長的程式，甚至會超出螢幕的寬度，因此變成要水平滑動，閱讀和編輯程式時有點不方便。

還好現在大部分的 IDE 都有提供程式自動換行的 Wrap 功能，當我們調整 IDE 的視窗大小時，程式也會跟著縮排換行。

以下我們分別介紹 Xcode、VS Code & Android Studio 打開 Wrap 功能的方法。

Xcode

從 Quick Actions 點選 Wrap Lines。

VS Code

從 Settings 將 Word Wrap 選擇 on。

Android Studio

從 Find Action 點選 Configure Soft Wraps。

從 Soft-wrap these files 設定自動換行的檔案類型。

ps: 記得要用分號分隔，不是用逗號分隔。
