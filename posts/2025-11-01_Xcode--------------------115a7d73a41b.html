Xcode 快速複製貼上 { } 區塊程式的方法

初學者在練習開發 iOS App 時，時常會複製貼上 { } 區塊的程式，比方以下製作球類運動清單的例子，複製貼上顯示桌球的 HStack，然後再改成棒球。

不過手動拖曳觸控版選取程式其實不太好選，按下 cmd + c 複製再按 cmd + v 貼上也有點麻煩，因此以下彼得潘另外介紹五種更快的方法。

  • 按住 shift 選取再按 cmd + d 貼上。
  • 折疊程式後雙擊選取，再按 cmd + d 貼上。
  • 雙擊 {，按 option + shift + 向左鍵選取，再按 cmd + d 貼上。
  • 使用 Vim 模式貼上多個。
  • 請 AI 幫忙。

按住 shift 選取再按 cmd + d 貼上。

在 HStack { } 後空白一行。

點擊 HStack 的 H 左邊，按住 shift 鍵，再點擊 HStack } 的下一行選取整段程式。

放開 shift 鍵，按下 cmd + d 貼上選取的程式。

將貼上的內容改成棒球。

折疊程式後雙擊選取，再按 cmd + d 貼上

點選 HStack 左邊的箭頭折疊。

ps: 如果沒看到箭頭，請參考以下連結打開 Code folding ribbon。

Xcode 的 Code folding ribbon
我們常常不知不覺寫出長長的一大串程式，不過沒關係，Xcode 的 Code folding 功能可以幫我們將整段程式收起(折疊)，等想念它的時候再將它展開。medium.com

雙擊選取折疊的程式。

按下 cmd + d 貼上選取的程式。

雙擊 {，按 option + shift + 向左鍵選取，再按 cmd + d 貼上

雙擊 { 選取 { } 包含的程式。

按 option + shift + 向左鍵選取 { 前的 HStack。

按下 cmd + d 貼上選取的程式。

按下向左鍵 + return 換行。

使用 Vim 模式貼上多個

進入 vim 模式。(ps: 點選 Vim mode 可進入跟離開 Vim 模式)

在 HStack 上按住 shift 後按 v % y %。(ps: shift v 是選取整行，% 會跳到對應的括號，y 是複製)

放開 shift，按 p 貼上。

若想一次貼上多個，可加上數字，比方 2p 會貼上 2 個。
