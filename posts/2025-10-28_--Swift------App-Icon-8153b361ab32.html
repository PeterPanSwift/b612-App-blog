從 Swift 程式讀取 App Icon

最近彼得潘將 Xcode 的 App Icon 換成坐在 Swift 上的小王子時，發現 Xcode 連歡迎畫面的圖片也變了。Cool ! 看來我們可以從 Swift 程式讀取 App Icon 後顯示在 App 裡。

在 iOS App 實際測試後，發現以下兩種設定的 App Icon 得到的 iconFiles 都是 60*60 大小的 App Icon。

  • 在 Assets 設定 1024*1024 的 AppIcon。

得到的 iconFiles: ["AppIcon60x60"]。

  • 在 Project navigator 加入 .icon 的圖檔， 然後在 General 頁面的 App Icon 輸入檔名。

得到的 iconFiles: ["Landmarks App Icon60x60"]。

以下分別為 SwiftUI & UIKit 在 App 裡顯示 App Icon 的寫法。

  • SwiftUI

必須先生成 UIImage，再從 UIImage 生成 Image。

struct ContentView: View {
    var appIconUIImage: UIImage? {
        if let iconsDictionary = Bundle.main.infoDictionary?["CFBundleIcons"] as? [String: Any],
           let primaryIconsDictionary = iconsDictionary["CFBundlePrimaryIcon"] as? [String: Any],
           let iconFiles = primaryIconsDictionary["CFBundleIconFiles"] as? [String],
           let appIconName = iconFiles.last {
            UIImage(named: appIconName)
        } else {
            nil
        }
    }
    
    var body: some View {
        if let appIconUIImage {
            Image(uiImage: appIconUIImage)
        }
    }
}
  • UIKit
class ViewController: UIViewController {

    override func viewDidLoad() {
        super.viewDidLoad()

        if let iconsDictionary = Bundle.main.infoDictionary?["CFBundleIcons"] as? [String: Any],
           let primaryIconsDictionary = iconsDictionary["CFBundlePrimaryIcon"] as? [String: Any],
           let iconFiles = primaryIconsDictionary["CFBundleIconFiles"] as? [String],
           let appIconName = iconFiles.last,
           let uiImage = UIImage(named: appIconName) {
            let imageView = UIImageView(image: uiImage)
            imageView.center = view.center
            view.addSubview(imageView)
        }
    }
}
