請給 Xcode 多一點點時間認得我們新定義的東西

當我們開發 Swift iOS App 時，時常會定義新的型別和 function，而 Xcode 也會貼心認得這些新發明的東西，提供自動完成，讓我們少打幾個字。但是就像彼得潘記得學生的名字需要好幾個星期，到了最後一堂課要離別時才記得，Xcode 也需要點時間認得這些新發明的東西。

例如以下例子，我們定義一個新的型別 Song。

struct Song {
   var name: String
   var singer: String
}

然後我們很快地切換到 ViewController ，想要產生一首新歌。然而當我們輸入 Son 時，卻發現自動完成的選單裡沒有 Song。

請給 Xcode 多一點點時間，大概再等個幾十秒，不超過一分鐘，你就可在自動完成的選單裡看到 Song。

如果你是一秒鐘幾十萬上下的 iOS App 工程師，那麼你還可試試這招。直接按 cmd + b，強制 Xcode 編譯程式，如此馬上就可讓 Xcode 認得我們剛剛定義的型別和 function。
