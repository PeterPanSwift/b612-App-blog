申請 Google Maps API key

透過 Google Maps API，我們可以開發許多有趣的功能，比方抓取附近好吃的餐廳，使用 Google Map 顯示地圖等。

為了使用 Google Maps API，我們必須先取得它的 API key，步驟如下:

連到 Getting started with Google Maps Platform 網頁

Getting started with Google Maps Platform | Google Developers
"type": "thumb-down", "id": "missingTheInformationINeed", "label":"Missing the information I need" },{ "type"…developers.google.com

點選 Get Started

點選 CREATE PROJECT 建立新專案或選擇之前建立的 PROJECT

輸入新專案的名字，點選 Create。

接著會跳出 Enable Google Maps Platform 視窗，此時有一個特別的 bug，它要我們再建立一次 project。反正建專案不用錢，那就再建一次吧。

建立計費帳號(billing account)

Google Maps API 很方便，但是 Google 也要吃飯，所以它會跟我們收錢。我們必須建立計費帳號，填寫信用卡資訊。

一開始 Google 會免費送我們 $300 美金的額度。之後如果花完 $300，或是超過 90 天，Google 會詢問我們是否升級付費帳號，只要不升級即不會花到錢。

點選 START MY FREE TRIAL 後，信用卡將被刷 40 塊台幣。別難過，這是為了確認信用卡可用，之後會馬上退刷的。

勾選想使用的功能

有 Maps，Routes & Places 三種。彼得潘貪心地全部勾選。

填寫問卷

點選 ENABLE，正式啟用 Google Map API

取得 API key

Cool，我們順利拿到珍貴的 API key，之後串接 Google Map API 必須搭配此 key。

之後如果忘了 API key，可透過以下方式查詢。

  • 連到 Google Developers Console 。
Google Cloud Platform
Google Cloud Platform lets you build, deploy, and scale applications, websites, and services on the same infrastructure…console.developers.google.com
  • 從右上選擇 project，切換到 Credentials 分頁。

點選下圖的紅色框框複製 key。
