Xcode 專案的 Include Tests 設定

有同學建立 Xcode 專案後，看到底下長出兩個以 Tests 結尾的奇怪資料夾，比方下圖的 DemoTests & DemoUITests。

為什麼會有這些 Tests 資料夾呢 ? 它們是為了包含測試 App 功能的檔案。當我們在建立專案時勾選 Include Tests 即會產生 Tests 資料夾。

測試 App 功能的相關程式屬於較進階的功能，初學者一開始還用不到，因此可以不理會 Tests 相關的資料夾。若是不想看到它們，也可以在建立專案時取消 Include Tests。
