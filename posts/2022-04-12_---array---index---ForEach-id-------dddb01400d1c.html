使用 array 的 index 當 ForEach id 的特別問題

當我們用 SwiftUI 的 ForEach 顯示 array 的內容時，通常需要指定區分成員的 id。

struct Song {
    let name: String
    let singer: String
}

不過當我們使用 array 的 index 當 ForEach 的 id 時，卻可能產生特別的問題，以下我們以永遠的天王劉德華的情歌歌單為例說明。

struct ContentView: View {
    @State private var songs = [
        Song(name: "真我的風采", singer: "劉德華"),
        Song(name: "回到你身邊", singer: "劉德華"),
        Song(name: "相思成災", singer: "劉德華"),
        Song(name: "在乎您", singer: "劉德華"),
        Song(name: "真情難收", singer: "劉德華")
    ]

    var body: some View {
        List {
            ForEach(songs.indices, id: \.self) { index in
                HStack {
                    Image(songs[index].name)
                        .resizable()
                        .scaledToFill()
                        .frame(width: 100, height: 100)
                        .clipShape(Circle())
                    Text("\(songs[index].name)")
                    
                }
                .onTapGesture {
                    withAnimation {
                        songs.swapAt(0, index)
                    }
                }
                
            }
        }
    }
}

以上程式使用 ForEach(songs.indices, id: \.self)，因此將以 array 的 index 數字當 id，歌曲真我的風采 id 是 0，回到你身邊的 id 是 1。這樣的寫法看來沒什麼問題，App 畫面也如下圖正常顯示。

然而此寫法卻可能產生特別的問題，尤其牽扯到動畫時。透過以下程式，點選歌曲時將以動畫讓歌曲成為列表的第一首歌。不過實際測試後，我們將發現動畫效果不如預期，它可能會沒有動畫，也可能是左右移動，而不是我們想要的上下移動。

.onTapGesture {
     withAnimation {
         songs.swapAt(0, index)
     }
}

若想讓動畫完美地以上下移動的方式調整順序，請記得別用 array 的 index 當 id，比方以下例子使用 Song 的 name 當 id。(ps: 假設歌曲的名字不會重覆）

struct ContentView: View {
    @State private var songs = [
        Song(name: "真我的風采", singer: "劉德華"),
        Song(name: "回到你身邊", singer: "劉德華"),
        Song(name: "相思成災", singer: "劉德華"),
        Song(name: "在乎您", singer: "劉德華"),
        Song(name: "真情難收", singer: "劉德華")
    ]

    var body: some View {
        List {
            ForEach(songs, id: \.name) { song in
                HStack {
                    Image(song.name)
                        .resizable()
                        .scaledToFill()
                        .frame(width: 100, height: 100)
                        .clipShape(Circle())
                    Text("\(song.name)")
                    
                }
                .onTapGesture {
                    withAnimation {
                        let index = songs.firstIndex {
                            $0.name == song.name
                        }
                        songs.swapAt(0, index!)
                    }
                }
                
            }
        }
    }
}
