如何將檔案加到 Xcode 專案的 project navigator

方法1: 拖曳大法

將檔案拖曳到專案的 project navigator 下。

記得勾選 Copy items if needed & Add to targets。

結果

方法2: Add Files to xxx

從專案的 project navigator 點選想要檔案加入的位置，比方點選下圖的 Demo ，然後從右鍵選單選擇 Add Files to xxx。

找到想加入的檔案後，點選右下的 Add 加入。
