使用 PyInstaller 將 Python 程式變成 Mac App

在 Mac 上執行 Python 程式有許多方法，最常見的莫過於從 terminal 輸入指令，比方以下指令將執行 main.py。

python3 main.py

不過最近有同學問彼得潘，能將 python 程式變成 Mac App，方便使用者雙擊 App Icon 執行嗎 ?

當然可以呀，而且方法不只一種。以下我們介紹利用 PyInstaller 將 Python 程式變成 Mac App。

PyInstaller Manual - PyInstaller 6.10.0 documentation
PyInstaller is tested against Windows, MacOS X, and Linux. However, it is not a cross-compiler; to make a Windows app…pyinstaller.org

我們用以下的 main.py 為例，它執行時會在 Mac 上建立伺服器，然後打開 Safari 連到 http://localhost:8080/，顯示小王子的名言只有用心看，才能看清楚。重要的東西是眼睛看不見的。

import http.server
import socketserver
import webbrowser

# 設定處理程序，會回應 "Hello World"
class MyHandler(http.server.SimpleHTTPRequestHandler):
    def do_GET(self):
        # 回應 200 狀態碼（成功）
        self.send_response(200)
        # 設定內容類型為 text/html 並加上 utf-8 編碼
        self.send_header("Content-type", "text/html; charset=utf-8")
        self.end_headers()
        # 回應內容 "只有用心看，才能看清楚。重要的東西是眼睛看不見的。"
        message = "只有用心看，才能看清楚。重要的東西是眼睛看不見的。"
        self.wfile.write(message.encode("utf-8"))

# 埠號，可以使用任何非佔用的本地埠
PORT = 8080

# 創建伺服器
with socketserver.TCPServer(("", PORT), MyHandler) as httpd:
    print(f"Serving at port {PORT}")

    # 使用 Safari 開啟本地連結
    webbrowser.get("safari").open(f"http://localhost:{PORT}")

    # 啟動伺服器並等待請求
    httpd.serve_forever()

為了用 PyInstaller 將 main.py 變成 Mac App，我們必須先用 pip 安裝 pyinstaller。

pip3 install pyinstaller

裝好 pyinstaller 後，輸入以下指令生成 Mac App，參數 — onefile 表示要生成單一執行檔，參數 — windowed 表示 App 啟動時不要顯示 terminal 視窗。

pyinstaller --onefile --windowed main.py

生成的 App 將在 dist 資料夾下，之後將下圖的 main application 傳給朋友，對方即可在 Mac 上執行 App。

由於這是未上架的 App，macOS 預設會阻止 App 啟動，因此啟動時可能會出現以下錯誤。
