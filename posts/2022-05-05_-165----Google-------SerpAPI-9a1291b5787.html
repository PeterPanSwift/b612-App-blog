#165 抓取 Google 搜尋結果的 SerpAPI

最近發現了強大的 SerpAPI，透過它我們可以抓取 Google 豐富的搜尋結果。如下圖所示，Google 搜尋查到的資料都可以從 SerpAPI 回傳的 JSON 取得。

SerpApi: Google Search API
JSON Results Regular organic results are available as well as Maps, Local, Stories, Shopping, Direct Answer, and…serpapi.com

除了抓取 Google 搜尋的結果，SerpAPI 還支援其它常見的搜尋應用，比方抓取 YouTube 影片跟 Google 地圖的餐廳等。

API 文件

SerpAPI 的完整說明，可參考以下連結。

接下來讓我們以搜尋可愛的角落生物為例，看看 SerpAPI 回傳的 JSON 長什麼樣子吧。

註冊帳號，認證 email 後點選 Subscribe

進入 dashboard 頁面，取得 API key

Google Search API - SerpApi
Scrape Google and other search engines from our fast, easy, and complete API.serpapi.com

API 範例

在 q 後輸入搜尋的文字，api_key 帶入剛剛取得的 API key。(ps: 以下例子的 key 123 是假的)

https://serpapi.com/search.json?engine=google&q=角落生物&api_key=123

JSON 解析

  • 知識圖譜

Google 搜尋結果的右邊常會顯示整理過的知識圖譜(Knowledge Graph)，比方下圖是角落生物的知識圖譜，從 JSON 的 key knowledge_graph 可取得知識圖譜。

  • 相關影片

從 JSON 的 key inline_videos 可取得影片資訊。

  • 搜尋結果

從 JSON 的 key organic_results 可取得搜尋結果。

有些搜尋結果會顯示縮圖，可從 JSON 的 thumbnail 欄位取得。

有些搜尋結果會顯示多張圖片，可從 JSON 的 images 欄位取得。

  • 相關的搜尋關鍵字

從 JSON 的 key related_searches 可取得相關的搜尋關鍵字。

  • 抓取下一頁的資料

從 JSON 的 key next_link 或 next 可取得下一頁的 API 網址。

分頁的關鍵在參數 start，預設一頁是十筆資料，因此抓取第二頁時可傳入 start=10，第三頁時傳入 start=20。

資料抓取的上限

沒付錢的話一個月只能搜尋 100 次。因此建議剛開始練習時先將 JSON 檔抓下來加入專案，這樣測試時比較不會浪費流量。

讀取專案裡的檔案測試 JSON 解碼
為了測試 JSON 資料對應的 Decodable(或 Codable) 型別是否有問題，除了實際串接 API 抓資料，我們也可以先將 JSON 檔加入專案裡，方便沒有網路時進行測試。medium.com
