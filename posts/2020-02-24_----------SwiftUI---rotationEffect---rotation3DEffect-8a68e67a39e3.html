轉吧七彩霓虹燈 ! SwiftUI 的 rotationEffect & rotation3DEffect

開發 SwiftUI App 時，利用 rotationEffect & rotation3DEffect 可實現各種有趣的旋轉效果。我們不只能控制旋轉的角度，還可以控制它延著 x、y、z 軸 & 某個 anchor 點旋轉。

接下來就讓我們以 B612 星球的小王子為例，看著他像旋轉木馬般精彩地旋轉吧。

struct ContentView: View {
    var body: some View {
        Image(.littlePrince)
    }
}

呼叫 rotationEffect 控制旋轉的角度

rotationEffect 的宣告如下。

func rotationEffect(_ angle: Angle, anchor: UnitPoint = .center) -> some View
rotationEffect(_:anchor:) | Apple Developer Documentation
Rotates a view's rendered output in two dimensions around the specified point.developer.apple.com

型別 Angle 的參數 angle 控制旋轉的角度，以下程式將讓小王子旋轉 45 度。

Image(.littlePrince)
    .rotationEffect(.degrees(45))

接下來我們讓小王子轉個夠，旋轉 45，90 & -45 度。

struct ContentView: View {
    let angels: [Angle] = [
        .degrees(45),
        .degrees(90),
        .degrees(-45)
    ]
    var body: some View {
        VStack {
            ForEach(angels, id: \.self) { angel in
                HStack {
                    Text(angel.degrees.formatted())
                        .font(.largeTitle)
                    Image(.littlePrince)
                        .rotationEffect(angel)
                }
            }
        }
    }
}

當角度大於 0 時元件將順時針旋轉，小於 0 時將逆時針旋轉。

產生 Angel 有很多寫法，可以設定角度(degree) 或弧度(radian)。設定弧度時，Double.pi 代表 180 度。以下幾種寫法全都代表角度 45 度。

var angel = Angle.degrees(45)
angel = .degrees(45)
angel = Angle(degrees: 45)
angel = Angle.radians(.pi / 4)
angel = .radians(.pi / 4)
angel = Angle(radians: .pi / 4)

如果角度超過 360 呢 ? 它會聰明地幫我們換算，比方 405 度會變成 45 度。

var angel = Angle.degrees(405)

繞著 anchor 點旋轉

rotationEffect 預設將繞著元件的中心點旋轉，不過我們也可以透過參數 anchor 控制它繞著其它點旋轉。

anchor 預設為 .center，以下例子傳入 .topLeading ，因此它將繞著元件左上角的頂點旋轉。

struct ContentView: View {
    let angels: [Angle] = [
        .degrees(45),
        .degrees(90),
        .degrees(-45)
    ]
    var body: some View {
        VStack {
            ForEach(angels, id: \.self) { angel in
                HStack {
                    Text(angel.degrees.formatted())
                        .font(.largeTitle)
                    Image(.littlePrince)
                        .rotationEffect(angel, anchor: .topLeading)
                }
            }
        }
    }
}
Image(.littlePrince)
    .rotationEffect(angel, anchor: .top)

anchor 傳入 .top ，它將繞著元件上邊界的中心點旋轉。

延著 x, y, z 軸 3D 旋轉的 rotation3DEffect

覺得平面旋轉不夠看嗎 ? 讓我們瞧瞧更厲害的 rotation3DEffect，控制元 件延著 x, y, z 軸旋轉。

rotation3DEffect(_:axis:anchor:anchorZ:perspective:) | Apple Developer Documentation
Renders a view's content as if it's rotated in three dimensions around the specified axis.developer.apple.com
struct ContentView: View {
    var body: some View {
        VStack {
            Image(.littlePrince)
                .rotation3DEffect(.degrees(45), axis: (x: 1, y: 0, z: 0))
            Image(.littlePrince)
                .rotation3DEffect(.degrees(45), axis: (x: 0, y: 1, z: 0))
            Image(.littlePrince)
                .rotation3DEffect(.degrees(45), axis: (x: 0, y: 0, z: 1))
        }
    }
}

參數 axis 控制旋轉的軸， 以上程式依序沿著 x 軸、y 軸、z 軸旋轉 45 度，延著 z 軸旋轉的效果將和 rotationEffect 一模一樣。

當元件沿著 x 軸旋轉 90 度時會怎樣呢 ? 不要問，很可怕，此時元件會整個不見。

Image(.littlePrince)
    .rotation3DEffect(.degrees(90), axis: (x: 1, y: 0, z: 0))

沿著 x 軸旋轉 180 度呢 ? 此時將變成上下 flip (鏡像上下翻轉，mirror) 的效果。

Image(.littlePrince)
    .rotation3DEffect(.degrees(180), axis: (x: 1, y: 0, z: 0))

我們也可以實現左右 flip(鏡像左右翻轉)，只要角度傳入 180，y 傳入 1，讓它沿著 y 軸旋轉 180 度。

Image(.littlePrince)
    .rotation3DEffect(.degrees(180), axis: (x: 0, y: 1, z: 0))

我們也可以混合不同軸的旋轉，比方以下例子。

  • 延著 x、y、z 軸轉 45 度。
Image(.littlePrince)
    .rotation3DEffect(.degrees(45), axis: (x: 1, y: 1, z: 1))

延著 z 軸轉 45 度，然後再水平 flip 180 度。

Image(.littlePrince)
    .rotationEffect(.degrees(45))
    .rotation3DEffect(.degrees(180), axis: (x: 0, y: 1, z: 0))
