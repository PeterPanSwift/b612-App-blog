利用 Xcode 的 Format File with swift-format 縮排程式 — Xcode 16 新功能

Xcode 從 16 開始內建縮排程式的工具 swift-format，現在縮排程式變得更加容易，按下快速鍵 ctrl + shift + i 即可用 Format File with swift-format 縮排整個檔案。

比方以下原本是縮排慘不忍睹的程式。

struct ContentView: View {
var body: some View {
VStack {
            Image(systemName: "globe")
    .imageScale(.large)
                       .foregroundStyle(.tint)
            Text("Hello, world!")
           }
              .padding()
    }
}

按下 ctrl + shift + i 後馬上變身成完美縮排的程式。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "globe")
                .imageScale(.large)
                .foregroundStyle(.tint)
            Text("Hello, world!")
        }
        .padding()
    }
}

除了按下 ctrl + shift + i ，從 Quick Actions 或 Xcode 的 menu 也可以執行 Format File with swift-format。

  • 從 Quick Actions 輸入 ffw。
  • 從 Xcode menu 的 Editor > Structure > Format File with swift-format。

format 整個專案的程式

若想 format 整個專案的程式，可在 Terminal 切到專案資料夾下，執行以下指令。

swift format -i -r .

在 Terminal 執行 swift format 預設會產生 2 個空白的縮排，若想改成 4 個空白，可進行以下步驟。

在專案資料夾下執行以下指令產生縮排的設定檔 .swift-format。

swift format dump-configuration > .swift-format

以下我們將 space 改成 4 讓縮排變成 4 個空白。

"indentation" : {
  "spaces" : 4
},

之後當我們用 Xcode 或 Terminal 對專案進行 swift-format 縮排時，它都會依據專案資料夾下 .swift-format 的設定，因此縮排也變成了 4 個空白。

如果希望所有專案都依照 .swift-format 的設定，則可將 .swift-format 放在自己的 home 目錄下。比方彼得潘將它放在 /Users/shih-yingpan/.swift-format，之後所有專案的縮排都會依據 .swift-format。

利用 Xcode Behavior format 整個專案的程式

我們也可以將 format 整個專案程式的動作變成 Xcode Behavior，方便之後一鍵縮排，步驟如下。

  • 在 Document 資料夾下新增 swift-format-project.sh。
#!/bin/bash

# 獲取當前項目目錄
PROJECT_DIR="$1"
cd "$PROJECT_DIR"

# 執行 swift-format
echo "Running swift-format..."
swift format -i -r .
echo "Format completed!"
  • 將 swift-format-project.sh 變成可執行的檔案。
chmod +x swift-format-project.sh
  • 新增 Xcode Behavior。

從 Quick Actions 點選Edit Behaviors。

點選 Custom。

點選左下的 +。

取名 format project。

點選左下的 +，點選 Run Script。

點選 No selection 旁的箭頭，點選 Choose Script 後從 Finder 裡選擇 swift-format-project.sh。

大功告成 ! 現在開始我們隨時可用 format project 縮排整個專案的程式。
