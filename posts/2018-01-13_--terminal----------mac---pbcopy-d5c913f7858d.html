將 terminal 指令的結果複製到 mac 的 pbcopy

當我們在 terminal 輸入指令時，為了將結果複製，傳統的做法是辛苦地將結果選取後再按下 cmd + c 複製。

例子：抓取 JSON 資料。

curl "https://itunes.apple.com/search?term=戴佩妮&media=music"

其實用不著這麼麻煩，只要在指令後接上 | pbcopy，即可將結果複製 !

curl "https://itunes.apple.com/search?term=戴佩妮&media=music" | pbcopy

在 JSON Editor Online 貼上剛剛指令複製的結果，生成漂亮的 JSON 格式。
