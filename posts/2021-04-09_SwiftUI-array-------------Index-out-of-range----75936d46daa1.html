SwiftUI array 移除最後一個成員造成的 Index out of range 問題

開發 SwiftUI App 時，我們時常利用 List 或 ForEach 顯示 array 的資料。不過當我們將 array 的最後一個成員移除時，在某些情況卻可能造成可怕的 Index out of range 閃退。

ps: 此問題在 Xcode 14 的 iOS 16 & Xcode 13 的 iOS 15.5 測試時並不會發生，可能新版不用再擔心此問題。

讓我們看看以下鬼滅之刃殺鬼的例子。

假設 Ghost 的型別如下。

struct Ghost: Identifiable {
    let id = UUID()
    var number: Int
    var name: String
}

畫面的設計如下，按下殺鬼 button 時將殺鬼，將 array 的最後一個成員移除。

struct ContentView: View {
    
    @State private var ghosts = [
        Ghost(number: 1, name: "黑死牟"),
        Ghost(number: 2, name: "童磨"),
        Ghost(number: 3, name: "猗窩座"),
　　]
    
    
    var body: some View {
        VStack(alignment: .leading) {
            ForEach(ghosts){ ghost in
                HStack {
                    Image(systemName: "\(ghost.number).circle.fill")
                        .resizable()
                        .frame(width: 50, height: 50)
                    Text(ghost.name)
                }
            }
            
            Button {
                ghosts.removeLast()
            } label: {
                Text("殺鬼")
            }
            .disabled(ghosts.isEmpty)
        }
        .font(.largeTitle)
    }
    
}

以上程式看來一切正常，我們可以盡情殺鬼。然而若是 ForEach 裡顯示 array 成員時使用到 Binding，那就不行了，只要 array 的最後一個成員被移除，程式就會閃退。若是移除其它的 array 成員，或是增加成員，則不會有問題，看來 Swift 特別討厭最後一名的人。

struct ContentView: View {
    
    @State private var ghosts = [
        Ghost(number: 1, name: "黑死牟"),
        Ghost(number: 2, name: "童磨"),
        Ghost(number: 3, name: "猗窩座"),
        
    ]
    
    private func binding(for ghost: Ghost) -> Binding<Ghost> {
        guard let index = ghosts.firstIndex(where: { $0.id == ghost.id }) else {
            fatalError("Can't find")
        }
        return $ghosts[index]
    }
    
    var body: some View {
        VStack(alignment: .leading) {
            ForEach(ghosts){ ghost in
                HStack {
                    Image(systemName: "\(ghost.number).circle.fill")
                        .resizable()
                        .frame(width: 50, height: 50)
                    TextField("name", text: binding(for: ghost).name)
                        .textFieldStyle(RoundedBorderTextFieldStyle())
                }
            }
            
            Button {
                ghosts.removeLast()
            } label: {
                Text("殺鬼")
            }
            .disabled(ghosts.isEmpty)
        }
        .font(.largeTitle)
        .padding()
    }
}

按下殺鬼 button 時 array 的最後一個成員猗窩座被移除，因此程式閃退。

錯誤訊息如下

Swift/ContiguousArrayBuffer.swift:580: Fatal error: Index out of range

新版解法(Xcode 13，可相容 iOS 13)

從 Xcode 13 開始，List & ForEach 等接受 array 的集合元件可以傳入 array 的 Binding，透過此寫法可避免閃退問題。

struct Ghost: Identifiable {
    let id = UUID()
    var number: Int
    var name: String
}
struct ContentView: View {
    
    @State private var ghosts = [
        Ghost(number: 1, name: "黑死牟"),
        Ghost(number: 2, name: "童磨"),
        Ghost(number: 3, name: "猗窩座"),
    ]
    
    var body: some View {
        VStack(alignment: .leading) {
            ForEach($ghosts) { $ghost in
                HStack {
                    Image(systemName: "\(ghost.number).circle.fill")
                        .resizable()
                        .frame(width: 50, height: 50)
                    TextField("name", text: $ghost.name)
                        .textFieldStyle(.roundedBorder)
                }
            }
            
            Button(action: {
                ghosts.removeLast()
            }, label: {
                Text("殺鬼")
            })
            .disabled(ghosts.isEmpty)
        }
        .font(.largeTitle)
        .padding()
        
    }
    
}

舊版解法: custom binding

不要用 $ 產生 Binding，而是自己產生 custom Binding。

struct ContentView: View {
    
    @State private var ghosts = [
        Ghost(number: 1, name: "黑死牟"),
        Ghost(number: 2, name: "童磨"),
        Ghost(number: 3, name: "猗窩座"),
    ]
    
    var body: some View {
        VStack(alignment: .leading) {
            ForEach(ghosts){ ghost in
                HStack {
                    Image(systemName: "\(ghost.number).circle.fill")
                        .resizable()
                        .frame(width: 50, height: 50)
                    TextField("name", text:
                                Binding(get: {
                                    ghost.name
                                }, set: {
                                    guard let index = ghosts.firstIndex(where: { $0.id == ghost.id }) else {
                                        fatalError("Can't find")
                                    }
                                    ghosts[index].name = $0
                                }))
                        .textFieldStyle(RoundedBorderTextFieldStyle())
                }
            }
            
            
            Button(action: {
                let index = Int.random(in: ghosts.indices)
                ghosts.removeLast()
            }, label: {
                Text("殺鬼")
            })
            .disabled(ghosts.isEmpty)
        }
        .font(.largeTitle)
        .padding()
        
        
    }
    
}
