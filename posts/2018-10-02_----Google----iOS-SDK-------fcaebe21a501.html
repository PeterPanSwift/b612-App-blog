如何在 Google 查詢 iOS SDK 的相關說明

如果想從 Google 查詢 iOS SDK，找出某個型別，屬性，或方法在 Apple Developer Documentation 裡的說明，可採用以下方法快速找到目標:

1 查詢型別

輸入型別名字。

例子: AVSpeechUtterance

搜尋結果裡，標題以 Apple Developer Documentation 結尾的即是我們要找的目標。

AVSpeechUtterance - AVFoundation | Apple Developer Documentation

說明文字

AVSpeechUtterance
A chunk of text to be spoken, along with parameters that affect its speech.
2 查詢屬性

輸入型別名和屬性名。

voice - AVSpeechUtterance | Apple Developer Documentation
Edit descriptiondeveloper.apple.com

說明文字

voice
The voice used to speak the utterance.
3 查詢方法

輸入型別名，func 和方法名。

speak(_:) - AVSpeechSynthesizer | Apple Developer Documentation
Edit descriptiondeveloper.apple.com

說明文字

speak(_:)
Enqueues an utterance to be spoken.
