新增 Cocoa Touch Class 的步驟說明

當我們想要新增類別，並讓它繼承 Cocoa Touch framework 的類別時，新增的步驟如下:

1 新增檔案。

方法 1: 從 Quick Actions 選 New > File from Template。

方法 2: 點選 File > New > File from Template 新增檔案。(或是快速鍵 cmd + N)

方法 3: 點選 Project navigator 左下的 +，或是從 Project navigator 叫出右鍵選單，然後點選 New File from Template。

2 選擇 Cocoa Touch Class。

因為想讓新增的類別繼承自 Cocoa Touch framework 的類別，所以選擇 Cocoa Touch Class。iOS SDK 裡大部分的類別，都來自 Cocoa Touch framework，比方 UIViewController & UIImageView。

3 從 Subclass of 欄位設定繼承的父類別。

方法1: 從下拉清單選擇。

方法2: 輸入前幾個字，讓 Xcode 自動補上後面的字。

比方輸入 uiviewc，自動就會變成 UIViewController。

4 從 Class 欄位輸入類別的名字。

Class 欄位將依我們選擇的父類別而改變，比方當父類別是 UIViewController 時，Class 欄位將自動變成 ViewController，此時我們只要在 ViewController 前再補上適合的名字，比方 Login，變成 LoginViewController 即可。

又比方父類別為 UITableViewController 時，Class 將自動變成 TableViewController。

5 點選右下角的 Next。
6 點選右下角的 Create 建立類別的 swift 檔。

經過剛剛的步驟，順利生成 LoginViewController.swift，檔名將和類別的名稱一致。
