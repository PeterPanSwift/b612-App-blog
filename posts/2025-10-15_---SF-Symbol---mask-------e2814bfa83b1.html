利用 SF Symbol 和 mask 裁切多邊形

設計 iOS App 時，將圖片裁切成多邊形有時能帶來更有設計感的視覺效果。

為了實現多邊形圖片，我們必須使用多邊形當圖片的遮罩。那多邊形哪裡來呢 ? 通常有三種做法。

  • 請美術提供多邊形的圖片。
  • 用程式繪製多邊形。
  • 使用 SF Symbol。

其中 SF Symbol 應該是最簡單的做法，它本身提供許多常見的形狀，例如以下例子。

  • 三角形(triangle.fill、righttriangle.fill)。
  • 菱形(rhombus.fill)。
  • 五邊形(pentagon.fill)。
  • 六邊形(hexagon.fill)。
  • 八邊形(octagon.fill)。
  • 寶石(diamond.fill)。
  • 盾牌(shield.fill)。
  • 封印形(seal.fill)。
  • 星形(star.fill)。
  • 膠囊(capsule.fill)。
  • 愛心(heart.fill)。
struct ContentView: View {
    let shapes = [
        "triangle",
        "righttriangle",
        "rhombus",
        "pentagon",
        "hexagon",
        "octagon",
        "diamond",
        "shield",
        "seal",
        "capsule",
        "star",
        "heart"
    ]
    
    var body: some View {
        VStack(alignment: .leading) {
            ForEach(shapes, id: \.self) { shape in
                HStack {
                    Text(shape)
                        .font(.largeTitle)
                    Image(systemName: shape)
                        .resizable()
                        .scaledToFit()
                        .symbolVariant(.fill)
                        .foregroundStyle(.orange.gradient)
                    
                }
            }
        }
    }
}

以下分別為 SwiftUI & UIKit 使用多邊形 SF Symbol 當遮罩的寫法。

  • SwiftUI。

在 mask modifier 傳入多邊形的 SF Symbol，搭配 symbolVariant(.fill) 設定填滿的形狀。

struct ContentView: View {
    
    var body: some View {
        Image(.prince)
            .resizable()
            .scaledToFill()
            .frame(width: 300, height: 300)
            .mask {
                Image(systemName: "seal")
                    .resizable()
                    .scaledToFit()
                    .symbolVariant(.fill)
            }
    }
}

加上多邊形邊框。mask 和 background 都是一樣形狀的 SF Symbol，但是 background 的尺寸較大，因此成為多邊形的邊框。

struct ContentView: View {
    let borderWidth: Double = 10
    let size: Double = 300
    
    var body: some View {
        Image(.prince)
            .resizable()
            .scaledToFill()
            .frame(width: size, height: size)
            .mask {
                Image(systemName: "seal")
                    .resizable()
                    .scaledToFit()
                    .symbolVariant(.fill)
            }
            .background {
                Image(systemName: "seal")
                    .resizable()
                    .scaledToFit()
                    .frame(width: size + borderWidth, height: size + borderWidth)
                    .foregroundStyle(.orange.gradient)
            }
            .shadow(radius: 5)
    }
}

ps: SF Symbol 的線條也可用 fontWeight 調整粗細。

struct ContentView: View {
  
    var body: some View {
        VStack {
            Image(systemName: "seal")
                .resizable()
                .scaledToFit()
                .fontWeight(.heavy)
            Image(systemName: "seal")
                .resizable()
                .scaledToFit()
            Image(systemName: "seal")
                .resizable()
                .scaledToFit()
                .fontWeight(.ultraLight)
        }
        
    }
}

我們也可以將包含多個元件的區塊裁切成多邊形，以下例子對 ZStack 進行裁切。

struct ContentView: View {

    var body: some View {
        ZStack {
            Image(.prince)
                .resizable()
                .scaledToFill()
                .frame(width: 300, height: 300)
            Text("Little Prince")
                .font(.largeTitle)
                .foregroundStyle(.white)
        }
        .mask {
            Image(systemName: "seal")
                .resizable()
                .scaledToFit()
                .symbolVariant(.fill)
        }
    }
}
  • UIKit

將 princeImageView 的 mask 設為多邊形的 SF Symbol。

class ViewController: UIViewController {

    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        let size = 300
        let princeImageView = UIImageView(frame: CGRect(x: 0, y: 0, width: size, height: size))
        princeImageView.image = .prince
        princeImageView.contentMode = .scaleAspectFill
        princeImageView.center = view.center
        view.addSubview(princeImageView)
        
        let maskImageView = UIImageView(frame: CGRect(x: 0, y: 0, width: size, height: size))
        maskImageView.image = UIImage(systemName: "seal.fill")
        maskImageView.contentMode = .scaleAspectFit
        princeImageView.mask = maskImageView

    }
}
