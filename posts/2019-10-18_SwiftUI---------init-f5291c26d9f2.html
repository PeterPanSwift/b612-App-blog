SwiftUI 搭配字串參數的 init

SwiftUI 很多元件在生成時可以設定顯示的內容，比方設定 button 要顯示文字還是圖片。由於顯示文字是最常見的應用，因此很多 SwiftUI 的元件都提供搭配字串參數的 init 方便我們傳入字串，不用再費心生成 Text。接下來就讓我們看看一些例子吧。

Button

比較麻煩的寫法: 生成 Text 描述 button 顯示。(ps: 文字內容是周興哲的終於了解自由歌詞)

Button {
            
} label: {
    Text("我終於了解自由，不是都是心碎了後才懂")
}

比較簡單的寫法: 直接傳入字串。

Button("我終於了解自由，不是都是心碎了後才懂") {
}

Toggle

Toggle("心碎", isOn: $isHeartBreak)
   .padding()

DatePicker

DatePicker("生日", selection: $birthday, displayedComponents: .date)

TextField

TextField("你的名字", text: $name)
   .textFieldStyle(.roundedBorder)
   .padding()
