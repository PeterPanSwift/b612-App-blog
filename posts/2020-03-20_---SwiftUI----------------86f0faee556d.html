利用 SwiftUI 繪製同時填滿顏色和線條的形狀

利用 SwiftUI 的 Path 我們可以繪製任何形狀，然後用 fill 填滿顏色或是用 stroke 繪製線條。

struct ContentView: View {
    var body: some View {
        VStack {
            Path { path in
                path.move(to: CGPoint(x: 0, y: 0))
                path.addLine(to: CGPoint(x: 20, y: 300))
                path.addLine(to: CGPoint(x: 300, y: 300))
                path.closeSubpath()
            }
            .fill(.red)
            
            Path { path in
                path.move(to: CGPoint(x: 0, y: 0))
                path.addLine(to: CGPoint(x: 20, y: 300))
                path.addLine(to: CGPoint(x: 300, y: 300))
                path.closeSubpath()
            }
            .stroke(lineWidth: 5)
        }
    }
}

不過很可惜的，我們不能同時呼叫 fill & stroke，讓它同時填滿顏色跟包含線條。

想讓剛剛畫出的三角形同時有顏色跟線條，我們得辛苦一點，畫兩個三角形，一個用 fill 填色，一個用 stroke 畫線，然後搭配 ZStack 或 overlay 疊在一起。

方法一: 使用 ZStack

struct ContentView: View {
    var body: some View {
        
        ZStack {
            Path { path in
                path.move(to: CGPoint(x: 0, y: 0))
                path.addLine(to: CGPoint(x: 20, y: 300))
                path.addLine(to: CGPoint(x: 300, y: 300))
                path.closeSubpath()
            }
            .fill(.red)
            
            Path { path in
                path.move(to: CGPoint(x: 0, y: 0))
                path.addLine(to: CGPoint(x: 20, y: 300))
                path.addLine(to: CGPoint(x: 300, y: 300))
                path.closeSubpath()
            }
            .stroke(lineWidth: 5)
            
        }
        
        
    }
}

方法二: 使用 ZStack，將 Path 定義成形狀

懶得將繪製三角形的 Path 重覆寫兩次的話，也可以將它定義成遵從 protocol Shape 的形狀，如此即可重覆使用。

我們將剛剛繪製三角形的 Path 定義成型別 Triangle，裡面的 Path 路徑先寫死，想做得更彈性的朋友則可依參數 rect 的尺寸調整繪製的座標。

struct Triangle: Shape {
    func path(in rect: CGRect) -> Path {
        
        Path { path in
            path.move(to: CGPoint(x: 0, y: 0))
            path.addLine(to: CGPoint(x: 20, y: 300))
            path.addLine(to: CGPoint(x: 300, y: 300))
            path.closeSubpath()
        }
    }    
}
struct ContentView: View {
    var body: some View {
        
        ZStack {
            Triangle()
                .fill(.red)
            
            Triangle()
                .stroke(lineWidth: 5)
            
        }
        
    }
}

方法三: 使用 overlay，將 Path 定義成形狀

struct Triangle: Shape {
    func path(in rect: CGRect) -> Path {
        
        Path { path in
            path.move(to: CGPoint(x: 0, y: 0))
            path.addLine(to: CGPoint(x: 20, y: 300))
            path.addLine(to: CGPoint(x: 300, y: 300))
            path.closeSubpath()
        }
    }
}
struct ContentView: View {
    var body: some View {
        
        Triangle()
            .fill(.red)
            .overlay(
                Triangle()
                    .stroke(lineWidth: 5)
            )
        
    }
}
