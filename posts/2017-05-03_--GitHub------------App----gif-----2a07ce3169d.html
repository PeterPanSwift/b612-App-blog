讓 GitHub 的專案頁面顯示美美的 App 操作 gif 或圖片

範例:

deeplovepan/BackToFirstPage
Contribute to BackToFirstPage development by creating an account on GitHub.github.com

步驟:

  1. 將 iOS App 專案上傳 (push) 到 GitHub。
  2. 在 Medium 發表文章，文章裡附上 App 操作 gif 或圖片。文章發表後，在文章的圖片上點選右鍵，點選 Copy Image Address，複製圖片的連結。

3. 連到 GitHub 上的專案頁面，點選右下角的按鈕 Add a README。到時候 README 的內容將是專案頁面呈現的介紹內容。

4. 加入剛剛複製的圖片連結，格式如下

![](圖片連結)

範例:

![](https://cdn-images-1.medium.com/max/1600/1*nLddRij6zs0K7A-DsMLj0A.gif)

點選 Preview 可預覽畫面。

5. 點選下方的 Commit new file，儲存剛剛做的修改。

大功告成 ! 現在我們的專案頁面終於可以呈現美美的 App 動畫 !

注意:

如果圖片太大，GitHub 會無法顯示，出現 Content length exceeded 的錯誤訊息。可先從以下連結將圖片壓縮。(圖片最好可以控制在 5MB 以下)
