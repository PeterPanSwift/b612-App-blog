Xcode 的專案名稱衝突問題

開發 iOS App 的第一步是建立專案(project)，不過專案名稱取名時可要小心，如果專案名稱和型別或 framework 一樣的話，將產生可怕的錯誤。

專案名和 framework 一樣

範例: 專案取名為 SwiftUI，和 iOS SDK 的 SwiftUI framework 衝突。

警告訊息提到 File ‘ContentView.swift’ is part of module ‘SwiftUI’; ignoring import，因此它無法 import SwiftUI，造成程式裡使用到 SwiftUI 的程式皆產生錯誤。

專案名和型別一樣

範例: 專案取名為 ContentView。專案裡有個以 struct 定義的型別 ContentView，因此產生衝突。

App 可以執行，但是無法預覽。

錯誤訊息提到 circular reference。
