iOS 推播可以顯示 gif 動畫了 !

iOS 的推播可以顯示多媒體的內容，除了我們熟悉的 jpg, png 圖片，mp3 音樂，以及 mpeg 影片，現在連 gif 都支援了 !

當收到 gif 推播時，一開始顯示的縮圖並不會動。但當我們利用 3D touch 按住推播展開後，即可看到生動的動畫。

範例程式

import UIKit
import UserNotifications
class ViewController: UIViewController {
   override func viewDidLoad() {
      super.viewDidLoad()
      if let url = Bundle.main.url(forResource: "demo", withExtension: "gif"), let attachment = try? UNNotificationAttachment(identifier: "", url: url, options: nil) {
         let content = UNMutableNotificationContent()
         content.attachments = [attachment]
         content.title = "動畫"
         let trigger = UNTimeIntervalNotificationTrigger(timeInterval: 10, repeats: false)
         let request = UNNotificationRequest(identifier: "notification", content: content, trigger: trigger)
         UNUserNotificationCenter.current().add(request, withCompletionHandler: nil)
   }
}
