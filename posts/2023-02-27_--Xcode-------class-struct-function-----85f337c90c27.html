在 Xcode 裡複製整段 class/struct/function 的程式

彼得潘寫 Blog 文章或製作投影片時，常常會從 Xcode 複製整段 class/struct/function 的程式，之前傻傻地將整段程式選取後再複製，遇到太長的程式還要捲動畫面，最近終於發現比較方便的方法了。

透過以下三個步驟可以快速複製整段 class/struct/function 的程式。

  • 將 class/struct/function 整段程式折疊。
  • 選取折疊後的 ContentView。(觸控板按 3 下)
  • 按下 cmd + c 複製(copy)。

以下我們以複製 struct ContentView 的程式說明。

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "globe")
                .imageScale(.large)
                .foregroundColor(.accentColor)
            Text("Hello, world!")
        }
        .padding()
    }
}
  • 將 ContentView { } 的程式折疊。
利用 Xcode 的 fold 折疊 { } 程式
