儲存去背的圖片到 iOS 的 photo album

儲存圖片到 iOS 的 photo album 聽起來是件超簡單的事，只要以下兩個步驟。

  • 在 Info 頁面設定權限 Privacy — Photo Library Additions Usage Description 的說明文字。
  • 呼叫 UIImageWriteToSavedPhotosAlbum 將照片存入相簿。
if let image = UIImage(named: "peter") {
    UIImageWriteToSavedPhotosAlbum(image, nil, nil, nil)
}

不過如果想儲存去背的圖片到相簿，那就得額外做一些處理了。

以儲存 asset 裡帥氣的去背 peter 為例。

雖然 peter 是去背的，呼叫 UIImageWriteToSavedPhotosAlbum 存入相簿後卻會變成白色的背景。

為了順利儲存去背的圖片，我們必須經過以下程式的層層轉換。

func savePngImage(originalImage: UIImage?) {
    guard let originalImage,
          let ciImage = CIImage(image: originalImage),
          let data = UIImage(ciImage: ciImage).pngData(),
          let saveImage = UIImage(data: data) else {
        return
    }     
    UIImageWriteToSavedPhotosAlbum(saveImage, nil, nil, nil) 
}
  • 從 UIImage 生成 CIImage。
  • 從 CIImage 生成 UIImage。
  • 從 UIImage 生成 png data。
  • 從 Data 生成 UIImage。

呼叫以下程式後，成功地在 photo album 存入去背的圖片。

savePngImage(originalImage: UIImage(named: "peter"))
