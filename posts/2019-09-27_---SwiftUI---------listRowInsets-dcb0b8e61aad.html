設定 SwiftUI 表格內容邊距的 listRowInsets

當我們利用 SwiftUI 的 List 或 Form 呈現表格和表單時，預設它會讓內容和表格邊邊有段安全距離。

let names = ["Federer", "Nadal", "Djokovic", "Murray"]

var body: some View {
   List(names, id: \.self) { name in
      Image(name)
         .resizable()
         .scaledToFill()
         .frame(width: 200, height: 200)
         .clipped()
   }
}

如下圖所示，帥氣的網球四大天王全都乖乖地和左邊邊界保持距離。

表格內容和邊邊的距離可透過 listRowInsets 設定。

listRowInsets(_:) | Apple Developer Documentation
Applies an inset to the rows in a list.developer.apple.com

比方我們在以下程式的 Image 呼叫 listRowInsets，傳入 EdgeInsets()，指定它跟邊邊不要有任何距離。

List(names, id: \.self) { name in
   Image(name)
      .resizable()
      .scaledToFill()
      .frame(width: 200, height: 200)
      .clipped()
      .listRowInsets(EdgeInsets())
}

不過很遺憾的，以上程式沒有帶來任何改變，圖片和邊邊仍然保持著距離。

想讓 listRowInsets 發揮作用，我們必須在 List 後直接接 { }，然後在 { } 裡的元件設定 listRowInsets，比方以下例子在 ForEach 呼叫 listRowInsets。

List {
   ForEach(names, id: \.self) { name in
      Image(name)
         .resizable()
         .scaledToFill() 
         .frame(width: 200, height: 200)
         .clipped()
    }
   
    .listRowInsets(EdgeInsets())
}

如下圖所示，四大天王終於完美地緊貼著左邊邊界跟彼此，因為 EdgeInsets() 表示 Image 跟邊邊完全沒有任何距離。

當然，我們也可以傳入 EdgeInsets 時指定上下左右的距離，例如以下例子。

List {
   ForEach(names, id: \.self) { name in
      Image(name)
         .resizable()
         .scaledToFill()
         .frame(width: 200, height: 200)
         .clipped() 
   }
   .listRowInsets(EdgeInsets(top: 30, leading: 30, bottom: 30, trailing: 30))
}

當表格或表單以 Section 分類時，我們也可以從 Section 呼叫 listRowInsets，範例如下。

Form {
   Section {
      Image(names[0])
         .resizable()
         .scaledToFill()
         .frame(width: 200, height: 200)
         .clipped()
   }
   .listRowInsets(EdgeInsets())

   Section {
      Image(names[1])
         .resizable()
         .scaledToFill()
         .frame(width: 200, height: 200)
         .clipped()
   }
   .listRowInsets(EdgeInsets())
}
