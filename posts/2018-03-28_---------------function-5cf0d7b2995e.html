如何分辨自動完成選單裡相似的 function

開發 Swift iOS App 時，有時會遇到一些參數很多的 function。此時如果剛好螢幕較小，字體又設成比較大的字體，將可能遇到自動完成選單列出的 function 長得差不多，不知該選誰的窘境。

其實只要放大眼睛，還是有辦法分辨的。請留意自動完成選單裡，針對目前選擇對象的文字說明，從說明大概可猜出此 function 的作用和參數。

說明提到 physical spring，所以可猜到參數和 spring 有關。
