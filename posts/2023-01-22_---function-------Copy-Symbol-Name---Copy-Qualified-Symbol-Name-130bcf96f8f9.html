複製 function 完整名字的 Copy Symbol Name & Copy Qualified Symbol Name

利用 Xcode 的 Copy Symbol Name & Copy Qualified Symbol Name，我們可以快速取得 function 的完整名字。

Copy Symbol Name

滑鼠移到 function 上按右鍵，從選單裡選擇 Copy Symbol Name。

上圖的 function 會得到以下名字。

imagePickerController(_:didFinishPickingMediaWithInfo:)

Copy Qualified Symbol Name

滑鼠移到 function 上按右鍵，從選單裡選擇 Copy Qualified Symbol Name。

上圖的 function 會得到以下名字，包含定義 function 的型別名。

ViewController.imagePickerController(_:didFinishPickingMediaWithInfo:)
