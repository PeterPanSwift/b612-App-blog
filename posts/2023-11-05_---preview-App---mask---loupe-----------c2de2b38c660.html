使用 preview App 的 mask 和 loupe 突顯和放大重要區塊

彼得潘由於程式教學的關係，常常會擷取軟體操作的畫面，然後再用工具突顯和放大圖片裡重要的區塊。

有什麼方法可以突顯和放大圖片裡的區塊呢 ? 剛好彼得潘使用 Mac，Mac 內建的編輯軟體 Preview 就有這些功能，以下我們分別介紹利用 mask 和 loupe 突顯和放大重要區塊。

  • 使用 mask(遮罩)突顯重要區塊。
  • 使用 loupe(放大鏡)放大重要區塊。

使用 mask(遮罩)突顯重要區塊

我們想突顯下圖的 Interface 欄位。

用 Preview 打開圖片，點選右上的鉛筆圖示(Show Markup Toolbar)，然後點選下圖的藍色框框出現選單後，再點選代表 mask 的黃色框框。

此時會出現遮罩的框框，框框以外的區域會變暗，也就是框框內的內容會被突顯。我們可調整框框的大小，讓它突顯重要的地方。

以下我們讓框框突顯 Interface 欄位。

成功實現 Interface 被突顯的說明圖片，提示讀者 Interface 欄位要選 SwiftUI。

使用 loupe(放大鏡) 放大重要區塊

我們想放大預覽畫面左上的圖釘(Pin Preview)按鈕。

跟剛剛遮罩的操作類似，放大鏡的按鈕就在遮罩按鈕的右邊。

點選後將出現放大鏡的圓形泡泡，我們可移動和調整它的大小。

以下我們將泡泡移到圖釘按鈕上，讀者將能更清楚看到它。
