SwiftUI 設定 UI 元件樣式(加入 modifier)的各種方法

從 SwiftUI 設定 UI 元件樣式(加入 modifier)時，我們可以從 Preview 畫面調整，也可以從程式調整，調整後 Preview 畫面和程式將連動更新，同時滿足喜歡寫程式跟懶得打字的朋友。

接下來就讓我們以調整文字顏色為例，將彼得潘的名言 Dreams do come true, if only we wish hard enough. 變成橘色，認識 SwiftUI 設定 UI 元件樣式的各種方法。

  • 方法 1: 從 Modifiers library 加入 modifier。
  • 方法 2: 直接輸入程式。

方法 1: 從 Modifiers library 加入 modifier

從 Quick Actions 點選 Show Library。(快速鍵 cmd + shift + L)

  • 切換到 Modifiers library 分頁。
  • 將 modifier 加到元件上。

將 modifier 加到元件的方法有以下幾種。

方法1: 拖曳到程式要輸入的位置。

如下圖所示，我們將 Foreground Style 拖曳到 Text 下。

方法2: 將滑鼠游標停在程式要輸入的地方，然後從 Modifiers library 視窗雙擊某個 modifier。

方法3: 將滑鼠游標停在程式要輸入的地方，從 Modifiers library 選擇某個 modifier 後按 enter。

方法 2: 直接輸入程式

當我們已經很熟練時，直接輸入程式反而是最快的方法。比方我們輸入 .fore，然後從自動完成選單選擇 foregroundStyle 按 Enter，然後再輸入 .orange。

struct ContentView: View {
    var body: some View {
        Text("Dreams do come true, if only we wish hard enough.")
            .foregroundStyle(.orange)
    }
}
