安裝 GitHub Copilot for Xcode

從 CopilotForXcode 的 GitHub 頁面下載 GitHub Copilot for Xcode。

GitHub - github/CopilotForXcode: Xcode extension for GitHub Copilot
Xcode extension for GitHub Copilot. Contribute to github/CopilotForXcode development by creating an account on GitHub.github.com

點選 the latest release 下載。

打開 dmg 檔，將 GitHub Copilot for Xcode 移到 Applications 下。

啟動 GitHub Copilot for Xcode，點選 Open。

點選 Continue。

同意 GitHub Copilot for Xcode 需要的權限

點選 Accessibility Permission。

打開 GitHub Copilot for Xcode Extension 的開關。

點選 Extension Permission。

打開 GitHub Copilot for Xcode 的開關。

登入 GitHub Copilot

點選 Login to GitHub。

點選 Copy Code and Open 複製驗證碼並打開 GitHub 的驗證頁面。

ps: 如果網頁沒有打開，請從瀏覽器連到上圖的網址 https://github.com/login/device。

點選 Continue。

按下 cmd + v 貼上剛剛複製的驗證碼，點選 Continue。

點選 Authorize GitHub Copilot Plugin。

關掉 Xcode 的 Predictive code completion

為了避免 Xcode 內建的 AI 提示 Predictive code completion 跟 GitHub Copilot 的提示干擾，建議打開 Xcode 的 Settings > Editing > Completion。

將 Display code predictions 設為 Never。

使用 Copilot 的程式提示

經歷了千辛萬苦，現在我們終於能在 Xcode 使用 Copilot 寫程式了。

重新啟動 Xcode，打開專案，點選 Allow 同意 GitHub Copilot for Xcode 存取檔案的權限。

Xcode 的 menu 多出了 GitHub Copilot 的相關指令。

比方 Enable/Disable Completions 可以打開/關閉 Copilot 的自動完成功能。

通常我們會透過兩種方法請 Copilot 輸入程式。

  • 輸入注解後按 enter，請 Copilot 建議相關的程式。
  • 輸入部分程式，請 Copilot 建議相關的程式。

struct ContentView: View {
    // 檢查是否為正確的 email 格式
    func isValidEmail(_ email: String) -> Bool {
        let emailRegEx = "[A-Z0-9a-z._%+-]+@[A-Za-z0-9.-]+\\.[A-Za-z]{2,64}"
        let emailPred = NSPredicate(format:"SELF MATCHES %@", emailRegEx)
        return emailPred.evaluate(with: email)
    }

關閉 Copilot 的程式提示

  • 方法 1: 從 Quick Actions 點選 GitHub Copilot > Enable/Disable Completions。
  • 方法 2: 從 GitHub Copilot for Xcode 的 Settings Advanced 分頁關掉 Request suggestions with typing。

使用 Copilot 修改專案程式和問問題

從 Quick Actions 點選 GitHub Copilot > Open Chat。

從右下方可選擇 model。

AI 可分成 Ask 和 Agent 模式，Ask 模式只會回答問題，Agent 模式可以直接修改專案程式和執行 terminal 的指令。

範例: 寫一個猜數字 App。

點選 Keep 接受 AI 的修改，點選 Undo 拒絕 AI 的修改。

在 Ask mode 輸入 / 可選擇指令。

點選迴紋針按鈕可選擇檔案。

升級更新 GitHub Copilot for Xcode

點選 Mac 右上方的 GitHub Copilot icon，點選 Check for Updates。
