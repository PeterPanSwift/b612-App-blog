移除 SwiftUI Text 顯示數字時添加的逗號

當我們利用 SwiftUI Text 顯示數字時，它會貼心地幫我們做些微調，比方每三位數字加個逗號。

struct ContentView: View {
    let price = 1000
    
    var body: some View {
        Text("價錢:\(price)")
    }
}

在某些情況，比方顯示金額時，每三位數字加逗號的確能讓使用者更容易看出實際的價錢。

然而並非所有數字都適合加上逗號，比方以下例子的電影年份加上逗號就滿怪的。

struct ContentView: View {
    let name = "小丑：雙重瘋狂"
    let year = 2024
    
    var body: some View {
        VStack {
            Text(name)
            Text("上映年份:\(year)")
        }
    }
}

想讓 Text 顯示數字時不加逗號有許多方法，以下我們示範三種方法。

  • 方法 1。
Text("上映年份:\(String(year))")
  • 方法 2。
Text("上映年份:\(year.formatted(.number.grouping(.never)))")
  • 方法 3。
Text("上映年份:\(year, format: .number.grouping(.never)))")
