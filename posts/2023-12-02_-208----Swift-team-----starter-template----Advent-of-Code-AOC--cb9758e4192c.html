什麼是 Advent of Code(AOC)

為了迎接聖誕老公公的到來，巧克力、餅乾和化妝品公司常常會推出有趣的聖誕倒數月曆(Advent Calendar)。

身為工程師，比起巧克力、餅乾和化妝品，你更喜歡有趣的程式解題嗎 ? 那你不妨考慮參加 Eric 大大每年在十二月推出的 Advent of Code。從 12/1 開始每天一道跟聖誕節有關的題目，然後持續到 12/25，幫助我們在聖誕節和另一半吃聖誕大餐前多寫一些程式減肥。

害怕題目太難嗎？別擔心，Advent of Code 的難度是逐步增加，前幾天的題目很適合給初學者練功增加信心。而且每天的題目還分成 part1 和 part2，part1 的難度較低，過關之後才會看到進階的 part2。

現在我們可以在 Day01.swift 撰寫解題的程式，主要修改 property entities 和 function part1 & part2。

我們會將 input 檔案的字串解析成方便程式解題的 array，存在 property entities。

Day 1 的 input 適合用 \n 分割後存成字串 array，以下是兩種可能的寫法。

  • 寫法 1。
var entities: [String] {
    data.components(separatedBy: .newlines)
}
  • 寫法 2。
 var entities: [String] {
     data.split(separator: "\n").map(String.init)
 }

在 function part1 & part2 輸入 part1 和 part2 的解題程式，function 的回傳結果即為題目要求的結果。

以下為彼得潘修改後的 part1 程式。

func part1() -> Any {
    entities.reduce(0) { partialResult, line in
        let first = line.first { $0.isNumber }
        let last = line.last { $0.isNumber }
        if let first,
           let last,
           let number = Int("\(first)\(last)") {
            return partialResult + number
        } else {
            return partialResult
        }
    }
}

要完成 part1 才能看到 part2 的題目。因此請先修改 AdventOfCode.swift 的 allChallenges，將測試的題目改成 Day01，然後執行程式印出 part1 的結果。

let allChallenges: [any AdventDay] = [
  Day01()
]
