設定 VS Code 修改程式後自動存檔(Auto Save)

VS Code 預設不會自動存檔，我們必須手動按下 cmd + s 才會存檔。不過 VS Code 其實也能設定修改程式後自動存檔，方法如下。

從 Command Palette 點選 Open Settings。

輸入 auto save，找到 Files: Auto Save 區塊。

有三種方式可觸發 VS Code 自動儲存。

  • afterDelay。

修改程式後過一段時間儲存。

  • onFocusChange。

離開修改的程式視窗時儲存，比方選擇另一個檔案或是切換到另一個 App。

  • onWindowChange。

切換視窗時儲存。

通常想修改程式後自動存檔，我們會選擇 afterDelay。

預設的 delay 是 1000 ms，因此修改程式後過 1 秒會自動儲存。從 Files: Auto Save Delay 欄位可調整 delay 的時間。
