SwiftUI 顯示圖文並排的 Label

開發 iOS App 時，我們時常看到圖片在左，文字在右的設計。這樣的設計，可以簡單地透過 HStack 呈現。

HStack {
    Image(systemName: "books.vertical.fill")
    Text("books")
}

不過在 SwiftUI 2，有更簡單的做法了。SwiftUI 提供了專門顯示圖文並排的 Label 元件。

Apple Developer Documentation
Edit descriptiondeveloper.apple.com

顯示圖文並排的 Label，搭配 SF Symbols

Label("books", systemImage: "books.vertical.fill")

修改 Label 的圖片顏色

Label("books", systemImage: "books.vertical.fill")
    .foregroundColor(.red)

顯示圖文並排的 Label，搭配自訂的 image

Label("peter", image: "peter")

搭配自訂的元件

生成 Label 時，我們也可以在參數 title & icon 傳入自訂的元件，比方以下例子。

struct ContentView: View {
    
    
    var body: some View {
        Label(
            title: {
                VStack(alignment: .leading) {
                    Text("the murder of roger ackroyd")
                    Text("Agatha Christie")
                }
            },
            icon: {
                Image("book")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 200, height: 200)
            }
        )
    }
}

List 裡自動對齊的 label

當 List 的內容以 Label 呈現時，它還多了自動對齊的好處。

比方以 HStack 呈現圖文時，由於圖片大小不一樣，所以文字的左邊將無法對齊。

List {
            HStack {
                Image(systemName: "arrow.up")
                Text("arrow")
            }
            HStack {
                Image(systemName: "books.vertical.fill")
                Text("books")
            }
            
}

同樣的內容，改以 Label 實現時，不費吹灰之力，文字就能完美對齊。

List {
    Label("arrow", systemImage: "arrow.up")
    Label("books", systemImage: "books.vertical.fill")
            
}

調整 List 裡所有圖片的顏色。

List {
   Label("arrow", systemImage: "arrow.up")
   Label("books", systemImage: "books.vertical.fill")
}
.accentColor(.red)

調整圖片的顏色。

List {
   Label("arrow", systemImage: "arrow.up")
       .listItemTint(.green)
   Label("books", systemImage: "books.vertical.fill")
       .listItemTint(.purple)
}
