從 Android Studio 建立 Flutter 專案和啟動模擬器執行 App

  • 從 Android Studio 建立 Flutter 專案。
  • 啟動 iOS 模擬器執行 App。
  • 啟動 Android 模擬器執行 App。

建立 Flutter 專案

  • 打開 Android Studio，在 Projects 分頁點選 New Flutter Project。

ps: 如果沒看到 New Flutter Project，請先參考以下連結準備好 Flutter App 的開發環境和安裝 Flutter Plugin。

在 Mac 上準備 Flutter App 的開發環境
透過 Google 推出的 Flutter framework 我們可以更方便地開發跨平台 App，開發出在 iOS，Android，Web，Windows 和 Linus 上都能執行的 App。apppeterpan.medium.com
在 Windows 上準備 Flutter App 的開發環境
透過 Google 推出的 Flutter framework 我們可以更方便地開發跨平台 App，開發出在 iOS、Android、Web、macOS、Windows 和 Linus 上都能執行的 App。medium.com
  • 選擇 Flutter，確認 Flutter SDK path 後按 Next。
  • 設定專案資訊。

彼得潘的目標是開發 iOS & Android App，因此 Project type 選擇 Application，Android language 選擇 Kotlin，iOS language 選擇 Swift，Platforms 只勾選 Android 和 iOS。Project location 要避免存在跟 iCloud 或 Dropbox 等雲端空間同步的地方，雲端同步將影響之後 Flutter App 的執行，出現 Target debug_unpack_ios failed、Cannot open file 等錯誤訊息。

點選 Create 後，成功建立 Flutter 專案，出現下圖的專案視窗。

ps: 除了從 IDE 建立專案，我們也可以從 terminal 輸入指令。

打開 Terminal，輸入指令 flutter create + 專案名稱建立專案。

flutter create first_app

結果如下，成功產生專案的資料夾 first_app，資料夾下包含開發 App 會用到的相關檔案。

  $ cd first_app

ps: 專案名稱的英文只能小寫，請用 _ 分隔單字，而且不能用數字開頭。

啟動 iOS 模擬器執行 App

  • 啟動 iOS Simulator。

我們必須先啟動 iOS Simulator，之後才能將 App 裝到啟動的 iOS Simulator 。請點選下圖的 Open iOS Simulator。

啟動成功後出現 iOS Simulator。

  • 選擇 iPhone 16 Pro (mobile)。
  • 啟動 App。

點選下圖的三角形啟動 App。(快速鍵 ctrl + r)

啟動時 Console 出現以下訊息。

模擬器上出現了彼得潘開發的第一個 Flutter iOS App。

點選右下的 + 可增加數字，每按一次加 1，彼得潘開心地一直按呀按，加到了 10。

  • hot reload。

開發 Flutter App 時，我們可利用 hot reload 功能加快開發速度。當我們修改程式時，hot reload 會直接更新 App，因此我們不用再花時間重新啟動 App。

以下圖為例，我們將 App 畫面中間的 You have pushed the button this many times: 改成 I love you。

App 的畫面馬上神奇地更新成 I love you。等下彼得潘要努力地按到 3000，跟電影鋼鐵人的 I love you 3000 看齊。

如果 App 畫面沒有自動更新，我們也可以自己按下圖的閃電按鈕觸發 hot reload。

啟動 Android 模擬器執行 App

  • 從上方的裝置清單選擇 Open Android Emulator。

如果沒看到 Open Android Emulator 的選項，請先點選 Refresh 重新整理。

  • 顯示 Android Emulator。

點選 Running Devices 按鈕顯示剛剛啟動的 Android Emulator。

  • 選擇剛剛啟動的 Emulator。
  • 啟動 App。

點選下圖的三角形啟動 App。(快速鍵 ctrl + r)

啟動時 Console 出現以下訊息。

模擬器上出現了彼得潘開發的第一個 Flutter Android App，點選右下的 + 一樣可增加數字，每按一次加 1。

若想讓 Running Devices 變成獨立視窗，可在 Pixel 的頁籤上按出右鍵選單，從 View Mode 選擇 Window。

獨立視窗的 Running Devices 還可以全螢幕顯示。

  • hot reload。

修改程式後，hot reload 會直接更新 App，因此我們將畫面中間的文字改成 I love you 後，畫面直接更新了。
