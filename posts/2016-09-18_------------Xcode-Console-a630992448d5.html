還模擬器一個乾乾淨淨的 Xcode Console

Xcode 的世界，就像現實世界一樣，是有差別待遇的。在模擬器執行 App 時，Console 會印出一堆我們不需要的訊息，但在實機執行 App 時，Console 卻變得乾乾淨淨，只印出我們程式裡 print 的訊息。

想要公平，不再有差別待遇，讓模擬器的 Console 乾乾淨淨也是可以的。只要照著以下步驟將 OS_ACTIVITY_MODE 關掉。

1 點選 App 的 Scheme ，選擇 Edit Scheme。

2 切換到 Run 的 Arguments 分頁。

3. 點選 Environment Variables 的 + 新增環境變數，Name 欄位輸入變數的名稱 OS_ACTIVITY_MODE，Value 欄位輸入變數的內容 disable。

從今而後，當我們在模擬器執行 App 時，Console 再也不會出現莫名奇妙的訊息了。
