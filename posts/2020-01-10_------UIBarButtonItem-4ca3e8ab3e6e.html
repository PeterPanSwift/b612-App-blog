顯示圖片的 UIBarButtonItem

想在 UIBarButtonItem 上顯示圖片，看似簡單，卻比想像中複雜許多，接下來就讓我們以彼得潘的圖片為例，看看如何在 UIBarButtonItem 正確顯示圖片吧。

假設我們想在 UIBarButtonItem 上顯示圖片 peter。

以下程式設定 navigation bar 右上角的 button，生成 UIBarButtonItem 時傳入 peter 的 UIImage。

@objc func buttonTap() {
        
}

override func viewDidLoad() {
    super.viewDidLoad()
    navigationItem.rightBarButtonItem = UIBarButtonItem(image: UIImage(named: "peter"), style: .plain, target: self, action: #selector(buttonTap))
}

然而結果卻令人失望，圖片的顏色變成一片藍，而且還偏移了。

為什麼會這樣呢 ? 這和以下兩個原因有關:

  • 圖片的尺寸太大。
  • UIBarButtonItem 預設會將圖片變成單一顏色。

因此我們有以下幾種解法，讓 UIBarButtonItem 顯示原本圖片的顏色，而且位置不會偏移。

解法 1: 將 assets 的圖片縮小，利用 withRenderingMode 讓圖片顯示原本的顏色

根據 Apple 官方的建議，navigation bar 上 button 的圖片最好是 24 * 24 ~ 28 * 28 points。

Custom Icons — Icons and Images — iOS — Human Interface Guidelines — Apple Developer
In iOS 13 or later, prefer using SF Symbols to represent tasks and modes in your app. If your app is running in iOS 12…developer.apple.com

因此我們將 peter 的大小改成 28 * 45。

然後在設定 UIBarButtonItem 的圖片時利用withRenderingMode(.alwaysOriginal) 讓圖片顯示原本的顏色。

navigationItem.rightBarButtonItem = UIBarButtonItem(image: UIImage(named: "peter")?.withRenderingMode(.alwaysOriginal), style: .plain, target: self, action: #selector(buttonTap))

剛剛的做法看起來挺好的，但只適用 assets 裡的圖片，而且無法應付以下兩種情況:

  • 來自網路下載或手機相簿的大尺寸圖片。
  • 無法將圖片變成其它比例或形狀，比方將長方形的圖片變成正方形或圓形。

沒關係，我們還有解法 2 & 3。

解法 2: 利用 UIGraphicsImageRenderer 將圖片縮小

當圖片尺寸很大時，我們可以參考以下連結，利用 UIGraphicsImageRenderer 將 UIImage 縮小。

利用 UIGraphicsImageRenderer & draw(in:) 縮小圖片
當我們利用 UIImagePickerController 拍照時，得到的照片大小為 3024 * 4032 pixel，然而一般我們的 App 畫面或是上傳到後台的圖片並不需要那麼大，因此我們時常會將拍照得到的照片縮小。medium.com
func resizeImage(image: UIImage, width: CGFloat) -> UIImage {
    let size = CGSize(width: width, height:
                        image.size.height * width / image.size.width)
    let renderer = UIGraphicsImageRenderer(size: size)
    let newImage = renderer.image { (context) in
        image.draw(in: renderer.format.bounds)
    }
    return newImage
}

呼叫 function resizeImage，將圖片變成 28 * 45 points。

if let image = UIImage(named: "peter") {
    let smallImage = resizeImage(image: image, width: 28)
    navigationItem.rightBarButtonItem = UIBarButtonItem(image: smallImage.withRenderingMode(.alwaysOriginal), style: .plain, target: self, action: #selector(buttonTap))
}

然而有時剛剛兩種解法都派不上用場，比方當我們想將圖片變成其它比例或形狀時，此時我們需要 customView !

解法 3: 利用 customView

如下圖所示，我們希望將長方形的照片變成圓形。

let button = UIButton()
button.imageView?.contentMode = .scaleAspectFill
button.imageView?.layer.cornerRadius = 14
button.translatesAutoresizingMaskIntoConstraints = false
button.setImage(UIImage(named: "peter"), for: .normal)
button.widthAnchor.constraint(equalToConstant: 28).isActive = true
button.heightAnchor.constraint(equalToConstant: 28).isActive = true
button.addTarget(self, action: #selector(buttonTap), for: .touchUpInside)
self.navigationItem.rightBarButtonItem = UIBarButtonItem(customView: button)

產生 UIBarButtonItem 時，我們可以利用參數 customView 傳入任何想顯示的元件。不過有一點要特別注意，元件的尺寸必須以 auto layout 的條件設定，不能直接設定 frame。

比方以下程式以 frame 設定 button 的大小為 28 * 28，最後顯示在 bar 的圖片卻完全不是我們想要的尺寸。

let button = UIButton()
button.imageView?.contentMode = .scaleAspectFill
button.imageView?.layer.cornerRadius = 14
button.setImage(UIImage(named: "peter"), for: .normal)
button.frame = CGRect(x: 0, y: 0, width: 28, height: 28)
button.addTarget(self, action: #selector(buttonTap), for: .touchUpInside)
self.navigationItem.rightBarButtonItem = UIBarButtonItem(customView: button)

解法 4: 利用 UIGraphicsImageRenderer 繪製縮小的圓形圖片

在解法 2 我們利用 UIGraphicsImageRenderer 將圖片縮小，其實它也可以將圖片變成圓形，如此不需透過 customView 也能顯示圓形圖片。有興趣的朋友可參考以下大大將 UIImage 變成圓形的 function circle。
