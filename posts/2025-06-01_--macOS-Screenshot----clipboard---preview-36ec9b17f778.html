將 macOS Screenshot 存到 clipboard 或 preview

彼得潘因為寫 blog 和製作上課投影片的關係，常常對 IDE 和 App 畫面進行螢幕截圖，因此之前常造成桌面充滿凌亂的截圖檔。

這些螢幕截圖大部分只是暫時需要的圖片，加入 blog 文章和投影片後即可刪除。彼得潘花了點時間研究後，發現 macOS 的 Screenshot 其實可以存到 clipboard 或 preview，以下是相關的操作方法。

  • cmd + ctrl + shift + 4 將選取的區域截圖存到 clipboard。

點選 cmd + v 可將截圖貼到 blog 文章或投影片，若想用 Preview App 編輯截圖，打開 Preview 後按下 cmd + n 即可開啟圖片進行編輯。

  • cmd + ctrl + shift + 3 將螢幕截圖存到 clipboard。
  • 設定預設的截圖存到 clipboard 或 preview。

前面介紹的兩種快速鍵按法有點難按，若想好按一點，我們可設定預設的截圖存到 clipboard 或 preview，如此預設的截圖快速鍵 cmd + ctrl + 4 和 cmd + ctrl + 3 即可將截圖存到 clipboard 或 preview。

點選 cmd + shift + 5 叫出截圖選單，點選 Options，Save to 選擇 Clipboard 或 Preview。

選擇 Preview 的話， 按下 cmd + ctrl + 4 和 cmd + ctrl + 3 截圖後將立即用 Preview 開啟圖片，方便我們編輯修改圖片。
