iOS App 如何產生獨一無二的 ID

開發 iOS App 時，有時我們需要產生獨一無二，不會跟別人同名的 ID。比方我們做了一個拍照 App，為了讓照片的檔名不會重覆，我們可用以下方法產生獨一無二的 ID。

方法1:
UUID().uuidString

結果

1D793079-C3C0–4D91-B2E1-A0F53A3A164C

uuidString | Apple Developer Documentation
Returns a string created from the UUID, such as "E621E1F8-C36C-495A-93FC-0C247A3E6E5F"developer.apple.com
方法2:
ProcessInfo().globallyUniqueString

結果

EC4AE0DA-98AE-4115–9D05–4B8CB84C1F7E-97986–0000E17D13E1E934

globallyUniqueString | Apple Developer Documentation
Global unique identifier for the process.developer.apple.com

ps: 除了寫程式產生 UUID，mac 上也有內建 uuidgen 的指令可以產生 UUID。

利用 uuidgen 產生 UUID
在 Mac 上要產生 UUID 十分容易，只要打開 terminal，輸入uuidgen，即可產生 UUID。medium.com
