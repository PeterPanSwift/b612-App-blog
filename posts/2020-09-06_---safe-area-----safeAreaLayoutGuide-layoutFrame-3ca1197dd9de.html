取得 safe area 大小的 safeAreaLayoutGuide.layoutFrame

開發 iOS App 時，有時我們會想知道 safe area 的大小。

我們可從 view.safeAreaLayoutGuide.layoutFrame 得到 safe area 的大小，不過記得要在 viewDidLayoutSubviews 或 viewDidAppear 裡讀取才能得到正確的大小。

Apple Developer Documentation
Edit descriptiondeveloper.apple.com

範例

設定 collection view 的 cell 大小為 safe area 的大小。由於 collection view 的大小等於 safe area 的大小，因此分頁的寬度等於 safe area 的寬度，剛好滑動一個 cell 等於滑動一個分頁。

class FullPhotoViewController: UIViewController, UICollectionViewDataSource {
   @IBOutlet weak var collectionViewLayout: UICollectionViewFlowLayout!
   override func viewDidLayoutSubviews() {
        super.viewDidLayoutSubviews()
        collectionViewLayout.itemSize = view.safeAreaLayoutGuide.layoutFrame.size
   }

ps: 剛剛的例子也可以直接設定 collection view 的 cell 大小等於 collection view 的大小。

class FullPhotoViewController: UIViewController, UICollectionViewDataSource {
   @IBOutlet weak var bookCollectionView: UICollectionView!
   @IBOutlet weak var collectionViewLayout: UICollectionViewFlowLayout!
   override func viewDidLayoutSubviews() {
        super.viewDidLayoutSubviews()
        collectionViewLayout.itemSize = bookCollectionView.bounds.size
    }
