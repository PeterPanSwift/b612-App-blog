從 Attributes inspector 移除 modifier 的兩種方法

方法1: 點選實心圓圈上的 X

目標移除範例裡的 padding。

struct ContentView: View {
    var body: some View {
        Text("Hello, world!")
            .padding()
    }
}

點選 text ，切換到它的 Attributes inspector 頁面

實心圓圈表示有加入的 modifier，因此目前只有加入 Padding。

滑鼠游標移到實心圓圈上，點選 X 刪除 modifier

圓圈變成空心，表示 padding modifier 已順利移除。

此時程式碼裡再也看不到 padding() 了。

struct ContentView: View {
    var body: some View {
        Text("Hello, world!")
    }
}

方法2: 點選 Delete

目標移除範例裡的 clipped。

struct ContentView: View {
    
    var body: some View {
        
        Image("peter")
            .resizable(resizingMode: .stretch)
            .aspectRatio(contentMode: .fill)
            .frame(width: 300.0, height: 300.0)
            .clipped()
    }
}

點選 Image ，切換到它的 Attributes inspector 頁面，找到 Clipped 區塊

滑鼠游標移到 Clipped 區塊的右上方，點選 Delete 刪除 modifier

此時程式碼裡再也看不到 clipped() 了。

struct ContentView: View {
    
    var body: some View {
        
        Image("peter")
            .resizable(resizingMode: .stretch)
            .aspectRatio(contentMode: .fill)
            .frame(width: 300.0, height: 300.0)
            
    }
}
