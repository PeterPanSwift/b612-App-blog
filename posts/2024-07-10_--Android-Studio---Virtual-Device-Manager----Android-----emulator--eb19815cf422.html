從 Android Studio 的 Virtual Device Manager 新增 Android 模擬器(emulator)

Android 有許多不同機型的模擬器，我們可從 Android Studio 的 Virtual Device Manager 新增 Android 模擬器。

ps: Android 模擬器的英文是 emulator、Android Virtual Device(AVD)。

打開 Android Studio 後，打開 Virtual Device Manager。

依據是否已開啟專案視窗，有不同的方法。

  • 未開啟專案視窗。

方法 1: 從右上的 More Actions 點選 Virtual Device Manager。

方法 2: 從 Find Action 點選 Virtual Device Manager。

  • 已開啟專案視窗。

方法 1: 從 Find Action 點選 Device Manager。

方法 2: 點選右側清單的 Device Manager。

打開 Virtual Device Manager後，點選 +，選擇 Create Virtual Device 新增模擬器。

選擇裝置。

設定相關資訊，然後點選 Finish 完成。

Device Manager 視窗出現我們剛剛新增的 Pixel 9API 35。
