隱藏 & 顯示向右箭頭的 navigationLinkIndicatorVisibility

當 SwiftUI List 包住 NavigationLink 時，NavigationLink 的右邊將貼心地長出提示跳到下一頁的向右箭頭 disclosure indicator。

比方以下例子彼得潘最愛的運動旁將顯示向右箭頭。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            List {
                NavigationLink("彼得潘最愛的運動") {
                    VStack {
                        Image(systemName: "figure.table.tennis.circle.fill")
                            .resizable()
                            .scaledToFit()
                        Text("桌球")
                            .font(.largeTitle)
                    }
                    .foregroundStyle(.orange)
                }
            }
        }
    }
}

然而有時我們會希望不要顯示向右箭頭。從 Xcode 26 開始，我們終於可以利用全新的 modifier navigationLinkIndicatorVisibility 控制箭頭的顯示，而且它可向前支援到 iOS 17。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            List {
                NavigationLink("彼得潘最愛的運動") {
                    VStack {
                        Image(systemName: "figure.table.tennis.circle.fill")
                            .resizable()
                            .scaledToFit()
                        Text("桌球")
                            .font(.largeTitle)
                    }
                    .foregroundStyle(.orange)
                }
                .navigationLinkIndicatorVisibility(.hidden)
            }
            
        }
    }
}

navigationLinkIndicatorVisibility 除了加在 NavigationLink，也可加在 List 或 ForEach 上。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            List {
                NavigationLink("彼得潘最愛的運動") {
                    VStack {
                        Image(systemName: "figure.table.tennis.circle.fill")
                            .resizable()
                            .scaledToFit()
                        Text("桌球")
                            .font(.largeTitle)
                    }
                    .foregroundStyle(.orange)
                }
            }
            .navigationLinkIndicatorVisibility(.hidden)
        }
    }
}

相反的，當 ScrollView 包住 NavigationLink 時，預設並不會長出向右箭頭。若想要顯示箭頭，我們可另外添加 .navigationLinkIndicatorVisibility(.visible)。

struct ContentView: View {
    var body: some View {
        NavigationStack {
            ScrollView {
                NavigationLink("彼得潘最愛的運動") {
                    VStack {
                        Image(systemName: "figure.table.tennis.circle.fill")
                            .resizable()
                            .scaledToFit()
                        Text("桌球")
                            .font(.largeTitle)
                    }
                    .foregroundStyle(.orange)
                }
            }
            .navigationLinkIndicatorVisibility(.visible)
        }
    }
}
