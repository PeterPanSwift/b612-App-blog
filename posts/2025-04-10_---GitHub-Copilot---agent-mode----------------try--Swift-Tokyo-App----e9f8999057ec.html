App 的講者圖片存在哪

原來是透過 rootViewController present SFSafariViewController 呀。

schedule 清單項目的背景色改成淡藍色漸層

ps: VS Code 在解析 swift 程式時預設會有一些錯誤，造成修改程式時出問題。因此建議先用 Xcode-Build-Server 生成 buildServer.json，讓 VS Code 能正確解析和識別 Xcode 專案的程式。

請在 terminal 輸入以下指令。

xcode-build-server config -workspace trySwiftTokyo.xcworkspace -scheme App
使用 VS Code 搭配 Xcode-Build-Server 開發 Xcode 專案的程式
