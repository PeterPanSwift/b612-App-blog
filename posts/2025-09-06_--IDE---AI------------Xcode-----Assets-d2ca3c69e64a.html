請 IDE 的 AI 從網路找圖後下載加到 Xcode 專案的 Assets

開發 iOS App 時，為了在 App 顯示美美的圖片，我們時常要辛苦地花時間做以下三件事。

  • 從網路找圖。
  • 下載圖片。
  • 將圖片加到 Xcode 專案的 Assets.xcassets。

不過這都是過去式了，隨著 AI 的進步，現在它已經能一條龍做完網路找圖、下載圖片、加到 Xcode 專案的 Assets 了。

以下我們以收集超多圖片的的 Openverse 為例，示範用 GitHub Copilot for Xcode、VS Code + Codex、VS Code + Claude Code 透過 Openverse API 將可愛的企鵝圖片加到 Assets。

Openverse API 的說明如下。

Openverse API
Edit descriptionapi.openverse.org
