黃色警告 file does not have a valid extension

當我們加到 assets 裡的圖片附檔名是大寫的 JPG / PNG 時，將出現黃色警告，比方上圖所示的 Ambiguous Content: The image set “cat” references a file “cat.JPG”, but that file does not have a valid extension.

解決的方法很簡單，只要將圖片的附檔名改成小寫的 jpg / png 即可。
