方便編輯 xliff 檔的 XLIFFTool

從 Xcode 13 開始，我們可以方便地從 xcloc 設定 iOS App 多國語言的翻譯文字。

從 xcloc 設定 iOS App 多國語言的翻譯文字
開發 iOS App 時，我們時常使用 xcloc (Xcode Localization Catalog) 設定多國語言的翻譯文字。不過它的格式是 xml，編輯起來並不是很方便。medium.com

不過如果是使用舊版的 Xcode 呢 ? 那就比較麻煩了，可參考以下的說明從 xliff 檔編輯翻譯的文字。

開發支援多國語言的 iOS App 時，我們通常會利用 Xcode 產生 xliff 檔，然後再請翻譯人員在 xliff 檔裡填寫翻譯的內容。(ps: xliff 的全名是 XML Localisation Interchange File Format)

比方我們用以下的 SwiftUI 程式為例，我們希望動物的名字有多國語言，英文版顯示 penguin & tiger，中文版顯示企鵝 & 老虎。

import SwiftUI
struct ContentView: View {
    var body: some View {
        HStack {
            VStack {
                Image("企鵝")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 150.0, height: 150.0)
                Text("penguin")
            }
            VStack {
                Image("老虎")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 150.0, height: 150.0)
                Text("tiger")
            }
        }
    }
}

Xcode 將幫我們產生包含繁體中文翻譯的 xliff 檔。

以上圖為例，翻譯人員可從 <source></source> 包含的文字找到需翻譯的文字，然後再手動在下方輸入 <target></target> 包含的翻譯內容。

<trans-unit id="penguin" xml:space="preserve">
<source>penguin</source>
<target>企鵝</target>
<note>No comment provided by engineer.</note>
</trans-unit>
<trans-unit id="tiger" xml:space="preserve">
<source>tiger</source>
<target>老虎</target>
<note>No comment provided by engineer.</note>
</trans-unit>

不過直接編輯 xliff 還是有點辛苦，所以接下來我們將從 App Store 找個好用的 xliff 檔編輯軟體。

在 App Store 搜尋 xliff 可找到很多 App，既然 XLIFFTool 排在第一個，而且又是免費，我們就試試它吧。

‎XLIFFTool
‎A simple tool for editing XLIFF files generated by Xcode 6+. From dev to dev :-) Features: * Spell-Checking in editing…apps.apple.com

啟動 XLIFFTool，點選 File > Open 開啟 xliff 檔

選擇欲開啟的 xliff 檔

xliff 的編輯畫面

從下圖的 Source 欄位可看到需要翻譯的文字。

在 Target 欄位輸入翻譯的內容

點選 File > Save 存檔

接著我們 import 剛剛編輯的 xliff 檔後，Xcode 將自動生成 Localizable.strings，包含我們設定的翻譯內容。
