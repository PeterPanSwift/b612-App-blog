生成 MusicKit App 需要的 developer token

建立 developer token 需經過以下三個步驟:

1 建立 Music ID。

2 建立 MusicKit 的 private key。

3 建立 developer token。

建立 Music ID

1 登入 Apple 的開發網站。

Apple Developer
There's never been a better time to develop for Apple Platforms.developer.apple.com

2 點選 Certificates, IDs & Profiles。

ps: 付費的開發會員才能進入 Certificates, IDs & Profiles 頁面，因此付費會員才有資格開發 MusicKit App。

3 新增 Music ID。

點選左邊側邊欄的 Music IDs 後，點選右上角的 + 新增 Music ID。

4 填寫 Description 和 ID。

新增的 Music ID 出現在列表。

建立 MusicKit 的 private key

1 新增 MusicKit 的 private key。

點選左邊側邊欄的 Keys > All 後，點選右上角的 + 新增 key。

2 輸入 key 的名字，勾選 Music Kit。

點選右邊的 Configure，選擇此 key 對應的 music id。

3 點選 Download，下載剛剛生成的 key。

記得要好好保存下載的 key，把它當成你的真愛 ! 因為 ... 你只有一次下載的機會，之後無法重新下載。

After downloading your key, it cannot be re-downloaded as
the server copy is removed.

下載的 key 是個附檔名 p8 的檔案。

建立 developer token

developer token 的格式為 JWT(JSON Web Tokens)。

JWT.IO

1 安裝 python 的套件管理工具 PIP (python package manager)

Installation - pip 9.0.1 documentation
get-pip.py will also install setuptools [3] and , if they're not already. setuptools is required to install source…pip.pypa.io

下載 get-pip.py

https://bootstrap.pypa.io/get-pip.py

sudo python get-pip.py

2. 安裝製作 JWT 的 python 套件 pyjwt。

sudo pip install pyjwt

3 安裝加密的 python 套件 cryptography。

sudo pip install cryptography

4 下載製作 developer token 需要的 music_token.py。

pelauimagineering/apple-music-token-generator
Contribute to apple-music-token-generator development by creating an account on GitHub.github.com

5 編輯 music_token.py，設定以下欄位

(1) secret (private key)

我們經歷千辛萬苦產生的 MusicKit private key，就是為了用在這裡。請複製 private key 的字串，將它貼到 secret 欄位。

(2) keyID

在 Apple 開發網頁的 Keys > All 頁面，可找到 key 的 key ID。

(3) teamId

在 Apple 開發網頁的 Membership 頁面，可找到 Team ID。

secret = """-----BEGIN PRIVATE KEY-----
MIGTAgEAMBMGByqGSM49AgEGCCqGSM49AwEHBHkwdwIBAQQgBJabyDbmG59WgHaL+QfFJkzlXWHubcadHhSO61K9CY6gCgYIKoZIzj0DAQehRANCAASNNOdHUZV9/Sgo4R+ulD1UX18ngMRCBlKecxU5EuRYQMyk7AulcXkWq8ZyjjmftmhpHQZBVEfyA+kOEm7ockU8
-----END PRIVATE KEY-----"""
keyId = "5Z9X3HX2RC"
teamId = "G4HL98LX6L"

生成 developer token，需要 alg，kid，iss，exp 和 iat。Apple 要求 alg 採用 ES256 加密，iat 決定 token 過期的時間，最長可設定 6 個月。我們只要完成剛剛 secret，keyID 和 teamId 的設定，其它部分此 python script 都幫我們處理好了。

6 產生 token。

python music_token.py

此 token 將幫助我們串接 Apple Music 的 API。若想檢查此 token 是否能順利串接 ，可用剛剛 python 指令輸出結果的 curl 字串測試。

有了 developer token，之後我們只要在 request 將 http header 的 Authorization 設為 Bearer 加 token，即可順利串接 Apple Music 的 API。

例子:
