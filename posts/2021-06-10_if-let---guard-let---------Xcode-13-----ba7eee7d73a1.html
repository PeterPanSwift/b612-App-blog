if let & guard let 的自動完成 — Xcode 13 新功能

Xcode 13 變得更方便了。當我們輸入 guard let 跟 if let 時，只要在 let 後輸入 optional 的名字，比方 name，就可以從選單選擇 name = name 的選項了。

if let 的自動完成

struct Baby {
    var name: String?
    
    func showName() {
        if let name = name {
            
        }
    }
}

guard let 的自動完成

struct Baby {
    var name: String?
    
    func showName() {
        guard let name = name else {
            return
        }
    }
}
