短影片來源

pixabay API。

抓取影片和圖片的 pixabay API
連到 pixabay 網站，註冊一個帳號medium.com

功能和畫面需求

  • 上下滑動切換影片，一頁顯示一個影片。
  • 影片會自動播放。
  • 影片播完會重覆播放。
  • 點選可暫停 / 播放影片。
  • 下方有 tab bar，切換到別的 tab 時暫停播放，回到影片的 tab 頁面時繼續播放。
  • 分頁功能，比方先抓 20 筆資料，滑到底再抓下 20 筆資料。
  • 右邊顯示一排按鈕，其中有一顆是顯示留言的按鈕。
  • 點選留言按鈕後會顯示留言列表。

可從 Dummy API 取得假的留言資料。
