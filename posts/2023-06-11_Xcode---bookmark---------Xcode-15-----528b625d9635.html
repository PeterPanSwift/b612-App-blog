Xcode 的 bookmark(書籤)功能 — Xcode 15 新功能

Xcode 15 加入了方便的 bookmark(書籤)功能，就像 Safari 可以將常去的網站加入 bookmark，現在 Xcode 也能將專案的程式加入 bookmark，方便我們隨時瀏覽正在開發研究的程式段落。

以下我們以 WWDC23 的 Backyard Birds 為例，示範如何在 Xcode 15 管理 bookmark。我們將介紹以下功能。

  • 新增 bookmark，可加入整個檔案，某行程式或是 find 的結果。
  • 瀏覽和管理 bookmark 清單。
  • 修改 bookmark 的描述。
  • 將 bookmark 打勾。

新增 bookmark

我們可以將整個檔案，某行程式或是 find 的結果加入 bookmark。

  • 將檔案加入 bookmark。

方法 1: 從右鍵選單選擇 Bookmark xxx.swift。

方法 2: 叫出 quick actions 選單後(cmd + shift + a) 輸入 boo。

  • 將某一行加入 bookmark。

選擇 Bookmark xxx Line xxx。

當某行程式被加入 bookmark，它的右邊會出現紫色的 bookmark icon。

  • 將 find 結果加入 bookmark。

搜尋的結果也可以加入 bookmark，比方我們想將出現 @Model 的地方加入 bookmark。

首先我們切到 Find navigator 搜尋 @Model。

從右鍵選單選擇 Bookmark Find xxx 即可將包含 @Model 的搜尋結果加入 bookmark。

瀏覽 bookmark 清單

從 Bookmark navigator 可瀏覽 bookmark 清單，下圖顯示我們剛剛加入的三種 bookmark，BirdsNavigationStack.swift，BirdFoodPickerSheet 的 153 行和 @Model Find 的結果。

我們可以上下拖曳調整 bookmark 的順序，當項目太多時我們也可以建立 Group 分類管理 bookmark。

當程式修改後，若想更新 bookmark 裡的 find 結果，則要點選下圖紅色框框的更新圖案。

修改 bookmark 的描述

bookmark 被加入後會有預設的描述，不過我們也可以修改，方法如下。

切到 Bookmark navigator，點選想修改的 bookmark。

再點擊一下會出現描述的輸入框。

從右鍵選單點選 Edit Description 也可以輸入描述。

除了剛剛的方法，當 bookmark 是某行程式時，我們也可以點選它右邊的 bookmark icon 修改描述。

我們在 Description 框框輸入研究 birdFoodShopLink，之後它將清楚地顯示在 bookmark icon 旁。

Bookmark navigator 裡的 bookmark 描述也更新成研究 birdFoodShopLink。

將 bookmark 打勾

bookmark 也可以當成 to do list 管理。將滑鼠游標移到 bookmark 左邊會出現小圓圈，我們可以勾選它表示項目已完成。

從右鍵選單點選 Mark as Complete 也可以打勾。
