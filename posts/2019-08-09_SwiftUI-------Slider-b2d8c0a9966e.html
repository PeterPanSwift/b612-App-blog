Slider | Apple Developer Documentation
A control for selecting a value from a bounded linear range of values.developer.apple.com

綁定浮點數的 Slider

Slider 的 init 接受型別 Binding<BinaryFloatingPoint> 的參數，當使用者滑動 slider 時，slider 綁定的資料會更新。我們必須傳入遵從 protocol BinaryFloatingPoint 的變數讓它綁定，像 Float，Double，CGFloat 這類浮點數型別都遵從 protocol BinaryFloatingPoint，因此可以跟 Slider 綁定。

讀取 slider 的內容

彼得潘一輩子的宿敵是討厭的虎克船長，接下來我們將用 slider 將 hook 縮小，跟討厭的 hook 說再見 !

struct ContentView: View {
    @State private var scale: Double = 1
    
    var body: some View {
        VStack {
            Image(.hook)
                .resizable()
                .scaleEffect(scale)
            Slider(value: $scale, in: 0...1)
        }
    }
}

說明

@State private var scale: Double = 1

slider 綁定的變數必須為浮點數，在此我們宣告型別 Double 的 state property scale。

scaleEffect(scale)

利用 scaleEffect 縮放圖片，比方 scale 等於 0.5 時圖片將變成一半的大小。

func scaleEffect(_ s: CGFloat, anchor: UnitPoint = .center) -> some View
scaleEffect(_:anchor:) | Apple Developer Documentation
Scales this view's rendered output by the given vertical and horizontal size amounts, relative to an anchor point.developer.apple.com
Slider(value: $scale, in: 0...1)

參數 in 控制 slider 的範圍，我們傳入 0…1，讓 hook 最大時是圖片原本的大小，最小時則完全消失在我們眼前。由於 in 參數的預設值為 0…1，所以我們也可省略它，變成 Slider(value: $scale)，它也會有一樣的效果。

顯示 Slider 數值時控制小數位數

當我們想在畫面上顯示 slider 的數值時，最好控制它顯示的位數，不然預設會顯示到小數點後好幾位。

Swift 指定小數位數的方法
使用 Swift 開發 iOS App 時，我們有幾種方法可指定小數的位數。medium.com
  • 方法 1: 使用 formatted function 或 format 參數。(iOS 15 以上)

從數字呼叫 formatted function，傳入 .number.precision(.fractionLength(2)) 得到顯示到小數點第二位的字串。

struct ContentView: View {
    @State private var scale: Double = 1
    
    var body: some View {
        VStack {
            Image(.hook)
                .resizable()
                .scaledToFit()
                .scaleEffect(scale)
            Slider(value: $scale, in: 0...1)
            let scaleString = scale.formatted(.number.precision(.fractionLength(2)))
            Text(scaleString)
        }
    }
}

從 format 參數傳入 .number.precision(.fractionLength(2)) 也可以得到顯示到小數點第二位的字串。

Text(scale, format: .number.precision(.fractionLength(2)))

若想要指定小數點位數最多兩位，但是 .0 的浮點數變整數字串，可在呼叫 formatted 時傳入 .number.precision(.fractionLength(0...2))，利用 .fractionLength(0...2)設定小數點位數的範圍。

struct ContentView: View {
    @State private var scale: Double = 1
    
    var body: some View {
        VStack {
            Image(.hook)
                .resizable()
                .scaledToFit()
                .scaleEffect(scale)
            Slider(value: $scale, in: 0...1)
            let scaleString = scale.formatted(.number.precision(.fractionLength(0...2)))
            Text(scale.formatted())
        }
    }
}

從 format 參數傳入 .number.precision(.fractionLength(0…2))) 也有一樣的效果。

Text(scale, format: .number.precision(.fractionLength(0...2)))
  • 方法 2: 使用 string interpolation。

以下程式我們在參數 specifier 傳入 %.2f，表示顯示到小數點後兩位。

struct ContentView: View {
    @State private var scale: Double = 1
    
    var body: some View {
        VStack {
            Image(.hook)
                .resizable()
                .scaledToFit()
                .scaleEffect(scale)
            Slider(value: $scale, in: 0...1)
            Text("\(scale, specifier: "%.2f")")
        }
    }
}

設定 slider 的顏色

利用 tint 控制 slider 的顏色。

struct ContentView: View {
   @State private var scale: Double = 1
    
   var body: some View {
      VStack {
         Image(.hook)
            .resizable()
            .scaleEffect(scale)
         Slider(value: $scale, in: 0...1)
              .tint(.orange)
      }
   }
}

隱藏拖曳鈕的 sliderThumbVisibility — iOS 26 以上

透過 sliderThumbVisibility(.hidden) 可隱藏拖曳鈕。

struct ContentView: View {
    @State private var scale: Double = 0.5

    var body: some View {
        Slider(value: $scale, in: 0...1)
            .sliderThumbVisibility(.hidden)
            .padding()

    }
}

沒有滑桿的 slider 其實常常看到，iOS 內建的音樂 App 調整播放進度和音量使用的正是沒滑桿的 slider。

設定 slider 移動的 step(距離)

有時我們想控制 slider 移動的距離，比方設定年紀的 slider，我們想要它是整數，像是 18，19 歲，而不是奇怪的 18.5 歲。如此小小的心願，透過 init 的參數 step 即可實現。

init<V>(value: Binding<V>, in bounds: ClosedRange<V>, step: V.Stride = 1, @ViewBuilder label: () -> Label, onEditingChanged: @escaping (Bool) -> Void = { _ in }) where V : BinaryFloatingPoint, V.Stride : BinaryFloatingPoint

以下程式參數 step 傳入 1 讓 slider 每次滑動的距離為 1，因此猜測彼得潘的年齡時，我們只能猜 10 ~ 20 之間的整數。當 age 是整數時，呼叫 formatted 將得到整數字串。而參數 label 傳入的 closure 產生的 Text("age") 並不會顯示，它的目的是為了 accessibility，比方搭配 VoiceOver 功能。

struct ContentView: View {
    @State private var age: Double = 18
    
    var body: some View {
        VStack {
            Text("彼得潘的真實年紀:\(age.formatted())")
            Slider(value: $age, in: 10...20, step: 1) {
                Text("age")
            }
        }
        .padding()
    }
}

如下圖所示，slider 滑動時只會停在整數的年紀。

顯示 slider 的最小值 & 最大值

我們也可以在 slider 的起點跟終點顯示最小值 & 最大值的文字，透過 init 的參數 minimumValueLabel & maximumValueLabel。

init<V>(value: Binding<V>, in bounds: ClosedRange<V>, step: V.Stride = 1, @ViewBuilder label: () -> Label, @ViewBuilder minimumValueLabel: () -> ValueLabel, @ViewBuilder maximumValueLabel: () -> ValueLabel, onEditingChanged: @escaping (Bool) -> Void = { _ in }) where V : BinaryFloatingPoint, V.Stride : BinaryFloatingPoint

我們在 minimumValueLabel 傳入 Text("18")，maximumValueLabel 傳入 Text("20")，讓 slider 顯示最小值 18 & 最大值 20。

struct ContentView: View {
    @State private var age: Double = 18
    
    var body: some View {
        VStack {
            Text("彼得潘的真實年紀:\(age.formatted())")
            Slider(value: $age, in: 18...20, step: 1) {
                Text("age")
            } minimumValueLabel: {
                Text("18")
            } maximumValueLabel: {
                Text("20")
            }
        }
    }
}

以圖片顯示 slider 的最小值 & 最大值

我們也可以在參數 minimumValueLabel & maximumValueLabel 傳入 Image 顯示圖片。

struct ContentView: View {
    @State private var age: Double = 18
    
    var body: some View {
        VStack {
            Text("彼得潘的真實年紀:\(age.formatted())")
            Slider(value: $age, in: 18...20, step: 1) {
                Text("age")
            } minimumValueLabel: {
                Image(systemName: "18.circle")
            } maximumValueLabel: {
                Image(systemName: "20.circle")
            }
        }
    }
}
