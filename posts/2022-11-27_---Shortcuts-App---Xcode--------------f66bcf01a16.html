利用 Shortcuts App 幫 Xcode 增添查詢單字和中翻英功能

撰寫 swift 程式時，我們時常花時間在以下兩件事情。

  • 查詢程式的某個單字是什麼意思。
  • 為了命名變數，function，型別，查詢某個單字怎麼拼。

這些都是簡單的小事，但是我們必須離開 Xcode，跳到 Google 上查詢。有沒有比較快的方法呢？

彼得潘研究一下後，發現 Shortcuts App 可以幫我們直接在 Xcode 完成這些事情。接下來我們將一步步示範利用 Shortcuts App 幫 Xcode 增添查詢單字和中翻英功能。

查詢單字意思

  • 新增 shortcut。

啟動 Shortcuts App，點選 Quick Actions，然後點選 +。

  • 輸入 shortcut 的名字。

命名為 show chinese。

  • 加入 Translate Text。

在 Action Library 頁面搜尋 Translate Text，將它拖曳到 Receive Any input 下。

  • 將翻譯的語言改成台灣中文。

將 to English (US) 改成 Chinese (Mandarin — Taiwan)。

  • 加入 Show Result。
  • Xcode 測試。

假設我們對不起英文老師，看不懂 preview。此時我們可選取 preview。

然後從右鍵選單選擇 Services > show chinese。

原來 preview 的意思是預覽呀。

將中文變成英文

  • 新增 shortcut translate to english。

類似剛剛的步驟，只是將翻譯的語言改成英文， Show Result 改成 Stop and Output。

  • Xcode 測試。

我們想在 Baby 裡宣告屬性血型，但是不知道血型怎麼拼。(再次對不起英文老師)此時我們可先寫中文，然後選取血型。

從右鍵選單選擇 Services > translate to english。

Cool，血型自動變成 Blood type 了。

struct Baby {
   
    let Blood type: String
}

然後我們再手動將它改成 bloodType 即可。

struct Baby {
   
    let bloodType: String
}
