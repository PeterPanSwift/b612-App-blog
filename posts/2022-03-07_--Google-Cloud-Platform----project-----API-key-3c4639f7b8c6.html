在 Google Cloud Platform 建立 project 和產生 API key

登入 google 帳號，連到 Google Cloud Platform

https://console.cloud.google.com

勾選 agree，點選 AGREE AND CONTINUE

點選 CREATE PROJECT

輸入專案名稱，點選 CREATE

建立專案後，若發現畫面一直停在轉轉轉的狀態，可直接點選網頁的重新整理。

專案成功建立後的畫面如下。

點選 API & Services

點選 Credentials

點選 CREATE CREDENTIALS，然後點選 API Key

產生 API Key，點選右邊的 copy 按鈕可將它複製。
