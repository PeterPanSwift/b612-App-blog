控制 SKScene 顯示時如何縮放的 scaleMode

使用 SpriteKit 開發 iOS 遊戲時，我們主要利用 SKScene 設計畫面，然後利用 SKView 將 SKScene 顯示出來。當 SKView 顯示 SKScene 時會將 scene 進行縮放，縮放的方式則跟 SKScene 的 scaleMode 有關。接下來我們將一一介紹各種 scaleMode 的縮放效果。

建立 SpriteKit 的 Game App 專案，設計 GameScene.sks 的遊戲畫面

scene 的大小設為 830 * 830。

加入顯示圖片的 SKSpriteNode，大小一樣設為 830 * 830，佔滿整個 scene。

在 GameScene 的 didMove 裡印出 scene 在螢幕可視範圍的寬高

剛剛我們將 scene 的大小設為 830 * 830，不過螢幕上顯示的是縮放後的 scene，在螢幕的可視範圍(也就是 view)不一定能完整顯示 830 * 830 的 scene，因此我們利用 function convertPoint ＆ view 的左上和右下計算可視範圍內 scene 的寬高。

class GameScene: SKScene {
    
    private var label : SKLabelNode?
    private var spinnyNode : SKShapeNode?
    
    override func didMove(to view: SKView) {
        let topLeft = convertPoint(fromView: .zero)
        let bottomRight = convertPoint(fromView: CGPoint(x: view.bounds.width, y: view.bounds.height))
        
        print("width", bottomRight.x - topLeft.x)
        print("height", topLeft.y - bottomRight.y)

aspectFill

scaleMode 設為 aspectFill。

class GameViewController: UIViewController {
   override func viewDidLoad() {
        super.viewDidLoad()
        
        if let view = self.view as! SKView? {
            if let scene = SKScene(fileNamed: "GameScene") {
                scene.scaleMode = .aspectFill
                view.presentScene(scene)
            }
        }
    }

結果

印出

width 383.53082275390625
height 830.0

scene 將維持比例縮放並佔滿 view，因此它會有部分區塊超出 view。在 iPhone 13 顯示時，830 * 830 的 scene 在螢幕裡只顯示了 383.53 * 830，左右各超出了 (830–383.53)/2。由於 iPhone 13 是 390 * 844 point，我們可算出它維持比例縮放並佔滿 view 時，螢幕裡顯示的寬度為 383.53(390 / 844.0 * 830)。

aspectFit

scaleMode 設為 aspectFit。

scene.scaleMode = .aspectFit

結果

印出

width 830.0
height 1796.205078125

scene 將維持比例縮放並完整顯示在 view 裡。在 iPhone 13 顯示時，830 * 830 的 scene 在螢幕裡可完整顯示，然後上下留白。由於 iPhone 13 是 390 * 844 point，我們可算出它維持比例並完整顯示在 view 時，螢幕裡顯示的高度為 1796.2( 844 / 390.0 * 830)。

fill

scaleMode 設為 fill。

scene.scaleMode = .fill

結果

印出

width 829.9999389648438
height 829.9999389648438

scene 將變形填滿整個 view，不管在哪個尺寸的 iPhone，view 裡都可看到 scene 的完整內容。

resizeFill

scaleMode 設為 resizeFill。

scene.scaleMode = .resizeFill

結果

印出

width 390.0
height 843.9999389648438

resizeFill 將讓 scene 的尺寸跟 view 一樣，因此在 iPhone 13 時 scene 的寬高會變成 390 * 844。
