#242 使用 AI 創作人生第一個 App(Vibe coding)

彼得潘在教 SwiftUI、UIKit、Flutter 程式時，會出很多很多的作業。不過第一堂課的作業往往是最難想的，因為剛上完第一堂課的同學會的東西還不多，能做的東西有限。

然而隨著 AI 愈來愈聰明，不懂變數和 function 的程式麻瓜也能透過 AI 開發一些簡單的 App。因此彼得潘決定之後程式課的第一個作業是【使用 AI 創作人生第一個 App】!

  • 顯示另外加入的圖片和網路的圖片。(ps: 也可以利用 AI 生圖)

建立 App 專案

使用 Xcode 建立 SwiftUI / UIKit(storyboard) 專案，或用 VS Code 建立 Flutter 專案。

使用 VS Code 打開專案資料夾，用 Agent mode 修改程式創作 App

點選 VS Code menu 的 Open Folder 打開專案資料夾。

打開要修改的檔案，建議 SwiftUI 打開 ContentView.swift，Flutter 打開 main.dart，UIKit(storyboard) 打開 ViewController.swift。

點選上方的 Copilot 按鈕打開 Copilot 視窗。

從右下的按鈕切換 Agent / Ask / Edit，選擇可直接修改專案程式的 Agent 或 Edit 模式。建議使用最厲害的 Agent 模式，AI 可自動執行指令和搜索研究專案的檔案。
