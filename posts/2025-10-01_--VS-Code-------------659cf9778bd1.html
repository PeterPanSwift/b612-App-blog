從 VS Code 啟動本機網站的各種方法

彼得潘常常從 VS Code 利用 AI 開發各種方便的網站，這些網站大部分只是彼得潘自己需要的小工具，因此只需跑在本機，不需要部署到網路上。

彼得潘研究一下後，整理了幾種在本機啟動網站的方法。以下彼得潘以 AI 開發的個人 Medium 文章搜尋網站示範。

利用 AI 在本機建立個人的 Medium 文章搜尋網站
彼得潘利用 Medium 撰寫許多的文章和筆記，因此滿希望能對全文進行比對，找出包含某某關鍵字的文章清單。medium.com

從 VS Code 打開網站的資料夾，網頁為 little_prince_posts.html。

啟動網站的方法有以下三種。

  • 使用 Python 啟動。
  • 請 AI 啟動。
  • 使用 VS Code 的外掛 Live Server。

使用 Python 啟動

在 Terminal 輸入以下指令。

python3 -m http.server

網站的網址如下。

http://localhost:8000/little_prince_posts.html

請 AI 啟動

輸入以下 prompt。

啟動網站

AI 的回答。

網站已經用 python3 -m http.server 在 http://localhost:8000/little_prince_posts.html 服務中（PID 77020）。想看狀態可 tail -f /tmp/http_server.log，要關掉時 kill 77020 即可。

網站的網址如下。

http://localhost:8000/little_prince_posts.html

使用 VS Code 的外掛 Live Server

從 Extension 頁面安裝 Live Server。

點選要顯示的網頁 little_prince_posts.html。

點選右下的 Go Live 啟動網站。

此時將自動打開瀏覽器顯示網頁，網址如下。

http://127.0.0.1:5500/little_prince_posts.html
