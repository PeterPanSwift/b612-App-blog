利用 JSON Editor Online 排版美美的 JSON 資料
開發 iOS App 時，我們時常串接 API 抓取 JSON 格式的資料。不過 JSON 卻讓我們又愛又恨，它豐富了我們的 App 內容，但要看懂 JSON 卻不是件易事。medium.com

從下圖展開的 JSON 樹狀圖，我們可以看出資料的型別，比方 results 是 array，kind 是字串，artistId 是數字。

看出資料的型別其實有更簡單的方法，透過 Smart JSON Editor，它將直接顯示每個欄位的資料型別。

  • 從 Mac 的 App Store 下載 Smart JSON Editor。
‎Smart JSON Editor
‎Powerful JSON Tree tool for app and web developers. Create testing JSON trees, use value transformers to generate /…apps.apple.com
  • 啟動 App，點選 File > New 打開新視窗。
  • 在瀏覽器，Postman 或是其它方法抓取 JSON，然後按 cmd + c 複製 JSON 內容。
  • 貼上 JSON 的文字內容。

點選按鈕 + ，然後點選 Model from Clipboard。

  • 查看 JSON 資料型別。

此時不只 JSON 變成方便瀏覽的樹狀圖，每個欄位都清楚地標示出它的 type，顯示它是 Array，Object，String，Number，Bool 或是 Null。

  • 點選右上的 Text View 可切換到 JSON 的文字瀏覽模式。
