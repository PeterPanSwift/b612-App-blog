在 Mac 上觸發 VS Code / Project IDX trigger suggest 的快速鍵

在 VS Code / Project IDX 撰寫程式時，我們常常會用 trigger suggest 叫出下圖的自動完成選單。

不過在 Mac 上用快速鍵觸發 trigger suggest 卻會遇到一些問題。

trigger suggest 有多種快速鍵，但是很多都不能用，ctrl + space 跟 Mac 的輸入法切換衝突，cmd + I(大寫的 i) 跟 GitHub Copilot 的快速鍵衝突。

若想維持輸入法切換跟 GitHub Copilot 預設的快速鍵，觸發 trigger suggest 剩下以下 4 種方法。

  • 方法 1: 使用另一個快速鍵 option + esc。
  • 方法 2: 自訂快速鍵。
  • 方法 3: 從 command palette 點選 suggest。
  • 方法 4: 重新輸入參數。

方法 1: 使用另一個快速鍵 option + esc

缺點是比較不好按。

方法 2: 自訂快速鍵

從 command palette 點選 Open Keyboard Shortcuts。

搜尋 trigger sug，找到搭配ctrl + space 的 Trigger Support 後雙擊。

輸入新的快速鍵，在此彼得潘輸入 ctrl + .。

之後按 ctrl + . 即可叫出自動完成的選單。

方法 3: 從 command palette 點選 suggest

按 cmd + shift + p 叫出 command palette，然後點選 Trigger Suggest。

  • 方法 4: 重新輸入參數。

比方重新輸入 TextButton 的參數名字 onPressed。

點選 onPressed 輸入後，trigger suggest 將自動觸發，提示輸入 ( ) => 或 ( ) { } 。
