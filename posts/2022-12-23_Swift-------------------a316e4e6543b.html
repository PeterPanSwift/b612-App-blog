Swift 的中文字排序 — 筆畫，注音，拼音

在 Swift 裡我們可利用 locale 指定排序的方法，以下我們分別介紹筆畫，注音 & 拼音三種中文排序的寫法。

以下的 array names 儲存世界知名大學的名字，我們希望能以筆畫，注音 & 拼音排序。

var names = ["劍橋", "牛津", "麻省理工", "史丹福", "哈佛", "加州理工", "倫敦帝國", "倫敦大學", "蘇黎世理工", "芝加哥"]

注音排序

let locale = Locale(identifier: "zh@collation=zhuyin")
names.sort {
    $0.compare($1, locale: locale) == .orderedAscending
}

說明

let locale = Locale(identifier: "zh@collation=zhuyin")

zh 表示中文，collation=zhuyin 表示以注音排序。

names.sort {
    $0.compare($1, locale: locale) == .orderedAscending
}

orderedAscending 表示以由小到大的方式排序。

結果

["麻省理工", "牛津", "倫敦大學", "倫敦帝國", "哈佛", "加州理工", "劍橋", "芝加哥", "史丹福", "蘇黎世理工"]

筆畫排序

stroke 表示以筆畫排序。

let locale = Locale(identifier: "zh@collation=stroke")
names.sort {
    $0.compare($1, locale: locale) == .orderedAscending
}

結果

["牛津", "加州理工", "史丹福", "芝加哥", "哈佛", "倫敦大學", "倫敦帝國", "麻省理工", "劍橋", "蘇黎世理工"]

拼音排序

pinyin 表示以拼音排序。

let locale = Locale(identifier: "zh@collation=pinyin")
names.sort {
    $0.compare($1, locale: locale) == .orderedAscending
}

結果

["哈佛", "加州理工", "劍橋", "倫敦大學", "倫敦帝國", "麻省理工", "牛津", "史丹福", "蘇黎世理工", "芝加哥"]
