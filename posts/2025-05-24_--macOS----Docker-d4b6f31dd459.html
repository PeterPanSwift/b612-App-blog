在 macOS 安裝 Docker

連到 Docker 網站。

Home
Docker is a platform designed to help developers build, share, and run container applications. We handle the tedious…www.docker.com

點選 Download Docker Desktop 的 Download for Mac。

雙擊下載的 Docker.dmg，將 Docker 移到 Applications。

啟動 Docker App。

點選 Accept。

選擇 Use recommended settings，點選 Finish。

現在 Docker 已成功運行，上方的選單列會看到可愛的 Docker 鯨魚。

點擊 Docker 鯨魚可看到狀態顯示 Docker Desktop is running。

打開 Terminal，確認是否可執行 docker 指令，比方輸入 docker — version 檢查 docker 版本。

docker --version

結果。

Docker version 28.1.1, build 4eba377

之後若想要每次登入使用電腦時都自動啟動 Docker，可在 Docker Desktop 的設定頁面勾選 Start Docker Desktop when you sign in to your computer。
