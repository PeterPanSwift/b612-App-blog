無法安裝最新版的 Xcode ? 請先升級 macOS 或安裝舊版 Xcode

為了開發 iOS App，我們的第一步往往是刷卡買 Mac，然後下載安裝開發工具 Xcode。

不過有些朋友卻遇到問題，發現自己被排擠了，無法安裝最新版的 Xcode。此時我們有以下兩個解決的方法。

方法 1: 升級到最新的 macOS

Xcode 有嚴格的安裝需求，比方要安裝 Xcode 26，macOS 至少要 15.6。

Xcode 各版本的安裝需求，iOS SDK & Swift

  • 查詢 macOS 版本。

如果不清楚自己 macOS 版本，可點選螢幕左上角的蘋果圖案，然後點選 About This Mac，查詢 macOS 的版本。

  • 升級 macOS。

點選下圖的 System Settings。

搜尋 Software Update，若看到可更新的項目再點選 Update Now。

Update macOS on Mac
Before following these steps, it's a good idea to back up your Mac. From the Apple menu  in the corner of your screen…support.apple.com

升級 macOS beta 的相關說明可參考以下連結。

升級安裝 macOS beta
從 macOS 13.4 Ventura 開始，升級 beta 版的 macOS 變得更容易了，我們可以直接從 System Settings App 升級 beta 版的作業系統。medium.com

方法 2: 安裝舊版的 Xcode

有些朋友因為 mac 比較舊，無法升級到最新版的 macOS，因此只能委屈地安裝舊版的 Xcode。從 macOS 的 wiki 可查詢 mac 能安裝的 macOS， 比方下圖顯示 2019 之後的 Macbook Pro 才能安裝 macOS 26。

macOS - Wikipedia
macOS (; previously Mac OS X and later OS X) is a proprietary graphical operating system developed and marketed by…en.wikipedia.org
