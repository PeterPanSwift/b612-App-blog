predictive code completion 的強項

  • 專精 Swift 和 Apple SDKs。

Swift & Apple SDK 的相關程式都會出現聰明的提示，SwiftUI 和 UIKit 都難不倒它。

  • 在本機端運行，速度超快且不需連網。
  • Xcode 會自動更新 code completion 使用的機器學習 model，因此它能提示最新的 SDK 和 Swift 寫法。

使用 predictive code completion 的條件

執行 predictive code completion 需要一定的效能，因此 Mac 必須滿足以下條件。

  • Apple 晶片的 CPU。
  • macOS 15 以上。

心只有 Swift 的 predictive code completion

predictive code completion 全心全意專注在 Swift，沒心支援其它程式，彼得潘測試後發現它無法撰寫其它程式語言。

Xcode 會依據檔案判斷適合的自動完成，比方 BmiView.swift 會出現 calcutateBmi 的提示。

BmiViewController.m 卻沒有任何提示，因為它是 Objective-C。

下載 Predictive Code Completion Model

初次打開 Xcode 時會出現 Predictive Code Completion Model 的選項，勾選它即可安裝。

如果忘了勾選，可從 Xcode 的 Settings > Components 點選 Predictive Code Completion Model 的 Get 安裝。

啟用 Predictive Code Completion

下載後 Predictive Code Completion 會預設啟用，從 Xcode 的 Settings 可設定是否啟用。

從 Settings 的 Editing 分頁點選 Completion。

從 Display code predictions 設定。
