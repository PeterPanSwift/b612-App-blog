CoreGraphics 方便的 zero property

從程式製作 iOS App 畫面時，我們時常使用型別 CGRect，CGSize 和 CGPoint。在使用它們時，我們有時要生成內容都是 0 的資料，比方以下例子，分別傳入 4 個 0，將 x, y, width, height 設為 0。

let loadingView = UIView(frame: CGRect(x: 0, y: 0, width: 0, height: 0))

其實有更簡單的方法，CoreGraphics framework 早已幫 CGRect，CGSize ，CGPoint 和 CGVector 加入內容為 0 的 static property zero，我們在開發時只要輸入 .zero 即可。

CGRect

let loadingView = UIView(frame: .zero)

CoreGraphics framework 裡的宣告

extension CGRect {
   public static var zero: CGRect { get }
}

CGSize

loadingView.frame.size = .zero

CGPoint

loadingView.frame.origin = .zero

CGVector

CGVector 比較少用到，不過它也有 zero property。

ps: 其實 Int 也有 zero property，不喜歡寫 0 的話，也可以輸入 Int.zero。
