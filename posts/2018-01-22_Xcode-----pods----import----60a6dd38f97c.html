Xcode 殘留的 pods 套件 import 錯誤

有同學在順利加入 CocoaPods 套件，以 xcworkspace 打開專案，開心地 import 套件後，出現如圖顯示的紅色錯誤，說它找不到套件，

No such module 'SDWebImage'

別擔心，一切都是假象 ! 如果你執行 App，將發現 App 還是可以順利生成啟動，只是 import SDWebImage 旁邊那行紅色錯誤遲遲不肯消失。

想讓它消失很簡單。只要點選視窗左上角的紅色 X，將視窗關掉，然後再重新開啟 xcworkspace，再按 cmd + b build App，即可發現紅色錯誤消失無蹤。
