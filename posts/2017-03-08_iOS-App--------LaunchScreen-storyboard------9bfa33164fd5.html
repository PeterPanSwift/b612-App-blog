iOS App 問題集: 在 LaunchScreen.storyboard 設定圓角

在 storyboard 製作 App 畫面時，將方方正正的圖片變成可愛的圓角，甚至變成圓形，並不是太難的事情。只要在 Identify inspector 頁面的 User Defined Runtime Attributes 區塊新增 layer.cornerRadius，設定圓角的數字即可。

但是，當你在 LaunchScreen.storyboard 做這件事時，將產生可怕的紅色錯誤，讓 App 無法執行 !

紅色錯誤訊息: Launch screens may not use user defined runtime attributes.

解決的方法:

不要在 LaunchScreen.storyboard 設定 User Defined Runtime Attributes !

請記得 LaunchScreen.storyboard 只是開頭畫面，只適合做單純的畫面排版，不適合做太複雜的操作。像是設定 User Defined Runtime Attributes， 設定 segue，將 controller 的類別設成自訂類別這類動作，對它來說都太強人所難了 !
