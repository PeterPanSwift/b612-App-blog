Xcode playground 顯示大圖問題

當圖片太大時，從 Xcode playground 查看 image view 的內容時可能會有問題，比方以下例子的 burger.png。

import UIKit
let image = UIImage(named: "burger.png")
let imageView = UIImageView(image: image)

此時可修改 image view 的 frame 調整大小，即可讓它順利顯示。

imageView.frame = CGRect(x: 0, y: 0, width: 538, height: 358)
