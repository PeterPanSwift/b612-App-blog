iOS 的 UI 元件都繼承 View，因此都擁有 View 的欄位

iOS App 的世界裡， View 元件(類別為 UIView)是最基本的 UI 元件，其它的 UI 元件都繼承 View，因此它們皆得到 View 擁有的功能和欄位。

以 Image View 為例，從下圖的 Attributes inspector 頁面，我們看到除了 專屬於 Image View 的欄位外，它也擁有 View 的欄位，因此它也可以設定背景顏色(Background)。
