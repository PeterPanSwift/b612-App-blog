切換 Object Library 元件的顯示方式

Xcode 的 Object Library 住著幫助我們開發 App 的各式元件。在顯示這些元件時，我們可以點選右上角的按鈕切換顯示方式。

總共有兩種可以選擇:

1 Icon View
格子狀排列，只顯示元件的 Icon。

2 List View
列表，顯示元件的 Icon，名稱和簡介。
