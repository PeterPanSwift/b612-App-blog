VS Code / Firebase Studio 只在程式儲存時觸發 hot reload，然而我們希望在修改程式後自動觸發，因此我們要先打開修改程式後自動儲存的功能。

設定自動儲存時觸發 hot reload

在設定視窗輸入 hot reload，找到 Flutter Hot Reload On Save 區塊。預設的設定是 manual，因此只有手動儲存時會觸發。

選擇 allIfDirty，讓手動或自動儲存都觸發 hot reload。
