請 AI 用 mermaid 生成 SwiftUI 的 view hierarchy 和 Flutter 的 widget tree

mermaid 很適合用來生成程式相關的圖表，因此彼得潘想到也許可以請 AI 用 mermaid 生成 SwiftUI 的 view hierarchy 和 Flutter 的 widget tree。

以下我們試試請 AI 生成兩種版本的 view hierarchy 和 widget tree 。

  • 只有型別的 view hierarchy 和 widget tree。
  • 包含型別和圖文名的 view hierarchy 和 widget tree。
  • 自訂生成兩種 view hierarchy / widget tree 的 GPT — view hierarchy / widget tree creator。

只有型別的 view hierarchy 和 widget tree

請用 mermaid 生成 SwiftUI 的 view hierarchy 或 Flutter 的 widget tree，框框裡只要寫 view / widget 的型別名字。

以下是參考的範例，請依範例的規則產生 mermaid。

1. SwiftUI 程式

import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "swift")
                .foregroundStyle(.orange)
            AppleView()
        }
    }
}

struct AppleView: View {
    var body: some View {
        HStack {
            Image(systemName: "apple.logo")
            Text("Apple")
                .font(.largeTitle)
        }
    }
}


產生的 mermaid。

    A[ContentView]--> B[VStack]
    B --> C[Image]
    B --> D[AppleView]
    D --> E[HStack]
    E --> F[Image]
    E --> G[Text]

請用 mermaid 生成 SwiftUI 的 view hierarchy 或 Flutter 的 widget tree，框框裡只要寫 view / widget 的型別名字。當參數是字串時，請寫出字串。

以下是參考的範例，請依範例的規則產生 mermaid。

    var body: some View {
        VStack {

}

struct AppleView: View {
    var body: some View {
        HStack {
            Image(systemName: "apple.logo")
