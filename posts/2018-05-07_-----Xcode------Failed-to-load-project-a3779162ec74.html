打不開的 Xcode 專案 — Failed to load project

GitHub 上有許多精彩的 Swift iOS App 專案，方便我們學習參考，但是有時候我們將專案下載後，開心地雙擊藍色專案檔時，卻發現專案無法開啟，出現以上的錯誤訊息，Failed to load project ... incompatible project version。

這是因為下載的專案採用的 Xcode 版本比我們自己的 Xcode 版本還新的關係，只要你將 Xcode 升級，確保你的 Xcode 版本不會比專案的 Xcode 版本舊，即可像雷神索爾拿起雷神之槌一樣，順利地開啟專案。
