利用 border，stroke & strokeBorder 實現 SwiftUI 邊框

以 ZAYIN 無法抗拒的藍色眼睛為例，認識使用 SwiftUI 設定元件的邊框。

利用 border，stroke & strokeBorder 實現 SwiftUI 邊框

以 ZAYIN 無法抗拒的藍色眼睛 & 偶像劇想見你為例，認識 SwiftUI 設定邊框的 border & stroke。

利用 border 設定邊框顏色跟寬度

struct ContentView: View {
    var body: some View {
        Text("無法抗拒 妳藍色的眼睛 ◉◉")
            .foregroundStyle(.blue)
            .border(.blue, width: 5)
    }
}
border(_:width:) | Apple Developer Documentation
Adds a border to this view with the specified style and width.developer.apple.com

如下圖所示，邊框預設將直接加在元件的框框裡，因此會檔到部分的文字。

利用 padding 增加元件上下左右的邊距

我們可以利用 padding 增加元件上下左右的邊距，然後再設定 border，如此 border 即可跟文字保持安全距離，不再檔到文字。

struct ContentView: View {
    var body: some View {
        Text("無法抗拒 妳藍色的眼睛 ◉◉")
            .foregroundStyle(.blue)
            .padding()
            .border(.blue, width: 5)
    }
}
padding(_:_:) | Apple Developer Documentation
Adds an equal padding amount to specific edges of this view.developer.apple.com

利用 overlay(或 background)搭配 stroke (或 strokeBorder)設定圓角邊框

  • 利用 overlay & stroke。
struct ContentView: View {
    var body: some View {
        Text("無法抗拒 妳藍色的眼睛 ◉◉")
            .foregroundStyle(.blue)
            .padding()
            .overlay {
                RoundedRectangle(cornerRadius: 20)
                    .stroke(.blue, lineWidth: 5)
            }
    }
}

除了 stroke，strokeBorder 也可以設定形狀的邊框。

struct ContentView: View {
    var body: some View {
        VStack {
            Text("無法抗拒 妳藍色的眼睛 ◉◉")
                .foregroundStyle(.blue)
                .padding()
                .overlay {
                    RoundedRectangle(cornerRadius: 20).stroke(.yellow, lineWidth: 10)
                }
            Text("無法抗拒 妳藍色的眼睛 ◉◉")
                .foregroundStyle(.blue)
                .padding()
                .overlay {
                    RoundedRectangle(cornerRadius: 20).strokeBorder(.yellow, lineWidth: 10)
                }
        }
    }
}

從下圖可看出 stroke & strokeBorder 的差別。使用 stroke 畫出的邊框將超出原本 Text 的長方形框框，而 strokeBorder 畫出的邊框則在 Text 的長方形框框內。

  • 利用 background & stroke 也可以。
struct ContentView: View {
    var body: some View {
        Text("無法抗拒 妳藍色的眼睛 ◉◉")
            .foregroundStyle(.blue)
            .padding()
            .background {
                RoundedRectangle(cornerRadius: 20)
                    .stroke(.blue, lineWidth: 5)
            }
    }
}
  • 利用 background(_:in:fillStyle:)。

圓角邊框加背景顏色

struct ContentView: View {
    var body: some View {
        Text("無法抗拒 妳藍色的眼睛 ◉◉")
            .foregroundStyle(.blue)
            .padding()
            .background(.yellow)
            .clipShape(.rect(cornerRadius: 20))
            .overlay {
                RoundedRectangle(cornerRadius: 20)
                    .stroke(.blue, lineWidth: 5)
            }
    }
}

我們還要另外呼叫 cornerRadius 設定背景顏色的圓角，否則將如以下例子，text 元件的背景色填滿了整個長方形。

圓角邊框加陰影

struct ContentView: View {
    var body: some View {
        Text("無法抗拒 妳藍色的眼睛 ◉◉")
            .foregroundStyle(.blue)
            .padding()
            .background(.yellow)
            .clipShape(.rect(cornerRadius: 20))
            .overlay {
                RoundedRectangle(cornerRadius: 20)
                    .stroke(.blue, lineWidth: 5)
            }
            .shadow(radius: 30)
    }
}

搭配邊框的 Button

點選 button 時若想要邊框一起變色，邊框 border 必須設定在 Button 裡的元件。

Button(action: {}) {
   Text("Border Button")
      .foregroundStyle(.blue)
      .padding()
      .border(.blue, width: 5)
}

若是直接從 button 呼叫 border，點選 button 時 border 將不會變色。

Button(action: {}) {
   Text("Border Button")
      .foregroundStyle(.blue)
      .padding()
}
.border(.blue, width: 5)
