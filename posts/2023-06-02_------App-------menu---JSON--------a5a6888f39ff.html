開發一個完整的訂飲料 App 需要許多技術，比方頁面的製作跟串接 RESTful API。不過初期的準備工作也滿花時間的，我們需要先收集整理飲料的 menu 資料跟照片。

從前沒有 AI 時，很多同學都是自己辛苦地一筆筆建立資料。最近厲害的 GPT 4 出現後，我們可以快速地產生訂飲料 App 需要的素材。

以下彼得潘以製作好喝的迷客夏 App 為例說明。

將 JSON 存到 GitHub

參考以下連結的方法將 JSON 放到 GitHub，以下我們在 GitHub 的 repository JSON_API 下新增 Milksha.json。

#176 使用 GitHub 存放方便 App 讀取的 JSON & 圖片 — 以UNIQLO App 為例
開發 iOS App 時，我們時常需要連線到後台抓取 JSON 資料。然而如果資料都由後台產生，不是由使用者生成，其實可以將 JSON 資料和圖片存在免費的 GitHub，然後再從 iOS App 連線讀取。medium.com

JSON 檔的網址如下，之後 App 透過它即可呈現迷客夏的 menu。

https://raw.githubusercontent.com/PeterPanSwift/JSON_API/main/Milksha.json

將飲料圖片存到 GitHub

從迷客夏網站下載飲料圖片。

飲品介紹 | 迷客夏Milksha
Edit descriptionwww.milksha.com

將圖片存到 GitHub

將飲料圖片存到 GitHub repository JSON_API 下。為了方便程式之後抓圖，記得讓圖片名字等於 JSON 裡的飲料名字。

比方珍珠紅茶拿鐵的圖片網址如下。

https://raw.githubusercontent.com/PeterPanSwift/JSON_API/main/珍珠紅茶拿鐵.png

每個飲料的前面網址都是固定的 https://raw.githubusercontent.com/PeterPanSwift/JSON_API/main/，我們只要加上飲料名字.png 即可抓取飲料圖片。
