Hot Reload & Hot Restart

以下程式 hot reload 會有問題，另外定義 StatelessWidget 或 StatefulWidget ，hot reload 才會有效果。

void main() {
  runApp(
    MaterialApp(
      home: Scaffold(
        backgroundColor: Colors.blueGrey,
        appBar: AppBar(
          title: Text('I AM Rich'),
          backgroundColor: Colors.blueGrey[900],
        ),
        body: Center(
          child: Image(
            image: AssetImage('images/diamond.png'),
          ),
        ),
      ),
    ),
  );
}

自動生成 StatelessWidget

hot reload 會維持 state，hot restart 會重置 state 。例子: counter 的數字。

Container widget

Container 沒有設定 child

Containers with no children try to be as big as possible unless the incoming constraints are unbounded, in which case they try to be as small as possible. 

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        backgroundColor: Colors.red,
        body: Container(
          color: Colors.white,
        ),
      ),
    );
  }
}

Containers 有 child 時，大小會是 child 的大小。

Containers with children size themselves to their children

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        backgroundColor: Colors.red,
        body: Container(

          child: Text('Hello'),

      ),
    );
  }
}

margin: 設定 container 和其它元件的間距

class MyApp extends StatelessWidget {

      ),
    );
  }

EdgeInsets 的各種選項。

margin 設定 container 和畫面的左邊界間距 20。

點選 Flutter Inspector 顯示 guideline 的 button，顯示出 margin 的線條。

padding: 設定 container 內容和 container 邊界的間距。

margin is for the outside of your widget, and padding is for the inside of your widget。

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        backgroundColor: Colors.red,
        body: SafeArea(
          child: Container(
            height: 100,
            width: 100,
            margin: EdgeInsets.all(20),
            padding: EdgeInsets.all(20),
            color: Colors.white,
            child: Text('Hello'),
          ),
        ),
      ),
    );
  }
}

padding 設定文字和 container 的間距 20。
