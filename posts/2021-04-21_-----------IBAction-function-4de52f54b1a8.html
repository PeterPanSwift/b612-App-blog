將多個元件連到同一個 IBAction function

當多個 button 有類似的功能時，我們通常會將它們連到同一個 IBAction function，比方以下的 App Guess The Logo — Logo Quiz，畫面裡有顯示 4 個 logo 的 button，玩家必須點選 button 猜哪一個是麥當勞。由於四個 button 的功能類似，所以我們可以將它們連到同一個 IBAction function。

將多個元件連到同一個 IBAction function 的方法如下:

從第一個元件連到 assistant editor 拉 IBAction

@IBAction func selectLogo(_ sender: UIButton) {
    
}

可以從左邊拉到右邊，也可以從右邊拉到左邊。

方法1: 從 IBAction 的圓圈連到其它 button(右到左）

方法2: 從 storyboard 的元件連到 IBAction(左到右）

檢視連線狀態

當 selectLogo 連到 4 個 button 後，若我們將滑鼠移到 IBAction 左邊的圓圈，將看到它連到的 4 個 button 都被選取。

我們也可以點選 view controller 後，從它的 Connections inspector 頁面看到 selectLogo 連到了 4 個 button。
