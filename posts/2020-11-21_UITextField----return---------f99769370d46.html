不過只拉 IBAction 還不夠，這時候只是 return 鍵按下時會呼叫 dismissKeyboard，我們得在裡面加上收鍵盤的程式，view.endEditing(true)。

@IBAction func dismissKeyboard(_ sender: Any) {
        view.endEditing(true)
}
