完成申請後，MovieGlu 將馬上寄出 email，告訴我們機密的 API key。

MovieGlu 提供兩種 API，正式跟測試的 API。如下圖所示，正式 API 的 territory 是 UK，只能發送 75 個 request，發送更多的 request 需要額外付費。測試 API 的 territory 是 XX，一個月可發送 1000 個 request。因此若只是練習，可優先使用測試的 API。

串接 API 需設定的 HTTP header

串接 API 時，HTTP header 需包含 client，x-api-key，authorization，territory，api-version，geolocation & device-datetime。
