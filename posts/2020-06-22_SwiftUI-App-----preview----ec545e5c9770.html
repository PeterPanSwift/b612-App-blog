SwiftUI App 改名的 preview 問題

更新: 此問題在新版 Xcode 已修正。

有朋友從 Display Name 修改了 App 的名字，因為想做個送給女朋友的 App，所以將 App 的名字取名為只想愛你。

沒想到改名這點小事竟讓 SwiftUI 的 preview 功能出問題，錯誤顯示 Compiling failed: no such module 或 Linking failed。

解法 1

點選 Xcode menu 的 Quit Xcode 關掉 Xcode。

重新啟動 Xcode，打開專案，然後點選 Preview 的 Resume 即可看到預覽畫面正常顯示。

解法 2

如果剛剛的解法 1 依然有問題，請不要從 Display Name 設定名字，改從 App 的 Info 頁面新增 Bundle display name，它的 Value 即為 App 的名字。

ps: 新增的方法是在 Info 的 table 裡點選 +。
