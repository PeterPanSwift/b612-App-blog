利用 stack view 的 Layout Margins 客製間距

以上的畫面十分單純，只不過是 image view 加上 label，但當我們把它們包在 vertical 的 stack view 裡，將遇到一些問題。

如下圖所示，不管我們如何調整 Alignment，都無法達成 label 彼得潘和圖片左邊界間距 10 的效果。

alignment 只能做到完全對齊， 因此當我們把它調為 Leading 時，它將讓 label 和 image view 的左邊界完全對齊。

不然我們試著另外在 label 身上，加上間距 10 的條件好了。不加還好，加了將產生可怕的紅色錯誤。因為它會和原來 alignment leading 的條件衝突， alignment leading 其實背後會幫我們另外加上 label 和 image view 的左邊界對齊的條件，所以不可能同時對齊，又同時間距 10。

解法

剛剛的問題，可利用 stack view 的 Layout Margins 解決，步驟如下:

1 將 label 加在 stack view 裡

2 點選包著 label 的 stack view 後，切換到 Size inspector 分頁。

3 設定 stack view 的 Layout Margins。

將 Layout Margins 設為 Fixed 後，即可控制 stack view 上下左右的 margin。

你可以把 margin 想成留白的區塊，stack view 裡的元件將從留白區塊之後開始擺放。當我們設定 Left margin 為 10 時，stack view 裡 label 的左邊將多出 10 points 的空間，實現我們想要的間距 10 效果。
