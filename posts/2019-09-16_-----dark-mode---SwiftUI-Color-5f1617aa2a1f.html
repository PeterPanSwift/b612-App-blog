採用 Color 內建支援 dark mode 的顏色

SwiftUI 的顏色型別是 Color，它貼心地提供許多在 light & dark mode 自動變色的顏色，比方下圖顯示的顏色清單。

在清單裡我們看到許多熟悉的顏色，比方 red，不過它並非百分百的紅色，而是 Apple 精心調配，在 light & dark mode 有著不同 RGB 的紅色。

Color | Apple Developer Documentation
Judicious use of color can enhance communication, evoke your brand, provide visual continuity, communicate status and…developer.apple.com

Text 預設採用 primary color，它的 primary color 在 light mode 是黑色，在 dark mode 則是白色，因此剛剛的文字才能在 dark mode 變成白色。

Text("你說藍色是你最愛的顏色")
    .foregroundStyle(.primary)

primary color 是主要顏色的意思，就像女朋友一樣，只有一個選擇是不夠的(誤)。因此 Color 還提供了第二選擇，secondary color，讓我們將文字改成 .secondary 試試。

Text("你說藍色是你最愛的顏色")
   .foregroundStyle(.secondary)

文字變成了灰色。

既然周興哲怎麼了的歌詞提到藍色，接著我們再試試 .blue 的效果吧 。

Text("你說藍色是你最愛的顏色")
   .foregroundStyle(.blue)

自訂 dark mode 的顏色和圖片

我們也可以在 Assets.xcassets 裡新增顏色和圖片，自訂 light & dark mode 有著不同的顏色和圖片，相關操作可參考以下連結。

Xcode 11 Asset 有趣的新功能
Xcode 11 Asset 多了不少有趣的新功能，比方搜尋和改名，多國語言的支援， copy paste，以及設定 dark mode 的圖片 & color(dark mode 在 Xcode 10 就可以設定，不過 iOS 13…medium.com

比方我們在 Assets.xcassets 新增了顏色你愛的藍，設定它在 light mode 是深藍色，dark mode 是淡藍色。

在生成 Color 時將顏色的名字當參數傳入。

  • 寫法 1
Text("你說藍色是你最愛的顏色")
   .foregroundStyle(.你愛的藍)
  • 寫法 2
Text("你說藍色是你最愛的顏色")
   .foregroundStyle(Color("你愛的藍"))
