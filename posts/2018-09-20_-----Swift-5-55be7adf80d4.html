搶先體驗 Swift 5

隨著 Xcode 10 的推出，現在我們可以使用 Swift 4.2 的新功能。不過其實我們已經可以開始體驗將在 2019 推出的 Swift 5 的部分功能 ，因為 Swift 的官網會定期更新，放上開發中的 Swift 5 讓我們下載，接下來就讓我們用 Xcode 10 搶先體驗 Swift 5 吧。

apple/swift-evolution
This maintains proposals for changes and user-visible enhancements to the Swift Programming Language. …github.com

1 連到 Swift 的下載網頁，點選 Snapshots Trunk Development (master) 下的 Xcode。

Trunk Development (master) 會定期更新，放上最新版本的 Swift。

Swift.org
Swift is a general-purpose programming language built using a modern approach to safety, performance, and software…swift.org

2 點選 swift 的 pkg 檔安裝。

3 從 Xcode 的 menu Xcode > Toolchains 切換 swift 版本。

點選 Swift Development Snapshot 2018–09–19 (a) 切換為 Swift 5。

4 實驗 Swift 5 的新功能。

Swift 5 SE-0225 的 Adding isMultiple to BinaryInteger 增加 function isMultiple(of)， 幫助我們判斷數字 A 是否是數字 B 的倍數。

apple/swift-evolution
This maintains proposals for changes and user-visible enhancements to the Swift Programming Language. …github.com

比方想判斷數字是否是偶數，只要呼叫 isMultiple(of: 2) ，回傳 true 時即為偶數，因為 2 的倍數就是偶數。

let number = 7
if number.isMultiple(of: 2) {
   print("Even")
} else {
   print("Odd")
}

新的寫法比以前用 % 2 判斷餘數為 0 的寫法直覺多了。

let number = 7
if number % 2 == 0 {
   print("Even")
} else {
   print("Odd")
}
