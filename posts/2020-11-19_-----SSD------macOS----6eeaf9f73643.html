利用外接 SSD 硬碟安裝 macOS 開機

為了開發 iOS App，我們時常需要安裝 beta 版的 Xcode & macOS，造成 mac 不太穩定，而且開發 App 時還會產生佔據大量空間的檔案，讓我們的硬碟空間不夠用。

有沒有什麼好方法呢? 有的，我們可以將 macOS 安裝在大容量的外接硬碟，然後透過外接硬碟開機。如此即可享有以下好處：

  • 在外接硬碟做各種有趣的實驗，就算做壞了還可以用 mac 內建硬碟開機。
  • 享受外接硬碟的大容量。

為了達到不錯的效能，彼得潘接下來將採用三星搭配 SSD & Thunderbolt 3 的 X5 外接硬碟，然後在上面安裝 macOS Big Sur。

接下來就讓我們鼓起勇氣，從最困難的第一步刷卡買硬碟開始吧。

購買外接硬碟

彼得潘前往附近的光華商場購買 X5。雖然 1T 很美好，不過由於預算有限，彼得潘還是先買 500 G 就好。

將外接硬碟接到 Mac

打開 Disk Utility

硬碟必須先格式化才能安裝 macOS，所以請先打開格式化高手 Disk Utility。

選擇 Show All Devices

從左邊的選單選擇外接硬碟，然後點選上方的 Erase(清除）

設定硬碟的名稱和格式

Scheme 選擇 GUID Partition Map，Format 選擇 APFS，然後點選 Erase。

成功格式化硬碟後，接下來可以開始安裝 macOS 了。

下載 & 安裝 macOS Big Sur

下載 macOS Big Sur 後，然後點選 Install macOS Big Sur 進行安裝。

選擇安裝的硬碟

選擇剛剛熱騰騰建立的 Samsung X5。

點選 Restart，開始安裝 macOS

經過一段時間的安裝和等待，我們終於看到心上人了 ~

macOS 11.0 Big Sur 現身!

One more thing，下次開機會是採用內建還是外接硬碟呢 ？

  • 沒有接外接硬碟時，mac 將選擇內建硬碟的 macOS 開機 。
  • 接外接硬碟時，mac 將選擇外接硬碟的 macOS 開機 。

另外我們也可從 System Preferences 的 Startup Disk 選擇開機的硬碟。

關於開機硬碟的設定，也可參考以下 Apple 官方的說明。
