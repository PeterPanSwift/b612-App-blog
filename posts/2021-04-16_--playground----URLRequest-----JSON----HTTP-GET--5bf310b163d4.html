從 playground 預覽 URL & URLRequest(GET) 取得的 JSON 資料

最近發現 playground 有個滿貼心的功能，當我們產生 URL 或 URLRequest(GET) 時，利用 playground 的 Quick Look 或 Show Result 就可看到取得的資料，比方看到 API 回傳的 JSON 。

不過此功能也有限制，只有滿足以下條件有方便的預覽功能。

  • HTTP method 是 GET。
  • 不需要另外設定 HTTP header。
