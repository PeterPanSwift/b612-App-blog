打造純文字介面的 SwiftTUI — 在 Linux 上也能練習 SwiftUI 了

最近發現有個模仿 SwiftUI 的第三方套件 SwiftTUI，透過它我們可以使用熟悉的 SwiftUI 寫法打造純文字介面的 terminal App。雖然它的畫面和功能比 SwiftUI 陽春許多，但是它有個特別的優點，沒有 Mac 和 iPad 的朋友在 Linux 上也能用 SwiftUI 寫法製作 App 了 !

GitHub - rensbreur/SwiftTUI: SwiftUI for terminal applications

我們可以在 macOS 和 Linux 上使用 SwiftTUI 打造 terminal App，以下彼得潘示範從 Xcode 建立 macOS 的 Command Line Tool，用 SwiftTUI 開發幾個有趣的 App。

  • Hello Neverland。
  • 剪刀石頭布(RPS)。
  • 猜數字。

建立 macOS 的 Command Line Tool 專案

加入 SwiftTUI package

https://github.com/rensbreur/SwiftTUI

Hello Neverland

import Foundation
import SwiftTUI

struct ContentView: View {
    var body: some View {
        Text("Hello Neverland")
            .padding()
            .background(.yellow)
            .border(.blue)
    }
}

Application(rootView: ContentView()).start()

定義 ContentView，使用 Text 和一些常用的 modifier，然後再用 Application(rootView: ContentView()).start() 設定程式啟動後顯示 ContentView。

不過要記得，SwiftTUI 只是弱化版的 SwiftUI，目前可用的功能可參考以下說明。

剛剛寫的程式必須從 Terminal App 啟動，若是直接按三角形執行，將出現以下錯誤。

正確執行的方法如下。

點選 Xcode menu 的 Product > Show Build Folder in Finder。

Finder 會顯示 App 的 Build 資料夾，從 Build > Products > Debug 下找到 App 的執行檔。

雙擊執行檔，Terminal 成功啟動顯示 Hello Neverland。

剪刀石頭布

import Foundation
import SwiftTUI

enum Sign: String, CaseIterable {
    case rock
    case paper
    case scissors
}

// 感謝 AI 提供莎士比亞風格的英文
enum GameResult: String {
    case win = "Forsooth, triumph doth belong to us!"
    case lose = "Alas, we are vanquished, but we shall return with vigor!"
    case draw = "A noble contest of equals, with neither side yielding!"
}

struct ContentView: View {
    @State private var computerSign: Sign?
    @State private var userSign: Sign?
    @State private var result = ""
    
    var body: some View {
        VStack {
            Text("computer \(computerSign?.rawValue ?? "?")")
            Text("player \(userSign?.rawValue ?? "?")")
            HStack {
                ForEach(Sign.allCases, id: \.self) { sign in
                    Button(sign.rawValue) {
                        userSign = sign
                        play()
                    }
                    .padding()
                    .border(.magenta)
                }
            }
            Text(result)
                .background(.yellow)
        }
    }
    
    func play() {
        guard let userSign else { return }
        computerSign = Sign.allCases.randomElement()
        if userSign == computerSign {
            result = GameResult.draw.rawValue
        } else {
            switch userSign {
            case .rock:
                if computerSign == .scissors {
                    result = GameResult.win.rawValue
                    return
                }
            case .paper:
                if computerSign == .rock {
                    result = GameResult.win.rawValue
                    return
                }
            case .scissors:
                if computerSign == .paper {
                    result = GameResult.win.rawValue
                    return
                }
            }
            result = GameResult.lose.rawValue
        }
    }
}

Application(rootView: ContentView()).start()

使用方向鍵選擇剪刀石頭布，按 enter 點擊 button。

值得注意的，顯示中文目前有一些排版問題，因此建議儘量顯示英文。

猜數字

import Foundation
import SwiftTUI

struct ContentView: View {
    @State private var guessNumber: Int?
    @State private var result = ""
    var answer = Int.random(in: 0...100)
    
    var body: some View {
        VStack(alignment: .center, spacing: 1) {
            Text("Guess how many roses fox will give you")
            TextField { content in
                if let number = Int(content) {
                    guessNumber = number
                }
            }
            Button("Guess") {
                if let guessNumber {
                    if guessNumber == answer {
                        result = "Correct!"
                    } else if answer > guessNumber {
                        result = "The fox loves you, guess higher!"
                    } else {
                        result = "The fox doesn't love you enough, guess lower!"
                    }
                }
            }
            .background(.yellow)
            
            Text(result)
        }
    }
}

Application(rootView: ContentView()).start()

VStack(alignment: .center, spacing: 1) 造成畫面排版時置中並且元件間有 1 行的間隔。TextField 的下方有 ＿，輸入後按 enter 才會觸發 TextField 的 closure 程式，將輸入的數字存入 guessNumber。
