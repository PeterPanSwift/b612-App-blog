利用 WKWebView 播放 YouTube 影片 & 自動播放

以有趣的電影預告【每天回家老婆都在裝死】為例。

影片的網址為

https://www.youtube.com/watch?v=JsX2lsLxLBI

v= 之後為影片的 id。為了讓顯示網頁的 WKWebView 只顯示影片，

請將影片 id 加到 https://www.youtube.com/embed/，組合出以下網址。

https://www.youtube.com/embed/JsX2lsLxLBI

播放 YouTube 影片的程式碼。

import UIKit
import WebKit

class ViewController: UIViewController {
    
    @IBOutlet weak var webView: WKWebView!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        let url = URL(string: "https://www.youtube.com/embed/JsX2lsLxLBI")!
        let request = URLRequest(url: url)
        webView.load(request)
    }
}

web view 在播放影片前顯示的畫面。

點選 web view 後將以全螢幕播放影片。

若想自動開始播放，可在網址的最後加上 ?autoplay=1，例如以下例子。

let url = URL(string: "https://www.youtube.com/embed/JsX2lsLxLBI?autoplay=1")!
