將網站變成獨立的 web app — macOS 14 Sonoma 新功能

從 macOS Sonoma 開始，我們可以將網站變成獨立的 web app，方便之後隨時瀏覽和切換 app。

以下彼得潘以最近正在研讀的 Swift Concurrency by Example 為例，將它變成有著 Swift icon 的帥氣 app。

打開 Safari，連到網站 Swift Concurrency by Example

https://www.hackingwithswift.com/quick-start/concurrency

點選 File > Add to Dock

設定 app 名字

點選 Add 後，Swift Concurrency 變成了 app，出現在下方的 Dock。

啟動 App

點選 dock 上的 Swift Concurrency 啟動 App，如下圖所示，它變成了獨立的 App，它不再是 Safari 裡開啟的小小網頁，上方的 menu 霸氣地顯示它的大名，Swift Concurrency by Example，

獨立 App 有許多好處，比方我們可按 cmd + tab 切換 App。

之後啟動 App 也很方便，從 Launchpad 和 Spotlight 都可快速啟動。
