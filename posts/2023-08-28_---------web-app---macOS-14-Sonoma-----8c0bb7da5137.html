將網站變成獨立的 web app — macOS 14 Sonoma 新功能

從 macOS Sonoma 開始，我們可以將網站變成獨立的 web app，方便之後隨時瀏覽和切換 app。

以下彼得潘以自己的個人網站為例，將它變成有著帥氣 icon 的 app。

打開 Safari，連到彼得潘的個人網站

https://apppeterpan.mystrikingly.com/

點選 Add to Dock

點選右上方的 Share button，然後點選 Add to Dock。

設定 app 名字和 icon

點選 icon 圖片可選擇圖片。

點選 Add 後，網站變成了 app，出現在下方的 Dock。

啟動 App

點選 dock 上的 icon 啟動 App，如下圖所示，它變成了獨立的 App，它不再是 Safari 裡開啟的小小網頁，上方的 menu 霸氣地顯示它的大名，愛瘋一切為蘋果的彼得潘。

獨立 App 有許多好處，比方我們可按 cmd + tab 切換 App。之後啟動 App 也很方便，從 Apps 和 Spotlight 都可快速啟動。
