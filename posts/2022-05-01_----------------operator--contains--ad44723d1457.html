判斷資料是否在某個範圍的 ~= operator (contains)

開發 iOS App 時，有時我們會想判斷資料是否在某個範圍，比方判斷數字是否在 1…5 之間。

Swift 提供了一個特別的 operator ~=，方便我們判斷資料是否在某個範圍。

程式的寫法如下，~= 的前面寫範圍，後面寫比較的對象，以下例子將印出 3 is in 1…5。

let number = 3
if 1...5 ~= number {
    print("\(number) is in 1...5")
} else {
    print("\(number) is not in 1...5")
}

以下例子將印出 6 is not in 1…5。

let number = 6
if 1...5 ~= number {
    print("\(number) is in 1...5")
} else {
    print("\(number) is not in 1...5")
}
