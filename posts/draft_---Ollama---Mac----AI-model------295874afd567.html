執行 Ollama App

點選 Install。

安裝 model。

library
Get up and running with large language models.ollama.com
llama3.1
Llama 3.1 is a new state-of-the-art model from Meta available in 8B, 70B and 405B parameter sizes.ollama.com

打開 Terminal，輸入以下指令安裝 llama3.1 8b。

ollama run llama3.1

開始聊天。

當 model 已經裝好後，之後輸入 ollama run llama3.1 即可馬上開始聊天。

如果只需要一次性的回答，也可以在 ollama run llama3.1 之後接問題。

ollama run llama3.1 請說明圍棋的玩法

安裝 mistral-nemo。
