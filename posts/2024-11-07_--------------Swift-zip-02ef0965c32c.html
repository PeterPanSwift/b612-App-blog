將兩個集合的成員依序配對的 Swift zip

透過 swift 的 zip function 我們可以將兩個集合的成員依序配對後產生新的集合。

zip(_:_:) | Apple Developer Documentation
Creates a sequence of pairs built out of two underlying sequences.developer.apple.com

以金庸小說的男女主角配對為例，zip(maleCharacters, femaleCharacters) 會產生新的集合，集合的成員為男女主角依序配對，型別為 Tuple。因此 (郭靖、黃蓉) 是一對，(楊過、小龍女) 是一對，(令狐冲、任盈盈) 是一對。

let maleCharacters = ["郭靖", "楊過", "令狐冲"]
let femaleCharacters = ["黃蓉", "小龍女", "任盈盈"]
let couples = zip(maleCharacters, femaleCharacters)
for couple in couples {
    print("\(couple.0) ❤️ \(couple.1)")
}

以上程式的 couple 型別為 (String, String)，couple.0 可取得男主名字，couple.1 可取得女主名字。

程式印出的結果如下。

郭靖 ❤️ 黃蓉
楊過 ❤️ 小龍女
令狐冲 ❤️ 任盈盈

剛剛的 for in 也可以改成以下寫法，直接在 for in 中間宣告 Tuple 型別的 (male, female) 存取男女主角的名字。

for (male, female) in couples {
    print("\(male) ❤️ \(female)")
}

如果 zip 配對時，發現兩個集合的成員數量不一致呢 ? 就像以下程式男主有 3 個，女主有 2 個。

let maleCharacters = ["郭靖", "楊過", "令狐冲"]
let femaleCharacters = ["黃蓉", "小龍女"]
let couples = zip(maleCharacters, femaleCharacters)
for (male, female) in couples {
    print("\(male) ❤️ \(female)")
}

此時會以成員數較少的集合決定配對的組數，因此只會配對兩組，印出以下結果，令狐冲只能孤單一人去深山練劍。

郭靖 ❤️ 黃蓉
楊過 ❤️ 小龍女

開發 App 時，zip 可以簡化我們的程式，不用再麻煩地用 index 分別取出兩個集合的成員，以下我們舉兩個例子說明。

  • Apple 的 Alphabetizer 範例。
Complete a game with logic | Apple Developer Documentation
In this tutorial, you’ll build a fully functioning iPad game for kids learning to spell, called Alphabetizer. Working…developer.apple.com

沒有 zip 時，我們必須用 index 取得兩個集合 tiles & newWords 的成員。

for index in 0..<tileCount {
    let tile = tiles[index]
    let word = newWords[index]
    tile.word = word
}

使用 zip 後簡單許多，直接就能取得集合的成員。

寫法 1: 搭配 for in。

for (tile, word) in zip(tiles, newWords) {
    tile.word = word
}

寫法 2: 搭配 forEach。

zip(tiles, newWords).forEach { tile, word in
    tile.word = word
}

ps: 除了 zip，使用 enumerated 也是另一種解法。

寫法 1: 搭配 for in。

for (i, tile) in tiles.enumerated() {
    tile.word = newWords[i]
}

寫法 2: 搭配 forEach。

tiles.enumerated().forEach { i, tile in
    tile.word = newWords[i]
}

以下是 Alphabetizer 另一個使用 zip 的片段。

for (tile, correctTile) in zip(
    userSortedTiles, alphabeticallySortedTiles) {
    let tileIsAlphabetized = tile == correctTile
    tile.flipped = tileIsAlphabetized
}
  • 將字串 array 的內容顯示到 label array。
class LyricsTableViewController: UIViewController {
    let lyrics = ["那麼多如果", "可能如果我", "可惜沒如果", "只剩下結果"]
    @IBOutlet var labels: [UILabel]!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        
        for (label, lyric) in zip(labels, lyrics) {
            label.text = lyric
        }
    }
}

ps: 除了 zip，使用 enumerated 也是一種解法。

class LyricsTableViewController: UIViewController {
    let lyrics = ["那麼多如果", "可能如果我", "可惜沒如果", "只剩下結果"]    
    @IBOutlet var labels: [UILabel]!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        
        for (i, label) in labels.enumerated() {
            label.text = lyrics[i]
        }
    }
}
