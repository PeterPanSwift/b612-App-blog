方便存取 menu command 的 Show Quick Actions — Xcode 15 新功能

Xcode 15 推出了方便存取 menu command 的 Show Quick Actions，只要按下快速鍵 cmd + shift + a，即可叫出 Quick Actions 選單，執行想做的動作。

Quick Actions 帶來了以下好處。

  • 不用動滑鼠就能執行 menu 的 command。
  • 只要記得 command 的關鍵字就能輸入，比起記快速鍵容易許多。
  • 沒有快速鍵的 command 也能快速執行。

以下我們試試幾個常見的應用。

輸入 emoji

輸入 emoji 按 enter，顯示 emoji 視窗。

改名(rename)

輸入 ren 按 enter，比方修改 ContentView 的名字。

產生 Memberwise initializer

輸入 mem 按 enter，產生 Memberwise initializer。

class Movie {
    internal init(title: String, year: Int, director: String, genre: String) {
        self.title = title
        self.year = year
        self.director = director
        self.genre = genre
    }
    
    let title: String
    let year: Int
    let director: String
    let genre: String
}

加入 package

輸入 addpa 按 enter，顯示 add package 視窗。

執行 git commit

輸入 comm 按 enter 執行 git commit。
