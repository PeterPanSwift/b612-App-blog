iOS 14 變成月曆的 date picker

在 iOS 14 選擇日期時間變得更方便了。內建的 date picker 提供了精美的月曆讓我們選日期，再也不用自己辛苦打造月曆畫面了。

以下我們分別示範 SwiftUI & UIKit 的範例，用 date picker 選擇彼得潘告白的時間吧。

SwiftUI

struct ContentView: View {
    
    @State private var loveTime = Date()
    
    var body: some View {       
        Form {
            HStack {
                Image("peter")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 150)
                Image(systemName: "heart.text.square.fill")
                    .resizable()
                    .frame(width: 150, height: 150)
                    .foregroundStyle(.pink)
            }
            DatePicker("跟 Wendy 告白", selection: $loveTime)
        }
    }
}

date picker 預設將以精簡的文字顯示。

點選時間將出現漂亮的月曆畫面。彼得潘點選 6/12，因為那天是彼得潘跟 Wendy 說我喜歡你的日子。

點選 Time，輸入告白的那一刻， 8:40。

點選畫面空白的地方，date picker 將自動收起，回到原來的畫面。

dark mode 的月曆

只輸入日期

DatePicker("跟 Wendy 告白", selection: $loveTime, displayedComponents: .date)

只輸入時間

DatePicker("跟 Wendy 告白", selection: $loveTime, displayedComponents: .hourAndMinute)

設定 date picker 的樣式

  • 滾輪樣式

設定 wheel。

DatePicker("跟 Wendy 告白", selection: $loveTime)
    .datePickerStyle(.wheel)
  • 直接顯示月曆

設定 GraphicalDatePickerStyle。

struct ContentView: View {
    @State private var loveTime = Date()
    
    var body: some View {
        
        VStack {
            HStack {
                Image("peter")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 150)
                Image(systemName: "heart.text.square.fill")
                    .resizable()
                    .frame(width: 150, height: 150)
                    .foregroundStyle(.pink)
            }
            Text(loveTime.description)
            DatePicker("跟 Wendy 告白", selection: $loveTime, displayedComponents: .date)
                .datePickerStyle(.graphical)
        }
    }
}

利用 tint 調整選取日期的顏色。

DatePicker("跟 Wendy 告白", selection: $loveTime, displayedComponents: .date)
    .datePickerStyle(.graphical)
    .tint(.red)

利用 background 調整日曆的背景顏色。

DatePicker("跟 Wendy 告白", selection: $loveTime, displayedComponents: .date)
    .datePickerStyle(.graphical)
    .background(Color.yellow)

設定採用哪一國的月曆

台灣的月曆。

DatePicker("生日", selection: $birthday, displayedComponents: .date)
    .environment(\.locale, Locale(identifier: "zh_Hant_TW"))
    .environment(\.calendar, Calendar(identifier: .republicOfChina))

日本的月曆。

DatePicker("生日", selection: $birthday, displayedComponents: .date)
    .environment(\.locale, Locale(identifier: "zh_Hant_TW"))
    .environment(\.calendar, Calendar(identifier: .japanese))

UIKit

在 storyboard 上加入的 date picker 將以精簡的文字顯示，點選後顯示美美的月曆畫面。

設定 date picker 的樣式

設定 preferredDatePickerStyle。

  • 顯示滾輪
class ViewController: UIViewController {

    @IBOutlet weak var datePicker: UIDatePicker!
    
    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.
        
        datePicker.preferredDatePickerStyle = .wheels
    }

在 Interface Builder 也可以從 date picker 的 Style 欄位調整樣式。

  • 精簡模式
datePicker.preferredDatePickerStyle = .compact
  • 顯示月曆。
datePicker.preferredDatePickerStyle = .inline

設定採用哪一國的月曆

台灣的月曆。

datePicker.locale = Locale(identifier: "zh_Hant_TW")
datePicker.calendar = Calendar(identifier: .republicOfChina)

日本的月曆。

datePicker.locale = Locale(identifier: "zh_Hant_TW")
datePicker.calendar = Calendar(identifier: .japanese)
