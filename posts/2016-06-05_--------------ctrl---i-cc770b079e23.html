讓程式碼變美的自動縮排 : ctrl + i

如果說App展現的是外在美，內在美想當然是製作App的Swift程式碼。我們都知道，內在美比外在美還重要，接下來就讓我們實施微整型，手術時間只要1秒鐘，按下ctrl + i ，即可將原本亂成一團的程式碼整齊縮排，變成內在美一百分 !

  1. function數字情歌numberSong裡的程式碼慘不忍睹，雜亂無章。

2. 雙擊 { ，選取function { }裡的程式碼。

3. 按下快速鍵 ctrl + i 或從Xcode的menu選取Editor -> Structure -> Re-Indent，自動縮排 { }裡的程式碼。

手術成功 ! function numberSong成為整齊縮排，內在美一百分的程式碼 !
