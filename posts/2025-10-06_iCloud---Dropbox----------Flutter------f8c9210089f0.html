iCloud 和 Dropbox 等雲端空間造成的 Flutter 專案問題

開發 Flutter App 時，專案資料夾要避免放在跟 iCloud、Dropbox 等雲端空間同步的地方，不然在執行 App 將遇到錯誤，出現 Failed to build iOS app、Target debug_unpack_ios failed、Cannot open file 等錯誤訊息。

比方以下為 Mac 桌面開啟 iCloud 同步時，執行存在桌面的專案出現的錯誤訊息。

Failed to build iOS app
Uncategorized (Xcode): Exited with status code 255

Could not build the application for the simulator.
Error launching application on iPhone 17 Pro.

Exited (1).

Mac 關閉桌面 iCloud 的方法如下。

打開 System Settings App，點選側邊欄的個人帳號，然後點選 iCloud。

點選 Drive。

關閉 Desktop & Documents Folders 的開關。
