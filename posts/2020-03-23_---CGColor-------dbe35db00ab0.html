產生 CGColor 的兩種方法

使用 UIKit 開發 iOS App 時，我們主要使用的是 UIColor，不過有些時候我們卻必須使用 CGColor，比方 CALayer 的 borderColor 或 CAGradientLayer 的 colors。

產生 CGColor 的方法有以下兩種:

直接生成 CGColor

參數的範圍是 0 ~ 1，不過此方法要 iOS 13 以上才能使用。

let color = CGColor(srgbRed: 120/255, green: 0, blue: 0, alpha: 1)
init(srgbRed:green:blue:alpha:) | Apple Developer Documentation
Creates a color in the sRGB color space.developer.apple.com

從 UIColor 得到 CGColor

從 UIColor 的 property cgColor 得到 CGColor。

let color = UIColor(red: 120/255, green: 0, blue: 0, alpha: 1).cgColor
cgColor | Apple Developer Documentation
The Quartz color that corresponds to the color object.developer.apple.com
