從 Xcode 切換到 Terminal 專案資料夾的兩種方法

開發 iOS App 時，我們偶爾要切換到只有文字的 Terminal 畫面，比方進行 git 或 cocoapods 的相關操作。從 Xcode 切換到 Terminal 專案資料夾有 2 種方法，一種是基本的手動切換，另一種則是先設定 Xcode 的 behavior，之後即可從 Xcode 一鍵啟動 Terminal。以下我們逐一介紹這兩種方法。

方法一: 打開 terminal，手動切換到專案資料夾

  • 打開 Terminal。

點選右上角的放大鏡，輸入 ter 啟動 Terminal。

  • 在 terminal 輸入 cd 和一個空白。

cd 是切換資料夾的指令，它的後面要接我們想進入的資料夾路徑。

  • 從 Xcode 將藍色資料夾 icon 拖曳到 terminal。
  • 移除 .xcodeproj 結尾的檔名。

此在我們移除 Demo.xcodeproj。

  • 按下 enter。

經歷了千辛萬苦，終於成功切換到專案資料夾 Demo 下。

方法二: 設定 behavior，從 Xcode 啟動 terminal

感謝以下網站介紹了如何設定從 Xcode 啟動 terminal 的 behavior。

Xcode - Open Terminal in current Xcode project folder | xcode Tutorial
Xcode have ability to run any script with hotkey. Here is example how to assign hotkey ⌘+⌥+⌃+⇧+T to open Terminal app…riptutorial.com

以下彼得潘照著網站的步驟操作一次:

  • 在某個資料夾下建立 bash 檔。

比方我們在 Documents 下建立 xcodeTerminal.sh，內容如下:

#!/bin/bash
# Project Name:  $XcodeProject
# Project Dir:   $XcodeProjectPath
# Workspace Dir: $XcodeWorkspacePath
open -a Terminal "$(dirname $XcodeProjectPath)"
  • 打開 xcodeTerminal.sh 的執行權限。
chmod +x xcodeTerminal.sh
  • 打開 Behaviors 設定視窗。

Xcode > Behaviors > Edit Behaviors。

  • 點選左下的 + 新增 Behavior。
  • 輸入 Behavior 的名稱。

比方輸入 open terminal。

  • 勾選 Run，點選 Choose Script 選擇 xcodeTerminal.sh
  • 從 Xcode 啟動 open terminal 的 behavior。

Xcode > Behaviors > open terminal。

Cool，不用再像第一個方法打一堆字，現在只要點選 Xcode 的選單即可啟動 terminal。

透過快速鍵啟動 Terminal

如果覺得不滿足，還想要更快，我們也可以設定快速鍵，在剛剛的 behavior 設定視窗裡指定啟動 terminal 的快速鍵。
