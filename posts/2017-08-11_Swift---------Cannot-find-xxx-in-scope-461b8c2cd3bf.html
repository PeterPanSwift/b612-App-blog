Swift 的名字問題 : Cannot find xxx in scope

初學者剛開始學習程式時，特別容易撞到名字問題，出現不認得名字的紅色錯誤:

Cannot find xxx in scope

初學者通常因為以下兩種原因遇到這樣的錯誤訊息。

  • 大小寫打錯或單字拼錯。
  • 參考範例輸入程式時，自己宣告的變數名或型別名和範例不同，或是你忘了建立型別和產生變數，但你照抄的程式裡卻使用到範例的型別和變數。

以下分別說明三種常見不認得名字的例子:

不認得型別

沒有 Baby 型別。這時候你可能需要 import 定義 Baby 型別的 framework，或是自己定義 Baby 型別，比方用 class, struct 或 enum 定義。

不認得變數或常數

不認得有 cuteBaby 這樣的變數或常數存在。你應該先用 var 或 let 宣告 cuteBaby，才能存取它。

不認得 function

不認得有 printer 這樣的 function 存在。你必須另外定義 function printer，才能呼叫它。
