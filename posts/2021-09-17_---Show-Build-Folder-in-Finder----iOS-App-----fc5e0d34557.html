利用 Show Build Folder in Finder 查詢 iOS App 的內容

當我們使用 Xcode 開發 iOS App 時，生成的 iOS App 將存在 Mac 的某個資料夾裡，我們可利用 Show Build Folder in Finder 打開它。

從 Quick Actions 點選 Show Build Folder in Finder

此時 Finder 將顯示 App 的 Build 資料夾，

在彼得潘的 Mac，Build 資料夾的路徑為

/Users/shih-yingpan/Library/Developer/Xcode/DerivedData/Demo-egmvhhfzunqgifbpmitcjzxeaxfh/Build

每個生成的 App 將在 DerivedData 下生成一個 App 資料夾，名稱為專案名加上一組亂碼，例如剛剛的 Demo-egmvhhfzunqgifbpmitcjzxeaxfh。

生成的 App 將在 Build > Products > Debug-iphonesimulator 下，比方下圖的 DemoButton。

iOS App 資料夾的內容

Demo 是 iOS App，附檔名是 app，它其實也是一個資料夾，我們可以查詢它的內容，從右鍵選單點選 Show Package Contents 可查看它底下藏的東西。

Demo.app 的內容如下，Demo 是 App 的執行檔，Assets.car 是 Assets 的資料，至於其它在 Project navigator 加入的素材檔也會出現在裡面，比方小幸運.mp3。

若有使用到 storyboard，則可在 Base.lproj 下看到 storyboard。
