SwiftUI 實現特定形狀背景的各種方法

開發 SwiftUI App 時，我們常常會設計特定形狀的背景，比方圓角矩形和圓形。以下我們以 VStack 為例，示範讓它搭配黃色圓角矩形背景的各種方法。

  • background(_:in:fillStyle:) 傳入顏色和形狀。
  • background 傳入顏色，clipShape 裁切形狀。
  • background(alignment:content:) 傳入某個顏色的形狀。
  • background 傳入顏色，mask 設定形狀。
  • background 傳入顏色，containerShape 設定形狀。
  • ZStack 搭配某個顏色的形狀 & fixedSize。

background(_:in:fillStyle:) 傳入顏色和形狀

VStack {
    Image(systemName: "star.fill")
    Text("You, only you, will have stars that can laugh!")
}
.padding()
.background(.yellow, in: .rect(cornerRadius: 20))
利用 SwiftUI 的 background(_:in:fillStyle:) 設定背景，用顏色、漸層、圖片填滿形狀

background 傳入顏色，clipShape 裁切形狀

VStack {
    Image(systemName: "star.fill")
    Text("You, only you, will have stars that can laugh!")
}
.padding()
.background(.yellow)
.clipShape(.rect(cornerRadius: 20))

background(alignment:content:) 傳入某個顏色的形狀

VStack {
    Image(systemName: "star.fill")
    Text("You, only you, will have stars that can laugh!")
}
.padding()
.background {
    RoundedRectangle(cornerRadius: 20)
        .fill(.yellow)
}

background 傳入顏色，mask 設定形狀

VStack {
    Image(systemName: "star.fill")
    Text("You, only you, will have stars that can laugh!")
}
.padding()
.background(.yellow)
.mask {
    RoundedRectangle(cornerRadius: 20)
}

background 傳入顏色，containerShape 設定形狀

VStack {
    Image(systemName: "star.fill")
    Text("You, only you, will have stars that can laugh!")
}
.padding()
.background(.yellow)
.containerShape(.rect(cornerRadius: 20))

ZStack 搭配某個顏色的形狀 & fixedSize

在 ZStack 的第一個成員加入形狀，然後在 ZStack 呼叫 fixedSize，讓 ZStack 的大小剛好等於內容的大小。

ZStack {
    RoundedRectangle(cornerRadius: 20)
        .fill(Color.yellow)
    VStack {
        Image(systemName: "star.fill")
        Text("You, only you, will have stars that can laugh!")
    }
    .padding()
}
.fixedSize()
