利用 Bool 或 Optional 控制 alert，actionSheet，sheet，fullScreenCover，popover & NavigationLink 的畫面是否顯示

開發 SwiftUI App 時，我們可以利用 Bool 或 Optional 控制畫面是否顯示，只要 Bool 為 true 或 Optional 有值即顯示。此方法可運用在以下兩種 case:

  • 利用 Bool 控制 alert 或 confirmation dialog 是否顯示。
  • 利用 Bool 或 Optional 控制 sheet，fullScreenCover，popover & NavigationLink 搭配的畫面是否顯示。

接下來就讓我們以鬼滅之刃英勇的九柱為例，說明 SwiftUI 如何利用 Bool 或 Optional 控制畫面的顯示。

宣告遵從 Identifiable 的資料型別

當 alert，actionSheet，popover，sheet & fullScreenCover 利用 Optional 是否有值顯示畫面時，Optional 的資料型別必須遵從 Identifiable，因此我們定義以下的 Hero 型別儲存柱的名字。

struct Hero: Identifiable {
    let id = UUID()
    let name: String
}

宣告九柱的 array

struct ContentView: View {
    
    let heroes = ["水柱", "蛇柱", "蟲柱", "炎柱", "戀柱", "岩柱", "風柱", "霞柱", "音柱"].map(Hero.init(name:))

alert — Bool 為 true 時顯示 alert

func alert<A, M>(
    _ titleKey: LocalizedStringKey,
    isPresented: Binding<Bool>,
    @ViewBuilder actions: () -> A,
    @ViewBuilder message: () -> M
) -> some View where A : View, M : View

範例

struct ContentView: View {
    let heroes = ["水柱", "蛇柱", "蟲柱", "炎柱", "戀柱", "岩柱", "風柱", "霞柱", "音柱"].map(Hero.init(name:))
    
    @State private var isShow = false
    
    var body: some View {
        NavigationStack {
            Button(action: {
                isShow = true
            }, label: {
                Text("九柱")
            })
            .navigationTitle("鬼滅之刃")
            .alert("九柱", isPresented: $isShow, actions: {
                Button("OK") { }
            }, message: {
                Text(heroes.map{ $0.name }.joined(separator: " "))
            })
        }
    }
}

點選九柱顯示 alert。

confirmationDialog — Bool 為 true 時顯示 confirmation dialog

func confirmationDialog<A>(
    _ titleKey: LocalizedStringKey,
    isPresented: Binding<Bool>,
    titleVisibility: Visibility = .automatic,
    @ViewBuilder actions: () -> A
) -> some View where A : View

範例

struct ContentView: View {
    let heroes = ["水柱", "蛇柱", "蟲柱", "炎柱", "戀柱", "岩柱", "風柱", "霞柱", "音柱"].map(Hero.init(name:))
    
    @State private var isShow = false
    
    var body: some View {
        NavigationStack {
            Button(action: {
                isShow = true
            }, label: {
                Text("九柱")
            })
            .navigationTitle("鬼滅之刃")
            .confirmationDialog("鬼滅之刃", isPresented: $isShow, actions: {
                ForEach(heroes, id: \.name) { hero in
                    Button(hero.name) { }
                }
            })
        }
    }
}

sheet — Optional 有值時顯示新畫面

func sheet<Item, Content>(item: Binding<Item?>, onDismiss: (() -> Void)? = nil, content: @escaping (Item) -> Content) -> some View where Item : Identifiable, Content : View

範例

struct ContentView: View {
    
    let heroes = ["水柱", "蛇柱", "蟲柱", "炎柱", "戀柱", "岩柱", "風柱", "霞柱", "音柱"].map(Hero.init(name:))
    
    @State private var selectHero: Hero?
    
    var body: some View {
        NavigationStack {
            List(heroes) { hero in
                HStack {
                    Text(hero.name)
                    Spacer()
                }
                .background(Color.yellow)
                .onTapGesture {
                    selectHero = hero
                }
            }
            .navigationTitle("鬼滅之刃")
            .sheet(item: $selectHero) { hero in
                Image(hero.name)
                    .resizable()
                    .scaledToFit()
            }
        }
    }
}

點選水柱顯示水柱，點選蟲柱顯示蟲柱。

NavigationLink — Optional 內容等於 tag 時顯示新畫面

init<V>(destination: Destination, tag: V, selection: Binding<V?>, label: () -> Label) where V : Hashable

範例

NavigationLink 搭配的 Optional 資料型別必須遵從 Hashable。

struct Hero: Identifiable, Hashable {
    let id = UUID()
    let name: String
}

點選 NavigationLink 時，selectHero 將自動被設為 NavigationLink 對應的 tag。除此之外，我們另外加了一個 button 水柱是最帥的，點選後將 selectHero 設為水柱，顯示水柱的帥氣圖片。

struct ContentView: View {
    
    let heroes = ["水柱", "蛇柱", "蟲柱", "炎柱", "戀柱", "岩柱", "風柱", "霞柱", "音柱"].map(Hero.init(name:))
    
    @State private var selectHero: Hero?
    
    var body: some View {
        NavigationStack {
            List {
                ForEach(heroes) { hero in
                    NavigationLink(
                        destination: Image(hero.name)
                            .resizable().scaledToFit(),
                        tag: hero,
                        selection: $selectHero,
                        label: {
                            HStack {
                                Text(hero.name)
                                Spacer()
                            }
                            .background(Color.yellow)
                    })
                }
                Button(action: {
                    selectHero = heroes[0]
                }, label: {
                    Text("水柱是最帥的")
                })
                
            }
            .navigationTitle("鬼滅之刃")
        }
    }
}

點選水柱是最帥的顯示水柱。
