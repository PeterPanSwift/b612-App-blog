Xcode 11 Asset 有趣的新功能

Xcode 11 Asset 多了不少有趣的新功能，比方搜尋和 find & replace，多國語言的支援， copy paste，以及設定 dark mode 的圖片 & color(dark mode 在 Xcode 10 就可以設定，不過 iOS 13 才支援 dark mode)。接下來就讓我們一步步認識這些有趣的新功能吧。

可以搜尋和 find & replace asset 裡的東西

現在我們可以搜尋 asset 裡的東西，找圖變得更加方便。如下圖所示，當我們在 Find navigator 裡搜尋 tree 時，它將同時搜尋 asset 裡 tree 相關的東西。(舊版的 Xcode 無法搜尋 asset 的東西)

我們也可以 find & replace asset 裡東西的名字，步驟如下

1. 點選 Assets.xcassets 後，點選 cmd + f 進入搜尋模式。
2. 點選 Find，切換成 Replace。
3. 輸入 Tree & tree，準備將大寫的 Tree 換成小寫的 tree。
4. 點選 All 將所有的 Tree 都變成 tree。
Xcode can find assets in your workspace or project using the Find navigator. The Asset catalog Editor also supports find and replace, and you can rename assets using Replace

asset 可以針對多國語言設定不同圖片

比方我們的 App 有英文 & 中文版，因此我們在專案的 Localization 加入 Chinese(Traditional)。

我們想設定 asset 裡的 peter 圖片，讓它在英文和中文版顯示不同圖片。

點選 peter 後，在右邊的 Attributes inspector 分頁點選 Localize 按鈕。

此時將出現 Chinese & English 的選項。

勾選 Chinese(Traditional)。

peter 圖片多了 Chinese(Traditional) 的 1x, 2x, 3x 格子。

如下圖所示，現在我們可以設定 peter 在中文時顯示另一張圖片了。

You can now localize assets in asset catalogs. Localization is enabled in the attribute inspector.

asset 裡的東西可以 copy paste

在 Xcode 11，我們終於可以在 asset 裡做複製貼上了。

我們可以對格子裡的圖片複製貼上，如下圖所示，我們點選 peter 格子裡的圖片後按 cmd + c，然後再用 cmd + v 貼到 hook 的格子。

我們也可以點選 asset 側邊欄的圖片按 cmd + c，然後再按 cmd + v 在側邊欄貼上複製的圖片(image set)。

在不同專案間也可以複製貼上 assets 的圖片，有興趣的朋友可參考以下連結。

在 Xcode 專案間複製貼上 assets 圖片
從 Xcode 11 開始，assets 裡的圖片也可以複製貼上了。medium.com
Assets can now be cut, copied, pasted, and duplicated using the menu or keyboard shortcuts.

從 asset 設定 dark mode 的圖片 & color

  • 設定 dark mode 的圖片

點選圖片 peter。

從右邊 Attributes inspector 分頁的 Appearances 欄位設定 dark mode 的圖片。

從 Appearances 的下拉選單，我們可選擇 Any, Dark 或 Any, Light, Dark 設定 dark mode 的圖片。

在此我們選擇 Any, Dark。

選擇 Any, Dark 後，圖片多了 Dark Appearance 的 1x, 2x, 3x 格子，我們可在格子裡放入 dark mode 的圖片，而 light mode 或是舊版不支援 dark mode 的 iOS 則會採用 Any Appearance 裡的圖片。

  • 設定 dark mode 的顏色

我們也可以為自訂的顏色另外指定 dark mode 的顏色。

首先我們點選下方的 +，選擇 Color Set 新增顏色

顏色預設會有兩個格子，其中的 Dark Appearance 代表 dark mode 的顏色。

如果不想設定 dark mode，可從 Attributes inspector 將 Appearances 設為 None。
