#97 UIPickerView 練習

目的: 學習使用 UIPickerView。

基本題

  • 在畫面上顯示單欄的 Picker View。

提示:

UIPickerViewDataSource | Apple Developer Documentation
The interface for a picker view's data source.developer.apple.com
pickerView(_:numberOfRowsInComponent:) | Apple Developer Documentation
Asks the data source for the number of rows for a specified component.developer.apple.com
numberOfComponents(in:) | Apple Developer Documentation
Asks the data source for the number of components in the picker view.developer.apple.com
UIPickerViewDelegate | Apple Developer Documentation
The interface for a picker view's delegate.developer.apple.com
pickerView(_:titleForRow:forComponent:) | Apple Developer Documentation
Called by the picker view when it needs the title to use for a given row in a given component.developer.apple.com
  • 滑動 Picker View 時更新欄位內容。

範例: 練習選擇身高。

提示:

pickerView(_:didSelectRow:inComponent:) | Apple Developer Documentation
Called by the picker view when the user selects a row in a component.developer.apple.com
  • 在畫面上顯示多欄的 Picker View。

進階題

  • 點選畫面後顯示 Picker View。
  • Picker View 的上方加入包含完成 & 取消 button 的 bar，按下完成時更新畫面的內容。
  • 顯示多欄的 Picker View，第二欄的 picker view 隨著第一欄而變化。

範例: 選擇火車起站或台灣行政區。
