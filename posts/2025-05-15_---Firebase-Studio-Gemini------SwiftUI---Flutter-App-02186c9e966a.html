利用 Firebase Studio Gemini 畫圖開發 SwiftUI / Flutter App

Firebase Studio Gemini 內建 Google 強大的 AI Gemini，最近彼得潘發現它除了常見的上傳圖檔功能，還可以畫圖請它生成 App 畫面呢 !

以下我們試試畫圖生成 App 的登入畫面。

打開 Firebase Studio Gemini 的 Gemini 聊天視窗，點選 Gemini: Chat with Gemini。

點擊輸入框的 Attach 按鈕，選擇 Drawing。

開始畫畫吧。

發揮彼得潘的畫畫天份繪製漂亮的登入畫面。

prompt 輸入 create SwiftUI App，AI 的回答如下。

生成的程式。

import SwiftUI

struct LoginView: View {
    @State private var account: String = ""
    @State private var password: String = ""

    var body: some View {
        VStack {
            TextField("account", text: $account)
                .padding()
                .border(Color.gray, width: 1)

            SecureField("password", text: $password)
                .padding()
                .border(Color.gray, width: 1)

            Button("login") {
                // Action for login button
            }
            .padding()
        }
        .padding()
    }
}

struct LoginView_Previews: PreviewProvider {
    static var previews: some View {
        LoginView()
    }
}

將程式貼到 Xcode 後呈現的登入畫面。

prompt 輸入 create Flutter App，AI 的回答如下。

生成的程式。

import 'package:flutter/material.dart';

void main() {
  runApp(const LoginApp());
}

class LoginApp extends StatelessWidget {
  const LoginApp({super.key});
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Login Page',
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(seedColor: Colors.deepPurple),
      ),
      home: const LoginPage(),
    );
  }
}

class LoginPage extends StatefulWidget {
  const LoginPage({super.key});

  @override
  State<LoginPage> createState() => _LoginPageState();
}

class _LoginPageState extends State<LoginPage> {
  final TextEditingController _accountController = TextEditingController();
  final TextEditingController _passwordController = TextEditingController();

  void _login() {
    // Implement your login logic here
    String account = _accountController.text;
    String password = _passwordController.text;
    print('Account: $account, Password: $password');
    // You would typically validate the credentials and navigate to another screen
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        backgroundColor: Theme.of(context).colorScheme.inversePrimary,
        title: const Text('Login Page'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            SizedBox(
              width: 300.0, // Adjust width as needed
              child: Padding(
                padding: const EdgeInsets.all(8.0),
                child: TextField(
                  controller: _accountController,
                  decoration: const InputDecoration(
                    labelText: 'Account',
                    border: OutlineInputBorder(),
                  ),
                ),
              ),
            ),
            SizedBox(
              width: 300.0, // Adjust width as needed
              child: Padding(
                padding: const EdgeInsets.all(8.0),
                child: TextField(
                  controller: _passwordController,
                  decoration: const InputDecoration(
                    labelText: 'Password',
                    border: OutlineInputBorder(),
                  ),
                  obscureText: true,
                ),
              ),
            ),
            ElevatedButton( // Adjust button width as needed
              onPressed: _login,
              child: const Text('Login'),
            ),
          ],
        ),
      ),
    );
  }
}
