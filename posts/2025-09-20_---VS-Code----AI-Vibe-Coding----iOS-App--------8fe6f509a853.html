以下我們說明使用 VS Code 搭配 AI Vibe Coding 開發 iOS App 的主要步驟。

  • 用 Xcode 建立 iOS App 專案。
  • 用 VS Code 打開專案資料夾。
  • 將 VS Code 的 Files: Auto Save 設為 afterDelay。
  • 打開 AI 視窗，輸入 prompt 請 AI 寫程式。
  • 從 Xcode 或 iOS Simulator 查看 App 畫面。
  • 修正 AI 製造的紅色錯誤。
  • 安裝 Xcode-Build-Server。

用 Xcode 建立 iOS App 專案

用 VS Code 打開專案資料夾

將專案資料夾拖曳到 VS Code 的 App Icon 或從 Command Palette(cmd + shift + p)點選 File: Open Folder。

將 VS Code 的 Files: Auto Save 設為 afterDelay

從 Command Palette 點選 Open Settings (UI)，將 Files: Auto Save 設為 afterDelay。

為了讓 AI 修改程式後自動存檔，方便 Xcode 載入修改後的程式，建議將 Files: Auto Save 設為 afterDelay，省去另外按 cmd + s 存檔的麻煩。

打開 AI 視窗，輸入 prompt 請 AI 寫程式

從 VS Code 打開 GitHub Copilot、Codex、Claude Code、Gemini Code Assist。

從 Xcode 或 iOS Simulator 查看 App 畫面

SwiftUI App 請切回 Xcode 查看 Preview，立即能把玩 AI 開發的 App。

UIKit App 則須將 App 裝到模擬器才能操作。(ps: 我們也可以請 AI 加上 #Preview，不過目前 AI 有時會寫錯。)

修正 AI 製造的紅色錯誤

從 Xcode 發現 App 跑不起來，程式出現紅色錯誤時可以請 AI 修正問題。

  • 方法 1: 貼圖請 AI 修正。

安裝 Xcode-Build-Server

若想直接在 VS Code 看到專案的程式錯誤，不用再切回 Xcode 查看，請再安裝 Xcode-Build-Server。
