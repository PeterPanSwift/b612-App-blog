在 iOS App 加入生動的 TW Icon Fonts

最近網路上出現了很有名的 Taiwan Icon Font，由深愛台灣的日本人和英國人合作設計。

Taiwan Icon Font
我們支持「臺灣」，因為臺灣有好大的吸引力，而且台灣就是我們的好朋友！2019年散步在東京會看到好多人在喝珍珠奶茶，去台灣旅遊的日本人每年越來越多（參考：台灣交通部觀光局的統計）。日本人選的首相，安倍晋三叫臺灣「好朋友」，每個日本人都還沒忘記…www.twicon.page

彼得潘看到這些有趣的 icon，決定來試試將它們加到開發的 iOS App 。

下載 icon 檔

TW Icon Fonts 可以免費下載，請放心勇敢地從以下連結點選 Download 下載。

Get Started
Download 我們為了電腦上會用的OTF檔案和為了網站或是服務會用的WOF跟WOF2檔案，為了開發很多方面的開發會用的SVG，把這些三個檔案我們提供給世界裡愛台灣的人。從下個「Download」按下載一下。 For Web ...www.twicon.page

下載後打開 twicon 資料夾，適合 iOS App 的是其中的 twicon.otf，因為 iOS App 只支援 otf & ttf 格式的字型檔。

將 twicon.otf 加到 Xcode 專案裡

參考以下連結的說明將 twicon.otf 加到 Xcode 專案。

為 iOS App 加入客製字型 (custom font)
覺得 iOS App 內建的中文字型不夠可愛嗎 ? 沒關係，接下來我們將從日本的字體網站，下載可愛的粉筆字體，在 iOS App 上顯示迷死人不償命的粉筆字 !medium.com

twicon.otf 將出現在 Project navigator 的檔案清單。

在 App 的 Info 頁面加入 Fonts provided by application，設定 Item 0 的 value 為 twicon.otf。

現在我們已經可以在 App 裡使用 TW Icon Fonts，接下來就讓我們分別在 SwiftUI & Storyboard 試試吧。

在 SwiftUI 使用 TW Icon Fonts

Text("Hello, World!")
   .font(.custom("Taiwanicon-20", size: 30))

TW Icon Fonts 的字型名稱是 Taiwanicon-20，因此我們利用 custom 指定字型和大小。

如何利用 Font Book 知道字型的名字
想在 iOS App 裡使用特別的字型並不難，只要能將字體的 otf (Open Type Font)或 ttf(True Type Font) 檔加到專案裡，並在 App 的 info 頁面做設定。medium.com

如下圖所示，Hello, World! 看起來沒什麼變化。這是當然的呀，TW Icon Fonts 並不是拿來顯示英文的，它是用來顯示珍奶，歪腰郵筒等有趣的台灣事物。

接下來我們就以顯示台灣黑熊為例子示範吧。首先我們得先找出台灣黑熊 icon 對應的文字。請打開 twicon 資料夾下的 List.html。

網頁裡表格的第一個 column 將列出每個 icon 對應的文字。

找到可愛的台灣黑熊，選取它後從右鍵選單 copy。

然後貼到 Text 顯示的字串裡。

Text("ꁐ")
    .font(.custom("Taiwanicon-20", size: 30))

Cool，台灣黑熊現身了 !

覺得黑熊太小嗎 ? 讓我們將 size 調成 200，瞧瞧威猛的大黑熊。

Text("ꁐ")
   .font(.custom("Taiwanicon-20", size: 200))

最近很多人將 iPhone 調成 dark mode，慘了，這樣台灣黑熊不就看不到了 ? 別擔心，SwiftUI 對 dark mode 有很好的支援，Text 預設會在 light / dark mode 切換黑 / 白色。

當然，我們也可以設定顏色，比方將台灣黑熊變成粉紅熊。

Text("ꁐ")
   .font(.custom("Taiwanicon-20", size: 200))
   .foregroundColor(.pink)

在 Storyboard 使用 TW Icon Fonts

設定 Font 的 Family，選擇 Taiwan icon。

調整 Color，變成台灣藍熊。
