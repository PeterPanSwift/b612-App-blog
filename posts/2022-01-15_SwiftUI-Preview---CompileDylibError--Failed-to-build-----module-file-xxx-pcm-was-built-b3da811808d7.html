SwiftUI Preview 的 CompileDylibError: Failed to build 問題: module file xxx.pcm was built

當我們利用 SwiftUI Preview 預覽 App 畫面時，有時會看到以下錯誤。

SwiftUI Preview 的 CompileDylibError: Failed to build 問題: module file xxx.pcm was built

當我們利用 SwiftUI Preview 預覽 App 畫面時，有時會看到以下錯誤。

點選 Diagnostics 的畫面如下，問題出在 xxx.pcm 檔。

解法很簡單，我們只要把 pcm 檔刪除即可。

打開 Terminal App，輸入以下指令，rm 後貼上從 Diagnostics 視窗複製的 pcm 檔路徑。

rm /var/folders/fj/bn09hxb93xngptj1c7mrx4xm0000gn/C/clang/ModuleCache/HP5PGJ664UGB/SwiftShims-2TTN5UXQBRCCQ.pcm

接著點選 Try Again，即可看到 preview 畫面順利出現。
