如何知道 ( ) 前的東西是不是型別

開發 iOS App 時，我們時常看到 xxx() 的語法，主要分成以下兩種:

  • xxx 是型別，xxx() 將生成型別 xxx 的東西。

Int 是型別，Int(3.5) 將生成整數 3。

let number = Int(3.5)
  • xxx 是某個 function，xxx() 將呼叫 function 做某件事。

print 是 function，以下程式將列印訊息。

print("上邪，我欲與君相知，長命無絕衰。
山無陵，江水爲竭。冬雷震震，夏雨雪。天地合，乃敢與君絕。")

如何區別 xxx 是型別還是 function

透過以下幾個小技巧，我們可以快速看出 xxx 的型別還是 function，明白 xxx( ) 是生成東西還是呼叫 function。

  • 判斷第一個字是大寫還是小寫。

在 Swift 的世界裡，通常型別的第一個字會大寫，function 的第一個字會小寫。

Int() 的 I 是大寫，因此 Int 是型別。print() 的 p 是小寫，因此 print 是 function。

  • 按住 option 鍵，點選 xxx 查說明。

顯示 struct，class，enum 時表示它是型別。

顯示 func 時表示它是 function。

  • 輸入程式時觀看提示。

格子裡顯示 S，C，E 表示它是 struct，class，enum，因此是型別。

格子裡顯示 f，M 表示它是 function。
