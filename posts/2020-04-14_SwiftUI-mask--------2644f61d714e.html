SwiftUI mask 的粉紅彼得潘

最近粉紅口罩很紅，彼得潘也來試試利用 SwiftUI 將粉紅色跟彼得潘圖片 mask 成粉紅彼得潘 ~

struct ContentView: View {
        
    var body: some View {
        Color(red: 244/255, green: 177/255, blue: 187/255)
            .frame(width: 400, height: 700)
            .mask(
                VStack {
                    Image("peter")
                        .resizable()
                        .scaledToFit()
                    Text("是你讓我的世界從那刻變成粉紅色")
                        .font(.system(size: 40))
                        .bold()
                        .offset(y: -50)
                        .rotation3DEffect(.degrees(30), axis: (x: 1, y: 0, z: 0))
                }
        )
        .shadow(radius: 5)
        
    }
}

By 彼得潘的 iOS App Neverland on April 14, 2020.
