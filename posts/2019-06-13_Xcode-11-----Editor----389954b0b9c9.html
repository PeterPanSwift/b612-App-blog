Xcode 11 的多重 Editor 視窗

在新版的Xcode 11，在 source editor 同時瀏覽編輯多個檔案變得更方便了。

新增視窗

如下圖所示，當我們點選紅色圈圈標示的 Add Editor 按鈕時，它將在右邊新開一個和原本視窗一樣檔案的編輯視窗。

當我們點選 AppDelegate.swift 的 Add Editor 按鈕後，它會將原本的視窗切割成左右 2 塊，同時顯示 AppDelegate.swift。

當然我們不會閒著沒事做，想要看著兩個一模一樣的檔案。一般我們接著會點選某個視窗，然後從左邊的 Project navigator 選擇它要顯示的檔案，比方我們點選 LaunchScreen.storyboard，將右邊的視窗內容變成開頭畫面。

除了剛剛的方法，還有一個改變視窗內容的小技巧。當我們按住 option + shift，然後從 Project navigator 點選檔案，此時將出現藍色框框標示檔案將顯示的視窗。

我們可從滑鼠或方向鍵調整框框標示的視窗，然後按下 enter 或滑鼠左鍵，讓檔案顯示到對應的視窗。

上下分割新增視窗

每個 source editor 視窗的右上角都有 Add Editor 按鈕，因此我們可以不斷增加新的視窗。除了左右分割視窗，我們也可以按住 option 再點選 Add Editor，在原本視窗的下半部切割新視窗。

當我們分割出愈多視窗時，雖然可以一次看到愈多檔案的內容，但每個視窗也會變得愈小，無法顯示太多內容。

放大視窗

還好 Xcode 11 多了一個方便的放大視窗功能，我們可特別放大目前想編輯的某個視窗。

如下圖所示，當我們點選 AppDelegate.swift 視窗左上角的 focus 按鈕時，此視窗將放大佔滿整個空間，隱藏其它的分割視窗。

別擔心，此時其它的視窗並沒有被關閉。當我們編輯完 AppDelegate.swift 後，可再點選左上角的 Hide Focus 按鈕回到多個分割視窗。

至於已經編輯完用不到的視窗，我們也可點選左上角的 X 關閉視窗。
