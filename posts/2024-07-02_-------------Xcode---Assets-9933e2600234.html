將網頁上的圖片複製貼上到 Xcode 的 Assets

當我們在網頁上看到不錯的圖片時，利用複製貼上可以快速將圖片加到 Xcode 專案的 Assets。

不過彼得潘研究後，發現從 Safari & Chrome 複製圖片貼到 Assets 時，會有一些小小的差異。

從 Chrome 點選 Copy Image 後，回到 Xcode 點選 Assets 的側邊欄，然後按 cmd + v 即可貼上圖片。

然而同樣的操作，從 Safari 點選 Copy Image 再按 cmd + v 貼到 Assets 卻會失敗。

這應該算 Safari 的小小 bug，點選另一個選項 Copy Subject 是沒問題的，按下 cmd + v 可以將去背的圖片貼到 Assets。

如果還是偏愛 Safari，不喜歡 Chrome，想從 Safari Copy Image 也是可以，只是稍微麻煩一點，步驟如下。

  • 從 Safari 點選 Copy Image 複製圖片。
  • 點選 Assets 清單下方的 +，然後點選 Image Set。

輸入圖片名字。

點選 1x、2x 或 3x 的格子。

按下 cmd + v 將圖片貼到格子。
