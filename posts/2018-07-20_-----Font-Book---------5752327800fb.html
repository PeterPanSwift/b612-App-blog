如何利用 Font Book 知道字型的名字

想在 iOS App 裡使用特別的字型並不難，只要能將字體的 otf (Open Type Font)或 ttf(True Type Font) 檔加到專案裡，並在 App 的 info 頁面做設定。

為 iOS App 加入客製字型 (custom font)
覺得 iOS App 內建的中文字型不夠可愛嗎 ? 沒關係，接下來我們將從日本的字體網站，下載可愛的粉筆字體，在 iOS App 上顯示迷死人不償命的粉筆字 !medium.com

但有個小問題要解決，我們需知道字型的名字，才能在 Interface Builder 或程式裡設定。之前曾介紹過從程式利用 CGDataProvider 取得名字的方法，其實如果懶得寫程式，還有另一個簡單的做法。

利用 CGDataProvider 取得客製字型的名字
開發 iOS App 時，我們可以另外加入自己喜歡的字型檔。但如果想在程式裡指定我們另外加入的字型，還有個問題要解決，我們需找出字型的名字，因為它跟字型檔的檔名是不同的 !medium.com

1 雙擊字體檔，將它安裝到 Mac。

點選右下角的 Install Font。

2 此時 Mac 的 Font Book App 將顯示剛安裝字型的預覽。

3 點選左上角的 i 按鈕，查詢字型的相關資訊。

如下圖所示，Family 和 PostScript name 是我們想知道的名字。

在 Interface Builder 設定字體時使用 Family。

在程式設定時則用 PostScript name。

  • SwiftUI
Text("With great power, comes great responsibility")      
   .font(.custom("Nagurigaki-Crayon", size: 30))
  • UIKit
let font = UIFont(name: "Nagurigaki-Crayon", size: 19)
