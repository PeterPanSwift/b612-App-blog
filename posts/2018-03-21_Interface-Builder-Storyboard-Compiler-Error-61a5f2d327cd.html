Interface Builder Storyboard Compiler Error

開發 iOS App 時，有時會莫名奇妙出現 Interface Builder Storyboard Compiler Error 的錯誤，害得 App 跑不起來，但是明明程式和 storyboard 都沒有問題呀，甚至連重新建一個專案也還是出現錯誤。

此時不妨試試重新開機。就像我們如果重新投胎，有機會當上英國王子一樣，重開機之後，錯誤就消失了 !
