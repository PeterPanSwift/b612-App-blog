Flutter App 快速 wrap widget / remove widget / move widget 的方法

製作 Flutter App 畫面時，我們時常需要 wrap widget(用某個 widget 包另一個 widget) 和 remove widget(移除 widget)，因此 IDE 早為我們提供了方便 wrap widget、remove widget 和 move widget 的方法。

以下我們用 VS Code / Firebase Studio 示範三種常見的方法。(ps: Android Studio 也有類似的方法。)

  • 按 Quick Fix 的快速鍵。
  • 按燈泡(Code Action lightbulb)。
  • 從 Flutter Outline。

範例的的程式如下，我們想將原本置頂的圖片變成置中，因此目標是用 Center 包住 Image 裡。

import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: SafeArea(
          child: Image.asset("assets/pairProgramming.png"),
        ),
      ),
    );
  }
}

wrap widget

  • 方法 1: 按 Quick Fix 的快速鍵。

點選想包住的元件，在此我們點選 Image。

然後按住 cmd + . 鍵，從 Wrap with 選單點選 Wrap with Center。

ps: Windows 的快速鍵是 ctrl + .。

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: SafeArea(
          child: Center(child: Image.asset("assets/pairProgramming.png")),
        ),
      ),
    );
  }
}

Image 成功包在 Center 裡，圖片置中了 !

Wrap with xxx 選單提供許多常用的選項，像是 Center、Column、Row、Container、SizeBox、Padding 等。不過 Flutter 有那麼多的 widget，若是選單上找不到想加入的 widget，則有兩種選擇。

  1. 想要生成搭配 child 的 widget。

點選 Wrap with widget。

程式將生成 widget 的框框，搭配參數 child，我們可自由在框框填入 widget 的名字。

2. 想要生成搭配 children 的 widget。

點選 Wrap with Column，程式將生成搭配參數 children 的 Column，再將 Column 改成我們想要的 widget 名字。

Wrap 功能也可以一次包含多個 widget，比方選取下圖的兩個 Text 後，再從 Wrap with 選單點選。

  • 方法 2: 按燈泡(Code Action lightbulb)

點選 Image，然後點選左邊的燈泡，從選單點選 Wrap with Center。

  • 方法 3: 從 Flutter Outline

從 command palette 點選 Flutter: Focus on Outline View。

左下方出現 Flutter Outline。

從 Image 上的右鍵選單點選 Wrap with Center。

從右上的三個點也可叫出選單點選 Wrap with Center。

remove widget

除了 wrap widget，我們也可以 remove widget。比方我們想反悔，不想要圖片置中，透過剛剛的三個方法也可以 remove Center widget。

  • 方法 1 & 2。

點選 Center，然後按快速鍵或按燈泡，從選單點選 Remove this widget。

  • 方法 3。

打開 Flutter Outline，點選 Center，然後叫出選單點選 Remove this widget。

move widget

有時我們會想調整 Multi-child layout widget children 的順序，比方以下例子 Column 的 children。

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(
      home: Scaffold(
        body: SafeArea(
          child: Column(
            children: [
              Text('關於我轉生變成史萊姆這檔事'),
              Text('Re:從零開始的異世界生活'),
              Text('彼得潘')
            ],
          ),
        ),
      ),
    );
  }
}

點選 Text('Re:從零開始的異世界生活')，按燈泡出現選單後可看到選項 Move widget down 和 Move widget up。
