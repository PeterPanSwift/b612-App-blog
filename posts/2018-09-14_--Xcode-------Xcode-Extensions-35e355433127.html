幫 Xcode 擴充功能的 Xcode Extensions

就像蜘蛛人穿上新版的蜘蛛裝可以變得更厲害，Xcode 也可以安裝 Extension 變得更強大呢 !

在 Xcode 10，我們可以從 Xcode > Xcode Extensions 啟動 App Store，打開幫 Xcode 升級的 Extensions 介紹頁面，找尋喜歡的 extension 安裝。

接下來就讓我們下載免費的 PasteTheType，用它幫我們的 class 生成寫起來有點辛苦的 init 吧。

‎PasteTheType
‎Read reviews, compare customer ratings, see screenshots and learn more about PasteTheType. Download PasteTheType for…itunes.apple.com
1 點選右上角的 GET。

然後點選 INSTALL。

然後點選 OPEN。

Extension 成功安裝 !

可惜此時 Xcode 還不知道 PasteTheType 的存在，我們要先從 System Preferences App 解除它的封印。

2 打開 System Preferences App，點選 Extensions。
3 從左邊的側邊欄點選 Xcode Source Editor，然後在右邊的清單勾選 PasteTheType。
4 回到 Xcode，現在 menu Editor 下多了 PasteTheType !

PasteTheType 可以幫我們生成 init，比方以下 class Baby 的例子。

Class Baby 沒有 init，所以產生錯誤。沒關係，現在有了 PasteTheType，我們再也不用自己寫了。

1 選取 Baby 的 property age & name，然後按下 cmd + c 複製。
2 點選 Editor > PasteTheType > to init。

PasteTheType 聰明地從剛剛複製的 property 生成 init(age: Int, name: String) 的程式碼。從今以後就算 class 有一百個 property 我們也不用怕了，一百行 self.xxx = xxx 都可以無痛生成。

class Baby {
   var age: Int
   var name: String
   init(age: Int, name: String) {
      self.age = age
      self.name = name
   }
}
