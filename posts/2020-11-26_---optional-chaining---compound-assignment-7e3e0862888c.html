結合 optional chaining & compound assignment

當我們使用 optional chaining 時，讀取的內容會變成 optional，不過若是內容的型別當初宣告時不是 optional，我們將可方便地使用 compound assignment 做運算。

接下來就讓我們以增加 Baby 的年紀為例說明吧。

cuteBaby?.age 是 optional，因此無法加 1。此問題可透過 optional binding 修正。

struct Baby {
    var age = 1
}

var cuteBaby: Baby? = Baby()
if let age = cuteBaby?.age {
    cuteBaby?.age = age + 1
}

不過殺雞焉用牛刀，使用 compound assignment 可以更簡單呢。

struct Baby {
    var age = 1
}

var cuteBaby: Baby? = Baby()
cuteBaby?.age += 1
