使用 Chrome 快速全選複製 Apple 網頁的 Swift 範例程式

Apple 使用 DocC 生成許多優質的程式教學網頁，學習時的體驗超棒的，可以一步步呈現程式的變化和 App 畫面的預覽。

不過它有個小小缺點，如果想複製某一步全部的程式， 按下 cmd + a 全選會變成選取整個網頁。

若是只想選取整段程式，必須先點選第一行，然後再按住 shift 點選最後一行。有沒有更快的方法呢 ?

彼得潘研究一下後，發現手機版的網頁可以點選左上的檔名顯示全螢幕的程式，因此我們可在 Mac 上用 Chrome 瀏覽網頁，將它調成手機模式後，即可按 cmd + a 全選整段程式。

以下為操作的步驟。

從 Chrome 連到剛剛的網頁，從右鍵選單點選 Inspect。

點擊下圖紅色框框的 device 按鈕。

從左上將裝置的大小改成 iPhone 14 Pro Max。

點擊程式左上的檔名 MeetingView.swift。

程式變成全螢幕顯示。

按下 cmd + a 全選，然後再按 cmd + c 複製。

按下 cmd + v 貼上，成功取得整段程式，不過第一行會多出檔名，再將它刪除即可。

MeetingView.swift
import SwiftUI


struct MeetingView: View {
    var body: some View {
        VStack {
            ProgressView(value: 5, total: 15)
            HStack {
                VStack(alignment: .leading) {
                    Text("Seconds Elapsed")
                        .font(.caption)
                    Label("300", systemImage: "hourglass.tophalf.fill")
                }
                Spacer()
                VStack(alignment: .trailing) {
                    Text("Seconds Remaining")
                        .font(.caption)
                    Label("600", systemImage: "hourglass.bottomhalf.fill")
                }
            }
            Circle()
                .strokeBorder(lineWidth: 24)
            HStack {
                Text("Speaker 1 of 3")
                Spacer()
                Button(action: {}) {
                    Image(systemName: "forward.fill")
                }
            }
        }
        .padding()
    }
}
