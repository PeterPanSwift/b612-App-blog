Swift Programming Language Book — 閱讀世界上最棒的 Swift 程式書

想學習 Swift，當然要閱讀世界上最棒的 Swift 程式書呀。世界上最棒的Swift 程式書是誰寫的呢？

很可惜不是彼得潘。寫得最棒的當然是 Apple 官方寫的 Swift 電子書。

打開 Swift Programming Language Book

不用花錢，也不用走路出門去書局買，我們可以舒服地躺在床上，從 Xcode 打開 Swift Programming Language Book 即可開始一邊喝咖啡，一邊展開美好的 Swift 旅程。

  • 方法 1

從 Quick Actions 點選 Swift Programming Language Book。

  • 方法 2

從 menu 的 Help 點選 Swift Programming Language Book 。

快速切換到某個章節

從右上的 Search icon 或左下的 Filter 框框輸入章節關鍵字。

搜尋文章內容

使用網頁的搜尋功能搜尋文章內容。點選 cmd + f 打開上方的搜尋框，然後輸入文字搜尋頁面，比方下圖將搜尋 Closures 章節裡 escap 出現的地方。

取得章節裡某個小節的連結

點選小節的標題即可複製小節的連結。

比方 property observers 的連結如下。

https://docs.swift.org/swift-book/documentation/the-swift-programming-language/properties#Property-Observers

章節網址的規則

單元名稱全部小寫，有多個單字時連在一起。小節名稱則維持小節的大小寫，空白換成 — 。

因此如果想查詢 Closures 單元的 Escaping Closures，可輸入以下網址，單元名稱為 closures，小節名稱為 Escaping-Closures。

https://docs.swift.org/swift-book/documentation/the-swift-programming-language/closures/#Escaping-Closures
