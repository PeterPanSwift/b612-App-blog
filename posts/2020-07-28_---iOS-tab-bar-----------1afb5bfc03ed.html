storyboard & UITabBarController

storyboard，UITabBarController & storyboard reference

UITabBarController 連到 storyboard reference，每個 tab 分頁的畫面對應不同的 storyboard。

xib & UITabBarController

點選 Tab Bar Controller 下的 controller 可設計分頁的內容，設定 controller 的類別，將畫面上的元件拉 outlet & action 到 controller 的程式。

SceneDelegate.swift

生成 TabController.xib 的 UITabBarController，將它設為 App 一開始的畫面。

func scene(_ scene: UIScene, willConnectTo session: UISceneSession, options connectionOptions: UIScene.ConnectionOptions) {
    guard let windowScene = (scene as? UIWindowScene) else { return }
    window = UIWindow(frame: UIScreen.main.bounds)
    let tabController = Bundle.main.loadNibNamed("TabController", owner: nil, options: nil)?.first as? UITabBarController
    window?.rootViewController = tabController
    window?.makeKeyAndVisible()
    window?.windowScene = windowScene
}

從程式設定 UITabBarController

SceneDelegate.swift

從程式生成 UITabBarController，將每個分頁的 controller 組合成 array，存入 UITabBarController 的 viewControllers。

func scene(_ scene: UIScene, willConnectTo session: UISceneSession, options connectionOptions: UIScene.ConnectionOptions) {
       
    guard let windowScene = (scene as? UIWindowScene) else { return }
    window = UIWindow(frame: UIScreen.main.bounds)
        
    let tabController = UITabBarController()
    let firstVC = FirstViewController()
    firstVC.tabBarItem = UITabBarItem(title: "星矢", image: nil, selectedImage: nil)
    let secondVC = SecondViewController()
    secondVC.tabBarItem = UITabBarItem(title: "紫龍", image: nil, selectedImage: nil)
    let thirdVC = ThirdViewController()
    thirdVC.tabBarItem = UITabBarItem(title: "冰河", image: nil, selectedImage: nil)
    let fourthVC = FourthViewController()
    fourthVC.tabBarItem = UITabBarItem(title: "瞬", image: nil, selectedImage: nil)
    let fifthVC = FifthViewController()
    fifthVC.tabBarItem = UITabBarItem(title: "一輝", image: nil, selectedImage: nil)
    tabController.viewControllers = [
        firstVC,
        secondVC,
        thirdVC,
        fourthVC,
        fifthVC
     ]
     window?.rootViewController = tabController
     window?.makeKeyAndVisible()
     window?.windowScene = windowScene      
}

tab 下每個分頁的 controller 也可以搭配 UINavigationController，比方以下程式的第一個 tab 搭配 UINavigationController。

func scene(_ scene: UIScene, willConnectTo session: UISceneSession, options connectionOptions: UIScene.ConnectionOptions) {
           
     guard let windowScene = (scene as? UIWindowScene) else { return }
     window = UIWindow(frame: UIScreen.main.bounds)
            
     let tabController = UITabBarController()
     let firstVC = FirstViewController()
     let firstTabNavVC = UINavigationController(rootViewController: firstVC)      
     firstTabNavVC.tabBarItem = UITabBarItem(title: "星矢", image: nil, selectedImage: nil)
     let secondVC = SecondViewController()
     secondVC.tabBarItem = UITabBarItem(title: "紫龍", image: nil, selectedImage: nil)
     let thirdVC = ThirdViewController()
     thirdVC.tabBarItem = UITabBarItem(title: "冰河", image: nil, selectedImage: nil)
     let fourthVC = FourthViewController()
     fourthVC.tabBarItem = UITabBarItem(title: "瞬", image: nil, selectedImage: nil)
     let fifthVC = FifthViewController()
     fifthVC.tabBarItem = UITabBarItem(title: "一輝", image: nil, selectedImage: nil)
     tabController.viewControllers = [
         firstTabNavVC,
         secondVC,
         thirdVC,
         fourthVC,
         fifthVC
     ]
     window?.rootViewController = tabController
     window?.makeKeyAndVisible()
     window?.windowScene = windowScene          
}
