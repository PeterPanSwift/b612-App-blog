不分大小寫的字串包含判斷， localizedCaseInsensitiveContains

開發 iOS App 時，有時我們會用到不分大小寫的字串包含判斷，比方以下 TV App 搜尋影片時，我們輸入 Spi 或 spi 都可以找到蜘蛛人的電影。

我們可以利用 function localizedCaseInsensitiveContains 進行不分大小寫的字串包含判斷。

Apple Developer Documentation
Edit descriptiondeveloper.apple.com
  • 範例 1
"Spider Man".localizedCaseInsensitiveContains("Spi")
"Spider Man".localizedCaseInsensitiveContains("spi")
  • 範例 2
let heroes = ["Spider Man", "Thor", "Iron Man"]
let filterHeroes1 = heroes.filter { $0.localizedCaseInsensitiveContains("man")}
filterHeroes1
let filterHeroes2 = heroes.filter { $0.localizedCaseInsensitiveContains("man")}
filterHeroes2
