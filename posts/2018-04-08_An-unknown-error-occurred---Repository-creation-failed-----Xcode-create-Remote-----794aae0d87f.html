An unknown error occurred 或 Repository creation failed — 從 Xcode create Remote 的錯誤

有些朋友從 Xcode 點選 Create xxx Remote，準備將專案傳到 Remote 時，出現錯誤訊息 Repository creation failed 或 An unknown error occurred。

發生此問題的解法有兩種

解法1: 換個 Repository Name

你的 Remote 帳號曾經建立過名字一樣的 Repository Name， 所以請換個 Repository Name，即可解決此問題。

解法2: 先 commit 再 create Remote

如果 An unknown error occurred 出現錯誤訊息 reference refs/heads/master not found，表示你忘了 commit。請先做 commit，然後再 create Remote。
