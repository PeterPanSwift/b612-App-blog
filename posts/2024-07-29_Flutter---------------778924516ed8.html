Flutter 列印大量資料的文字截斷問題

當我們在 Flutter 列印大量資料時，它將發生文字截斷的問題。

用以下程式為列，將 0 ~ 199 的數字組合並以逗號分隔將可完整列印。

void main() {
  final message = List.generate(
    200,
    (index) => index.toString(),
  ).join(',');
  print(message);
  runApp(const MyApp());
}

然而將範圍擴充到 0 ~ 299 就失敗了，只能印到 276。

void main() {
  final message = List.generate(
    300,
    (index) => index.toString(),
  ).join(',');
  print(message);
  runApp(const MyApp());
}

因此若想查看包含大量資料的變數內容，我們必須使用其它方法。

  • 方法 1: 使用 log。
  • 方法 2: 設定中斷點查看變數內容。

使用 log

imprt 'dart:developer'，然後用 function log 列印。

import 'dart:developer';
import 'package:flutter/material.dart';

void main() {
  final message = List.generate(
    300,
    (index) => index.toString(),
  ).join(',');
  log(message);
  runApp(const MyApp());
}

設定中斷點查看變數內容

查看變數內容的方法有兩種。

  • 從右鍵選單點選 Copy Value。
  • 在 DEBUG CONSOLE 輸入變數名。
