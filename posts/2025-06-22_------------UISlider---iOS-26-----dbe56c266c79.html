可調刻度和可移除滑桿的 UISlider — iOS 26 新功能

UISlider 在 iOS 26 有許多不錯的新功能，最大的兩個亮點是可調刻度和可移除滑桿。

SwiftUI 的 Slider 原本就可設定刻度，比方滑動 slider 設定彼得潘的年紀，範圍是 18 ~ 25。我們希望年紀是整數，想要 slider 停在 19，20 這樣的整數，別停在 18.1，18.2 這種奇怪的年紀。

UIKit 的 UISlider 其實也能做到，只是我們必須自己寫程式控制它只能滑到某些位置。

設定 iOS App slider 的移動距離
iOS App 的 slider 預設是連續滑動的，比方 slider 的範圍是 18 ~ 20，因此當我們從 18 滑到 20 時，可以停在 18.1，18.2 等中間小數點的數字。medium.com

從 iOS 26 開始，UISlider 終於也內建設定刻度的功能，透過 slider 的 trackConfiguration 可設定刻度。

以下例子我們透過參數 numberOfTicks 設定範圍 0 ~ 2 的 slider 有 3 個刻度，因此滑桿只能滑到 0、1、2 這 3 個位置。

class ViewController: UIViewController {
    @IBOutlet weak var slider: UISlider!

    override func viewDidLoad() {
        super.viewDidLoad()
        // Do any additional setup after loading the view.

        slider.minimumValue = 0
        slider.maximumValue = 2
        slider.value = 0
        slider.trackConfiguration = .init(numberOfTicks: 3)
    }
}

另一種設定刻度位置的方法是透過參數 ticks，它的型別為 [UISlider.TrackConfiguration.Tick]。以下例子設定了 3 個刻度，position 的數字範圍是 0 ~ 1，因此 0、0.5、1 對應的刻度位置是 slider 的起點，中間點，跟終點。

slider.trackConfiguration = .init(ticks: [.init(position: 0), .init(position: 0.5), .init(position: 1)])

trackConfiguration 還可設定一個特別的參數 neutralValue，它將讓 slider 標示的顏色範圍變成 neutralValue 和滑桿間的範圍。neutralValue 的數字範圍是 0 ~ 1，因此以下例子 neutralValue 0.5 表示 slider 的中間點，顏色範圍變成滑桿到 slider 中間的區塊。

slider.minimumValue = 0
slider.maximumValue = 10
slider.value = 0
slider.trackConfiguration = .init(neutralValue: 0.5, numberOfTicks: 11)

除了刻度的亮點，UISlider 另一個新功能是 sliderStyle。將 sliderStyle 設為 thumbless 可移除滑桿，讓 slider 長得像進度條。

slider.sliderStyle = .thumbless

slider 滑動時還會有變粗的效果。

沒有滑桿的 slider 其實常常看到，iOS 內建的音樂 App 調整播放進度和音量使用的正是沒滑桿的 slider。
