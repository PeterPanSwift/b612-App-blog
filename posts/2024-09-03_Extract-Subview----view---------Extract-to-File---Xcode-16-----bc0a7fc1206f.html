Extract Subview 後將 view 型別搬到新檔的 Extract to File — Xcode 16 新功能

開發 SwiftUI App 時，我們常會將 subview 變成獨立的型別，增加程式的可讀性，方便未來重覆使用。

用以下程式的小王子角色介紹為例，每個角色由 VStack 搭配 Image 和 Text 組成，我們可將它變成獨立的型別 CharacterView。

import SwiftUI

struct ContentView: View {
    var body: some View {
        HStack {
            VStack {
                Image(.prince)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 100, height: 100)
                    .clipShape(.circle)
                    .shadow(color: .primary.opacity(0.7), radius: 20)
                Text("小王子")
            }
            VStack {
                Image(.fox)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 100, height: 100)
                    .clipShape(.circle)
                    .shadow(color: .primary.opacity(0.7), radius: 20)
                Text("狐狸")
            }
            VStack {
                Image(.rose)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 100, height: 100)
                    .clipShape(.circle)
                    .shadow(color: .primary.opacity(0.7), radius: 20)
                Text("玫瑰")
            }
        }
    }
}

將 VStack 的內容定義成顯示角色頭像和名字的 CharacterView，之後只要用 CharacterView 即可生成小王子故事的可愛角色。

struct ContentView: View {
    var body: some View {
        HStack {
            CharacterView(avatar: .prince, name: "小王子")
            CharacterView(avatar: .fox, name: "狐狸")
            CharacterView(avatar: .rose, name: "玫瑰")
        }
    }
}

struct CharacterView: View {
    let avatar: ImageResource
    let name: String
    
    var body: some View {
        VStack {
            Image(avatar)
                .resizable()
                .scaledToFill()
                .frame(width: 100, height: 100)
                .clipShape(.circle)
                .shadow(color: .primary.opacity(0.7), radius: 20)
            Text(name)
        }
    }
}

subview 的型別可以在同一檔案，也可在不同檔案。

從 Xcode 16 開始，我們可透過 Extract to File 將選取的程式搬到新檔。因此我們可在 CharacterView 上按出右鍵選單，點選 Refactor > Extract to File。

Xcode 聰明地將 struct CharacterView 搬到新檔 CharacterView.swift。

現在 ContentView.swift 的程式也變精簡了，只剩下 struct ContentView 和 preview。

import SwiftUI

struct ContentView: View {
    var body: some View {
        HStack {
            CharacterView(avatar: .prince, name: "小王子")
            CharacterView(avatar: .fox, name: "狐狸")
            CharacterView(avatar: .rose, name: "玫瑰")
        }
    }
}

#Preview(traits: .sizeThatFitsLayout) {
    ContentView()
}
