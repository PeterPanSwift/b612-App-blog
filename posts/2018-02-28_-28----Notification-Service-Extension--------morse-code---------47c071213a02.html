#28 利用 Notification Service Extension 解密摩斯密碼(morse code)推播和下載圖片

目的: 學習利用 Notification Service Extension 修改推播內容後顯示。

假設推播的內容是摩斯密碼, 請利用 Notification Service Extension 解密和下載圖片後在推播上顯示，比方 .. .-.. — — …- . -. — — — ..- 將變成 i love you。

{
   "aps" : {
      "alert" : "..   .-.. --- ...- .   -.-- --- ..-",
   }
}

利用 Notification Service Extension，我們可在顯示推播前，修改推播的內容，實現以下功能:

  • 解密被加密的推播訊息
  • 下載推播的圖片或影片後，顯示在推播上
  • 修改顯示的文字
