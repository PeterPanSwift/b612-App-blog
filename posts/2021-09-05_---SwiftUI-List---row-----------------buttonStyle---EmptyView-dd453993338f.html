設定 SwiftUI List 的 row 有多個可點選的區塊 — 利用 buttonStyle & EmptyView

設計 iOS App 時，我們時常利用表格(List)顯示豐富的資料內容。如下圖所示，小說精選列表顯示一本本精彩的小說，點選作者名跳到作者頁面，點選整本書的 row 跳到書本頁面。

在 SwiftUI 利用 NavigationStack，List & NavigationLink 可以快速實現點選切換頁面的列表。但若想讓一個 row 裡有多個可點選的區塊，點選會做不同的事，卻需要額外費一番功夫。接下來我們將示範如何讓 List 的 row 有多個可點選的區塊。

新版做法

點選 List row 不同區塊跳到不同頁面 — 使用 NavigationStack 搭配 path 或 navigationDestination(item:destination:)

以下是比較麻煩的舊版做法。

準備動作: 製作一個可以點選 row 跳到下一頁的表格

定義 Book

import Foundation

struct Book: Identifiable {
    let id = UUID()
    let name: String
    let author: String
}
extension Book {
    static var demoBook: Book {
        Book(name: "Lush Life", author: "伊坂幸太郎")
    }
}

定義 BookView

BookView 將顯示書本的詳細內容。

import SwiftUI

struct BookView: View {
    let book: Book
    
    var body: some View {
        VStack {
            Image(book.name)
                .resizable()
                .scaledToFit()
            Text(book.author)
        }
        .navigationTitle(book.name)
    }
}

#Preview {
    NavigationStack {
        BookView(book: .demoBook)
    }
}

定義 BookRow

表格顯示的每一本書以 BookRow 設計。

import SwiftUI

struct BookRow: View {
    let book: Book
   
    var body: some View {
        VStack {
            HStack {
                Image(book.name)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 100, height: 100)
                    .clipped()
                Text(book.name)
                Spacer()
                Text(book.author)
                
            }
            Button {
                print("share")
            } label: {
                Image(systemName: "square.and.arrow.up")
            }
        }
    }
}

#Preview(traits: .sizeThatFitsLayout) {
    BookRow(book: .demoBook)
}

定義 BookList

顯示小說清單的 BookList。

import SwiftUI

struct BookList: View {
    let books = [
        Book(name: "Lush Life", author: "伊坂幸太郎"),
        Book(name: "白井小姐", author: "乙一")
    ]
   
    var body: some View {
        NavigationStack {
            List {
                ForEach(books) { book in
                    NavigationLink {
                        BookView(book: book)
                    } label: {
                        BookRow(book: book)
                    }
                }
                
            }
            .navigationTitle("小說精選")
        }
    }
}

#Preview {
    BookList()
}

執行 App

點選表格裡的小說將跳到顯示大圖的小說頁面。

現在我們已經做出一個可以點選的表格，不過它有以下缺點:

  • 點選 row 的任何位置都會跳到書本頁面。待會我們將修改程式，實現點選作者時跳到作者頁面，點選 row 的其它位置時跳到書本頁面。
  • 點選分享的 button 也會跳到書本頁面，不會觸發分享 button 設定的 closure 程式。
  • row 的右邊顯示向右的箭頭(disclosure indicator)。待會我們將修改程式，將箭頭藏起來。

讓 NavigationLink 裡的 button 可以單獨點選 — buttonStyle(.plain)

List 的 row 加了 NavigationLink 後，預設會變成點選 row 裡的任何地方都會跳到下一頁。若想讓 row 上的 button 單獨點選，不會觸發切換頁面，則須另外在 NavigationLink 上呼叫 .buttonStyle(.plain)。

NavigationLink {
    BookView(book: book)
} label: {
    BookRow(book: book)
}
.buttonStyle(.plain)

執行 App

現在點選分享 button 可以做別的事情，不會再跳到書本頁面了。

點選 row 裡不同地方跳到不同的頁面 & 隱藏向右的箭頭

我們想要點選作者時跳到作者頁面，點選 row 的其它位置時跳到書本頁面，並且隱藏向右的箭頭(disclosure indicator)。

定義 AuthorView

AuthorView 顯示作者的相關資訊，在此我們偷懶一下，只顯示他最重要的資訊，名字 !

import SwiftUI

struct AuthorView: View {
    let author: String
    var body: some View {
        Text(author)
            .font(.system(size: 40))
    }
}

#Preview {
    AuthorView(author: "伊坂幸太郎")
}

修改 BookList

import SwiftUI

struct BookList: View {
    let books = [
        Book(name: "Lush Life", author: "伊坂幸太郎"),
        Book(name: "白井小姐", author: "乙一")
    ]
    @State private var selectAuthor: String?
    
    var body: some View {
        NavigationStack {
            List {
                ForEach(books) { book in
                    ZStack {
                        
                        BookRow(book: book, selectAuthor: $selectAuthor)
                        
                        NavigationLink {
                            BookView(book: book)
                        } label: {
                            EmptyView()
                        }
                        .opacity(0)
                        
                        NavigationLink(tag: book.author, selection: $selectAuthor) {
                            AuthorView(author: book.author)
                        } label: {
                            EmptyView()
                        }
                        .hidden()
                    }
                    .buttonStyle(.plain)
                }
                
            }
            .navigationTitle("小說精選")
        }
    }
}

#Preview {
    BookList()
}

說明。

@State private var selectAuthor: String?

宣告 selectAuthor 儲存點選的作者名稱。

ZStack {           
     BookRow(book: book, selectAuthor: $selectAuthor)
         .buttonStyle(.plain)
                        
     NavigationLink {
         BookView(book: book)
     } label: {
         EmptyView()
     }
     .opacity(0)
                        
     NavigationLink(tag: book.author, selection: $selectAuthor) {
         AuthorView(author: book.author)
     } label: {
         EmptyView()
     }
     .hidden()
}
.buttonStyle(.plain)

以 ZStack 包裝顯示的 row，ZStack 裡包含 BookRow 和 2 個 NavigationLink。第一個 NavigationLink 顯示 EmptyView，透明度設為 0，它將在 BookRow 蓋上一層看不到但可點選的區塊，點選後將跳到 BookView，而且 row 上也不會出現向右的箭頭。

第二個 NavigationLink 設為 hidden，因此它不能點選，它的目的是控制作者頁面的顯示。在此我們幫每個 row 的 NavigationLink 加上 tag，以 book.author 當 tag。如下圖所示，第一個 row 的 NavigationLink tag 為伊坂幸太郎，第二個為乙一。

NavigationLink(tag: book.author, selection: $selectAuthor) {
   AuthorView(author: book.author)
} label: {
   EmptyView()
}
.hidden()

參數 selection 控制著 NavigationLink 的切換頁面是否觸發。我們在參數 selection 傳入 $selectAuthor，當 selectAuthor 有值時，它將觸發 tag 等於 selectAuthor 的 NavigationLink 切換頁面。

舉例來說，當 selectAuthor 等於伊坂幸太郎時，將觸發第一個 row 的 NavigationLink 切換頁面，因為它的 NavigationLink tag 為伊坂幸太郎。

現在我們只差最後一步，在使用者點選作者名字時設定 selectAuthor，讓我們回頭修改 BookRow。

修改 BookRow

import SwiftUI

struct BookRow: View {
    let book: Book
    @Binding var selectAuthor: String?
    var body: some View {
        VStack {
            HStack {
                Image(book.name)
                    .resizable()
                    .scaledToFill()
                    .frame(width: 100, height: 150)
                    .clipped()
                Text(book.name)
                Spacer()
                Button {
                    selectAuthor = book.author
                } label: {
                    Text(book.author)
                }
                .foregroundColor(.blue)
            }
            Button {
                print("share")
            } label: {
                Image(systemName: "square.and.arrow.up")
            }
            .foregroundStyle(.blue)
        }
        
    }
}

#Preview(traits: .sizeThatFitsLayout) {
    BookRow(book: .demoBook, selectAuthor: .constant(nil))
}

執行 App

現在一個 row 裡終於可以有多個可點選的區塊了，點作者名跳到作者頁面，點分享 button 觸發分享功能，點其它區塊跳到書本頁面。
