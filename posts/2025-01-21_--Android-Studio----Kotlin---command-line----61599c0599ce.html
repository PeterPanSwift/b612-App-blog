從 Android Studio 練習 Kotlin 的 command line 程式

Android Studio 為了開發 Android App 而生，不過剛開始學習開發 Android App 時，我們通常會從 command line 程式練習 Kotlin 語法。

以下我們介紹兩種在 Android Studio 練習 Kotlin command line 程式的方法。

  • 方法 1: 在 Empty Activity 專案定義 main function。
  • 方法 2: 在 No Activity 專案新增 Kotlin 檔，然後再定義 main function。

方法 1: 在 Empty Activity 專案定義 main function

建立專案時選擇開發 Android App 常用的 Empty Activity。

在 MainActivity.kt 定義 main function。

fun main() {
    print("輸入你的名字: ")
    val name = readln()
    print("認識你真好，$name")
}

執行 main function，方法有兩種。

  • 方法 1: 點選 main function 旁的三角形。

選擇 Run 'MainActivityKt'。

  • 方法 2: 上方的 Run / Debug Configurations 選擇 MainActivityKt。

點選三角形。

執行後下方會出現 Console 區塊，它將顯示程式列印的內容，我們也能在其中輸入程式需要的資料。

方法 2: 在 No Activity 專案新增 Kotlin 檔，然後再定義 main function

建立專案時選擇 No Activity。

No Activity 預設不會有任何 Kotlin 檔，因此我們要自己新增。

從下圖的紅色框框叫出右鍵選單。

點選 New > Kotlin Class/File。

選擇 File，輸入自訂的檔名。在此我們輸入 Demo。

生成 Demo.kt。

定義 main function。

fun main() {
    print("輸入你的名字: ")
    val name = readln()
    print("認識你真好，$name")
}

接下來跟方法 1 一樣，點選 main function 旁的三角形即可執行程式。
